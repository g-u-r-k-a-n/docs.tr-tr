## <a name="example"></a><span data-ttu-id="7ec2c-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="7ec2c-105">Example</span></span>  

 <span data-ttu-id="7ec2c-106">Aşağıdaki örnek CS1686 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7ec2c-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
