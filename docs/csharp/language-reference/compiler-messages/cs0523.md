# <a name="compiler-error-cs0523"></a><span data-ttu-id="4203f-102">Derleyici Hatası CS0523</span><span class="sxs-lookup"><span data-stu-id="4203f-102">Compiler Error CS0523</span></span>
<span data-ttu-id="4203f-103">' Struct1 ' türünün ' Struct2 Field ' yapı üyesi yapı düzeninde döngüye neden oluyor</span><span class="sxs-lookup"><span data-stu-id="4203f-103">Struct member 'struct2 field' of type 'struct1' causes a cycle in the struct layout</span></span>  
  
 <span data-ttu-id="4203f-104">İki yapının tanımları özyinelemeli başvuruları içerir.</span><span class="sxs-lookup"><span data-stu-id="4203f-104">The definitions of two structs include recursive references.</span></span> <span data-ttu-id="4203f-105">[Yapı](../builtin-types/struct.md) tanımlarını, her biri diğeri üzerinde tanımlamayan şekilde değiştirin.</span><span class="sxs-lookup"><span data-stu-id="4203f-105">Change the [struct](../builtin-types/struct.md) definitions such that each does not define itself on the other.</span></span> <span data-ttu-id="4203f-106">Yapılar değer türleri olduğundan, bu sınırlama yalnızca yapılar için geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="4203f-106">This limitation applies only to structs, since structs are value types.</span></span> <span data-ttu-id="4203f-107">Özyinelemeli başvuruları kullanıyorsanız, türlerinizi sınıflar olarak bildirin.</span><span class="sxs-lookup"><span data-stu-id="4203f-107">If using recursive references, declare your types as classes.</span></span>  
  
 <span data-ttu-id="4203f-108">Aşağıdaki örnek CS0523 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4203f-108">The following sample generates CS0523:</span></span>  
  
```csharp  
// CS0523.cs  
// compile with: /target:library

{  
   public RecursiveLayoutStruct2 field;  
}  
  
struct RecursiveLayoutStruct2  
{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
