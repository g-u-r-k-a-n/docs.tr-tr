# <a name="how-to-autogenerate-columns-in-a-data-bound-windows-forms-datagridview-control"></a>Nasıl yapılır: Veri Bağlantılı Windows Forms DataGridView Denetiminde Sütunları Otomatik Olarak Oluşturma
Aşağıdaki kod örneği, <xref:System.Windows.Forms.DataGridView> denetimindeki bir bağlantılı veri kaynağından sütunların nasıl görüntüleneceğini gösterir. <xref:System.Windows.Forms.DataGridView.AutoGenerateColumns%2A> Özellik değeri `true` (varsayılan) olduğunda, veri kaynağı tablosundaki her sütun için bir <xref:System.Windows.Forms.DataGridViewColumn> oluşturulur.  
  
 <xref:System.Windows.Forms.DataGridViewComboBoxColumn.DataSource%2A> özelliğini ayarladığınızda <xref:System.Windows.Forms.DataGridView> denetiminde zaten sütun varsa, varolan bağlantılı sütunlar veri kaynağındaki sütunlarla karşılaştırılır ve bir eşleşme olduğunda korunur. İlişkisiz sütunlar her zaman korunur. Veri kaynağında eşleşme olmayan bağlantılı sütunlar kaldırılır. Denetimde eşleşme olmayan veri kaynağındaki sütunlar, <xref:System.Windows.Forms.DataGridView.Columns%2A> koleksiyonun sonuna eklenen yeni <xref:System.Windows.Forms.DataGridViewColumn> nesneleri oluşturur.
