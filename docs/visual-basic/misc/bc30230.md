# <a name="inherits-not-valid-in-modules"></a>' Inherits ' modüllerde geçerli değildir
Bir `Inherits` modül içinde bir ifade oluşur. Modüller sınıflardan devralınabilir.  
  
 **Hata kimliği:** BC30230  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfadeyi kaldırın `Inherits` veya modülü bir sınıf olarak yeniden yazın.
