# <a name="compiler-warning-level-2-cs0467"></a><span data-ttu-id="99bc8-102">Derleyici Uyarısı (düzey 2) CS0467</span><span class="sxs-lookup"><span data-stu-id="99bc8-102">Compiler Warning (level 2) CS0467</span></span>
<span data-ttu-id="99bc8-103">Yöntem 'yöntem' ve yöntem dışı 'non-method' arasındaki belirsizlik.</span><span class="sxs-lookup"><span data-stu-id="99bc8-103">Ambiguity between method 'method' and non-method 'non-method'.</span></span> <span data-ttu-id="99bc8-104">Yöntem grubunu kullanma.</span><span class="sxs-lookup"><span data-stu-id="99bc8-104">Using method group.</span></span>  
  
 <span data-ttu-id="99bc8-105">Aynı imzaya sahip farklı arabirimlerden devralınan üyeler belirsizlik hatasına neden olur.</span><span class="sxs-lookup"><span data-stu-id="99bc8-105">Inherited members from different interfaces that have the same signature  cause an ambiguity error.</span></span>  
  
## <a name="example"></a><span data-ttu-id="99bc8-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="99bc8-106">Example</span></span>  
 <span data-ttu-id="99bc8-107">Aşağıdaki örnek, CS0467 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="99bc8-107">The following example generates CS0467.</span></span>  
  
```csharp  
// CS0467.cs  
interface IList
{  
    int Count { get; set; }  
}  
  
interface ICounter  
{  
    void Count(int i);  
}  
  
interface IListCounter : IList, ICounter {}  
  
class Driver
{  
    void Test(IListCounter x)  
    {  
        // The following line causes the warning. The assignment also  
        // causes an error because you can't assign a value to a method.  
        x.Count = 1;  
        x.Count(3);
        // To resolve the warning, you can change the name of the method or
        // the property.  
  
        // You also can disambiguate by specifying IList or ICounter.  
        ((IList)x).Count = 1;  
        ((ICounter)x).Count(3);  
    }  
  
    static void Main()
    {  
    }  
}  
```
