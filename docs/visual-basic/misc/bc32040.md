# <a name="the-keyword-keyword-is-used-to-overload-inherited-members-do-not-use-the-keyword-keyword-when-overloading-sub-new"></a>' \<keyword> ' Anahtar sözcüğü devralınan üyeleri aşırı yüklemek için kullanılır; \<keyword> ' Sub New ' öğesini aşırı yüklerken ' ' anahtar sözcüğünü kullanmayın

Bir Oluşturucu anahtar sözcüğüyle birlikte bildirilmiştir `Overloads` .  
  
 Visual Basic oluşturucuların devralınmasını veya aşırı yüklenmesini desteklemez.  
  
 **Hata kimliği:** BC32040  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Overloads`Tüm Oluşturucu bildirimlerinden anahtar sözcüğünü kaldırın.
