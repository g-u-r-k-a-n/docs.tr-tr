# <a name="coiniticor-enumeration"></a>COINITICOR Numaralandırması

## <a name="syntax"></a>Sözdizimi  
  
```cpp  
typedef enum tagCOINITCOR  
{  
    COINITCOR = 0x0  
} COINITICOR;  
```
