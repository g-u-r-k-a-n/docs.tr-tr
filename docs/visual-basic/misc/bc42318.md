# <a name="xml-comment-type-parameter-must-have-a-name-attribute"></a>XML açıklama türü parametresinin bir ' name ' özniteliği olmalıdır

XML açıklama türü parametresinin bir ' name ' özniteliği olmalıdır. XML açıklaması yoksayılacak.  
  
 XML açıklama etiketi, `name` eksik veya tamamlanmamış bir özniteliği gerektiriyor.  
  
 **Hata kimliği:** BC42318  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Etiketine gerekli özniteliği ekleyin veya bu `name` özniteliği doldurun.
