# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>' End operator ' satırdaki ilk ifade olmalıdır

Bir `End Operator` ifade, iki nokta üst üste ( `:` ) bildirim ayırıcısını izler. `End Operator` kaynak satırındaki tek bir ifade olmalıdır.  
  
 **Hata kimliği:** BC33006  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyimi farklı satırlara ayırın.
