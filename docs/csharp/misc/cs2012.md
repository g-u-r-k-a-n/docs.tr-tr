# <a name="compiler-error-cs2012"></a>Derleyici hatası CS2012

' Dosya ' yazma için açılamıyor  
  
 [-Bugreport:](../language-reference/compiler-options/bugreport-compiler-option.md) `file` derleyici seçeneğini kullanırken dosya yazma için açılamadı. Geçerli bir dosya adı belirttiğinizden ve dosyanın salt okunmadığından emin olun.
