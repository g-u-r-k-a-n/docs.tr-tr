# <a name="property-parameters-cannot-have-the-name-value"></a>Özellik parametrelerinin adı ' Value ' olamaz
Deyimdeki bir parametre `Property` adı `Value` . Bu ad, yordamda özelliğe atanmakta olan yeni değer için bir özellik tanımı içinde ayrılmıştır `Set` .  
  
 **Hata kimliği:** BC32201  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Deyimdeki parametrenin adını değiştirin `Property` .
