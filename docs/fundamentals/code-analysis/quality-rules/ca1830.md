## <a name="how-to-fix-violations"></a><span data-ttu-id="95c43-116">İhlalleri çözme</span><span class="sxs-lookup"><span data-stu-id="95c43-116">How to fix violations</span></span>

<span data-ttu-id="95c43-117">Çağrıdan gereksiz öğesini silin `ToString()` .</span><span class="sxs-lookup"><span data-stu-id="95c43-117">Delete the unnecessary `ToString()` from the invocation.</span></span>

```csharp
using System.Text;

class C
{
    int _value;

    // Violation
    public void Log(StringBuilder destination)
    {
        destination.Append("Value: ").Append(_value.ToString()).AppendLine();
    }

    // Fixed
    public void Log(StringBuilder destination)
    {
        destination.Append("Value: ").Append(_value).AppendLine();
    }
}
```
