# <a name="fusion-structures"></a>Fusion Yapıları

Bu bölümde Fusion API 'sinin kullandığı yönetilmeyen yapılar açıklanmaktadır.  
  
## <a name="in-this-section"></a>Bu Bölümde  

 [ASSEMBLY_INFO Yapısı](assembly-info-structure.md)  
 Genel derleme önbelleğinde kayıtlı olan bir derleme hakkındaki bilgileri içerir.  
  
 [FUSION_INSTALL_REFERENCE Yapısı](fusion-install-reference-structure.md)  
 Uygulamanın, uygulamanın genel derleme önbelleğinde yüklediği bir derlemeye yaptığı bir başvuruyu temsil eder.  
  
 [IDENTITY_ATTRIBUTE Yapısı](identity-attribute-structure.md)  
 Bir [IDefinitionIdentity](idefinitionidentity-interface.md) örneği hakkında meta veri özniteliği bilgilerini içerir.  
  
 [IDENTITY_ATTRIBUTE_BLOB Yapısı](identity-attribute-blob-structure.md)  
 İlişkili bir [IDENTITY_ATTRIBUTE](identity-attribute-structure.md) yapısına ilişkin konum bilgilerini içerir.
