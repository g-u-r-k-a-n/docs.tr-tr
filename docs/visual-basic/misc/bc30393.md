# <a name="exit-try-can-only-appear-inside-a-try-statement"></a>' Exit Try ' yalnızca bir ' TRY ' ifadesinin içinde yer alabilir

`Exit Try` yalnızca bir `Try` Block ifadesinde bulunabilir. Bir yedekli `Exit Try` deyime sahipsiniz ya da `Exit Try` deyiminiz karşılık gelen bloğunun sınırları dışında görünüyor `Try` .  
  
 **Hata kimliği:** BC30393  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Gereksiz deyimin yerini bulun ve kaldırın `Exit Try` .  
  
2. `Exit Try`Deyiminizi kodunuzun içindeki uygun yere taşıyın.
