# <a name="compiler-error-cs1032"></a>Derleyici hatası CS1032

Dosyadaki ilk belirteçten sonra önişlemci sembolleri tanımlanamaz/tanımlanamıyor  
  
 `#define`Ve `#undef` [Önişlemci yönergeleri](../language-reference/preprocessor-directives/index.md) , bir programın başlangıcında, ad alanı bildiriminde kullanılanlar gibi diğer anahtar sözcüklerden önce kullanılmalıdır.  
  
 Aşağıdaki örnek CS1032 oluşturur:  
  
```csharp  
// CS1032.cs  
namespace x  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace  
      public static void Main()  
      {  
      }  
   }  
}  
```
