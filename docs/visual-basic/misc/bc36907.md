<span data-ttu-id="fc2ad-103">Bir tür bağımsız değişkeni, genel parametreleri olmayan ya da bir türün zaten belirtilmediği genel parametreleri olmayan bir genişletme yöntemine yapılan çağrıda belirtildi.</span><span class="sxs-lookup"><span data-stu-id="fc2ad-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="fc2ad-104">Örneğin, aşağıdaki kod bu hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="fc2ad-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _

    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
