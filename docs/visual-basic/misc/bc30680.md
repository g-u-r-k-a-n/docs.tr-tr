# <a name="end-region-must-be-preceded-by-a-matching-region"></a>' #End Region ' öncesinde eşleşen bir ' #Region ' olmalıdır

İle `#Region` Visual Studio Code düzenleyicisinin ana hat özelliğini kullanırken genişleyebilir veya daraltılacak bir kod bloğu belirtebilirsiniz. Başlangıç ve bitiş `#Region` deyimleri aynı kod bloğunda olmalıdır.  
  
 **Hata kimliği:** BC30680  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `#Region`Karşılık gelen deyimden önce uygun yere ekleyin `#End Region` .
