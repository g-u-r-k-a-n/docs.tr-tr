# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebound-overload-resolution"></a>.NET Compact Framework hedeflenen sürümü, geç bağlanan aşırı yükleme çözümlemesini desteklemiyor

.NET Compact Framework hedeflenen sürümü, geç bağlantılı aşırı yükleme çözümlemesini desteklemiyor.  
  
 **Hata kimliği:** BC30764  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesne türündeki parametreleri uygun türe atayın.
