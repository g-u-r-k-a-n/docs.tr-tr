# <a name="type-typename-is-not-supported-because-it-either-directly-or-indirectly-inherits-from-itself"></a>' \<typename> ' Türü, doğrudan veya dolaylı olarak kendisinden devraldığı için desteklenmiyor
Bir sınıf veya arabirim kendisinden veya sonunda devralan başka bir sınıftan ya da arabirimden devralır.  
  
 Visual Basic, dairesel devralmayı desteklemez.  
  
 **Hata kimliği:** BC30916  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Devralma yapısını, başka bir sınıftan veya arabirimden devralmayan bir taban sınıf veya arabirim üzerinde oluşturulmuş olacak şekilde değiştirin.
