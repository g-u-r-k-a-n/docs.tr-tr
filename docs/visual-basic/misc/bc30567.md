# <a name="array-initializer-is-missing-number-elements"></a>Dizi başlatıcıda \<number> öğe yok

Diziniz için başlatıcıda yeterli sayıda öğe sağlamadınız.  
  
 **Hata kimliği:** BC30567  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Eksik bağımsız değişkeni veya bağımsız değişkenleri sağlayın.  
  
- `ReDim`Dizinin boyutunu değiştirmek için kullanın.
