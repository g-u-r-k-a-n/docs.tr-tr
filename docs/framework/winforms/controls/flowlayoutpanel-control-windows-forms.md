# <a name="flowlayoutpanel-control-windows-forms"></a>FlowLayoutPanel Denetimi (Windows Forms)
`FlowLayoutPanel` denetimi içeriğini yatay veya dikey bir akış yönünde düzenler. İçeriği bir satırdan sonrakine veya bir sütundan sonrakine kaydırılmış olabilir. Alternatif olarak, içeriği sarmalanabilir yerine kırpılabilir.  
  
 Bu bölümdeki konularda, uygulamalarınızda `FlowLayoutPanel` özellikler oluşturmanıza izin veren kavramlar ve teknikler açıklanır.
