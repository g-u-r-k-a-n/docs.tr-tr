# <a name="cannot-rename-with-different-drive"></a>Farklı sürücü ile yeniden adlandırılamıyor

`Rename`İfade dosya yolunda bir sürücü harfi belirtemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sürücü harfini dosya yolundan kaldırın.
