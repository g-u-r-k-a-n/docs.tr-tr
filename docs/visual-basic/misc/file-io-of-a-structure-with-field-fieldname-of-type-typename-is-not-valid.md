# <a name="file-io-of-a-structure-with-field-fieldname-of-type-typename-is-not-valid"></a>' ' Türündeki ' ' alanına sahip bir yapının dosya g/ç 'i \<fieldname> \<typename> geçerli değil

Bir dosyaya yazmaya çalıştığınız yapı desteklenmeyen bir tür içeriyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Desteklenmeyen türün nerede oluştuğunu öğrenmek için yapıyı denetleyin ve sonra geçerli bir türle değiştirin.
