# <a name="type-expected"></a>Tür bekleniyor

Kaynak kodu karakterleri tür adı olarak tanınabilir olmayan bir durum oluşur. Bağlam, bir türü temsil eden bir Öğesel veri türü veya tanımlanmış bir öğe adı için çağırır.  
  
 **Hata kimliği:** BC30182  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Türün tanımlandığından emin olun.  
  
- Tür adının doğru yazıldığından emin olun.
