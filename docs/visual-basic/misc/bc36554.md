## <a name="to-correct-this-error"></a><span data-ttu-id="13606-112">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="13606-112">To correct this error</span></span>

- <span data-ttu-id="13606-113">Parametre dizisi genişletmek istediğiniz veri türünü belirtmezse, bu türü belirten yeni bir ilk parametre ekleyin.</span><span class="sxs-lookup"><span data-stu-id="13606-113">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>

  ```vb
  <Extension()>
  Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)
  ' Concatenate the strings in addOns to str.
  End Sub
  ```

- <span data-ttu-id="13606-114">Parametre dizisi genişletmek istediğiniz veri türünü belirtse, bir parametre dizisi yerine bir bağımsız değişken gerektiren normal bir dizi olarak değiştirmeyi düşünün.</span><span class="sxs-lookup"><span data-stu-id="13606-114">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="13606-115">Normal diziler genişletilebilir.</span><span class="sxs-lookup"><span data-stu-id="13606-115">Regular arrays can be extended.</span></span>

  ```vb
  <Extension()>
  Public Function Sum(ByVal ints() As Integer) As Integer
      Dim total As Integer = 0
      For Each i As Integer In ints
          total = total + i
      Next i
      Return total
  End Function
  ```
