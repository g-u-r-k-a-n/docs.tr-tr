# <a name="catch-block-never-reached-exception-handled-above-in-the-same-try-statement"></a>' Catch ' bloğuna hiç ulaşılamadı; \<exception>yukarıda aynı ' TRY ' ifadesinde işlendi
`Catch`Önceki bir blokta işlendiği için koddaki bloğa ulaşılamıyor `Try` .  
  
Bu ileti, varsayılan olarak bir uyarıdır. Uyarıları gizleme veya uyarıları hata olarak değerlendirme hakkında daha fazla bilgi için bkz. [Visual Basic uyarıları yapılandırma](/visualstudio/ide/configuring-warnings-in-visual-basic).
  
 **Hata kimliği:** BC42031  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ifadesini kaldırın.
