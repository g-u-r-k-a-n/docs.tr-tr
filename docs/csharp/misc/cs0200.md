## <a name="examples"></a><span data-ttu-id="5a4ef-107">Örnekler</span><span class="sxs-lookup"><span data-stu-id="5a4ef-107">Examples</span></span>

<span data-ttu-id="5a4ef-108">Aşağıdaki örnek CS0200 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5a4ef-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="5a4ef-109">Aşağıdaki örnek [Otomatik uygulanan özellikleri](../programming-guide/classes-and-structs/auto-implemented-properties.md) ve [nesne başlatıcıları](../programming-guide/classes-and-structs/object-and-collection-initializers.md) KULLANıR ve yine de CS0200 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5a4ef-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md) and [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md) and still generates CS0200:</span></span>

```csharp

{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }

```

<span data-ttu-id="5a4ef-110">Salt okunurdur ' Property ' özelliğine veya dizin oluşturucusuna atamak için bir küme erişimcisi ekleyin veya nesne oluşturucusunda değeri atayın.</span><span class="sxs-lookup"><span data-stu-id="5a4ef-110">To assign to a property or indexer 'property' that's read-only, add a set accessor or assign the value in the object's constructor.</span></span>

    {

    }
