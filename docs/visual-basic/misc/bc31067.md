# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>Yapıdaki metot ' protected ' veya ' Protected Friend ' olarak bildirilemez
Yapıdaki bir yöntem veya olarak bildirildi `Protected` `Protected Friend` .  
  
 **Hata kimliği:** BC31067  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Veya gibi yapılar içinde yöntemleri `Public` bildirin `Private` .
