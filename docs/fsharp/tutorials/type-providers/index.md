# <a name="type-providers"></a><span data-ttu-id="64f95-103">Tür Sağlayıcıları</span><span class="sxs-lookup"><span data-stu-id="64f95-103">Type Providers</span></span>

<span data-ttu-id="64f95-104">Bir F# tür sağlayıcısı programınız içinde kullanmanız için türler, özellikler ve yöntemler sağlayan bir bileşendir.</span><span class="sxs-lookup"><span data-stu-id="64f95-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="64f95-105">Tür sağlayıcıları, F # derleyicisi tarafından oluşturulan ve bir dış veri kaynağını temel alan, **sunulan türler**olarak bilinen öğeleri oluşturur.</span><span class="sxs-lookup"><span data-stu-id="64f95-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="64f95-106">Örneğin, SQL için bir F # tür sağlayıcısı, ilişkisel bir veritabanındaki tabloları ve sütunları temsil eden türler oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="64f95-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="64f95-107">Aslında, bu, [SqlProvider](https://fsprojects.github.io/SQLProvider/) tür sağlayıcısının yaptığı şeydir.</span><span class="sxs-lookup"><span data-stu-id="64f95-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="64f95-108">Sunulan türler, bir tür sağlayıcısına giriş parametrelerine bağımlıdır.</span><span class="sxs-lookup"><span data-stu-id="64f95-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="64f95-109">Bu giriş bir örnek veri kaynağı (bir JSON şema dosyası gibi), doğrudan bir dış hizmete işaret eden bir URL veya bir veri kaynağına bağlantı dizesi olabilir.</span><span class="sxs-lookup"><span data-stu-id="64f95-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="64f95-110">Bir tür sağlayıcısı ayrıca tür gruplarının isteğe bağlı olarak genişletildiğinden de emin olabilir; diğer bir deyişle, türlerine gerçekten programınızın başvurduğu durumlarda genişletilir.</span><span class="sxs-lookup"><span data-stu-id="64f95-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="64f95-111">Bu çevrimiçi veri marketleri gibi büyük ölçekli bilgi uzaylarının doğrudan, istek anında bütünleştirmesini türü kesin belirlenmiş olarak sağlar.</span><span class="sxs-lookup"><span data-stu-id="64f95-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="64f95-112">Genel ve silinmiş tür sağlayıcıları</span><span class="sxs-lookup"><span data-stu-id="64f95-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="64f95-113">Tür sağlayıcıları iki şekilde gelir: genel ve silinmiş.</span><span class="sxs-lookup"><span data-stu-id="64f95-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="64f95-114">Genel tür sağlayıcıları, üretildikleri derlemeye .NET türleri olarak yazılabilen türler üretir.</span><span class="sxs-lookup"><span data-stu-id="64f95-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="64f95-115">Bu, diğer derlemelerdeki koddan tüketilebilmesi için izin verir.</span><span class="sxs-lookup"><span data-stu-id="64f95-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="64f95-116">Bu, veri kaynağının türü belirlenmiş gösteriminin genellikle .NET türleriyle temsil edilebilir bir değer olması gerektiği anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="64f95-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="64f95-117">Tür sağlayıcılarının silinmesinin, yalnızca oluşturulduğu derlemede veya projede tüketilebilen türler üretir.</span><span class="sxs-lookup"><span data-stu-id="64f95-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="64f95-118">Türler kısa ömürlü; diğer bir deyişle, bir derlemeye yazılmazlar ve diğer derlemelerdeki kodla tüketilemez.</span><span class="sxs-lookup"><span data-stu-id="64f95-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="64f95-119">Bu kişiler *gecikmeli* Üyeler içerebilir ve bu bilgiler, potansiyel olarak sonsuz bir bilgi alanından sağlanmış türleri kullanmanıza olanak sağlar.</span><span class="sxs-lookup"><span data-stu-id="64f95-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="64f95-120">Büyük ve bağlanmış bir veri kaynağının küçük bir alt kümesini kullanmak için faydalıdır.</span><span class="sxs-lookup"><span data-stu-id="64f95-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
