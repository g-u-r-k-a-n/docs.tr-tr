# <a name="compiler-error-cs1708"></a><span data-ttu-id="d7556-102">Derleyici Hatası CS1708</span><span class="sxs-lookup"><span data-stu-id="d7556-102">Compiler Error CS1708</span></span>
<span data-ttu-id="d7556-103">Sabit boyut arabelleklerine yalnızca yerel halk veya alanlar üzerinden erişilebilir</span><span class="sxs-lookup"><span data-stu-id="d7556-103">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="d7556-104">C# 2.0'daki yeni bir özellik, bir `struct`.'ün içinde bir satır içi diziyi tanımlayabilme yeteneğidir.</span><span class="sxs-lookup"><span data-stu-id="d7556-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="d7556-105">Bu tür dizilere yalnızca yerel değişkenler veya alanlar üzerinden erişilebilir ve bir ifadenin sol tarafındaki ara değerler olarak başvurulabilir.</span><span class="sxs-lookup"><span data-stu-id="d7556-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="d7556-106">Ayrıca, diziler veya `static` . `readonly`</span><span class="sxs-lookup"><span data-stu-id="d7556-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="d7556-107">Bu hatayı gidermek için bir dizi değişkeni tanımlayın ve satır içi diziyi değişkene atayın.</span><span class="sxs-lookup"><span data-stu-id="d7556-107">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="d7556-108">Veya, satır `static` `readonly` içi diziyi temsil eden alandan veya değiştiriyi kaldırın.</span><span class="sxs-lookup"><span data-stu-id="d7556-108">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d7556-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="d7556-109">Example</span></span>  
 <span data-ttu-id="d7556-110">Aşağıdaki örnek CS1708 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="d7556-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
