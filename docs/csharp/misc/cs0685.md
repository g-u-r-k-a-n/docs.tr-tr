# <a name="compiler-error-cs0685"></a><span data-ttu-id="04416-103">Derleyici hatası CS0685</span><span class="sxs-lookup"><span data-stu-id="04416-103">Compiler Error CS0685</span></span>

<span data-ttu-id="04416-104">' Member ' koşullu üyesinde out parametresi olamaz</span><span class="sxs-lookup"><span data-stu-id="04416-104">Conditional member 'member' cannot have an out parameter</span></span>  
  
 <span data-ttu-id="04416-105"><xref:System.Diagnostics.ConditionalAttribute>Özniteliği bir yöntemde kullanılırken, bu yöntemin out parametresi bulunmayabilir.</span><span class="sxs-lookup"><span data-stu-id="04416-105">When using the <xref:System.Diagnostics.ConditionalAttribute> attribute on a method, that method may not have an out parameter.</span></span> <span data-ttu-id="04416-106">Bunun nedeni, out parametresi için kullanılan değişken değerinin, yöntem çağrısının Nothing olarak derlenmesi durumunda tanımlanmamalıdır.</span><span class="sxs-lookup"><span data-stu-id="04416-106">This is because the value of the variable used for the out parameter would not be defined in the case that the method call is compiled to nothing.</span></span> <span data-ttu-id="04416-107">Bu hatayı önlemek için, koşullu yöntem bildiriminden Out parametresini kaldırın veya koşullu özniteliği kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="04416-107">To avoid this error, remove the out parameter from the conditional method declaration, or don't use the Conditional Attribute.</span></span>
