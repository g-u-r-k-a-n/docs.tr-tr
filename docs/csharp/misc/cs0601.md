# <a name="compiler-error-cs0601"></a>Derleyici hatası CS0601
DllImport özniteliği ' static ' ve ' extern ' olarak işaretlenmiş bir yöntemde belirtilmelidir  
  
 `DllImport`Öznitelik, doğru erişim anahtar sözcüklere sahip olmayan bir yöntemde kullanıldı.  
  
 Aşağıdaki örnek CS0601 oluşturur:  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
