# <a name="property-array-index-is-not-valid"></a>Özellik dizisi dizini geçerli değil

Sağlanan değer, bir özellik dizisi dizini için geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizininizin belirtilen özellik için geçerli aralıkta olduğundan emin olmak için bileşenin belgelerini denetleyin.
