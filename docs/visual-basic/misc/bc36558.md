## <a name="example"></a><span data-ttu-id="4c089-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="4c089-111">Example</span></span>

<span data-ttu-id="4c089-112">Aşağıdaki örnek, `AttributeUsage` yeni sürümünün hangi program öğelerine uygulanacağını belirtmek için özniteliğini kullanır `ExtensionAttribute` .</span><span class="sxs-lookup"><span data-stu-id="4c089-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="4c089-113">Örnek, numaralandırmanın üç üyesini belirtir `AttributeTargets` : `Assembly` , `Class` ve `Method` .</span><span class="sxs-lookup"><span data-stu-id="4c089-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="4c089-114">Bu öğelerden herhangi birinin atlanması bu hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="4c089-114">The omission of any one of these elements will cause this error.</span></span>

```vb
Namespace System.Runtime.CompilerServices
    <AttributeUsage(AttributeTargets.Assembly Or _
        AttributeTargets.Class Or AttributeTargets.Method)>
    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```

<span data-ttu-id="4c089-115">Alternatif olarak, `ExtensionAttribute` öğesinin üyesini kullanarak tüm program öğelerine uygulanmasına izin verebilirsiniz `All` `AttributeTargets` .</span><span class="sxs-lookup"><span data-stu-id="4c089-115">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>

```xml

```

<span data-ttu-id="4c089-116">`AttributeUsage`Aşağıdaki kodda gösterildiği gibi satırı silmek, aynı sonucu üretir.</span><span class="sxs-lookup"><span data-stu-id="4c089-116">Deleting the `AttributeUsage` line, as shown in the following code, produces the same result.</span></span>
