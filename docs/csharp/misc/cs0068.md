 <span data-ttu-id="fdc3a-107">Aşağıdaki örnek CS0068 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="fdc3a-107">The following sample generates CS0068:</span></span>  
  
```csharp

{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068

{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
