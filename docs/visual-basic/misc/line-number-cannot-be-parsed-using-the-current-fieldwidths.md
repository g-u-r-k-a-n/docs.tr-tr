# <a name="line-number-cannot-be-parsed-using-the-current-fieldwidths"></a>Satır \<number> , geçerli Fieldgenişlikleri kullanılarak ayrıştırılamıyor

Belirtilen satır, alanları belirtilenden farklı genişlikler içerdiğinden ayrıştırılamıyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Satırı `FieldWidths` işlemek için satırın doğru ayrıştırılabilmesi için ayarlayın veya özel durum işleme kodu ekleyin.
