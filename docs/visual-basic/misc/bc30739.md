# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>' RaiseEvent ' deyimleri Komut penceresinde geçerli değildir
`RaiseEvent` deyimleri **komut** penceresinde kullanılamaz.  
  
 **Hata kimliği:** BC30739  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- **Kod düzenleyicisindeki**koddan test `RaiseEvent` deyimleri.
