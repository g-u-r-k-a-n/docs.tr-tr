# <a name="end-removehandler-must-be-preceded-by-a-matching-removehandler-declaration"></a>' End RemoveHandler ' öncesinde eşleşen bir ' RemoveHandler ' bildirimi olmalıdır

`End RemoveHandler`Karşılık gelen bir bildirime sahip olmayan bir ifade oluştu `RemoveHandler` . `End RemoveHandler` önce karşılık gelen bir `RemoveHandler` deyimden önce gelmelidir.  
  
 **Hata kimliği:** BC31125  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Önceki `RemoveHandler` deyimin geçerli ve doğru yazıldığından emin olun.
