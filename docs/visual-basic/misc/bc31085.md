# <a name="date-constant-is-not-valid"></a>Tarih sabiti geçerli değil

Tarih sabiti geçersiz bir tarihle birlikte bildiriliyor.  
  
 **Hata kimliği:** BC31085  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sabitte kullanılan tarihi düzeltin.
