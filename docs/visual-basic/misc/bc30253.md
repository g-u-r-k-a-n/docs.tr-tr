# <a name="interface-must-end-with-a-matching-end-interface"></a>' Interface ', eşleşen bir ' End Interface ' ile bitmelidir

`Interface`Karşılık gelen bir ifade olmadan bir ifade oluşur `End Interface` . `End Interface`Arabirim tanımını sonlandırmak için bir ifade kullanmanız gerekir.  
  
 **Hata kimliği:** BC30253  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Interface`Üyelerin doğru biçimlendirildiğinden emin olun.  
  
2. `End Interface`Arabirim tanımının sonuna bir ifade ekleyin.
