# <a name="compiler-warning-level-3-cs0675"></a><span data-ttu-id="2fd71-102">Derleyici Uyarısı (düzey 3) CS0675</span><span class="sxs-lookup"><span data-stu-id="2fd71-102">Compiler Warning (level 3) CS0675</span></span>
<span data-ttu-id="2fd71-103">Bitwise veya operatör bir işaret genişletilmiş operand kullanılan; önce daha küçük bir imzasız türe döküm yapmayı düşünün</span><span class="sxs-lookup"><span data-stu-id="2fd71-103">Bitwise-or operator used on a sign-extended operand; consider casting to a smaller unsigned type first</span></span>  
  
 <span data-ttu-id="2fd71-104">Derleyici örtülü olarak genişledi ve bir değişkeni işaretledi ve sonra elde edilen değeri bitwise VEYA işleminde kullandı.</span><span class="sxs-lookup"><span data-stu-id="2fd71-104">The compiler implicitly widened and sign-extended a variable, and then used the resulting value in a bitwise OR operation.</span></span> <span data-ttu-id="2fd71-105">Bu beklenmeyen davranışlara neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="2fd71-105">This can result in unexpected behavior.</span></span>  
  
 <span data-ttu-id="2fd71-106">Aşağıdaki örnek CS0675 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2fd71-106">The following sample generates CS0675:</span></span>  
  
```csharp  
// CS0675.cs  
// compile with: /W:3  
using System;  
  
public class sign  
{  
   public static void Main()  
   {  
      int hi = 1;  
      int lo = 1;  
      long value = (((long)hi) << 32) | lo;              // CS0675  
      // try the following line instead  
      // long value = (((long)hi) << 32) | ((uint)lo);   // correct  
   }  
}  
```
