 <span data-ttu-id="3e299-105">Bu hata, genel bir sınıf aynı genel arabirimde birden fazla Parametreleştirme uygularsa ve iki arabirimi özdeş hale getiren bir tür parametresi değişimi varsa oluşur.</span><span class="sxs-lookup"><span data-stu-id="3e299-105">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="3e299-106">Bu hatayı önlemek için arabirimlerden yalnızca birini uygulayın veya çakışmayı önlemek için tür parametrelerini değiştirin.</span><span class="sxs-lookup"><span data-stu-id="3e299-106">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>
