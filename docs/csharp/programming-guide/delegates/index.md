- <span data-ttu-id="0a55d-121">Temsilciler C++ işlev işaretçilerine benzer, ancak temsilciler tamamen nesne yönelimlidir ve üye işlevlere c++ işaretçilerinin aksine, temsilciler hem nesne örneğini hem de yöntemi kapsüller.</span><span class="sxs-lookup"><span data-stu-id="0a55d-121">Delegates are similar to C++ function pointers, but delegates are fully object-oriented, and unlike C++ pointers to member functions, delegates encapsulate both an object instance and a method.</span></span>
