# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Win32 bildirimini katıştırma hatası:/win32manifest seçeneği/nowin32manifest ile çakışıyor
Visual Basic Derleyicisi hem `/win32manifest` derleyici seçeneği hem de `/nowin32manifest` derleyici seçeneği geçirildi. Bu seçeneklerden yalnızca biri Visual Basic derleyicisine geçirilebilir.  
  
 **Hata kimliği:** BC2033  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. @No__t_0 derleyici seçeneğini veya `/nowin32manifest` derleyici seçeneğini kaldırın.
