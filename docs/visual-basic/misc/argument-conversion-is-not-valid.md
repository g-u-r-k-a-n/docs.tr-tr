# <a name="argument-conversion-is-not-valid"></a>' Dönüştürme ' bağımsız değişkeni geçerli değil

`VbStrConv`Numaralandırma değeri, muhtemelen sıfırdan küçük olduğu için geçerli değil. Sabit listesi değerleri sıfırdan büyük veya sıfıra eşit olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Değerin sıfırdan büyük veya sıfıra eşit olduğundan emin olun.
