# <a name="compiler-error-cs0407"></a><span data-ttu-id="f4eba-103">Derleyici hatası CS0407</span><span class="sxs-lookup"><span data-stu-id="f4eba-103">Compiler Error CS0407</span></span>

<span data-ttu-id="f4eba-104">' Return-Type Method ' yanlış dönüş türüne sahip</span><span class="sxs-lookup"><span data-stu-id="f4eba-104">'return-type method' has the wrong return type</span></span>  
  
 <span data-ttu-id="f4eba-105">Yöntem, temsilci türüyle uyumlu değildi.</span><span class="sxs-lookup"><span data-stu-id="f4eba-105">The method was not compatible with the delegate type.</span></span> <span data-ttu-id="f4eba-106">Bağımsız değişken türleri eşleşti, ancak dönüş türü bu temsilci için doğru dönüş türü değildi.</span><span class="sxs-lookup"><span data-stu-id="f4eba-106">The argument types matched, but the return type was not the correct return type for that delegate.</span></span> <span data-ttu-id="f4eba-107">Bu hatayı önlemek için, farklı bir yöntem kullanın, yöntemin dönüş türünü değiştirin ya da temsilcinin dönüş türünü değiştirin.</span><span class="sxs-lookup"><span data-stu-id="f4eba-107">To avoid this error, use a different method, change the method's return type, or change the delegate's return type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f4eba-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="f4eba-108">Example</span></span>  

 <span data-ttu-id="f4eba-109">Aşağıdaki örnek CS0407 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f4eba-109">The following sample generates CS0407:</span></span>  
  
```csharp  
// CS0407.cs  
public delegate int MyDelegate();  
  
class C  
{  
    MyDelegate d;  
  
    public C()  
    {  
        d = new MyDelegate(F);  // OK: F returns int  
        d = new MyDelegate(G);  // CS0407 – G doesn't return int  
    }  
  
    public int F()  
    {  
        return 1;  
    }  
  
    public void G()  
    {  
    }  
  
    public static void Main()  
    {  
        C c1 = new C();  
    }  
}  
```
