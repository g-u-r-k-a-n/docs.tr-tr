# <a name="compiler-error-cs1560"></a>Derleyici hatası CS1560
Önişlemci yönergesi için geçersiz dosya adı belirtildi. Dosya adı çok uzun veya geçerli bir dosya adı değil  
  
 [#Line](../language-reference/preprocessor-directives/preprocessor-line.md) ile belirtilen dosya adı _MAX_PATH (256 karakter) veya `#line` bulunan satırı 2000 karakteri aştı.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1560 oluşturur.  
  
```csharp  
// cs1560.cs
using System;
class MyClass
{
   public static void Main()
   {
      Console.WriteLine("Normal line #1.");
      #line 21 "MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890.txt"   // CS1560  
    }  
}  
```
