# <a name="property-value-is-not-valid"></a>Özellik değeri geçerli değil

Bir nesne veya denetimin özelliklerinden biri, aralığı dışında bir değere ayarlandı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özellik değerini geçerli bir ayarla değiştirin.
