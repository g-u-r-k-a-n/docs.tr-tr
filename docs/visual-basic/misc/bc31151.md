# <a name="element-is-missing-an-end-tag"></a>Öğede bir bitiş etiketi eksik

XML öğesi sabit değerinde bir XML End etiketi veya boş bir etiket tanımlayıcısı '/' eksik (örneğin, `<tagName />` ).  
  
 **Hata kimliği:** BC31151  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML etiketi için bitiş etiketini ekleyin veya XML etiketine boş bir etiket tanımlayıcısı '/' ekleyin.
