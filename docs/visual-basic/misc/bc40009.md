# <a name="possible-problem-detected-while-building-assembly-error"></a>Derleme oluşturulurken olası bir sorun algılandı: \<error>

Visual Basic Derleyicisi tarafından çağrılan ALINK Aracı, derlemeyi derlerken bir hata bildiriyor. Olası bir neden, imzasız bir derlemeye başvuru yapan imzalı bir derlemedir.  
  
 Bu ileti bir uyarıdır. Derleyici derlemeyi oluşturmaya devam ediyor. Uyarıları gizleme veya uyarıları hata olarak değerlendirme hakkında daha fazla bilgi için lütfen bkz. [Visual Basic uyarıları yapılandırma](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Hata kimliği:** BC40009  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Alıntı yapılan hata iletisini inceleyin ve uygun eylemi gerçekleştirin.  
  
2. Hatanın yinelenip yinelenmeyeceğini görmek için programı yeniden derleyin.  
  
3. Hata yinelenirse Visual Basic derleyicisini yeniden yükleyin.  
  
4. Yeniden yükleme sonrasında hata devam ederse, koşullara ilişkin bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
