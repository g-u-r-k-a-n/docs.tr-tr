# <a name="argument-access-is-not-valid-valid-values-for-output-mode-are-openaccesswrite-and-openaccessdefault"></a>' Access ' bağımsız değişkeni geçerli değil. Çıkış modu için geçerli değerler şunlardır ' OpenAccess. Write ' ve ' OpenAccess. Default '

Çıkış modunda bir erişim değeri geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Değeri ya da olarak değiştirin `OpenAccess.Write` `OpenAccess.Default` .
