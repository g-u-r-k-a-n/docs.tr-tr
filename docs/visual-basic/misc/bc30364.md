# <a name="sub-new-cannot-be-declared-modifier"></a>' Sub New ', ' ' olarak bildirilemez \<modifier>

`Sub New`Geçerli olmayan bir değiştirici ile bir (gibi) bildirmeye çalıştınız `Overrides` .  
  
 **Hata kimliği:** BC30364  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştiricisini kaldırın.
