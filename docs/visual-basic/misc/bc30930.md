# <a name="name-necessary-for-compiling-this-construct-cannot-be-found"></a>\<name>Bu yapıyı derlemek için gerekli olan ' ' bulunamadı

Derleme için gereken bir öğe bulunamıyor.  
  
 **Hata kimliği:** BC30930  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Gerekli yardımcı öğeyi sağlayın.  
  
2. Kodun geçerli yönetilen kod olduğundan emin olun.
