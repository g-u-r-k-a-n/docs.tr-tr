# <a name="attribute-attributename-is-not-valid-error"></a>' \<attributename> ' Özniteliği geçerli değil: \<error>

Ortak dil çalışma zamanı, geçerli bağlamdaki özniteliği kabul etmez.  
  
 **Hata kimliği:** BC30127  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özniteliği bildirimden kaldırın veya bağlamı için geçerli olan bir öznitelik ile değiştirin.  
  
2. Hata iletisini inceleyin ve uygun eylemi gerçekleştirin.
