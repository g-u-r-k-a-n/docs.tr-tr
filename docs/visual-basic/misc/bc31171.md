# <a name="character-sequence----is-not-allowed-in-an-xml-comment"></a>XML açıklamasında '--' karakter dizisine izin verilmez

XML açıklama değişmez değeri, desteklenmeyen '--' karakter dizisini içeriyor.  
  
 **Hata kimliği:** BC31171  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- '--' Karakter dizisini XML açıklama değişmez değerinden kaldırın.
