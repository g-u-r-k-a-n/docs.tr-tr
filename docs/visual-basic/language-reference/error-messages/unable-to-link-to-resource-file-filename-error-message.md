# <a name="unable-to-link-to-resource-file-filename-error-message"></a>'\<filename>' kaynak dosyasına bağlanamıyor: \<error message>

Visual Basic derleyici, bildirime sahip bir derleme oluşturmak için derleme bağlayıcı (Al.exe, Alink olarak da bilinir) çağırır. Bağlayıcı, derlemeden yerel bir COM+ kaynak dosyasına bağlanırken bir hata bildirdi.  
  
 **Hata kimliği:** BC30144  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Alıntı yapılan hata iletisini inceleyin ve [Al.exe](../../../framework/tools/al-exe-assembly-linker.md)konusuna başvurun. daha fazla açıklama ve öneri.  
  
2. Hata devam ederse, koşullar hakkındaki bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
