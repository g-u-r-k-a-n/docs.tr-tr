# <a name="classes-cannot-be-declared-specifier"></a>Sınıflar ' ' olarak bildirilemez \<specifier>

Bir `Class` veya `Module` geçersiz tanımlayıcı ile bir veya daha fazla bildirmeye çalıştınız.  
  
 **Hata kimliği:** BC30461  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirteci kaldırın.
