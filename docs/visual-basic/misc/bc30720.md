# <a name="on-error-statements-are-not-valid-in-the-immediate-window"></a>' On Error ' deyimleri Komut penceresinde geçerli değildir

`On Error` deyimler yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30720  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `On Error`Hata ayıklama kodundaki deyimleri kaldır.
