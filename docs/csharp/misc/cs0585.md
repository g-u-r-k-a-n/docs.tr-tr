# <a name="compiler-error-cs0585"></a>Derleyici hatası CS0585

Derleyici iç hatası: Aşama ' aşama '

 Derleyicinin beklenmeyen sözdizimini ayrıştırmamasından dolayı başarısız olup olmadığını belirlemeyi deneyin. Böyle bir durum söz konusu değilse [bize konuşmayı](/visualstudio/ide/feedback-options)deneyin.
 
> [!NOTE]
> Bu derleyici hatası artık Roslyn içinde kullanılmıyor.
