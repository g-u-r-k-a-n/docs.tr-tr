# <a name="lib-expected"></a>' Lib ' bekleniyor

Bir `Declare` ifade, anahtar sözcüğünü kullanmadan bir kitaplık adı belirtir `Lib` .  
  
 **Hata kimliği:** BC30218  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Lib`Kitaplık adından önce anahtar sözcüğünü ekleyin.
