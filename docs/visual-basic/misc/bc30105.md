# <a name="number-of-indices-is-less-than-the-number-of-dimensions-of-the-indexed-array"></a>Dizin sayısı, dizinlenmiş dizinin boyut sayısından daha az

Bir dizi öğesine erişmek için kullanılan dizin sayısı, dizi sırasıyla tam olarak aynı olmalıdır, diğer bir deyişle, kendisi için belirtilen boyut sayısı.  
  
 **Hata kimliği:** BC30105  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Alt simgelerin toplam sayısı dizi derecesine eşit olana kadar dizi başvurusuna alt simge ekleyin.
