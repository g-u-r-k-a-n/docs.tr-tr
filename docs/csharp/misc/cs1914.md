## <a name="example"></a><span data-ttu-id="f5a6d-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="f5a6d-109">Example</span></span>  
 <span data-ttu-id="f5a6d-110">Aşağıdaki kod CS1914 oluşturur, çünkü baş harfalan `TestClass.Number` alanı başlatmaya `static`çalışır:</span><span class="sxs-lookup"><span data-stu-id="f5a6d-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
