# <a name="unused-local-constant-constant"></a>Kullanılmayan yerel sabit: ' \<constant> '
Kodunuzda herhangi bir yerde kullanılmayan yerel bir sabit bildirilmiştir.  
  
 **Hata kimliği:** BC42099  
  
### <a name="to-remove-this-warning"></a>Bu uyarıyı kaldırmak için  
  
1. Bu hata bir uyarıdır ve yoksayılabilir. Uyarıyı kaldırmak için, koddan kullanılmayan sabit bildirimini kaldırın.
