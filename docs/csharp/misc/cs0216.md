# <a name="compiler-error-cs0216"></a><span data-ttu-id="138be-103">Derleyici hatası CS0216</span><span class="sxs-lookup"><span data-stu-id="138be-103">Compiler Error CS0216</span></span>

<span data-ttu-id="138be-104">' Operator ' işleci, eşleşen ' missing_operator ' işlecinin de tanımlanmasını gerektirir</span><span class="sxs-lookup"><span data-stu-id="138be-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="138be-105">Kullanıcı tanımlı bir [==](../language-reference/operators/equality-operators.md#equality-operator-) operatör Kullanıcı tanımlı [! =](../language-reference/operators/equality-operators.md#inequality-operator-) işleci gerektirir ve tam tersi de geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="138be-105">A user-defined [==](../language-reference/operators/equality-operators.md#equality-operator-) operator requires a user-defined [!=](../language-reference/operators/equality-operators.md#inequality-operator-) operator, and vice versa.</span></span>  
 <span data-ttu-id="138be-106">Aynı zamanda Kullanıcı tanımlı bir [true](../language-reference/operators/true-false-operators.md) işlecine ve Kullanıcı tanımlı bir [false](../language-reference/operators/true-false-operators.md) işlecine de uygulanır.</span><span class="sxs-lookup"><span data-stu-id="138be-106">The same applies also to a user-defined [true](../language-reference/operators/true-false-operators.md) operator and a user-defined [false](../language-reference/operators/true-false-operators.md) operator.</span></span>  
  
 <span data-ttu-id="138be-107">Aşağıdaki örnek CS0216 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="138be-107">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator == (MyClass MyIntLeft, MyClass MyIntRight)   // CS0216  
   {  
      return MyIntLeft == MyIntRight;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator != (MyClass MyIntLeft, MyClass MyIntRight)  
   {  
      return MyIntLeft != MyIntRight;  
   }  
   */  
  
   public override bool Equals (object obj)  
   {  
      return base.Equals (obj);  
   }  
  
   public override int GetHashCode()  
   {  
      return base.GetHashCode();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
