# <a name="compiler-error-cs0315"></a><span data-ttu-id="1bbe3-103">Derleyici hatası CS0315</span><span class="sxs-lookup"><span data-stu-id="1bbe3-103">Compiler Error CS0315</span></span>

<span data-ttu-id="1bbe3-104">' ValueType ' türü, ' TypeorMethod ' genel türünde veya yönteminde ' 'T ' tür parametresi olarak kullanılamaz \<T> .</span><span class="sxs-lookup"><span data-stu-id="1bbe3-104">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="1bbe3-105">' ValueType ' öğesinden ' referenceType ' öğesine paketleme dönüştürmesi yok.</span><span class="sxs-lookup"><span data-stu-id="1bbe3-105">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="1bbe3-106">Bu hata, genel bir türü belirli bir sınıfa kısıtlamak ve kendisine örtülü olarak kutulanmamış bir değer türü kullanarak bu sınıfın bir örneğini oluşturmaya çalıştığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="1bbe3-106">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="1bbe3-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="1bbe3-107">To correct this error</span></span>  
  
1. <span data-ttu-id="1bbe3-108">Bir çözüm, yapıyı bir sınıf olarak yeniden tanımlamanız.</span><span class="sxs-lookup"><span data-stu-id="1bbe3-108">One solution is to redefine the struct as a class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1bbe3-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="1bbe3-109">Example</span></span>  

 <span data-ttu-id="1bbe3-110">Aşağıdaki örnek CS0315 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1bbe3-110">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
