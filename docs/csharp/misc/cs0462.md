# <a name="compiler-error-cs0462"></a><span data-ttu-id="38ea8-102">Derleyici Hatası CS0462</span><span class="sxs-lookup"><span data-stu-id="38ea8-102">Compiler Error CS0462</span></span>
<span data-ttu-id="38ea8-103">Devralınan üyeler 'member1' ve 'member2' türünde aynı imzaya sahiptir, bu nedenle geçersiz kılınmaz</span><span class="sxs-lookup"><span data-stu-id="38ea8-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="38ea8-104">Bu hata jenerik lerin getirilmesiyle ortaya çıkar.</span><span class="sxs-lookup"><span data-stu-id="38ea8-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="38ea8-105">Normalde, aynı imzaya sahip bir sınıfta bir yöntemin iki sürümü olamaz.</span><span class="sxs-lookup"><span data-stu-id="38ea8-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="38ea8-106">Ancak genel bilgilerle, belirli bir türle anında kullanılabilirse başka bir yöntemi yineleyebilecek genel bir yöntem belirtebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="38ea8-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="38ea8-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="38ea8-107">Example</span></span>  
 <span data-ttu-id="38ea8-108">Anında `C<int>` yapıldığında, yöntemin `F` iki sürümü aynı imzayla oluşturulur, bu nedenle sınıftaki `D` geçersiz kılma geçersiz kılma nın hangisine uygulanacağına karar veremez.</span><span class="sxs-lookup"><span data-stu-id="38ea8-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="38ea8-109">Aşağıdaki örnek CS0462 üretir.</span><span class="sxs-lookup"><span data-stu-id="38ea8-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
