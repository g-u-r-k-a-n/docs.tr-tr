# <a name="compiler-warning-level-2-cs0472"></a>Derleyici Uyarısı (seviye 2) CS0472
'value2' türündeki bir değer hiçbir zaman 'değer3' türünün 'null'una eşit olmadığından ifadenin sonucu her zaman 'değer1' dir.  
  
 Derleyici, sabit null değeri olan bir işleç kullanırsanız uyarmalıdır.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0472 üretir.  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;
        return counter;  
    }  
}  
```
