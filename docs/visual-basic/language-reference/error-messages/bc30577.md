# <a name="bc30577-addressof-operand-must-be-the-name-of-a-method-without-parentheses"></a><span data-ttu-id="578f7-102">BC30577: ' AddressOf ' işleneni bir metot adı olmalıdır (parantezler olmadan)</span><span class="sxs-lookup"><span data-stu-id="578f7-102">BC30577: 'AddressOf' operand must be the name of a method (without parentheses)</span></span>

<span data-ttu-id="578f7-103">`AddressOf`İşleci, belirli bir yordama başvuran bir yordam temsilci örneği oluşturur.</span><span class="sxs-lookup"><span data-stu-id="578f7-103">The `AddressOf` operator creates a procedure delegate instance that references a specific procedure.</span></span> <span data-ttu-id="578f7-104">Söz dizimi şu şekildedir:</span><span class="sxs-lookup"><span data-stu-id="578f7-104">The syntax is as follows:</span></span>

```vb
AddressOf procedurename
```

<span data-ttu-id="578f7-105">Aşağıdaki bağımsız değişkeni çevresine parantez eklediniz `AddressOf` , burada hiçbiri gerekli değildir.</span><span class="sxs-lookup"><span data-stu-id="578f7-105">You inserted parentheses around the argument following `AddressOf`, where none are needed.</span></span>

<span data-ttu-id="578f7-106">**Hata kimliği:** BC30577</span><span class="sxs-lookup"><span data-stu-id="578f7-106">**Error ID:** BC30577</span></span>

## <a name="example"></a><span data-ttu-id="578f7-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="578f7-107">Example</span></span>

<span data-ttu-id="578f7-108">Aşağıdaki örnek BC30577 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="578f7-108">The following example generates bc30577:</span></span>

```vb
Public Sub CountZeroToTen()
    For i = 0 To 10
        Console.WriteLine($"Counted: {i}")
        Threading.Thread.Sleep(500)
    Next
End Sub

Sub Main()
    ' Any of the following two lines generates bc30577.
    'Dim t As New Threading.Thread(AddressOf(CountZeroToTen))
    'Dim t As New Threading.Thread(AddressOf CountZeroToTen())
    t.Start()
End Sub
```

## <a name="to-correct-this-error"></a><span data-ttu-id="578f7-109">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="578f7-109">To correct this error</span></span>

1. <span data-ttu-id="578f7-110">Aşağıdaki örnekte gösterildiği gibi, bağımsız değişkenin etrafındaki parantezleri kaldırın `AddressOf` :</span><span class="sxs-lookup"><span data-stu-id="578f7-110">Remove the parentheses around the argument following `AddressOf` as shown in the following example:</span></span>

    ```vb
    Public Sub CountZeroToTen()
        For i = 0 To 10
            Console.WriteLine($"Counted: {i}")
            Threading.Thread.Sleep(500)
        Next
    End Sub

    Sub Main()
        Dim t As New Threading.Thread(AddressOf CountZeroToTen)
        t.Start()
    End Sub
    ```

2. <span data-ttu-id="578f7-111">Bağımsız değişkenin bir yöntem adı olduğundan emin olun.</span><span class="sxs-lookup"><span data-stu-id="578f7-111">Make sure the argument is a method name.</span></span>
