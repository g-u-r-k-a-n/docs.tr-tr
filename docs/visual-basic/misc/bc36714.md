# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="a22c9-102">Genişletilmiş özellikler başlatılamıyor</span><span class="sxs-lookup"><span data-stu-id="a22c9-102">Expanded properties cannot be initialized</span></span>

<span data-ttu-id="a22c9-103">Otomatik uygulanan bir özellik, bildiriminin bir parçası olarak başlatılabilir, ancak genişletilmiş bir özellik olamaz.</span><span class="sxs-lookup"><span data-stu-id="a22c9-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="a22c9-104">**Hata kimliği:** BC36714</span><span class="sxs-lookup"><span data-stu-id="a22c9-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a22c9-105">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="a22c9-105">To correct this error</span></span>  
  
- <span data-ttu-id="a22c9-106">Otomatik uygulanan bir özellik kullanın veya başlatma özelliğini özellik bildiriminden kaldırın.</span><span class="sxs-lookup"><span data-stu-id="a22c9-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a22c9-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="a22c9-107">Example</span></span>  

 <span data-ttu-id="a22c9-108">Aşağıdaki örneklerde hem otomatik uygulanan hem de genişletilmiş özellikler gösterilmektedir.</span><span class="sxs-lookup"><span data-stu-id="a22c9-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="a22c9-109">Otomatik uygulanan özellikler atama veya `New` yan tümce kullanılarak başlatılabilir, ancak genişletilmiş özellikler olamaz.</span><span class="sxs-lookup"><span data-stu-id="a22c9-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
