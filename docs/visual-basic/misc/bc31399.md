 <span data-ttu-id="16d89-105">Bu durum, hedeflenen temel sınıfın hiçbir Oluşturucusu yoksa veya başka bir sınıftan erişimi önleyen erişim düzeylerine sahip oluşturucuları varsa meydana gelir.</span><span class="sxs-lookup"><span data-stu-id="16d89-105">This can happen if the intended base class has no constructors or if it has constructors with access levels that prevent access from another class.</span></span>
