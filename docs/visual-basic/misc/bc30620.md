# <a name="option-strict-can-be-followed-only-by-on-or-off"></a>' Option Strict ' arkasından yalnızca ' on ' veya ' off ' gelebilir

`Option Strict` deyimler ya da belirtmelidir `On` `Off` . Diğer tüm değerler geçersiz.  
  
 **Hata kimliği:** BC30620  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Deyimin birini ya da ya da belirtin `On` `Off` `Option Strict` .
