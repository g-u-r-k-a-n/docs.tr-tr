## <a name="remarks"></a><span data-ttu-id="af87c-122">Açıklamalar</span><span class="sxs-lookup"><span data-stu-id="af87c-122">Remarks</span></span>

<span data-ttu-id="af87c-123">Tablodaki tüm türler, `object` ve `string`hariç, basit türler olarak adlandırılır.</span><span class="sxs-lookup"><span data-stu-id="af87c-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>

<span data-ttu-id="af87c-124">.NET türleri ve bunların C# tür anahtar sözcük diğer adları değiştirilebilir.</span><span class="sxs-lookup"><span data-stu-id="af87c-124">The .NET types and their C# type keyword aliases are interchangeable.</span></span> <span data-ttu-id="af87c-125">Örneğin, aşağıdaki bildirimlerden birini kullanarak bir tamsayı değişkeni bildirebilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="af87c-125">For example, you can declare an integer variable by using either of the following declarations:</span></span>

```csharp
int x = 123;
System.Int32 y = 123;
```

<span data-ttu-id="af87c-126">Belirtilen türü temsil eden <xref:System.Type?displayProperty=nameWithType> örneğini almak için [typeof](../operators/type-testing-and-cast.md#typeof-operator) işlecini kullanın:</span><span class="sxs-lookup"><span data-stu-id="af87c-126">Use the [typeof](../operators/type-testing-and-cast.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Type stringType = typeof(string);
Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
