# <a name="systemservicemodeltxcompletionstatusabortedonsessionclose"></a>System.ServiceModel.TxCompletionStatusAbortedOnSessionClose

Oturum kapatıldığında tamamlanmamış olduğundan ve TransactionAutoCompleteOnSessionClose OperationBehaviorAttribute yanlış olarak ayarlandığı için belirtilen işlem iptal edildi.  
  
## <a name="description"></a>Açıklama  

 Geçerli etkin oturum kapalıysa izlenen, işlem tamamlanmadıysa ve TransactionAutoCompleteOnSessionClose olarak ayarlanır `false` .  
  
## <a name="troubleshooting"></a>Sorun giderme  

 Bu izleme, araştırılması gereken olası bir uygulama hatasını gösterir.
