# <a name="too-many-arguments-to-extension-method-methodname-defined-in-typename"></a>' ' İçinde tanımlanan ' ' genişletme metodunda çok fazla bağımsız değişken \<methodname> var \<typename>

Bir genişletme yöntemine yapılan çağrıda sağlanan bağımsız değişkenlerin sayısı, yöntemi için tanımlanan bağımsız değişkenlerin sayısını aşıyor.  
  
 **Hata kimliği:** BC36582  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Uzantı yönteminin tanımına başvurun ve çağrıdan ek bağımsız değişkenleri kaldırın.
