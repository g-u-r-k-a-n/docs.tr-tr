 <span data-ttu-id="d0495-104">Bu bağımsız <xref:System.Diagnostics.ConditionalAttribute> değişkene veya <xref:System.Runtime.CompilerServices.IndexerNameAttribute> özniteliklere geçtiğiniz herhangi bir bağımsız değişken geçerli bir tanımlayıcı olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="d0495-104">Any argument that you pass to the <xref:System.Diagnostics.ConditionalAttribute> or <xref:System.Runtime.CompilerServices.IndexerNameAttribute> attributes must be a valid identifier.</span></span> <span data-ttu-id="d0495-105">Bu, tanımlayıcılarda meydana geldiklerinde "+" gibi yasadışı karakterler içermeyebileceği anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="d0495-105">This means that it may not contain characters such as "+" that are illegal when they occur in identifiers.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d0495-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="d0495-106">Example</span></span>  
 <span data-ttu-id="d0495-107">Bu örnek, CS0633 <xref:System.Diagnostics.ConditionalAttribute>kullanarak göstermektedir.</span><span class="sxs-lookup"><span data-stu-id="d0495-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="d0495-108">Aşağıdaki örnek CS0633 üretir.</span><span class="sxs-lookup"><span data-stu-id="d0495-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="d0495-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="d0495-109">Example</span></span>  
 <span data-ttu-id="d0495-110">Bu örnek, CS0633 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>kullanarak göstermektedir.</span><span class="sxs-lookup"><span data-stu-id="d0495-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]
   {
      get { return i; }
   }  
}  
```
