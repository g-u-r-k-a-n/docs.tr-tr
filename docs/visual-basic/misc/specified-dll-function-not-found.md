# <a name="specified-dll-function-not-found"></a>Belirtilen DLL işlevi bulunamadı

Bir kullanıcı kitaplığı başvurusunda dinamik bağlantı kitaplığı (DLL) bulundu, ancak belirtilen DLL işlevi DLL içinde bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İşlev bildiriminde geçerli bir sıra sayısı belirtin.  
  
2. DLL adının ve diğer adın doğru olduğundan emin olun.
