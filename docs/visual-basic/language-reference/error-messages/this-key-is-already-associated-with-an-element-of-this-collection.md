# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>Bu anahtar bu koleksiyonun bir öğesi ile zaten ilişkili

Bir koleksiyon üyesi için belirtilen bir anahtar zaten koleksiyonun başka bir üyesini tanımlıyor. Anahtar, `Add` bir koleksiyonun belirli bir üyesini benzersiz bir şekilde tanımlayan yöntemde belirtilen bir dizedir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu üye için farklı bir anahtar kullanın.
