# <a name="compiler-error-cs0599"></a>Derleyici hatası CS0599
' Argument ' adlandırılmış öznitelik bağımsız değişkeni için geçersiz değer  
  
 Bir özniteliğe geçersiz bir bağımsız değişken geçirildi.
