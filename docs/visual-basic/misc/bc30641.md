# <a name="byval-and-byref-cannot-be-combined"></a>' ByVal ' ve ' ByRef ' birleştirilemez

`ByVal`Ve `ByRef` anahtar sözcükleri birbirini dışlıyor.  
  
 **Hata kimliği:** BC30641  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ya da `ByVal` `ByRef` ' ı belirtin, ancak ikisini birden belirtmeyin.
