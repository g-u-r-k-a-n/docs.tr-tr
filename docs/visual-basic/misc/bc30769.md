# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-end-statement"></a>.NET Compact Framework hedeflenen sürümü ' End ' ifadesini desteklemiyor

Üzerinde çalıştığınız .NET Compact Framework sürümü, `End` ifadesini desteklemez.  
  
 **Hata kimliği:** BC30769  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfadesini kaldırın.
