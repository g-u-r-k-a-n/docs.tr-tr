# <a name="icorprofilerobjectenumreset-method"></a>ICorProfilerObjectEnum::Reset Yöntemi
Moves this enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT Reset ();  
```
