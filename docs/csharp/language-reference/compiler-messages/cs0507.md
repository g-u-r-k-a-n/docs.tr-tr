# <a name="compiler-error-cs0507"></a><span data-ttu-id="f6a64-102">Derleyici Hatası CS0507</span><span class="sxs-lookup"><span data-stu-id="f6a64-102">Compiler Error CS0507</span></span>
<span data-ttu-id="f6a64-103">'function1' : 'access' devralınan üye 'function2' geçersiz kılındığında erişim değiştiriciler değiştiremezsiniz</span><span class="sxs-lookup"><span data-stu-id="f6a64-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="f6a64-104">Bir yöntem geçersiz kılmada erişim belirtimini değiştirmek için bir girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="f6a64-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f6a64-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="f6a64-105">Example</span></span>  
 <span data-ttu-id="f6a64-106">Aşağıdaki örnek CS0507 üretir.</span><span class="sxs-lookup"><span data-stu-id="f6a64-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="f6a64-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="f6a64-107">Example</span></span>  
 <span data-ttu-id="f6a64-108">CS0507, bir sınıf başvurulan meta verilerde `protected internal` tanımlandığı şekilde işaretlenmiş bir yöntemi geçersiz kılmaya çalışırsa da oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="f6a64-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="f6a64-109">Bu durumda, geçersiz kılma yöntemi olarak `protected`işaretlenmelidir.</span><span class="sxs-lookup"><span data-stu-id="f6a64-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="f6a64-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="f6a64-110">Example</span></span>  
 <span data-ttu-id="f6a64-111">Aşağıdaki örnek CS0507 üretir.</span><span class="sxs-lookup"><span data-stu-id="f6a64-111">The following sample generates CS0507.</span></span>
