# <a name="compiler-warning-level-1-cs1635"></a>Derleyici Uyarısı (düzey 1) CS1635
Genel olarak devre dışı bırakıldığı için ' uyarı kodu ' uyarısı geri yüklenemiyor  
  
 Bu uyarı, tüm derleme birimi için bir uyarı devre dışı bırakmak üzere **/nowarn** komut satırı seçeneğini veya proje ayarını kullanırsanız, ancak bu `#pragma warning restore` uyarıyı geri yüklemeyi denemek için kullandığınızda oluşur. Bu hatayı çözmek için,/nowarn komut satırı seçeneğini veya proje ayarını kaldırın veya `#pragma warning restore` komut satırı veya proje ayarları aracılığıyla devre dışı bırakamakta olduğunuz tüm uyarılar için öğesini kaldırın. Daha fazla bilgi için [#pragma uyarı](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) konusuna bakın.  
  
 Aşağıdaki örnek CS1635 oluşturur:  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
