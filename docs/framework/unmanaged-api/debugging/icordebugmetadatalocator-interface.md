# <a name="icordebugmetadatalocator-interface"></a>ICorDebugMetaDataLocator Arabirimi
Hata ayıklayıcıya meta veri bilgileri sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetMetaData Yöntemi](icordebugmetadatalocator-getmetadata-method.md)|Hata ayıklayıcının istenen bir işlemi tamamlaması için gereken meta verileri bir modülün tam yolunu döndürmesini ister.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
> Bu arabirim, çapraz makine ya da çapraz işlem için uzaktan çağrılmakta değil.
