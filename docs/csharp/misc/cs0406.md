 <span data-ttu-id="7f5a4-105">Genel bir tür veya metodun bir sınıf türü kısıtlaması olduğunda, önce Bu kısıtlama listelenmelidir.</span><span class="sxs-lookup"><span data-stu-id="7f5a4-105">When a generic type or method has a class type constraint, that constraint must be listed first.</span></span> <span data-ttu-id="7f5a4-106">Bu hatadan kaçınmak için, sınıf türü kısıtlamasını kısıtlama listesinin başlangıcına taşıyın.</span><span class="sxs-lookup"><span data-stu-id="7f5a4-106">To avoid this error, move the class type constraint to the beginning of the constraint list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7f5a4-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="7f5a4-107">Example</span></span>  
 <span data-ttu-id="7f5a4-108">Aşağıdaki örnek CS0406 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7f5a4-108">The following sample generates CS0406.</span></span>  
  
```csharp

interface I {}  
class C {}

```
