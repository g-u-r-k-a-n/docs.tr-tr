# <a name="argument-argumentname-cannot-be-converted-to-type-typename"></a>' ' Bağımsız değişkeni \<argumentname> ' ' türüne dönüştürülemez \<typename>
Bir dönüştürme bir bağımsız değişkeni geçerli olmayan bir türe değiştirmeye çalışır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Türü geçerli olan bir türle değiştirin.
