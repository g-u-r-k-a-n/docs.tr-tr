# <a name="compiler-error-cs1651"></a><span data-ttu-id="f4848-103">Derleyici hatası CS1651</span><span class="sxs-lookup"><span data-stu-id="f4848-103">Compiler Error CS1651</span></span>
<span data-ttu-id="f4848-104">' Identifier ' statik salt okunur alanının alanları ref veya out olarak geçirilemez (statik oluşturucu içinde olması dışında)</span><span class="sxs-lookup"><span data-stu-id="f4848-104">Fields of static readonly field 'identifier' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="f4848-105">Bir statik salt okunur alanın üyesi olan bir işleve bir değişkeni bir ref bağımsız değişkeni olarak geçirirseniz bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="f4848-105">This error occurs if you pass a variable to a function that is a member of a static readonly field as a ref argument.</span></span> <span data-ttu-id="f4848-106">Başvuru parametreleri işlev tarafından değiştirilemediğinden, buna izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="f4848-106">Since ref parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="f4848-107">Bu hatayı çözmek için alandaki **ReadOnly** anahtar sözcüğünü kaldırın veya ReadOnly alanının üyelerini işleve geçirmeyin.</span><span class="sxs-lookup"><span data-stu-id="f4848-107">To resolve this error, remove the **readonly** keyword on the field, or do not pass the members of the readonly field to the function.</span></span> <span data-ttu-id="f4848-108">Örneğin, aşağıdaki örnekte gösterildiği gibi, değiştirilebilen ve geçici olarak bir ref bağımsız değişkeni olarak geçen geçici bir değişken oluşturmayı deneyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f4848-108">For example, you might try creating a temporary variable which can be modified and passing the temporary as a ref argument, as shown in the following example.</span></span>  
  
 <span data-ttu-id="f4848-109">Aşağıdaki örnek CS1651 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f4848-109">The following sample generates CS1651:</span></span>  
  
```csharp  
// CS1651.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{
  public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void f(ref int iref)  
   {  
   }  
  
   static void Main()  
   {  
      f(ref Outer.inner.i);  // CS1651  
      // Try this instead:  
      // int tmp = Outer.inner.i;  
      // f(ref tmp);  
   }  
}  
```
