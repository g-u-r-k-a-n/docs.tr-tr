# <a name="compiler-error-cs0040"></a>Derleyici hatası CS0040

Hata ayıklama bilgileri dosyası oluşturulurken beklenmeyen hata oluştu — ' Reason '  
  
 [**DEBUGTYPE**](../language-reference/compiler-options/code-generation.md#debugtype) derleyici seçeneği kullanılırken bu hata oluşabilir ve derleyicinin. pdb dosyasına yazamayacağını belirtir. Bu hatanın olası çözümleri Visual Studio 'Yu yeniden yüklemeyi, derleyicinin bir dosya veya dizine yazma erişimi olmasını veya/debugile derlenmemesinin sağlamamasını içerir.
