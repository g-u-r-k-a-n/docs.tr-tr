# <a name="end-event-must-be-preceded-by-a-matching-custom-event"></a>' End Event ' öncesinde eşleşen bir ' Custom Event ' olmalıdır
`End Event`Karşılık gelen bir ifade olmadan bir ifade oluşur `Custom Event` . `End Event`önce karşılık gelen bir `Custom Event` deyimden önce gelmelidir.  
  
 **Hata kimliği:** BC31123  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Önceki bir `Custom Event` deyimin geçerli olduğundan ve doğru yazıldığından emin olun.
