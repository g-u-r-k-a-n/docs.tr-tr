# <a name="cannot-call-friend-function-on-object-which-is-not-an-instance-of-defining-class"></a>Tanımlama sınıfının bir örneği olmayan nesnede Friend işlevi çağrılamaz

`Friend`Bir sınıfın yordamını çağırmayı denediniz veya bir `Friend` özellik ya da yönteme çapraz işlem veya çapraz iş parçacığı ile erişmeye çalıştınız. Bir `Friend` yordam, sınıfının dışındaki bir modülden çağrılabilir, ancak sınıfın tanımlandığı projenin bir parçasıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sınıfın tanımlandığı projenin parçası olan bir modülden çağırtığınızdan veya yordam ile eriştiğiniz emin olun.
