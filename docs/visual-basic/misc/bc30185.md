# <a name="enum-must-end-with-a-matching-end-enum"></a>' Enum ' öğesi eşleşen bir ' End Enum ' ile bitmelidir

`Enum`Karşılık gelen bir ifade olmadan bir ifade oluşur `End Enum` . Bir `End Enum` deyimin numaralandırmayı sonlandırmak için kullanılması gerekir.  
  
 **Hata kimliği:** BC30185  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Enum`Üyelerin doğru biçimlendirildiğinden emin olun.  
  
2. `End Enum`Numaralandırmanın sonuna bir ifade ekleyin.
