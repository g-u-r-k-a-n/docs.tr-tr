<span data-ttu-id="4bf63-104">Genel bir arabirim, başka bir genel arabirimden birden çok kez kalıtımla alır ve tür bağımsız değişkenlerin belirli değerleri için devralmalardan ikisi çelişebilir.</span><span class="sxs-lookup"><span data-stu-id="4bf63-104">A generic interface inherits more than once from another generic interface, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 `Public Interface interfaceA(Of u)`  
  
 `End Interface`  
  
 `Public Interface derivedInterface(Of t1, t2)`  
  
 `Inherits interfaceA(Of t1), interfaceA(Of t2)`  
  
 `End Interface`  
  
 <span data-ttu-id="4bf63-106">`derivedInterface`Oluşturulmuş veya uygulanmış ise, her ikisine de aynı tür sağlamak `t1` için `t2` , aynı tür bağımsız değişkenleriyle iki sürümünü devralması gerekir `interfaceA` .</span><span class="sxs-lookup"><span data-stu-id="4bf63-106">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="4bf63-107">Bunun yapılması, hangi sürümün erişecağına bir belirsizlik üretir.</span><span class="sxs-lookup"><span data-stu-id="4bf63-107">Doing so would produce an ambiguity about which version to access.</span></span>
