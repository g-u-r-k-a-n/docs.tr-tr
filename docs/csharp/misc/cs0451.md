# <a name="compiler-error-cs0451"></a><span data-ttu-id="adbff-103">Derleyici hatası CS0451</span><span class="sxs-lookup"><span data-stu-id="adbff-103">Compiler Error CS0451</span></span>

<span data-ttu-id="adbff-104">' New () ' kısıtlaması ' struct ' kısıtlamasıyla kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="adbff-104">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="adbff-105">Genel türde kısıtlamalar belirtirken `new()` kısıtlama yalnızca sınıf türü kısıtlamaları, arabirim türü kısıtlamaları, başvuru türü kısıtlamaları ve tür parametresi kısıtlamaları ile kullanılabilir ancak değer türü kısıtlamalarına sahip değildir.</span><span class="sxs-lookup"><span data-stu-id="adbff-105">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="adbff-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="adbff-106">Example</span></span>  

 <span data-ttu-id="adbff-107">Aşağıdaki örnek CS0451 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="adbff-107">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
