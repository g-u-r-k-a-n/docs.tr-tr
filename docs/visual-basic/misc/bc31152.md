# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>' Prefix ' XML ad alanı öneki XML tarafından kullanılmak üzere ayrılmıştır ve ad alanı URI 'SI değiştirilemez
Bir XML sabit değerinde, gibi ayrılmış bir XML ad alanı öneki `xmlns` kullanılmıştır.  
  
 **Hata kimliği:** BC31152  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML ad alanı önekini ayrılmamış bir değere değiştirin.
