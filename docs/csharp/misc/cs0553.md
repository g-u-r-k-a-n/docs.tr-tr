# <a name="compiler-error-cs0553"></a>Derleyici hatası CS0553

' dönüştürme yordamı ': taban sınıfına/öğesinden Kullanıcı tanımlı dönüştürme  
  
 Bir taban sınıfın değerlerine Kullanıcı tanımlı Dönüştürmelere izin verilmiyor; Böyle bir işlece gerek yoktur.  
  
 Aşağıdaki örnek CS0553 oluşturur:  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
