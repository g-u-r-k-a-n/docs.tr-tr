# <a name="get-is-already-declared"></a>' Get ' zaten tanımlanmış
Bir `Get` ifade, kod bloğunda birden çok kez görünür. `Get`İfade, `Get` bir özelliğin değerini almak için kullanılan bir özellik yordamı bildirir.  
  
 **Hata kimliği:** BC30443  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ifadesini kaldırın `Get` .
