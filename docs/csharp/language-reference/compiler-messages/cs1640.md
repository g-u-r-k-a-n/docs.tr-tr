# <a name="compiler-error-cs1640"></a><span data-ttu-id="4203a-102">Derleyici Hatası CS1640</span><span class="sxs-lookup"><span data-stu-id="4203a-102">Compiler Error CS1640</span></span>
<span data-ttu-id="4203a-103">foreach deyimi 'tür' değişkenleri üzerinde çalışamaz çünkü 'arayüz' birden fazla anlık uygular, belirli bir arabirim anında döküm deneyin</span><span class="sxs-lookup"><span data-stu-id="4203a-103">foreach statement cannot operate on variables of type 'type' because it implements multiple instantiations of 'interface', try casting to a specific interface instantiation</span></span>  
  
 <span data-ttu-id="4203a-104">Tür, iEnumerator\<T>'nin iki veya daha fazla örneğinden miras kalır, bu da `foreach` kullanabileceği nizaminin benzersiz bir numaralandırması olmadığı anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="4203a-104">The type inherits from two or more instances of IEnumerator\<T>, which means there is not a unique enumeration of the type that `foreach` could use.</span></span> <span data-ttu-id="4203a-105">IEnumerator\<T> türünü belirtin veya başka bir döngü yapısı kullanın.</span><span class="sxs-lookup"><span data-stu-id="4203a-105">Specify the type of IEnumerator\<T> or use another looping construct.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4203a-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="4203a-106">Example</span></span>  
 <span data-ttu-id="4203a-107">Aşağıdaki örnek CS1640 üretir:</span><span class="sxs-lookup"><span data-stu-id="4203a-107">The following sample generates CS1640:</span></span>  
  
```csharp  
// CS1640.cs  
  
using System;  
using System.Collections;  
using System.Collections.Generic;  
  
public class C : IEnumerable, IEnumerable<int>, IEnumerable<string>  
{  
    IEnumerator<int> IEnumerable<int>.GetEnumerator()  
    {  
        yield break;  
    }  
  
    IEnumerator<string> IEnumerable<string>.GetEnumerator()  
    {  
        yield break;  
    }  
  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        return (IEnumerator)((IEnumerable<string>)this).GetEnumerator();  
    }  
}  
  
public class Test  
{  
    public static int Main()  
    {  
        foreach (int i in new C()){}    // CS1640  
  
        // Try specifying the type of IEnumerable<T>  
        // foreach (int i in (IEnumerable<int>)new C()){}  
        return 1;  
    }  
}  
```
