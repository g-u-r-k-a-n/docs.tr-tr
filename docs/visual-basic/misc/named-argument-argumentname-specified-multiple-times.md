# <a name="named-argument-argumentname-specified-multiple-times"></a>Adlandırılmış bağımsız değişken ' \<argumentname> ' birden çok kez belirtildi
Adlandırılmış bir bağımsız değişkeni birden çok kez belirttiniz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirtimlerinden biri hariç tümünü kaldırın.
