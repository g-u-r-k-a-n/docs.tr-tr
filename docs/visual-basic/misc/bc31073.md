# <a name="name1-conflicts-with-a-type-by-the-same-name-defined-in-name2"></a>' \<name1> ' \<type> , ' ' içinde tanımlanan aynı adla ' ' ile \<name2> çakışıyor

Aynı ada sahip bir türle çakışan bir ada rastlandı.  
  
 **Hata kimliği:** BC31073  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Benzersiz bir ad bildirin.  
  
2. Çakışan üye adı etrafında köşeli ayraç kullanın.
