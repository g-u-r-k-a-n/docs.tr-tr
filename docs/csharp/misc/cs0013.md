# <a name="compiler-error-cs0013"></a>Derleyici hatası CS0013
' ModelStore.dll ' dosyasına meta veriler yazılırken beklenmeyen hata--' daha fazla kullanıcı dizesi oluşturmak için mantıksal alan kalmadı.  
  
 .NET Framework ortak dil çalışma zamanı meta verileri yaymadı. Yolun doğru olduğundan ve diskin dolu olmadığından emin olmak için denetleyin. Sorun devam ederse, Visual Studio 'Yu ve/veya .NET Framework onarmanız veya yeniden yüklemeniz gerekebilir.
