# <a name="end-enum-must-be-preceded-by-a-matching-enum"></a>' End Enum ' öncesinde eşleşen bir ' Enum ' olmalıdır

`End Enum`Karşılık gelen bir ifade olmadan bir ifade oluşur `Enum` . `End Enum` önce karşılık gelen bir `Enum` deyimden önce gelmelidir.  
  
 **Hata kimliği:** BC30184  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Önceki bir `Enum` deyimin yanlış yazılmadığından veya geçersiz olduğundan emin olun.  
  
2. `Enum`Üyelerin doğru biçimlendirildiğinden emin olun.
