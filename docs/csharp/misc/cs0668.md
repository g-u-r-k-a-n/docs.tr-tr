<span data-ttu-id="1a2c2-104">İki dizin erişimcisi farklı adlara sahiptir; IndexerName özniteliği bir tür içindeki her dizin oluşturucuda aynı adla kullanılmalıdır</span><span class="sxs-lookup"><span data-stu-id="1a2c2-104">Two indexers have different names; the IndexerName attribute must be used with the same name on every indexer within a type</span></span>  
  
 <span data-ttu-id="1a2c2-105">**IndexerName** özniteliğine geçirilen değerler, bir türdeki tüm dizin oluşturucular için aynı olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="1a2c2-105">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="1a2c2-106">**IndexerName** özniteliği hakkında daha fazla bilgi için bkz. [IndexerNameAttribute nesnesinin oluşturucusu Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="1a2c2-106">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="1a2c2-107">Aşağıdaki örnek CS0668 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1a2c2-107">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```
