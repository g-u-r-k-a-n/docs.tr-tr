## <a name="example"></a><span data-ttu-id="b4fd7-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="b4fd7-109">Example</span></span>

<span data-ttu-id="b4fd7-110">Aşağıdaki örnekler CS8892 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b4fd7-110">The following examples generates CS8892:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Program
{
    // CS8892: Method 'Program.Main()' will not be used as an entry point because a synchronous entry point 'Program.Main(string[])' was found.
    public static async Task Main()
    {
        await Task.Delay(1);
    }

    public static void Main(string[] args)
    {
        Console.WriteLine(2);
    }
}
```

## <a name="how-to-fix-this-warning"></a><span data-ttu-id="b4fd7-111">Bu uyarıyı çözme</span><span class="sxs-lookup"><span data-stu-id="b4fd7-111">How to fix this warning</span></span>

```csharp
    public static async Task SomeOtherNameAsync()

        await Task.Delay(1);

    }

```
