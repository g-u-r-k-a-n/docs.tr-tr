# <a name="the-file-is-already-open"></a>Dosya zaten açık

Zaten açık olan bir dosyayı açma girişiminde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dosyayı açmayı denemeden önce kapatın.  
  
     -veya-  
  
- Gereksiz `FileOpen` ifadesini kodınızdan kaldırın.
