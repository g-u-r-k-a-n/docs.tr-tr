# <a name="compiler-error-cs0750"></a><span data-ttu-id="c8958-103">Derleyici hatası CS0750</span><span class="sxs-lookup"><span data-stu-id="c8958-103">Compiler Error CS0750</span></span>
<span data-ttu-id="c8958-104">Kısmi bir yöntemin erişim değiştiricileri ya da Virtual, abstract, override, New, Sealed veya extern değiştiricileri olamaz.</span><span class="sxs-lookup"><span data-stu-id="c8958-104">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="c8958-105">Özel davranışları nedeniyle, kısmi Yöntemler kabul ettikleri değiştiricilere göre kısıtlamalara tabidir.</span><span class="sxs-lookup"><span data-stu-id="c8958-105">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c8958-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="c8958-106">To correct this error</span></span>  
  
1. <span data-ttu-id="c8958-107">Yöntem bildiriminden yetkisiz değiştiricisini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="c8958-107">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c8958-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="c8958-108">Example</span></span>  
 <span data-ttu-id="c8958-109">Aşağıdaki örnek CS0750 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c8958-109">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
