# <a name="compiler-error-cs0554"></a><span data-ttu-id="7a9a7-103">Derleyici hatası CS0554</span><span class="sxs-lookup"><span data-stu-id="7a9a7-103">Compiler Error CS0554</span></span>

<span data-ttu-id="7a9a7-104">' dönüştürme yordamı ': türetilmiş sınıfa/öğesinden Kullanıcı tanımlı dönüştürme</span><span class="sxs-lookup"><span data-stu-id="7a9a7-104">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="7a9a7-105">Türetilmiş bir sınıfın değerlerine Kullanıcı tanımlı Dönüştürmelere izin verilmiyor; Böyle bir işlece gerek yoktur.</span><span class="sxs-lookup"><span data-stu-id="7a9a7-105">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="7a9a7-106">Kullanıcı tanımlı dönüştürmeler hakkında daha fazla bilgi için C# dil belirtiminde Bölüm 6 ' ya bakın.</span><span class="sxs-lookup"><span data-stu-id="7a9a7-106">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="7a9a7-107">Aşağıdaki örnek CS0554 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7a9a7-107">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
