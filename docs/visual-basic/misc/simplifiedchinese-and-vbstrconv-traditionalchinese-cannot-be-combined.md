# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>SimplifiedChinese ve VbStrConv. TraditionalChinese birleştirilemez
Uygulamanız, birbirini dışlayan `VbStrConv` numaralandırma üyelerini `SimplifiedChinese` ve `TraditionalChinese`birleştirmeye çalışıyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `VbStrConv.SimplifiedChinese` ya da `VbStrConv.TraditionalChinese`kaldırın.
