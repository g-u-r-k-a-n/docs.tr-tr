# <a name="array-initializer-has-too-many-dimensions"></a>Dizi başlatıcıda çok fazla boyut içeriyor

Diziniz için başlatıcıda çok fazla boyut sağladınız.  
  
 **Hata kimliği:** BC30566  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaç boyutun gerekli olduğunu öğrenmek için dizi başlatıcısını denetleyin.  
  
- Ek bağımsız değişkeni veya bağımsız değişkenleri kaldırın.
