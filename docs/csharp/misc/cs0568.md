# <a name="compiler-error-cs0568"></a><span data-ttu-id="dcae7-103">Derleyici hatası CS0568</span><span class="sxs-lookup"><span data-stu-id="dcae7-103">Compiler Error CS0568</span></span>
<span data-ttu-id="dcae7-104">Yapılar, açık parametresiz oluşturucular içeremez</span><span class="sxs-lookup"><span data-stu-id="dcae7-104">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="dcae7-105">Her [yapının](../language-reference/builtin-types/struct.md) zaten nesneyi sıfıra Başlatan parametresiz bir Oluşturucusu vardır.</span><span class="sxs-lookup"><span data-stu-id="dcae7-105">Each [struct](../language-reference/builtin-types/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="dcae7-106">Bu nedenle, bir yapı için oluşturabileceğiniz oluşturucuların bir veya daha fazla parametre olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="dcae7-106">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="dcae7-107">Aşağıdaki örnek CS0568 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="dcae7-107">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
