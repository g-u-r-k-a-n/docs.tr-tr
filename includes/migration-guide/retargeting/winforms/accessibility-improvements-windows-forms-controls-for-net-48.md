### <a name="accessibility-improvements-in-windows-forms-controls-for-net-48"></a>.NET 4.8 için Windows Forms denetimlerinde erişilebilirlik iyileştirmeleri

|   |   |
|---|---|
|Ayrıntılar|Windows Forms Framework, Windows Forms müşterilerini daha iyi desteklemek için erişilebilirlik teknolojileri ile nasıl çalıştığını geliştirmeye devam etmektedir. Bunlar aşağıdaki değişiklikleri içerir:<ul><li>Yüksek Karşıtlık modu sırasında ekranı iyileştirmek için değişiklikler.</li><li>Ekran Okuyucusu ile etkileşimde değişiklikler.</li><li>Erişilebilir hiyerarşisindeki değişiklikler (UI Otomasyonu ağacında gezinmeyi geliştirme).</li></ul>|
|Öneri|**Bu değişiklikleri nasıl tercih edin veya çıkar**<br>Uygulamanın bu değişikliklerden yararlanabilmesi için .NET Framework 4.8 üzerinde çalışması gerekir. Uygulama aşağıdaki yollardan birini bu değişiklikleri kabul edebilir:<ul><li>.NET Framework 4.8'i hedeflemek için yeniden derlenmiştir. Bu erişilebilirlik değişiklikleri, .NET Framework 4.8'i hedefleyen Windows Forms uygulamalarında varsayılan olarak etkinleştirilir.</li><li>.NET Framework 4.7.2 veya önceki sürümü hedefler ve aşağıdaki app config dosyasının <code>&lt;runtime&gt;</code> bölümüne aşağıdaki [AppContext anahtarını](https://docs.microsoft.com/dotnet/framework/configure-apps/file-schema/runtime/appcontextswitchoverrides-element) ekleyerek ve <code>false</code>aşağıdaki örnekte görüldüğü gibi , eski erişilebilirlik davranışlarını devre dışı bırakmaz.</li></ul><pre><code class="lang-xml">&lt;?xml version=&quot;1.0&quot; encoding=&quot;utf-8&quot;?&gt;&#13;&#10;&lt;configuration&gt;&#13;&#10;&lt;startup&gt;&#13;&#10;&lt;supportedRuntime version=&quot;v4.0&quot; sku=&quot;.NETFramework,Version=v4.7&quot;/&gt;&#13;&#10;&lt;/startup&gt;&#13;&#10;&lt;runtime&gt;&#13;&#10;&lt;!-- AppContextSwitchOverrides value attribute is in the form of &#39;key1=true/false;key2=true/false  --&gt;&#13;&#10;&lt;AppContextSwitchOverrides value=&quot;Switch.UseLegacyAccessibilityFeatures=false;Switch.UseLegacyAccessibilityFeatures.2=false;Switch.UseLegacyAccessibilityFeatures.3=false&quot; /&gt;&#13;&#10;&lt;/runtime&gt;&#13;&#10;&lt;/configuration&gt;&#13;&#10;</code></pre>.NET Framework 4.8'de eklenen erişilebilirlik özelliklerini tercih etmek için .NET Framework 4.7.1 ve 4.7.2'nin erişilebilirlik özelliklerini de tercih etmeniz gerektiğini unutmayın. .NET Framework 4.8'i hedefleyen ve eski erişilebilirlik davranışını korumak isteyen uygulamalar, bu AppContext anahtarını <code>true</code>açıkça ayarlayarak eski erişilebilirlik özelliklerinin kullanımını seçebilir. Klavye ToolTip çağırma desteğini etkinleştirmek <code>Switch.System.Windows.Forms.UseLegacyToolTipDisplay=false</code> için appContextSwitchOverrides değerine satır eklenmesi ni gerektirir:<pre><code class="lang-xml">&#39;&lt;AppContextSwitchOverrides value=&quot;Switch.UseLegacyAccessibilityFeatures=false;Switch.UseLegacyAccessibilityFeatures.2=false;Switch.UseLegacyAccessibilityFeatures.3=false;Switch.System.Windows.Forms.UseLegacyToolTipDisplay=false&quot; /&gt;&#39;&#13;&#10;</code></pre>Bu özelliği etkinleştirmek için .NET Framework 4.7.1 - 4.8'in yukarıda belirtilen erişilebilirlik özelliklerini etkinleştirme gerektiğini unutmayın. Ayrıca, erişilebilirlik özelliklerinden herhangi biri seçilmezse ancak araç ipucu görüntüleme özelliği kabul edilirse, bu özelliklere ilk erişime bir çalışma zamanı <xref:System.NotSupportedException> atılır. Özel durum iletisi, klavye Araç İpuçlarının etkinleştirilen düzey 3'ün erişilebilirlik geliştirmelerini gerektirdiğini gösterir. **Yüksek Karşıtlık temalarında işletim sistemi tanımlı renklerin kullanımı**<ul><li>Geliştirilmiş yüksek kontrastlı temalar.</li></ul>**Geliştirilmiş Ekran Okuyucusu desteği**<ul><li>Anlatıcı şimdi erişilebilir bir <xref:System.Windows.Forms.DataGridViewColumn> adı duyurduğunda tür <xref:System.Windows.Forms.DataGridViewCell>yönünü duyurur.</li></ul>**Geliştirilmiş CheckedListBox Erişilebilirlik desteği**<ul><li><xref:System.Windows.Forms.CheckedListBox> Denetim için Geliştirilmiş Ekran Okuyucudesteği. Klavyeyi <xref:System.Windows.Forms.CheckedListBox> kullanarak denetime gezinirken, Ekran <xref:System.Windows.Forms.CheckedListBox> Okuyucuöğe odaklanır ve duyurur.</li><li>Boş bir CheckedListBox denetimi artık denetim odaklandığında sanal bir ilk öğe için çizilmiş bir odak dikdörtgeni vardır.</li></ul>**Geliştirilmiş ComboBox Erişilebilirlik desteği**<ul><li>UI Automation bildirimlerini <xref:System.Windows.Forms.ComboBox> ve diğer UI Otomasyon özelliklerini kullanabilme özelliğiyle kontrol için Etkin UI Otomasyon desteği.</li></ul>**Geliştirilmiş DataGridView Erişilebilirlik desteği**<ul><li>UI Automation bildirimlerini ve diğer UI Otomasyon özelliklerini kullanabilme özelliğiyle kontrol için <xref:System.Windows.Forms.DataGridView> Etkin UI Otomasyon desteği.</li><li>Karşılık gelen <xref:System.Windows.Forms.DataGridViewComboBoxEditingControl> veya <xref:System.Windows.Forms.DataGridViewTextBoxEditingControl> şimdi karşılık gelen düzenleme hücresinin bir alt öğesi olan UI Otomasyon öğesi.</li></ul>**Geliştirilmiş LinkLabel Erişilebilirlik desteği**<ul><li>Geliştirilmiş <xref:System.Windows.Forms.LinkLabel> denetim erişilebilirliği: Ekran okuyucusu, ilgili <xref:System.Windows.Forms.LinkLabel> denetim devre dışı bırakılırsa bağlantıiçin devre dışı bırakma durumunu bildirir.</li></ul>**Geliştirilmiş ProgressBar Erişilebilirlik desteği**<ul><li>UI Automation bildirimlerini <xref:System.Windows.Forms.ProgressBar> ve diğer UI Otomasyon özelliklerini kullanabilme özelliğiile kontrol için Etkin UI Otomasyon desteği. Geliştiriciler artık, Ekran Okuyucusu'nun ilerlemeyi belirtmek için duyurabileceği UI Otomasyonu bildirimlerini kullanabilir.</li></ul>UI otomasyon bildirim olayları da dahil olmak üzere UI otomasyon olaylarına genel bakış için [UI Otomasyon Etkinliklerine Genel Bakış](https://docs.microsoft.com/windows/desktop/WinAuto/uiauto-eventsoverview)bölümüne bakın. **Geliştirilmiş PropertyGrid Erişilebilirlik desteği**<ul><li>UI Automation bildirimlerini <xref:System.Windows.Forms.PropertyGrid> ve diğer UI Otomasyon özelliklerini kullanabilme özelliğiyle kontrol için Etkin UI Otomasyon desteği.</li><li>Şu anda düzenlenen özelliğe karşılık gelen UI Otomasyon öğesi artık ilgili özellik öğesi UI Automation öğesinin bir alt öğesidir.</li><li>Üst <xref:System.Windows.Forms.PropertyGrid> denetim kategori görünümüne ayarlanmışsa, UI Automation özellik öğesi öğesi artık ilgili kategori öğesinin bir alt öğesidir.</li></ul>**Geliştirilmiş ToolStrip desteği**<ul><li>UI Automation bildirimlerini <xref:System.Windows.Forms.ToolStrip> ve diğer UI Otomasyon özelliklerini kullanabilme özelliğiyle kontrol için Etkin UI Otomasyon desteği.</li><li>Öğeler arasında <xref:System.Windows.Forms.ToolStrip> geliştirilmiş gezinme.</li><li>Öğeler modunda, Ekran Okuyucusu odağı kaybolmaz ve gizli öğelere gitmez.</li></ul>**Geliştirilmiş Görsel ipuçları**<ul><li>Boş <xref:System.Windows.Forms.CheckedListBox> bir denetim artık odak aldığında bir odak göstergesi görüntüler.</li></ul>Not: UI otomasyon desteği çalışma zamanındaki denetimler için etkinleştirilir, ancak tasarım zamanında kullanılmaz. UI otomasyonuna genel bir bakış için [UI Automation Genel Bakış](https://docs.microsoft.com/dotnet/framework/ui-automation/ui-automation-overview)bölümüne bakın.</p>**Klavyeile kumandaların Araç İpuçlarını çağırmak**<ul><li>Denetim araç ucu artık klavye ile kontrol odaklanarak çağrılabilir.
