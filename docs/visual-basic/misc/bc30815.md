# <a name="functionname-is-not-declared-visual-basic-error"></a>' \<functionname> ' bildirilmemiş (Visual Basic hatası)

' \<functionname> ' bildirilmemiş. Dosya g/ç işlevselliği, Microsoft. VisualBasic ad alanında kullanılabilir.  
  
 **Hata kimliği:** BC30815  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İçinde tanımlanan dosya g/ç işlevlerini `My.Computer.FileSystem` , `System.IO` ad alanını Visual Basic çalışma zamanı işlevlerini veya `Microsoft.VisualBasic.FileSystem` nesneyi kullanın.
