# <a name="compiler-error-cs0201"></a>Derleyici Hatası CS0201

Yalnızca atama, arama, artış, karar ve yeni nesne ifadeleri deyim olarak kullanılabilir  
  
 Derleyici geçersiz bir deyimle karşılaştığında bir hata oluşturur. Geçersiz bir deyim, bir atamayı temsil etmeyen bir yarı nokta üzerinde[=](../operators/assignment-operator.md)biten satır veya satır dizisidir ( ), yöntem çağrısı [()](../operators/member-access-operators.md#invocation-operator-), [yeni](../operators/new-operator.md), [--](../operators/arithmetic-operators.md#decrement-operator---) veya [++](../operators/arithmetic-operators.md#increment-operator-) işlem. Daha fazla bilgi için [Deyimler, İfadeler ve İşleçler'e](../../programming-guide/statements-expressions-operators/index.md)bakın.  
  
## <a name="example"></a>Örnek

 Aşağıdaki örnek CS0201 oluşturur, çünkü 2 * 3 bir ifadedir, bir deyim değil. Kodu derlemek için ifadenin değerini bir değişkene atamayı deneyin.  

```csharp
// CS0201.cs  
public class MainClass  
{  
   public static void Main()  
   {  
      2 * 3;   // CS0201  
      // Try the following line instead.  
      //   int i = 2 * 3;  
   }  
}  
```

## <a name="example"></a>Örnek

 Bir artış işlemi yle parametrelendirilmiş olsa bile, tek başına denetlenen bir deyim olmadığından aşağıdaki örnek CS0201 oluşturur.  

```csharp
// CS0201_b.cs  
// compile with: /target:library  
public class MyList<T>
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
