## <a name="example"></a><span data-ttu-id="16906-113">Örnek</span><span class="sxs-lookup"><span data-stu-id="16906-113">Example</span></span>  

 <span data-ttu-id="16906-114">Aşağıdaki örnek CS0280 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="16906-114">The following sample generates CS0280.</span></span>  
  
```csharp  
// CS0280.cs  
using System;  
using System.Collections;  
  
public class ValidBase: IEnumerable  
{  
   IEnumerator IEnumerable.GetEnumerator()  
   {  
      yield return 0;  
   }  
  
   internal IEnumerator GetEnumerator()  
   {  
      yield return 0;  
   }  
}  
  
class Derived : ValidBase  
{  
   // field, not method  
   new public int GetEnumerator;  
}  
  
public class Test  
{  
   public static void Main()  
   {  
      foreach (int i in new Derived()) {}   // CS0280  
   }  
}  
```
