 <span data-ttu-id="d2567-224">Derlemenin yüklenme şekli, tam zamanında (JıT) derlenmiş kodun, işlemdeki birden çok uygulama etki alanı tarafından paylaşılıp paylaşılamayacağını belirler.</span><span class="sxs-lookup"><span data-stu-id="d2567-224">The way the assembly is loaded determines whether its just-in-time (JIT) compiled code can be shared by multiple application domains in the process.</span></span>  
  
- <span data-ttu-id="d2567-225">Bir derleme etki alanı nötr olarak yüklenirse, aynı güvenlik izni kümesini paylaşan tüm uygulama etki alanları JıT ile derlenen aynı kodu paylaşabilir.</span><span class="sxs-lookup"><span data-stu-id="d2567-225">If an assembly is loaded domain-neutral, all application domains that share the same security grant set can share the same JIT-compiled code.</span></span> <span data-ttu-id="d2567-226">Bu, uygulamanın gerektirdiği belleği azaltır.</span><span class="sxs-lookup"><span data-stu-id="d2567-226">This reduces the memory required by the application.</span></span>  
  
- <span data-ttu-id="d2567-227">Bir derleme etki alanı nötr olarak yüklenmediyse, uygulamasının yüklendiği her uygulama etki alanında JıT olarak derlenmesi ve yükleyicinin, uygulama etki alanları arasında iç kaynakları paylaşmamalıdır olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="d2567-227">If an assembly is not loaded domain-neutral, it must be JIT-compiled in every application domain in which it is loaded and the loader must not share internal resources across application domains.</span></span>  
  
 <span data-ttu-id="d2567-228">1 olarak ayarlandığında, COMPLUS_LoaderOptimization Environment bayrağı, çalışma zamanı konağını, SingleDomain olarak bilinen, etki alanı olmayan bağımsız şekilde tüm derlemeleri yüklemeye zorlar.</span><span class="sxs-lookup"><span data-stu-id="d2567-228">When set to 1, the COMPLUS_LoaderOptimization environment flag forces the runtime host to load all assemblies in non-domain-neutral way known as SingleDomain.</span></span> <span data-ttu-id="d2567-229">SingleDomain, her zaman etki alanı nötr olarak yüklenen mscorlib dışında, hiçbir derlemeyi etki alanı nötr olarak yüklemez.</span><span class="sxs-lookup"><span data-stu-id="d2567-229">SingleDomain loads no assemblies as domain-neutral, except Mscorlib, which is always loaded domain-neutral.</span></span> <span data-ttu-id="d2567-230">Bu ayar, ana bilgisayar işlemde yalnızca tek bir uygulama çalıştırırken yaygın olarak kullanıldığından, tek etki alanı olarak adlandırılır.</span><span class="sxs-lookup"><span data-stu-id="d2567-230">This setting is called single domain because it is commonly used when the host is running only a single application in the process.</span></span>  
  
> [!CAUTION]
> <span data-ttu-id="d2567-231">COMPLUS_LoaderOptimization ortam bayrağı, tanılama ve test senaryolarında kullanılmak üzere tasarlanmıştır.</span><span class="sxs-lookup"><span data-stu-id="d2567-231">The COMPLUS_LoaderOptimization environment flag was designed to be used in diagnostic and test scenarios.</span></span> <span data-ttu-id="d2567-232">Bayrağın açık olması ciddi yavaşlamaya ve bellek kullanımında artışa neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="d2567-232">Having the flag turned on can cause severe slow-down and increase in memory usage.</span></span>  
  
### <a name="code-example"></a><span data-ttu-id="d2567-233">Kod örneği</span><span class="sxs-lookup"><span data-stu-id="d2567-233">Code example</span></span>

 <span data-ttu-id="d2567-234">Tüm derlemelerin yüklenmelerini zorlamak için, IISADMIN hizmeti için etki alanı nötr olarak, `COMPLUS_LoaderOptimization=1` HKEY_LOCAL_MACHINE \System\currentcontrolset\services\ıısadmın anahtarındaki ortamın çok dizeli değerine eklenerek elde edilebilir.</span><span class="sxs-lookup"><span data-stu-id="d2567-234">To force all assemblies not to be loaded as domain-neutral for the IISADMIN service can be achieved by appending `COMPLUS_LoaderOptimization=1` to the Environment’s Multi-String Value in the HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\IISADMIN key.</span></span>
