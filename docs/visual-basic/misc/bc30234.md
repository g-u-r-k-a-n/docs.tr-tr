# <a name="specifier-is-not-valid-on-a-withevents-declaration"></a>\<specifier>WithEvents bildiriminde ' ' geçerli değil

Bir ifade, gibi anahtar `Dim` `ReadOnly` kelimesiyle birlikte geçersiz bir anahtar sözcük içerir `WithEvents` .  
  
 **Hata kimliği:** BC30234  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Deyimden geçersiz anahtar sözcüğü Kaldır `Dim` .
