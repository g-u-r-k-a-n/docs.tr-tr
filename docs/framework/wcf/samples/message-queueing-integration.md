# <a name="message-queueing-integration"></a>İleti Kuyruğa Alma Tümleştirme
Bu bölüm Message Queuing ve Windows Communication Foundation (WCF) arasındaki tümleştirmeyi gösteren örnekler içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Windows Communication Foundation'a İleti Kuyruğa Alma](message-queuing-to-wcf.md)  
 Bir Message Queuing (MSMQ) uygulamasının bir WCF hizmetine nasıl MSMQ mesajı gönderebileceğinizi gösterir.
  
 [Windows Communication Foundation'dan İleti Kuyruğuna](wcf-to-message-queuing.md)  
 Bir WCF uygulamasının bir Message Queuing (MSMQ) uygulamasına nasıl ileti gönderebileceğinizi gösterir.  
  
 [İleti Bağıntısı](message-correlation.md)  
 Bir Message Queuing (MSMQ) uygulamasının bir WCF hizmetine MSMQ iletisi gönderebildiği ve bir istek/yanıt senaryosunda gönderici ve alıcı uygulamaları arasında iletilerin nasıl bağıntılı olduğunu gösterir.
