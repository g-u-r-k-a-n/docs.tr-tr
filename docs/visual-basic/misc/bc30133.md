# <a name="goto-statements-are-not-valid-in-the-immediate-window"></a>' GoTo ' deyimleri Komut penceresinde geçerli değildir

`GoTo`İfade, dallanma gerçekleştirir ve hata ayıklama bağlamında buna izin verilmez.  
  
 **Hata kimliği:** BC30133  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `GoTo` **Komut** penceresinde bir ifade vermeyin.
