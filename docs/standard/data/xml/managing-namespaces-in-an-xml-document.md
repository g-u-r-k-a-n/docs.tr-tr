# <a name="managing-namespaces-in-an-xml-document"></a><span data-ttu-id="644d8-104">XML Belgesinde Ad Alanlarını Yönetme</span><span class="sxs-lookup"><span data-stu-id="644d8-104">Managing Namespaces in an XML Document</span></span>

<span data-ttu-id="644d8-105">XML ad alanları, bir XML belgesindeki öğe ve öznitelik adlarını özel ve önceden tanımlanmış URI 'Ler ile ilişkilendirir.</span><span class="sxs-lookup"><span data-stu-id="644d8-105">XML namespaces associate element and attribute names in an XML document with custom and predefined URIs.</span></span> <span data-ttu-id="644d8-106">Bu ilişkilendirmeleri oluşturmak için, ad alanı URI 'Leri için ön ekleri tanımlar ve bu önekleri, XML verilerinde öğe ve öznitelik adlarını nitelemek için kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="644d8-106">To create these associations, you define prefixes for namespace URIs, and use those prefixes to qualify element and attribute names in XML data.</span></span> <span data-ttu-id="644d8-107">Ad alanları öğe ve öznitelik adı çakışmalarını önler ve aynı ada sahip öğelerin ve özniteliklerin işlenmesini ve farklı şekilde doğrulanmasını etkinleştirir.</span><span class="sxs-lookup"><span data-stu-id="644d8-107">Namespaces prevent element and attribute name collisions, and enable elements and attributes of the same name to be handled and validated differently.</span></span>

## <a name="declaring-namespaces"></a><span data-ttu-id="644d8-108">Ad alanlarını bildirme</span><span class="sxs-lookup"><span data-stu-id="644d8-108">Declaring namespaces</span></span>  

 <span data-ttu-id="644d8-109">Bir öğe üzerinde bir ad alanı bildirmek için `xmlns:` özniteliğini kullanın:</span><span class="sxs-lookup"><span data-stu-id="644d8-109">To declare a namespace on an element, you use the `xmlns:` attribute:</span></span>  
  
 `xmlns:<name>=<"uri">`  
  
 <span data-ttu-id="644d8-110">Burada `<name>` ad alanı öneki ve `<"uri">` ad ALANıNı tanımlayan URI 'dir.</span><span class="sxs-lookup"><span data-stu-id="644d8-110">where `<name>` is the namespace prefix and `<"uri">` is the URI that identifies the namespace.</span></span> <span data-ttu-id="644d8-111">Ön eki bildirdikten sonra, bir XML belgesindeki öğeleri ve öznitelikleri nitelemek ve bunları ad alanı URI 'siyle ilişkilendirmek için kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="644d8-111">After you declare the prefix, you can use it to qualify elements and attributes in an XML document and associate them with the namespace URI.</span></span> <span data-ttu-id="644d8-112">Ad alanı ön eki bir belge boyunca kullanıldığından, bu değer kısa bir süre olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="644d8-112">Because the namespace prefix is used throughout a document, it should be short in length.</span></span>  
  
 <span data-ttu-id="644d8-113">Bu örnek iki `BOOK` öğe tanımlar.</span><span class="sxs-lookup"><span data-stu-id="644d8-113">This example defines two `BOOK` elements.</span></span> <span data-ttu-id="644d8-114">İlk öğe önek tarafından nitelenir, `mybook` ve ikinci öğe önek tarafından nitelenir `bb` .</span><span class="sxs-lookup"><span data-stu-id="644d8-114">The first element is qualified by the prefix, `mybook`, and the second element is qualified by the prefix, `bb`.</span></span> <span data-ttu-id="644d8-115">Her önek farklı bir ad alanı URI 'siyle ilişkilendirilir:</span><span class="sxs-lookup"><span data-stu-id="644d8-115">Each prefix is associated with a different namespace URI:</span></span>  
  
```xml  
<mybook:BOOK xmlns:mybook="http://www.contoso.com/books.dtd">  
<bb:BOOK xmlns:bb="urn:blueyonderairlines" />
</mybook>
```  
  
 <span data-ttu-id="644d8-116">Bir öğenin belirli bir ad alanının parçası olduğunu belirtmek için, buna ad alanı öneki ekleyin.</span><span class="sxs-lookup"><span data-stu-id="644d8-116">To signify that an element is a part of a particular namespace, add the namespace prefix to it.</span></span> <span data-ttu-id="644d8-117">Örneğin, bir `Author` öğe `mybook` ad alanına aitse, olarak belirtilir `<mybook:Author>` .</span><span class="sxs-lookup"><span data-stu-id="644d8-117">For example, if a `Author` element belongs to the `mybook` namespace, it is declared as `<mybook:Author>`.</span></span>

## <a name="declaration-scope"></a><span data-ttu-id="644d8-118">Bildirim kapsamı</span><span class="sxs-lookup"><span data-stu-id="644d8-118">Declaration scope</span></span>  

 <span data-ttu-id="644d8-119">Bir ad alanı, içinde bildirildiği öğenin sonuna kadar bildirim noktasından etkilidir.</span><span class="sxs-lookup"><span data-stu-id="644d8-119">A namespace is effective from its point of declaration until the end of the element it was declared in.</span></span> <span data-ttu-id="644d8-120">Bu örnekte, öğesinde tanımlanan ad alanı öğesi gibi `BOOK` öğe dışındaki öğelere uygulanmaz `BOOK` `Publisher` :</span><span class="sxs-lookup"><span data-stu-id="644d8-120">In this example, the namespace defined in the `BOOK` element doesn't apply to elements outside the `BOOK` element, such as the `Publisher` element:</span></span>  
  
```xml  
<Author>Joe Smith</Author>  
<BOOK xmlns:book="http://www.contoso.com">  
    <title>My Wonderful Day</title>  
      <price>$3.95</price>  
</BOOK>  
<Publisher>  
    <Name>MSPress</Name>  
</Publisher>  
```  
  
 <span data-ttu-id="644d8-121">Ad alanı kullanılmadan önce bildirilmelidir, ancak XML belgesinin en üstünde görünmesini gerekmez.</span><span class="sxs-lookup"><span data-stu-id="644d8-121">A namespace must be declared before it can be used, but it doesn't have to appear at the top of the XML document.</span></span>  
  
 <span data-ttu-id="644d8-122">Bir XML belgesinde birden çok ad alanı kullandığınızda, temizleyici bir arama belgesi oluşturmak için bir ad alanını varsayılan ad alanı olarak tanımlayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="644d8-122">When you use multiple namespaces in an XML document, you can define one namespace as the default namespace to create a cleaner looking document.</span></span> <span data-ttu-id="644d8-123">Varsayılan ad alanı, kök öğesinde bildirilmiştir ve belgedeki tüm nitelenmemiş öğeler için geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="644d8-123">The default namespace is declared in the root element and applies to all unqualified elements in the document.</span></span> <span data-ttu-id="644d8-124">Varsayılan ad alanları özniteliklere değil yalnızca öğeler için geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="644d8-124">Default namespaces apply to elements only, not to attributes.</span></span>  
  
 <span data-ttu-id="644d8-125">Varsayılan ad alanını kullanmak için, öneki ve öğesindeki bildiriminden iki nokta üst üste atlayın:</span><span class="sxs-lookup"><span data-stu-id="644d8-125">To use the default namespace, omit the prefix and the colon from the declaration on the element:</span></span>  
  
```xml  
<BOOK xmlns="http://www.contoso.com/books.dtd">  
...
</BOOK>
```
