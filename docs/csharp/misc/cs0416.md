# <a name="compiler-error-cs0416"></a><span data-ttu-id="3ef42-103">Derleyici hatası CS0416</span><span class="sxs-lookup"><span data-stu-id="3ef42-103">Compiler Error CS0416</span></span>
<span data-ttu-id="3ef42-104">' tür parametresi ': öznitelik bağımsız değişkeni tür parametreleri kullanamaz</span><span class="sxs-lookup"><span data-stu-id="3ef42-104">'type parameter': an attribute argument cannot use type parameters</span></span>  
  
 <span data-ttu-id="3ef42-105">Bir tür parametresi, izin verilmeyen bir öznitelik bağımsız değişkeni olarak kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="3ef42-105">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="3ef42-106">Genel olmayan bir tür kullanın.</span><span class="sxs-lookup"><span data-stu-id="3ef42-106">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="3ef42-107">Aşağıdaki örnek CS0416 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3ef42-107">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}  
  
class G<T>  
{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
