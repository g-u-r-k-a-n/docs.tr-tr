# <a name="iactiononclrevent-interface"></a>IActionOnCLREvent Arabirimi
[ICLROnEventManager:: RegisterActionOnEvent](iclroneventmanager-registeractiononevent-method.md) metoduna yapılan bir çağrı kullanılarak kaydedilmiş olaylarda geri çağırmaları gerçekleştiren [IActionOnCLREvent:: OnEvent](iactiononclrevent-onevent-method.md) metodunu sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Description|  
|------------|-----------------|  
|[OnEvent Yöntemi](iactiononclrevent-onevent-method.md)|Kayıtlı bir olay için geri arama gerçekleştirir.|
