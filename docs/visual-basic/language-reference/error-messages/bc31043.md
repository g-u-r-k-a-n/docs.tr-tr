# <a name="bc31043-arrays-declared-as-structure-members-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="4b7c3-103">BC31043: yapı üyeleri olarak belirtilen diziler bir başlangıç boyutuyla bildirilemez</span><span class="sxs-lookup"><span data-stu-id="4b7c3-103">BC31043: Arrays declared as structure members cannot be declared with an initial size</span></span>

<span data-ttu-id="4b7c3-104">Yapıdaki bir dizi, başlangıç boyutuyla birlikte bildirilmiştir.</span><span class="sxs-lookup"><span data-stu-id="4b7c3-104">An array in a structure is declared with an initial size.</span></span> <span data-ttu-id="4b7c3-105">Herhangi bir yapı öğesini başlatamıyor ve bir dizi boyutunun bildirilmesi, tek bir başlatma biçimidir.</span><span class="sxs-lookup"><span data-stu-id="4b7c3-105">You cannot initialize any structure element, and declaring an array size is one form of initialization.</span></span>

<span data-ttu-id="4b7c3-106">**Hata kimliği:** BC31043</span><span class="sxs-lookup"><span data-stu-id="4b7c3-106">**Error ID:** BC31043</span></span>

## <a name="example"></a><span data-ttu-id="4b7c3-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="4b7c3-107">Example</span></span>

<span data-ttu-id="4b7c3-108">Aşağıdaki örnek BC31043 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4b7c3-108">The following example generates BC31043:</span></span>

```vb

    Public demoArray(9) As Integer

```

## <a name="to-correct-this-error"></a><span data-ttu-id="4b7c3-109">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="4b7c3-109">To correct this error</span></span>

1. <span data-ttu-id="4b7c3-110">Yapıdaki diziyi dinamik (başlangıç boyutu olmadan) olarak tanımlayın.</span><span class="sxs-lookup"><span data-stu-id="4b7c3-110">Define the array in your structure as dynamic (no initial size).</span></span>

2. <span data-ttu-id="4b7c3-111">Belirli bir dizi boyutuna ihtiyacınız varsa, kodunuz çalışırken bir [ReDim ifadesiyle](../statements/redim-statement.md) dinamik bir diziyi yeniden Dimension uygulayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="4b7c3-111">If you require a certain size of array, you can redimension a dynamic array with a [ReDim Statement](../statements/redim-statement.md) when your code is running.</span></span> <span data-ttu-id="4b7c3-112">Aşağıdaki örnekte bu gösterilmektedir:</span><span class="sxs-lookup"><span data-stu-id="4b7c3-112">The following example illustrates this:</span></span>

    ```vb
    Structure DemoStruct
        Public demoArray() As Integer
    End Structure
    Sub UseStruct()
        Dim struct As DemoStruct
        ReDim struct.demoArray(9)
        Struct.demoArray(2) = 777
    End Sub
    ```
