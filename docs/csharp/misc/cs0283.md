# <a name="compiler-error-cs0283"></a><span data-ttu-id="dba7a-102">Derleyici hatası CS0283</span><span class="sxs-lookup"><span data-stu-id="dba7a-102">Compiler Error CS0283</span></span>
<span data-ttu-id="dba7a-103">' Type ' türü const olarak bildirilemez</span><span class="sxs-lookup"><span data-stu-id="dba7a-103">The type 'type' cannot be declared const</span></span>  
  
<span data-ttu-id="dba7a-104">[Sabit](../language-reference/keywords/const.md) bildiriminde belirtilen tür `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, bir [sabit listesi](../language-reference/builtin-types/enum.md) türü veya `null`değeri atanan bir başvuru türü olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="dba7a-104">The type specified in a [constant](../language-reference/keywords/const.md) declaration must be `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, an [enum](../language-reference/builtin-types/enum.md) type, or a reference type that is assigned a value of `null`.</span></span> <span data-ttu-id="dba7a-105">Her sabit ifade, hedef türünde bir değer veya hedef türüne örtük olarak dönüştürülebilir bir tür değeri vermelidir.</span><span class="sxs-lookup"><span data-stu-id="dba7a-105">Each constant expression must yield a value of the target type or of a type that is implicitly convertible to the target type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dba7a-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="dba7a-106">Example</span></span>  
 <span data-ttu-id="dba7a-107">Aşağıdaki örnek CS0283 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="dba7a-107">The following example generates CS0283.</span></span>  
  
```csharp  
// CS0283.cs  
struct MyTest  
{  
}  
class MyClass   
{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.  
    const MyTest test = new MyTest();   // CS0283  
  
    public static int Main() {  
        return 1;  
    }  
}  
```
