# <a name="could-not-complete-operation-since-target-directory-is-under-source-directory"></a>Hedef dizin kaynak dizin altında olduğundan işlem tamamlanamadı

Döngüsel bir işlem başarısız oldu. Döngüsel işlemler çevrimi ve bu nedenle tamamlanamıyor. Örneğin, A nesnesi B nesnesinden devralma girişiminde bulunabilir ve bu nesne A nesnesinden devralır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Devralma sırasında, döngüsel başvuru bulunmadığından emin olun.
