# <a name="argument-not-specified-for-parameter-parametername-of-methodname"></a>' ' Öğesinin ' ' parametresi için bağımsız değişken belirtilmedi \<parametername> \<methodname>
Gerekli bir parametre için bir bağımsız değişken sağlamadınız.  
  
 **Hata kimliği:** BC30455  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirtilen parametre için bir bağımsız değişken sağlayın.
