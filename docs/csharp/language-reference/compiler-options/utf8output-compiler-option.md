# <a name="-utf8output-c-compiler-options"></a>-utf8output (C# Derleyici Seçenekleri)
**-utf8output** seçeneği, UTF-8 kodlamasını kullanarak derleyici çıktısını görüntüler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```console  
-utf8output  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bazı uluslararası yapılandırmalarda derleyici çıktısı konsolda doğru şekilde görüntülenemez. Bu yapılandırmalarda **-utf8output** kullanın ve derleyici çıktısını bir dosyaya yönlendirin.  
  
 Bu derleyici seçeneği Visual Studio'da kullanılamaz ve programlı olarak değiştirilemez.
