# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="43357-103">Derleyici Uyarısı (düzey 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="43357-103">Compiler Warning (level 2) CS0251</span></span>

<span data-ttu-id="43357-104">Bir diziyi negatif bir dizinle dizine ekleme (dizi dizinleri her zaman sıfırdan başlar)</span><span class="sxs-lookup"><span data-stu-id="43357-104">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="43357-105">Bir dizide dizin eklemek için negatif bir sayı kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="43357-105">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="43357-106">Aşağıdaki örnek CS0251 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="43357-106">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
