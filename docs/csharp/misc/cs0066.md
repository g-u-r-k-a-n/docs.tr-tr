# <a name="compiler-error-cs0066"></a><span data-ttu-id="747d3-103">Derleyici hatası CS0066</span><span class="sxs-lookup"><span data-stu-id="747d3-103">Compiler Error CS0066</span></span>

<span data-ttu-id="747d3-104">' Event ': olay bir temsilci türünde olmalıdır</span><span class="sxs-lookup"><span data-stu-id="747d3-104">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="747d3-105">Event anahtar sözcüğü bir [temsilci](../language-reference/builtin-types/reference-types.md) türü gerektirir.</span><span class="sxs-lookup"><span data-stu-id="747d3-105">The event keyword requires a [delegate](../language-reference/builtin-types/reference-types.md) type.</span></span> <span data-ttu-id="747d3-106">Daha fazla bilgi için bkz. [Olaylar](../programming-guide/events/index.md) ve [Temsilciler](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="747d3-106">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="747d3-107">Aşağıdaki örnek CS0066 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="747d3-107">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
