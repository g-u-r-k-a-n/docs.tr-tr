# <a name="interface-interfacename-is-not-implemented-by-this-class"></a>' \<interfacename> ' Arabirimi bu sınıf tarafından uygulanmıyor

Bu sınıf veya yapının üyesi, sınıfın veya yapının uygulamayan bir arabirimin üyesini uygulamaya çalışır.  
  
 **Hata kimliği:** BC31035  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Implements`Sınıf veya yapının başına, uygun arabirimi uyguladığından bir ifade ekleyin.  
  
- `Implements`Bu hataya neden olan üyenin anahtar sözcüğünü kaldırın.
