# <a name="webhostfailedtolisten"></a>WebHostFailedToListen

Kimlik: 112  
  
 Önem derecesi: Hata  
  
 Kategori: ServiceModel  
  
## <a name="description"></a>Açıklama  

 Bu olay, belirli URL 'YI dinlemeye çalışırken bir hata oluştuğunu gösterir. Bu çalışan işlem sonlandırıldı. Olayda özel durum, işlem adı ve işlem KIMLIĞI listelenmektedir.
