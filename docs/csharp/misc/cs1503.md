# <a name="compiler-error-cs1503"></a>Derleyici hatası CS1503

' Number ' bağımsız değişkeni TypeA 'dan TypeB 'ye dönüştürülemez  
  
 Bir yöntemde bir bağımsız değişkenin türü, sınıf örneği oluşturulurken geçirilen tür ile eşleşmiyor. Bu hata genellikle CS1502 ile birlikte görüntülenir. Bu hatayı giderme hakkında tartışmak için bkz. [CS1502](../language-reference/compiler-messages/cs1502.md) .
