# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Yan etkiler bu bağlamda ifade değerlendirmesi sırasında geçerli değil

Bir nesnenin durumunu değiştiren deyimler bu bağlamda kullanılamaz.  
  
 **Hata kimliği:** BC30700  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Deyimlerinizi diğer öğelerin durumu değiştirilmeden değerlendirilecek şekilde değiştirin.
