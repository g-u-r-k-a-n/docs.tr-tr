# <a name="parameter-parametername-of-methodname-already-has-a-matching-argument"></a>' ' \<parametername> Öğesinin ' ' parametresinin \<methodname> zaten eşleşen bir bağımsız değişkeni var

Bir bağımsız değişken aynı yordam çağrısında ad ile birden çok kez geçirilir.  
  
 **Hata kimliği:** BC30274  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken adlarından birinin yanlış yazılmış olup olmadığını kontrol edin.  
  
- Bağımsız değişken listesinden yinelenen bağımsız değişkeni kaldırın.
