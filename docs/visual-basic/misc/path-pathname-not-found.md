# <a name="path-pathname-not-found"></a>' \<pathname> ' Yolu bulunamadı

İşletim sistemi, dosya erişimi veya disk erişim işlemi sırasında belirtilen yol yolunu bulamıyor. Bir dosyanın yolu, dosyayı bulmak için geçmesi gereken sürücü belirtimini ve dizin ve alt dizinleri içermelidir. Yol göreli veya mutlak olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Yolun gerekli öğeleri içerdiğinden emin olun.
