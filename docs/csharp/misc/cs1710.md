# <a name="compiler-warning-level-2-cs1710"></a><span data-ttu-id="5593d-103">Derleyici Uyarısı (düzey 2) CS1710</span><span class="sxs-lookup"><span data-stu-id="5593d-103">Compiler Warning (level 2) CS1710</span></span>

<span data-ttu-id="5593d-104">' Type ' üzerindeki XML açıklamasında ' Parameter ' için yinelenen typeparam etiketi vardır</span><span class="sxs-lookup"><span data-stu-id="5593d-104">XML comment on 'type' has a duplicate typeparam tag for 'parameter'</span></span>  
  
 <span data-ttu-id="5593d-105">Genel bir türün belgeleri, tür parametresi için yinelenen bir etiket içerir.</span><span class="sxs-lookup"><span data-stu-id="5593d-105">The documentation of a generic type includes a duplicate tag for the type parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5593d-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="5593d-106">Example</span></span>  

 <span data-ttu-id="5593d-107">Aşağıdaki kod, uyarı CS1710 görünmesine neden olur.</span><span class="sxs-lookup"><span data-stu-id="5593d-107">The following code will cause warning CS1710 to appear.</span></span>  
  
```csharp  
// CS1710.cs  
// compile with: /doc:cs1710.xml  
// To resolve this warning, delete one of the duplicate <typeparam>'s.  
using System;  
class Stack<ItemType>  
{  
}  
  
/// <typeparam name="MyType">can be an int</typeparam>  
/// <typeparam name="MyType">can be an int</typeparam>  
class MyStackWrapper<MyType>  
{  
    // Open constructed type Stack<MyType>.  
    Stack<MyType> stack;  
    public MyStackWrapper(Stack<MyType> s)  
    {  
        stack = s;  
    }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        // Closed constructed type Stack<int>.  
        Stack<int> stackInt = new Stack<int>();  
        MyStackWrapper<int> MyStackWrapperInt =  
            new MyStackWrapper<int>(stackInt);  
    }  
}  
```
