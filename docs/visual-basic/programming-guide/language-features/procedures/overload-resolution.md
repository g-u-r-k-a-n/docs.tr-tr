# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="66b3f-102">Aşırı Yükleme Çözümü (Visual Basic Başvurusu)</span><span class="sxs-lookup"><span data-stu-id="66b3f-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="66b3f-103">Visual Basic derleyicisi, aşırı yüklü birkaç sürümde tanımlanan bir yordam için bir çağrıyla karşılaştığında, derleyici nin hangi aşırı yüklemeden arayacağına karar vermesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="66b3f-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="66b3f-104">Bunu aşağıdaki adımları gerçekleştirerek yapar:</span><span class="sxs-lookup"><span data-stu-id="66b3f-104">It does this by performing the following steps:</span></span>  
  
1. <span data-ttu-id="66b3f-105">**Erişilebilir -lik.**</span><span class="sxs-lookup"><span data-stu-id="66b3f-105">**Accessibility.**</span></span> <span data-ttu-id="66b3f-106">Arama kodunun aramasını engelleyen bir erişim düzeyine sahip aşırı yüklemeyi ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="66b3f-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2. <span data-ttu-id="66b3f-107">**Parametrelerin Sayısı.**</span><span class="sxs-lookup"><span data-stu-id="66b3f-107">**Number of Parameters.**</span></span> <span data-ttu-id="66b3f-108">Çağrıda verilenden farklı sayıda parametre tanımlayan aşırı yüklemeyi ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="66b3f-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3. <span data-ttu-id="66b3f-109">**Parametre Veri Türleri.**</span><span class="sxs-lookup"><span data-stu-id="66b3f-109">**Parameter Data Types.**</span></span> <span data-ttu-id="66b3f-110">Derleyici, örnek yöntemleri ni genişletme yöntemlerine göre tercih sağlar.</span><span class="sxs-lookup"><span data-stu-id="66b3f-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="66b3f-111">Yordam çağrısıyla eşleşecek şekilde yalnızca genişletme dönüşümleri gerektiren herhangi bir örnek yöntemi bulunursa, tüm uzantı yöntemleri bırakılır ve derleyici yalnızca örnek yöntemi adaylarıyla devam eder.</span><span class="sxs-lookup"><span data-stu-id="66b3f-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="66b3f-112">Böyle bir örnek yöntemi bulunamazsa, hem örnek hem de uzantı yöntemleriyle devam e-devam edin.</span><span class="sxs-lookup"><span data-stu-id="66b3f-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="66b3f-113">Bu adımda, arama bağımsız değişkenlerinin veri türlerinin aşırı yükte tanımlanan parametre türlerine dönüştürülemeyeceği aşırı yüklemeyi ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="66b3f-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4. <span data-ttu-id="66b3f-114">**Dönüşümleri Daraltma.**</span><span class="sxs-lookup"><span data-stu-id="66b3f-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="66b3f-115">Çağrı bağımsız değişkeni türlerinden tanımlı parametre türlerine daraltma dönüştürme gerektiren aşırı yüklemeyi ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="66b3f-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="66b3f-116">Bu, tür kontrol anahtarının[(Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) veya `On` . `Off`</span><span class="sxs-lookup"><span data-stu-id="66b3f-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5. <span data-ttu-id="66b3f-117">**En az genişletme.**</span><span class="sxs-lookup"><span data-stu-id="66b3f-117">**Least Widening.**</span></span> <span data-ttu-id="66b3f-118">Derleyici, kalan aşırı yüklemeleri çiftler halinde dikkate alır.</span><span class="sxs-lookup"><span data-stu-id="66b3f-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="66b3f-119">Her çift için, tanımlanan parametrelerin veri türlerini karşılaştırır.</span><span class="sxs-lookup"><span data-stu-id="66b3f-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="66b3f-120">Aşırı yüklerden birinde bulunan türler diğerindeki karşılık gelen türlere genişlerse, derleyici ikincisini ortadan kaldırır.</span><span class="sxs-lookup"><span data-stu-id="66b3f-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="66b3f-121">Diğer bir süre, en az genişletme gerektiren aşırı yükü korur.</span><span class="sxs-lookup"><span data-stu-id="66b3f-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="66b3f-122">**Tek Aday.**</span><span class="sxs-lookup"><span data-stu-id="66b3f-122">**Single Candidate.**</span></span> <span data-ttu-id="66b3f-123">Yalnızca bir aşırı yükleme kalana kadar çiftler halinde aşırı yüklemeleri göz önünde bulundurmaya devam eder ve aşırı yükleme çağrısını çözer.</span><span class="sxs-lookup"><span data-stu-id="66b3f-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="66b3f-124">Derleyici aşırı yüklemeleri tek bir adaya indiremiyorsa, bir hata oluşturur.</span><span class="sxs-lookup"><span data-stu-id="66b3f-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
