# <a name="end-class-must-be-preceded-by-a-matching-class"></a>' End Class ' öncesinde eşleşen bir ' class ' olmalıdır

`End Class` bir bloğu tamamlaması için kullanılır `Class` ; Bu nedenle yalnızca bloğun sonunda yer alabilir. Bir yedekli ya da `End Class` `End Class` deyiminiz ilgili bloğunun sınırları dışında görünüyor `Class` .  
  
 **Hata kimliği:** BC30460  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm gereksiz deyimleri bulun ve kaldırın `End Class` .  
  
- `End Class`Deyiminizi kodunuzda uygun konuma taşıyın.
