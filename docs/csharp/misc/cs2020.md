# <a name="compiler-error-cs2020"></a>Derleyici hatası CS2020

Yalnızca ilk giriş dosyası kümesi ' Module ' dışında bir hedef oluşturabilir  
  
 Çok çıktılı bir derlemede, ilk çıkış dosyası [-target: exe](../language-reference/compiler-options/output.md), [-target: winexe](../language-reference/compiler-options/output.md)veya [-target: Library](../language-reference/compiler-options/output.md)ile oluşturulmalıdır. Sonraki çıkış dosyalarının, [-target: Module](../language-reference/compiler-options/output.md)ile oluşturulması gerekir.
