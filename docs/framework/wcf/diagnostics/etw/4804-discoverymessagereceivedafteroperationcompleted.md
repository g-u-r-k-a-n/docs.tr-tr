|||  
|-|-|  
|ID|4804|  
|Anahtar sözcükler|Bulma|  
|Level|Bilgi|

## <a name="description"></a>Description  

 Bu olay, işlem tamamlandıktan sonra bulma iletisi alındığında yayınlanır.  
  
## <a name="message"></a>İleti  

 MessageID = ' %2 ' olan bir %1 iletisi, karşılık gelen %3 işlemi tamamlandığından DiscoveryClient tarafından bırakıldı.
