# <a name="compiler-error-cs8411"></a><span data-ttu-id="b3756-102">Derleyici hatası CS8411</span><span class="sxs-lookup"><span data-stu-id="b3756-102">Compiler Error CS8411</span></span>

<span data-ttu-id="b3756-103">' Type ', ' GetAsyncEnumerator ' için uygun bir ortak örnek tanımı içermediğinden, zaman uyumsuz foreach ifadesi ' Type ' türündeki değişkenlerde çalışamaz</span><span class="sxs-lookup"><span data-stu-id="b3756-103">Asynchronous foreach statement cannot operate on variables of type 'type' because 'type' does not contain a suitable public instance definition for 'GetAsyncEnumerator'</span></span>

<span data-ttu-id="b3756-104">`await foreach` deyimlerde yalnızca tanımına sahip olan türler üzerinde çalışır `GetAsyncEnumerator` `IAsyncEnumerable<T>` .</span><span class="sxs-lookup"><span data-stu-id="b3756-104">`await foreach` statement operates only on types having a definition of `GetAsyncEnumerator`, such as `IAsyncEnumerable<T>`.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="b3756-105">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="b3756-105">To correct this error</span></span>

<span data-ttu-id="b3756-106">`await foreach` yerine `foreach` yazın.</span><span class="sxs-lookup"><span data-stu-id="b3756-106">Replace `await foreach` with `foreach`.</span></span>

## <a name="example"></a><span data-ttu-id="b3756-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="b3756-107">Example</span></span>

```csharp
using System.Collections.Generic;
using System.Threading.Tasks;

class Program
{
    async Task Example(IAsyncEnumerator<int> enumerator)
    {
        // error CS8411: Asynchronous foreach statement cannot operate on variables
        // of type 'IAsyncEnumerator<T>' because 'IAsyncEnumerator<T>' does not
        // contain a suitable public instance definition for 'GetAsyncEnumerator'
        await foreach (int i in enumerator)
        {
        }
    }
}
