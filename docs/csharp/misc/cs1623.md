# <a name="compiler-error-cs1623"></a><span data-ttu-id="3122b-103">Derleyici hatası CS1623</span><span class="sxs-lookup"><span data-stu-id="3122b-103">Compiler Error CS1623</span></span>
<span data-ttu-id="3122b-104">Yineleyicilerin Ref veya out parametreleri olamaz</span><span class="sxs-lookup"><span data-stu-id="3122b-104">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="3122b-105">Bu hata, bir yineleyici yöntemi bir `in` , `ref` veya parametresi alırsa oluşur `out` .</span><span class="sxs-lookup"><span data-stu-id="3122b-105">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="3122b-106">Bu hatadan kaçınmak için,, `in` `ref` veya `out` anahtar sözcüğünü Yöntem imzasıyla kaldırın.</span><span class="sxs-lookup"><span data-stu-id="3122b-106">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3122b-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="3122b-107">Example</span></span>  
 <span data-ttu-id="3122b-108">Aşağıdaki örnek CS1623 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3122b-108">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
