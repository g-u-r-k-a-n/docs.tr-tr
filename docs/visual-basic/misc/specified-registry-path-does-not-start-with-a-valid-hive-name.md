# <a name="specified-registry-path-does-not-start-with-a-valid-hive-name"></a>Belirtilen kayıt defteri yolu geçerli bir Hive adı ile başlamıyor

Belirtilen kayıt defteri yolu geçerli bir Hive adı ile başlamıyor. Geçerli Hive adları şunlardır:  
  
- `HKEY_CLASSES_ROOT`  
  
- `HKEY_CURRENT_CONFIG`  
  
- `HKEY_CURRENT_USER`  
  
- `HKEY_DYNDATA`  
  
- `HKEY_LOCAL_MACHINE`  
  
- `HKEY_PERFORMANCE_DATA`  
  
- `HKEY_USERS`  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli bir Hive adı olduğunu doğrulayın.
