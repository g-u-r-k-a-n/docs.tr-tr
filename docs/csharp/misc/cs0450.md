# <a name="compiler-error-cs0450"></a><span data-ttu-id="f6542-102">Derleyici Hatası CS0450</span><span class="sxs-lookup"><span data-stu-id="f6542-102">Compiler Error CS0450</span></span>
<span data-ttu-id="f6542-103">'Tip Parametre Adı': hem bir kısıtlama sınıfı hem de 'sınıf' veya 'struct' kısıtlaması belirtilemez</span><span class="sxs-lookup"><span data-stu-id="f6542-103">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="f6542-104">Bir tür parametresi yapı türü kısıtlaması tarafından sınırlandırılırsa, yapı ve sınıf birbirini dışlayan kategoriler olduğundan, belirli bir sınıf türü tarafından da sınırlandırılması mantıksal olarak çelişkilidir.</span><span class="sxs-lookup"><span data-stu-id="f6542-104">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="f6542-105">Bir tür parametresi belirli bir sınıf türü kısıtlaması tarafından sınırlandırılırsa, sınıf türü kısıtlaması tarafından sınırlandırılmış tanım gereğidir ve bu nedenle sınıf türü kısıtlamasını belirtmek gereksizdir.</span><span class="sxs-lookup"><span data-stu-id="f6542-105">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f6542-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="f6542-106">Example</span></span>  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
