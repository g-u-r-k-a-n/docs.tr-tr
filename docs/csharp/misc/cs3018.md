## <a name="example"></a><span data-ttu-id="3ddbb-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="3ddbb-108">Example</span></span>  

 <span data-ttu-id="3ddbb-109">Aşağıdaki örnek CS3018 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="3ddbb-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
