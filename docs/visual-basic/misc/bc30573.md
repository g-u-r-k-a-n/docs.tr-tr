# <a name="xml-namespace-prefix-prefix-is-already-declared"></a>XML ad alanı öneki ' ön eki ' zaten tanımlanmış
Bir ifade tarafından tanımlanan bir XML ad alanı öneki, `Imports` önceden tanımlanmış BIR XML ad alanı önekiyle eşleşir.  
  
 **Hata kimliği:** BC30573  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML ad alanı öneki için benzersiz bir ad kullanın.
