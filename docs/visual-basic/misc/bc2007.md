# <a name="unrecognized-option-optionname-ignored"></a>Tanınmayan seçenek \<optionname> ; yoksayıldı

Komut satırından geçerli olmayan bir seçenek belirtildi.  
  
 **Hata kimliği:** BC2007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli olmayan bir tane bulmak için belirtilen seçenekleri işaretleyin.
