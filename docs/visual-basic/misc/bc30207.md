# <a name="option-compare-must-be-followed-by-text-or-binary"></a>' Option Compare ' arkasından ' text ' veya ' binary ' gelmelidir

Bir `Option Compare` ifade yanlış bir ayar veya ayar içeriyor. ' De izin verilen tek `Option Compare` Ayarlar `Text` ve ' dir `Binary` .  
  
 **Hata kimliği:** BC30207  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Ayar tanımlayıcısının yanlış yazılmış olup olmadığını denetleyin.  
  
2. `Text` `Binary` Deyimden ya da ifadeye ekleyin `Option Compare` ; Örneğin, `Option Compare Text` .
