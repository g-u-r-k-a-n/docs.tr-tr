<span data-ttu-id="feffe-119">Proje oluşturulduğunda, bir Dockerfile ve ile `dotnet`yapılandırılan çalışan çalışma zamanı dahil edilir.</span><span class="sxs-lookup"><span data-stu-id="feffe-119">When the project is created, it will include a Dockerfile and the worker runtime configured to `dotnet`.</span></span> <span data-ttu-id="feffe-120">Şimdi, işlevinizi yerel olarak oluşturup test edebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="feffe-120">Now, you can create and test your function locally.</span></span> <span data-ttu-id="feffe-121">`docker build` Ve `docker run` komutlarını kullanarak derleyin ve çalıştırın.</span><span class="sxs-lookup"><span data-stu-id="feffe-121">Build and run it using the  `docker build` and `docker run` commands.</span></span> <span data-ttu-id="feffe-122">Docker desteğiyle Azure Işlevleri oluşturmaya başlamanıza yönelik ayrıntılı adımlar için bkz.
