# <a name="tlbexp-helper-functions-unmanaged-api-reference"></a>Tlbexp Yardımcı İşlevleri (Yönetilmeyen API Başvurusu)
[Tür kitaplığı verme programı aracı](../../tools/tlbexp-exe-type-library-exporter.md) (Tlbexp. exe), TlbRef. dll adlı bir dinamik bağlantı kitaplığı yükler. Bu DLL iki yardımcı işlev ve derleme-tür kitaplığı dönüştürme işlemi sırasında dışarı aktarma aracının kullandığı bir arabirim içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [GetTypeLibInfo İşlevi](gettypelibinfo-function.md)  
 Bir tür kitaplığı için yerelleştirme ve işletim sistemi bilgilerini sağlar.  
  
 [LoadTypeLibWithResolver İşlevi](loadtypelibwithresolver-function.md)  
 Başvurulan tür kitaplıklarını çözümlemek için [ıtypeelibresolver arabiriminin](itypelibresolver-interface.md) bir uygulamasını kullanarak bir tür kitaplığı yükler.  
  
 [ITypeLibResolver Arabirimi](itypelibresolver-interface.md)  
 Bir tür kitaplığının tam yolunu döndüren [Resolvettypeınfo lib yöntemini](resolvetypelib-method.md)sağlar.
