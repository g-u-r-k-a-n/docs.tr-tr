# <a name="compiler-error-cs1909"></a><span data-ttu-id="d807f-103">Derleyici hatası CS1909</span><span class="sxs-lookup"><span data-stu-id="d807f-103">Compiler Error CS1909</span></span>

 <span data-ttu-id="d807f-105">`DefaultValue`Parametre türü üzerinde geçerli olmayan bir özniteliği KULLANDıĞıNıZDA CS1909 oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="d807f-105">CS1909 is generated when you use a `DefaultValue` attribute that is not applicable on the parameter type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d807f-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="d807f-106">Example</span></span>  

 <span data-ttu-id="d807f-107">Aşağıdaki örnek CS1909 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="d807f-107">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
