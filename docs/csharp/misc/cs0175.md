# <a name="compiler-error-cs0175"></a><span data-ttu-id="7b3ff-103">Derleyici hatası CS0175</span><span class="sxs-lookup"><span data-stu-id="7b3ff-103">Compiler Error CS0175</span></span>
<span data-ttu-id="7b3ff-104">' Base ' anahtar sözcüğünün kullanımı bu bağlamda geçerli değil</span><span class="sxs-lookup"><span data-stu-id="7b3ff-104">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="7b3ff-105">Taban [anahtar sözcüğünün, temel](../language-reference/keywords/base.md) sınıfın belirli bir üyesini belirtmek için kullanılması gerekir.</span><span class="sxs-lookup"><span data-stu-id="7b3ff-105">The [base](../language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="7b3ff-106">Daha fazla bilgi için bkz. [oluşturucular](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="7b3ff-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="7b3ff-107">Aşağıdaki örnek CS0175 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7b3ff-107">The following sample generates CS0175:</span></span>  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
