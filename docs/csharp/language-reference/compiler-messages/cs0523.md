# <a name="compiler-error-cs0523"></a><span data-ttu-id="252b7-103">Derleyici Hatası CS0523</span><span class="sxs-lookup"><span data-stu-id="252b7-103">Compiler Error CS0523</span></span>

<span data-ttu-id="252b7-104">' Struct1 ' türünün ' Struct2 Field ' yapı üyesi yapı düzeninde döngüye neden oluyor</span><span class="sxs-lookup"><span data-stu-id="252b7-104">Struct member 'struct2 field' of type 'struct1' causes a cycle in the struct layout</span></span>  
  
 <span data-ttu-id="252b7-105">İki yapının tanımları özyinelemeli başvuruları içerir.</span><span class="sxs-lookup"><span data-stu-id="252b7-105">The definitions of two structs include recursive references.</span></span> <span data-ttu-id="252b7-106">[Yapı](../builtin-types/struct.md) tanımlarını, her biri diğeri üzerinde tanımlamayan şekilde değiştirin.</span><span class="sxs-lookup"><span data-stu-id="252b7-106">Change the [struct](../builtin-types/struct.md) definitions such that each does not define itself on the other.</span></span> <span data-ttu-id="252b7-107">Yapılar değer türleri olduğundan, bu sınırlama yalnızca yapılar için geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="252b7-107">This limitation applies only to structs, since structs are value types.</span></span> <span data-ttu-id="252b7-108">Özyinelemeli başvuruları kullanıyorsanız, türlerinizi sınıflar olarak bildirin.</span><span class="sxs-lookup"><span data-stu-id="252b7-108">If using recursive references, declare your types as classes.</span></span>  
  
 <span data-ttu-id="252b7-109">Aşağıdaki örnek CS0523 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="252b7-109">The following sample generates CS0523:</span></span>  
  
```csharp

{  
   public RecursiveLayoutStruct2 field;  
}

{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
