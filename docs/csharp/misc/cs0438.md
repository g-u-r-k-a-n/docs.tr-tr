 <span data-ttu-id="8f49a-108">Aşağıdaki örnek CS0438 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8f49a-108">The following example generates CS0438:</span></span>

 <span data-ttu-id="8f49a-110">Sonra bu dosyayı derleyin:</span><span class="sxs-lookup"><span data-stu-id="8f49a-110">Then compile this file:</span></span>

```csharp  
   public class A { }  
}  
```  
  
 <span data-ttu-id="8f49a-111">Sonra bu dosyayı derleyin:</span><span class="sxs-lookup"><span data-stu-id="8f49a-111">And then compile this file:</span></span>

```csharp  
      Console.WriteLine(typeof(Util.A));   // CS0438
```
