# <a name="compiler-error-cs1009"></a><span data-ttu-id="80af8-102">Derleyici Hatası CS1009</span><span class="sxs-lookup"><span data-stu-id="80af8-102">Compiler Error CS1009</span></span>
<span data-ttu-id="80af8-103">Tanınmayan kaçış sırası</span><span class="sxs-lookup"><span data-stu-id="80af8-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="80af8-104">Beklenmeyen bir karakter bir\\ [dize](../builtin-types/reference-types.md#the-string-type)içinde bir ters çizgi ( ) izler.</span><span class="sxs-lookup"><span data-stu-id="80af8-104">An unexpected character follows a backslash (\\) in a [string](../builtin-types/reference-types.md#the-string-type).</span></span> <span data-ttu-id="80af8-105">Derleyici geçerli kaçış karakterlerinden birini bekler.</span><span class="sxs-lookup"><span data-stu-id="80af8-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="80af8-106">Daha fazla bilgi için [Karakter Kaçışları'na](../../../standard/base-types/character-escapes-in-regular-expressions.md)bakın.</span><span class="sxs-lookup"><span data-stu-id="80af8-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="80af8-107">Aşağıdaki örnek CS1009 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="80af8-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="80af8-108">Aşağıdaki örnekte görüldüğü gibi, bu hatanın yaygın bir nedeni, bir dosya adındaki ters eğik çizgi karakterini kullanmaktır.</span><span class="sxs-lookup"><span data-stu-id="80af8-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="80af8-109">Bu hatayı gidermek için\\\\aşağıdaki @-quoted örnekte görüldüğü gibi " " veya dize literal'i kullanın.</span><span class="sxs-lookup"><span data-stu-id="80af8-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
