## <a name="example"></a><span data-ttu-id="7d082-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="7d082-108">Example</span></span>  
 <span data-ttu-id="7d082-109">Aşağıdaki kod ve beyannamesinde `incorrect_1` CS0746 `incorrect_2`oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7d082-109">The following code generates CS0746 in the declaration of `incorrect_1` and `incorrect_2`.</span></span> <span data-ttu-id="7d082-110">Aşağıdaki bildirimler, anonim bir türü bildirmenin iki doğru yolunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="7d082-110">The following declarations show two of the correct ways to declare an anonymous type.</span></span>  
  
```csharp  
// cs0746.cs  
public class C  
{  
    public static int Main()  
    {  
        int i = 100;  
        string s = "Bottles of beer.";  
  
        var incorrect_1 = new { a.b = 1 }; // CS0746
        var incorrect_2 = new {100, "Bottles of beer."}; // CS0746  
        var correct_1 = new { i, s }; //OK  
        var correct_2 = new {num = i, message = s}; // OK  
  
        return 1;  
    }  
}  
```
