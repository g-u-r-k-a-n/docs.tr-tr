|||  
|-|-|  
|ID|2576|  
|Anahtar sözcükler|WFActivities|

## <a name="description"></a>Açıklama  

 TryCatch etkinliğinin bir özel durum yakalandığını belirtir.

 ' %1 ' TryCatch etkinliği ' %2 ' türünde bir özel durum yakaladı.
