## <a name="example"></a><span data-ttu-id="967b1-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="967b1-108">Example</span></span>  

 <span data-ttu-id="967b1-109">Aşağıdaki örnek CS0411 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="967b1-109">The following sample generates CS0411:</span></span>  
  
```csharp

{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
