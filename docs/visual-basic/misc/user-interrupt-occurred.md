# <a name="user-interrupt-occurred"></a>Kullanıcı kesmesi gerçekleşti

Kullanıcı CTRL + BREAK veya başka bir kesme tuşuna bastın.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İşlemi yeniden deneyin.
