 <span data-ttu-id="e5669-105">[Sealed](../language-reference/keywords/sealed.md) , [geçersiz kılma](../language-reference/keywords/override.md)ile işaretlenmeyen bir üyede kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="e5669-105">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="e5669-106">Daha fazla bilgi için bkz.  <span data-ttu-id="e5669-107">Aşağıdaki örnek CS0238 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e5669-107">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs

{  
   public abstract void f();  
}

{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
