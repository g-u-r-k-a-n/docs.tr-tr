## <a name="description"></a>Description  

 Bu hata, Aktarım Protokolü yanlış yapılandırıldığında oluşur.  
  
## <a name="troubleshooting"></a>Sorun giderme  

 web.config aktarım yapılandırma parametrelerini denetleyin ve onarın.
