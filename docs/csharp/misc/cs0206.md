## <a name="example"></a><span data-ttu-id="f61a4-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="f61a4-107">Example</span></span>  

 <span data-ttu-id="f61a4-108">Aşağıdaki örnek CS0206 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f61a4-108">The following sample generates CS0206:</span></span>  
  
```csharp  
// CS0206.cs  
public class MyClass  
{  
    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMethod(ref int i)  
    // public static void MyMethod(int i)  
    {  
    }  
  
    public static void Main()  
    {  
        MyMethod(ref P);   // CS0206  
        // try the following line instead  
        // MyMethod(P);   // CS0206  
    }  
}  
```
