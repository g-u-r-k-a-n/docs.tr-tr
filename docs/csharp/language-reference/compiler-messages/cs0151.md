## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="c65cf-107">Belirsiz dönüştürme örneği</span><span class="sxs-lookup"><span data-stu-id="c65cf-107">Example of ambiguous conversion</span></span>

<span data-ttu-id="c65cf-108">Bu hata, dönüştürme olmadığında ya da kullanılabilir örtük dönüştürmeler belirsiz bir durumla sonuçlandıysa ortaya çıkabilir.</span><span class="sxs-lookup"><span data-stu-id="c65cf-108">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="c65cf-109">Aşağıdaki örnek CS0151 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c65cf-109">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```
