# <a name="runtime-activities-in-wf"></a>WF 'de çalışma zamanı etkinlikleri

[!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] , kalıcılık ve sonlandırma gibi iş akışı çalışma zamanının özelliklerine erişmek için sistem tarafından sağlanan çeşitli etkinlikler sağlar.  
  
|Etkinlik|Description|  
|--------------|-----------------|  
|<xref:System.Activities.Statements.Persist>|İş akışının durumunu kalıcı olarak sürdürmediğini açıkça ister.|  
|<xref:System.Activities.Statements.TerminateWorkflow>|Çalışan iş akışı örneğini sonlandırır.|  
|<xref:System.Activities.Statements.NoPersistScope>|Alt etkinliklerin kalıcı olmasını önleyen bir kapsayıcı etkinliği.|
