# <a name="queue-dropped-messages"></a>Kuyruktan Bırakılan İletiler

Sayaç adı: kuyruğa alınan Iletiler bırakıldı.  
  
## <a name="description"></a>Açıklama  

 Bu hizmette sıraya alınan aktarım tarafından bırakılan ileti sayısı.  
  
 İletiler ne zaman bırakıldığında hakkında daha fazla bilgi için bkz. [Poison-Message Handling](../../feature-details/poison-message-handling.md).
