## <a name="example"></a><span data-ttu-id="00759-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="00759-107">Example</span></span>  

 <span data-ttu-id="00759-108">Aşağıdaki örnek CS0082 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="00759-108">The following example generates CS0082:</span></span>  
  
```csharp  
//cs0082.cs

{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get

        return 2;  
    }

    {  
        return 1;  
    }  
}  
```
