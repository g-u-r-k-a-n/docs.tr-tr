# <a name="compiler-error-cs1528"></a><span data-ttu-id="971e1-103">Derleyici hatası CS1528</span><span class="sxs-lookup"><span data-stu-id="971e1-103">Compiler Error CS1528</span></span>

<span data-ttu-id="971e1-104">Beklenen OR = (bildirimde Oluşturucu bağımsız değişkenleri belirtilemez)</span><span class="sxs-lookup"><span data-stu-id="971e1-104">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="971e1-105">Sınıfa yönelik bir nesne oluşturulmakta gibi bir sınıfa yönelik bir başvuru oluşturulmuş.</span><span class="sxs-lookup"><span data-stu-id="971e1-105">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="971e1-106">Örneğin, bir değişkene bir değişken geçirme girişimi vardı.</span><span class="sxs-lookup"><span data-stu-id="971e1-106">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="971e1-107">Bir sınıfın nesnesini oluşturmak için [New](../language-reference/operators/new-operator.md) işlecini kullanın.</span><span class="sxs-lookup"><span data-stu-id="971e1-107">Use the [new](../language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="971e1-108">Aşağıdaki örnek CS1528 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="971e1-108">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
