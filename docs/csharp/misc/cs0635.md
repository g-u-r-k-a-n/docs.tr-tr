# <a name="compiler-error-cs0635"></a>Derleyici hatası CS0635

' Attribute ': System. Interop. UnmanagedType. CustomMarshaller adlandırılmış bağımsız değişkenler ComType ve Marshal gerektirir  
  
 Sıralama biçimi **System. Interop. UnmanagedType. CustomMarshaller**olduğunda **Comtype** ve **Marshal** bağımsız değişkenlerinin belirtilmesi gerekir.
