# <a name="compiler-error-cs1599"></a>Derleyici hatası CS1599
Yöntem veya temsilci ' Type ' türünü döndüremiyor  
  
 .NET sınıf kitaplığındaki bazı türler, örneğin, <xref:System.TypedReference> <xref:System.RuntimeArgumentHandle> ve, <xref:System.ArgIterator> güvenli olmayan işlemleri gerçekleştirmek için kullanılabilecek potansiyel olarak kullanılabilecek bir dönüş türü olarak kullanılamaz.  
  
 Aşağıdaki örnek CS1599 oluşturur:  
  
```csharp  
// CS1599.cs  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```
