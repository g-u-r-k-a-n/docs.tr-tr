# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametre, tanımlayan işlevle aynı ada sahip olamaz

Aynı adı hem parametrede hem de tanımlayan işlevinde kullandınız.  
  
 **Hata kimliği:** BC30530  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İşlev içindeki parametre adlarını denetleyin ve çakışan parametrenin adını değiştirin.
