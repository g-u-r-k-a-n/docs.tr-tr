 <span data-ttu-id="077a1-106">Derleyici, `override` olarak işaretlenmiş bir yöntem buldu `obsolete` .</span><span class="sxs-lookup"><span data-stu-id="077a1-106">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="077a1-107">Ancak, geçersiz kılma yöntemi kendisi kullanım dışı olarak işaretlenmemiştir.</span><span class="sxs-lookup"><span data-stu-id="077a1-107">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="077a1-108">Eğer çağrılırsa, geçersiz kılma yöntemi hala [CS0612](./cs0612.md)oluşturur.</span><span class="sxs-lookup"><span data-stu-id="077a1-108">The overriding method will still generate [CS0612](./cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="077a1-109">Yöntem bildirimlerinizi gözden geçirin ve bir yöntemin (ve tüm geçersiz kılmaların tümünün) işaretlenip işaretlenmeyeceğini açıkça belirtin `obsolete` .</span><span class="sxs-lookup"><span data-stu-id="077a1-109">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="077a1-110">Aşağıdaki örnek CS0672 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="077a1-110">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
