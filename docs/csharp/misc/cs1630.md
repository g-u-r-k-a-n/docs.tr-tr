# <a name="compiler-error-cs1630"></a>Derleyici hatası CS1630

**Errorreport** için ' Option ' geçersiz seçeneği; **Prompt**, **Send**, **Queue** veya **none** olmalıdır  
  
 Komut satırı seçeneği [**errorreport**](../language-reference/compiler-options/advanced.md#errorreport) ' ın ardından **Prompt**, **Send**, **Queue** veya **none** gelmelidir ve bir iç derleyici hatası oluştuğunda hangi eylemin yapılacağını belirtin.
