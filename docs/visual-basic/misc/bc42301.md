# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Dil öğesi başına yalnızca bir XML açıklama bloğuna izin verilir

Bir dil öğesine birden fazla XML açıklama bloğu uygulandı.  
  
 **Hata kimliği:** BC42301  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz XML açıklama bloğunu kaldırın.
