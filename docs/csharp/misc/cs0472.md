<span data-ttu-id="dcc51-104">' Değer2 ' türündeki bir değer hiçbir zaman ' value3 ' türündeki ' null ' değerine eşit olmadığından, ifadenin sonucu her zaman ' değer1 ' olur</span><span class="sxs-lookup"><span data-stu-id="dcc51-104">The result of the expression is always 'value1' since a value of type 'value2' is never equal to 'null' of type 'value3'</span></span>

## <a name="example"></a><span data-ttu-id="dcc51-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="dcc51-106">Example</span></span>  
 <span data-ttu-id="dcc51-107">Aşağıdaki örnek CS0472 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="dcc51-107">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;
        return counter;  
    }  
}  
```
