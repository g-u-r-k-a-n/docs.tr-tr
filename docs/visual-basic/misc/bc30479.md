# <a name="shared-sub-new-cannot-have-any-parameters"></a>Paylaşılan ' Sub New ' parametre içeremez
Paylaşılan bir deyime bir parametre veya parametreler eklemiş olabilirsiniz; `Sub New` Bu geçerli değildir.  
  
 **Hata kimliği:** BC30479  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Parametre veya parametreleri kaldırın.
