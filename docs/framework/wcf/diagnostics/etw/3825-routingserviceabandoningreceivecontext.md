# <a name="3825---routingserviceabandoningreceivecontext"></a>3825 - RoutingServiceAbandoningReceiveContext

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|3825|  
|Anahtar sözcükler|Yönlendirme hizmetleri|  
|Level|Uyarı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Description  

 Bu olay, yönlendirme hizmeti belirtilen KIMLIĞE sahip bir ileti için ReceiveContext. Abandon öğesini çağırırken yayınlanır.  
  
## <a name="message"></a>İleti  

 Yönlendirme hizmeti KIMLIĞI ' %1 ' olan Ileti için ReceiveContext. Abandon 'ı çağırıyor.
