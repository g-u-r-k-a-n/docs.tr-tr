# <a name="property-propertyname-is-readonly"></a>' \<propertyname> ' Özelliği ' ReadOnly '
Belirtilen bir özelliğe değer atamaya çalıştınız `ReadOnly` .  
  
 **Hata kimliği:** BC30526  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `ReadOnly`Özellik bildiriminden belirteci kaldırın.
