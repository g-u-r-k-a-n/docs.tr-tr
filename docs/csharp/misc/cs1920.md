# <a name="compiler-error-cs1920"></a><span data-ttu-id="3084d-103">Derleyici hatası CS1920</span><span class="sxs-lookup"><span data-stu-id="3084d-103">Compiler Error CS1920</span></span>

<span data-ttu-id="3084d-104">Öğe başlatıcısı boş olamaz.</span><span class="sxs-lookup"><span data-stu-id="3084d-104">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="3084d-105">Koleksiyon Başlatıcısı, bir dizi öğe başlatıcısından oluşur.</span><span class="sxs-lookup"><span data-stu-id="3084d-105">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="3084d-106">Öğe başlatıcılarının atama ifadesi içermediği müddetçe küme ayraçları içine alınması gerekmez.</span><span class="sxs-lookup"><span data-stu-id="3084d-106">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="3084d-107">Ancak, küme ayraçları yaparsanız boş olamaz.</span><span class="sxs-lookup"><span data-stu-id="3084d-107">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="3084d-108">Öğe başlatıcısı bir nesne başlatıcısiyse, başlatıcı yeni bir nesne oluşturma ifadesi içerdiği sürece küme ayraçları boş olabilir.</span><span class="sxs-lookup"><span data-stu-id="3084d-108">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3084d-109">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="3084d-109">To correct this error</span></span>

- <span data-ttu-id="3084d-111">İfadenin bir nesne Başlatıcısı olması amaçlanıyorsa, küme ayracın önüne yeni nesne oluşturma ifadesini ekleyin.</span><span class="sxs-lookup"><span data-stu-id="3084d-111">If the expression is intended to be an object initializer, add the new object creation expression in front of the braces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3084d-112">Örnek</span><span class="sxs-lookup"><span data-stu-id="3084d-112">Example</span></span>  

 <span data-ttu-id="3084d-113">Aşağıdaki örnek CS1920 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3084d-113">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```
