# <a name="member-implicitly-defined-for-eventname-cannot-shadow-a-mustoverride-method-in-the-base-class-classname"></a>' ' \<member> için örtük olarak tanımlanan ' ', ' \<eventname> ' tabanında bir ' MustOverride ' metodunu gölgeleyemez \<class> \<classname>
Belirtilen olay, değiştiriciyle belirtilen bir yöntemle aynı ada sahip bir üyeyi örtük olarak bildiriyor `MustOverride` .  
  
 **Hata kimliği:** BC31413  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `MustOverride`Değiştiriciyi taban sınıftaki yöntemden kaldırın veya özelliğe ya da yönteme benzersiz bir ad verin.
