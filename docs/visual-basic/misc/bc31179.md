# <a name="expected--at-start-of-an-embedded-expression"></a>Katıştırılmış ifadenin başlangıcında '% = ' bekleniyor

Katıştırılmış bir ifadenin başlangıç tanımlayıcısı `<%=` doğru şekilde kodlanmadı. Yüzde karakterini (%) kullanmayacağınızı unutmayın bir XML öğesi sabit değerinin adında.  
  
 **Hata kimliği:** BC31179  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Katıştırılmış ifade için başlangıç tanımlayıcısının olarak kodlanmış olduğundan emin olun `<%=` .
