# <a name="return-statements-are-not-valid-in-the-immediate-window"></a>' Return ' deyimleri Komut penceresinde geçerli değildir

`Return`İfade, dallanma gerçekleştirir ve hata ayıklama bağlamında buna izin verilmez.  
  
 **Hata kimliği:** BC30147  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Return` **Komut** penceresinde bir ifade vermeyin.
