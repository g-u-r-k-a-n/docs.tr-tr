# <a name="comma-expected"></a>Virgül bekleniyor
Bağımsız değişken listesinde bağımsız değişkenlerin ikisi arasında virgül (,) eksik.  
  
 **Hata kimliği:** BC30196  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm bağımsız değişkenlerin virgülle ayrıldığından emin olun.
