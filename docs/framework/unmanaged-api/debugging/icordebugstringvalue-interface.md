## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetLength Yöntemi](icordebugstringvalue-getlength-method.md)|Bu tarafından başvurulan dizedeki karakter sayısını alır `ICorDebugStringValue` .|
