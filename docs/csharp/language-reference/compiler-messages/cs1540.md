# <a name="compiler-error-cs1540"></a><span data-ttu-id="59483-102">Derleyici Hatası CS1540</span><span class="sxs-lookup"><span data-stu-id="59483-102">Compiler Error CS1540</span></span>
<span data-ttu-id="59483-103">Korumalı üye 'üye'ye 'type1' türünden bir niteleyici aracılığıyla erişemez; niteleyici türü 'type2' (veya ondan türetilmiş) olmalıdır</span><span class="sxs-lookup"><span data-stu-id="59483-103">Cannot access protected member 'member' via a qualifier of type 'type1'; the qualifier must be of type 'type2' (or derived from it)</span></span>  
  
 <span data-ttu-id="59483-104">Türetilen [bir sınıf,](../keywords/class.md) taban sınıfın bir örneği aracılığıyla taban sınıfının korumalı üyelerine erişemez.</span><span class="sxs-lookup"><span data-stu-id="59483-104">A derived [class](../keywords/class.md) cannot access protected members of its base class through an instance of the base class.</span></span> <span data-ttu-id="59483-105">Türetilmiş sınıfta bildirilen taban sınıfın bir örneği, çalışma zamanında, aynı tabandan türetilen ancak türemiş sınıfla başka bir şekilde ilişkili olmayan başka bir türe örnek olabilir.</span><span class="sxs-lookup"><span data-stu-id="59483-105">An instance of the base class declared in the derived class might, at run time, be an instance of another type that is derived from the same base but is not otherwise related to the derived class.</span></span> <span data-ttu-id="59483-106">Korumalı üyelere yalnızca türetilmiş türler tarafından erişilebildiği için, çalışma zamanında geçerli olmayan korumalı üyelere erişme girişimleri derleyici tarafından geçerli değil olarak işaretlenir.</span><span class="sxs-lookup"><span data-stu-id="59483-106">Because protected members can be accessed only by derived types, any attempts to access protected members that might not be valid at run time are marked by the compiler as not valid.</span></span>  
  
 <span data-ttu-id="59483-107">Aşağıdaki `Employee` örnekte sınıfta `emp2` `emp3` ve her ikisi `Person` de derleme zamanda `emp2` türü `Manager` var, ancak çalışma zamanında türü vardır.</span><span class="sxs-lookup"><span data-stu-id="59483-107">In the `Employee` class in the following example, `emp2` and `emp3` both have type `Person` at compile time, but `emp2` has type `Manager` at run time.</span></span> <span data-ttu-id="59483-108">Türetilmedığından, `Employee` `Manager` sınıfın bir örneği aracılığıyla taban `Person`sınıfın korumalı üyelerine erişemez. `Manager`</span><span class="sxs-lookup"><span data-stu-id="59483-108">Because `Employee` is not derived from `Manager`, it cannot access the protected members of the base class, `Person`, through an instance of the `Manager` class.</span></span> <span data-ttu-id="59483-109">Derleyici, iki `Person` nesnenin çalışma zamanı türünün ne olacağını belirleyemez.</span><span class="sxs-lookup"><span data-stu-id="59483-109">The compiler cannot determine what the run-time type of the two `Person` objects will be.</span></span> <span data-ttu-id="59483-110">Bu nedenle, hem `emp2` arama ve `emp3` neden derleyici hatası CS1540 gelen arama.</span><span class="sxs-lookup"><span data-stu-id="59483-110">Therefore, both the call from `emp2` and the call from `emp3` cause compiler error CS1540.</span></span>  
  
```csharp  
using System;  
using System.Text;  
  
namespace CS1540  
{  
    class Program1  
    {  
        static void Main()  
        {  
            Employee.PreparePayroll();  
        }  
    }  
  
    class Person  
    {  
        protected virtual void CalculatePay()
        {  
            Console.WriteLine("CalculatePay in Person class.");  
        }  
    }  
  
    class Manager : Person  
    {  
        protected override void CalculatePay()
        {  
            Console.WriteLine("CalculatePay in Manager class.");
  
        }  
    }  
  
    class Employee : Person  
    {  
        public static void PreparePayroll()  
        {  
            Employee emp1 = new Employee();  
            Person emp2 = new Manager();  
            Person emp3 = new Employee();  
            // The following line calls the method in the Employee base class,  
            // Person.  
            emp1.CalculatePay();
  
            // The following lines cause compiler error CS1540. The compiler
            // cannot determine at compile time what the run-time types of
            // emp2 and emp3 will be.  
            //emp2.CalculatePay();
            //emp3.CalculatePay();  
  
        }  
    }  
}  
```
