## <a name="example"></a><span data-ttu-id="624bd-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="624bd-109">Example</span></span>  

 <span data-ttu-id="624bd-110">Aşağıdaki örnek CS0131 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="624bd-110">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```
