# <a name="core-communications-channel-framework"></a>Temel İletişimler: Kanal Çerçevesi

Bu konu, Windows Communication Foundation (WCF) Kanal çerçevesi tarafından oluşturulan tüm özel durumları listeler.  
  
## <a name="exception-list"></a>Özel durum listesi  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|Receiverequesttimeın|Belirtilen yerel adreste alınan istek belirtilen süreden sonra zaman aşımına uğradı. Bu işlem için ayrılan süre daha uzun bir zaman aşımı değerinin bir bölümü olabilir.|  
|ReceiveTimedOutNoLocalAddress|Alma işlemi belirtilen süreden sonra zaman aşımına uğradı. Bu işlem için ayrılan süre daha uzun bir zaman aşımı değerinin bir bölümü olabilir.|  
|UnableToDemuxChannel|Belirtilen eyleme sahip iletiyi kabul etmek için kullanılabilir kanal yok.|
