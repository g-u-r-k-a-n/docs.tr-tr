# <a name="arrays-used-as-attribute-arguments-are-required-to-explicitly-specify-values-for-all-elements"></a>Öznitelik bağımsız değişkenleri olarak kullanılan diziler, tüm öğelerin değerlerini açıkça belirtmek için gereklidir
Öznitelik bağımsız değişkeni olarak kullanılan bir dizi, belirtilmeyen bir değere sahip bir öğe içeriyor.  
  
 **Hata kimliği:** BC31110  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm öğe değerlerini belirtin.
