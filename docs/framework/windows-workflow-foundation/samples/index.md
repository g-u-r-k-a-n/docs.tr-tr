# <a name="windows-workflow-wf-samples"></a>Windows Workflow (WF) Örnekleri

Windows Workflow Foundation (WF) çeşitli yönlerini gösteren yönergeler sağlayan [Windows Iş akışı örneklerini indirebilirsiniz](https://www.microsoft.com/download/details.aspx?id=21459) .

Bu bölümdeki makalelerde, indirme paketindeki bazı örnekler açıklanır. Tüm örnekleri içeren bir tüm belge kümesi için [.NET Framework 4 WF örnekleri](/previous-versions/dotnet/netframework-4.0/dd483375(v=vs.100)) bölümüne bakın.

> [!NOTE]
> İndirilebilir örnekler Visual Studio 2010 ve .NET Framework 4 ile oluşturulmuştur, ancak Visual Studio 'nun sonraki sürümleriyle ve .NET Framework uyumludur. Windows Workflow Foundation için ek örnekler [!INCLUDE[netfx_current_short](../../../../includes/netfx-current-short-md.md)] , [MSDN kod örneklerinde](/samples/browse/?redirectedfrom=MSDN-samples)bulunabilir.
