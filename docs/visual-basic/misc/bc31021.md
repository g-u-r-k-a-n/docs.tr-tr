# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-error"></a>Visual Basic derleyicisi aşağıdaki hatadan kurtumı yapamıyor: \<error>

Visual Basic Derleyicisi şu hatayı kurtaramıyor: <`error`>.  
  
 Bazı IDE özellikleri devre dışı bırakıldı.  
  
 Visual Basic derleyicisinde bir iç hata oluştu.  
  
 **Hata kimliği:** BC31021  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Çalışmanızı kaydedin ve Visual Studio 'Yu yeniden başlatın.  
  
2. Hata yinelenirse, Visual Studio 'Yu yeniden yükleyin.  
  
3. Yeniden yükleme sonrasında hata devam ederse, Microsoft Ürün Destek Hizmetleri 'ne bildirin.
