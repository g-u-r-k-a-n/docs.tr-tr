## <a name="example"></a><span data-ttu-id="80912-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="80912-111">Example</span></span>  

 <span data-ttu-id="80912-112">Aşağıdaki kod, `Program.testMethod` statik olarak bildirildiği IÇIN CS0736 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="80912-112">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>

```csharp  
{
  
    interface ITest  
    {  
        int testMethod(int x);

        // Try the following line instead.

    }
}  
```
