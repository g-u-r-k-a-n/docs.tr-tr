# <a name="option-option-cannot-be-followed-by--or---ignored"></a>' \<option> ' seçeneğinin arkasından ' + ' veya '-' gelemez; yoksayıldı

Derleyici seçeneği değerle belirtildi `+` `-` ; Bu değerler desteklenmez.  
  
 Derleyici seçenekleri ve destekledikleri değerler hakkında daha fazla bilgi için [alfabetik olarak listelenen Visual Basic derleyici seçenekleri](../reference/command-line-compiler/compiler-options-listed-alphabetically.md)bölümüne bakın.  
  
 **Hata kimliği:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Bu hatayı düzeltmek ve seçeneğini kullanmak için  
  
- Derleyici seçeneğini hiçbir değer olmadan belirtin.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Bu hatayı düzeltmek ve seçeneğini kullanmayın  
  
- Derleyici seçeneğini kaldırın.
