# <a name="compiler-error-cs1664"></a>Derleyici hatası CS1664

' Length ' uzunluğunda ve ' Type ' türünde sabit boyutlu arabellek çok büyük  
  
 Sabit boyutlu bir arabelleğin en büyük boyutu (uzunluğu öğe boyutuyla çarpılarak belirlendiği şekilde) 2 ^ 31 = 268435455 olur.
