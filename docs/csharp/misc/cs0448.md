# <a name="compiler-error-cs0448"></a>Derleyici hatası CS0448
+ + Veya--işleci için dönüş türü, kapsayan tür veya kapsayan türden türetilmiş olmalıdır  
  
 `++`Veya işleçlerini geçersiz kıldığınızda `--` , kapsayan tür ile aynı türü döndürmelidir veya kapsayan türden türetilmiş bir tür döndürmelidir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0448 oluşturur.  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0448 oluşturur.  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
