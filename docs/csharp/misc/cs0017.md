# <a name="compiler-error-cs0017"></a><span data-ttu-id="2fcfe-103">Derleyici hatası CS0017</span><span class="sxs-lookup"><span data-stu-id="2fcfe-103">Compiler Error CS0017</span></span>

<span data-ttu-id="2fcfe-104">' Çıkış dosyası adı ' programında tanımlanmış birden fazla giriş noktası yok.</span><span class="sxs-lookup"><span data-stu-id="2fcfe-104">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="2fcfe-105">Giriş noktasını içeren türü belirtmek için/Main ile derleyin.</span><span class="sxs-lookup"><span data-stu-id="2fcfe-105">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="2fcfe-106">Programda yalnızca bir tane [ana](../programming-guide/main-and-command-args/index.md) yöntem olabilir.</span><span class="sxs-lookup"><span data-stu-id="2fcfe-106">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="2fcfe-107">Bu hatayı çözmek için, kodunuzdaki tüm ana yöntemleri silebilir ya da hangi ana yöntemi kullanmak istediğinizi belirtmek için [**StartupObject**](../language-reference/compiler-options/advanced.md#mainentrypoint-or-startupobject) derleyici seçeneğini kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="2fcfe-107">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [**StartupObject**](../language-reference/compiler-options/advanced.md#mainentrypoint-or-startupobject) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="2fcfe-108">Aşağıdaki örnek CS0017 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2fcfe-108">The following sample generates CS0017:</span></span>

```csharp  
{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
