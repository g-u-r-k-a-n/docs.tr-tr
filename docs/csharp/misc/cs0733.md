# <a name="compiler-error-cs0733"></a>Derleyici hatası CS0733
Genel tür iletilemez, ' GenericType<> '  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0733 oluşturur. İlk dosyayı bir kitaplık olarak derleyin ve ardından ikinci dosyayı derlerken ona başvurun.  
  
```csharp  
// CS0733a.cs  
// compile with: /target:library  
public class GenericType<T>
{  
}  
```  
  
```csharp  
// CS0733.cs  
// compile with: /target:library /r:CS0733a.dll  
[assembly: System.Runtime.CompilerServices.TypeForwardedTo(typeof(GenericType<int>))]   // CS0733  
```
