# <a name="compiler-warning-level-3-cs1702"></a>Derleyici Uyarısı (düzey 3) CS1702

"Derleme adı #1" derleme başvurusunun "derleme adı #2" ile eşleştiği varsayılarak, çalışma zamanı ilkesi sağlamanız gerekebilir  
  
 İki derleme başvurusu farklı derleme ve/veya düzeltme numaralarına sahiptir, bu nedenle otomatik olarak birleşmez. Application. config dosyasındaki yönergeleri kullanarak birleşme zorlamak için çalışma zamanı ilkesi sağlamanız gerekebilir.
