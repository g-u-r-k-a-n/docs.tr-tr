# <a name="compiler-error-cs1719"></a>Derleyici hatası CS1719

' Dosya adı ' Win32 kaynak dosyası okunurken hata--' Reason '  
  
 "Dosya bulunamadı" veya "erişim engellendi" gibi, hatada verilen nedenden dolayı Win32 kaynak dosyasını okuma girişimi başarısız oldu. Bu hata, neden tarafından açıklanan sorun düzeltilerek çözümlenir.
