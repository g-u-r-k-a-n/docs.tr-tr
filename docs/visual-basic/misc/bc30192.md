# <a name="end-of-parameter-list-expected"></a>Parametre listesi sonu bekleniyor

Bir `ParamArray` belirtim, yordam bildiriminde bir veya daha fazla bağımsız değişken belirtimleri izler.  
  
 **Hata kimliği:** BC30192  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öğesini `ParamArray` bağımsız değişken listesinin sonuna taşı.
