# <a name="argument-argumentname-is-not-valid-for-the-array"></a>' ' Bağımsız değişkeni \<argumentname> dizi için geçerli değil
Bir dizi geçerli olmayan bir bağımsız değişken içeriyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfadedeki bağımsız değişkenlerin yazımını denetleyin. Yanlış yazılmış bir değişken adı, sıfır olarak başlatılan bir sayısal değişkeni örtülü olarak oluşturabilir.
