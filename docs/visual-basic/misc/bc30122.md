# <a name="stop-statements-are-not-valid-in-the-immediate-window"></a>' Stop ' deyimleri Komut penceresinde geçerli değildir

`Stop`Ve `End` deyimleri yürütmeyi askıya alır ve bir hata ayıklama bağlamında buna izin verilmez.  
  
 **Hata kimliği:** BC30122  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir `Stop` veya `End` ifadesini **komut** penceresinde vermeyin.
