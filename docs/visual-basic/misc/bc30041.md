# <a name="maximum-number-of-errors-has-been-exceeded"></a>En fazla hata sayısı aşıldı

En fazla derleme hatası sayısı 100 ' dir. Derleyici, hata raporlarının boyutunu azaltmak için bu sınırı uygular.  
  
 **Hata kimliği:** BC30041  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu derleme tarafından oluşturulan diğer hataları gözden geçirin ve mümkün olduğunca fazla çözün. Kaynak kodu 100 'den fazla hata içeriyorsa, sonraki derlemede daha fazla hata görebilirsiniz.
