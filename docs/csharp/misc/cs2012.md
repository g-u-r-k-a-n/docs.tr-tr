# <a name="compiler-error-cs2012"></a>Derleyici hatası CS2012

' Dosya ' yazma için açılamıyor  
  
`-bugreport:`Derleyici seçeneğini kullanırken, `file` dosya yazmak için açılamadı. Geçerli bir dosya adı belirttiğinizden ve dosyanın salt okunmadığından emin olun.

> [!IMPORTANT]
> `bugreport`Seçenek artık desteklenmiyor. Bunun yerine [**errorreport**](../language-reference/compiler-options/advanced.md#errorreport) seçeneğini kullanın.
