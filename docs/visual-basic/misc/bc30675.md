# <a name="synclock-statement-must-end-with-a-matching-end-synclock"></a>' SyncLock ' ifadesinin eşleşen bir ' End SyncLock ' ile bitmesi gerekir

`SyncLock` bloklar `SyncLock` anahtar sözcüğüyle başlar ve yapı ile biter `End SyncLock` .  
  
 **Hata kimliği:** BC30675  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `SyncLock`Bloğun bir yapı ile sonlantığınızdan emin olun `End SyncLock` .
