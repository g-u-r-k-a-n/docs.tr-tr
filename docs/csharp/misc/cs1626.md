# <a name="compiler-error-cs1626"></a><span data-ttu-id="7a190-103">Derleyici hatası CS1626</span><span class="sxs-lookup"><span data-stu-id="7a190-103">Compiler Error CS1626</span></span>

<span data-ttu-id="7a190-104">Catch yan tümcesiyle bir try bloğunun gövdesinde değer değerlendirilemiyor</span><span class="sxs-lookup"><span data-stu-id="7a190-104">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="7a190-105">Try bloğu ile ilişkili bir catch yan tümcesi varsa, try bloğunda bir yield ifadesine izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="7a190-105">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="7a190-106">Bu hatayı önlemek için, yield ifadesini try/catch/finally bloğunun dışına taşıyın ya da catch bloğunu kaldırın.</span><span class="sxs-lookup"><span data-stu-id="7a190-106">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="7a190-107">Aşağıdaki örnek CS1626 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7a190-107">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
