# <a name="collection-index-must-be-in-the-range-1-to-the-size-of-the-collection"></a>Koleksiyon dizini, koleksiyonun boyutunun 1 aralığında olmalıdır

Sağlanan dizin değeri, koleksiyonun boyutuyla 1 olan geçerli aralığın dışında kalıyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Sağlanan değeri geçerli bir değerle değiştirin.
