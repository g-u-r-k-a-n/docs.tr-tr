# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>' \<typename> ' Türü, ' Public ' olarak bildirildiği için öznitelik olarak kullanılamaz
Özniteliklerin ortak olması gerekir.  
  
 **Hata kimliği:** BC31516  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Public`Değiştiricisini Attribute sınıfına ekleyin.
