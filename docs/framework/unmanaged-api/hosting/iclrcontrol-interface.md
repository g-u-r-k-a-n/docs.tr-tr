# <a name="iclrcontrol-interface"></a>ICLRControl Arabirimi

, Bir konağın ortak dil çalışma zamanı (CLR) için başvuruları almasını ve yönlerini yapılandırmasına izin veren yöntemler sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetCLRManager Yöntemi](iclrcontrol-getclrmanager-method.md)|Konağın CLR 'yi yapılandırmak için kullanabileceği bir yönetici türü örneğine yönelik bir arabirim işaretçisi alır.|  
|[SetAppDomainManagerType Yöntemi](iclrcontrol-setappdomainmanagertype-method.md)|<xref:System.AppDomainManager>Uygulama etki alanı yöneticileri için tür olarak türetilmiş bir tür belirler.|
