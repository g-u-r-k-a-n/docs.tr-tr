## <a name="pseudo-code-examples"></a><span data-ttu-id="1b0e2-126">Sözde kod örnekleri</span><span class="sxs-lookup"><span data-stu-id="1b0e2-126">Pseudo-code examples</span></span>

### <a name="potential-reference-cycle-violation"></a><span data-ttu-id="1b0e2-127">Olası başvuru çevrimi ihlali</span><span class="sxs-lookup"><span data-stu-id="1b0e2-127">Potential reference cycle violation</span></span>

```csharp
using System;

[Serializable()]
class ExampleClass
{
    public ExampleClass ExampleProperty {get; set;}

    public int NormalProperty {get; set;}
}

class AnotherClass
{
    // The argument passed by could be `JsonConvert.DeserializeObject<ExampleClass>(untrustedData)`.
    public void AnotherMethod(ExampleClass ec)
    {
        while(ec != null)
        {
            Console.WriteLine(ec.ToString());
            ec = ec.ExampleProperty;
        }
    }
}
```
