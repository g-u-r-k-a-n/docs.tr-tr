# \<policyImporters>

Bu yapılandırma öğesi bağlamalar hakkında özel ilke onaylamalarını denetleyen tüm ilke içe aktarımlarını belirler. Her alt öğe, `policyImporter` bağlama özellikleriyle ilgili özel ilke onaylamalarını aramak için kullanılan bir <> ve onaylama için gereken özellikleri uygulayan özel bir bağlama öğesi ekler.
