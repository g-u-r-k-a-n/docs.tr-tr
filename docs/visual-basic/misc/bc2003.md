# <a name="response-file-filename-included-multiple-times"></a>Yanıt dosyası \<filename> birden çok kez eklendi
Bir yanıt dosyası birden çok kez belirtildi. Yanıt dosyaları derlemek için derleyici seçeneklerini ve kaynak kodu dosyalarını içerir.  
  
 **Hata kimliği:** BC2003  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz yanıt dosyası belirticilerini kaldırın.
