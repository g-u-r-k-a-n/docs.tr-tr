# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Yöntem bildirim deyimleri mantıksal satırda ilk kez olmalıdır

`Function`Or `Sub` ifadesinin iki nokta üst üste (:) Ekstre ayırıcı. `Function` ya da `Sub` kaynak satırındaki tek bir ifade olmalıdır.  
  
 **Hata kimliği:** BC32009  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyimi farklı satırlara ayırın.
