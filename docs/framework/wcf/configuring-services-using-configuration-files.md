 <span data-ttu-id="b6976-178">WCF, yapılandırmada bir başvuru sistemi kullanarak uç noktalar arasında yapılandırmaların paylaşılmasını kolaylaştırır.</span><span class="sxs-lookup"><span data-stu-id="b6976-178">WCF makes it easy to share configurations between endpoints using a reference system in configuration.</span></span> <span data-ttu-id="b6976-179">Yapılandırma değerlerini bir uç noktaya doğrudan atamak yerine, bağlama ilgili yapılandırma değerleri, `bindingConfiguration` bölümündeki öğelerde gruplandırılır `<binding>` .</span><span class="sxs-lookup"><span data-stu-id="b6976-179">Rather than directly assigning configuration values to an endpoint, binding-related configuration values are grouped in `bindingConfiguration` elements in the `<binding>` section.</span></span> <span data-ttu-id="b6976-180">Bağlama yapılandırması, bağlamada adlandırılmış bir ayar grubudur.</span><span class="sxs-lookup"><span data-stu-id="b6976-180">A binding configuration is a named group of settings on a binding.</span></span> <span data-ttu-id="b6976-181">Uç noktalar daha sonra `bindingConfiguration` adına göre başvuru yapabilir.</span><span class="sxs-lookup"><span data-stu-id="b6976-181">Endpoints can then reference the `bindingConfiguration` by name.</span></span>

 <span data-ttu-id="b6976-185">`behaviorConfiguration`, Aşağıdaki örnekte gösterildiği gibi aynı şekilde uygulanır.</span><span class="sxs-lookup"><span data-stu-id="b6976-185">A `behaviorConfiguration` is implemented the same way, as illustrated in the following sample.</span></span>
