# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Win32 bildirimini katıştırma hatası:/win32manifest seçeneği/nowin32manifest ile çakışıyor
Visual Basic Derleyicisi hem bir `/win32manifest` derleyici seçeneğini hem de bir `/nowin32manifest` derleyici seçeneğini geçti. Bu seçeneklerden yalnızca biri Visual Basic derleyicisine geçirilebilir.  
  
 **Hata kimliği:** BC2033  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `/win32manifest`Derleyici seçeneğini veya `/nowin32manifest` derleyici seçeneğini kaldırın.
