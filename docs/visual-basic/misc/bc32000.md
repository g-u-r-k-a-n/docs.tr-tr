# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>' ' Yerel değişkenine \<variablename> bildirilmeden önce başvuru yapılamaz

Yordamın içindeki bir değişken, bildiriminden önce kullanılır.  
  
 **Hata kimliği:** BC32000  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimi değişkene ilk başvurunun önüne taşı.
