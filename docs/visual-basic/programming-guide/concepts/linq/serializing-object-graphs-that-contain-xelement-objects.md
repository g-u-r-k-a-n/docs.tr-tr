# <a name="serializing-object-graphs-that-contain-xelement-objects-visual-basic"></a>XElement nesneleri içeren nesne grafiklerini serileştirme (Visual Basic)
Bu konu, türündeki nesnelere yönelik başvuruları içeren nesne grafiklerini serileştirme yeteneğini tanıtır <xref:System.Xml.Linq.XElement> . Bu tür serileştirmek sağlamak için, <xref:System.Xml.Linq.XElement> arabirimini uygular <xref:System.Xml.Serialization.IXmlSerializable> .  
  
 Yalnızca <xref:System.Xml.Linq.XElement> sınıfın serileştirme uyguladığı unutulmamalıdır.
