# <a name="compiler-error-cs0174"></a>Derleyici hatası CS0174
' Base ' başvurusu için temel sınıf gerekir  
  
 Bu hata yalnızca .NET çalışma zamanı, `System.Object` temel sınıfı olmayan tek sınıf olan sınıf için kaynak kodunu derlediğinde oluşur.
