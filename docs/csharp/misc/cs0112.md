# <a name="compiler-error-cs0112"></a><span data-ttu-id="759f8-103">Derleyici hatası CS0112</span><span class="sxs-lookup"><span data-stu-id="759f8-103">Compiler Error CS0112</span></span>

<span data-ttu-id="759f8-104">' Function ' statik üyesi geçersiz kılma, sanal veya soyut olarak işaretlenemez</span><span class="sxs-lookup"><span data-stu-id="759f8-104">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="759f8-105">`override`, **Virtual**veya **abstract** anahtar sözcüğünü kullanan herhangi bir yöntem bildirimi de **static** anahtar sözcüğünü kullanamaz.</span><span class="sxs-lookup"><span data-stu-id="759f8-105">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="759f8-106">Daha fazla bilgi için bkz. [Yöntemler](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="759f8-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="759f8-107">Aşağıdaki örnek CS0112 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="759f8-107">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
