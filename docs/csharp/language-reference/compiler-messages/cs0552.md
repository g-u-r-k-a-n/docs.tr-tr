# <a name="compiler-error-cs0552"></a>Derleyici Hatası CS0552
' dönüştürme yordamı ': arabirime/arabirimden Kullanıcı tanımlı dönüşüm  
  
 Bir arabirimden veya arabiriminden Kullanıcı tanımlı bir dönüşüm oluşturamazsınız. Dönüştürme yordamına ihtiyacınız varsa, arabirimi bir sınıf haline getirerek veya arabirimden bir sınıf türeleyerek bu hatayı çözün.  
  
 Aşağıdaki örnek CS0552 oluşturur:  
  
```csharp  
// CS0552.cs  
public interface ii  
{  
}  
  
public class a  
{  
   // delete the routine to resolve CS0552  
   public static implicit operator ii(a aa) // CS0552  
   {  
      return new ii();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
