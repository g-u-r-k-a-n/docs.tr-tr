# <a name="compiler-warning-level-1-cs3005"></a>Derleyici Uyarısı (düzey 1) CS3005

Yalnızca büyük/küçük harf bakımından farklı ' Identifier ' tanımlayıcısı CLS uyumlu değil  
  
 [public](../language-reference/keywords/public.md) [protected](../language-reference/keywords/protected.md) `protected internal` `public` `protected` `protected internal` Yalnızca bir veya daha fazla harf durumunda olan bir veya daha fazla harften farklı olan ortak, KORUMALı veya tanımlayıcı, ortak dil belirtimi (CLS) ile uyumlu değildir. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [Dil bağımsızlığı ve dilden bağımsız bileşenler](../../standard/language-independence.md).
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS3003 oluşturur:  
  
```csharp  
// CS3005.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int a1 = 0;  
    public static int A1 = 1;   // CS3005  
  
    public static void Main()  
    {  
        Console.WriteLine(a1);  
        Console.WriteLine(A1);  
    }  
}  
```
