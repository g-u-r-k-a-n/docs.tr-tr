 <span data-ttu-id="9993b-104">Sorgu ifadesinde Aralık değişkeni sorgu ifadesi sonlanana kadar kapsamdadır.</span><span class="sxs-lookup"><span data-stu-id="9993b-104">The range variable in a query expression is in scope until the query expression terminates.</span></span> <span data-ttu-id="9993b-105">Bu nedenle benzersiz bir tanımlayıcıya sahip olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="9993b-105">It must therefore have a unique identifier.</span></span>

## <a name="example"></a><span data-ttu-id="9993b-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="9993b-108">Example</span></span>  
 <span data-ttu-id="9993b-109">Aşağıdaki örnek, tanımlayıcı `num` `from` yan tümcesindeki Aralık değişkeni için ve `let` yan tümcesiyle tanıtılan Aralık değişkeni için kullanıldığından, CS1930 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="9993b-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```
