# <a name="file-already-open"></a>Dosya zaten açık

Bazen bir dosyanın başka bir `FileOpen` işlemin gerçekleşebilmesi için kapatılmış olması gerekir. Bu hatanın olası nedenleri arasında:  
  
- `FileOpen`Zaten açık olan bir dosya için sıralı çıkış modu işlemi yürütüldü  
  
- Bir ifade açık bir dosyaya başvurur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İfadeyi yürütmeden önce dosyayı kapatın.
