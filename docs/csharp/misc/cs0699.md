# <a name="compiler-error-cs0699"></a>Derleyici hatası CS0699
' Generic ', ' Identifier ' tür parametresini tanımlamıyor  
  
 Tür parametresi, bu genel için tür parametrelerinin bildirim listesinde bulunmayan bir genel tanımda kullanıldı. Tür parametresi için kullanılan ad tutarsızsa bu durum oluşabilir.  
  
 Aşağıdaki örnek CS0699 oluşturur:  
  
```csharp  
// CS0699.cs  
class C<T> where U : I   // CS0699 – U is not a valid type parameter  
{  
}  
```
