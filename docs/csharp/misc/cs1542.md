# <a name="compiler-error-cs1542"></a>Derleyici hatası CS1542
' dll ' zaten bir derleme olduğundan bu derlemeye eklenemez; Bunun yerine '/R ' seçeneğini kullanın  
  
 [/Addmodule](../language-reference/compiler-options/addmodule-compiler-option.md) derleyici seçeneği ile başvurulan dosya [/target: Module](../language-reference/compiler-options/target-module-compiler-option.md)ile oluşturulmamış. Bu derlemedeki dosyaya başvurmak için [başvuru](../language-reference/compiler-options/reference-compiler-option.md) kullanın.
