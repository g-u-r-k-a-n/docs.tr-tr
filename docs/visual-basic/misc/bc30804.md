# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>' VARIANT ' artık desteklenen bir tür değil; Bunun yerine ' Object ' türünü kullanın

`Object`Veri türü yerini alır `Variant` .  
  
 **Hata kimliği:** BC30804  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öğeyi olarak bildirin `Object` .
