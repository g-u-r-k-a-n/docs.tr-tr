# <a name="expression-is-not-a-method"></a>İfade bir metot değil

Yöntem çağrın hedefi bir yöntem değil.  
  
 **Hata kimliği:** BC30454  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yöntem adını yanlış yazdığınızdan emin olun.
