# <a name="iclrstrongname2-interface"></a>ICLRStrongName2 Arabirimi

, SHA-2 güvenli karma algoritmaları grubunu (SHA-256, SHA-384 ve SHA-512) kullanarak tanımlayıcı adlar oluşturma olanağı sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[StrongNameGetPublicKeyEx Yöntemi](strongnamegetpublickeyex-method.md)|Ortak/özel anahtar çiftinden ortak anahtarı alır ve bir karma algoritmasını ve bir imza algoritmasını belirtir.|  
|[StrongNameSignatureVerificationEx2 Yöntemi](strongnamesignatureverificationex2-method.md)|Kesin adlandırılmış bir derlemenin imzasını doğrular ve ECMA anahtarından gerçek bir anahtara bir eşleme sağlar.|
