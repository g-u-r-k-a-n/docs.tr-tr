# <a name="metadata"></a>Meta veri

Windows Communication Foundation (WCF), hizmet meta verilerini dışarı aktarmak, yayımlamak, almak ve içeri aktarmak için bir altyapı sağlar. WCF Hizmetleri, Svcutil.exe gibi araçların hizmete erişmek için otomatik olarak istemci kodu oluşturabileceği şekilde hizmetin uç noktalarıyla nasıl etkileşim kuracağınızı anlatan meta verileri kullanır.
