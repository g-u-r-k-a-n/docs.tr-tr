# <a name="39460---trackingvaluenotserializable"></a>39460 - TrackingValueNotSerializable

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|39460|  
|Anahtar sözcükler|WFTracking|  
|Düzey|Uyarı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Açıklama  

 Bir izleme kaydındaki ayıklanan bağımsız değişkenin/değişkenin seri hale getirilmediğini belirtir.  
  
## <a name="message"></a>İleti  

 Ayıklanan bağımsız değişken/değişken ' %1 ' seri hale getirilebilir değil.
