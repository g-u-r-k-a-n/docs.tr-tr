# <a name="integrating-user-help-in-windows-forms"></a>Windows Forms'ta Kullanıcı Yardımını Tümleştirme
Önemli olan, ancak genellikle daha fazla bakmış olan, Windows tabanlı uygulamalar oluşturmanın en çok olduğu, kullanıcıların karışıklık süreleriyle ilgili olarak yardım ettiği yerdir. Windows Forms, her biri [HelpProvider bileşeni](../controls/helpprovider-component-windows-forms.md)tarafından sağlanan Iki farklı yardım türünü destekler. Birincisi, kullanıcının HTML veya HTML Yardımı 1 ' in bir yardım dosyasına işaret eder. *x* veya daha büyük biçim. İkincisi, tek denetimler üzerinde kısa "Bu nedir"-Type yardımını gösterebilir; Bu, özellikle iletişim kutularında yararlı olur. Her iki tür yardım da aynı formda kullanılabilir.  
  
 Ayrıca, [araç Ipucu bileşeni](../controls/tooltip-component-windows-forms.md) Windows Forms denetimleri Için bireysel yardım sağlamak üzere kullanılabilir.
