# <a name="else-c-reference"></a>#else (C# Başvurusu)
`#else`bileşik koşullu yönerge oluşturmanıza olanak sağlar, böylece, yukarıdaki [#if](./preprocessor-if.md) veya (isteğe bağlı) `true` [#elif](./preprocessor-elif.md) yönergelerinde ifadelerhiçbiri değerlendirmek, derleyici ve sonraki arasındaki `#else` `#endif`tüm kodu değerlendirecektir .  
  
## <a name="remarks"></a>Açıklamalar  
 [#endif](./preprocessor-endif.md) sonra `#else`bir sonraki önişlemci yönergesi olmalıdır. Nasıl [#if](./preprocessor-if.md) kullanılacağına bir örnek `#else`için #if bakın.
