<span data-ttu-id="adce4-104">' Function ' bir Oluşturucu, yıkıcı, işleç veya açık arabirim uygulama olduğundan koşullu öznitelik geçerli değil</span><span class="sxs-lookup"><span data-stu-id="adce4-104">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>

```csharp  
// CS0577.cs  
// compile with: /target:library

{  
   void m();  
}

}  
```
