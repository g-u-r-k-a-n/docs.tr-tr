## <a name="cause"></a><span data-ttu-id="eff4c-111">Nedeni</span><span class="sxs-lookup"><span data-stu-id="eff4c-111">Cause</span></span>

<span data-ttu-id="eff4c-112">[P/Invoke](../../../standard/native-interop/pinvoke.md) dize parametresi değeri ile geçirilir ve ile işaretlenir <xref:System.Runtime.InteropServices.OutAttribute> .</span><span class="sxs-lookup"><span data-stu-id="eff4c-112">A [P/Invoke](../../../standard/native-interop/pinvoke.md) string parameter is passed by value and marked with <xref:System.Runtime.InteropServices.OutAttribute>.</span></span>

## <a name="rule-description"></a><span data-ttu-id="eff4c-113">Kural açıklaması</span><span class="sxs-lookup"><span data-stu-id="eff4c-113">Rule description</span></span>

<span data-ttu-id="eff4c-114">.NET çalışma zamanı otomatik olarak [dize](/dotnet/api/system.string.intern#remarks)oluşturma işlemi gerçekleştirir.</span><span class="sxs-lookup"><span data-stu-id="eff4c-114">The .NET runtime automatically performs [string interning](/dotnet/api/system.string.intern#remarks).</span></span> <span data-ttu-id="eff4c-115">İle işaretlenmiş bir dize <xref:System.Runtime.InteropServices.OutAttribute> bir P/Invoke değerine göre geçirildiyse, çalışma zamanı kararsız hale gelebilirler.</span><span class="sxs-lookup"><span data-stu-id="eff4c-115">If an interned string marked with <xref:System.Runtime.InteropServices.OutAttribute> is passed by value to a P/Invoke, the runtime can be destabilized.</span></span>

## <a name="how-to-fix-violations"></a><span data-ttu-id="eff4c-116">İhlalleri çözme</span><span class="sxs-lookup"><span data-stu-id="eff4c-116">How to fix violations</span></span>

<span data-ttu-id="eff4c-117">Değiştirilen dize verilerinin çağırana geri sıralaması gerekliyse, bunun yerine dizeyi başvuruya göre geçirin.</span><span class="sxs-lookup"><span data-stu-id="eff4c-117">If marshaling of modified string data back to the caller is required, pass the string by reference instead.</span></span> <span data-ttu-id="eff4c-118">Aksi takdirde, <xref:System.Runtime.InteropServices.OutAttribute> diğer herhangi bir değişiklik yapılmadan kaldırılabilir.</span><span class="sxs-lookup"><span data-stu-id="eff4c-118">Otherwise, the <xref:System.Runtime.InteropServices.OutAttribute> can be removed without any other changes.</span></span>

```csharp
 // Violation
[DllImport("MyLibrary")]
private static extern void Foo([Out] string s);

// Fixed: passed by reference
[DllImport("MyLibrary")]
private static extern void Foo(out string s);

// Fixed: marshaling data back to caller is not required
[DllImport("MyLibrary")]
private static extern void Foo(string s);
```
