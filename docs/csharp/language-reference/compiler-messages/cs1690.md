# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="8730d-102">Derleyici Uyarısı (düzey 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="8730d-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="8730d-103">Bir üyeye 'üye' olarak erişilmesi, mareşal-by-reference sınıfının bir alanı olduğundan çalışma zamanı özel bir özel durum neden olabilir</span><span class="sxs-lookup"><span data-stu-id="8730d-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="8730d-104">Bu uyarı, bir sınıfın <xref:System.MarshalByRefObject>bir üyesiüzerinde bir yöntem, özellik veya dizinleyici çağırmaya çalıştığınızda oluşur ve üye bir değer türüdür.</span><span class="sxs-lookup"><span data-stu-id="8730d-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="8730d-105">Devralınan `MarshalByRefObject` nesneler genellikle bir uygulama etki alanında başvuru tarafından marshaled olması amaçlanır.</span><span class="sxs-lookup"><span data-stu-id="8730d-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="8730d-106">Herhangi bir kod, bir uygulama etki alanında böyle bir nesnenin değer türü <xref:System.InvalidOperationException> üyesine doğrudan erişmeye çalışırsa, çalışma zamanı oluşur.</span><span class="sxs-lookup"><span data-stu-id="8730d-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="8730d-107">Uyarıyı çözmek için önce üyeyi yerel bir değişkene kopyalayın ve bu değişkendeki yöntemi arayın.</span><span class="sxs-lookup"><span data-stu-id="8730d-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="8730d-108">Aşağıdaki örnek CS1690 üretir:</span><span class="sxs-lookup"><span data-stu-id="8730d-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {
        AppDomain domain = AppDomain.CreateDomain("MyDomain");
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;
    }
}
```
