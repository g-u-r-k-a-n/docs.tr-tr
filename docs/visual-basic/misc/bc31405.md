# <a name="optional-parameters-cannot-have-structure-types"></a>İsteğe bağlı parametreler yapı türlerine sahip olamaz

İsteğe bağlı parametreler yapı türleri olamaz.  
  
 **Hata kimliği:** BC31405  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Yapı türü parametrelerini gerektiği şekilde bildirin.  
  
2. Bir yapının yerini almak için bir statik sınıf üyesi bildirin.
