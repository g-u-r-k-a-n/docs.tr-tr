> [!NOTE]
> <span data-ttu-id="506b7-108">Bu örneğe ilişkin Kurulum yordamı ve derleme yönergeleri bu konunun sonunda bulunur.</span><span class="sxs-lookup"><span data-stu-id="506b7-108">The setup procedure and build instructions for this sample are located at the end of this topic.</span></span>  
  
 <span data-ttu-id="506b7-109">Veri bağlama bir veri kaynağının bir kullanıcı arabirimine otomatik olarak bağlanmasını sağlar.</span><span class="sxs-lookup"><span data-stu-id="506b7-109">Data binding allows a data source to be automatically bound to a UI.</span></span> <span data-ttu-id="506b7-110">Bu, programlama modelini basitleştiğinden, her UI öğesini bir veri nesnesinden veya bir dizi veri nesnesinden verilerle programlı bir şekilde güncelleştirmenizi gerektirmez.</span><span class="sxs-lookup"><span data-stu-id="506b7-110">This simplifies the programming model because it does not require that you programmatically update each UI element with the data from a data object or an array of data objects.</span></span> <span data-ttu-id="506b7-111">Bir nesneyi tek bir UI öğesine veya bir diziye, `ListBox`gibi birden çok girişi alan bir denetime bağlayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="506b7-111">You can bind an object to a single UI element or an array to a control that takes multiple inputs, such as a `ListBox`.</span></span> <span data-ttu-id="506b7-112">Aşağıdaki kod, bir kullanıcı arabirimi öğesinin `DataContext` nasıl veri bağlanacağını gösterir.</span><span class="sxs-lookup"><span data-stu-id="506b7-112">The following code shows how to bind data to the `DataContext` of a UI element.</span></span>  
  
```csharp  
// Event handler executed when call is complete  
void client_GetAlbumListCompleted(object sender, GetAlbumListCompletedEventArgs e)  
{  
    // This is on the UI thread, myPanel can be accessed directly  
    myPanel.DataContext = e.Result;   
}  
```  
  
 <span data-ttu-id="506b7-113">Önceki örnekte, `myPanel` adlı `grid` düzen öğesi için `DataContext` `GetAlbumList` metodu tarafından döndürülen verilere ayarlanır.</span><span class="sxs-lookup"><span data-stu-id="506b7-113">In the previous sample, the `DataContext` for the `grid` layout element named `myPanel` is set to the data returned by the `GetAlbumList` method.</span></span> <span data-ttu-id="506b7-114">`DataContext`, öğelerin bağlama için kullanılan veri kaynağı ve yol gibi bağlamanın diğer özellikleri hakkında üst öğelerinden bilgi devralmasını sağlar.</span><span class="sxs-lookup"><span data-stu-id="506b7-114">The `DataContext` allows elements to inherit information from their parent elements about the data source that is used for binding, as well as other characteristics of the binding such as the path.</span></span> <span data-ttu-id="506b7-115">Sunucudaki verilerin her güncelleştirildiği her seferinde kod satırının yürütülmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="506b7-115">The line of code must be executed every time the data on the server is updated.</span></span> <span data-ttu-id="506b7-116">Örneğin, pencere başlatıldığında ve yeni bir albüm eklendiğinde yürütülür.</span><span class="sxs-lookup"><span data-stu-id="506b7-116">For example, it is executed when the window is initialized and when a new album is added.</span></span>  
  
 <span data-ttu-id="506b7-117">Aşağıdaki örnek XAML kodunda, `ListBox` `ItemsSource="{Binding }"`belirtir.</span><span class="sxs-lookup"><span data-stu-id="506b7-117">In the following sample XAML code, the `ListBox` specifies `ItemsSource="{Binding }"`.</span></span>  
  
```xml  
<ListBox   
          ItemTemplate="{StaticResource AlbumStyle}"  
          ItemsSource="{Binding }"   
          IsSynchronizedWithCurrentItem="true" />  
```  
  
 <span data-ttu-id="506b7-118">Bu, üst düzey UI öğesine bağlanan verilerin de bu denetime (yani Albümler dizisi) bağlandığını belirtir.</span><span class="sxs-lookup"><span data-stu-id="506b7-118">This specifies that the data bound to the top-level UI element is also bound to this control (that is, the array of Albums).</span></span> <span data-ttu-id="506b7-119">Ayrıca, `ItemTemplate="{StaticResource AlbumStyle}"` `ListBox`her öğe için kullanılacak veri şablonunu belirtir.</span><span class="sxs-lookup"><span data-stu-id="506b7-119">In addition, `ItemTemplate="{StaticResource AlbumStyle}"` specifies the data template to be used for each item in the `ListBox`.</span></span> <span data-ttu-id="506b7-120">Verilerin nasıl biçimlendirilmesi gerektiğini belirtmek için veri şablonları da tanımlayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="506b7-120">You can also define data templates to specify how the data should be formatted.</span></span> <span data-ttu-id="506b7-121">Bu veri şablonları, uygulamadaki diğer kullanıcı arabirimi öğeleri için yeniden kullanılabilir. avantajı, veri şablonunun tek bir yerde tanımlanması ve saklanması olabilir.</span><span class="sxs-lookup"><span data-stu-id="506b7-121">These data templates can be reused for other UI elements in the application, the advantage is that the data template is defined and maintained in one place.</span></span>  
  
 <span data-ttu-id="506b7-122">`AlbumStyle` veri şablonu iki `TextBlock`yan yana bir kılavuz yerleştirir.</span><span class="sxs-lookup"><span data-stu-id="506b7-122">The `AlbumStyle` data template lays out a grid with two `TextBlock`s side by side.</span></span> <span data-ttu-id="506b7-123">Bir tane, albümün adını ve albümdeki diğer parça sayısını belirtir.</span><span class="sxs-lookup"><span data-stu-id="506b7-123">One specifies the name of the Album and the other the number of Tracks in the album.</span></span>  
  
```xaml  
<DataTemplate x:Key="AlbumStyle">  
    <Grid>  
        <Grid.ColumnDefinitions>  
            <ColumnDefinition Width="260" />  
            <ColumnDefinition Width="60" />  
        </Grid.ColumnDefinitions>  
        <TextBlock Grid.Column="0" TextContent="{Binding Path=Title}" />  
        <TextBlock Grid.Column="1" TextContent="{Binding Path=Tracks#.Count}" HorizontalAlignment="Right" />  
    </Grid>  
</DataTemplate>  
```  
  
 <span data-ttu-id="506b7-124">Aşağıdaki XAML kodu ikinci bir `ListBox`oluşturur.</span><span class="sxs-lookup"><span data-stu-id="506b7-124">The following XAML code creates a second `ListBox`.</span></span>  
  
```xaml  
<ListBox Grid.Row="2"   
            Grid.ColumnSpan="2"   
            ItemTemplate="{StaticResource TrackStyle}"  
            ItemsSource="{Binding Path=Tracks}" />  
```  
  
 <span data-ttu-id="506b7-125">Kod `ItemsSource`için bir yol belirtir.</span><span class="sxs-lookup"><span data-stu-id="506b7-125">The code specifies a path for the `ItemsSource`.</span></span> <span data-ttu-id="506b7-126">Bu, bu denetime bağlanan verilerin en üst düzey veriler olmadığını, `Tracks`adlı en üst düzey verilerin bir özelliğini olduğunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="506b7-126">This indicates that the data bound to this control is not the top-level data but a property of the top-level data named `Tracks`.</span></span> <span data-ttu-id="506b7-127">Bu özellik, albümün içindeki izlemelerin dizisini temsil eder.</span><span class="sxs-lookup"><span data-stu-id="506b7-127">This property represents the array of tracks contained within the album.</span></span> <span data-ttu-id="506b7-128">Ayrıca, `TrackStyle` adlı farklı bir `DataTemplate` belirtilir.</span><span class="sxs-lookup"><span data-stu-id="506b7-128">In addition, a different `DataTemplate` named `TrackStyle` is specified.</span></span> <span data-ttu-id="506b7-129">`TrackStyle` şablonun düzeni `AlbumStyle` şablonuyla benzerdir, ancak `TextBlock`s farklı özelliklere bağlanır.</span><span class="sxs-lookup"><span data-stu-id="506b7-129">The layout of the `TrackStyle` template is similar to that of the `AlbumStyle` template, but the `TextBlock`s are bound to different properties.</span></span> <span data-ttu-id="506b7-130">Bunun nedeni, iki şablonun farklı veri nesneleriyle kullanıllarıdır.</span><span class="sxs-lookup"><span data-stu-id="506b7-130">This is because the two templates are used with different data objects.</span></span>
