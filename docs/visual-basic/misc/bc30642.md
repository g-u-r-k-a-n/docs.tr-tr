# <a name="optional-and-paramarray-cannot-be-combined"></a>' Optional ' ve ' ParamArray ' birleştirilemez
`Optional`Ve `ParamArray` değiştiricileri birbirini dışlıyor.  
  
 **Hata kimliği:** BC30642  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ya da `ParamArray` `Optional` ' ı belirtin, ancak ikisini birden belirtmeyin.
