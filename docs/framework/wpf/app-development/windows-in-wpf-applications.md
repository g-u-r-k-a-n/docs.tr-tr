# <a name="windows-in-wpf-applications"></a>WPF Uygulamalarında Pencereler
Kullanıcılar, Windows ile uygulamalarla etkileşime geçer. Pencerenin temel amacı, içeriği barındırmak ve görüntülemektir. Pencerenin barındırdığı içerik türü, bir uygulamanın üzerinde çalıştığı verilerin türüne bağlıdır. Bu, medya, Extensible Application Markup Language (XAML) sayfaları, Web sayfaları, belgeler, veritabanı tabloları ve kayıtları ve sistem bilgileri içerebilir.
