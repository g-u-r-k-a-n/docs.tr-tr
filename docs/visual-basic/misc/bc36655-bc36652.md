<span data-ttu-id="0f94d-105">Tür çıkarımı, genel genişletme yöntemine yapılan çağrıda tür parametresinin (veya parametrelerinin) türünü (veya türlerini) belirlemekte kullanılacak bir girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="0f94d-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="0f94d-106">Derleyici bir veya daha fazla tür parametresi için birden fazla olası veri türü bulur ve bu hatayı raporlar.</span><span class="sxs-lookup"><span data-stu-id="0f94d-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> <span data-ttu-id="0f94d-107">Bağımsız değişkenlerin belirtilmesi bir seçenek değil (örneğin, sorgu ifadelerinde sorgu işleçleri için), ikinci tümce olmadan hata iletisi görüntülenir.</span><span class="sxs-lookup"><span data-stu-id="0f94d-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

```vb

    Sub Main()

        Dim caller As New Class1
        '' Not valid.
        'caller.targetExtension(1, "2")

    End Sub

    Class Class1
    End Class
```
