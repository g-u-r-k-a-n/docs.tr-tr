# <a name="option-infer-can-be-followed-only-by-on-or-off"></a>' Option Infer ' arkasından yalnızca ' on ' veya ' off ' gelebilir
`Option Infer`deyimler ya da belirtmelidir `On` `Off` . Diğer değerler geçerli değil.  
  
 **Hata kimliği:** BC30979  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ya da `Option Infer On` belirtin `Option Infer Off` .
