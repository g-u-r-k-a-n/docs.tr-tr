# <a name="paramarray-parameter-must-be-an-array"></a>ParamArray parametresi bir dizi olmalıdır

`ParamArray`Bağımsız değişken, tek boyutlu dizi türü olarak bildirilmelidir. Bildirim, bağımsız değişken adından sonra parantez içermez `ParamArray` .  
  
 **Hata kimliği:** BC30050  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken adından sonra bir çift ayraç ekleyin `ParamArray` .
