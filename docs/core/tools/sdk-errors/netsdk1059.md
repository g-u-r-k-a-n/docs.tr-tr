# <a name="netsdk1059-project-contains-obsolete-net-cli-tool"></a>NETSDK1059: proje artık kullanılmıyor .NET CLı aracı içeriyor

**Bu makale şu şekilde geçerlidir:** ✔️ .NET Core 2.1.100 SDK ve sonraki sürümleri

.NET SDK 'da uyarı NETSDK1059 olduğunda, projeniz eski bir .NET CLı aracı içerir. .NET Core 2,1 itibariyle, bu araçlar .NET SDK 'ya dahil edilmiştir ve projenin açıkça başvurulması gerekmez. .NET Core 2,1 ' ye geçiş hakkında daha fazla bilgi [burada](https://aka.ms/dotnetclitools-in-box)bulunabilir.
