# <a name="xml-name-expected"></a>XML adı bekleniyor
Bir XML sabit değeri veya XML ekseni özelliği bir ad olmadan belirtildi.  
  
 **Hata kimliği:** BC31146  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML değişmez değeri veya XML eksen özelliği için bir ad ekleyin.
