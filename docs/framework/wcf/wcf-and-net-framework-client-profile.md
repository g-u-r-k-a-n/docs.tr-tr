# <a name="wcf-and-net-framework-client-profile"></a>WCF ve .NET Framework İstemci Profili

.NET Framework Istemci profili, tüm Framework 'ü gerektirmeyen istemciler için tasarlanan tam .NET Framework basit bir sürümüdür. Windows Communication Foundation hepsi istemci çerçevesi tarafından desteklenmez.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>.NET Framework Istemci profili tarafından desteklenen WCF özellikleri  

 Aşağıdaki Windows Communication Foundation Özellikler .NET Framework Istemci profili tarafından desteklenir:  
  
- Tüm WCF, CardSpace ve Web barındırma için desteklenir.  
  
- TCP/IP kanallarının uzaktan iletişimi desteklenir.  
  
- Asmx (Web Hizmetleri) desteklenmez.
