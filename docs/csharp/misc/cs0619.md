# <a name="compiler-error-cs0619"></a><span data-ttu-id="51309-103">Derleyici hatası CS0619</span><span class="sxs-lookup"><span data-stu-id="51309-103">Compiler Error CS0619</span></span>

<span data-ttu-id="51309-104">' Member ' artık kullanılmıyor: ' text '</span><span class="sxs-lookup"><span data-stu-id="51309-104">'member' is obsolete: 'text'</span></span>

<span data-ttu-id="51309-105">Sınıf üyesi, sınıf üyesine başvuruluyorsa bir hata verilecek şekilde, [eski](../language-reference/attributes/general.md#obsolete-attribute) özniteliğiyle işaretlendi.</span><span class="sxs-lookup"><span data-stu-id="51309-105">A class member was marked with the [Obsolete](../language-reference/attributes/general.md#obsolete-attribute) attribute, such that an error will be issued when the class member is referenced.</span></span>

## <a name="example"></a><span data-ttu-id="51309-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="51309-106">Example</span></span>

<span data-ttu-id="51309-107">Aşağıdaki örnek CS0619 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="51309-107">The following sample generates CS0619:</span></span>  
  
```csharp
using System;

public class C
{
    [Obsolete("Use NewMethod instead", true)] // generates an error on use
    public static void OldMethod()
    {
    }

    // this is the method you should be using
    public static void NewMethod()
    {
    }  
}

class MyClass
{
   public static void Main()
   {
      C.OldMethod();   // CS0619
   }
}
```
