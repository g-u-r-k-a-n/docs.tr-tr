## <a name="example"></a><span data-ttu-id="1a100-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="1a100-108">Example</span></span>  
 <span data-ttu-id="1a100-109">İkinci dosya, önceki örnekte oluşturulan sınıftan `B` türetilmiş bir sınıf `C` tanımlayan bir DLL oluşturur.</span><span class="sxs-lookup"><span data-stu-id="1a100-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>
