# <a name="compiler-error-cs0016"></a>Derleyici Hatası CS0016

Çıktı dosyasına yazamadı 'dosya' — 'neden'

 Derleyici bir çıktı dosyasına yazamadı. Var olduğundan emin olmak için dosyaya giden yolu denetleyin. Daha önce oluşturulmuş bir dosya zaten konumdaysa, dosyanın yazılabilir olduğundan ve şu anda dosyada kilit bulunmadığından emin olun. Örneğin, oluşturmaya çalıştığınızda çalıştırılabileninizin çalışmadığından emin olun.
