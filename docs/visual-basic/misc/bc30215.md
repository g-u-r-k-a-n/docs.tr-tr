# <a name="sub-or-function-expected"></a>' Sub ' veya ' function ' bekleniyor
Bir `Declare` ifade ya da belirtmeyin `Sub` `Function` . `Declare`bir dış yordam tanımlar ve hangi tür yordamın gerektiğini belirtmelidir.  
  
 **Hata kimliği:** BC30215  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Sub` `Function` İfadesine or anahtar sözcüğünü ekleyin `Declare` .
