# <a name="compiler-error-cs1536"></a>Derleyici hatası CS1536
Geçersiz parametre türü void  
  
 Void işaretçisi dışında [void](../language-reference/builtin-types/void.md) bir parametre belirtmek için gerekli veya geçerli değildir.  
  
 Aşağıdaki örnek CS1536 oluşturur:  
  
```csharp  
// CS1536.cs  
class a  
{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
