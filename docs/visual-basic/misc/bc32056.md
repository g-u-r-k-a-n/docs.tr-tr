# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="2bdb7-102">' Implements ' yan tümcesinde tür parametresine izin verilmiyor</span><span class="sxs-lookup"><span data-stu-id="2bdb7-102">Type parameter not allowed in 'Implements' clause</span></span>

<span data-ttu-id="2bdb7-103">`Implements`Genel türde bir yan tümce, uygulanacak üye olarak bir tür parametresi belirtir.</span><span class="sxs-lookup"><span data-stu-id="2bdb7-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="2bdb7-104">Bir `Implements` yan tümce bir arabirim ve üye belirtmelidir.</span><span class="sxs-lookup"><span data-stu-id="2bdb7-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="2bdb7-105">Arabirime bir tür parametresi geçirebilir, ancak üyeyi üyeye geçiremez ve üyenin adı olarak kullanamaz.</span><span class="sxs-lookup"><span data-stu-id="2bdb7-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="2bdb7-106">Aşağıdaki deyimler bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="2bdb7-106">The following statements can generate this error.</span></span>  
  
```vb  
Class c1(Of t)  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="2bdb7-107">**Hata kimliği:** BC32056</span><span class="sxs-lookup"><span data-stu-id="2bdb7-107">**Error ID:** BC32056</span></span>
