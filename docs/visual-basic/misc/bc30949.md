# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="590d4-103">Sabit olmayan bir boyut için dizi başlatıcısı belirtilemiyor; ' ' boş başlatıcısını kullan {}</span><span class="sxs-lookup"><span data-stu-id="590d4-103">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>

<span data-ttu-id="590d4-104">Dizi, derleme zamanında bilinmeyen bir boyutu başlatır.</span><span class="sxs-lookup"><span data-stu-id="590d4-104">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="590d4-105">Aşağıdaki kod bu hatayı oluşturur.</span><span class="sxs-lookup"><span data-stu-id="590d4-105">The following code generates this error.</span></span>  
  
```vb  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="590d4-106">Aşağıdaki kod hatayı önler.</span><span class="sxs-lookup"><span data-stu-id="590d4-106">The following code avoids the error.</span></span>  
  
```vb  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="590d4-107">**Hata kimliği:** BC30949</span><span class="sxs-lookup"><span data-stu-id="590d4-107">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="590d4-108">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="590d4-108">To correct this error</span></span>  
  
- <span data-ttu-id="590d4-109">Mümkünse, dizi bildiriminde sabit bir boyut belirtin.</span><span class="sxs-lookup"><span data-stu-id="590d4-109">If possible, specify a constant dimension in the array declaration.</span></span>  
  
- <span data-ttu-id="590d4-110">Sabit boyutlu bir boyut belirtmezseniz, sabit olmayan boyut bilindiğinde diziyi bir döngü kullanarak başlatmalısınız.</span><span class="sxs-lookup"><span data-stu-id="590d4-110">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
