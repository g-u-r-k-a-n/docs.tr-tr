# <a name="repair-the-net-framework"></a>.NET Framework’ü Onarma

Bazı durumlarda .NET Framework yüklemeniz hasar görebilir ve onarım gerektirebilir. Bunu başlattıktan sonra uygulamanız doğru kilitlenirse veya daha yeni .NET Framework sürümleri yükleyemezseniz bu durum olabilir.

[.NET Framework onarım aracını](https://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe)kullanarak .NET Framework yüklemenizi onarabilirsiniz.

Uygulamanız .NET Framework onarılırken hala başlatılamıyorsa, uygulama bir sorun olabilir. Bu durumda, uygulama yayımcısına başvurmanız gerekir.
