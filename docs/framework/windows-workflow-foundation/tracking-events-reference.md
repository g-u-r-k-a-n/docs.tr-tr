# <a name="tracking-events-reference"></a>İzleme Olayları Başvurusu

Yürütme sırasında, içinde bir iş akışı [!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] , ömrü boyunca çeşitli aşamalar arasında gezindiğinde olayları izleme işlemleri gerçekleştirir. Ana bilgisayar bu olaylara abone olabilir ve ömrü boyunca iş akışının ilerleme durumu üzerinde güncel kalır. Oluşturulan izleme olayları bu bölümde ele alınmıştır.
