# <a name="compiler-error-cs0311"></a><span data-ttu-id="f84c7-103">Derleyici Hatası CS0311</span><span class="sxs-lookup"><span data-stu-id="f84c7-103">Compiler Error CS0311</span></span>

<span data-ttu-id="f84c7-104">' Type1 ' türü, ' ' genel türünde veya yönteminde ' 'T ' tür parametresi olarak kullanılamaz \<name> .</span><span class="sxs-lookup"><span data-stu-id="f84c7-104">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="f84c7-105">' Type1 ' öğesinden ' type2 ' öğesine örtük bir başvuru dönüştürmesi yok.</span><span class="sxs-lookup"><span data-stu-id="f84c7-105">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="f84c7-106">Bir kısıtlama genel tür parametresine uygulandığında, örtük bir kimlik veya başvuru dönüştürmesi somut bağımsız değişkenden kısıtlama türüne sahip olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="f84c7-106">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f84c7-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="f84c7-107">To correct this error</span></span>

2. <span data-ttu-id="f84c7-109">Sınıfa sahipseniz, kısıtlamayı kaldırabilir veya örtük bir başvuruyu veya kimlik dönüştürmeyi etkinleştirmek için bir şey yapabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f84c7-109">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="f84c7-110">Örneğin, ikinci türün birinciden devralmasını sağlayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f84c7-110">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f84c7-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="f84c7-111">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="f84c7-112">Değer türünde bir bağımsız değişkeni kullanmaya çalışırken bu hata oluşursa, örneğin öğesinden `short` `int` , bir genel tür parametresini yerine getirdiğine yönelik örtük bir sayısal dönüştürmenin.</span><span class="sxs-lookup"><span data-stu-id="f84c7-112">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
