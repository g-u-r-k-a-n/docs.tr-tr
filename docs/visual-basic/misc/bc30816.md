# <a name="membername-is-not-declared"></a>' \<membername> ' bildirilmemiş

' `<membername>` ' bildirilmemiş. `Debug` nesne işlevselliği `System.Diagnostics.Debug` `System` derlemede kullanılabilir.  
  
 Belirtilen üye adı bulunamadı.  
  
 **Hata kimliği:** BC30816  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Üyenin yazımını doğrulayın.  
  
2. Bir `Imports` ifade kullanın veya diğer ad alanlarında tanımlanan üyeleri tamamen niteleyin. Örneğin, `WriteLine` işlev `System.Diagnostics.Debug` ad alanında bildirilmiştir.
