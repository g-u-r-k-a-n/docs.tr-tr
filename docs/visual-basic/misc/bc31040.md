# <a name="membername-exists-in-multiple-base-interfaces"></a>' \<membername> ' birden çok temel arabirimde bulunuyor

' \<membername> ' birden çok temel arabirimde bulunuyor. \<membername>Türetilmiş arabirimin adı yerine ' Implements ' yan tümcesinde ' ' bildiren arabirimin adını kullanın.  
  
 Bu arabirim, birden çok arabirimden aynı ada sahip üyeleri devralır, belirsizlik oluşturur.  
  
 **Hata kimliği:** BC31040  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Implements`Türetilmiş arabirimin adı yerine yan tümcelerde tanımlama arabirimi adı ' nı kullanın.
