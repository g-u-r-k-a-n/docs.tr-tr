## <a name="example"></a><span data-ttu-id="229b2-119">Örnek</span><span class="sxs-lookup"><span data-stu-id="229b2-119">Example</span></span>

```csharp
// Code with violations
switch (o)
{
    case int _:
        Console.WriteLine("Value was an int");
        break;
    case string _:
        Console.WriteLine("Value was a string");
        break;
}

// Fixed code
switch (o)
{
    case int:
        Console.WriteLine("Value was an int");
        break;
    case string:
        Console.WriteLine("Value was a string");
        break;
}
```
