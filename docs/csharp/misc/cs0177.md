# <a name="compiler-error-cs0177"></a>Derleyici hatası CS0177

' Parameter ' out parametresi, denetimin geçerli yöntemi terk etmeden önce atanmalıdır  
  
 [Out](../language-reference/keywords/out-parameter-modifier.md) anahtar sözcüğüyle işaretlenmiş bir parametreye Yöntem gövdesinde bir değer atanmadı. Daha fazla bilgi için bkz. [parametreleri geçirme](../programming-guide/classes-and-structs/passing-parameters.md)  
  
 Aşağıdaki örnek CS0177 oluşturur:  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
