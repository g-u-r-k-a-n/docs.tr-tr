# <a name="compiler-error-cs1647"></a>Derleyici hatası CS1647

Bir ifade, ' Code ' yakınında derlemek için çok uzun veya karmaşık  
  
 Derleyicide kodunuzun işlendiği bir yığın taşması vardı. Bu hatayı çözmek için kodunuzu kolaylaştırın. Kodunuz geçerliyse, [Geliştirici topluluğu](https://developercommunity.visualstudio.com/spaces/61/index.html) sitesinde veya [StackOverflow](https://stackoverflow.com/)yardım alın.
