# <a name="isymunmanagedscope-interface"></a>ISymUnmanagedScope Arabirimi
Bir yöntem içindeki bir sözcük temelli kapsamı temsil eder.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetChildren Yöntemi](isymunmanagedscope-getchildren-method.md)|Bu kapsamın alt öğelerini alır.|  
|[GetEndOffset Yöntemi](isymunmanagedscope-getendoffset-method.md)|Bu kapsamın bitiş sapmasını alır.|  
|[GetLocalCount Yöntemi](isymunmanagedscope-getlocalcount-method.md)|Bu kapsam içinde tanımlanan yerel değişkenlerin sayısını alır.|  
|[GetLocals Yöntemi](isymunmanagedscope-getlocals-method.md)|Bu kapsam içinde tanımlanan yerel değişkenleri alır.|  
|[GetMethod Yöntemi](isymunmanagedscope-getmethod-method.md)|Bu kapsamı içeren yöntemi alır.|  
|[GetNamespaces Yöntemi](isymunmanagedscope-getnamespaces-method.md)|Bu kapsam içinde kullanılmakta olan ad alanlarını alır.|  
|[GetParent Yöntemi](isymunmanagedscope-getparent-method.md)|Bu kapsamın üst kapsamını alır.|  
|[GetStartOffset Yöntemi](isymunmanagedscope-getstartoffset-method.md)|Bu kapsamın başlangıç sapmasını alır.|  
  
## <a name="requirements"></a>Gereksinimler  
 **Üst bilgi:** CorSym. IDL, CorSym. h
