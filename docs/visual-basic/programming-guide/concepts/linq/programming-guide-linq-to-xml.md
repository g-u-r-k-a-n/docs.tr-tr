## <a name="who-should-read-this-documentation"></a>Bu belgeleri kimler okumalı  
 Bu belgede, Visual Basic zaten anlayan geliştiriciler ve .NET Framework bazı temel yönleri hedeflededir.  
  
 Bu belgenin amacı, [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] her türlü geliştirici için kullanımı kolay hale sağlamaktır. [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)], XML programlamayı kolaylaştırır. Bunu kullanmak için uzman bir geliştirici olmanız gerekmez.
