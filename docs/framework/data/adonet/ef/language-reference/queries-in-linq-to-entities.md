 <span data-ttu-id="00bc3-108">BIR LINQ sorgu işlemi üç eylemden oluşur: veri kaynağını veya kaynaklarını elde etmek, sorguyu oluşturmak ve sorguyu yürütmek.</span><span class="sxs-lookup"><span data-stu-id="00bc3-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>
