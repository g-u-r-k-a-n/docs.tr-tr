# <a name="next-expected"></a>' Next ' bekleniyor

Bir `On Error Resume` Yapı, sonuçlamadan oluşur `Next` .  
  
 **Hata kimliği:** BC30003  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Next`Deyimin sonuna anahtar sözcüğünü ekleyin `On Error` .
