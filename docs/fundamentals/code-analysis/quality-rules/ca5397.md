### <a name="enumeration-name-violation"></a><span data-ttu-id="a2506-133">Sabit listesi adı ihlali</span><span class="sxs-lookup"><span data-stu-id="a2506-133">Enumeration name violation</span></span>

```csharp
using System;
using System.Security.Authentication;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5397 violation for using Tls11
        SslProtocols protocols = SslProtocols.Tls11 | SslProtocols.Tls12;
    }
}
```
