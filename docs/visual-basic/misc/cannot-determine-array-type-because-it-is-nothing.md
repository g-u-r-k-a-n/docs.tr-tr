# <a name="cannot-determine-array-type-because-it-is-nothing"></a>Nothing olduğundan, dizi türü belirlenemiyor

Değer olarak ayarlandığı için, diziniz türü belirlenemiyor `Nothing` .  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dizi için dışında bir değer sağlayın `Nothing` .
