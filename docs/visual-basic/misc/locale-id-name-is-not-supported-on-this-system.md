# <a name="locale-id-name-is-not-supported-on-this-system"></a>'\<name > ' yerel ayar KIMLIĞI bu sistemde desteklenmiyor
`VbStrConv`kullanmaya çalışıyorsunuz, ancak sisteminiz belirtilen yerel ayarı desteklemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Yerel ayarı desteklemek için makineyi yükseltin veya uygulamayı belirtilen yerel ayarı destekleyen bir makinede çalıştırın.
