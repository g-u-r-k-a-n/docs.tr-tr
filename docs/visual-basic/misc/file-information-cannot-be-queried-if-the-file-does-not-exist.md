# <a name="file-information-cannot-be-queried-if-the-file-does-not-exist"></a>Dosya yoksa dosya bilgileri sorgulanamıyor
<xref:System.IO.FileInfo>Varolmayan bir dosya hakkında bilgi için bir nesneyi sorgulama girişiminde bulunuldu. Genellikle bunun nedeni dosya adı sağlanırken bir hata oluşur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Doğru dosya adını tedarik edin.
