# <a name="endregion-c-reference"></a>#endregion (C# Başvurusu)
`#endregion`[#region](./preprocessor-region.md) bloğunun sonunu işaretler. Örnek:  
  
```csharp
#region MyClass definition  
class MyClass
{  
    static void Main()
    {  
    }  
}  
#endregion  
```
