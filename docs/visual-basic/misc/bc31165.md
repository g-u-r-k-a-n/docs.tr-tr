# <a name="expected-beginning--for-an-xml-tag"></a>\<XML etiketi için ' ' başlangıcı bekleniyor

XML sabit değerinde XML etiketinde gereken ' < ' karakteri eksik.  
  
 **Hata kimliği:** BC31165  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML sabit değeri için XML etiketinin başlangıcına ' < ' karakterini ekleyin.
