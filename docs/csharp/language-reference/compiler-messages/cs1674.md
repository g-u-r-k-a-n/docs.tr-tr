# <a name="compiler-error-cs1674"></a><span data-ttu-id="ad9ba-102">Derleyici Hatası CS1674</span><span class="sxs-lookup"><span data-stu-id="ad9ba-102">Compiler Error CS1674</span></span>
<span data-ttu-id="ad9ba-103">'T': kullanılan bir ifadede kullanılan tür örtülü olarak 'System.IDisposable' dönüştürülebilir olmalıdır</span><span class="sxs-lookup"><span data-stu-id="ad9ba-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="ad9ba-104">Kullanım `using` [Bildirimi,](../keywords/using-statement.md) bloğun sonundaki bir nesnenin atılmasını sağlamak için kullanılmak üzere tasarlanmıştır, böylece böyle bir bildirimde yalnızca tek kullanımlık türler kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="ad9ba-104">The [using Statement](../keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="ad9ba-105">Örneğin, değer türleri tek kullanımlık değildir ve sınıf olarak sınırlandırılmamış tür parametreleri tek kullanımlık olarak kabul edilemez.</span><span class="sxs-lookup"><span data-stu-id="ad9ba-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ad9ba-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="ad9ba-106">Example</span></span>  
 <span data-ttu-id="ad9ba-107">Aşağıdaki örnek CS1674 üretir.</span><span class="sxs-lookup"><span data-stu-id="ad9ba-107">The following sample generates CS1674.</span></span>

```csharp  
// CS1674.cs

      a++;

```  
  
## <a name="example"></a><span data-ttu-id="ad9ba-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="ad9ba-108">Example</span></span>  
 <span data-ttu-id="ad9ba-109">Aşağıdaki örnek CS1674 üretir.</span><span class="sxs-lookup"><span data-stu-id="ad9ba-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
