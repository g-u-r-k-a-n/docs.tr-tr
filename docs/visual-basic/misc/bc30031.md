# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-number-characters"></a>Ad alanlarındaki türlerin tam nitelikli adları karakterden uzun olamaz \<number>

Türün tam adı, belirtilen karakter sayısından daha uzun olamaz.  
  
 **Hata kimliği:** BC30031  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Türün adını kısaltın.
