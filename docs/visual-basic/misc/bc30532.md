# <a name="conversion-from-date-to-double-requires-calling-the-datetooadate-method"></a>' Date ' türünden ' Double ' türüne dönüştürme için ' Date. ToOADate ' yönteminin çağrılması gerekir
Bir değeri bir değere dönüştürmeye çalıştınız `Date` `Double` , bu, yöntemi kullanılmadan gerçekleştirilemez <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> .  
  
 **Hata kimliği:** BC30532  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType>Değeri dönüştürmek için yöntemini kullanın.
