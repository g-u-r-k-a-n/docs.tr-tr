<span data-ttu-id="281a4-103">Genel bir arabirim iki veya daha fazla genel arabirimden kalıtımla alır ve tür bağımsız değişkenlerin belirli değerleri için devrasallarından ikisi çakışabilir.</span><span class="sxs-lookup"><span data-stu-id="281a4-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 <span data-ttu-id="281a4-104">Aşağıdaki deyimler bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="281a4-104">The following statements can generate this error.</span></span>

```vb

Public Interface interfaceX(Of v)
    Inherits interfaceA(Of v)
End Interface
Public Interface interfaceY(Of w)
    Inherits interfaceA(Of w)
End Interface
Public Interface derivedInterface(Of t1, t2)
    Inherits interfaceX(Of t1), interfaceY(Of t2)
End Interface
```

<span data-ttu-id="281a4-105">`derivedInterface`Oluşturulmuş veya uygulanmış ise, her ikisine de aynı tür sağlamak `t1` için `t2` , aynı tür bağımsız değişkenleriyle iki sürümünü devralması gerekir `interfaceA` .</span><span class="sxs-lookup"><span data-stu-id="281a4-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="281a4-106">Bunun yapılması, hangi sürümün erişecağına bir belirsizlik üretir.</span><span class="sxs-lookup"><span data-stu-id="281a4-106">Doing so would produce an ambiguity about which version to access.</span></span>
