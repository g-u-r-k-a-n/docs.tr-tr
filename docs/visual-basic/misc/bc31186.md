# <a name="-is-not-allowed"></a>': ' öğesine izin verilmiyor

': ' öğesine izin verilmiyor. XML nitelikli adları bu bağlamda kullanılamaz.  
  
 Geçersiz bir konumda nitelenmiş ada sahip bir XML sabit değeri belirtildi, örneğin, işlem yönergesi sabit değerinde bir hedef.  
  
 **Hata kimliği:** BC31186  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML sabit değerini tam adla kaldırın veya geçerli bir konuma taşıyın.  
  
     XML adlarında hangi karakterlerin geçerli olduğu hakkında bilgi için, bkz. [Genişletilebilir Biçimlendirme Dili (XML) 1,0](https://www.w3.org/TR/xml).
