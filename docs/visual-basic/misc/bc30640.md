# <a name="option-explicit-can-be-followed-only-by-on-or-off"></a>' Option Explicit ' arkasından yalnızca ' on ' veya ' off ' gelebilir

`Option Explicit` deyimler ya da belirtmelidir `On` `Off` . Diğer tüm değerler geçersiz.  
  
 **Hata kimliği:** BC30640  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ya da `Option Explicit On` belirtin `Option Explicit Off` .
