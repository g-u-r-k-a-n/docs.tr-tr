# <a name="compiler-error-cs0304"></a><span data-ttu-id="6926e-102">Derleyici Hatası CS0304</span><span class="sxs-lookup"><span data-stu-id="6926e-102">Compiler Error CS0304</span></span>

<span data-ttu-id="6926e-103">Yeni() kısıtlaması olmadığı için değişken türü 'türü' bir örnek oluşturamaz</span><span class="sxs-lookup"><span data-stu-id="6926e-103">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="6926e-104">Genel bir sınıf uyguladığınızda ve tür `new` `T`parametresi için sağlanan herhangi bir türde yeni bir örnek oluşturmak için anahtar sözcüğü `T` kullanmak istediğinizde, aşağıdaki örnekte gösterildiği gibi sınıf bildiriminde [yeni() kısıtlamayı](../keywords/new-constraint.md) uygulamanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="6926e-104">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../keywords/new-constraint.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="6926e-105">Kısıtlama, `new()` sağlanan `T` herhangi bir beton türünün varsayılan, parametresiz bir oluşturucuya sahip olduğunu garanti ederek tür güvenliğini zorlar.</span><span class="sxs-lookup"><span data-stu-id="6926e-105">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="6926e-106">`new` CS0304, `T` `new()` kısıtlamayı `T` belirtmediğinde bir tür parametresi örneği oluşturmak için sınıfın gövdesindeki işleci kullanmaya çalışırsanız oluşur.</span><span class="sxs-lookup"><span data-stu-id="6926e-106">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="6926e-107">İstemci tarafında, kod hiçbir parametresiz oluşturucu olan bir tür ile genel sınıf anlık çalışırsa, bu kod [Derleyici Hatası CS0310](./cs0310.md)oluşturur.</span><span class="sxs-lookup"><span data-stu-id="6926e-107">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](./cs0310.md).</span></span>  
  
 <span data-ttu-id="6926e-108">Aşağıdaki örnek, CS0304 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="6926e-108">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="6926e-109">İşleç `new` de sınıf yöntemleri izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="6926e-109">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="6926e-110">Hatayı önlemek için, aşağıdaki örnekte `new()` gösterildiği gibi kısıtlamayı kullanarak sınıfı bildirin.</span><span class="sxs-lookup"><span data-stu-id="6926e-110">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
