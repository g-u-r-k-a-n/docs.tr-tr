### <a name="improvements-to-grid-star-rows-space-allocating-algorithm"></a>Grid yıldız satırları alan ayırma algoritması geliştirmeleri

|   |   |
|---|---|
|Ayrıntılar|.NET Framework 4.7'de tanıtılan boyutlarda <xref:System.Windows.Controls.Grid> ) ayırmak [için algoritmadaki](https://github.com/Microsoft/dotnet/blob/master/Documentation/compatibility/wpf-grid-allocation-of-space-to-star-columns.md)bir hata düzeltildi.  Boş satırlar <code>Height=&quot;Auto&quot;</code> içeren bir Izgara gibi bazı durumlarda, satırlar yanlış konumda, muhtemelen Tamamen Izgara dışında düzenlenmiştir.|
|Öneri|Uygulamanın bu değişikliklerden yararlanabilmesi için .NET Framework 4.8 veya sonraki yerlerde çalışması gerekir.|
