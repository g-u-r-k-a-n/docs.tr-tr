<span data-ttu-id="d9020-181">Meta verileri ve ortak dil çalışma süresindeki rolünü daha iyi anlamak için, basit bir program oluşturmak ve meta verilerin çalışma süresini nasıl etkilediğini göstermek yararlı olabilir.</span><span class="sxs-lookup"><span data-stu-id="d9020-181">To better understand metadata and its role in the common language runtime, it might be helpful to construct a simple program and illustrate how metadata affects its run-time life.</span></span> <span data-ttu-id="d9020-182">Aşağıdaki kod örneği, bir sınıf `MyApp`içinde iki yöntem gösterir.</span><span class="sxs-lookup"><span data-stu-id="d9020-182">The following code example shows two methods inside a class called `MyApp`.</span></span> <span data-ttu-id="d9020-183">Yöntem `Main` sadece iki toplam bağımsız `Add` değişkenin toplamını döndürürken, yöntem program giriş noktasıdır.</span><span class="sxs-lookup"><span data-stu-id="d9020-183">The `Main` method is the program entry point, while the `Add` method simply returns the sum of two integer arguments.</span></span>

```vb
Public Class MyApp

      Dim ValueTwo As Integer = 20

   End Function

```

```csharp
using System;
public class MyApp
```
