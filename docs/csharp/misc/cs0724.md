# <a name="compiler-error-cs0724"></a>Derleyici hatası CS0724

Bağımsız değişken içermeyen bir throw deyimi, en yakın kapsayan catch yan tümcesinin içinde iç içe geçmiş bir finally yan tümcesinde kullanılamaz

## <a name="example"></a>Örnek

Aşağıdaki örnek, `finally` yan tümce bloğunun içindeki `throw` deyimi nedeniyle CS0724 oluşturuyor:

```csharp  
// CS0724.cs  
using System;  
  
class Program
{
    static void Test()
    {
        try
        {
            throw new Exception();
        }
        catch
        {
            try
            {
            }
            finally
            {
                throw; // CS0724
            }
        }
    }

    static void Main()
    {
    }
}
```
