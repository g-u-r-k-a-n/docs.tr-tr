# <a name="compiler-error-cs0404"></a>Derleyici hatası CS0404

' < ' beklenmiyordu: öznitelikler genel olamaz  
  
 Özniteliklerde genel tür parametrelerine izin verilmez. Tür parametresini ve açılı ayraçları kaldırın.  
  
 Aşağıdaki örnek CS0404 oluşturur:  
  
```csharp  
// CS0404.cs  
[MyAttrib<int>]  // CS0404  
class C  
{  
   public static void Main()  
   {  
  
   }  
}  
```
