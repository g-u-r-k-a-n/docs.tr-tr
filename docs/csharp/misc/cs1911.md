# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="89ecd-103">Derleyici Uyarısı (düzey 1) CS1911</span><span class="sxs-lookup"><span data-stu-id="89ecd-103">Compiler Warning (level 1) CS1911</span></span>

<span data-ttu-id="89ecd-104">Anonim bir yöntem, lambda ifadesi, sorgu ifadesi veya Yineleyici ile bir ' Base ' anahtar sözcüğü aracılığıyla ' name ' üyesine erişim doğrulanamayan koda neden olur.</span><span class="sxs-lookup"><span data-stu-id="89ecd-104">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="89ecd-105">Erişimi, kapsayan türdeki bir yardımcı yöntemine taşımayı düşünün.</span><span class="sxs-lookup"><span data-stu-id="89ecd-105">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="89ecd-106">`base`Bir yineleyici veya anonim metotların Yöntem gövdesinde anahtar sözcükle sanal işlevleri çağırmak doğrulanamayan koda neden olur.</span><span class="sxs-lookup"><span data-stu-id="89ecd-106">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="89ecd-107">Doğrulanamayan kod, kısmi güven ortamında çalıştırılamaz.</span><span class="sxs-lookup"><span data-stu-id="89ecd-107">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="89ecd-108">CS1911 için tek bir çözüm, sanal işlev çağrısını bir yardımcı işleve taşımadır.</span><span class="sxs-lookup"><span data-stu-id="89ecd-108">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="89ecd-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="89ecd-109">Example</span></span>  

 <span data-ttu-id="89ecd-110">Aşağıdaki örnek CS1911 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="89ecd-110">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
