# <a name="method-methodname-cannot-handle-event-eventname-because-they-do-not-have-the-same-signature"></a>' ' Metodu \<methodname> \<eventname> aynı imzaya sahip olmadıkları için ' ' olayını işleyemiyor

Belirtilen olay işleyicisinin parametreleri veya parametre türleri, belirtilen olay için tanımlananlarla eşleşmiyor.  
  
 **Hata kimliği:** BC31029  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Olayı veya olay işleyicisini parametre listelerinin eşleşmesi için yeniden tanımlayın.
