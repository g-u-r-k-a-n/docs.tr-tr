# <a name="internal-compiler-error-code-generator-received-malformed-input"></a>İç derleyici hatası: kod Oluşturucu hatalı biçimlendirilmiş giriş aldı

Visual Basic Derleyicisi, dahili olarak yazılmış dosyalardan birini başarıyla yorumlayamaz.  
  
 **Hata kimliği:** BC30011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Hatanın yinelenip yinelenmeyeceğini görmek için programı yeniden derleyin.  
  
2. Hata yinelenirse Visual Basic derleyicisini yeniden yükleyin.  
  
3. Yeniden yükleme sonrasında hata devam ederse, koşullara ilişkin bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
