 <span data-ttu-id="6b011-107">Aşağıdaki örnek CS0438 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6b011-107">The following example generates CS0438:</span></span>

 <span data-ttu-id="6b011-109">Ardından bu dosyayı derle:</span><span class="sxs-lookup"><span data-stu-id="6b011-109">Then compile this file:</span></span>

   public class A { }  
}  
```  
  
 <span data-ttu-id="6b011-110">Ve sonra bu dosyayı derle:</span><span class="sxs-lookup"><span data-stu-id="6b011-110">And then compile this file:</span></span>

{  
   public static void Main() {  
      Console.WriteLine(typeof(Util.A));   // CS0438  
   }

```
