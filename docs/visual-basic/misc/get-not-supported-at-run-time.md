# <a name="get-not-supported-at-run-time"></a>Çalışma zamanında desteklenmez

Çalışma zamanında yalnızca tasarım zamanında erişilebilen bir özelliği okumaya çalıştınız.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özelliği denetleyin ve hangi koşullara ayarlanbileceğinize göre belirleyin.  
  
2. Özelliği başvurusunu kaldırın.
