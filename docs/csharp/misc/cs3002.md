## <a name="example"></a><span data-ttu-id="486c8-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="486c8-107">Example</span></span>  

 <span data-ttu-id="486c8-108">Aşağıdaki örnek CS3002 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="486c8-108">The following example generates CS3002:</span></span>  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
