# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>Şirket içi IaaS VM altyapınızda Windows Kapsayıcıları ne zaman dağıtılmalıdır?

- Kuruluşunuz buluta taşınmaya hazır olmayabilir veya bir iş nedeni nedeniyle buluta taşıyamayabilir. Ancak, windows kapsayıcılarını kendi veri merkezlerinizde kullanmanın avantajlarından yine de yararlanabilirsiniz.

- Şirket içinde kullanılan ve buluta geçmeye çalıştığınızda sizi yavaşlatabilecek başka yapılar da olabilir. Örneğin, şirket içi Windows Server Etkin Dizini veya diğer şirket içi varlıklarla güvenlik veya kimlik doğrulama bağımlılıkları.

- Bugün Windows Kapsayıcıları kullanmaya başlarsanız, yarın çok daha iyi bir konumdan buluta aşamalı geçiş yapabilirsiniz. Windows Kapsayıcıları, kilitlenmeden herhangi bir bulut için bir dağıtım birimi haline geliyor.
