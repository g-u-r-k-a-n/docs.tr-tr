# <a name="properties-with-no-required-parameters-cannot-be-declared-default"></a>Gerekli parametreleri olmayan Özellikler ' default ' olarak bildirilemez

Gerekli parametreler olmadan ancak değiştiriciyle bir özellik belirtildi `Default` .  
  
 **Hata kimliği:** BC31048  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Varsayılan Özellikler için gerekli parametreleri bildirin.  
  
- `Default`Değiştirici Özellik tanımından kaldırın.
