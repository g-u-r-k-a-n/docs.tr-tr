# <a name="mylog-object"></a>My.Log Nesnesi

Uygulamanın günlük dinleyicilerine olay ve özel durum bilgilerini yazmak için bir özellik ve yöntemler sağlar.  
  
## <a name="remarks"></a>Açıklamalar  

 Nesnesinin yöntemleri ve özellikleri hakkında daha fazla bilgi için `My.Log` bkz <xref:Microsoft.VisualBasic.Logging.AspLog> ..  
  
 `My.Log`Nesne yalnızca ASP.NET uygulamaları için kullanılabilir. İstemci uygulamaları için [My. Application. log nesnesini](my-application-log-object.md)kullanın.
