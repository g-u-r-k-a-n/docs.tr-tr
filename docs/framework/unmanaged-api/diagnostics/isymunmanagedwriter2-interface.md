# <a name="isymunmanagedwriter2-interface"></a>ISymUnmanagedWriter2 Arabirimi

Bir sembol yazıcısını temsil eder ve belgeleri, dizi noktalarını, sözcük temelli kapsamları ve değişkenleri tanımlamak için yöntemler sağlar. Bu arabirim [ıstreamunmanagedwriter](isymunmanagedwriter-interface.md) arabirimini genişletir.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[DefineConstant2 Yöntemi](isymunmanagedwriter2-defineconstant2-method.md)|Sabit değer için bir ad tanımlar.|  
|[DefineGlobalVariable2 Yöntemi](isymunmanagedwriter2-defineglobalvariable2-method.md)|Tek bir genel değişkeni tanımlar.|  
|[DefineLocalVariable2 Yöntemi](isymunmanagedwriter2-definelocalvariable2-method.md)|Geçerli sözcük kapsamındaki tek bir değişkeni tanımlar.|  
  
## <a name="requirements"></a>Gereksinimler  

 **Üst bilgi:** CorSym. IDL, CorSym. h
