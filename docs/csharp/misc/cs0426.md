# <a name="compiler-error-cs0426"></a><span data-ttu-id="275e5-103">Derleyici hatası CS0426</span><span class="sxs-lookup"><span data-stu-id="275e5-103">Compiler Error CS0426</span></span>
<span data-ttu-id="275e5-104">Tür adı ' Identifier ', ' Type ' türünde yok</span><span class="sxs-lookup"><span data-stu-id="275e5-104">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="275e5-105">Bu hata, başka bir tür içinde iç içe geçmiş bir türe başvuru yaptığınızda oluşur, ancak böyle bir iç içe tür yok.</span><span class="sxs-lookup"><span data-stu-id="275e5-105">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="275e5-106">İç içe geçmiş türün adını yanlış yazarsanız bu durum oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="275e5-106">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="275e5-107">Kullanılan adların yazımını denetleyin ve kapsayan türün beklenen üyeye sahip olduğunu doğrulayın.</span><span class="sxs-lookup"><span data-stu-id="275e5-107">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="275e5-108">Aşağıdaki örnek CS0426 oluşturur, çünkü C sınıfı iç içe türünde bir tür içermez:</span><span class="sxs-lookup"><span data-stu-id="275e5-108">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
