# <a name="set-method-cannot-have-more-than-one-parameter"></a>' Set ' yönteminin birden fazla parametresi olamaz
Bir özellik yordamları için birden fazla parametre tanımlandı `Set` .  
  
 **Hata kimliği:** BC31063  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özellik yordamlarından ek parametreleri kaldırın `Set` .
