# <a name="integrating-with-com-applications"></a>COM Uygulamaları ile Tümleştirme

Windows Communication Foundation (WCF) Hizmetleri, WCF hizmeti bilinen adı kullanılarak doğrudan mevcut kodunuzla tümleştirilebilir. Hizmet bilinen adı, Office VBA, Visual Basic 6,0 veya Visual C++ 6,0 gibi çok çeşitli COM tabanlı geliştirme ortamlarından kullanılabilir.  
  
## <a name="in-this-section"></a>Bu Bölümde  

 [COM Uygulamaları ile Tümleştirme Genel Bakış](integrating-with-com-applications-overview.md)  
 Tümleştirme sürecinin ana bölümlerine genel bakış sunar.  
  
 [Nasıl yapılır: Hizmet Bilinen Adını Kaydetme ve Yapılandırma](how-to-register-and-configure-a-service-moniker.md)  
 Bir COM uygulamasında WCF hizmeti bilinen adını kullanmak için, gerekli olan türleri COM 'a kaydedin ve COM uygulamasını ve bilinen adı gerekli bağlama yapılandırmasıyla yapılandırın.  
  
 [Nasıl yapılır: Windows Communication Foundation Hizmeti Bilinen Adını Kaydolmadan Kullanma](use-the-wcf-service-moniker-without-registration.md)  
 Bir Web Hizmetleri tanım dili (WSDL) belgesi veya WS-MetadataExchange uç noktasından sözleşmenin tanımının nasıl alınacağını açıklar.  
  
 [Nasıl yapılır: WSDL Sözleşmeleriyle Hizmet Bilinen Adı Kullanma](how-to-use-a-service-moniker-with-wsdl-contracts.md)  
 WCF WSDL bilinen adı kullanılarak bir WCF örneğinin nasıl çağrılacağını açıklar.  
  
 [Nasıl yapılır: Meta Veri Değişimi Sözleşmeleriyle Hizmet Bilinen Adı Kullanma](how-to-use-a-service-moniker-with-metadata-exchange-contracts.md)  
 Bir MEX uç noktası belirten WCF bilinen adını kullanarak bir WCF örneğinin nasıl çağrılacağını açıklar.  
  
 [Nasıl yapılır: Kanal Güvenliği Kimlik Bilgilerini Belirtme](how-to-specify-channel-security-credentials.md)  
 WCF hizmeti bilinen adı, `IChannelCredentials` kanal kimlik bilgilerini belirtmek için bir dizi alternatif yöntemi sağlayan arabirimi destekler.
