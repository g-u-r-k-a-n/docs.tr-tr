# <a name="file-filename-could-not-be-found"></a>Dosya \<filename> bulunamadı
Belirtilen dosya bulunamadı. Bu, yanlış bir dosya yolu sağlamanın bir sonucu olabilir.  
  
 **Hata kimliği:** BC2001  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dosya adının ve yolun doğru olduğundan emin olun.
