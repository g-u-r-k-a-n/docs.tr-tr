# <a name="compiler-error-cs1906"></a>Derleyici hatası CS1906

' Option ' seçeneği geçersiz; Kaynak görünürlüğü ' Public ' veya ' Private ' olmalıdır  
  
 Bu hata, geçersiz bir [ **kaynakları** (çıkış için kaynak dosyasını katıştır)](../language-reference/compiler-options/resources.md#resources) veya [ **LinkResources** (.NET Framework kaynağa bağlantı)](../language-reference/compiler-options/resources.md#linkresources) komut satırı seçeneğini gösterir. **/Resource** veya **/linkresource** komut satırı seçeneğinin söz dizimini denetleyin ve kullanılan erişilebilirlik değiştiricisinin **ortak veya genel** olduğundan emin olun `private` .
