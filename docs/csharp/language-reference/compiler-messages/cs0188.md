 <span data-ttu-id="dd0ae-104">Oluşturucunun `struct`bir yöntemi çağırabilmesi için bir `struct` tüm alanların bir Oluşturucu tarafından atanması gerekir.</span><span class="sxs-lookup"><span data-stu-id="dd0ae-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>

## <a name="example"></a><span data-ttu-id="dd0ae-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="dd0ae-106">Example</span></span>

 <span data-ttu-id="dd0ae-107">Aşağıdaki örnek CS0188 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="dd0ae-107">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
