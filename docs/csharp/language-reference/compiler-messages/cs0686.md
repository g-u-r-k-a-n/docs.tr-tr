# <a name="compiler-error-cs0686"></a><span data-ttu-id="4551c-103">Derleyici Hatası CS0686</span><span class="sxs-lookup"><span data-stu-id="4551c-103">Compiler Error CS0686</span></span>
<span data-ttu-id="4551c-104">' Accessor ' erişimcisi ' Type ' türü için ' Member ' arabirim üyesini uygulayamaz.</span><span class="sxs-lookup"><span data-stu-id="4551c-104">Accessor 'accessor' cannot implement interface member 'member' for type 'type'.</span></span> <span data-ttu-id="4551c-105">Açık arabirim uygulamasını kullanın.</span><span class="sxs-lookup"><span data-stu-id="4551c-105">Use an explicit interface implementation.</span></span>  
  
 <span data-ttu-id="4551c-106">Önerilir: Bu hata, bir özellik veya olayla ilişkili otomatik olarak oluşturulan yöntemlerle çakışan yöntem adlarını içeren bir arabirim uygularken ortaya çıkabilir.</span><span class="sxs-lookup"><span data-stu-id="4551c-106">Suggested: This error can occur when implementing an interface that contains method names which conflict with the auto-generated methods associated with a property or event.</span></span> <span data-ttu-id="4551c-107">Özellikler için get/set yöntemleri get_property ve set_property olarak oluşturulur ve olaylar için ekleme/kaldırma yöntemleri add_event ve remove_event olarak oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="4551c-107">The get/set methods for properties are generated as get_property and set_property, and the add/remove methods for events are generated as add_event and remove_event.</span></span> <span data-ttu-id="4551c-108">Bir arabirim bu yöntemlerden birini içeriyorsa, bir çakışma oluşur.</span><span class="sxs-lookup"><span data-stu-id="4551c-108">If an interface contains either of these methods, a conflict occurs.</span></span> <span data-ttu-id="4551c-109">Bu hatayı önlemek için, yöntemleri açık bir arabirim uygulaması kullanarak uygulayın.</span><span class="sxs-lookup"><span data-stu-id="4551c-109">To avoid this error, implement the methods using an explicit interface implementation.</span></span> <span data-ttu-id="4551c-110">Bunu yapmak için, işlevi şöyle belirtin:</span><span class="sxs-lookup"><span data-stu-id="4551c-110">To do this, specify the function as:</span></span>  
  
```csharp  
Interface.get_property() { /* */ }  
Interface.set_property() { /* */ }  
```  
  
## <a name="example"></a><span data-ttu-id="4551c-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="4551c-111">Example</span></span>  
 <span data-ttu-id="4551c-112">Aşağıdaki örnek CS0686 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4551c-112">The following sample generates CS0686:</span></span>  
  
```csharp  
// CS0686.cs  
interface I  
{  
    int get_P();  
}  
  
class C : I  
{  
    public int P  
    {  
        get { return 1; }  // CS0686  
    }  
}  
// But the following is valid:  
class D : I  
{  
    int I.get_P() { return 1; }  
    public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="4551c-113">Örnek</span><span class="sxs-lookup"><span data-stu-id="4551c-113">Example</span></span>  
 <span data-ttu-id="4551c-114">Bu hata, olayları bildirirken de oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="4551c-114">This error can also occur when declaring events.</span></span>  <span data-ttu-id="4551c-115">Olay yapısı, `add_event` `remove_event` Aşağıdaki örnekte olduğu gibi bir arabirimdeki aynı ada sahip yöntemlerle çakışabilecek ve yöntemlerini otomatik olarak oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4551c-115">The event construct automatically generates the `add_event` and `remove_event` methods, which could conflict with the methods of the same name in an interface, as in the following sample:</span></span>  
  
```csharp  
// CS0686b.cs  
using System;  
  
interface I  
{  
    void add_OnMyEvent(EventHandler e);  
}  
  
class C : I  
{  
    public event EventHandler OnMyEvent  
    {  
        add { }  // CS0686  
        remove { }  
    }  
}  
  
// Correct (using explicit interface implementation):  
class D : I  
{  
    void I.add_OnMyEvent(EventHandler e) {}  
    public static void Main() {}  
}  
```
