<span data-ttu-id="f7ff6-104">İçindeki bir LINQ sorgusu `Structure` `Me` , yapının örnek üyesine veya başvurusunu içerir.</span><span class="sxs-lookup"><span data-stu-id="f7ff6-104">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="f7ff6-105">`Me`İçindeki sorgu ifadelerinde veya örnek üyelerine başvurulara izin verilmez `Structure` .</span><span class="sxs-lookup"><span data-stu-id="f7ff6-105">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="f7ff6-106">**Hata kimliği:** BC36535</span><span class="sxs-lookup"><span data-stu-id="f7ff6-106">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f7ff6-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="f7ff6-107">To correct this error</span></span>  
  
1. <span data-ttu-id="f7ff6-108">Örnek üyesinin veya başvuru tarafından döndürülen değerin bir kopyasını oluşturun `Me` ve aşağıdaki örnekte gösterildiği gibi sorgu ifadesindeki kopyayı kullanın.</span><span class="sxs-lookup"><span data-stu-id="f7ff6-108">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
