# <a name="an-embedded-expression-may-not-be-used-here"></a>Katıştırılmış bir ifade burada kullanılamaz

Katıştırılmış bir ifade, XML sabit değerinde desteklenmeyen bir konuma eklenmiş. Katıştırılmış ifadelerin bir XML ad alanı değeri belirtmek için kullanılamayacağını unutmayın.  
  
 **Hata kimliği:** BC31172  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Katıştırılmış ifadeyi desteklenmeyen konumdan kaldırın.
