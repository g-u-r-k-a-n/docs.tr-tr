# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>' \<name1> \<name2> ' İçinde ' ' için örtük olarak oluşturulan ' ' ile \<type> çakışıyor \<name3>

Bir tür üyesinin adı başka bir örtük olarak oluşturulan üyeyle çakışıyor.  
  
 **Hata kimliği:** BC31058  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Adlandırma çakışmasını kaldırmak için açıkça belirtilen üyeyi yeniden adlandırın.
