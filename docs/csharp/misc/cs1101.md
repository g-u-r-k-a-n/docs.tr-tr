# <a name="compiler-error-cs1101"></a><span data-ttu-id="86357-103">Derleyici hatası CS1101</span><span class="sxs-lookup"><span data-stu-id="86357-103">Compiler Error CS1101</span></span>

<span data-ttu-id="86357-104">' Ref ' parametre değiştiricisi ' this ' ile kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="86357-104">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="86357-105">`this`Anahtar sözcüğü bir statik yöntemin ilk parametresini değiştirdiğinde, yöntemin bir genişletme yöntemi olduğunu derleyiciye bildirir.</span><span class="sxs-lookup"><span data-stu-id="86357-105">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="86357-106">Genişletme yönteminin ilk parametresinde başka değiştiriciler gerekmez veya buna izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="86357-106">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="86357-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="86357-107">Example</span></span>  

 <span data-ttu-id="86357-108">Aşağıdaki örnek CS1101 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="86357-108">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
