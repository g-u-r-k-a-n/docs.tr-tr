# <a name="end-raiseevent-must-be-preceded-by-a-matching-raiseevent-declaration"></a>' End RaiseEvent ' öncesinde eşleşen bir ' RaiseEvent ' bildirimi olmalıdır

`End RaiseEvent`Karşılık gelen bir bildirime sahip olmayan bir ifade oluştu `RaiseEvent` . `End RaiseEvent` önce karşılık gelen bir `RaiseEvent` deyimden önce gelmelidir.  
  
 **Hata kimliği:** BC31126  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Önceki `RaiseEvent` deyimin geçerli ve doğru yazıldığından emin olun.
