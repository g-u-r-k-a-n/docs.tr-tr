# <a name="character-character-0xhexadecimal-is-not-allowed-in-an-xml-name"></a>XML adında ' character ' (0x \<hexadecimal> ) karakterine izin verilmiyor

Desteklenmeyen bir karakter, XML değişmez değeri veya XML ekseni özelliği adına eklenir.  
  
 **Hata kimliği:** BC31170  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML sabit değeri veya XML eksen özelliği adından desteklenmeyen karakteri kaldırın.
