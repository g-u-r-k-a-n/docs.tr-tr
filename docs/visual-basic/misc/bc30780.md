# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>' Continue ' deyimleri Komut penceresinde geçerli değildir
' Continue ' deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30780  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Hata ayıklama kodundan ' Continue ' ifadesini kaldırın.
