 [CLRRuntimeHost Coclass’ı](clrruntimehost-coclass.md)  
 Çalışma zamanı tarafından kod yürütmeyi yönetmek için arabirimler sağlar.  
  
 [ComCallUnmarshal Coclass’ı](comcallunmarshal-coclass.md)
