# <a name="addhandler-definition-missing-for-event-eventname"></a>' ' Olayı için ' AddHandler ' tanımı eksik \<eventname>
Bir olay olarak bildirilirse `Custom` , olay işleyicisi eklemek için bir yordam sağlamalıdır.  
  
 **Hata kimliği:** BC31130  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `AddHandler` `Custom Event` İfadesiyle ve ifadesiyle bir bildirim ekleyin `End Event` .  
  
2. Olay bildirimi içindeki diğer yordamların doğru şekilde bitdiğini doğrulayın.
