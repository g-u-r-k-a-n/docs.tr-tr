# <a name="compiler-error-cs0466"></a>Derleyici hatası CS0466

' Method1 ', ' Method2 ' olmadığı için bir params parametresine sahip olmamalıdır  
  
 `params`Uygulanan arabirim bunu kullanmıyorsa, sınıf üyesi üzerinde parametresini kullanamazsınız.  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS0466 oluşturur.  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
