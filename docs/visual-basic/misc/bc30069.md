# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>' ' For döngüsü denetim değişkeni \<variablename> zaten kapsayan bir for döngüsü tarafından kullanılıyor

İç içe `For` döngüler, benzersiz denetim değişkenleri kullanmalıdır.  
  
 **Hata kimliği:** BC30069  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Denetim değişkeni adlarından birini değiştirin.
