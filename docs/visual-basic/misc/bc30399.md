# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>' MustOverride ' olarak bildirildiği için ' MyBase ', ' ' yöntemiyle kullanılamaz \<methodname>

`MyBase`Tanımlanmış bir yöntemle kullanmayı denediniz `MustOverride` .  
  
 **Hata kimliği:** BC30399  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimi kaldırın `MustOverride` .
