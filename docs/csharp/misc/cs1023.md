# <a name="compiler-error-cs1023"></a><span data-ttu-id="70b72-103">Derleyici hatası CS1023</span><span class="sxs-lookup"><span data-stu-id="70b72-103">Compiler Error CS1023</span></span>

<span data-ttu-id="70b72-104">Gömülü ifade bir bildirim veya etiketli bir ifade olamaz</span><span class="sxs-lookup"><span data-stu-id="70b72-104">Embedded statement cannot be a declaration or labeled statement</span></span>  
  
 <span data-ttu-id="70b72-105">Bir **if** deyimini izleyen deyimler gibi gömülü bir deyim, hiçbir bildirim veya etiketli deyim içerebilir.</span><span class="sxs-lookup"><span data-stu-id="70b72-105">An embedded statement, such as the statements following an **if** statement, can contain neither declarations nor labeled statements.</span></span>  
  
 <span data-ttu-id="70b72-106">Aşağıdaki örnek, iki kez CS1023 üretir:</span><span class="sxs-lookup"><span data-stu-id="70b72-106">The following sample generates CS1023 twice:</span></span>  
  
```csharp  
// CS1023.cs  
public class a  
{  
   public static void Main()  
   {  
      if (1)  
         int i;      // CS1023, declaration is not valid here  
  
      if (1)  
         xx : i++;   // CS1023, labeled statement is not valid here  
   }  
}  
```
