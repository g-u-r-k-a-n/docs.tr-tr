# <a name="instance-members-and-me-cannot-be-used-within-a-lambda-expression-in-structures"></a><span data-ttu-id="dcd16-102">Örnek üyeleri ve ' Me ' yapılardaki bir lambda ifadesi içinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="dcd16-102">Instance members and 'Me' cannot be used within a lambda expression in structures</span></span>
<span data-ttu-id="dcd16-103">Bir yapının içinden, yapının örnek üyesine başvuran bir lambda ifadesi tanımladınız veya kullanır `Me` .</span><span class="sxs-lookup"><span data-stu-id="dcd16-103">From within a structure, you have defined a lambda expression that refers to an instance member of the structure or uses `Me`.</span></span> <span data-ttu-id="dcd16-104">Aşağıdaki kod, bu iki geçersiz başvuruyu gösterir.</span><span class="sxs-lookup"><span data-stu-id="dcd16-104">The following code illustrates these two invalid references.</span></span>  
  
```vb  
Structure Structure1  
  
    Public InstanceMember As Integer  
  
    Public Function ExampleFun() As Integer  
        '' The error is caused by use of InstanceMember.  
        'Dim fun1 = Function() InstanceMember  
        '' The error is caused by use of Me.  
        'Dim fun2 = Function() Me.InstanceMember  
        'Return fun1()  
    End Function  
  
End Structure  
```  
  
 <span data-ttu-id="dcd16-105">**Hata kimliği:** BC36638</span><span class="sxs-lookup"><span data-stu-id="dcd16-105">**Error ID:** BC36638</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="dcd16-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="dcd16-106">To correct this error</span></span>  
  
- <span data-ttu-id="dcd16-107">Örnek üyesini yerel bir değişkene atayın ve deyiminizdeki yerel değişkeni kullanın.</span><span class="sxs-lookup"><span data-stu-id="dcd16-107">Assign the instance member to a local variable, and use the local variable in your statement.</span></span>  
  
```vb  
Public Function ExampleFunFix() As Integer  
    Dim temp = InstanceMember  
    Dim fun1 = Function() temp  
    Return fun1()  
End Function  
```
