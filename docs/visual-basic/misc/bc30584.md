# <a name="type-cannot-be-inherited-more-than-once"></a>' \<type> ' birden çok kez devralınamaz

Bir ifadeyi tekrarlanmış olursunuz `Inherits` .  
  
 **Hata kimliği:** BC30584  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ifadesini kaldırın `Inherits` .
