<span data-ttu-id="37d83-104">' Method ' metodu ve ' yöntem olmayan ' yöntemi arasında belirsizlik.</span><span class="sxs-lookup"><span data-stu-id="37d83-104">Ambiguity between method 'method' and non-method 'non-method'.</span></span> <span data-ttu-id="37d83-105">Yöntem grubu kullanılıyor.</span><span class="sxs-lookup"><span data-stu-id="37d83-105">Using method group.</span></span>

## <a name="example"></a><span data-ttu-id="37d83-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="37d83-107">Example</span></span>  

 <span data-ttu-id="37d83-108">Aşağıdaki örnek CS0467 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="37d83-108">The following example generates CS0467.</span></span>  
  
```csharp  
// CS0467.cs  
interface IList
{  
    int Count { get; set; }  
}  
  
interface ICounter  
{  
    void Count(int i);  
}  
  
interface IListCounter : IList, ICounter {}  
  
class Driver
{  
    void Test(IListCounter x)  
    {  
        // The following line causes the warning. The assignment also  
        // causes an error because you can't assign a value to a method.  
        x.Count = 1;  
        x.Count(3);
        // To resolve the warning, you can change the name of the method or
        // the property.  
  
        // You can also disambiguate by specifying IList or ICounter.  
        ((IList)x).Count = 1;  
        ((ICounter)x).Count(3);  
    }  
  
    static void Main()
    {  
    }  
}  
```
