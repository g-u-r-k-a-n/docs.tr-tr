# <a name="compiler-error-cs0636"></a><span data-ttu-id="7b257-103">Derleyici hatası CS0636</span><span class="sxs-lookup"><span data-stu-id="7b257-103">Compiler Error CS0636</span></span>
<span data-ttu-id="7b257-104">FieldOffset özniteliği yalnızca StructLayout (LayoutKind. Explicit) ile işaretlenmiş türlerin üyelerine yerleştirilebilecek</span><span class="sxs-lookup"><span data-stu-id="7b257-104">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="7b257-105">**FieldOffset** özniteliğiyle işaretlenmiş herhangi bir üye içeriyorsa, struct bildiriminde **StructLayout (LayoutKind. Explicit)** özniteliğini kullanmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="7b257-105">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="7b257-106">Daha fazla bilgi için bkz. [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="7b257-106">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="7b257-107">Aşağıdaki örnek CS0636 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7b257-107">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
