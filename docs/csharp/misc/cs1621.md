# <a name="compiler-error-cs1621"></a><span data-ttu-id="7bf50-103">Derleyici hatası CS1621</span><span class="sxs-lookup"><span data-stu-id="7bf50-103">Compiler Error CS1621</span></span>
<span data-ttu-id="7bf50-104">Yield deyimi bir anonim yöntem veya lambda ifadesi içinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="7bf50-104">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="7bf50-105">Bir [anonim yöntemde](../language-reference/operators/delegate-operator.md) veya [lambda ifadesinde](../language-reference/operators/lambda-expressions.md) [yield](../language-reference/keywords/yield.md) deyimini kullanamazsınız.</span><span class="sxs-lookup"><span data-stu-id="7bf50-105">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../language-reference/operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="7bf50-106">Aşağıdaki örnek CS1621 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7bf50-106">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
