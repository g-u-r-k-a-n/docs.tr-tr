# <a name="the-file-is-currently-open-for-writing"></a>Dosya şu anda yazma için açık

Şu anda yazmak üzere açık olan bir dosyaya erişme girişiminde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dosyayı kapatın ve işlemi yeniden deneyin.
