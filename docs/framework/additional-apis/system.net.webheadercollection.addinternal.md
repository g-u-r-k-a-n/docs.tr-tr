# <a name="webheadercollectionaddinternal-method"></a>WebHeaderCollection. Addınternal yöntemi

Denetimleri atlayarak, belirtilen ad ve değere sahip yeni bir üst bilgi ekler.

```csharp
internal void AddInternal(string name, string value)
```

> [!WARNING]
> Bu yöntem dahili ve doğrudan kodunuzda kullanılmamalıdır.
>
> Microsoft, bu yöntemin herhangi bir koşulda bir üretim uygulamasında kullanımını desteklemez.

## <a name="parameters"></a>Parametreler

- `name` <xref:System.String>

  Üstbilginin adı.

- `value` <xref:System.String>

  Üst bilginin değeri.
