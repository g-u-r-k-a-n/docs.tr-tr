# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Modül içindeki metotlar arabirim üyeleri uygulayamaz

Arabirim üyelerinin sınıflar veya yapılar tarafından uygulanması gerekir.  
  
 **Hata kimliği:** BC31083  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sınıflardan veya yapılardan arabirim üyeleri uygulayın.
