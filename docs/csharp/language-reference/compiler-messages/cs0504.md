 <span data-ttu-id="8bbb1-105">Bir değişken [const](../keywords/const.md)ise, aynı zamanda [statiktir](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="8bbb1-105">If a variable is [const](../keywords/const.md), it is also [static](../keywords/static.md).</span></span> <span data-ttu-id="8bbb1-106">**Sabit** ve **statik** bir değişken istiyorsanız bu değişkeni **const** olarak bildirmeniz yeterlidir; istediğiniz şey **statik** bir değişkense, **statik** olarak işaretlemeniz yeterlidir.</span><span class="sxs-lookup"><span data-stu-id="8bbb1-106">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="8bbb1-107">Aşağıdaki örnek CS0504 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8bbb1-107">The following sample generates CS0504:</span></span>
