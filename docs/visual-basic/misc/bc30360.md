# <a name="property-propertyname-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-name"></a>' ' \<propertyname> Tabanında varsayılan bir özelliği geçersiz kıldığından ' ' özelliği ' default ' olarak bildirilmelidir \<name>

`Property`Kodunuzdaki bir ifade `Default` , temel sınıf, yapı veya arabirimdeki bir özelliği geçersiz kılar ve bu nedenle, bildirim alınması gerekir `Default` .  
  
 **Hata kimliği:** BC30360  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özelliği bildirin `Default` .
