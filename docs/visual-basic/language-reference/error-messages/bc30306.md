# <a name="array-subscript-expression-missing"></a>Dizi alt simge ifadesi eksik
Dizi başlatma, dizi sınırlarını tanımlayan bir veya daha fazla alt simge çıkar. Örneğin, deyimi `myArray (5,5,,10)` üçüncü alt simgeyi atan ifadesi içerebilir.  
  
 **Hata kimliği:** BC30306  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Eksik alt indisi sağlayın.
