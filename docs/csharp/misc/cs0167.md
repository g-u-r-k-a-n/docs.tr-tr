# <a name="compiler-error-cs0167"></a>Derleyici hatası CS0167
' Delegate ' temsilcisinin Invoke yöntemi eksik  
  
 Başka bir derleyici ile oluşturulmuş bir yönetilen program (.NET Framework ortak dil çalışma zamanı kullanan) içeri aktardınız ve kullandınız. Bu derleyici, hatalı oluşturulmuş bir [temsilciye](../language-reference/builtin-types/reference-types.md)izin verilir. Bu nedenle, `Invoke` Yöntem kullanılamıyor. Daha fazla bilgi için bkz. [Temsilciler](../programming-guide/delegates/index.md).
