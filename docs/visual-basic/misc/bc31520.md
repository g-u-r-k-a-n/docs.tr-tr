# <a name="unable-to-apply-security-attributes-to-name-error"></a>Güvenlik öznitelikleri ' ' öğesine uygulanamıyor \<name> : \<error>

Güvenlik öznitelikleri uygulanırken bir hata oluştu. Bu, güvenlik özniteliklerinin yanlış kullanılması veya derleyici ile ilgili bir sorun olduğunu gösteriyor olabilir.  
  
 **Hata kimliği:** BC31520  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özniteliğin sözdiziminin ve bağımsız değişkenlerinin olduğunu doğrulayın.  
  
2. Hatanın yeniden oluşup oluşmadığını görmek için programı yeniden derleyin.  
  
3. Hata devam ederse, çalışmanızı kaydedin ve Visual Studio 'Yu yeniden başlatın.  
  
4. Hata yeniden oluşursa Visual Basic yeniden yükleyin.  
  
5. Yeniden yükleme sonrasında hata devam ederse, Microsoft Ürün Destek Hizmetleri 'ne bildirin.
