<span data-ttu-id="0b142-104">Olası hatalı boş ifade</span><span class="sxs-lookup"><span data-stu-id="0b142-104">Possible mistaken empty statement</span></span>  
  
 <span data-ttu-id="0b142-105">Koşullu bir deyimden sonra noktalı virgül, kodunuzun istenenden farklı şekilde yürütülmesine neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="0b142-105">A semicolon after a conditional statement may cause your code to execute differently than intended.</span></span>  
  
 <span data-ttu-id="0b142-106">**/Nowarn** derleyici seçeneğini veya `#pragmas warning` Bu uyarıyı devre dışı bırakmak için bkz. [-nowarn (C# derleyici seçenekleri)](../language-reference/compiler-options/nowarn-compiler-option.md) veya daha fazla bilgi için [#pragma uyarısı](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) .</span><span class="sxs-lookup"><span data-stu-id="0b142-106">You can use **/nowarn** compiler option or `#pragmas warning` to disable this warning; see [-nowarn (C# Compiler Options)](../language-reference/compiler-options/nowarn-compiler-option.md) or [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) for more information.</span></span>  
  
 <span data-ttu-id="0b142-107">Aşağıdaki örnek CS0642 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0b142-107">The following sample generates CS0642:</span></span>  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
