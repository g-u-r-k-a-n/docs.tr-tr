# <a name="types-are-not-available-in-this-context"></a>Türler bu bağlamda kullanılamaz

Yönerge içindeki bir türe başvurmasına çalıştınız `#If` .  
  
 **Hata kimliği:** BC30458  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Başvuruyu kaldırın.
