# <a name="compiler-error-cs0555"></a>Derleyici hatası CS0555

Kullanıcı tanımlı işleç kapsayan türdeki bir nesneyi alamaz ve kapsayan türdeki bir nesneye dönüştüremez

Kapsayan sınıfın değerlerine Kullanıcı tanımlı Dönüştürmelere izin verilmiyor; Böyle bir işlece gerek yoktur.

Aşağıdaki örnek CS0555 oluşturur:

```csharp
// CS0555.cs
public class MyClass
{
   // delete the following operator to resolve this CS0555
   public static implicit operator MyClass(MyClass aa)   // CS0555
   {
      return new MyClass();
   }

   public static void Main()
   {
   }
}
```
