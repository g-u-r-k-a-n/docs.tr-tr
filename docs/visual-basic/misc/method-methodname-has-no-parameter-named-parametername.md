# <a name="method-methodname-has-no-parameter-named-parametername"></a>' \<methodname> ' Metodunun ' ' adlı parametresi yok \<parametername>

Sağlanan adlandırılmış bağımsız değişken, hedef metodun bağımsız değişkenlerinden hiçbiriyle eşleşmiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Adlandırılmış bağımsız değişkenin doğru verildiğinden emin olun.
