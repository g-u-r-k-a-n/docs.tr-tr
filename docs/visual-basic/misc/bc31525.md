# <a name="type-typename-is-imported-from-different-versions-of-assembly-assemblyname"></a>' ' Türü \<typename> , ' ' derlemesinin farklı sürümlerinden içeri aktarıldı \<assemblyname>

' ' Türü \<typename> , ' ' derlemesinin farklı sürümlerinden içeri aktarıldı \<assemblyname> . Aynı derlemenin farklı sürümleri aynı projede kullanılamaz.  
  
 Bir tür, aynı derlemenin farklı sürümlerinden içeri aktarıldı, belirsizlik oluşturuluyor.  
  
 **Hata kimliği:** BC31525  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Aynı derlemenin birden çok sürümüne başvuruları kaldırın.
