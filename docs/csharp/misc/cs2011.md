# <a name="compiler-error-cs2011"></a>Derleyici hatası CS2011

' Dosya ' yanıt dosyası açılamadı  
  
 Derlemede bir yanıt dosyası belirtildi, ancak derleyici dosyayı bulamıyor ve açamadı.  
  
 Yanıt dosyaları hakkında daha fazla bilgi için bkz. [ **ResponseFiles** (yanıt dosyası belirtme)](../language-reference/compiler-options/miscellaneous.md#responsefiles).
