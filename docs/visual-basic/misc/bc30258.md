# <a name="classes-can-inherit-only-from-other-classes"></a>Sınıflar yalnızca diğer sınıflardan kalıtımla alabilir

Bir `Inherits` ifade, temel türü olarak bir sınıf dışında bir türü belirtir.  
  
 **Hata kimliği:** BC30258  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Temel türü önceden tanımlanmış bir sınıfa değiştirin.
