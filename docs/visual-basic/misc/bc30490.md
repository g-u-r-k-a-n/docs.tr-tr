# <a name="default-cannot-be-combined-with-specifier"></a>' Default ', ' ' ile birleştirilemez \<specifier>

`Default`Anahtar sözcüğünü, veya gibi bir belirticiyle birleştirmeye çalıştınız, `Shared` `Private` Bu bağlamda geçerli değildir.  
  
 **Hata kimliği:** BC30490  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaldırın `Default` .  
  
     -veya-  
  
- Çakışan belirticisini kaldırın.
