# <a name="compiler-error-cs0441"></a>Derleyici hatası CS0441

' class ': bir sınıf hem static hem de Sealed olamaz  
  
 Hem statik hem de korumalı bir sınıf bildirdiğinizde bu hata oluşur. Statik sınıflar doğal olarak mühürlenmiş olduğundan, korumalı değiştirici gerekli değildir. Çözümlemek için yalnızca bir değiştirici kullanın.  
  
 Aşağıdaki örnek CS0441 oluşturur:  
  
```csharp  
// CS0441.cs  
sealed static class MyClass { }   // CS0441  
```
