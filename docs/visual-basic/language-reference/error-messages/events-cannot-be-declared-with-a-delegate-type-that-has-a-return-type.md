# <a name="bc31084-events-cannot-be-declared-with-a-delegate-type-that-has-a-return-type"></a>BC31084: olaylar, dönüş türü olan bir temsilci türüyle bildirilemez

Bir işlev yordamı için bir temsilci belirtildi.

 **Hata kimliği:** BC31084

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Bir yordam için temsilci belirtin `Sub` .
