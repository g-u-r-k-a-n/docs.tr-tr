## <a name="cause"></a><span data-ttu-id="4fa14-111">Nedeni</span><span class="sxs-lookup"><span data-stu-id="4fa14-111">Cause</span></span>

<span data-ttu-id="4fa14-112"><xref:System.Net.Http.HttpClient?displayProperty=fullName>, Bir platforma özel işleyici ( <xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> veya) sağlarken <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName> `CheckCertificateRevocationList` `true` , özelliği, iptal edilen sertifikaların geçerli olarak kabul edilmesine izin verecek şekilde ayarlanmamış <xref:System.Net.Http.HttpClient> .</span><span class="sxs-lookup"><span data-stu-id="4fa14-112">Using <xref:System.Net.Http.HttpClient?displayProperty=fullName> while providing a platform specific handler (<xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> or <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName>) whose `CheckCertificateRevocationList` property is not set to `true` will allow revoked certificates to be accepted by the <xref:System.Net.Http.HttpClient> as valid.</span></span>

<span data-ttu-id="4fa14-113">Bu kural [CA5400](ca5400.md)ile benzerdir, ancak analiz `CheckCertificateRevocationList` özelliğin kesinlikle veya ayarlanmamış olduğunu tespit edebilir `false` .</span><span class="sxs-lookup"><span data-stu-id="4fa14-113">This rule is similar to [CA5400](ca5400.md), but analysis can determine that the `CheckCertificateRevocationList` property is definitely `false` or not set.</span></span>
