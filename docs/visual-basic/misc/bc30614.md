# <a name="mustoverride-method-methodname-cannot-be-called-with-myclass"></a>' ' MustOverride ' metodu ' \<methodname> ', ' MyClass ' ile çağrılamaz
`MyClass`öğesine eşdeğerdir `Me` , ancak çağrılan yöntem olarak kabul edilir `NotOverridable` .  
  
 **Hata kimliği:** BC30614  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştiricisini kaldırın `MustOverride` veya bir temel sınıfta yöntemi bildirin ve bu sınıfı alıp geçersiz kılın.
