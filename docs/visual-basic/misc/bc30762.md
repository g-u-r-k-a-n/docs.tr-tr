# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebinding"></a>.NET Compact Framework hedeflenen sürümü geç bağlamayı desteklemiyor

Üzerinde çalıştığınız .NET Compact Framework sürümü geç bağlamayı desteklemiyor.  
  
 **Hata kimliği:** BC30762  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Nesne olarak belirtilen bir değişkende işlevleri, alt öğeleri veya özellikleri çağırmayı önleyin.  
  
2. Nesne değişkenini dizi olarak kullanmaktan kaçının.  
  
3. Nesne değişkenleriyle sözlük üyesi erişim ifadeleri kullanmaktan kaçının.
