 <span data-ttu-id="8a75d-107">Aşağıdaki örnek CS0127 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8a75d-107">The following sample generates CS0127:</span></span>

```csharp  
   public class MyClass  
   {  
      public int hiddenMember2

         }

            return 0;   // remove return statement to resolve this CS0127  
         }  
      }

      {  
      }  
   }  
}  
```
