# <a name="methods-or-events-that-implement-interface-members-cannot-be-declared-shared"></a>Arabirim üyelerini uygulayan metotlar veya olaylar ' Shared ' olarak bildirilemez
`Shared`Arabirim üyesini uygulayan bir yöntem veya olay olarak bildirmeye çalıştınız. Bir sınıfta uygulanan metotlar ve olaylar `Shared` `Private` , devredilemeyen bir sınıf dışında, ya da belirlenmemiş olamaz.  
  
 **Hata kimliği:** BC30505  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Shared`Anahtar sözcüğünü kaldırın.
