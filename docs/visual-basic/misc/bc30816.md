# <a name="membername-is-not-declared"></a>'\<MemberName > ' bildirilmemiş
'`<membername>`' bildirilmemiş. `Debug` nesne işlevselliği `System` derlemesinde `System.Diagnostics.Debug` kullanılabilir.  
  
 Belirtilen üye adı bulunamadı.  
  
 **Hata kimliği:** BC30816  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Üyenin yazımını doğrulayın.  
  
2. `Imports` bir ifade kullanın veya diğer ad alanlarında tanımlanan üyeleri tamamen niteleyin. Örneğin, `WriteLine` işlevi `System.Diagnostics.Debug` ad alanında bildirilmiştir.
