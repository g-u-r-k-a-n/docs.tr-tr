# <a name="project-level-conditional-compilation-constant-constantname-is-not-valid-error"></a>Proje düzeyi koşullu derleme sabiti ' \<constantname> ' geçerli değil: \<error>

Proje için geçersiz bir koşullu derleme seçeneği belirtildi.  
  
 **Hata kimliği:** BC31030  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bu projeyle kullanılan koşullu derleme sabitlerini doğrulayın.
