# <a name="compiler-error-cs0206"></a>Derleyici hatası CS0206

Bir özellik veya Dizin Oluşturucu, out veya ref parametresi olarak geçirilemez  
  
 Bir [özellik](../programming-guide/classes-and-structs/properties.md) , [ref](../language-reference/keywords/ref.md) veya [Out](../language-reference/keywords/out-parameter-modifier.md) parametresi olarak geçirilemeyebilir. Daha fazla bilgi için bkz. [parametreleri geçirme](../programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS0206 oluşturur:  
  
```csharp  
// CS0206.cs  
public class MyClass  
{  
    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMeth(ref int i)  
    // public static void MyMeth(int i)  
    {  
    }  
  
    public static void Main()  
    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```
