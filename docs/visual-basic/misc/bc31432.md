<span data-ttu-id="80619-103">`Private`Kısmi Yöntem bildirimlerinde erişim değiştiricisi gereklidir.</span><span class="sxs-lookup"><span data-stu-id="80619-103">The access modifier `Private` is required in partial method declarations.</span></span> <span data-ttu-id="80619-104">Aşağıdaki örnek, `Private` yöntemi imzasında ve uygulamanın uygulamasında kullanımını gösterir.</span><span class="sxs-lookup"><span data-stu-id="80619-104">The following example shows the use of `Private` in the method signature and its implementation.</span></span>  
  
```vb  
' Definition of the partial method signature.  
Partial Private Sub OnNameChanged()  
    ' The body of the signature is empty.  
End Sub  
```  
  
```vb  
' Implementation of the partial method.  
Private Sub OnNameChanged()  
    MsgBox("Name was changed to " & Me.Name)  
End Sub  
```
