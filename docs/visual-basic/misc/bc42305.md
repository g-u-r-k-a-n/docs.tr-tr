# <a name="xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>' ' XML açıklama etiketi \<tag> , aynı XML açıklama bloğunda birden çok kez aynı özniteliklerle görünüyor

XML açıklama etiketi \<tag> aynı özniteliklerle aynı XML açıklama bloğunda birden çok kez görünüyor. XML açıklaması yoksayılacak.  
  
 XML açıklamasında aynı özniteliklere sahip birden fazla etiket var.  
  
 **Hata kimliği:** BC42305  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yinelenen etiketi kaldırın.
