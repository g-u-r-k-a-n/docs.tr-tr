# <a name="compiler-error-cs1577"></a>Derleyici hatası CS1577

Derleme oluşturma başarısız oldu — neden  
  
 C# derleyicisi kaynak kodu başarıyla derlendikten sonra derlemenin derleme oluşturma bölümü başarısız oldu. Daha fazla bilgi için bkz. ALink yardımcı programı ([Al.exe araç hataları ve uyarıları](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)) için hatalar belgeleri.
