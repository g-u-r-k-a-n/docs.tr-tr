# <a name="white-space-may-not-appear-here"></a>Boşluk, burada görünmeyebilir
Bir XML öğesi sabit değerinin adına bir boşluk veya boşluk eklenir.  
  
 **Hata kimliği:** BC31177  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML öğesi değişmez değerinin adından boşluğu veya boşlukları kaldırın.
