# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Bu ifade değerlendirilirken sıfıra bölme gerçekleşti

Bu ifadeyi değerlendirmek için bir sayıyı sıfıra bölmeye çalıştınız.  
  
 **Hata kimliği:** BC30542  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Deyiminizin paydasında sıfırın ne olduğunu anlamak için sabit ve değişken değerlerinizi kontrol edin.
