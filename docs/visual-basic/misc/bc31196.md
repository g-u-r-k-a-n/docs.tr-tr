# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Bu derlemede kullanılmadığından, bu hata ayıklama oturumunda XML sabit değerleri ve XML eksen özellikleri kullanılamaz

Visual Basic özelliklerinin kullanılamadığı bir hata ayıklama oturumu sırasında **izleme** veya **anında** penceresinde bir XML sabit değeri veya XML ekseni özelliğine başvuruldu. Bu, Visual Basic özelliklerde XML kullanmayan veya bir yayın derlemesi olan bir derleme için durumdur.  
  
 **Hata kimliği:** BC31196  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Derlemenin hata ayıklama derlemesini kullanarak yeni bir hata ayıklama oturumu başlatın.
