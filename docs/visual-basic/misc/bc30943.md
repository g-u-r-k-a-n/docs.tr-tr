# <a name="generic-methods-cannot-be-exposed-to-com"></a>Genel metotlar COM 'a sunulamaz

Bir veya daha fazla genel yordam içeren .NET Framework bileşen bir COM bileşenine aktarılmalıdır.  
  
 Bileşen nesne modeli (COM) genel türleri desteklemez ve bunlarla birlikte çalışamaz.  
  
 **Hata kimliği:** BC30943  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- .NET Framework bileşeninden genel yordam veya yordamları kaldırın veya COM birlikte çalışması için kullanmayın.
