# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>Tür bildirimi karakterleri bu bağlamda geçerli değil

$ Gibi, bu bağlamda geçerli olmayan bir karakter kullanmaya çalıştınız.  
  
 **Hata kimliği:** BC30468  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Karakteri kaldırın.
