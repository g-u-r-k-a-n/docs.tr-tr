# <a name="compiler-error-cs1536"></a><span data-ttu-id="ab219-103">Derleyici hatası CS1536</span><span class="sxs-lookup"><span data-stu-id="ab219-103">Compiler Error CS1536</span></span>

<span data-ttu-id="ab219-104">Geçersiz parametre türü void</span><span class="sxs-lookup"><span data-stu-id="ab219-104">Invalid parameter type void</span></span>  
  
 <span data-ttu-id="ab219-105">Void işaretçisi dışında [void](../language-reference/builtin-types/void.md) bir parametre belirtmek için gerekli veya geçerli değildir.</span><span class="sxs-lookup"><span data-stu-id="ab219-105">It is not necessary or valid to specify a [void](../language-reference/builtin-types/void.md) parameter other than a void pointer.</span></span>  
  
 <span data-ttu-id="ab219-106">Aşağıdaki örnek CS1536 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ab219-106">The following sample generates CS1536:</span></span>  
  
```csharp  
// CS1536.cs  
class a  
{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
