# <a name="methodname-cannot-override-a-method-that-has-been-shadowed"></a>' \<methodname> ' gölgeli bir yöntemi geçersiz kılamaz
Bir yöntem, başka bir üyenin gölgesindeki bir yöntemi geçersiz kılmaya çalıştı.  
  
 **Hata kimliği:** BC31406  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Shadows`Gölgelendirilmiş üyeleri geçersiz kılmak için değiştiricisini kullanın.
