 <span data-ttu-id="d6f87-107">Aşağıdaki örnek CS0655 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d6f87-107">The following sample generates CS0655:</span></span>

```csharp  
    // decimal is not valid attribute parameter type

// Try the following line instead:  
// [My(e = 0)]

{  
    public static void Main()

    }  
}  
```
