# <a name="compiler-error-cs1617"></a>Derleyici hatası CS1617

/Langversioniçin geçersiz ' Option ' seçeneği. '/Langversion:? ' kullanın Desteklenen değerleri listelemek için.

[-Langversion](../language-reference/compiler-options/langversion-compiler-option.md) komut satırı anahtarını veya proje ayarını kullandıysanız ancak geçerli bir dil seçeneği belirtmediyseniz bu hata oluşur. Bu hatayı çözmek için komut satırı söz dizimini veya proje ayarını kontrol edin ve listede listelenen seçeneklerden birini değiştirin.

Örneğin, ile derleme `csc -langversion:ISO` CS1617 hatasını oluşturacaktır.
