# <a name="unable-to-include-xml-fragment-elementpath-of-file-commentfile"></a>' ' Dosyasının ' ' XML parçası eklenemiyor \<elementpath> \<commentfile>

`<include>`XML açıklamadaki bir etiket, varolmayan bir öğe için bir XPath ifadesi içeriyor veya dosya hatalı BIÇIMLENDIRILMIŞ XML içeriyor.  
  
 **Hata kimliği:** BC42320  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosyadaki XML 'nin geçerli olduğunu doğrulayın.  
  
2. XML parçasının dosyada bulunduğundan emin olun.
