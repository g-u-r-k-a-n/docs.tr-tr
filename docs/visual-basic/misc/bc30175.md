# <a name="type1-typename-and-type2-typename-declared-in-name-conflict-in-namespace-namespacename"></a>\<type1>' ' \<typename> içinde belirtilen ' ' ve ' ', ' ' \<type2> \<typename> \<name> içinde \<namespace> çakışıyor \<namespacename>

Bir sınıf ve modül gibi tanımlanmış iki öğe aynı ad alanında aynı ada sahiptir.  
  
 **Hata kimliği:** BC30175  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öğelerinden birinin bildirimini farklı bir ad alanına taşıyın.  
  
- Öğelerden birinin adını değiştirin.
