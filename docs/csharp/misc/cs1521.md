<span data-ttu-id="88b48-104">Geçersiz temel tür</span><span class="sxs-lookup"><span data-stu-id="88b48-104">Invalid base type</span></span>  
  
 <span data-ttu-id="88b48-105">[Temel](../language-reference/keywords/base.md) sınıf belirtimi hatalı biçimlendirilmiş.</span><span class="sxs-lookup"><span data-stu-id="88b48-105">A [base](../language-reference/keywords/base.md) class specification was ill formed.</span></span>  
  
 <span data-ttu-id="88b48-106">Aşağıdaki örnek CS1521 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="88b48-106">The following sample generates CS1521:</span></span>  
  
```csharp  
// CS1521.cs

{  
   public static void Main()  
   {  
   }  
}

{  
}

{  
}

{  
}  
```
