# <a name="compiler-warning-level-1-cs3027"></a><span data-ttu-id="79ab4-102">Derleyici Uyarısı (düzey 1) CS3027</span><span class="sxs-lookup"><span data-stu-id="79ab4-102">Compiler Warning (level 1) CS3027</span></span>

<span data-ttu-id="79ab4-103">' type_2 ' temel arabirimi CLS uyumlu olmadığından, ' type_1 ' CLS uyumlu değil</span><span class="sxs-lookup"><span data-stu-id="79ab4-103">'type_1' is not CLS-compliant because base interface 'type_2' is not CLS-compliant</span></span>

## <a name="example"></a><span data-ttu-id="79ab4-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="79ab4-105">Example</span></span>  

 <span data-ttu-id="79ab4-106">Aşağıdaki örnek, kendi imzasında CLS uyumlu olmayan bir tür kullanan bir arabirim içerir ve türü CLS uyumlu değil olarak yapar.</span><span class="sxs-lookup"><span data-stu-id="79ab4-106">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>

```csharp  
// compile with: /target:library

{  
   void IMethod(uint i);  
}  
```
