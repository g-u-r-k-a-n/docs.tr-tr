# <a name="compiler-warning-level-1-cs3004"></a>Derleyici Uyarısı (düzey 1) CS3004

Karışık ve ayrıştırılmış Unicode karakterleri CLS uyumlu değildir  
  
 [Ortak,](../language-reference/keywords/public.md) [korumalı](../language-reference/keywords/protected.md)veya `protected internal` tanımlayıcıların ortak dil belirtimi (CLS) ile uyumlu olması için yalnızca oluşturulan UNICODE karakterlere izin verilir. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [Dil bağımsızlığı ve dilden bağımsız bileşenler](../../standard/language-independence.md).
