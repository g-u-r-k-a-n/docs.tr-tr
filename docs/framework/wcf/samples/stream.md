 <span data-ttu-id="8ddb2-139">İşlemler `GetStream` `UploadStream` ve `EchoStream` tümü doğrudan bir dosyadan veri göndermeye veya alınan verileri doğrudan bir dosyaya kaydetmeye yöneliktir.</span><span class="sxs-lookup"><span data-stu-id="8ddb2-139">The operations `GetStream`, `UploadStream` and `EchoStream` all deal with sending data directly from a file or saving received data directly to a file.</span></span> <span data-ttu-id="8ddb2-140">Ancak bazı durumlarda, büyük miktarlarda veri gönderilmesi veya alınması ve verilerin gönderildiği veya alındığı şekilde öbeklerinde bazı işlemler gerçekleştirme gereksinimi vardır.</span><span class="sxs-lookup"><span data-stu-id="8ddb2-140">However in some cases, there is a requirement to send or receive large amounts of data and perform some processing on chunks of the data as it is being sent or received.</span></span> <span data-ttu-id="8ddb2-141">Bu tür senaryoları ele almanın bir yolu, <xref:System.IO.Stream> verileri okunan veya yazıldığı gibi işleyen özel bir akış (öğesinden türetilen bir sınıf) yazmak.</span><span class="sxs-lookup"><span data-stu-id="8ddb2-141">One way to address such scenarios is to write a custom stream (a class that derives from <xref:System.IO.Stream>) that processes data as it is read or written.</span></span> <span data-ttu-id="8ddb2-142">`GetReversedStream`İşlem ve `ReverseStream` sınıf buna bir örnektir.</span><span class="sxs-lookup"><span data-stu-id="8ddb2-142">The `GetReversedStream` operation and `ReverseStream` class are an example of this.</span></span>  
  
 <span data-ttu-id="8ddb2-143">`GetReversedStream` Yeni bir örneğini oluşturur ve döndürür `ReverseStream` .</span><span class="sxs-lookup"><span data-stu-id="8ddb2-143">`GetReversedStream` creates and returns a new instance of `ReverseStream`.</span></span> <span data-ttu-id="8ddb2-144">Bu nesneden sistem okuduğu için gerçek işlem gerçekleşir `ReverseStream` .</span><span class="sxs-lookup"><span data-stu-id="8ddb2-144">The actual processing happens as the system reads from that `ReverseStream` object.</span></span> <span data-ttu-id="8ddb2-145">`ReverseStream.Read`Uygulama, temel alınan dosyadaki bir bayt öbeğini okur, bunları tersine çevirir ve ters bayt döndürür.</span><span class="sxs-lookup"><span data-stu-id="8ddb2-145">The `ReverseStream.Read` implementation reads a chunk of bytes from the underlying file, reverses them, then returns the reversed bytes.</span></span> <span data-ttu-id="8ddb2-146">Bu, tüm dosya içeriğini tersine döndürmez; tek seferde bir bayt öbeğini tersine çevirir.</span><span class="sxs-lookup"><span data-stu-id="8ddb2-146">This does not reverse the entire file content; it reverses one chunk of bytes at a time.</span></span> <span data-ttu-id="8ddb2-147">Bu, içerik okunmakta veya akışa yazılırken akış işlemeyi nasıl gerçekleştirebileceğini gösteren bir örnektir.</span><span class="sxs-lookup"><span data-stu-id="8ddb2-147">This is an example to show how you can perform stream processing as the content is being read or written from and to the stream.</span></span>  
  
```csharp
class ReverseStream : Stream  
{  
  
    FileStream inStream;  
    internal ReverseStream(string filePath)  
    {  
        //Opens the file and places a StreamReader around it.  
        inStream = File.OpenRead(filePath);  
    }  
  
    // Other methods removed for brevity.  
  
    public override int Read(byte[] buffer, int offset, int count)  
    {  
        int countRead=inStream.Read(buffer, offset,count);  
        ReverseBuffer(buffer, offset, countRead);  
        return countRead;  
    }  
  
    public override void Close()  
    {  
        inStream.Close();  
        base.Close();  
    }  
    protected override void Dispose(bool disposing)  
    {  
        inStream.Dispose();  
        base.Dispose(disposing);  
    }  
    void ReverseBuffer(byte[] buffer, int offset, int count)  
    {  
        int i, j;  
        for (i = offset, j = offset + count - 1; i < j; i++, j--)  
        {  
            byte currenti = buffer[i];  
            buffer[i] = buffer[j];  
            buffer[j] = currenti;  
        }  
  
    }  
}  
```
