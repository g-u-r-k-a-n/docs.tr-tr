# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>XML yorumu satırdaki ilk ifade olmalıdır

XML yorumu satırdaki ilk ifade olmalıdır. XML açıklaması yoksayılacak.  
  
 XML yorumu, izin verilmeyen başka bir dil öğesinden sonra yerleştirildi.  
  
 **Hata kimliği:** BC42302  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML açıklamasını yeni bir satıra taşıyın.
