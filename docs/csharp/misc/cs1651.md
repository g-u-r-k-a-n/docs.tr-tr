 <span data-ttu-id="83f3a-104">Bu hata, bir değişkeni ref bağımsız değişkeni olarak statik bir salt alanın üyesi olan bir işleve geçirirseniz oluşur.</span><span class="sxs-lookup"><span data-stu-id="83f3a-104">This error occurs if you pass a variable to a function that is a member of a static readonly field as a ref argument.</span></span> <span data-ttu-id="83f3a-105">Ref parametreleri işlev tarafından değiştirilebileceğinden, buna izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="83f3a-105">Since ref parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="83f3a-106">Bu hatayı gidermek için, alandaki **salt** salt anahtar sözcüğü kaldırın veya yalnızca okunan alanın üyelerini işleve geçirmeyin.</span><span class="sxs-lookup"><span data-stu-id="83f3a-106">To resolve this error, remove the **readonly** keyword on the field, or do not pass the members of the readonly field to the function.</span></span> <span data-ttu-id="83f3a-107">Örneğin, aşağıdaki örnekte gösterildiği gibi değiştirilebilir ve geçici bir ref bağımsız değişkeni olarak geçebilecek geçici bir değişken oluşturmayı deneyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="83f3a-107">For example, you might try creating a temporary variable which can be modified and passing the temporary as a ref argument, as shown in the following example.</span></span>

```csharp  
  {  
    public int i;  
  }

{
  public static readonly Inner inner = new Inner();  
}

{  
   static void f(ref int iref)  
   {  
   }

      // int tmp = Outer.inner.i;  
      // f(ref tmp);  
   }  
}  
```
