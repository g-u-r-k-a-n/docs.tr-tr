# <a name="implements-not-valid-in-modules"></a>' Implements ' modüllerde geçerli değildir

`Implements`Bir modül içinde bir ifade veya yan tümce oluşur. Modüller arabirimler uygulayamaz.  
  
 **Hata kimliği:** BC30231  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Implements`Deyimi veya yan tümcesini kaldırın ya da modülü bir sınıf olarak yeniden yazın.
