# <a name="compiler-error-cs1655"></a><span data-ttu-id="6a389-103">Derleyici hatası CS1655</span><span class="sxs-lookup"><span data-stu-id="6a389-103">Compiler Error CS1655</span></span>

<span data-ttu-id="6a389-104">' Değişken ' alanları bir ' ReadOnly değişken türü ' olduğundan ref veya out bağımsız değişkeni olarak geçirilemez</span><span class="sxs-lookup"><span data-stu-id="6a389-104">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="6a389-105">Bu hata, bir [foreach](../language-reference/keywords/foreach-in.md) değişkeninin bir üyesini, bir [using](../language-reference/keywords/using-statement.md) değişkenini veya bir işlev için bir ref veya out bağımsız değişkeni olarak [sabit](../language-reference/keywords/fixed-statement.md) bir değişkeni geçirmeye çalışıyorsanız oluşur.</span><span class="sxs-lookup"><span data-stu-id="6a389-105">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="6a389-106">Bu değişkenler bu bağlamlarda salt okunurdur olarak kabul edildiğinden buna izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="6a389-106">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="6a389-107">Aşağıdaki örnek CS1655 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6a389-107">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
