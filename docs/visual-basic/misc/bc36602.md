# <a name="readonly-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a><span data-ttu-id="587f3-103">' ReadOnly ' değişkeni, bir oluşturucunun içindeki lambda ifadesinde bir atamanın hedefi olamaz</span><span class="sxs-lookup"><span data-stu-id="587f3-103">'ReadOnly' variable cannot be the target of an assignment in a lambda expression inside a constructor</span></span>

<span data-ttu-id="587f3-104">`ReadOnly`Lambda ifadesinin içinden bir değişkene başvurmadı; buna izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="587f3-104">You have referred to a `ReadOnly` variable from within a lambda expression, which is not permitted.</span></span> <span data-ttu-id="587f3-105">Aşağıdaki kod, `ReadOnly` değişkeni `m` bir parametreye bağımsız değişken olarak göndererek bu hataya neden olur `ByRef` .</span><span class="sxs-lookup"><span data-stu-id="587f3-105">The following code causes this error by sending the `ReadOnly` variable `m` as the argument to a `ByRef` parameter.</span></span>  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 <span data-ttu-id="587f3-106">**Hata kimliği:** BC36602</span><span class="sxs-lookup"><span data-stu-id="587f3-106">**Error ID:** BC36602</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="587f3-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="587f3-107">To correct this error</span></span>  
  
- <span data-ttu-id="587f3-108">Mümkünse, `n` `Test` `ByVal` aşağıdaki kodda gösterildiği gibi işlevdeki parametresini bir parametreye değiştirin.</span><span class="sxs-lookup"><span data-stu-id="587f3-108">If possible, change parameter `n` in function `Test` to a `ByVal` parameter, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="587f3-109">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="587f3-109">To correct this error</span></span>  
  
- <span data-ttu-id="587f3-110">`ReadOnly`Değişkenini `m` oluşturucuda yerel bir değişkene atayın ve `m` aşağıdaki kodda gösterildiği gibi yerel değişkenini yerine kullanın.</span><span class="sxs-lookup"><span data-stu-id="587f3-110">Assign the `ReadOnly` variable `m` to a local variable in the constructor, and use the local variable in place of `m`, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```
