# <a name="compiler-error-cs0711"></a><span data-ttu-id="90f90-103">Derleyici hatası CS0711</span><span class="sxs-lookup"><span data-stu-id="90f90-103">Compiler Error CS0711</span></span>

<span data-ttu-id="90f90-104">Statik sınıflar yıkıcı içeremez</span><span class="sxs-lookup"><span data-stu-id="90f90-104">Static classes cannot contain destructors</span></span>  
  
 <span data-ttu-id="90f90-105">Statik bir sınıf örneği oluşturulamıyor, bu nedenle oluşturucular veya Yıkıcılar için gerekli değildir.</span><span class="sxs-lookup"><span data-stu-id="90f90-105">A static class cannot be instantiated, hence it has no need for constructors or destructors.</span></span> <span data-ttu-id="90f90-106">Bu hatadan kaçınmak için, statik sınıflardan tüm yıkıcıları kaldırın veya gerçekten örnek oluşturup yok etmek istiyorsanız, sınıfı statik olmayan hale getirin.</span><span class="sxs-lookup"><span data-stu-id="90f90-106">To avoid this error, remove any destructors from static classes, or, if you really want to construct and destroy instances, make the class non-static.</span></span>  
  
 <span data-ttu-id="90f90-107">Aşağıdaki örnek CS0711 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="90f90-107">The following sample generates CS0711:</span></span>  
  
```csharp  
// CS0711.cs  
public static class C  
{  
   ~C()  // CS0711  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
