# <a name="compiler-error-cs0013"></a>Derleyici hatası CS0013
' ModelStore.dll ' dosyasına meta veriler yazılırken beklenmeyen hata--' daha fazla kullanıcı dizesi oluşturmak için mantıksal alan kalmadı.  
  
 .NET çalışma zamanı meta verileri yaymadı. Yolun doğru olduğundan ve diskin dolu olmadığından emin olmak için denetleyin. Sorun devam ederse, Visual Studio ve/veya .NET 'i onarmanız veya yeniden yüklemeniz gerekebilir.
