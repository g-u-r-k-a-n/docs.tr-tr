# <a name="compiler-error-cs1647"></a>Derleyici hatası CS1647

Bir ifade, ' Code ' yakınında derlemek için çok uzun veya karmaşık

Derleyicide kodunuzun işlendiği bir yığın taşması vardı. Bu hatayı çözmek için kodunuzu kolaylaştırın. Kodunuz geçerliyse, [Geliştirici topluluğu](https://aka.ms/feedback/report?space=61) sitesinde veya [StackOverflow](https://stackoverflow.com/)yardım alın.
