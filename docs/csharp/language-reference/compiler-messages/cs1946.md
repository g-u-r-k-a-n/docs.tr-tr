# <a name="compiler-error-cs1946"></a><span data-ttu-id="b2c5b-102">Derleyici Hatası CS1946</span><span class="sxs-lookup"><span data-stu-id="b2c5b-102">Compiler Error CS1946</span></span>

<span data-ttu-id="b2c5b-103">Anonim yöntem ifadesi bir ifade ağacına dönüştürülemez.</span><span class="sxs-lookup"><span data-stu-id="b2c5b-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="b2c5b-104">Anonim bir [yöntem](../operators/delegate-operator.md) bir deyim kümesini temsil eder, ancak bir ifade ağacı bir deyim içermemelidir.</span><span class="sxs-lookup"><span data-stu-id="b2c5b-104">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="b2c5b-105">Bu nedenle anonim bir yöntem bir ifade ağacı tarafından temsil edilemez.</span><span class="sxs-lookup"><span data-stu-id="b2c5b-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="b2c5b-106">Bu hatayı düzeltmek için anonim yöntemi [lambda ifadesine](../../programming-guide/statements-expressions-operators/lambda-expressions.md)değiştirin.</span><span class="sxs-lookup"><span data-stu-id="b2c5b-106">To correct this error, change the anonymous method to a [lambda expression](../../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="b2c5b-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="b2c5b-107">Example</span></span>

<span data-ttu-id="b2c5b-108">Aşağıdaki örnek CS1946 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b2c5b-108">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
