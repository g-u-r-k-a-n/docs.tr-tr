<span data-ttu-id="75515-103">Çağırma deyiminizden erişemezsiniz bir dönüş türüne sahip bir işlev çağırılır.</span><span class="sxs-lookup"><span data-stu-id="75515-103">You have called a function that has a return type that you cannot access from the calling statement.</span></span> <span data-ttu-id="75515-104">Örneğin, aşağıdaki kodda, dönüş türü `PrivateType`, sınıf `TestClass` `Private` erişim değiştiricisiyle bildirildiği için `Main` çağrısı `PublicMethod` başarısız olur.</span><span class="sxs-lookup"><span data-stu-id="75515-104">For example, in the following code, the call from `Main` to `PublicMethod` fails because the return type, `PrivateType`, is declared with the `Private` access modifier in class `TestClass`.</span></span> <span data-ttu-id="75515-105">Bu nedenle, `PrivateType` erişilebilen bağlam `TestClass` sınırlıdır.</span><span class="sxs-lookup"><span data-stu-id="75515-105">Therefore, the context within which `PrivateType` can be accessed is limited to `TestClass`.</span></span>

```vb
Class TestClass

    Dim pT As New PrivateType

    Private Class PrivateType
    End Class

    '' A corresponding error is returned in this line: 'PublicMethod
    '' cannot expose 'PrivateType' in namespace 'ConsoleApplication1'
    '' through class 'TestClass'.
    'Public Function PublicMethod() As PrivateType
    '    Return Nothing
    'End Function

End Class

Module Module1

    Sub Main()

        Dim tc As TestClass
        '' This error occurs here, because the data type returned by
        '' PublicMethod()is declared Private in class TestClass and
        '' cannot be accessed from here.
        'Console.WriteLine(tc.PublicMethod())

    End Sub

End Module
```
