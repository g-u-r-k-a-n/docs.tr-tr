# <a name="compiler-error-cs0549"></a><span data-ttu-id="c8e3d-103">Derleyici hatası CS0549</span><span class="sxs-lookup"><span data-stu-id="c8e3d-103">Compiler Error CS0549</span></span>

<span data-ttu-id="c8e3d-104">' function ', ' class ' Sealed sınıfındaki yeni bir sanal üyedir</span><span class="sxs-lookup"><span data-stu-id="c8e3d-104">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="c8e3d-105">[Sealed](../language-reference/keywords/sealed.md)bir[sınıf](../language-reference/keywords/class.md) temel sınıf olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="c8e3d-105">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="c8e3d-106">Bu nedenle, korumalı sınıfta sanal bir metoda sahip olmak işe yaramaz.</span><span class="sxs-lookup"><span data-stu-id="c8e3d-106">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="c8e3d-107">Aşağıdaki örnek CS0549 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c8e3d-107">The following sample generates CS0549:</span></span>  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
