# <a name="compiler-warning-level-3-cs0642"></a>Derleyici Uyarısı (düzey 3) CS0642

Olası hatalı boş ifade  
  
 Koşullu bir deyimden sonra noktalı virgül, kodunuzun istenenden farklı şekilde yürütülmesine neden olabilir.  
  
 **/Nowarn** derleyici seçeneğini veya `#pragmas warning` Bu uyarıyı devre dışı bırakmak için bkz. [-nowarn (C# derleyici seçenekleri)](../language-reference/compiler-options/nowarn-compiler-option.md) veya daha fazla bilgi için [#pragma uyarısı](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) .  
  
 Aşağıdaki örnek CS0642 oluşturur:  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
