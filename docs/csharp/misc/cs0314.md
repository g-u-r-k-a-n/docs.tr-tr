## <a name="example"></a><span data-ttu-id="e35db-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="e35db-109">Example</span></span>  
 <span data-ttu-id="e35db-110">Aşağıdaki kod CS0314 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e35db-110">The following code generates CS0314:</span></span>

```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.

{ }  
```
