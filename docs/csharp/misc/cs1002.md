# <a name="compiler-error-cs1002"></a>Derleyici Hatası CS1002
; Beklenen  
  
 Derleyici eksik bir yarım kolon algılandı. C#'daki her ifadenin sonunda bir yarı kolon gereklidir. Bir deyim birden fazla satıra yayılabilir.  
  
 Aşağıdaki örnek CS1002 oluşturur:  
  
```csharp  
// CS1002.cs  
namespace x  
{  
   abstract public class clx  
   {  
      int i   // CS1002, missing semicolon  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
