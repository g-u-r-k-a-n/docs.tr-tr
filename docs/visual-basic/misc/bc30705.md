# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>Başvurulan ' \<objectname> ' nesnesi ' Nothing ' değerine sahip
Başvurulan nesne ayarlanmadı veya değere ayarlandı `Nothing` .  
  
 **Hata kimliği:** BC30705  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesnenin adının doğru yazıldığını ve nesnenin ayarlandığını doğrulayın.
