# <a name="compiler-error-cs0004"></a>Derleyici hatası CS0004

Uyarı hata olarak değerlendirildi  
  
 Derleyicinin `/warnaserror` uyarıları hata olarak oluşturmasına neden olan derleyici seçeneğiyle derlediğiniz.
