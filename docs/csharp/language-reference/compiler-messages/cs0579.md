# <a name="compiler-error-cs0579"></a>Derleyici Hatası CS0579
Yinelenen 'öznitelik' özniteliği  
  
 Öznitelik **AllowMultiple=true'yu** [ÖznitelikKullanımı'nda](../../language-reference/attributes/general.md)belirtmediği sürece aynı özniteliği birden fazla belirtmek mümkün değildir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS0579 oluşturur.  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
