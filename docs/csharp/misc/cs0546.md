# <a name="compiler-error-cs0546"></a><span data-ttu-id="557d1-103">Derleyici hatası CS0546</span><span class="sxs-lookup"><span data-stu-id="557d1-103">Compiler Error CS0546</span></span>
<span data-ttu-id="557d1-104">' erişimci ': ' Property ' geçersiz kılınabilir bir set erişimcisine sahip olmadığından geçersiz kılınamaz</span><span class="sxs-lookup"><span data-stu-id="557d1-104">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="557d1-105">Erişimci geçersiz kılınamadığından, bir özellik için erişimci yöntemlerinden birini geçersiz kılma girişimi başarısız oldu.</span><span class="sxs-lookup"><span data-stu-id="557d1-105">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="557d1-106">Bu hata şu durumlarda oluşabilir:</span><span class="sxs-lookup"><span data-stu-id="557d1-106">This error can occur if:</span></span>  
  
- <span data-ttu-id="557d1-107">temel sınıf özelliği [sanal](../language-reference/keywords/virtual.md)olarak bildirilmemiş.</span><span class="sxs-lookup"><span data-stu-id="557d1-107">the base class property is not declared as [virtual](../language-reference/keywords/virtual.md).</span></span>  
  
- <span data-ttu-id="557d1-108">temel sınıf özelliği, geçersiz kılmaya çalıştığınız [Get](../language-reference/keywords/get.md) veya [set](../language-reference/keywords/set.md) erişimcisini bildirmiyor.</span><span class="sxs-lookup"><span data-stu-id="557d1-108">the base class property does not declare the [get](../language-reference/keywords/get.md) or [set](../language-reference/keywords/set.md) accessor you are trying to override.</span></span>  
  
 <span data-ttu-id="557d1-109">Temel sınıf özelliğini geçersiz kılmak istemiyorsanız, türetilmiş sınıftaki özelliğinden önce [New](../language-reference/keywords/new-modifier.md) anahtar sözcüğünü kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="557d1-109">If you do not want to override the base class property, you can use the [new](../language-reference/keywords/new-modifier.md) keyword before the property in derived class.</span></span>  
  
 <span data-ttu-id="557d1-110">Daha fazla bilgi için bkz. ## <a name="example"></a><span data-ttu-id="557d1-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="557d1-111">Example</span></span>  
 <span data-ttu-id="557d1-112">Aşağıdaki örnek, temel sınıf özelliği için bir set erişimcisi bildirmediğinden, CS0546 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="557d1-112">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
