# <a name="attribute-member-membername-cannot-be-the-target-of-an-assignment-because-it-is-not-declared-public"></a>' ' Öznitelik üyesi \<membername> , ' Public ' olarak bildirilmemiş olduğundan atamanın hedefi olamaz

Bir özniteliğin özel üyesine değer atama girişiminde bulunuldu.  
  
 **Hata kimliği:** BC31511  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Atamayı kaldırın.  
  
2. Geliştirdiğiniz özel öznitelikler kullanılıyorsa, öznitelik üyesinin erişim değiştiricisini olarak değiştirin `Public` .
