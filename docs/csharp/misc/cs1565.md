# <a name="compiler-error-cs1565"></a>Derleyici hatası CS1565

Çakışan seçenekler belirtildi: Win32 kaynak dosyası; Win32 simgesi  
  
 Aynı derlemede hem [**Win32Resource**](../language-reference/compiler-options/resources.md#win32resource) derleyici seçeneğini hem de [**Win32Icon**](../language-reference/compiler-options/resources.md#win32icon) derleyici seçeneğini belirtmek geçerli değildir.
