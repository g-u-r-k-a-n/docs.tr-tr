# <a name="error-number-must-be-within-the-range-0-and-65535"></a>Hata numarası 0 ve 65535 aralığında olmalıdır

Bir hata numarası, 0 ile 65535 arasında izin verilen değerin dışında kalıyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sayının geçerli Aralık içinde olduğundan emin olun.
