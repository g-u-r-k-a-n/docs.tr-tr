# <a name="partial-method-methodname-cannot-use-the-implements-keyword"></a>' ' Kısmi yöntemi \<methodname> ' Implements ' anahtar sözcüğünü kullanamaz
Kısmi yöntem tanımı bir arabirimi veya arabirim üyesini uygulayamaz.  
  
 **Hata kimliği:** BC31439  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Implements`İfadeyi kısmi yöntem tanımından kaldırın.
