# <a name="compiler-error-cs0712"></a>Derleyici hatası CS0712
' Static Class ' statik sınıfının örneği oluşturulamaz  
  
 Statik sınıfların örnekleri oluşturmak mümkün değildir. Statik sınıflar statik alanlar ve Yöntemler içerecek şekilde tasarlanmıştır, ancak örneklenemez.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0712 oluşturur:  
  
```csharp  
// CS0712.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
    public static void Main()  
    {  
        SC sc = new SC();  // CS0712  
    }  
}  
```
