# <a name="withevents-variable-does-not-raise-any-events"></a>' WithEvents ' değişkeni hiçbir olay oluşturmaz

Anahtar sözcüğüyle belirtilen nesne değişkeni, `WithEvents` olayları tetiklebilecek bir sınıf örneğine başvurmuyor.  
  
 **Hata kimliği:** BC30591  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Nesne değişkeninin olayları yükseltemeyen bir sınıf örneğine başvurduğundan emin olun.
