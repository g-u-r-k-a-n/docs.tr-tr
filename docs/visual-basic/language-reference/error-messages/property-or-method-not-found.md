# <a name="property-or-method-not-found"></a>Özellik veya yöntem bulunamadı
Başvurulan nesne yöntemi veya nesne özelliği tanımlı değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesnenin adını yanlış yazmış olabilirsiniz. Bir nesne için tanımlanan özellikleri ve yöntemleri görmek için Nesne Tarayıcısı görüntüleyin. Kullanılabilir özelliklerin ve yöntemlerin bir listesini görüntülemek için uygun nesne kitaplığını seçin.
