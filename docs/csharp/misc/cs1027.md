# <a name="compiler-error-cs1027"></a>Derleyici hatası CS1027

\#endif yönergesi bekleniyor  
  
 `#endif`Belirtilen yönerge için eşleşen bir [Önişlemci yönergesi](../language-reference/preprocessor-directives/index.md) bulunamadı `#if` . Ya da, `#endregion` `#region` bir blok içinde eşleşen yönerge olmadığında derleyici bir yönerge buldu olabilir `#if` .  
  
 Aşağıdaki örnek CS1027 oluşturur:  
  
```csharp  
// CS1027.cs  
#if true   // CS1027, uncomment next line to resolve  
// #endif  
  
namespace x  
{  
   public class clx  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
