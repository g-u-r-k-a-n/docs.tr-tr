# <a name="compiler-error-cs2017"></a>Derleyici hatası CS2017

Modül veya kitaplık oluşturuluyorsa/Main belirtilemiyor  
  
 Bir [-target: Library](../language-reference/compiler-options/target-library-compiler-option.md)oluştururken bir ana giriş noktası belirtemezsiniz.  
  
 Aşağıdaki örnek CS2017 oluşturur:  
  
```csharp  
// CS2017.cs  
// compile with: /main:MyClass /target:library  
// CS2017 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
