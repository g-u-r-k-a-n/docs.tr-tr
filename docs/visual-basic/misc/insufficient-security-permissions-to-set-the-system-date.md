# <a name="insufficient-security-permissions-to-set-the-system-date"></a>Sistem tarihini ayarlamak için güvenlik izinleri yetersiz

Güvenlik izinleri sistem tarihini ayarlamak için yetersizdir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Güvenlik izinlerini, sistem tarihinin ayarlanmasına izin verecek şekilde değiştirin.
