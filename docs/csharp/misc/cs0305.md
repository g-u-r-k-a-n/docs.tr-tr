## <a name="example"></a><span data-ttu-id="aef86-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="aef86-107">Example</span></span>  

 <span data-ttu-id="aef86-108">Aşağıdaki örnek CS0305 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="aef86-108">The following sample generates CS0305.</span></span>  
  
```csharp  
// CS0305.cs  
public class MyList<T> {}  
public class MyClass<T> {}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
