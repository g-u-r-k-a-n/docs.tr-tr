# <a name="analytic-tracing-with-etw"></a>ETW ile Çözümleme İzleme

Windows Communication Foundation (WCF) analitik izleme, bir WCF hizmetinin yürütülmesi sırasında tanılama bilgilerini yakalamak için bir yol sunar. WCF analitik izleme olayları, bir üretim ortamında WCF hizmetleri sorunlarını gidermeye olanak tanımak için WCF yığınındaki anahtar noktalarında dağıtılır. WCF Hizmetleri için analitik izleme, [!INCLUDE[netfx_current_long](../../../../../includes/netfx-current-long-md.md)] Bu olaylar Windows Için olay izleme (ETW) oturumuna çok verimli bir şekilde yayıldığından, WCF hizmetlerini barındıran bir ürün sunucusunun performansı üzerinde en az etkisi vardır.
