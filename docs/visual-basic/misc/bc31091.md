# <a name="import-of-type-typename-from-assembly-or-module-name-failed"></a>' ' Türünün ' ' \<typename> bütünleştirilmiş kodundan veya modülünden içeri aktarılması \<name> başarısız oldu
Başvurulan tüm kitaplıkların birbirleriyle uyumlu olduğundan emin olun.  
  
 **Hata kimliği:** BC31091  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Başvurulan türü içeren dosyaların bozuk olmadığından emin olun.
