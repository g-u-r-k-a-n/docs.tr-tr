# <a name="compiler-warning-level-1-cs1683"></a>Derleyici Uyarısı (düzey 1) CS1683
'Tür Adı' türüne başvuru, bu derlemede tanımlandığını iddia eder, ancak kaynakta veya eklenen modüllerde tanımlanmamıştır  
  
 Bu hata, şu anda derlemekte olduğunuz derlemeye geri başvuru içeren bir derleme içe aktarırken oluşabilir, ancak derlenen derleme başvuruyla eşleşen hiçbir şey içermez. Bu duruma gelmenin bir yolu, başlangıçta içe aktarılan derlemenin başvuruda bulunduğu üyeyi içeren derlemenizi derlemektir. Ardından, yanlışlıkla içe aktarılan derlemenin başlettiği üyeleri kaldırarak derlemenizi güncellersiniz.
