# <a name="internal-compiler-error"></a>İç derleyici hatası

 **Hata kimliği:** BC30747  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Visual Studio’yu yeniden başlatın.
