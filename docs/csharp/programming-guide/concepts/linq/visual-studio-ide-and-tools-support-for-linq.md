# <a name="visual-studio-ide-and-tools-support-for-linq-c"></a>LINQ için Visual Studio IDE ve Araçlar Desteği (C#)
Visual Studio entegre geliştirme ortamı (IDE), LINQ uygulama geliştirmeyi destekleyen aşağıdaki özellikleri sağlar:  
  
## <a name="object-relational-designer"></a>Object Relational Designer  
 Object İlişkisel Tasarımcı, Temel veritabanındaki ilişkisel verileri temsil eden C# sınıfları oluşturmak için [LINQ'dan SQL](../../../../framework/data/adonet/sql/linq/index.md) uygulamalarına kadar kullanabileceğiniz bir görsel tasarım aracıdır. Daha fazla bilgi için [Visual Studio'da LINQ'dan SQL Tools'a](/visualstudio/data-tools/linq-to-sql-tools-in-visual-studio2)bakın.  
  
## <a name="sqlmetal-command-line-tool"></a>SQLMetal Komut Satırı Aracı  
 SQLMetal, linq uygulamalarında kullanılmak üzere varolan veritabanlarından sınıflar oluşturmak için yapı işlemlerinde kullanılabilecek bir komut satırı aracıdır. Daha fazla bilgi için [sqlmetal.exe (Kod Oluşturma Aracı)](../../../../framework/tools/sqlmetal-exe-code-generation-tool.md)bakın.  
  
## <a name="linq-aware-code-editor"></a>LINQ-Aware Kod Düzenleyicisi  
 C# kod düzenleyicisi LINQ'u IntelliSense ve biçimlendirme özellikleriyle kapsamlı bir şekilde destekler.  
  
## <a name="visual-studio-debugger-support"></a>Visual Studio Debugger Desteği  
 Visual Studio hata ayıklama sorgu ifadelerinin hata ayıklama destekler. Daha fazla bilgi için, hata [ayıklama LINQ](/visualstudio/debugger/debugging-linq)bakın.
