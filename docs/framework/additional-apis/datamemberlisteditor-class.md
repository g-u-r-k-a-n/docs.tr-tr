# <a name="datamemberlisteditor-class"></a>DataMemberListEditor Sınıfı

Değer kaynağını seçmek için nesnenin `DataSource` `DataSource` tüm özelliklerini listeleyerek, veriye bağlı nesnelerin (null olmayan özelliklere sahip nesneler) özelliklerini düzenlemek için açılır kullanıcı arabirimi sağlar.  
  
## <a name="syntax"></a>Sözdizimi
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> Sınıf `DataMemberListEditor` dahilidir ve doğrudan kodunuzda kullanılmamalıdır.
>
> Microsoft, hiçbir koşulda bir üretim uygulamasında bu sınıfın kullanımını desteklemez.
