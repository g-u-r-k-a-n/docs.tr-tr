# <a name="ca5403-do-not-hard-code-certificate"></a>CA5403: Sertifikayı sabit olarak kodlamayın

| | Değer |
|-|-|
| **Kural Kimliği** |CA5403|
| **Kategori** |[Güvenlik](security-warnings.md)|
| **Düzeltilme veya kırılmamış** |Kırılmamış|

## <a name="cause"></a>Nedeni

`data` `rawData` Veya oluşturucusunun veya parametresi, <xref:System.Security.Cryptography.X509Certificates.X509Certificate> <xref:System.Security.Cryptography.X509Certificates.X509Certificate2> aşağıdakilerden biri tarafından sabit kodludur:

- Bayt dizisi.
- Char dizisi.
- <xref:System.Convert.FromBase64String(System.String)?displayProperty=fullName>.
- Tüm aşırı yüklemeleri <xref:System.Text.Encoding.GetBytes%2A?displayProperty=fullName> .

## <a name="rule-description"></a>Kural açıklaması

Sabit kodlanmış sertifikanın özel anahtarı kolayca keşfedilir. Derlenmiş ikililer de olsa, kötü niyetli kullanıcıların sabit kodlanmış bir sertifikanın özel anahtarını çıkartmasına çok kolay. Özel anahtar tehlikeye düştüğünde, bir saldırgan bu sertifikayı taklit edebilir ve bu sertifika tarafından korunan tüm kaynaklar veya işlemler saldırgan tarafından kullanılabilir olacaktır.

## <a name="how-to-fix-violations"></a>İhlalleri çözme

- Uygulamanızı, Azure Key Vault gibi bir güvenli anahtar yönetimi sistemi kullanacak şekilde yeniden tasarlamayı düşünün.
- Kimlik bilgilerini ve sertifikaları, kaynak kodınızdan ayrı güvenli bir konumda saklayın.

## <a name="when-to-suppress-warnings"></a>Uyarıların ne zaman bastırılamıyor

Sabit kodlanmış veriler sertifikanın özel anahtarını içermiyorsa, bu kuraldan bir uyarıyı bastırmak güvenlidir. Örneğin, veriler bir `.cer` dosyadan yapılır. Sabit kodlama ortak sertifika bilgileri, süreleri dolduğunda veya iptal edildiğinde sertifika döndürmek için hala bir zorluk oluşturabilir.

## <a name="pseudo-code-examples"></a>Sözde kod örnekleri

### <a name="hard-coded-by-byte-array"></a>Bayt dizisi tarafından sabit kodlanmış

```csharp
using System.IO;
using System.Security.Cryptography.X509Certificates;

class ExampleClass
{
    public void ExampleMethod(string path)
    {
        byte[] bytes = new byte[] {1, 2, 3};
        File.WriteAllBytes(path, bytes);
        new X509Certificate2(path);
    }
}
```

### <a name="hard-coded-by-char-array"></a>Char dizisi tarafından sabit kodlanmış

```csharp
using System.IO;
using System.Security.Cryptography.X509Certificates;
using System.Text;

class ExampleClass
{
    public void ExampleMethod(byte[] bytes, string path)
    {
        char[] chars = new char[] { '1', '2', '3' };
        Encoding.ASCII.GetBytes(chars, 0, 3, bytes, 0);
        File.WriteAllBytes(path, bytes);
        new X509Certificate2(path);
    }
}
```

### <a name="hard-coded-by-frombase64string"></a>FromBase64String tarafından sabit kodlanmış

```csharp
using System;
using System.IO;
using System.Security.Cryptography.X509Certificates;

class ExampleClass
{
    public void ExampleMethod(string path)
    {
        byte[] bytes = Convert.FromBase64String("AAAAAaazaoensuth");
        File.WriteAllBytes(path, bytes);
        new X509Certificate2(path);
    }
}
```

### <a name="hard-coded-by-getbytes"></a>GetBytes tarafından sabit kodlanmış

```csharp
using System;
using System.IO;
using System.Security.Cryptography.X509Certificates;
using System.Text;

class ExampleClass
{
    public void ExampleMethod(string path)
    {
        byte[] bytes = Encoding.ASCII.GetBytes("AAAAAaazaoensuth");
        File.WriteAllBytes(path, bytes);
        new X509Certificate2(path);
    }
}
```

### <a name="solution"></a>Çözüm

```csharp
using System.IO;
using System.Security.Cryptography.X509Certificates;

class ExampleClass
{
    public void ExampleMethod(string path)
    {
        new X509Certificate2("Certificate.cer");
    }
}
```
