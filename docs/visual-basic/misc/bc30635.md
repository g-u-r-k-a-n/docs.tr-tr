# <a name="readonly-and-writeonly-cannot-be-combined"></a>' ReadOnly ' ve ' WriteOnly ' birleştirilemez

`ReadOnly`Ve `WriteOnly` değiştiricileri birbirini dışlıyor.  
  
 **Hata kimliği:** BC30635  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ya da `ReadOnly` `WriteOnly` ' ı belirtin, ancak ikisini birden belirtmeyin.
