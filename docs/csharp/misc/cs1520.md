# <a name="compiler-error-cs1520"></a><span data-ttu-id="6fa5f-102">Derleyici Hatası CS1520</span><span class="sxs-lookup"><span data-stu-id="6fa5f-102">Compiler Error CS1520</span></span>
<span data-ttu-id="6fa5f-103">Yöntemin bir dönüş türü olmalı</span><span class="sxs-lookup"><span data-stu-id="6fa5f-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="6fa5f-104">Bir sınıf, yapı veya arabirimde bildirilen bir yöntemin açık bir dönüş türü olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="6fa5f-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="6fa5f-105">Aşağıdaki örnekte, Kare yöntemi [dize](../language-reference/builtin-types/reference-types.md)bir dönüş değeri vardır:</span><span class="sxs-lookup"><span data-stu-id="6fa5f-105">In the following example, the Square method has a return value of [string](../language-reference/builtin-types/reference-types.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="6fa5f-106">Aşağıdaki örnek CS1520 üretir:</span><span class="sxs-lookup"><span data-stu-id="6fa5f-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="6fa5f-107">Alternatif olarak, bir oluşturucunun adının durumu aşağıdaki örnekte olduğu gibi sınıf veya yapı bildiriminden farklı olduğunda bu hatayla karşılaşılabilir.</span><span class="sxs-lookup"><span data-stu-id="6fa5f-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="6fa5f-108">Ad sınıf adı ile tam olarak aynı olmadığından, derleyici bunu bir oluşturucu olarak değil, normal bir yöntem olarak yorumlar ve hatayı üretir:</span><span class="sxs-lookup"><span data-stu-id="6fa5f-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
