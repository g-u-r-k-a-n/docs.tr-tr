<span data-ttu-id="d04f7-112"><xref:System.Net.Http.HttpClient?displayProperty=fullName>, Bir platforma özel işleyici ( <xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> veya) sağlarken <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName> `CheckCertificateRevocationList` `true` , özelliği, iptal edilen sertifikaların geçerli olarak kabul edilmesine izin verecek şekilde ayarlanmamış <xref:System.Net.Http.HttpClient> .</span><span class="sxs-lookup"><span data-stu-id="d04f7-112">Using <xref:System.Net.Http.HttpClient?displayProperty=fullName> while providing a platform specific handler (<xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> or <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName>) whose `CheckCertificateRevocationList` property is not set to `true` will allow revoked certificates to be accepted by the <xref:System.Net.Http.HttpClient> as valid.</span></span>
