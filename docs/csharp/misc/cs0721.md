# <a name="compiler-error-cs0721"></a>Derleyici hatası CS0721

' Type ': statik türler parametre olarak kullanılamaz  
  
 Statik bir tür parametre olarak anlamlı değildir. Statik türlerin hiçbir örneği oluşturulamayabilir, hiçbir örnek parametre olarak geçirilmeyebilir.  
  
 Aşağıdaki örnek CS0721 oluşturur:  
  
```csharp  
// CS0721.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public void F(SC sc)  // CS0721  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
