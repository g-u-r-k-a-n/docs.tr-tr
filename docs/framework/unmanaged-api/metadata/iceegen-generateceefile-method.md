# <a name="iceegengenerateceefile-method"></a>ICeeGen::GenerateCeeFile Yöntemi
Şu anda bu [ICeeGen](../../../../docs/framework/unmanaged-api/metadata/iceegen-interface.md) arabirimine yüklenmiş kod tabanını içeren bir kod tabanı dosyası oluşturur.  
  
 Bu yöntem kullanılmıyor ve kullanılmamalıdır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT GenerateCeeFile ();  
```  
  
## <a name="parameters"></a>Parametreler  
 `GenerateCeeFile` hiçbir parametre alır.
