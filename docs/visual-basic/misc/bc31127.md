# <a name="addhandler-is-already-declared"></a>' AddHandler ' zaten tanımlanmış
`AddHandler`Özel bir olay bildiriminde birden fazla bildirim görüntülenir. Bir `AddHandler` bildirim, bir olay işleyicisi eklemek için kullanılan yordamı bildirir.  
  
 **Hata kimliği:** BC31127  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ifadesini kaldırın `AddHandler` .
