# <a name="compiler-error-cs0558"></a>Derleyici hatası CS0558
Kullanıcı tanımlı işleç ' operator ' statik ve ortak olarak bildirilmelidir  
  
 Hem **statik** hem de **genel** erişim [değiştiricilerin](../language-reference/keywords/index.md) Kullanıcı tanımlı işleçlerinde belirtilmesi gerekir.  
  
 Aşağıdaki örnek CS0558 oluşturur:  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
