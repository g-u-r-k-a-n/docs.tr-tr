# <a name="compiler-error-cs1626"></a><span data-ttu-id="d6b9d-102">Derleyici Hatası CS1626</span><span class="sxs-lookup"><span data-stu-id="d6b9d-102">Compiler Error CS1626</span></span>
<span data-ttu-id="d6b9d-103">Yakalama yan tümcesi ile deneme bloğunun gövdesinde bir değer veremiyor</span><span class="sxs-lookup"><span data-stu-id="d6b9d-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="d6b9d-104">Try bloğuyla ilişkili bir catch yan tümcesi varsa, try bloğunda bir verim deyimine izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="d6b9d-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="d6b9d-105">Bu hatayı önlemek için verim deyimini try/catch/finally block'un dışına taşıyın veya catch bloğunu kaldırın.</span><span class="sxs-lookup"><span data-stu-id="d6b9d-105">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="d6b9d-106">Aşağıdaki örnek CS1626 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d6b9d-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
