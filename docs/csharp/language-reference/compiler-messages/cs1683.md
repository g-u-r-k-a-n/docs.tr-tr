# <a name="compiler-warning-level-1-cs1683"></a>Derleyici Uyarısı (düzey 1) CS1683

' Tür adı ' türüne başvuru bu derlemede tanımlandığını bildiriyor, ancak kaynakta veya herhangi bir eklenen modülde tanımlı değil  
  
 Bu hata, şu anda derlediğiniz derlemeye geri başvuru içeren bir derlemeyi içeri aktardığınızda meydana gelir, ancak derlenen derleme başvuru ile eşleşen hiçbir şey içermiyor. Bu duruma getirmenin bir yolu, başlangıçta içeri aktarılmakta olan derlemenin başvurduğu üyeyi içeren derlemenizi derlemeye yöneliktir. Daha sonra, içeri aktarılan derlemenin başvurduğu üyeleri yanlışlıkla kaldırarak derlemenizi güncelleştirmiş olursunuz.
