## <a name="example"></a><span data-ttu-id="216a8-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="216a8-109">Example</span></span>

```csharp  
{  
    interface IFace1 { }  
    interface IFace2 { }  
    partial class PartialBadBounds<T> where T : IFace1 { } // CS0265

}  
```
