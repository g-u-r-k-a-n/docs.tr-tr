# <a name="compiler-error-cs0233"></a><span data-ttu-id="118cc-102">Derleyici Hatası CS0233</span><span class="sxs-lookup"><span data-stu-id="118cc-102">Compiler Error CS0233</span></span>
<span data-ttu-id="118cc-103">'tanımlayıcı' önceden tanımlanmış bir boyutu yoktur, bu nedenle boyutlar yalnızca güvenli olmayan bir bağlamda kullanılabilir</span><span class="sxs-lookup"><span data-stu-id="118cc-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="118cc-104">[Güvenli olmayan](../keywords/unsafe.md) bağlam olmadan, işleç [boyutları](../operators/sizeof.md) yalnızca boyutu derleme zamanı sabiti olan türler için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="118cc-104">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="118cc-105">Bu hatayı alıyorsanız, güvenli olmayan bir bağlam kullanın.</span><span class="sxs-lookup"><span data-stu-id="118cc-105">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="118cc-106">Aşağıdaki örnek CS0233 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="118cc-106">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
