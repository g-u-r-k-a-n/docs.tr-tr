# <a name="select-statements-are-not-valid-in-the-immediate-window"></a>' Select ' deyimleri Komut penceresinde geçerli değildir

`Select` deyimler yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30718  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Select`Hata ayıklama kodundaki deyimleri kaldır.
