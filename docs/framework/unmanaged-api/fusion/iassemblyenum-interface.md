# <a name="iassemblyenum-interface"></a>IAssemblyEnum Arabirimi
`IAssemblyName` nesnelerinin bir dizisi için bir Numaralandırıcı temsil eder.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Clone Yöntemi](iassemblyenum-clone-method.md)|Bu `IAssemblyEnum` nesnesinin basit bir kopyasını oluşturur.|  
|[GetNextAssembly Yöntemi](iassemblyenum-getnextassembly-method.md)|Bu `IAssemblyEnum` nesnesinin içindeki bir sonraki `IAssemblyName` yönelik bir işaretçi alır.|  
|[Reset Yöntemi](iassemblyenum-reset-method.md)|Bu `IAssemblyEnum` nesnesini başlangıç konumuna sıfırlar.|
