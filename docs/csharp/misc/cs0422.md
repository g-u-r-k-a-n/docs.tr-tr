# <a name="compiler-warning-level-4-cs0422"></a>Derleyici Uyarısı (seviye 4) CS0422

/artımlı seçenek artık desteklenmeyecek

 Artımlı derleme (**/incr** veya **/artımlı**) C# 2.0'da desteklenmez.

> [!NOTE]
> Bu derleyici uyarısı artık Roslyn'de kullanılmaz.
