# <a name="compiler-error-cs0432"></a><span data-ttu-id="56137-103">Derleyici hatası CS0432</span><span class="sxs-lookup"><span data-stu-id="56137-103">Compiler Error CS0432</span></span>

<span data-ttu-id="56137-104">' Identifier ' diğer adı bulunamadı</span><span class="sxs-lookup"><span data-stu-id="56137-104">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="56137-105">Bu hata, diğer ad olmayan bir tanımlayıcının sağında "::" kullandığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="56137-105">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="56137-106">Hatayı gidermek için, bunun yerine "." kullanın.</span><span class="sxs-lookup"><span data-stu-id="56137-106">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="56137-107">Aşağıdaki örnek CS0432 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="56137-107">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
