# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>Hiçbir erişilebilir ' \<method> ' Bu bağımsız değişkenlerle çağrılamayacağından aşırı yükleme çözümlemesi başarısız oldu:\<error>
Aşırı yüklenmiş bir yönteme çağrı yaptınız, ancak derleyici eşleşen bir bağımsız değişken listesi olan bir yöntem bulamıyor.  
  
 **Hata kimliği:** BC30518  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sağladığınız bağımsız değişken listesini denetleyin.
