# <a name="exit-statements-are-not-valid-in-the-immediate-window"></a>' Exit ' deyimleri Komut penceresinde geçerli değildir
`Exit`deyimler yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30713  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Exit`Hata ayıklama kodundan deyimleri kaldır.
