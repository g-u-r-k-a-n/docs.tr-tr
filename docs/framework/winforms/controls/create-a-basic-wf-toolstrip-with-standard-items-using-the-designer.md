# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>Nasıl yapılır: Tasarımcı Kullanarak Standart Öğelerle Temel bir Windows Formları ToolStrip Oluşturma
Aşağıdaki yordamda, bir <xref:System.Windows.Forms.ToolStrip> oluşturma ve tipik görevleri temsil eden yedi <xref:System.Windows.Forms.ToolStripButton> denetimi ekleme gösterilmektedir.

## <a name="to-add-standard-items-in-the-designer"></a>Tasarımcıya standart öğeler eklemek için

1. <xref:System.Windows.Forms.ToolStrip> denetimi oluşturun.

2. <xref:System.Windows.Forms.ToolStrip>**ToolStrip görevler** bölmesini göstermek için, sağ üst köşesinde akıllı görev okuna tıklayın.

3. **ToolStrip görevleri** bölmesinde **Standart öğe Ekle**' yi seçin.
