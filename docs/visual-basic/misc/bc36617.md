# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Toplama işlevi adı bir tür karakteriyle birlikte kullanılamaz
Bir toplama işlevinin adının sonuna bir tür karakteri eklenmiştir.  
  
 **Hata kimliği:** BC36617  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Toplama işlevi adından tür karakterini kaldırın.
