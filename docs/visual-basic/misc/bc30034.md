# <a name="bracketed-identifier-is-missing-closing-"></a>Parantez içine alınmış tanımlayıcıda kapatma '] ' eksik
Kaçış adında parantez, eşleşen çiftler halinde gerçekleşmelidir.  
  
 **Hata kimliği:** BC30034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaçışın adının sonuna bir kapanış ayracı koyun.
