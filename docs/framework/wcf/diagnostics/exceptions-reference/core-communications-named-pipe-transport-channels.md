# <a name="core-communications-named-pipe-transport-channels"></a>Temel İletişimler: Adlandırılmış Kanal Taşıma Kanalları

Bu konu, adlandırılmış kanal taşıma kanalları tarafından oluşturulan tüm özel durumları listeler.  
  
## <a name="exception-list"></a>Özel durum listesi  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|PipeCantCloseWithPendingWrite|Kanala yazma işlemi beklenirken kanal kapatılamıyor.|  
|Ipereadpending|Kanal için bir okuma işlemi devam ediyor.|  
|PipeShutdownReadError|Kanal ' kapatması ' göstergesinin okuma işlemi başarısız oldu.|  
|PipeShutdownWriteError|Kanal ' kapatması ' göstergesinin yazma işlemi başarısız oldu.|  
|PipeWritePending|Kanal için bir yazma işlemi devam ediyor.|
