# <a name="typename-is-a-type-in-containername-and-cannot-be-used-as-an-expression"></a>' \<typename> ', ' ' içindeki bir tür \<containername> ve ifade olarak kullanılamaz

Kullanılan adlardan biri zaten bir türün adı olarak tanımlanmış ve bir ifadede kullanılamaz.  
  
 **Hata kimliği:** BC30691  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Üyeyi benzersiz şekilde adlandırın.  
  
- Üyenin adı etrafında köşeli ayraç kullanın.
