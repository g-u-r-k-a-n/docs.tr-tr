## <a name="pnrp-ids"></a><span data-ttu-id="cadb3-118">PNRP Tİp'leri</span><span class="sxs-lookup"><span data-stu-id="cadb3-118">PNRP IDs</span></span>  
 <span data-ttu-id="cadb3-119">![PNRP Kimliği](./media/fdc9e8a0-4a1c-488d-a019-bc3a1973220c.gif "fdc9e8a0-4a1c-488d-a019-bc3a1973220c")</span><span class="sxs-lookup"><span data-stu-id="cadb3-119">![PNRP ID](./media/fdc9e8a0-4a1c-488d-a019-bc3a1973220c.gif "fdc9e8a0-4a1c-488d-a019-bc3a1973220c")</span></span>  
  
 <span data-ttu-id="cadb3-120">PNRP T'leri aşağıdakilerden oluşur:</span><span class="sxs-lookup"><span data-stu-id="cadb3-120">PNRP IDs are composed of the following:</span></span>  
  
- <span data-ttu-id="cadb3-121">Eşler arası (P2P) kimliği olarak bilinen yüksek sıralı 128 bit, bitiş noktasına atanan bir eş adının karmasIdır.</span><span class="sxs-lookup"><span data-stu-id="cadb3-121">The high-order 128 bits, known as the peer-to-peer (P2P) ID, are a hash of a peer name assigned to the endpoint.</span></span> <span data-ttu-id="cadb3-122">Eş adı aşağıdaki biçimi vardır: *Authority.Classifier*.</span><span class="sxs-lookup"><span data-stu-id="cadb3-122">The peer name has the following format: *Authority.Classifier*.</span></span> <span data-ttu-id="cadb3-123">Güvenli adlar için *Otorite,* altıgen karakterdeki eş adının ortak anahtarının Güvenli Karma Algoritma 1 (SHA1) karmasıdır.</span><span class="sxs-lookup"><span data-stu-id="cadb3-123">For secured names, *Authority* is the Secure Hash Algorithm 1 (SHA1) hash of the public key of the peer name in hexadecimal characters.</span></span> <span data-ttu-id="cadb3-124">Güvenli olmayan adlar için *Otorite* tek karakter "0"dır.</span><span class="sxs-lookup"><span data-stu-id="cadb3-124">For unsecured names, the *Authority* is the single character "0".</span></span> <span data-ttu-id="cadb3-125">*Sınıflandırıcı,* uygulamayı tanımlayan bir dizedir.</span><span class="sxs-lookup"><span data-stu-id="cadb3-125">*Classifier* is a string that identifies the application.</span></span> <span data-ttu-id="cadb3-126">Hiçbir eş adı sınıflandırıcı, `null` sonlandırıcı da dahil olmak üzere 149 karakterden uzun olamaz.</span><span class="sxs-lookup"><span data-stu-id="cadb3-126">No peer name classifier can be greater than 149 characters long, including the `null` terminator.</span></span>  
  
- <span data-ttu-id="cadb3-127">Düşük sıralı 128 bit, aynı buluttaki aynı P2P KIMLIĞI'nin farklı örneklerini tanımlayan oluşturulan bir sayı olan Hizmet Konumu için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="cadb3-127">The low-order 128 bits are used for the Service Location, which is a generated number that identifies different instances of the same P2P ID in the same cloud.</span></span>  
  
 <span data-ttu-id="cadb3-128">P2P Kimliği ve Hizmet Konumu'nun bu birleşimi, birden çok PNRP kimliğinin tek bir bilgisayardan kaydedilmesine olanak tanır.</span><span class="sxs-lookup"><span data-stu-id="cadb3-128">This combination of P2P ID and Service Location allows multiple PNRP IDs to be registered from a single computer.</span></span>
