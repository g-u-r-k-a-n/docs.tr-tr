# <a name="compiler-error-cs0459"></a><span data-ttu-id="d4219-103">Derleyici hatası CS0459</span><span class="sxs-lookup"><span data-stu-id="d4219-103">Compiler Error CS0459</span></span>

<span data-ttu-id="d4219-104">Salt okunurdur bir yerel değişkenin adresi alınamaz</span><span class="sxs-lookup"><span data-stu-id="d4219-104">Cannot take the address of a read-only local variable</span></span>

 <span data-ttu-id="d4219-105">C# dilinde salt okunurdur yerel değişkenler üreten üç yaygın senaryo vardır: `foreach` , `using` , ve `fixed` .</span><span class="sxs-lookup"><span data-stu-id="d4219-105">There are three common scenarios in the C# language that generate read-only local variables: `foreach`, `using`, and `fixed`.</span></span> <span data-ttu-id="d4219-106">Bu durumların her birinde, salt yazılır yerel değişkene yazma veya adresini alma izniniz yoktur.</span><span class="sxs-lookup"><span data-stu-id="d4219-106">In each of these cases, you are not allowed to write to the read-only local variable, or to take its address.</span></span> <span data-ttu-id="d4219-107">Bu hata, derleyici bir salt okunurdur yerel değişkenin adresini almaya çalışırken oluşur.</span><span class="sxs-lookup"><span data-stu-id="d4219-107">This error is generated when the compiler realizes you are trying to take the address of a read-only local variable.</span></span>

## <a name="example"></a><span data-ttu-id="d4219-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="d4219-108">Example</span></span>

 <span data-ttu-id="d4219-109">Aşağıdaki örnek, bir `foreach` döngüsünde ve bir ifade bloğunda salt okunurdur bir yerel değişkenin adresini almak için bir girişim YAPıLDıĞıNDA CS0459 oluşturur `fixed` :</span><span class="sxs-lookup"><span data-stu-id="d4219-109">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block:</span></span>

```csharp
// CS0459.cs
// compile with: /unsafe

class Program
{
    public unsafe void M1()
    {
        int[] ints = new int[] { 1, 2, 3 };
        foreach (int i in ints)
        {
            int *j = &i;  // CS0459
        }

        fixed (int *i = &_i)
        {
            int **j = &i;  // CS0459
        }
    }

    private int _i = 0;
}
```
