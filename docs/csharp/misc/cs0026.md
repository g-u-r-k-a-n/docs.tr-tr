## <a name="example"></a><span data-ttu-id="4b208-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="4b208-108">Example</span></span>  
 <span data-ttu-id="4b208-109">Aşağıdaki örnek CS0026 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4b208-109">The following example generates CS0026:</span></span>

```csharp  
   public static int i = 0;  
  
   public static void Main()  
   {

      // i = i + 1;  
   }  
}  
```
