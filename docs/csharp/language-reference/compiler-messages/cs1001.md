# <a name="compiler-error-cs1001"></a><span data-ttu-id="74b79-102">Derleyici Hatası CS1001</span><span class="sxs-lookup"><span data-stu-id="74b79-102">Compiler Error CS1001</span></span>

<span data-ttu-id="74b79-103">Tanımlayıcı bekleniyor</span><span class="sxs-lookup"><span data-stu-id="74b79-103">Identifier expected</span></span>

<span data-ttu-id="74b79-104">Bir tanımlayıcı sağlamadınız.</span><span class="sxs-lookup"><span data-stu-id="74b79-104">You did not supply an identifier.</span></span> <span data-ttu-id="74b79-105">Tanımlayıcı, sağladığınız sınıfın, yapının, Namespace, yöntemin, değişkenin ve bunun adıdır.</span><span class="sxs-lookup"><span data-stu-id="74b79-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>

<span data-ttu-id="74b79-106">Aşağıdaki örnek basit bir sınıf bildirir ancak sınıfa bir ad vermez:</span><span class="sxs-lookup"><span data-stu-id="74b79-106">The following example declares a simple class but does not give the class a name:</span></span>

```csharp
public class //CS1001
{
    public int Num { get; set; }
    void MethodA() {}
}
```

<span data-ttu-id="74b79-107">Aşağıdaki örnek CS1001 oluşturur çünkü bir numaralandırma bildirirken, üyeleri belirtmeniz gerekir:</span><span class="sxs-lookup"><span data-stu-id="74b79-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>

```csharp
public class Program
{
    enum Colors
    {
        'a', 'b' // CS1001, 'a' is not a valid int identifier
        // The following line shows examples of valid identifiers:
        // Blue, Red, Orange
    };

    public static void Main()
    {
    }
}
```

<span data-ttu-id="74b79-108">Derleyici, örneğin bir arabirim tanımında kullanılmasa bile parametre adları gereklidir.</span><span class="sxs-lookup"><span data-stu-id="74b79-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="74b79-109">Bu parametreler, arabirimi kullanan programcılar, Parametrelerin anlamları hakkında bir şeyi öğrenmesi için gereklidir.</span><span class="sxs-lookup"><span data-stu-id="74b79-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>

```csharp
interface IMyTest
{
    void TestFunc1(int, int);  // CS1001
    // Use the following line instead:
    // void TestFunc1(int a, int b);
}

class CMyTest : IMyTest
{
    void IMyTest.TestFunc1(int a, int b)
    {
    }
}
```
