- <span data-ttu-id="4206c-115">Mümkünse, yordam parametresiyle aynı türde bir çağırma bağımsız değişkeni kullanın, bu nedenle Visual Basic herhangi bir dönüştürme yapması gerekmez.</span><span class="sxs-lookup"><span data-stu-id="4206c-115">If possible, use a calling argument of the same type as the procedure parameter, so Visual Basic does not need to do any conversion.</span></span>

- <span data-ttu-id="4206c-117">Çağırma bağımsız değişkenine bir değer döndürmenize gerek duyuyorsanız, Visual Basic çağıran bağımsız değişken türüne geri dönüştürebilmesi için ters dönüştürme işlecini tanımlayın.</span><span class="sxs-lookup"><span data-stu-id="4206c-117">If you need to return a value into the calling argument, define the reverse conversion operator so Visual Basic can convert back to the calling argument type.</span></span>
