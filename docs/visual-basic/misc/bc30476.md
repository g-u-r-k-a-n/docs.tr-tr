# <a name="withevents-variables-cannot-be-typed-as-arrays"></a>' WithEvents ' değişkenleri dizi olarak türlenemez
Kullanarak bir dizi bildirmeye çalıştınız `WithEvents` . Kullanarak istediğiniz sayıda bağımsız değişken bildirebilirsiniz `WithEvents` , ancak bu şekilde dizileri bildiremezsiniz.  
  
 **Hata kimliği:** BC30476  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Değişkenleri ayrı ayrı bildirin.
