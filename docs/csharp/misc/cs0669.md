# <a name="compiler-error-cs0669"></a><span data-ttu-id="e8454-103">Derleyici hatası CS0669</span><span class="sxs-lookup"><span data-stu-id="e8454-103">Compiler Error CS0669</span></span>

<span data-ttu-id="e8454-104">ComImport özniteliğine sahip bir sınıfın Kullanıcı tanımlı Oluşturucusu olamaz.</span><span class="sxs-lookup"><span data-stu-id="e8454-104">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="e8454-105">Ortak dil çalışma zamanındaki COM birlikte çalışma katmanı, [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) sınıfları için Oluşturucu sağlar.</span><span class="sxs-lookup"><span data-stu-id="e8454-105">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="e8454-106">Sonuç olarak, bir COM nesnesi çalışma zamanında yönetilen bir nesne olarak kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="e8454-106">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="e8454-107">Aşağıdaki örnek CS0669 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e8454-107">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
