# <a name="a-delimiter-cannot-be-nothing-or-an-empty-string"></a>Bir sınırlayıcı Nothing veya boş dize olamaz

`TextFieldParser` `Delimiters` Özelliği, `Nothing` veya boş `String` ("") olarak ayarlandığı için dosyadan okuyamıyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İçin geçerli bir değer sağlayın `Delimiters` .
