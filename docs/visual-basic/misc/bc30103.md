# <a name="-requires-its-left-operand-to-have-a-type-parameter-class-or-interface-type-but-this-operand-has-the-type-type"></a>'! ', sol işlenenin bir tür parametresi, sınıf veya arabirim türünde olmasını gerektirir, ancak bu işlenen ' ' türüne sahip \<type>

Bir nokta () yerine ünlem işareti () kullanan sözlük üyesi erişimi, `!` `.` yalnızca bir sınıf veya arabirim üzerinde geçerlidir.  
  
 **Hata kimliği:** BC30103  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ünlem işaretinin solundaki ifadeyi tanımlanmış bir sınıf veya arabirim türü olarak değerlendirilen bir ile değiştirin.
