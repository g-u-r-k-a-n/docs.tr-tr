# <a name="goto-labelname-is-not-valid-because-labelname-is-inside-a-synclock-statement-that-does-not-contain-this-statement"></a>' \<labelname> \<labelname> ' Bu ifadeyi içermeyen bir ' SyncLock ' ifadesinin içinde olduğundan ' goto ' geçerli değil

Bir bloğa dallandırılamıyor `SyncLock` .  
  
 **Hata kimliği:** BC30755  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Etiketi bloğundan önce yeniden yapılandırmak için kodunuzu yeniden yapılandır `SyncLock` .
