# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Sorgu ifadeleri hata ayıklama pencereleri içinde geçerli değildir

Hata ayıklama oturumu sırasında **Watch** veya **anında** pencereye bir LINQ sorgusu girildi. Sorgular hata ayıklama penceresinde kullanılamaz.  
  
 **Hata kimliği:** BC36609
