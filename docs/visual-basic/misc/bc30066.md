# <a name="exit-property-is-not-valid-in-a-function-or-sub"></a>' Exit Property ', Function veya Sub içinde geçerli değildir

`Exit Property``Function`yordamda veya `Sub` yordamda görüntülenir. Bir `Exit` deyimin gerçekleştiği blokla eşleşmesi gerekir.  
  
 **Hata kimliği:** BC30066  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öğesini `Exit Property` `Exit Function` veya `Exit Sub` ifadesiyle uygun şekilde değiştirin.
