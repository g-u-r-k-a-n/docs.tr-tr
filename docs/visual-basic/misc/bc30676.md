# <a name="eventname-is-not-an-event-of-containername"></a>' \<eventname> ', ' ' bir olay değil \<containername>

Belirtilen olay, nesne için bildirilmemiş.  
  
 **Hata kimliği:** BC30676  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Olay adının yazımını denetleyin.  
  
2. Doğru nesneye erişmede emin olun. Nesneye tam nitelikli başvuruları kullanın veya `Imports` gerekirse uygun ad alanını içeri aktarmak için ifadesini kullanın.
