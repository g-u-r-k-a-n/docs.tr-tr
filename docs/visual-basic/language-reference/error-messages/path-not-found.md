# <a name="path-not-found"></a>Yol bulunamadı

Dosya erişimi veya disk erişimi işlemi sırasında, işletim sistemi belirtilen yolu bulamadı. Dosya yolu, sürücü belirtimini ve dosyayı bulmak için geçmesi gereken dizin ve alt dizinleri içerir. Yol göreli veya mutlak olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yolu doğrulayın ve belirtin.
