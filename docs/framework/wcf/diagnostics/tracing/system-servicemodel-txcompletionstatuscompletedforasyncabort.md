# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
Belirtilen işlem için belirtilen işlem zaman uyumsuz iptal nedeniyle tamamlandı.  
  
## <a name="description"></a>Açıklama  
 Geçerli işlem, başka bir katılımcının Iptali, zaman aşımları veya bir işlemin katılımcısı içinde başka bir hata nedeniyle sonlandırıldı.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Bu iptal beklenmiyorsa, İptalin gerçek nedenini öğrenmek için tüm sistem günlüklerine bakın.
