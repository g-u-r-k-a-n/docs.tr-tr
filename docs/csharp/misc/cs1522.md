<span data-ttu-id="0fbc9-104">Boş Switch bloğu</span><span class="sxs-lookup"><span data-stu-id="0fbc9-104">Empty switch block</span></span>  
  
 <span data-ttu-id="0fbc9-105">Derleyici, **Case** veya **Default** deyimleri olmayan bir [Switch](../language-reference/keywords/switch.md) bloğu algıladı.</span><span class="sxs-lookup"><span data-stu-id="0fbc9-105">The compiler detected a [switch](../language-reference/keywords/switch.md) block with no **case** or **default** statement.</span></span> <span data-ttu-id="0fbc9-106">Bir `switch` blok bir veya daha fazla **Case** veya **Default** deyimi içermelidir.</span><span class="sxs-lookup"><span data-stu-id="0fbc9-106">A `switch` block must have one or more **case** or **default** statements.</span></span>  
  
 <span data-ttu-id="0fbc9-107">Aşağıdaki örnek CS1522 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0fbc9-107">The following sample generates CS1522:</span></span>  
  
```csharp

// compile with: /W:1  
using System;  
class x  
{  
   public static void Main()  
   {  
      int i = 6;  
  
      switch(i)   // CS1522  
      {  
         // add something to the switch block, for example:  
         /*  
         case (5):  
            Console.WriteLine("5");  
            return;  
         default:  
            Console.WriteLine("not 5");  
            return;  
         */  
      }  
   }  
}  
```
