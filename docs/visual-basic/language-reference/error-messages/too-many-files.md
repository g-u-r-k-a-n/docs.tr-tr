# <a name="too-many-files"></a>Çok fazla sayıda dosya
Kök dizinde, işletim sisteminin izin verdiğinden daha fazla dosya oluşturuldu veya CONFIG 'inizdeki **dosyalar =** ayarında belirtilen sayıdan daha fazla dosya açıldı. SYS dosyası.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Programınız kök dizindeki dosyaları açıyor, kapatıyor veya kaydediyor ise, programınızı bir alt dizin kullanacak şekilde değiştirin.  
  
2. CONFIG 'inizdeki **Files =** ayarında belirtilen dosya sayısını artırın. SYS dosyası ve bilgisayarınızı yeniden başlatın.
