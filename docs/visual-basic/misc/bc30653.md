# <a name="reference-required-to-module-modulename-containing-the-type-membername"></a>' ' \<modulename> Türünü içeren ' ' modülüne başvuru gerekiyor \<membername>

' ' Türünü içeren ' ' modülüne başvuru gerekiyor \<modulename> \<membername> . Projenize bir tane ekleyin.  
  
 Proje, proje başvurularına dahil olmayan bir derlemeden başka bir türe bağımlı olan bir tür içerir.  
  
 **Hata kimliği:** BC30653  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Projedeki bağımlı türler için bir proje başvurusu ekleyin.
