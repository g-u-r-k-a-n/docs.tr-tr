# <a name="compiler-error-cs1583"></a>Derleyici hatası CS1583

' File ' geçerli bir Win32 kaynak dosyası değil  
  
 Bu hata, geçerli veya doğru biçimli bir Win32 kaynak dosyası olmayan [**Win32Resource**](../language-reference/compiler-options/resources.md#win32resource) derleyici seçeneğiyle bir dosya adı belirttiğinizde oluşur. Visual Studio 'da dosya adı, proje Tasarımcısı 'nın uygulama bölmesinde belirtilir.
