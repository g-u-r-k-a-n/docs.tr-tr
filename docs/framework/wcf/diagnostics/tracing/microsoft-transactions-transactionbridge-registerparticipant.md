# <a name="microsofttransactionstransactionbridgeregisterparticipant"></a>Microsoft.Transactions.TransactionBridge.RegisterParticipant

WS-Atomic Işlem protokolü hizmeti bir katılımcıyı denetim protokolü için başarıyla kaydetti.  
  
## <a name="description"></a>Açıklama  

 Yerel Işlem yöneticisi bir katılımcının bir kayıt isteğini bir 2 PC protokolüne başarıyla kabul edildiğinde izlenebilir.
