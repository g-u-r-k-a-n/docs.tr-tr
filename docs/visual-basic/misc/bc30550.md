# <a name="constructors-cannot-implement-interface-methods"></a>Oluşturucular arabirim yöntemleri uygulayamaz
`New`Anahtar sözcüğünü, geçerli olmayan bir arabirim yöntemi uygulamak için kullanmayı denediniz.  
  
 **Hata kimliği:** BC30550  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Anahtar sözcüğü ile arabirimleri uygulayın `Implements` .
