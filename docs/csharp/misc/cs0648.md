# <a name="compiler-error-cs0648"></a>Derleyici hatası CS0648

' Type ', dil tarafından desteklenmeyen bir türdür  
  
 Büyük olasılıkla C++ olan başka bir dilden oluşturulan meta veriler, yönetilen olarak işaretlenmemiş bir tür içeriyordu. Bu tür hakkındaki bilgiler meta verilerde bulunur, ancak tür C# dilinde yazılmış programlar için kullanılamaz.
