# <a name="compiler-error-cs0708"></a><span data-ttu-id="edf67-103">Derleyici hatası CS0708</span><span class="sxs-lookup"><span data-stu-id="edf67-103">Compiler Error CS0708</span></span>
<span data-ttu-id="edf67-104">' Field ': statik sınıfta örnek üyeleri bildirilemez</span><span class="sxs-lookup"><span data-stu-id="edf67-104">'field': cannot declare instance members in a static class</span></span>  
  
 <span data-ttu-id="edf67-105">Statik olarak belirtilen bir sınıfta statik olmayan bir üye bildirirseniz bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="edf67-105">This error occurs if you declare a non-static member in a class that is declared static.</span></span> <span data-ttu-id="edf67-106">Statik sınıfların örnekleri oluşturmak mümkün değildir, bu nedenle örnek değişkenleri anlamlı olmaz.</span><span class="sxs-lookup"><span data-stu-id="edf67-106">It is not possible to create instances of static classes, so instance variables would not be meaningful.</span></span> <span data-ttu-id="edf67-107">**Static** anahtar sözcüğünün tüm statik sınıfların üyelerine uygulanması gerekir.</span><span class="sxs-lookup"><span data-stu-id="edf67-107">The **static** keyword should be applied to all members of static classes.</span></span>  
  
 <span data-ttu-id="edf67-108">Aşağıdaki örnek CS0708 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="edf67-108">The following sample generates CS0708:</span></span>  
  
```csharp

{  
   int i;  // CS0708
```
