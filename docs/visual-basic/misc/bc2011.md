# <a name="unable-to-open-response-file-filename"></a>' ' Yanıt dosyası açılamadı \<filename>
Erişmeye çalıştığınız dosya kullanılamıyor.  
  
 **Hata kimliği:** BC2011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ağ bağlantınızın hala etkin olduğundan emin olun.  
  
- Dosya üzerindeki izinlerinizi denetleyin.
