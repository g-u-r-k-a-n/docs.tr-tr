## <a name="example"></a><span data-ttu-id="099d0-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="099d0-107">Example</span></span>  

 <span data-ttu-id="099d0-108">Aşağıdaki örnek CS0199 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="099d0-108">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
