# <a name="compiler-error-cs0150"></a><span data-ttu-id="7788f-103">Derleyici hatası CS0150</span><span class="sxs-lookup"><span data-stu-id="7788f-103">Compiler Error CS0150</span></span>
<span data-ttu-id="7788f-104">Sabit değer bekleniyor</span><span class="sxs-lookup"><span data-stu-id="7788f-104">A constant value is expected</span></span>  
  
 <span data-ttu-id="7788f-105">Bir sabit beklenirken bir değişken bulundu.</span><span class="sxs-lookup"><span data-stu-id="7788f-105">A variable was found where a constant was expected.</span></span> <span data-ttu-id="7788f-106">Daha fazla bilgi için bkz. [anahtar](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="7788f-106">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="7788f-107">Aşağıdaki örnek CS0150 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7788f-107">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="7788f-108">Bu hata, bir değişken değeri ile bir dizi boyutu belirtildiğinde ve dizi başlatıcısıyla başlatıldığında üretilir.</span><span class="sxs-lookup"><span data-stu-id="7788f-108">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="7788f-109">Hatayı kaldırmak için, diziyi ayrı bir deyimde veya deyimlerde başlatın.</span><span class="sxs-lookup"><span data-stu-id="7788f-109">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
