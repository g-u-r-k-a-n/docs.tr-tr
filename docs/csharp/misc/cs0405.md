# <a name="compiler-error-cs0405"></a>Derleyici hatası CS0405
' Type Parameter ' tür parametresi için yinelenen kısıtlama ' Constraint '  
  
 Genel bildirimindeki kısıtlamaların ikisi de aynıdır. Hatadan kurtulmak için, yinelemeyi kaldırın.  
  
 Aşağıdaki örnek CS0405 oluşturur:  
  
```csharp  
// CS0405.cs  
interface I  
{  
}  
  
class C<T> where T : I, I  // CS0405.cs  
{  
}  
```
