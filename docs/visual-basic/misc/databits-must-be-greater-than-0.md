# <a name="databits-must-be-greater-than-0"></a>DataBits 0 ' dan büyük olmalıdır

`DataBits`Yöntem için belirtilen bağımsız değişken `My.Computer.Ports.OpenSerialPort` sıfırdan büyük olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `DataBits`Bağımsız değişkenin değerini pozitif bir sayı olarak değiştirin.
