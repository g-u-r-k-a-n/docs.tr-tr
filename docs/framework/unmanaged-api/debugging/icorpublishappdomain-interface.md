|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetID Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icorpublishappdomain-getid-method.md)|Bu `ICorPublishAppDomain`için benzersiz tanımlayıcıyı alır.|
