# <a name="wcf-and-websockets"></a>WCF ve WebSockets
.NET Framework 4,5, Windows Communication Foundation için WebSockets desteğini tanıtır.  WebSockets, standart HTTP bağlantı noktaları 80 ve 443 üzerinden çift yönlü iletişim sağlayan verimli, standartlara dayalı bir teknolojidir. Standart HTTP bağlantı noktalarının kullanımı, WebSockets 'in aracılar aracılığıyla Web üzerinden iletişim kurmasına izin verir.  WebSocket aktarımı üzerinden iletişimi desteklemek için iki yeni standart bağlama eklenmiştir. <xref:System.ServiceModel.NetHttpBinding> ve <xref:System.ServiceModel.NetHttpsBinding>. WebSockets 'e özgü ayarlar, özelliğine erişerek üzerinde yapılandırılabilir <xref:System.ServiceModel.Channels.HttpTransportBindingElement> <xref:System.ServiceModel.Channels.HttpTransportBindingElement.WebSocketSettings%2A> .
