# <a name="file-information-cannot-be-queried-while-open-for-writing"></a>Yazma açıkken dosya bilgileri sorgulanamıyor

Şu anda yazmak üzere açık olan bir dosya hakkında bilgi edinmek için girişimde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dosyayı kapatın ve işlemi yeniden deneyin.
