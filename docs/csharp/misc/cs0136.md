## <a name="example"></a><span data-ttu-id="4e350-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="4e350-107">Example</span></span>  
 <span data-ttu-id="4e350-108">Aşağıdaki örnek CS0136 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4e350-108">The following sample generates CS0136:</span></span>  
  
```csharp
// CS0136.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }  
}  
```
