# <a name="const-declaration-cannot-have-an-array-initializer"></a>Const bildiriminde bir dizi başlatıcısı bulunamaz
Bir bildirimde dizi başlatıcısı kullanmaya çalıştınız `Const` .  
  
 **Hata kimliği:** BC30445  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Çevreleyen ayraçları kaldırarak başlatıcıyı geçerli bir ifadeyle değiştirin.
