# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="055b3-102">'ByRef' parametresi '\<parametre adı>' bir lambda ifadesinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="055b3-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="055b3-103">Bir veya işlev içinde `Sub` bildirilen bir `ByRef` lambda `Sub` ifadesi, bu veya işlevin herhangi bir parametresini kullanamaz.</span><span class="sxs-lookup"><span data-stu-id="055b3-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="055b3-104">Örneğin, `ByRef` lambda ifadesinde parametre `n` kullanıldığından aşağıdaki kod bu hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="055b3-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="055b3-105">**Hata Kimliği:** BC36639</span><span class="sxs-lookup"><span data-stu-id="055b3-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="055b3-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="055b3-106">To correct this error</span></span>  
  
- <span data-ttu-id="055b3-107">Parametreyi `ByRef` yerel bir değişkene atayın ve aşağıdaki kodda gösterildiği gibi lambda ifadesindeyerel değişkeni kullanın:</span><span class="sxs-lookup"><span data-stu-id="055b3-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
