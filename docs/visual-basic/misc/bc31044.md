# <a name="events-declared-with-an-as-clause-must-have-a-delegate-type"></a>' As ' yan tümcesiyle belirtilen olaylar bir temsilci türüne sahip olmalıdır
Temsilci olmayan bir tür belirtildi.  
  
 **Hata kimliği:** BC31044  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `As`Yan tümcesini kaldırın.  
  
2. Yan tümcesiyle bir temsilci belirtin `As` .
