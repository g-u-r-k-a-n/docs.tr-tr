# <a name="properties-declared-writeonly-cannot-have-a-get"></a>' WriteOnly ' olarak tanımlanan özelliklerde ' Get ' bulunamaz

`Get`Yordam bir özelliğin değerini okur. `WriteOnly` Özellikler okunamıyor.  
  
 **Hata kimliği:** BC30023  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `WriteOnly`Deyimden anahtar sözcüğü kaldırın `Property` veya `Get` yordamı Özellik gövdesinden kaldırın.
