# <a name="method-methodname1-must-be-declared-overloads-because-another-methodname2-is-declared-overloads"></a>\<method>\<methodname1>başka bir ' \<methodname2> ' ' overloads ' olarak bildirildiği için ' ' ' overloads ' olarak bildirilmelidir
Değiştiriciyle aynı ada sahip başka bir özellik veya yöntem tanımlanmış `Overloads` .  
  
 **Hata kimliği:** BC31409  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Overloads`Değiştirici özelliğine veya yöntemine ekleyin.  
  
2. `Overloads`Değiştiriciyi aynı ada sahip tüm özelliklerden veya metotlardan kaldırın.
