# <a name="method-call-did-not-return-a-value"></a>Yöntem çağrısı bir değer döndürmedi
Çağrılan yöntem beklenen şekilde bir değer döndürmedi.  
  
 **Hata kimliği:** BC30723  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Çağrılan yöntemin bir değer döndüren bir işlev olduğunu doğrulayın.
