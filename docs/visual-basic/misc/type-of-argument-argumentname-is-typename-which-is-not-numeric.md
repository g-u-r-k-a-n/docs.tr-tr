# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>' ' Bağımsız değişkeninin türü \<argumentname> \<typename> , sayısal olmayan ' '

Sayısal olmayan bir bağımsız değişken, sayısal bir bağımsız değişkenin beklenildiği bir durumda oluşur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişkenin sayısal olduğundan emin olun.
