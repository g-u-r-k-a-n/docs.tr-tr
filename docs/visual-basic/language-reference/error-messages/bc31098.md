# <a name="class-classname-cannot-be-found"></a>'\<ClassName > ' sınıfı bulunamıyor
'\<ClassName > ' sınıfı bulunamıyor. Bu durum genellikle eşleşmeyen bir ' Microsoft. VisualBasic. dll ' sonucudur.  
  
 Tanımlı üye bulunamadı.  
  
 **Hata kimliği:** BC31098  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Hatanın yinelenip yinelenmeyeceğini görmek için programı yeniden derleyin.  
  
2. Hata yinelenirse, çalışmanızı kaydedin ve Visual Studio 'Yu yeniden başlatın.  
  
3. Hata devam ederse Visual Basic yeniden yükleyin.  
  
4. Yeniden yükleme sonrasında hata devam ederse, Microsoft Ürün Destek Hizmetleri 'ne bildirin.
