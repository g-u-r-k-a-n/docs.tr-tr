# <a name="compiler-error-cs1517"></a>Derleyici hatası CS1517

Geçersiz Önişlemci ifadesi  
  
 Derleyici geçersiz bir Önişlemci ifadesiyle karşılaştı.  
  
 Daha fazla bilgi için bkz. [Önişlemci yönergeleri](../language-reference/preprocessor-directives/index.md).  
  
 Aşağıdaki örnekte, bazı geçerli ve geçersiz Önişlemci ifadeleri gösterilmektedir:  
  
```csharp  
// CS1517.cs  
#if symbol      // OK  
#endif  
#if !symbol     // OK  
#endif  
#if (symbol)    // OK  
#endif  
#if true        // OK  
#endif  
#if false       // OK  
#endif  
#if 1           // CS1517  
#endif  
#if ~symbol     // CS1517  
#endif  
#if *           // CS1517  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
