 <span data-ttu-id="1849f-107">Aşağıdaki örnek CS0157 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1849f-107">The following sample generates CS0157:</span></span>

```csharp  
{  
   public class MyClass2 : Exception  
   {  
   }

      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
