<span data-ttu-id="1e359-104">`New`Kısıtlama, bu tür parametresine sağlanan tür bağımsız değişkeninin, içinden nesneler oluşturan koda erişilebilen parametresiz bir oluşturucuyu kullanıma sunması gerektiği anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="1e359-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="1e359-105">Tüm değer türlerinde parametresiz oluşturucular vardır, ancak tüm başvuru türleri uygulanmaz.</span><span class="sxs-lookup"><span data-stu-id="1e359-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="1e359-106">Bu nedenle `Structure` kısıtlama kısıtlamayı karşılar `New` , ancak `Class` kısıtlama veya bir sınıf ya da arabirim adı değildir.</span><span class="sxs-lookup"><span data-stu-id="1e359-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>
