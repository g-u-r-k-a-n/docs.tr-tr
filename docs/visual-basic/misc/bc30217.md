# <a name="string-constant-expected"></a>Dize sabiti bekleniyor

Çift tırnak işareti (") dışında bir karakter, bağlamın bir dize sabiti gerektirdiğini ortaya çıkar. Olası nedenlerden biri, dizede tırnak işaretlerinin eksik olması olabilir.  
  
 **Hata kimliği:** BC30217  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dizenin çift tırnak işareti içine doğru şekilde yerleştirildiğinden emin olun.  
  
2. Bir dize sabiti sağlayın.
