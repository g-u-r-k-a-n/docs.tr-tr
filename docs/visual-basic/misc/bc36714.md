# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="8ce33-103">Genişletilmiş özellikler başlatılamıyor</span><span class="sxs-lookup"><span data-stu-id="8ce33-103">Expanded properties cannot be initialized</span></span>

<span data-ttu-id="8ce33-104">Otomatik uygulanan bir özellik, bildiriminin bir parçası olarak başlatılabilir, ancak genişletilmiş bir özellik olamaz.</span><span class="sxs-lookup"><span data-stu-id="8ce33-104">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="8ce33-105">**Hata kimliği:** BC36714</span><span class="sxs-lookup"><span data-stu-id="8ce33-105">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8ce33-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="8ce33-106">To correct this error</span></span>  
  
- <span data-ttu-id="8ce33-107">Otomatik uygulanan bir özellik kullanın veya başlatma özelliğini özellik bildiriminden kaldırın.</span><span class="sxs-lookup"><span data-stu-id="8ce33-107">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8ce33-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="8ce33-108">Example</span></span>  

 <span data-ttu-id="8ce33-109">Aşağıdaki örneklerde hem otomatik uygulanan hem de genişletilmiş özellikler gösterilmektedir.</span><span class="sxs-lookup"><span data-stu-id="8ce33-109">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="8ce33-110">Otomatik uygulanan özellikler atama veya `New` yan tümce kullanılarak başlatılabilir, ancak genişletilmiş özellikler olamaz.</span><span class="sxs-lookup"><span data-stu-id="8ce33-110">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
