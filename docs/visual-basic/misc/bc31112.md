# <a name="statement-cannot-appear-within-an-event-body"></a>İfade bir olay gövdesinde görünemez
İfade bir olay gövdesinde görünemez. Olay sonu varsayıldı.  
  
 Bir olay gövdesi içinde geçerli olmayan bir ifade, içinde yer alır.  
  
 **Hata kimliği:** BC31112  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `End Event`Deyimden önce ekleyin.
