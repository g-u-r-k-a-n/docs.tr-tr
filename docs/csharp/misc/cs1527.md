# <a name="compiler-error-cs1527"></a><span data-ttu-id="3bc47-103">Derleyici hatası CS1527</span><span class="sxs-lookup"><span data-stu-id="3bc47-103">Compiler Error CS1527</span></span>

<span data-ttu-id="3bc47-104">Bir ad alanında tanımlanan öğeler açıkça özel, korumalı, korunan iç veya özel korumalı olarak bildirilemez.</span><span class="sxs-lookup"><span data-stu-id="3bc47-104">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="3bc47-105">Bir ad alanındaki tür bildirimlerinin [ortak](../language-reference/keywords/public.md) veya [iç](../language-reference/keywords/internal.md) erişimi olabilir.</span><span class="sxs-lookup"><span data-stu-id="3bc47-105">Type declarations in a namespace can have either [public](../language-reference/keywords/public.md) or [internal](../language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="3bc47-106">Hiçbir erişilebilirlik belirtilmemişse, **iç** varsayılandır.</span><span class="sxs-lookup"><span data-stu-id="3bc47-106">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="3bc47-107">Aşağıdaki örnek CS1527 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3bc47-107">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="3bc47-108">Aşağıdaki örnek, program kodunuzda açıkça bir ad alanı bildirildiği zaman, tüm tür bildirimleri genel ad alanı içinde örtük olarak yer aldığı için CS1527 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="3bc47-108">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
