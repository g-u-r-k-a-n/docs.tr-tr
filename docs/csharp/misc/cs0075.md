# <a name="compiler-error-cs0075"></a><span data-ttu-id="8d2c8-103">Derleyici hatası CS0075</span><span class="sxs-lookup"><span data-stu-id="8d2c8-103">Compiler Error CS0075</span></span>
<span data-ttu-id="8d2c8-104">Negatif bir değeri dönüştürmek için değeri parantez içine almalısınız</span><span class="sxs-lookup"><span data-stu-id="8d2c8-104">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="8d2c8-105">Önceden tanımlanmış bir türü tanımlayan bir anahtar sözcük kullanarak atama yapıyorsanız, parantezler gerekmez.</span><span class="sxs-lookup"><span data-stu-id="8d2c8-105">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="8d2c8-106">Aksi takdirde, (x) – y, atama ifadesi olarak değerlendirilmeyecek, ancak ayraçları koymanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="8d2c8-106">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="8d2c8-107">C# belirtiminden, Section 7.6.6:</span><span class="sxs-lookup"><span data-stu-id="8d2c8-107">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="8d2c8-108">*Kesinleştirme kuralından sonra, x ve y tanımlayıcısa, (x) y, (x) (y) ve (x) (-y) atama ifadeleridir, ancak x bir türü tanımlarsa bile (x)-y değildir. Ancak, x önceden tanımlanmış bir türü (örneğin, int) tanımlayan bir anahtar sözcüktür, dört formun hepsi atama ifadeleridir (böyle bir anahtar sözcük muhtemelen bir ifade olamaz).*</span><span class="sxs-lookup"><span data-stu-id="8d2c8-108">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="8d2c8-109">Aşağıdaki kod CS0075 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8d2c8-109">The following code generates CS0075:</span></span>  
  
```csharp  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
