# <a name="inherits-statements-must-precede-all-declarations-in-an-interface"></a>' Inherits ' deyimleri arabirimdeki tüm bildirimlerden önce gelmelidir

Bir `Inherits` ifade, sınıf tanımında ilk boş olmayan, yorum olmayan satır olmalıdır.  
  
 **Hata kimliği:** BC30357  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Inherits`Deyiminizi tüm bildirimlerin önünde olacak şekilde taşıyın.
