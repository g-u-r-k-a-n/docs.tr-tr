# <a name="-expected"></a>') ' bekleniyor
Bir yordam tanımındaki bağımsız değişken listesi veya yordam çağrısı gibi bir liste, bir kapanış ayracı () karakteri ile sonlandırılmaz `)` .  
  
 **Hata kimliği:** BC30198  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Listenin sonuna bir kapanış ayracı karakteri ekleyin.
