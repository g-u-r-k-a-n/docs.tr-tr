### <a name="fixed-a-hang-when-listbox-contains-duplicate-value-types"></a>ListBox yinelenen değer türleri içerdiğinde asılı kalma düzeltildi

#### <a name="details"></a>Ayrıntılar

<xref:System.Windows.Controls.ItemsControl>Öğe koleksiyonu yinelenen değer türünde nesneler içerdiğinde, kaydırma sırasında bir sanallaştırmanın askıda kalmasına neden olan bir sorun düzeltildi.

| Name    | Değer       |
|:--------|:------------|
| Kapsam   |Ana|
|Sürüm|4,8|
|Tür|Çalışma Zamanı|

#### <a name="affected-apis"></a>Etkilenen API’ler

API analizi aracılığıyla algılanamaz.
