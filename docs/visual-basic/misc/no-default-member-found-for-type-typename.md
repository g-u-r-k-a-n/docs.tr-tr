# <a name="no-default-member-found-for-type-typename"></a>' ' Türü için varsayılan üye bulunamadı \<typename>

Geç bağlantılı bir dizin `Get` veya `Set` çağrı yapılmaya çalışıldı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Nesnesine `Get` bağlı olarak veya ' a sahip bir varsayılan özelliğe sahip olduğundan emin olun `Set` .
