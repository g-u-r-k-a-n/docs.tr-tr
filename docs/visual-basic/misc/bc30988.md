# <a name="type-or-with-expected"></a><span data-ttu-id="259a0-103">Tür veya ' with ' bekleniyor</span><span class="sxs-lookup"><span data-stu-id="259a0-103">Type or 'With' expected</span></span>

<span data-ttu-id="259a0-104">Bir sınıfın örneğini bildirdiğinizde, `New` anahtar kelimesinin ardından bir tür adı veya tarafından gelmelidir `With` .</span><span class="sxs-lookup"><span data-stu-id="259a0-104">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="259a0-105">Örneğin, aşağıdaki deyimler `client` sınıfının bir örneği olarak bildirir `Customer` .</span><span class="sxs-lookup"><span data-stu-id="259a0-105">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="259a0-106">Tür adı `Customer` aşağıdaki gibidir `New` .</span><span class="sxs-lookup"><span data-stu-id="259a0-106">The type name `Customer` follows `New`.</span></span>  
  
```vb  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="259a0-107">Visual Basic 2008 ' den başlayarak, bir nesneyi anonim bir türün örneği olacak şekilde bildirebilirsiniz, bu durumda bir veri türü belirtmezsiniz.</span><span class="sxs-lookup"><span data-stu-id="259a0-107">Beginning with Visual Basic 2008, you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="259a0-108">Anonim tür bildirimlerinde anahtar sözcüğü `With` aşağıdaki gibidir `New` .</span><span class="sxs-lookup"><span data-stu-id="259a0-108">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```vb  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```
