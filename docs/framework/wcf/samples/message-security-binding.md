# <a name="message-security-binding"></a>İleti Güvenliği Bağlama
Bu bölüm, WCF 'de Windows hizmetlerinde ileti güvenliği bağlamayı gösteren örnekler içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [İleti Güvenliği Anonim](message-security-anonymous.md)  
 Bu örnek, istemci kimlik doğrulaması olmadan ileti düzeyinde güvenlik kullanan, ancak sunucunun X. 509.440 sertifikasını kullanarak sunucu kimlik doğrulaması gerektiren bir Windows Communication Foundation (WCF) uygulamasının nasıl uygulanacağını gösterir.  
  
 [İleti Güvenliği Sertifikası](message-security-certificate.md)  
 Bu örnek, istemci için X. 509.440 v3 sertifika kimlik doğrulamasıyla WS-Security kullanan bir uygulamanın nasıl uygulanacağını gösterir ve sunucunun X. 509.440 v3 sertifikasını kullanarak sunucu kimlik doğrulaması gerektirir.  
  
 [İleti Güvenliği Kullanıcı Adı](message-security-user-name.md)  
 Bu örnek, istemci için Kullanıcı adı kimlik doğrulamasıyla WS-Security kullanan bir uygulamanın nasıl uygulanacağını gösterir ve sunucunun X. 509v3 sertifikasını kullanarak sunucu kimlik doğrulaması gerektirir.  
  
 [İleti Güvenliği Windows](message-security-windows.md)  
 Bu örnek <xref:System.ServiceModel.WSHttpBinding> , Windows kimlik doğrulamasıyla ileti düzeyi güvenliği kullanmak için bir bağlamanın nasıl yapılandırılacağını gösterir.
