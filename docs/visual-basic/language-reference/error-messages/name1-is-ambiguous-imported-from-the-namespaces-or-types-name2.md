# <a name="bc30561-name1-is-ambiguous-imported-from-the-namespaces-or-types-name2"></a>BC30561: ' ' \<name1> belirsiz, ' ' ad alanlarından veya türlerinden içeri aktarıldı \<name2>

Belirsiz bir ad sağladınız ve bu nedenle başka bir adla çakışıyor. Visual Basic derleyicisinde herhangi bir çakışma çözümleme kuralı yok; adları kendiniz kesinleştirmeniz gerekir.

 **Hata kimliği:** BC30561

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Ad alanı içeri aktarmaları kaldırılarak adı belirsizliğini kaldırır.

- Adı tam olarak nitelendirin.
