# <a name="compiler-error-cs0585"></a>Derleyici Hatası CS0585

İç Derleyici Hatası: aşama 'sahne'

 Derleyicinin beklenmeyen sözdizimini ayrıştıramadığı için başarısız olup olmadığını belirlemeye çalışın. Bu durumda değilse, [Bizimle Konuşun](/visualstudio/ide/feedback-options)deneyin.

> [!NOTE]
> Bu derleyici hatası artık Roslyn'de kullanılmaz.
