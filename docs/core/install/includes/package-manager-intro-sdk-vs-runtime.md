.NET Core uygulamalarını geliştirmek istiyorsanız SDK'yı (çalışma süresini içerir) yükleyin. Veya yalnızca uygulamaları çalıştırmanız gerekiyorsa, çalışma süresini yükleyin. Çalışma süresini yüklüyorsanız, hem .NET Core hem de ASP.NET Core [çalışma sürelerini içerdiğinden, ASP.NET Core çalışma süresini](#install-the-aspnet-core-runtime)yüklemenizi öneririz.

SDK'yı veya çalışma saatini zaten yüklediyseniz, hangi sürümlerin `dotnet --list-sdks` yüklü olduğunu görmek için ve `dotnet --list-runtimes` komutları kullanın. Daha fazla bilgi için [.NET Core'un zaten yüklü olup olmadığını nasıl kontrol edebilirsiniz.](../how-to-detect-installed-versions.md?pivots=os-linux)
