<span data-ttu-id="59110-103">İç içe döngüler, `Next` Aşağıdaki örnekte olduğu gibi, iç içe geçmiş döngüden daha fazla döngü değişkeni belirten tek bir ifadeyle sonlandırılır:</span><span class="sxs-lookup"><span data-stu-id="59110-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>

```vb  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```
