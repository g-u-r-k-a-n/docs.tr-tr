# <a name="compiler-error-cs1563"></a>Derleyici hatası CS1563

' Çıktı dosyası ' çıkışının hiç kaynak dosyası yok  
  
 [-Out](../language-reference/compiler-options/out-compiler-option.md) derleyici seçeneği belirtildi, ancak hiçbir kaynak kodu dosyası izmedi. **/Out**kullanmamalısınız ya da **/Out**'dan sonra kaynak kodu dosyalarını belirtmeniz gerekir.
