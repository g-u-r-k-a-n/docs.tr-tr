# <a name="type-arguments-unexpected"></a><span data-ttu-id="3913e-102">Tür bağımsız değişkenleri beklenmiyor</span><span class="sxs-lookup"><span data-stu-id="3913e-102">Type arguments unexpected</span></span>

<span data-ttu-id="3913e-103">Bir `Implements` yan tümce, uyguladığı arabirim üyesi için tür bağımsız değişkenleri sağlar.</span><span class="sxs-lookup"><span data-stu-id="3913e-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="3913e-104">`Implements`Yan tümce yalnızca uyguladığı arabirimi ve üyeyi tanımlamalıdır.</span><span class="sxs-lookup"><span data-stu-id="3913e-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="3913e-105">Bu, genel bir yordam bildirirken, `Of` bir arabirim yordamı uygulamadıysanız olduğu gibi, yan tümce ve tür bağımsız değişkenlerinin bildirimin ana bölümünde görünmesi gerektiğini gösterir.</span><span class="sxs-lookup"><span data-stu-id="3913e-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="3913e-106">Aşağıdaki kod bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="3913e-106">The following code can generate this error.</span></span>  
  
```vb  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="3913e-107">Yan tümcesinden önceki bildirimin, `Implements` erişim veya yordam değiştiricilerinden oluşan olası ekleme dışında, arabirim tanımı gibi görünmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="3913e-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="3913e-108">Aşağıdaki kod hatayı önler.</span><span class="sxs-lookup"><span data-stu-id="3913e-108">The following code avoids the error.</span></span>  
  
```vb  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="3913e-109">**Hata kimliği:** BC32088</span><span class="sxs-lookup"><span data-stu-id="3913e-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3913e-110">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="3913e-110">To correct this error</span></span>  
  
- <span data-ttu-id="3913e-111">Yan tümcesinden tür bağımsız değişkeni listesini kaldırın `Implements` .</span><span class="sxs-lookup"><span data-stu-id="3913e-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
- <span data-ttu-id="3913e-112">Arabirimin genel bir üyesini uygulamadıysanız, Type bağımsız değişkeni listesini bildirimin ana bölümüne koyun, `Implements` anahtar sözcüğünden önce.</span><span class="sxs-lookup"><span data-stu-id="3913e-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
