# <a name="unable-to-sink-events-of-object-because-the-object-is-already-firing-events-to-the-maximum-number-of-event-receivers-it-supports"></a>Nesne, desteklediği en fazla sayıda olay alıcısı için olayları zaten tetiklerken nesnenin olayları havuza alınamıyor
