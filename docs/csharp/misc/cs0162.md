## <a name="example"></a><span data-ttu-id="6b605-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="6b605-106">Example</span></span>

<span data-ttu-id="6b605-107">Aşağıdaki örnek CS0162 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6b605-107">The following example generates CS0162:</span></span>

```csharp
// CS0162.cs
// compile with: /W:2
public class Program
{
    public static void Main()
    {
        goto lab1;
        {
            // The following statements cannot be reached:
            int i = 9;   // CS0162
            i++;
        }
    lab1:
        {
        }
    }
}
```
