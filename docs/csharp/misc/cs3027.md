## <a name="example"></a><span data-ttu-id="f121b-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="f121b-106">Example</span></span>  

 <span data-ttu-id="f121b-107">Aşağıdaki örnek, kendi imzasında CLS uyumlu olmayan bir tür kullanan bir arabirim içerir ve türü CLS uyumlu değil olarak yapar.</span><span class="sxs-lookup"><span data-stu-id="f121b-107">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>
