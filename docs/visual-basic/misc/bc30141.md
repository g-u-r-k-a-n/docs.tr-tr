# <a name="unable-to-create-assembly-linker-object-error-message"></a>Bütünleştirilmiş kod bağlayıcı nesnesi oluşturulamıyor: \<error message>

Visual Basic derleyici, bildirime sahip bir derleme oluşturmak için derleme bağlayıcı (Al.exe, Alink olarak da bilinir) çağırır. Bağlayıcı, çalışma nesnesini oluştururken bir hata bildirdi. Bu, meta veri dosyasına başvuru veya bir içeri aktarma dosyası çağırma hatası olabilir.  
  
 **Hata kimliği:** BC30141  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Alıntı yapılan hata iletisini inceleyin ve daha fazla açıklama ve öneri için [Al.exe](../../framework/tools/al-exe-assembly-linker.md) konusuna başvurun.  
  
2. Hata devam ederse, koşullar hakkındaki bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
