# <a name="index-indexnumber-for-dimension-dimensionnumber-is-out-of-range"></a>' \<indexnumber> ' Boyutu için ' ' dizini \<dimensionnumber> Aralık dışında
Dizinlerden biri tanımlanan aralığı aşıyor.  
  
 **Hata kimliği:** BC30702  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yordamın mantığını, dizi sınırlarının aşılmayacak şekilde değiştirin.  
  
- Daha geniş bir değer aralığına izin vermek için diziyi yeniden tanımlayın.
