<span data-ttu-id="49693-140">Geliştiriciler genellikle <xref:System.IAsyncResult> çok yüksek performans gereksinimlerine sahip olan hizmetler için olan kalıbı kullanır.</span><span class="sxs-lookup"><span data-stu-id="49693-140">Developers often use the <xref:System.IAsyncResult> pattern for services that typically have very high performance requirements.</span></span> <span data-ttu-id="49693-141">Örneğin, tamamlama senaryosu için yoklama, yüksek performanslı bir sunucu tekniğidir.</span><span class="sxs-lookup"><span data-stu-id="49693-141">For example, the polling for completion scenario is a high-performance server technique.</span></span>
