# <a name="modules-cannot-be-declared-specifier"></a>Modüller ' ' olarak bildirilemez \<specifier>

Modül geçersiz bir erişim değiştiricisiyle bildirildi.  
  
 **Hata kimliği:** BC31052  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Erişim değiştiricisini, veya gibi geçerli bir erişim değiştiricisi ile değiştirin `Public` `Friend` .
