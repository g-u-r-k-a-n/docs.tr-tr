# <a name="get-started-wpf"></a>Kullanmaya başlama (WPF)

Windows Presentation Foundation (WPF), masaüstü istemci uygulamaları oluşturan bir kullanıcı bir kullanıcı birimi çerçevesidir. WPF geliştirme platformu, uygulama modeli, kaynaklar, denetimler, grafikler, düzen, veri bağlama, belgeler ve güvenlik dahil olmak üzere çok çeşitli uygulama geliştirme özelliklerini destekler. Bu .NET Framework'ün bir alt kümesidir, bu nedenle ASP.NET veya Windows Forms kullanarak .NET Framework ile daha önce uygulama inşa ettiyseniz, programlama deneyimi tanıdık olmalıdır. WPF, uygulama programlama için bildirimsel bir model sağlamak için Genişletilebilir Uygulama Biçimlendirme Dili'ni (XAML) kullanır. Bu bölümde WPF'yi tanıtmanıza ve işe başlamanıza yardımcı olan konular vardır.
