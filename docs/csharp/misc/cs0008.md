# <a name="compiler-error-cs0008"></a>Derleyici hatası CS0008

' Dosya ' dosyasından meta veriler okunurken beklenmeyen hata oluştu — ' Description '  
  
 Meta verileri almak için bir DLL başarıyla açıldı, ancak veriler bozuk. Daha fazla bilgi için bkz. [ **Başvurular** (C# derleyici seçenekleri)](../language-reference/compiler-options/inputs.md#references).
