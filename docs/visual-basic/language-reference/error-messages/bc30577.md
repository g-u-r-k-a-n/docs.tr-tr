# <a name="addressof-operand-must-be-the-name-of-a-method-without-parentheses"></a>'AddressOf' işleneni bir metot adı olmalıdır (parantezler olmadan)
@No__t-0 işleci, belirli bir yordama başvuran bir yordam temsilci örneği oluşturur. Sözdizimi aşağıdaki gibidir.  
  
 `AddressOf``procedurename`  
  
 @No__t-0 ' dan sonraki bağımsız değişkeni çevresine parantez eklediniz, burada hiçbiri gerekli değildir.  
  
 **Hata kimliği:** BC30577  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. @No__t-0 ' dan sonraki bağımsız değişkenin etrafındaki parantezleri kaldırın.  
  
2. Bağımsız değişkenin bir yöntem adı olduğundan emin olun.
