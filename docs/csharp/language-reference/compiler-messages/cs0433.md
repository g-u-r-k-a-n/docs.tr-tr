## <a name="examples"></a><span data-ttu-id="29cf8-111">Örnekler</span><span class="sxs-lookup"><span data-stu-id="29cf8-111">Examples</span></span>

 <span data-ttu-id="29cf8-112">Bu kod, belirsiz türün ilk kopyasıyla DLL 'yi oluşturur.</span><span class="sxs-lookup"><span data-stu-id="29cf8-112">This code creates the DLL with the first copy of the ambiguous type.</span></span>  
  
```csharp  
// CS0433_1.cs  
// compile with: /target:library  
namespace TypeBindConflicts
{  
   public class AggPubImpAggPubImp {}  
}  
```  
  
 <span data-ttu-id="29cf8-113">Bu kod, belirsiz türün ikinci kopyasıyla DLL 'yi oluşturur.</span><span class="sxs-lookup"><span data-stu-id="29cf8-113">This code creates the DLL with the second copy of the ambiguous type.</span></span>  
  
```csharp  
// CS0433_2.cs  
// compile with: /target:library  
namespace TypeBindConflicts
{  
   public class AggPubImpAggPubImp {}  
}  
```  
  
 <span data-ttu-id="29cf8-114">Aşağıdaki örnek CS0433 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="29cf8-114">The following example generates CS0433.</span></span>  
  
```csharp  
// CS0433_3.cs  
// compile with: /reference:cs0433_1.dll /reference:cs0433_2.dll  
using TypeBindConflicts;  
public class Test
{  
   public static void Main()
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
 <span data-ttu-id="29cf8-115">Aşağıdaki örnek, bu CS0433 hatasını çözmek için **/Reference** derleyici seçeneğinin Alias özelliğini nasıl kullanabileceğinizi gösterir.</span><span class="sxs-lookup"><span data-stu-id="29cf8-115">The following example shows how you can use the alias feature of the **/reference** compiler option to resolve this CS0433 error.</span></span>  
  
```csharp  
// CS0433_4.cs  
// compile with: /reference:cs0433_1.dll /reference:TypeBindConflicts=cs0433_2.dll  
using TypeBindConflicts;  
public class Test
{  
   public static void Main()
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();  
   }  
}  
```
