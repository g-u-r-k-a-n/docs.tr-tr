# <a name="xml-comment-parameter-must-have-a-name-attribute"></a>XML açıklama parametresinin bir ' name ' özniteliği olmalıdır
XML açıklama parametresinin bir ' name ' özniteliği olmalıdır. XML açıklaması yoksayılacak.  
  
 XML etiketinde gerekli bir `name` öznitelik eksik.  
  
 **Hata kimliği:** BC42308  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `name`Etiketi için gerekli özniteliği sağlayın.
