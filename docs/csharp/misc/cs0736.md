<span data-ttu-id="116cf-103">'tür adı' arabirim üyesi 'üye adı' uygulamaz.</span><span class="sxs-lookup"><span data-stu-id="116cf-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="116cf-104">'yöntem adı' statik olduğu için arabirim üyesi uygulayamaz.</span><span class="sxs-lookup"><span data-stu-id="116cf-104">'method name' cannot implement an interface member because it is static.</span></span>

## <a name="example"></a><span data-ttu-id="116cf-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="116cf-110">Example</span></span>  
 <span data-ttu-id="116cf-111">Statik olarak beyan `Program.testMethod` edildiğinden aşağıdaki kod CS0736 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="116cf-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```
