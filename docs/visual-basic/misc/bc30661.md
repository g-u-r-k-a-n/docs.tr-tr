# <a name="field-or-property-membername-is-not-found"></a>' ' Alanı veya özelliği \<membername> bulunamadı

Öznitelik için tanımsız bir alan veya özelliğe başvuruluyor.  
  
 **Hata kimliği:** BC30661  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Alanın veya özelliğin yazımını denetleyin.  
  
2. Projenizin doğru öznitelik sınıfına erişebildiğini doğrulayın ve gerekirse proje başvurularına ekleyin.
