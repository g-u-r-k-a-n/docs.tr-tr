# <a name="compiler-error-cs0569"></a>Derleyici hatası CS0569
' Method2 ': dil tarafından desteklenmediğinden ' Method1 ' geçersiz kılınamaz  
  
 Bu hata, başka bir dilde yazılmış bir temel sınıftan türettiğinizde ve derleyici geçersiz kılmaya çalıştığınız yöntemi desteklemediği zaman oluşur.
