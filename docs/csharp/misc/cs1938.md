 <span data-ttu-id="b30c1-112">Aşağıdaki kod CS1938 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b30c1-112">The following code generates CS1938:</span></span>  
  
```csharp

using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] sourceA = { 1, 2, 3, 4, 5 };  
        int[] sourceB = { 3, 4, 5, 6, 7 };  
  
        var query = from a in sourceA  
                    join b in sourceB on b equals a // CS1938  
                    // Try the following line instead.  
                    // join b in sourceB on a equals b  
                    select new { a, b };  
    }  
}  
```
