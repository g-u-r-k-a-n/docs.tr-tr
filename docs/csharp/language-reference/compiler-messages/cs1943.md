# <a name="compiler-error-cs1943"></a><span data-ttu-id="0d135-103">Derleyici Hatası CS1943</span><span class="sxs-lookup"><span data-stu-id="0d135-103">Compiler Error CS1943</span></span>
<span data-ttu-id="0d135-104">' Type ' kaynak türündeki bir sorgu ifadesindeki sonraki from yan tümcesinde ' Type ' türünde bir ifadeye izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="0d135-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="0d135-105">' Method ' çağrısında tür çıkarımı başarısız oldu.</span><span class="sxs-lookup"><span data-stu-id="0d135-105">Type inference failed in the call to 'method'.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="0d135-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="0d135-107">To correct this error</span></span>  
  
1. <span data-ttu-id="0d135-108">Türün `IEnumerable` , `IEnumerable<T>` ya da türetilmiş bir arabirim ya da kendisi için tanımlanmış bir sorgu düzenine sahip başka herhangi bir türü uygulayan bir sorgulanabilir tür olduğundan emin olun.</span><span class="sxs-lookup"><span data-stu-id="0d135-108">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>

## <a name="example"></a><span data-ttu-id="0d135-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="0d135-110">Example</span></span>  
 <span data-ttu-id="0d135-111">Aşağıdaki kod CS1943 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0d135-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
