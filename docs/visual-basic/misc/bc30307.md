# <a name="method1-cannot-override-method2-because-they-differ-by-the-default-values-of-optional-parameters"></a>\<method1> \<method2> isteğe bağlı parametrelerinin varsayılan değerleri farklı olduğundan ' ', ' ' öğesini geçersiz kılamaz

İsteğe bağlı parametrelerinin varsayılan değerlerinde birinciden farklı olan bir yöntemi geçersiz kılmayı denediniz, yani imzaları farklıdır. Bir tür, aynı ada ve imzaya sahip bir yöntemi bildirerek ve açıklamayı değiştiriciyle işaretleyerek devralınan geçersiz kılınabilir bir yöntemi geçersiz kılabilir `Overrides` .  
  
 **Hata kimliği:** BC30307  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İki yöntemin de aynı imzaya sahip olduğundan emin olun.
