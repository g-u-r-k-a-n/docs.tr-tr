# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>' Throw ' deyimleri Komut penceresinde geçerli değildir
`Throw` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30725  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Throw` deyimlerini hata ayıklama kodundan kaldırın.
