# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Değişken bildirim deyimleri Komut penceresinde geçerli değildir

Değişken bildirimleri **komut** penceresinde kullanılamaz.  
  
 **Hata kimliği:** BC30709  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- **Anında** pencerede değişkenleri bildirme.
