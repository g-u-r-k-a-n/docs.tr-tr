# <a name="compiler-error-cs1929"></a><span data-ttu-id="4d891-102">Derleyici hatası CS1929</span><span class="sxs-lookup"><span data-stu-id="4d891-102">Compiler Error CS1929</span></span>

 <span data-ttu-id="4d891-104">Bu hata, genişletmemiş bir sınıftan bir genişletme yöntemi çağırmaya çalıştığınızda oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="4d891-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="4d891-105">Burada gösterilen örnekte, genişletme yöntemi türetilmiş sınıf için tanımlanır `A` , ancak temel sınıf için değildir `B` .</span><span class="sxs-lookup"><span data-stu-id="4d891-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4d891-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="4d891-106">To correct this error</span></span>  
  
1. <span data-ttu-id="4d891-107">Çağırmak istediğiniz tür için yeni bir genişletme yöntemi oluşturun veya çağrıyı, mevcut yöntemin genişlettiği türün bir nesnesine taşıyın.</span><span class="sxs-lookup"><span data-stu-id="4d891-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4d891-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="4d891-108">Example</span></span>  

 <span data-ttu-id="4d891-109">Aşağıdaki kod CS1928 ve CS1929 üretir:</span><span class="sxs-lookup"><span data-stu-id="4d891-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
