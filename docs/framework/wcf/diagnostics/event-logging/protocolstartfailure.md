# <a name="protocolstartfailure"></a>ProtocolStartFailure

Kimlik: 143  
  
 Önem derecesi: Hata  
  
 Kategori: TransactionBridge  
  
## <a name="description"></a>Description  

 Bu olay WS-Atomic Işlem Protokolü hizmetinin başlayamadığını gösterir. Sonuç olarak, WS-Atomic Işlem işlevi devre dışı bırakılır. Olayda protokol KIMLIĞI, protokol adı, özel durum, işlem adı ve işlem KIMLIĞI listelenmektedir. .
