# <a name="inherits-can-appear-only-once-within-a-class-statement-and-can-only-specify-one-class"></a>' Inherits ', ' class ' ifadesinde yalnızca bir kez görünebilir ve yalnızca bir sınıf belirtebilir

`Inherits`Aynı sınıfta birden fazla ifade görünüyor veya bir `Inherits` ifade birden fazla sınıf belirtiyor. Bir sınıf, yalnızca bir temel sınıftan devralınabilir.  
  
 **Hata kimliği:** BC30121  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm ek `Inherits` deyimlerini kaldırın ve kalan `Inherits` deyimin yalnızca bir temel sınıf belirttiğinden emin olun.
