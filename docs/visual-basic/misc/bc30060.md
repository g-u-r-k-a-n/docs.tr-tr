# <a name="conversion-from-type1-to-type2-cannot-occur-in-a-constant-expression"></a>\<type1>Sabit bir ifadede ' ' ile ' ' arasında dönüştürme \<type2> gerçekleşemez

Bir deyimde başlatma ifadesi, `Const` sabit olarak belirtilen türe dönüştürülemeyen bir veri türü olarak değerlendirilir.  
  
 **Hata kimliği:** BC30060  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Sabiti, sabit için belirtilen türe dönüştürülebilecek bir veri türü ifadesiyle başlatın.
