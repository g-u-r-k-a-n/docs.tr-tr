# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="d5ce2-102">Derleyici Uyarısı (düzey 2) CS0253</span><span class="sxs-lookup"><span data-stu-id="d5ce2-102">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="d5ce2-103">Olası istenmeyen başvuru karşılaştırması; bir değer karşılaştırması elde etmek için, 'yazın' yazın için sağ tarafı döküm</span><span class="sxs-lookup"><span data-stu-id="d5ce2-103">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="d5ce2-104">Derleyici bir başvuru karşılaştırması yapıyor.</span><span class="sxs-lookup"><span data-stu-id="d5ce2-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="d5ce2-105">Dizelerin değerini karşılaştırmak istiyorsanız, ifadenin sağ tarafını `type`.</span><span class="sxs-lookup"><span data-stu-id="d5ce2-105">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="d5ce2-106">Aşağıdaki örnek CS0253 üretir:</span><span class="sxs-lookup"><span data-stu-id="d5ce2-106">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
