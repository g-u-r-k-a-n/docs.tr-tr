 <span data-ttu-id="54eaa-105">[Temel](../language-reference/keywords/base.md) sınıf belirtimi hatalı biçimlendirilmiş.</span><span class="sxs-lookup"><span data-stu-id="54eaa-105">A [base](../language-reference/keywords/base.md) class specification was ill formed.</span></span>
