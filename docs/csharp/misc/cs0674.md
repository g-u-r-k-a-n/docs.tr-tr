# <a name="compiler-error-cs0674"></a><span data-ttu-id="ffd07-103">Derleyici hatası CS0674</span><span class="sxs-lookup"><span data-stu-id="ffd07-103">Compiler Error CS0674</span></span>
<span data-ttu-id="ffd07-104">' System. ParamArrayAttribute ' kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="ffd07-104">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="ffd07-105">Bunun yerine ' params ' anahtar sözcüğünü kullanın.</span><span class="sxs-lookup"><span data-stu-id="ffd07-105">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="ffd07-106">C# derleyicisi kullanılmasına izin vermez <xref:System.ParamArrayAttribute?displayProperty=nameWithType> ; bunun yerine [params](../language-reference/keywords/params.md) kullanın.</span><span class="sxs-lookup"><span data-stu-id="ffd07-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="ffd07-107">Aşağıdaki örnek CS0674 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ffd07-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
