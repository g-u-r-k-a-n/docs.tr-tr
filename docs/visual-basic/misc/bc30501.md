# <a name="shared-cannot-be-combined-with-specifier-on-a-method-declaration"></a>Yöntem bildiriminde ' Shared ' ' ' ile birleştirilemez \<specifier>

Değiştirici,, `Shared` veya gibi bir tanımlayıcı ile birleştirmeye çalıştınız, örneğin, `Overridable` `NotOverridable` `MustOverride` bir yöntem bildiriminde bu bir kombinasyonda geçerli değildir.  
  
 **Hata kimliği:** BC30501  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirteci kaldırın.
