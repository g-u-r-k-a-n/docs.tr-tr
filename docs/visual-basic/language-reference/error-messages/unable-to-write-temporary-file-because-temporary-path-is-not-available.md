# <a name="bc30698-unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>BC30698: geçici yol kullanılabilir olmadığından geçici dosya yazılamıyor

Visual Basic geçici dosyaların depolandığı yolu belirleyemedi.

 **Hata kimliği:** BC30698

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Visual Studio’yu yeniden başlatın.

2. Sorun devam ederse, Visual Studio 'Yu yeniden yükleyin.
