# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<MemberName > ', bu \<ContainerName için oluşturulan '\<procedurename > ' tarafından zaten bildirildi >

Ortak dil çalışma zamanı tarafından belirtilen üyeyle aynı ada sahip bir üye bildirildi.

**Hata kimliği:** BC30733

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Çakışmayı kaldırmak için üyenin adını değiştirin.
