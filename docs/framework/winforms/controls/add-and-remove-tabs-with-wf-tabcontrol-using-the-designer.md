# <a name="how-to-add-and-remove-tabs-with-the-windows-forms-tabcontrol-using-the-designer"></a>Nasıl yapılır: Tasarımcı Kullanarak Windows Formları TabControl ile Sekme Ekleme ve Kaldırma
Formunuza bir denetim yerleştirdiğinizde <xref:System.Windows.Forms.TabControl> , varsayılan olarak iki sekme içerir. Tasarımcıyı kullanarak sekmeler ekleyebilir veya kaldırabilirsiniz.

 Aşağıdaki yordam, bir denetim içeren bir form ile **Windows uygulama** projesi gerektirir <xref:System.Windows.Forms.TabControl> . Böyle bir projeyi ayarlama hakkında daha fazla bilgi için bkz. [nasıl yapılır: oluşturma Windows Forms uygulama projesi](/visualstudio/ide/step-1-create-a-windows-forms-application-project) ve [nasıl yapılır: Windows Forms denetim ekleme](how-to-add-controls-to-windows-forms.md).

## <a name="to-add-or-remove-a-tab-using-the-designer"></a>Tasarımcıyı kullanarak sekme ekleme veya kaldırma

- Denetimin akıllı etiketinde **Sekme Ekle** veya **sekmeyi kaldır** ' a tıklayın.

     -veya-

     **Özellikler** penceresinde, **Ellipsis** ![ ](./media/visual-studio-ellipsis-button.png) <xref:System.Windows.Forms.TabControl.TabPages%2A> **TabPage koleksiyonu düzenleyicisini**açmak için özelliğinin yanındaki üç nokta düğmesini (... Özellikler penceresi) tıklayın... **Ekle** veya **Kaldır** düğmesine tıklayın.
