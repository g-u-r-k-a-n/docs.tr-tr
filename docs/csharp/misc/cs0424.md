# <a name="compiler-error-cs0424"></a><span data-ttu-id="0c5e2-103">Derleyici hatası CS0424</span><span class="sxs-lookup"><span data-stu-id="0c5e2-103">Compiler Error CS0424</span></span>

<span data-ttu-id="0c5e2-104">' class ': ComImport özniteliğine sahip sınıf bir temel sınıf belirtemez</span><span class="sxs-lookup"><span data-stu-id="0c5e2-104">'class': a class with the ComImport attribute cannot specify a base class</span></span>  
  
 <span data-ttu-id="0c5e2-105">Özniteliği belirtmek <xref:System.Runtime.InteropServices.ComImportAttribute> , sınıf için uygulamanın BIR com modülünden içeri aktarılacağını belirtir.</span><span class="sxs-lookup"><span data-stu-id="0c5e2-105">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="0c5e2-106">Temel sınıftan devralınan ek yöntemlerin veya alanların COM modülünde tanımlanan uygulamaya eklenmesine izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="0c5e2-106">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>

```csharp

using System.Runtime.InteropServices;

```
