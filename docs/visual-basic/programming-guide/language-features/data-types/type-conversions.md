# <a name="type-conversions-in-visual-basic"></a>Visual Basic'de Tür Dönüştürmeleri

Bir değeri bir veri türünden başka bir türe değiştirme işlemi *dönüştürme* olarak adlandırılır. Dönüşümler, dahil edilen türlerin veri kapasiteye bağlı olarak, *genişletme* veya *daraltma*. Bunlar ayrıca, kaynak kodundaki sözdizimine bağlı olarak *örtük* veya *açıktır*.  
  
## <a name="in-this-section"></a>Bu Bölümde

 Hedef türün verileri tutabilir olup olmadığı sınıflandırılan dönüştürmeleri açıklar.  
  
 [Örtük ve Açık Dönüştürmeler](implicit-and-explicit-conversions.md)  
 Visual Basic otomatik olarak gerçekleştirip gerçekleştirmediğini sınıflandırılan dönüştürmeleri açıklar.  
  
 [Dizeler ve Diğer Türler Arasında Dönüştürmeler](conversions-between-strings-and-other-types.md)  
 Dizeler ve sayısal, `Boolean` ya da tarih/saat değerleri arasında dönüştürme gösterir.  
  
 [Nasıl yapılır: Visual Basic'te Bir Nesneyi Başka Bir Türe Dönüştürme](how-to-convert-an-object-to-another-type.md)  
 Bir `Object` değişkenin diğer veri türlerine nasıl dönüştürüleceğini gösterir.  
  
 [Dizi Dönüştürmeler](array-conversions.md)  
 Farklı veri türlerinin dizileri arasında dönüştürme işleminde size adım adım.
