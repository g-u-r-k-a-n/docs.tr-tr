# <a name="compiler-error-cs1558"></a><span data-ttu-id="b6781-102">Derleyici hatası CS1558</span><span class="sxs-lookup"><span data-stu-id="b6781-102">Compiler Error CS1558</span></span>
<span data-ttu-id="b6781-103">' class ' uygun bir statik Main yöntemine sahip değil</span><span class="sxs-lookup"><span data-stu-id="b6781-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="b6781-104">[-Main](../language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği, **ana** yöntemin aranacağı bir sınıf belirtti.</span><span class="sxs-lookup"><span data-stu-id="b6781-104">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="b6781-105">Ancak [Main](../programming-guide/main-and-command-args/index.md) yöntemi doğru tanımlanmadı.</span><span class="sxs-lookup"><span data-stu-id="b6781-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="b6781-106">Aşağıdaki örnek, geçersiz dönüş türü nedeniyle CS1558 oluşturuyor.</span><span class="sxs-lookup"><span data-stu-id="b6781-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
