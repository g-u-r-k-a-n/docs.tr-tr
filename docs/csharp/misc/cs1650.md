<span data-ttu-id="06dba-104">' Identifier ' statik salt okunur alanının alanlarına atama yapılamaz (statik oluşturucu veya değişken başlatıcı içinde olması dışında)</span><span class="sxs-lookup"><span data-stu-id="06dba-104">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="06dba-105">Bu hata, salt okunur ve statik olan bir alanın bir üyesini değiştirmeye çalıştığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="06dba-105">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="06dba-106">Bu hatayı çözmek için, atamaları salt okunur alanlarla Oluşturucu veya değişken başlatıcısıyla sınırlayın veya `readonly` anahtar sözcüğünü alanın bildiriminden kaldırın.</span><span class="sxs-lookup"><span data-stu-id="06dba-106">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
