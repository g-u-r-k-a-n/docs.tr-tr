# <a name="compiler-error-cs0149"></a><span data-ttu-id="007b7-102">Derleyici Hatası CS0149</span><span class="sxs-lookup"><span data-stu-id="007b7-102">Compiler Error CS0149</span></span>
<span data-ttu-id="007b7-103">Yöntem adı bekleniyor</span><span class="sxs-lookup"><span data-stu-id="007b7-103">Method name expected</span></span>  
  
 <span data-ttu-id="007b7-104">[Bir temsilci](../language-reference/builtin-types/reference-types.md)oluştururken, bir yöntem belirtin.</span><span class="sxs-lookup"><span data-stu-id="007b7-104">When creating a [delegate](../language-reference/builtin-types/reference-types.md), specify a method.</span></span> <span data-ttu-id="007b7-105">Daha fazla bilgi için [bkz.](../programming-guide/delegates/index.md)</span><span class="sxs-lookup"><span data-stu-id="007b7-105">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="007b7-106">Aşağıdaki örnek CS0149 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="007b7-106">The following sample generates CS0149:</span></span>  
  
```csharp  
// CS0149.cs  
using System;  
  
delegate string MyDelegate(int i);  
  
class MyClass  
{  
   // class member-field of the declared delegate type  
   static MyDelegate dt;
  
   public static void Main()  
   {  
      dt = new MyDelegate(17.45);   // CS0149  
      // try the following line instead  
      // dt = new MyDelegate(Func2);  
      F(dt);  
   }  
  
   public static string Func2(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void F(MyDelegate myFunc)  
   {  
      myFunc(8);  
   }  
}  
```
