# <a name="expression-cannot-be-evaluated-at-this-time"></a>İfade şu anda değerlendirilemiyor

Belirtilen ifade geçerli bağlamda değerlendirilemiyor.  
  
 **Hata kimliği:** BC30736  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfadenin sözdizimini değiştirin.
