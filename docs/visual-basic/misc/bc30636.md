# <a name="-expected"></a>' > ' bekleniyor

Belirtilen kod için sözdizimi yanlış. Bu hata genellikle bir öznitelik veya XML öğesi değişmez değeri için kapanış açılı ayracın eksik olduğunu gösterir.  
  
 **Hata kimliği:** BC30636  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kullanılmakta olan kodun söz dizimini denetleyin.
