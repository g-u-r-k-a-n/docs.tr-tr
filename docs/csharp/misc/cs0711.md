 <span data-ttu-id="35f3d-105">Statik bir sınıf örneği oluşturulamıyor, bu nedenle oluşturucular veya Yıkıcılar için gerekli değildir.</span><span class="sxs-lookup"><span data-stu-id="35f3d-105">A static class cannot be instantiated, hence it has no need for constructors or destructors.</span></span> <span data-ttu-id="35f3d-106">Bu hatadan kaçınmak için, statik sınıflardan tüm yıkıcıları kaldırın veya gerçekten örnek oluşturup yok etmek istiyorsanız, sınıfı statik olmayan hale getirin.</span><span class="sxs-lookup"><span data-stu-id="35f3d-106">To avoid this error, remove any destructors from static classes, or, if you really want to construct and destroy instances, make the class non-static.</span></span>
