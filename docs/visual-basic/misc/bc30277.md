# <a name="type-character-charactername-does-not-match-declared-data-type-type"></a>' ' Tür karakteri \<charactername> bildirilmemiş ' ' veri türüyle eşleşmiyor \<type>

Bir değişken, bir veri türü ile tanımlanır, ancak uyumsuz bir veri türünü temsil eden tür karakteriyle (örneğin, `K$ = 10` ne zaman `K` bildirildiğinde) başvurulur `Integer` .  
  
 **Hata kimliği:** BC30277  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değişkenin tanımlanmış veri türünü değiştirin veya başvurudaki tür karakterini değiştirin veya kaldırın.
