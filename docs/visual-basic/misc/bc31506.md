# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>' \<typename> MustInherit ' olarak bildirildiği için ' ' öznitelik olarak kullanılamaz

Özel öznitelik sınıfları olarak bildirilemez `MustInherit` .  
  
 **Hata kimliği:** BC31506  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `MustInherit`Özel özniteliklerden değiştiricisini kaldırın.
