# <a name="only-one-of-notoverridable-mustoverride-or-overridable-can-be-specified"></a>Yalnızca bir ' NotOverridable ', ' MustOverride ' veya ' geçersiz kılınabilir ' belirtilebilir

Çakışan geçersiz kılma belirtimleri bir öğe bildiriminde oluşur. Geçersiz kılma anahtar sözcükleri birbirini dışlıyor.  
  
 **Hata kimliği:** BC30177  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir geçersiz kılma belirticisi seçin ve diğer anahtar sözcükleri bildirimden kaldırın.
