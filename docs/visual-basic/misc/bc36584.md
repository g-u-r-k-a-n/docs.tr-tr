# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>' ' \<parametername> İçinde tanımlanan ' ' genişletme yöntemindeki ' ' parametresinin \<methodname> \<typename> zaten eşleşen bir bağımsız değişkeni var

Bir bağımsız değişken aynı yordam çağrısında ad ile birden çok kez geçirilir.  
  
 **Hata kimliği:** BC36584  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken adlarından birinin yanlış yazılmış olup olmadığını kontrol edin.  
  
- Bağımsız değişken listesinden yinelenen bağımsız değişkeni kaldırın.
