# <a name="compiler-error-cs0449"></a><span data-ttu-id="e84e4-103">Derleyici hatası CS0449</span><span class="sxs-lookup"><span data-stu-id="e84e4-103">Compiler Error CS0449</span></span>

<span data-ttu-id="e84e4-104">' Class ' veya ' struct ' kısıtlaması diğer kısıtlamalardan önce gelmelidir</span><span class="sxs-lookup"><span data-stu-id="e84e4-104">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="e84e4-105">Genel bir türün veya metodun tür parametresindeki kısıtlamalar belirli bir sırada gerçekleşmelidir: `class` ya da ilk, varsa, `struct` herhangi bir arabirim kısıtlaması ve son olarak herhangi bir Oluşturucu kısıtlaması olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="e84e4-105">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="e84e4-106">Bu hata, `class` veya `struct` kısıtlaması ilk olarak görünmediğinde oluşur.</span><span class="sxs-lookup"><span data-stu-id="e84e4-106">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="e84e4-107">Bu hatayı çözmek için kısıtlama yan tümcelerini yeniden sıralayın.</span><span class="sxs-lookup"><span data-stu-id="e84e4-107">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e84e4-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="e84e4-108">Example</span></span>  

 <span data-ttu-id="e84e4-109">Aşağıdaki örnek CS0449 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="e84e4-109">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
