# <a name="compiler-error-cs0005"></a>Derleyici hatası CS0005
' Compiler_option ' derleyici seçeneğinin arkasından bir bağımsız değişken gelmelidir  
  
 Bazı derleyici seçenekleri parametre gerektirir. Derleyici seçeneği için gereken bağımsız değişkenleri geçirmezseniz, CS0005 oluşturulur.  
  
 Daha fazla bilgi için bkz. [C# derleyici seçenekleri](../language-reference/compiler-options/index.md).
