 <span data-ttu-id="6ca57-105">Aşağıdaki kod başarısız dönüştürmeleri gösterir.</span><span class="sxs-lookup"><span data-stu-id="6ca57-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="6ca57-106">Temsilci `FunDel` .</span><span class="sxs-lookup"><span data-stu-id="6ca57-106">The delegate is `FunDel`.</span></span>

 <span data-ttu-id="6ca57-107">Aşağıdaki işlevlerin her biri, `FunDel` Bu hataya neden olacak şekilde farklılık gösterir.</span><span class="sxs-lookup"><span data-stu-id="6ca57-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>

```vb  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date

```  
  
 <span data-ttu-id="6ca57-108">Aşağıdaki atama deyimlerinin her biri hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="6ca57-108">Each of the following assignment statements causes the error.</span></span>

```vb  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.
```  
