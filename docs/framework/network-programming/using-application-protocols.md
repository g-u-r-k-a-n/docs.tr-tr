# <a name="using-application-protocols"></a>Uygulama Protokolleri Kullanma
.NET Framework, yaygın olarak kullanılan Internet uygulama protokollerini destekler. Bu bölümde [HTTP](http.md), "TCP" ve "UDP" protokollerinin kullanımı yla ilgili bilgilerin yanı sıra özel protokolleri uygulamak için [Windows Soketleri](sockets.md) arabiriminin kullanımı yla ilgili bilgiler de yer almaktadır.
