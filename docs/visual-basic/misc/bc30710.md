# <a name="end-of-expression-expected"></a>İfade sonu bekleniyor

Başka bir deyişle, tamamlanmamış bir ifadenin sonunda çok fazla karakter bulundu.  
  
 **Hata kimliği:** BC30710  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesnenin veya deyimin sözdizimini doğrulayın.
