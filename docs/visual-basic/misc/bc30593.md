# <a name="variables-in-modules-cannot-be-declared-specifier"></a>Modül içindeki değişkenler ' ' olarak bildirilemez \<specifier>

Deyimdeki bir değişkende gibi bir tanımlayıcı kullandınız `MustInherit` `Module` . Modüller hiçbir şekilde örneklenemez, devralmayı desteklemez ve arabirimler uygulayamaz.  
  
 **Hata kimliği:** BC30593  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirteci kaldırın.
