# <a name="could-not-find-library-libraryname"></a>' ' Kitaplığı bulunamadı \<libraryname>

Belirtilen kitaplık bulunamadı.  
  
 Bu sorun, komut isteminden derlerken ve birçok derlemeye başvuran bir yanıt (. rsp) dosyası kullanılırken gerçekleşebilir  
  
 **Hata kimliği:** BC2017  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Doğru olduğundan emin olmak için kitaplık adını denetleyin.
