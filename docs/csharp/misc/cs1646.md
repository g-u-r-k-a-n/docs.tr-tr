# <a name="compiler-error-cs1646"></a>Derleyici hatası CS1646

Tam tanımlayıcıdan sonra anahtar sözcük, tanımlayıcı veya dize bekleniyor: \@  
  
 ' ' Tam belirticisinin kullanımı için bkz. dize sabit değerleri \@ . Tam belirticiye yalnızca bir dize, anahtar sözcük veya tanımlayıcıdan önce izin verilir. Bu hatayı çözmek için herhangi bir uygunsuz konumdan @ sembolünü kaldırın veya amaçlanan dize, anahtar sözcük veya tanımlayıcıyı ekleyin.  
  
 Aşağıdaki örnek CS1646 oluşturur:  
  
```csharp  
// CS1646  
class C  
{  
   int i = @5;  // CS1646  
   // Try this line instead:  
   // int i = 5;  
}  
```
