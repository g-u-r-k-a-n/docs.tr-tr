# <a name="key-cannot-be-deleted-because-it-has-subkeys"></a>Anahtar alt anahtarları içerdiğinden silinemiyor

Anahtar alt anahtarları içerdiğinden silinemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu işlemi denemeden önce belirtilen anahtarın alt anahtarlarını belirle ve Sil.
