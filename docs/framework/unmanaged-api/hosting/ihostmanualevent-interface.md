# <a name="ihostmanualevent-interface"></a>IHostManualEvent Arabirimi
Konağın el ile sıfırlama olayının temsili için bir uygulama sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Reset Yöntemi](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-reset-method.md)|Geçerli `IHostManualEvent` örneğini, sinyal olmayan bir duruma sıfırlar.|  
|[Set Yöntemi](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-set-method.md)|Geçerli `IHostManualEvent` örneğini bir sinyal durumuna ayarlar.|  
|[Wait Yöntemi](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-wait-method.md)|Geçerli `IHostManualEvent` örneğinin, ait olana veya belirli bir süre geçtikten sonra beklemesini sağlar.|
