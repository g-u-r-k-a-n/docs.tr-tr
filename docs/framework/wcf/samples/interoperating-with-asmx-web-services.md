> [!NOTE]

 <span data-ttu-id="6f38b-105">Bu örnek, Internet Information Services (IIS) tarafından barındırılan bir istemci konsol programı (.exe) ve bir hizmet kitaplığından (.dll) oluşur.</span><span class="sxs-lookup"><span data-stu-id="6f38b-105">This sample consists of a client console program (.exe) and a service library (.dll) hosted by Internet Information Services (IIS).</span></span> <span data-ttu-id="6f38b-106">Hizmet, istek-yanıt iletişim iseni tanımlayan bir sözleşme uygulayan bir ASMX Web Hizmetidir.</span><span class="sxs-lookup"><span data-stu-id="6f38b-106">The service is an ASMX Web Service that implements a contract that defines a request-reply communication pattern.</span></span> <span data-ttu-id="6f38b-107">Hizmet matematik işlemlerini ortaya`Add` `Subtract`çıkarır `Multiply`( `Divide`, , , ve ).</span><span class="sxs-lookup"><span data-stu-id="6f38b-107">The service exposes math operations (`Add`, `Subtract`, `Multiply`, and `Divide`).</span></span> <span data-ttu-id="6f38b-108">İstemci bir matematik işlemi için eşzamanlı isteklerde bulunmaz ve sonuç ile hizmet yanıtları.</span><span class="sxs-lookup"><span data-stu-id="6f38b-108">The client makes synchronous requests to a math operation and the service replies with the result.</span></span> <span data-ttu-id="6f38b-109">İstemci etkinliği konsol penceresinde görünür.</span><span class="sxs-lookup"><span data-stu-id="6f38b-109">Client activity is visible in the console window.</span></span>  
  
 <span data-ttu-id="6f38b-110">Aşağıdaki örnek kodda gösterilen ASMX Web hizmeti uygulaması, uygun sonucu hesaplar ve döndürür.</span><span class="sxs-lookup"><span data-stu-id="6f38b-110">The ASMX Web service implementation shown in the following sample code calculates and returns the appropriate result.</span></span>  
  
```csharp  
[WebService(Namespace="http://Microsoft.ServiceModel.Samples")]  
public class CalculatorService : System.Web.Services.WebService  
    {  
        [WebMethod]  
        public double Add(double n1, double n2)  
        {  
            return n1 + n2;  
        }  
        [WebMethod]  
        public double Subtract(double n1, double n2)  
        {  
            return n1 - n2;  
        }  
        [WebMethod]  
        public double Multiply(double n1, double n2)  
        {  
            return n1 * n2;  
        }  
        [WebMethod]  
        public double Divide(double n1, double n2)  
        {  
            return n1 / n2;  
        }  
    }  
```  
  
 <span data-ttu-id="6f38b-111">Yapılandırıldıkça, hizmete aynı makinedeki bir istemci `http://localhost/servicemodelsamples/service.asmx` tarafından erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="6f38b-111">As configured, the service can be accessed at `http://localhost/servicemodelsamples/service.asmx` by a client on the same machine.</span></span> <span data-ttu-id="6f38b-112">Uzak makinelerdeki istemcilerin hizmete erişebilmek için localhost yerine nitelikli bir etki alanı adı belirtilmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="6f38b-112">For clients on remote machines to access the service, a qualified domain name must be specified instead of localhost.</span></span>  
  
 <span data-ttu-id="6f38b-113">İletişim [ServiceModel Metadata Utility Tool (Svcutil.exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md)tarafından oluşturulan bir istemci üzerinden yapılır.</span><span class="sxs-lookup"><span data-stu-id="6f38b-113">Communication is done through a client generated by the [ServiceModel Metadata Utility Tool (Svcutil.exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).</span></span> <span data-ttu-id="6f38b-114">İstemci generatedClient.cs dosyada bulunur.</span><span class="sxs-lookup"><span data-stu-id="6f38b-114">The client is contained in the file generatedClient.cs.</span></span> <span data-ttu-id="6f38b-115">Güncelleştirilmiş meta verileri almak için kullanıldığından, proxy kodunu oluşturmak için ASMX hizmetinin kullanılabilir olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="6f38b-115">The ASMX service must be available to generate the proxy code, because it is used to retrieve the updated metadata.</span></span> <span data-ttu-id="6f38b-116">Yazılan proxy'yi oluşturmak için istemci dizinindeki bir komut isteminden aşağıdaki komutu çalıştırın.</span><span class="sxs-lookup"><span data-stu-id="6f38b-116">Run the following command from a command prompt in the client directory to generate the typed proxy.</span></span>  
  
```console  
svcutil.exe /n:http://Microsoft.ServiceModel.Samples,Microsoft.ServiceModel.Samples http://localhost/servicemodelsamples/service.svc?wsdl /out:generatedClient.cs  
```  
  
 <span data-ttu-id="6f38b-117">Oluşturulan istemciyi kullanarak, uygun adresi ve bağlamayı yapılandırarak bir hizmet bitiş noktasına erişebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="6f38b-117">By using the generated client, you can access a service endpoint by configuring the appropriate address and binding.</span></span> <span data-ttu-id="6f38b-118">Hizmet gibi istemci de iletişim kurmak için bitiş noktasını belirtmek için bir yapılandırma dosyası (App.config) kullanır.</span><span class="sxs-lookup"><span data-stu-id="6f38b-118">Like the service, the client uses a configuration file (App.config) to specify the endpoint to communicate with.</span></span> <span data-ttu-id="6f38b-119">İstemci bitiş noktası yapılandırması, aşağıdaki örnek yapılandırmada gösterildiği gibi hizmet bitiş noktası, bağlama ve sözleşme için mutlak bir adresten oluşur.</span><span class="sxs-lookup"><span data-stu-id="6f38b-119">The client endpoint configuration consists of an absolute address for the service endpoint, the binding, and the contract, as shown in the following sample configuration.</span></span>  
  
```xml  
<client>  
   <endpoint
      address="http://localhost/ServiceModelSamples/service.asmx"
      binding="basicHttpBinding"
      contract="Microsoft.ServiceModel.Samples.CalculatorServiceSoap" />  
</client>  
```  
  
 <span data-ttu-id="6f38b-120">İstemci uygulaması oluşturulan istemcinin bir örneğini oluşturuyor.</span><span class="sxs-lookup"><span data-stu-id="6f38b-120">The client implementation constructs an instance of the generated client.</span></span> <span data-ttu-id="6f38b-121">Oluşturulan istemci daha sonra hizmetle iletişim kurmak için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="6f38b-121">The generated client can then be used to communicate with the service.</span></span>  
  
```csharp  
// Create a client.  
CalculatorServiceSoapClient client = new CalculatorServiceSoapClient();  
  
// Call the Add service operation.  
double value1 = 100.00D;  
double value2 = 15.99D;  
double result = client.Add(value1, value2);  
Console.WriteLine("Add({0},{1}) = {2}", value1, value2, result);  
  
// Call the Subtract service operation.  
value1 = 145.00D;  
value2 = 76.54D;  
result = client.Subtract(value1, value2);  
Console.WriteLine("Subtract({0},{1}) = {2}", value1, value2, result);  
  
// Call the Multiply service operation.  
value1 = 9.00D;  
value2 = 81.25D;  
result = client.Multiply(value1, value2);  
Console.WriteLine("Multiply({0},{1}) = {2}", value1, value2, result);  
  
// Call the Divide service operation.  
value1 = 22.00D;  
value2 = 7.00D;  
result = client.Divide(value1, value2);  
Console.WriteLine("Divide({0},{1}) = {2}", value1, value2, result);  
  
//Closing the client gracefully closes the connection and cleans up resources.  
client.Close();  
  
Console.WriteLine();  
Console.WriteLine("Press <ENTER> to terminate client.");  
Console.ReadLine();  
```  
  
 <span data-ttu-id="6f38b-122">Örneği çalıştırdığınızda, işlem istekleri ve yanıtları istemci konsol penceresinde görüntülenir.</span><span class="sxs-lookup"><span data-stu-id="6f38b-122">When you run the sample, the operation requests and responses are displayed in the client console window.</span></span> <span data-ttu-id="6f38b-123">İstemciyi kapatmak için istemci penceresinde ENTER tuşuna basın.</span><span class="sxs-lookup"><span data-stu-id="6f38b-123">Press ENTER in the client window to shut down the client.</span></span>  
  
