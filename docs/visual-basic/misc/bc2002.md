# <a name="source-file-filename-specified-multiple-times"></a>Kaynak dosya \<filename> birden çok kez belirtildi

Bir dosya birden çok kez belirtildi.  
  
 **Hata kimliği:** BC2002  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz dosya belirticilerini kaldırın.
