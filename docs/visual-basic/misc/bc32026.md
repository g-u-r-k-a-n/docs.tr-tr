# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>Belirticiler ve öznitelikler ' namespace ' deyimlerinde geçerli değildir
Ad alanları, program blokları ve programlama öğelerinin hiyerarşik organizasyonu için kullanılır. Herhangi bir değiştirici veya öznitelik almaz.  
  
 **Hata kimliği:** BC32026  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm anahtar sözcükleri ve öznitelik bloklarını `Namespace` deyimden kaldırın.
