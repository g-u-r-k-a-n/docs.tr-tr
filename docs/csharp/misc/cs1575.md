# <a name="compiler-error-cs1575"></a><span data-ttu-id="9ba58-103">Derleyici hatası CS1575</span><span class="sxs-lookup"><span data-stu-id="9ba58-103">Compiler Error CS1575</span></span>

<span data-ttu-id="9ba58-104">Stackalloc ifadesi, türden sonra [] gerektirir</span><span class="sxs-lookup"><span data-stu-id="9ba58-104">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="9ba58-105">[Stackalloc](../language-reference/operators/stackalloc.md)ile istenen ayırmanın boyutu köşeli ayraç içinde belirtilmelidir.</span><span class="sxs-lookup"><span data-stu-id="9ba58-105">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="9ba58-106">Aşağıdaki örnek CS1575 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9ba58-106">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
