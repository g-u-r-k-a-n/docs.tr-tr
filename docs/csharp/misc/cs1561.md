# <a name="compiler-error-cs1561"></a>Derleyici hatası CS1561

Çıkış dosya adı çok uzun veya geçersiz  
  
 Çıkış dosyası adı 256 karakterden uzun olamaz ve diğer bir deyişle, Euro simgesi, soru işareti veya ters eğik çizgi gibi geçersiz karakterler içeremez.
