# <a name="compiler-error-cs0412"></a><span data-ttu-id="df314-103">Derleyici hatası CS0412</span><span class="sxs-lookup"><span data-stu-id="df314-103">Compiler Error CS0412</span></span>

<span data-ttu-id="df314-104">' Generic ': bir parametre veya yerel değişken bir yöntem türü parametresiyle aynı ada sahip olamaz</span><span class="sxs-lookup"><span data-stu-id="df314-104">'generic': a parameter or local variable cannot have the same name as a method type parameter</span></span>  
  
 <span data-ttu-id="df314-105">Bir genel yöntemin tür parametresi ve yöntemdeki bir yerel değişken ya da yöntemin parametrelerinden biri arasında bir ad çakışması var.</span><span class="sxs-lookup"><span data-stu-id="df314-105">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="df314-106">Bu hatayı önlemek için, çakışan parametreleri veya yerel değişkenleri yeniden adlandırın.</span><span class="sxs-lookup"><span data-stu-id="df314-106">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="df314-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="df314-107">Example</span></span>  

 <span data-ttu-id="df314-108">Aşağıdaki örnek CS0412 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="df314-108">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
