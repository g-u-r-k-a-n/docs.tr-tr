# <a name="the-path-has-not-been-set"></a>Yol ayarlanmadı

Dosya taşıma veya kopyalama gibi bir dosya işlemi, gerekli yolu sağlamadan yapılmaya çalışıldı. Bu, yolun yanlış ayrıştırılmasından kaynaklanabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dosya yolunun sağlanmış ve doğru olduğundan emin olun.
