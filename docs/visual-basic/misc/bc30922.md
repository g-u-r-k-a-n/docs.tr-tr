# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="b8896-103">' ' \<derivedtypename> , ' ' \<type> \<constructedbasetypename> türünün erişimini \<internaltypename> derlemenin dışına genişlettiğinden ' ' öğesinden ' ' öğesinden devralma yapılamaz</span><span class="sxs-lookup"><span data-stu-id="b8896-103">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>

<span data-ttu-id="b8896-104">Türetilmiş bir sınıf veya arabirim, bir temel sınıf veya arabirime tür bağımsız değişkeni olarak kullanarak kısıtlanmış bir türün erişim düzeyini genişletmeye çalışır.</span><span class="sxs-lookup"><span data-stu-id="b8896-104">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="b8896-105">Aşağıdaki kod bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="b8896-105">The following code can generate this error.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="b8896-106">Derlemenin dışındaki kodun erişimine izin verilmiyor `restrictedStructure` .</span><span class="sxs-lookup"><span data-stu-id="b8896-106">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="b8896-107">Bununla birlikte, `derivedClass` başvuruda bulunan herhangi bir koddan erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="b8896-107">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="b8896-108">Bu nedenle, `derivedClass` `baseClass` `restrictedStructure` herhangi bir derlemede kod açığa çıkmasına neden olabileceğinden, tür bağımsız değişkeni olarak kullanıyorsa, devralma yapılamaz `restrictedStructure` .</span><span class="sxs-lookup"><span data-stu-id="b8896-108">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="b8896-109">**Hata kimliği:** BC30922</span><span class="sxs-lookup"><span data-stu-id="b8896-109">**Error ID:** BC30922</span></span>

- <span data-ttu-id="b8896-111">Türetilmiş türün, kısıtlanmış türün erişim düzeyini genişletmemesi için sınıfların veya arabirimlerin erişim düzeylerini ayarlayın.</span><span class="sxs-lookup"><span data-stu-id="b8896-111">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>

- <span data-ttu-id="b8896-113">Erişim düzeylerini ayarlayaerişemiyorsanız, temel sınıfı veya arabirimi oluştururken kısıtlanmış türü tür bağımsız değişkeni olarak kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="b8896-113">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
