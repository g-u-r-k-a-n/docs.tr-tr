# <a name="compiler-error-cs0662"></a><span data-ttu-id="e38c2-103">Derleyici hatası CS0662</span><span class="sxs-lookup"><span data-stu-id="e38c2-103">Compiler Error CS0662</span></span>

<span data-ttu-id="e38c2-104">' Method ' bir ref parametresinde yalnızca Out özniteliği belirtemez.</span><span class="sxs-lookup"><span data-stu-id="e38c2-104">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="e38c2-105">Hem gelen hem de giden özniteliklerini kullanın veya hiçbirini kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="e38c2-105">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="e38c2-106">Arabirim yönteminde yalnızca [Out](xref:System.Runtime.InteropServices.OutAttribute) özniteliğiyle [ref](../language-reference/keywords/ref.md) kullanan bir parametre bulunur.</span><span class="sxs-lookup"><span data-stu-id="e38c2-106">An interface method has a parameter that uses [ref](../language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="e38c2-107">`ref` **Out** özniteliğini kullanan bir parametre de [ın](xref:System.Runtime.InteropServices.InAttribute) özniteliğini kullanmalıdır.</span><span class="sxs-lookup"><span data-stu-id="e38c2-107">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="e38c2-108">Aşağıdaki örnek CS0662 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e38c2-108">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
