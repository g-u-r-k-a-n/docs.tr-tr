# <a name="unable-to-load-referenced-library-filename-error"></a>Başvurulan ' ' kitaplığı yüklenemiyor \<filename> : \<error>

Bu proje tarafından başvurulan bir Kitaplık yüklenemedi.  
  
 **Hata kimliği:** BC31011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosyanın belirtilen konumda olduğunu ve başka bir uygulama tarafından kilitlenmediğini doğrulayın.
