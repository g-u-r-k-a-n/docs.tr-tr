# <a name="referenced-membername-has-a-value-of-nothing"></a>Başvurulan ' \<membername> ' değeri ' Nothing ' değerine sahip

Başvurulan öğeye bir değer atanmamış ya da değere atandı `Nothing` .  
  
 **Hata kimliği:** BC30759  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öğenin adının yazıldığını ve doğru şekilde ayarlandığını doğrulayın.
