# <a name="compiler-error-cs0504"></a><span data-ttu-id="07bee-102">Derleyici Hatası CS0504</span><span class="sxs-lookup"><span data-stu-id="07bee-102">Compiler Error CS0504</span></span>
<span data-ttu-id="07bee-103">Sabit 'değişken' statik işaretlenemez</span><span class="sxs-lookup"><span data-stu-id="07bee-103">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="07bee-104">Bir değişken [const](../keywords/const.md)ise, aynı zamanda [statiktir.](../keywords/static.md)</span><span class="sxs-lookup"><span data-stu-id="07bee-104">If a variable is [const](../keywords/const.md), it is also [static](../keywords/static.md).</span></span> <span data-ttu-id="07bee-105">Bir **const** ve **statik** değişken istiyorsanız, bu değişkeni **const**olarak bildirmeniz; tek istediğiniz **statik** bir değişkense, **statik**olarak işaretlemeniz sadece .</span><span class="sxs-lookup"><span data-stu-id="07bee-105">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="07bee-106">Aşağıdaki örnek CS0504 üretir:</span><span class="sxs-lookup"><span data-stu-id="07bee-106">The following sample generates CS0504:</span></span>  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
