<span data-ttu-id="505cc-103">Bu derlemenin dışından görülemediğinden, CLS uyumluluk denetimi ' Type ' üzerinde gerçekleştirilmeyecektir.</span><span class="sxs-lookup"><span data-stu-id="505cc-103">CLS compliance checking will not be performed on 'type' because it is not visible from outside this assembly.</span></span>  
  
 <span data-ttu-id="505cc-104">Bu uyarı, özniteliğe sahip bir tür veya üye <xref:System.CLSCompliantAttribute> başka bir derlemeden görünür olmadığında oluşur.</span><span class="sxs-lookup"><span data-stu-id="505cc-104">This warning occurs when a type or a member that has the <xref:System.CLSCompliantAttribute> attribute is not visible from another assembly.</span></span> <span data-ttu-id="505cc-105">Bu hatayı çözmek için, diğer derlemeden görünmeyen herhangi bir sınıftaki veya üyenin özniteliğini kaldırın veya tür ya da üyeleri görünür hale getirin.</span><span class="sxs-lookup"><span data-stu-id="505cc-105">To resolve this error, remove the attribute on any classes or members that are not visible from the other assembly, or make the type or members visible.</span></span> <span data-ttu-id="505cc-106">CLS uyumluluğu hakkında daha fazla bilgi için bkz. ## <a name="example"></a><span data-ttu-id="505cc-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="505cc-107">Example</span></span>  

 <span data-ttu-id="505cc-108">Aşağıdaki örnek CS3019 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="505cc-108">The following sample generates CS3019:</span></span>  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line  
[CLSCompliant(true)]  // CS3019  
class C  
{  
    [CLSCompliant(false)]  // CS3019  
    void Foo()  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
