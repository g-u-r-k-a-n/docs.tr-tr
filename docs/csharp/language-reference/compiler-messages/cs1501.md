# <a name="compiler-error-cs1501"></a><span data-ttu-id="4fa8d-103">Derleyici Hatası CS1501</span><span class="sxs-lookup"><span data-stu-id="4fa8d-103">Compiler Error CS1501</span></span>
<span data-ttu-id="4fa8d-104">' Method ' metodu için hiçbir aşırı yükleme ' number ' bağımsız değişken almaz</span><span class="sxs-lookup"><span data-stu-id="4fa8d-104">No overload for method 'method' takes 'number' arguments</span></span>  
  
 <span data-ttu-id="4fa8d-105">Bir sınıf yöntemine bir çağrı yapıldı, ancak yöntemin tanımı belirtilen sayıda bağımsız değişken almıyor.</span><span class="sxs-lookup"><span data-stu-id="4fa8d-105">A call was made to a class method, but no definition of the method takes the specified number of arguments.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4fa8d-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="4fa8d-106">Example</span></span>  
 <span data-ttu-id="4fa8d-107">Aşağıdaki örnek CS1501 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="4fa8d-107">The following sample generates CS1501.</span></span>  
  
```csharp  
using System;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            ExampleClass ec = new ExampleClass();  
            ec.ExampleMethod();  
            ec.ExampleMethod(10);  
            // The following line causes compiler error CS1501 because
            // ExampleClass does not contain an ExampleMethod that takes  
            // two arguments.  
            ec.ExampleMethod(10, 20);  
        }  
    }  
  
    // ExampleClass contains two overloads for ExampleMethod. One of them
    // has no parameters and one has a single parameter.  
    class ExampleClass  
    {  
        public void ExampleMethod()  
        {  
            Console.WriteLine("Zero parameters");  
        }  
  
        public void ExampleMethod(int i)  
        {  
            Console.WriteLine("One integer parameter.");  
        }  
  
        //// To fix the error, you must add a method that takes two arguments.  
        //public void ExampleMethod (int i, int j)  
        //{  
        //    Console.WriteLine("Two integer parameters.");  
        //}  
    }  
}  
```
