# <a name="bc31043-arrays-declared-as-structure-members-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="f3b36-102">BC31043: yapı üyeleri olarak belirtilen diziler bir başlangıç boyutuyla bildirilemez</span><span class="sxs-lookup"><span data-stu-id="f3b36-102">BC31043: Arrays declared as structure members cannot be declared with an initial size</span></span>

<span data-ttu-id="f3b36-103">Yapıdaki bir dizi, başlangıç boyutuyla birlikte bildirilmiştir.</span><span class="sxs-lookup"><span data-stu-id="f3b36-103">An array in a structure is declared with an initial size.</span></span> <span data-ttu-id="f3b36-104">Herhangi bir yapı öğesini başlatamıyor ve bir dizi boyutunun bildirilmesi, tek bir başlatma biçimidir.</span><span class="sxs-lookup"><span data-stu-id="f3b36-104">You cannot initialize any structure element, and declaring an array size is one form of initialization.</span></span>

<span data-ttu-id="f3b36-105">**Hata kimliği:** BC31043</span><span class="sxs-lookup"><span data-stu-id="f3b36-105">**Error ID:** BC31043</span></span>

## <a name="example"></a><span data-ttu-id="f3b36-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="f3b36-106">Example</span></span>

<span data-ttu-id="f3b36-107">Aşağıdaki örnek BC31043 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f3b36-107">The following example generates BC31043:</span></span>

```vb
Structure DemoStruct
    Public demoArray(9) As Integer
End Structure
```

## <a name="to-correct-this-error"></a><span data-ttu-id="f3b36-108">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="f3b36-108">To correct this error</span></span>

1. <span data-ttu-id="f3b36-109">Yapıdaki diziyi dinamik (başlangıç boyutu olmadan) olarak tanımlayın.</span><span class="sxs-lookup"><span data-stu-id="f3b36-109">Define the array in your structure as dynamic (no initial size).</span></span>

2. <span data-ttu-id="f3b36-110">Belirli bir dizi boyutuna ihtiyacınız varsa, kodunuz çalışırken bir [ReDim ifadesiyle](../statements/redim-statement.md) dinamik bir diziyi yeniden Dimension uygulayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f3b36-110">If you require a certain size of array, you can redimension a dynamic array with a [ReDim Statement](../statements/redim-statement.md) when your code is running.</span></span> <span data-ttu-id="f3b36-111">Aşağıdaki örnekte bu gösterilmektedir:</span><span class="sxs-lookup"><span data-stu-id="f3b36-111">The following example illustrates this:</span></span>

    ```vb
    Structure DemoStruct
        Public demoArray() As Integer
    End Structure
    Sub UseStruct()
        Dim struct As DemoStruct
        ReDim struct.demoArray(9)
        Struct.demoArray(2) = 777
    End Sub
    ```
