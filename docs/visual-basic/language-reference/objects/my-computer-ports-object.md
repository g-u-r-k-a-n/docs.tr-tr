# <a name="mycomputerports-object"></a>My.Computer.Ports Nesnesi

## <a name="remarks"></a>Açıklamalar  

 Nesnesinin yöntemleri ve özellikleri hakkında daha fazla bilgi için `My.Computer.Ports` bkz <xref:Microsoft.VisualBasic.Devices.Ports> ..  
  
 Daha fazla bilgi için bkz. [bilgisayarın bağlantı noktalarına erişme](../../developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md).
