# <a name="modifier-is-not-valid-on-a-local-constant-declaration"></a>' \<modifier> ' yerel bir sabit bildiriminde geçerli değil

`Const`Yordam içindeki bir ifade, gibi geçersiz bir anahtar sözcük içeriyor `Overloads` .  
  
 **Hata kimliği:** BC30246  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Deyimden geçersiz anahtar sözcüğü Kaldır `Const` .
