## <a name="example"></a><span data-ttu-id="f4ae8-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="f4ae8-107">Example</span></span>  
 <span data-ttu-id="f4ae8-108">Aşağıdaki örnek CS0241 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="f4ae8-108">The following sample generates CS0241.</span></span> <span data-ttu-id="f4ae8-109">Buna ek olarak, örnek, varsayılan bağımsız değişkenleri olan bir yöntem, aşırı yükleme ile nasıl simüle edilir gösterir.</span><span class="sxs-lookup"><span data-stu-id="f4ae8-109">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {
      B x = new B();  
      x.Test();  
   }  
}  
```
