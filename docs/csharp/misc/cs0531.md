# <a name="compiler-error-cs0531"></a><span data-ttu-id="90ea3-103">Derleyici hatası CS0531</span><span class="sxs-lookup"><span data-stu-id="90ea3-103">Compiler Error CS0531</span></span>

<span data-ttu-id="90ea3-104">' üye ': arabirim üyelerinin tanımı olamaz</span><span class="sxs-lookup"><span data-stu-id="90ea3-104">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="90ea3-105">Bir [arabirimde](../language-reference/keywords/interface.md) bildirildiği Yöntemler, kendisinden devralan ve arabirimin kendisinde yer alan bir sınıfta uygulanmalıdır.</span><span class="sxs-lookup"><span data-stu-id="90ea3-105">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="90ea3-106">Aşağıdaki örnek CS0531 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="90ea3-106">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
