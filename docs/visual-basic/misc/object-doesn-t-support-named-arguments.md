# <a name="object-doesnt-support-named-arguments"></a>Nesne adlandırılmış bağımsız değişkenleri desteklemiyor
Metotları adlandırılmış bağımsız değişkenleri desteklemeyen bir nesneye erişmeye çalıştınız.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu nesnedeki yöntemlere erişirken pozitif bağımsız değişkenleri belirtin. Bağımsız değişken konumları ve türleri hakkında daha fazla bilgi için nesnenin belgelerine bakın.
