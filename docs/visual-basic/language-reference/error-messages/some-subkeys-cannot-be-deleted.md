# <a name="some-subkeys-cannot-be-deleted"></a>Bazı alt anahtarlar silinemiyor

Kayıt defteri anahtarını silme girişiminde bulunuldu, ancak bazı alt anahtarlar silinemediği için işlem başarısız oldu. Genellikle bunun nedeni izinlerin olmamasından kaynaklanır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirtilen alt anahtarları silmek için yeterli izinlere sahip olduğunuzdan emin olun.
