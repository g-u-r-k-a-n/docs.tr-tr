# <a name="serializing-xml-trees-visual-basic"></a>XML ağaçlarını serileştirme (Visual Basic)
Xml ağacının serileştirilmesi xml ağacının XML oluşturma anlamına gelir. Bir dosyaya, <xref:System.IO.TextWriter> sınıfının somut bir uygulamasına veya bir <xref:System.Xml.XmlWriter>somut uygulamasına seri hale getirebilirsiniz.  
  
 Serileştirme çeşitli yönlerini denetleyebilirsiniz. Örneğin, seri hale getirilmiş XML 'in girintisini ve bir XML bildirimi mi yazılacağını denetleyebilirsiniz.
