# <a name="object-doesnt-support-current-locale-setting-visual-basic"></a>Nesne geçerli yerel ayarı desteklemiyor (Visual Basic)

Geçerli proje için yerel ayar ayarını desteklemeyen bir nesneye erişmeye çalıştınız.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesnenin desteklediği yerel ayar ayarlarını kontrol edin.
