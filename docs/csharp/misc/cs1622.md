# <a name="compiler-error-cs1622"></a><span data-ttu-id="72a30-103">Derleyici hatası CS1622</span><span class="sxs-lookup"><span data-stu-id="72a30-103">Compiler Error CS1622</span></span>

<span data-ttu-id="72a30-104">Bir yineleyiciden değer döndüremez.</span><span class="sxs-lookup"><span data-stu-id="72a30-104">Cannot return a value from an iterator.</span></span> <span data-ttu-id="72a30-105">Bir değer döndürmek için yield return ifadesini veya yinelemeyi sonlandırmak için bırakma kesmeyi kullanın.</span><span class="sxs-lookup"><span data-stu-id="72a30-105">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="72a30-106">Yineleyici, return ifadesinin yerine yield ifadesiyle bir değer döndüren özel bir işlevdir.</span><span class="sxs-lookup"><span data-stu-id="72a30-106">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="72a30-107">Daha fazla bilgi için bkz. **yineleyiciler**.</span><span class="sxs-lookup"><span data-stu-id="72a30-107">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="72a30-108">Aşağıdaki örnek CS1622 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="72a30-108">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
