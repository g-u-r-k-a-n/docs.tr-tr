# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>XML sabit değerinde ' character ' (0x \<hexadecimal> ) karakterine izin verilmiyor

XML sabit değerinde desteklenmeyen bir karakter kullanıldı.  
  
 **Hata kimliği:** BC31176  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Desteklenmeyen karakteri kaldırın.
