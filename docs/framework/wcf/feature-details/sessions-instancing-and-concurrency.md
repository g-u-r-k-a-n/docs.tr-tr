## <a name="instancing"></a><span data-ttu-id="a7402-127">Örnek Oluşturma</span><span class="sxs-lookup"><span data-stu-id="a7402-127">Instancing</span></span>

## <a name="concurrency"></a><span data-ttu-id="a7402-149">Eşzamanlılık</span><span class="sxs-lookup"><span data-stu-id="a7402-149">Concurrency</span></span>  
 <span data-ttu-id="a7402-150">Eşzamanlılık, bir <xref:System.ServiceModel.InstanceContext> etkin olan iş parçacıklarının sayısını herhangi bir zamanda denetler.</span><span class="sxs-lookup"><span data-stu-id="a7402-150">Concurrency is the control of the number of threads active in an <xref:System.ServiceModel.InstanceContext> at any one time.</span></span> <span data-ttu-id="a7402-151">Bu, <xref:System.ServiceModel.ConcurrencyMode> numaralandırmasında <xref:System.ServiceModel.ServiceBehaviorAttribute.ConcurrencyMode%2A?displayProperty=nameWithType> kullanılarak denetlenir.</span><span class="sxs-lookup"><span data-stu-id="a7402-151">This is controlled by using the <xref:System.ServiceModel.ServiceBehaviorAttribute.ConcurrencyMode%2A?displayProperty=nameWithType> with the <xref:System.ServiceModel.ConcurrencyMode> enumeration.</span></span>  
  
 <span data-ttu-id="a7402-152">Aşağıdaki üç eşzamanlılık modu kullanılabilir:</span><span class="sxs-lookup"><span data-stu-id="a7402-152">The following three concurrency modes are available:</span></span>
