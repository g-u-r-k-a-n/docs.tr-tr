# <a name="end-property-must-be-preceded-by-a-matching-property"></a>' End Property ' öncesinde eşleşen bir ' Property ' olmalıdır

`End Property`Kodunuzda, öncesinde eşleşen bir bildirim olmadan bir ifade görüntülenir `Property` .  
  
 **Hata kimliği:** BC30431  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ise, `End Property` ifadeyi kaldırın.  
  
- Eksik `Property` yordamı bir varsa sağlayın.  
  
- Öğesini `End Property` koddaki uygun yere taşıyın.
