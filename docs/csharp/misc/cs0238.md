# <a name="compiler-error-cs0238"></a>Derleyici hatası CS0238
' Member ' korumalı olamaz çünkü bir geçersiz kılma değil  
  
 [Sealed](../language-reference/keywords/sealed.md) , [geçersiz kılma](../language-reference/keywords/override.md)ile işaretlenmeyen bir üyede kullanıldı. Daha fazla bilgi için bkz. [Devralma](../programming-guide/classes-and-structs/inheritance.md).  
  
 Aşağıdaki örnek CS0238 oluşturur:  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
