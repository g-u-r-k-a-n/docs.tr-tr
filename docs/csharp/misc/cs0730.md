## <a name="example"></a><span data-ttu-id="899ca-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="899ca-106">Example</span></span>  
 <span data-ttu-id="899ca-107">Aşağıdaki örnek CS0730 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="899ca-107">The following sample generates CS0730.</span></span> <span data-ttu-id="899ca-108">İki kaynak dosyadan oluşur.</span><span class="sxs-lookup"><span data-stu-id="899ca-108">It consists of two source files.</span></span> <span data-ttu-id="899ca-109">İlk olarak, kitaplık dosyasını derleyin `CS0730a.cs` ve `CS0730.cs` kitaplık dosyasına başvuran dosyayı derleyin.</span><span class="sxs-lookup"><span data-stu-id="899ca-109">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
