# <a name="type-in-a-module-cannot-be-declared-specifier"></a>Modül içindeki tür ' ' olarak bildirilemez \<specifier>

Bir modül içinde geçersiz değiştiricilere sahip bir tür bildirildi.  
  
 **Hata kimliği:** BC30735  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Protected` `Protected Friend` Modül içinde belirtilen türler üzerinde veya değiştiricilerini kullanmayın.
