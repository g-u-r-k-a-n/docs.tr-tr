# <a name="the-source-folder-does-not-exist"></a>Kaynak klasör yok

Mevcut olmayan bir kaynak klasöre başvuran bir işlem denediniz. Bu, yanlış bir dosya yolu sağlamaya neden olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özellikle göreli yollar kullanıyorsanız, doğru yolu kullandığınızı kontrol edin.
