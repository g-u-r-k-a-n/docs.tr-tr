# <a name="compiler-warning-level-1-cs0184"></a>Derleyici Uyarısı (düzey 1) CS0184
Verilen ifade hiçbir şekilde sağlanan (' Type ') türünde değil  
  
 Test ettiğiniz değişken ***tür*** olarak bildirilmemiş veya ***türden***türetilmediği için ifade hiçbir şekilde **doğru** olamaz.  
  
 Aşağıdaki örnek CS0184 oluşturur:  
  
```csharp  
// CS0184.cs  
// compile with: /W:1  
class MyClass  
{  
   public static void Main()  
   {  
      int i = 0;  
      if (i is string)   // CS0184  
         i++;  
   }  
}  
```
