# <a name="compiler-error-cs0743"></a><span data-ttu-id="3d7e2-103">Derleyici hatası CS0743</span><span class="sxs-lookup"><span data-stu-id="3d7e2-103">Compiler Error CS0743</span></span>
<span data-ttu-id="3d7e2-104">' On ' bağlamsal anahtar sözcüğü bekleniyordu</span><span class="sxs-lookup"><span data-stu-id="3d7e2-104">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="3d7e2-105">Yan tümce için bir model `join` `join` . `in` .. ...`on` ... `equals` , bu örnekte gösterildiği gibi:</span><span class="sxs-lookup"><span data-stu-id="3d7e2-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3d7e2-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="3d7e2-106">To correct this error</span></span>  
  
1. <span data-ttu-id="3d7e2-107">`on`Anahtar sözcüğünü `join` yan tümcesine ekleyin.</span><span class="sxs-lookup"><span data-stu-id="3d7e2-107">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3d7e2-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="3d7e2-108">Example</span></span>  
 <span data-ttu-id="3d7e2-109">Aşağıdaki kod CS0743 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3d7e2-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
