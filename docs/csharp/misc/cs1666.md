# <a name="compiler-error-cs1666"></a><span data-ttu-id="301b0-102">Derleyici Hatası CS1666</span><span class="sxs-lookup"><span data-stu-id="301b0-102">Compiler Error CS1666</span></span>
<span data-ttu-id="301b0-103">Sabit olmayan ifadelerde bulunan sabit boyutlu arabellekleri kullanamazsınız.</span><span class="sxs-lookup"><span data-stu-id="301b0-103">You cannot use fixed size buffers contained in unfixed expressions.</span></span> <span data-ttu-id="301b0-104">Sabit deyimi kullanmayı deneyin.</span><span class="sxs-lookup"><span data-stu-id="301b0-104">Try using the fixed statement.</span></span>  
  
 <span data-ttu-id="301b0-105">Sabit boyutlu arabellek kendisi sabit olmayan bir sınıf içeren bir ifadede kullanırsanız bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="301b0-105">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="301b0-106">Çalışma süresi sabit boyutlu arabellek kullanırken hatalara neden olabilir bellek erişimini en iyi duruma getirmek için sabitlenmemiş sınıf taşımak için ücretsizdir.</span><span class="sxs-lookup"><span data-stu-id="301b0-106">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="301b0-107">Bu hatayı önlemek için `fixed` deyimdeki anahtar kelimeyi kullanın.</span><span class="sxs-lookup"><span data-stu-id="301b0-107">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="301b0-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="301b0-108">Example</span></span>  
 <span data-ttu-id="301b0-109">Aşağıdaki örnek CS1666 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="301b0-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);
   }
}  
```
