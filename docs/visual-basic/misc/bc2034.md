# <a name="option-win32manifest-ignored"></a>/Win32manifest seçeneği yoksayıldı
/Win32manifest seçeneği yoksayıldı. Yalnızca hedef bir bütünleştirilmiş kod olduğunda belirtilebilir.  
  
 @No__t_1 seçeneği `module` olarak ayarlandığında Visual Basic Derleyicisi `/win32manifest` derleyici seçeneğini geçirdi.  
  
 **Hata kimliği:** BC2034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. @No__t_0 derleyici seçeneğini kaldırın veya `/target` seçeneğini `exe`, `winexe` veya `library` olarak ayarlayın.
