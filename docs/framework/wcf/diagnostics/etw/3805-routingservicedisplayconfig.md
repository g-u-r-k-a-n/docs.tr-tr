# <a name="3805---routingservicedisplayconfig"></a>3805 - RoutingServiceDisplayConfig

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|3805|  
|Anahtar sözcükler|Yönlendirme hizmetleri|  
|Düzey|Ayrıntılı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Açıklama  

 Bu olay, yönlendirme hizmeti RouteOnHeadersOnly ile yapılandırıldığında yayınlanır.  
  
## <a name="message"></a>İleti  

 Yönlendirme hizmeti RouteOnHeadersOnly: %1, SoapProcessingEnabled: %2, EnsureOrderedDispatch: %3 ile yapılandırıldı.
