# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="d947f-103">Derleyici Uyarısı (düzey 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="d947f-103">Compiler Warning (level 2) CS0469</span></span>

<span data-ttu-id="d947f-104">' Goto case ' değeri ' Type ' türüne örtük olarak dönüştürülebilir değil</span><span class="sxs-lookup"><span data-stu-id="d947f-104">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="d947f-105">Kullandığınızda `goto case` , goto case değerinden anahtar türüne örtük bir dönüştürme olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="d947f-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d947f-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="d947f-106">Example</span></span>  

 <span data-ttu-id="d947f-107">Aşağıdaki örnek CS0469 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="d947f-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
