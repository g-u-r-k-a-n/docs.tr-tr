# <a name="compiler-error-cs0761"></a><span data-ttu-id="40be7-103">Derleyici hatası CS0761</span><span class="sxs-lookup"><span data-stu-id="40be7-103">Compiler Error CS0761</span></span>

 <span data-ttu-id="40be7-105">Kısmi bir yöntemin bir uygulamaya sahip olması durumunda, genel tür kısıtlamasının Yöntem imzasında tanımlanan kısıtlamayla aynı olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="40be7-105">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
