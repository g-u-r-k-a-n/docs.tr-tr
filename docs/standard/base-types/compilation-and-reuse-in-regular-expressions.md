## <a name="the-regular-expressions-cache"></a><span data-ttu-id="d1a2a-119">Normal İfadeler Önbelleği</span><span class="sxs-lookup"><span data-stu-id="d1a2a-119">The Regular Expressions Cache</span></span>

 <span data-ttu-id="d1a2a-123">Önbelleğe alınan en fazla normal ifade sayısı `static` (`Shared` Visual Basic'teki) <xref:System.Text.RegularExpressions.Regex.CacheSize%2A?displayProperty=nameWithType> özelliğinin değerine göre belirlenir.</span><span class="sxs-lookup"><span data-stu-id="d1a2a-123">The maximum number of cached regular expressions is determined by the value of the `static` (`Shared` in Visual Basic) <xref:System.Text.RegularExpressions.Regex.CacheSize%2A?displayProperty=nameWithType> property.</span></span> <span data-ttu-id="d1a2a-124">Varsayılan olarak, normal ifade altyapısı 15'e kadar derlenmiş normal ifadeleri önbelleğe alır.</span><span class="sxs-lookup"><span data-stu-id="d1a2a-124">By default, the regular expression engine caches up to 15 compiled regular expressions.</span></span> <span data-ttu-id="d1a2a-125">Derlenen normal ifadelerin sayısı önbellek boyutunu aşıyorsa, en son kullanılan normal ifade atılır ve yeni normal ifade önbelleğe çıkar.</span><span class="sxs-lookup"><span data-stu-id="d1a2a-125">If the number of compiled regular expressions exceeds the cache size, the least recently used regular expression is discarded and the new regular expression is cached.</span></span>
