# <a name="compiler-error-cs1655"></a><span data-ttu-id="6fbe9-102">Derleyici Hatası CS1655</span><span class="sxs-lookup"><span data-stu-id="6fbe9-102">Compiler Error CS1655</span></span>
<span data-ttu-id="6fbe9-103">'Yalnızca değişken türü' olduğu için ref veya dış bağımsız değişken olarak 'değişken' alanlarını geçiremez</span><span class="sxs-lookup"><span data-stu-id="6fbe9-103">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="6fbe9-104">Bu hata, [foreach](../language-reference/keywords/foreach-in.md) değişkeninin, [bir kullanarak](../language-reference/keywords/using-statement.md) değişkenin veya [sabit](../language-reference/keywords/fixed-statement.md) bir değişkenin bir üyesini ref veya out bağımsız değişkeni olarak bir işleve geçirmeye çalışıyorsanız oluşur.</span><span class="sxs-lookup"><span data-stu-id="6fbe9-104">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="6fbe9-105">Bu değişkenler bu bağlamlarda salt okunur olarak kabul edilir, bu izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="6fbe9-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="6fbe9-106">Aşağıdaki örnek CS1655 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6fbe9-106">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
