# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Lambda ifadelerinin değerlendirmesi hata ayıklayıcıda geçerli değil

Bir lambda ifadesi, hata ayıklama oturumu sırasında **Watch** veya **anında** pencereye girilmiştir. Lambda ifadeleri hata ayıklama penceresinde kullanılamaz.  
  
 **Hata kimliği:** BC36627  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kod düzenleyicisinde test lambda ifadeleri.
