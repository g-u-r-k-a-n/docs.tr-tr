 <span data-ttu-id="97e2c-105">Kısıtlama, tür `Structure` bağımsız değişkeninin tür parametresine geçirilen değer türü olmasını gerektirir.</span><span class="sxs-lookup"><span data-stu-id="97e2c-105">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="97e2c-106">Ancak, bir değer türü uygulanamıyor veya devralınamaz, bu nedenle onu uygulamak veya ondan devralması için diğer tür parametresinin gerekli olduğu bir kısıtlama olarak kullanılması anlamlı değildir.</span><span class="sxs-lookup"><span data-stu-id="97e2c-106">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="97e2c-107">Bu durumun yalnızca anlamlı yorumu, her iki tür bağımsız değişkeni de tam olarak aynı türde olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="97e2c-107">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="97e2c-108">Bu durumda, genel türlerinizin yalnızca bir tür parametresi olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="97e2c-108">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="97e2c-109">Aşağıdaki ifade bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="97e2c-109">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="97e2c-110">Geçirilen tür `t2` `t1` `t1` bir değer türü olması gerektiğinden, geçirilen tür ' e uygulanamıyor veya devralma türünü alamaz.</span><span class="sxs-lookup"><span data-stu-id="97e2c-110">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>
