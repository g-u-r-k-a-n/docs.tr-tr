# <a name="compiler-error-cs0555"></a><span data-ttu-id="03c35-103">Derleyici hatası CS0555</span><span class="sxs-lookup"><span data-stu-id="03c35-103">Compiler Error CS0555</span></span>

<span data-ttu-id="03c35-104">Kullanıcı tanımlı işleç kapsayan türdeki bir nesneyi alamaz ve kapsayan türdeki bir nesneye dönüştüremez</span><span class="sxs-lookup"><span data-stu-id="03c35-104">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>

<span data-ttu-id="03c35-105">Kapsayan sınıfın değerlerine Kullanıcı tanımlı Dönüştürmelere izin verilmiyor; Böyle bir işlece gerek yoktur.</span><span class="sxs-lookup"><span data-stu-id="03c35-105">User-defined conversions to values of the enclosing class are not allowed; you do not need such an operator.</span></span>

<span data-ttu-id="03c35-106">Aşağıdaki örnek CS0555 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="03c35-106">The following sample generates CS0555:</span></span>

```csharp
// CS0555.cs
public class MyClass
{
   // delete the following operator to resolve this CS0555
   public static implicit operator MyClass(MyClass aa)   // CS0555
   {
      return new MyClass();
   }

   public static void Main()
   {
   }
}
```
