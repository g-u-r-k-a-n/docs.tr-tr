 <span data-ttu-id="c2ae8-105">Veri kaynağının hiyerarşik bir nesne modeli olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="c2ae8-105">The data source must have a hierarchical object model.</span></span> <span data-ttu-id="c2ae8-106">Örneğin, bir `Employee` sınıfı bir çalışanın doğrudan raporu olan çalışan nesnelerinin bir koleksiyonunu içerebilir.</span><span class="sxs-lookup"><span data-stu-id="c2ae8-106">For example, an `Employee` class might contain a collection of Employee objects that are the direct reports of an employee.</span></span> <span data-ttu-id="c2ae8-107">Veriler hiyerarşik olmayan bir şekilde gösteriliyorsa, verilerin hiyerarşik bir gösterimini derlemeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="c2ae8-107">If the data is represented in a way that is not hierarchical, you must build a hierarchical representation of the data.</span></span>
