# <a name="systemservicemodelactivationwebhostprotocolmisconfigured"></a>System.ServiceModel.Activation.WebHostProtocolMisconfigured
Protokol, Windows Işlem etkinleştirme hizmeti 'nde (WAS) yanlış yapılandırılmıştır.  
  
## <a name="description"></a>Açıklama  
 Bu hata, Aktarım Protokolü yanlış yapılandırıldığında oluşur.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Web. config dosyasındaki aktarım yapılandırma parametrelerini denetleyin ve onarın.
