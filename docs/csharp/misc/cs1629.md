 <span data-ttu-id="a35f7-105">C# dil belirtimi Yineleyicilerde güvenli olmayan koda izin vermiyor.</span><span class="sxs-lookup"><span data-stu-id="a35f7-105">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="a35f7-106">Aşağıdaki örnek CS1629 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a35f7-106">The following sample generates CS1629:</span></span>  
  
```csharp

class C
{  
   IEnumerator<int> IteratorMeth() {

      }  
   }  
}
```
