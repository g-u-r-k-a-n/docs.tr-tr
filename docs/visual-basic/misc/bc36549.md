# <a name="anonymous-type-property-propertyname-cannot-be-used-in-the-definition-of-a-lambda-expression-within-the-same-initialization-list"></a><span data-ttu-id="01a4b-102">' ' Anonim tür özelliği \<propertyname> aynı başlatma listesindeki bir lambda ifadesinin tanımında kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="01a4b-102">Anonymous type property '\<propertyname>' cannot be used in the definition of a lambda expression within the same initialization list</span></span>

<span data-ttu-id="01a4b-103">Anonim bir türün başlatma listesinde tanımlanan özellikler aynı listedeki bir lambda ifadesi tanımının parçası olamaz.</span><span class="sxs-lookup"><span data-stu-id="01a4b-103">Properties defined in the initialization list of an anonymous type cannot be part of a lambda expression definition in the same list.</span></span> <span data-ttu-id="01a4b-104">Örneğin, aşağıdaki kodda, özelliği `Num` tanımına dahil edilemez `LambdaFun` .</span><span class="sxs-lookup"><span data-stu-id="01a4b-104">For example, in the following code, property `Num` cannot be included in the definition of `LambdaFun`.</span></span>

```vb
' Not valid.
'Dim anon = New With {.Num = 4, .LambdaFun = Function() .Num > 0}
```

<span data-ttu-id="01a4b-105">**Hata kimliği:** BC36549</span><span class="sxs-lookup"><span data-stu-id="01a4b-105">**Error ID:** BC36549</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="01a4b-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="01a4b-106">To correct this error</span></span>

1. <span data-ttu-id="01a4b-107">Anonim türü iki parçaya bölmeyi göz önünde bulundurun:</span><span class="sxs-lookup"><span data-stu-id="01a4b-107">Consider splitting the anonymous type into two parts:</span></span>

    ```vb
    Dim anon1 = New With {.Num = 4}
    Dim anon2 = New With {.LambdaFun = Function() anon1.Num > 0}
    ' - or -
    Dim anon3 = New With {.lambdaFun = Function(n As Integer) n > 0}
    Console.WriteLine((anon2.LambdaFun)())
    Console.WriteLine(anon3.lambdaFun(anon1.Num))
    anon1.Num = -5
    Console.WriteLine((anon2.LambdaFun)())
    Console.WriteLine(anon3.lambdaFun(anon1.Num))
    ```

    <span data-ttu-id="01a4b-108">`anon1.Num`Özellik olarak bildirirseniz `Key` , değerinin değiştirilemeyeceğini unutmayın.</span><span class="sxs-lookup"><span data-stu-id="01a4b-108">Note that if you declare `anon1.Num` as a `Key` property, its value cannot be changed.</span></span>

2. <span data-ttu-id="01a4b-109">Diğer bir seçenek de, anonim tür özelliğine erişmek için normal işlev deyimlerinin kullanılması:</span><span class="sxs-lookup"><span data-stu-id="01a4b-109">An alternative is to use a regular function statement to access the anonymous type property:</span></span>

    ```vb
    Function testNum(ByVal n As Integer) As Boolean
        Return n > 0
    End Function
    Console.WriteLine(testNum(anon1.Num))
    ```

3. <span data-ttu-id="01a4b-110">Benzer şekilde, anonim bir tür dışında tanımlanan bir Lambda işlevi de kullanabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="01a4b-110">Similarly, you can use a lambda function that is defined outside the anonymous type:</span></span>

    ```vb
    Dim lambdaFun1 = Function() anon1.Num > 0
    Dim lambdaFun2 = Function(n As Integer) n > 0
    ```
