## <a name="syntax"></a>Söz dizimi  
  
```cpp  
HRESULT GetCount (  
    [out] ULONG   *pcelt  
);  
```  
  
## <a name="parameters"></a>Parametreler

 dışı Numaralandırmadaki öğelerin sayısına yönelik bir işaretçi.
