# <a name="parallel-diagnostic-tools"></a>Paralel Tanılama Araçları
Visual Studio, hata ayıklama ve profil oluşturma için çok iş parçacığı uygulamaları için kapsamlı destek sağlar.  
  
## <a name="debugging"></a>Hata ayıklama  
 Visual Studio hata ayıklama paralel uygulamaları hata ayıklama için yeni pencereler ekler. Daha fazla bilgi edinmek için aşağıdaki kaynaklara bakın:  
  
- [Paralel Yığınlar Penceresini Kullanma](/visualstudio/debugger/using-the-parallel-stacks-window)  
  
- [Görevleri Penceresini Kullanma](/visualstudio/debugger/using-the-tasks-window)  
  
- [Walkthrough: Paralel Uygulama Hata Ayıklama](/visualstudio/debugger/walkthrough-debugging-a-parallel-application).  
  
## <a name="profiling"></a>Profil oluşturma  
 EşzamanlıLık Görselleştiricisi rapor görünümleri, paralel bir programdaki iş parçacıklarının birbiriyle ve sistemdeki diğer işlemlerden gelen iş parçacıklarıyla nasıl etkileşimde bulundurak olduğunu görselleştirmenizi sağlar. Daha fazla bilgi için [Concurrency Visualizer'a](/visualstudio/profiling/concurrency-visualizer)bakın.
