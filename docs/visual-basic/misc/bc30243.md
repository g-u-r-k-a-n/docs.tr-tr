# <a name="modifier-is-not-valid-on-an-event-declaration"></a>' \<modifier> ' olay bildiriminde geçerli değil

Bir `Event` ifade gibi geçersiz bir anahtar sözcük içeriyor `ReadOnly` .  
  
 **Hata kimliği:** BC30243  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Deyimden geçersiz anahtar sözcüğü Kaldır `Event` .
