# <a name="compiler-error-cs0120"></a><span data-ttu-id="980c2-102">Derleyici Hatası CS0120</span><span class="sxs-lookup"><span data-stu-id="980c2-102">Compiler Error CS0120</span></span>

<span data-ttu-id="980c2-103">Statik olmayan alan, yöntem veya özellik 'üye' için bir nesne başvurusu gereklidir</span><span class="sxs-lookup"><span data-stu-id="980c2-103">An object reference is required for the nonstatic field, method, or property 'member'</span></span>

 <span data-ttu-id="980c2-104">Statik olmayan bir alan, yöntem veya özellik kullanmak için önce bir nesne örneği oluşturmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="980c2-104">In order to use a non-static field, method, or property, you must first create an object instance.</span></span> <span data-ttu-id="980c2-105">Statik yöntemler hakkında daha fazla bilgi için statik [sınıflar ve Statik Sınıf Üyeleri'ne](../../programming-guide/classes-and-structs/static-classes-and-static-class-members.md)bakın.</span><span class="sxs-lookup"><span data-stu-id="980c2-105">For more information about static methods, see [Static Classes and Static Class Members](../../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span> <span data-ttu-id="980c2-106">Sınıf örnekleri oluşturma hakkında daha fazla bilgi için [Bkz. Örnek Oluşturucular.](../../programming-guide/classes-and-structs/instance-constructors.md)</span><span class="sxs-lookup"><span data-stu-id="980c2-106">For more information about creating instances of classes, see [Instance Constructors](../../programming-guide/classes-and-structs/instance-constructors.md).</span></span>

 <span data-ttu-id="980c2-107">Aşağıdaki örnek CS0120 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="980c2-107">The following sample generates CS0120:</span></span>

```csharp
// CS0120_1.cs
public class MyClass
{
    // Non-static field.
    public int i;
    // Non-static method.
    public void f() {}
    // Non-static property.
    int Prop
    {
        get
        {
            return 1;
        }
    }

    public static void Main()
    {
        i = 10;   // CS0120
        f();   // CS0120
        int p = Prop;   // CS0120
        // Try the following lines instead.
        // var mc = new MyClass();
        // mc.i = 10;
        // mc.f();
        // int p = mc.Prop;
    }
}
```

 <span data-ttu-id="980c2-108">Statik bir yöntemden statik olmayan bir yönteme çağrı varsa, Aşağıdaki gibi CS0120 de oluşturulur:</span><span class="sxs-lookup"><span data-stu-id="980c2-108">CS0120 will also be generated if there is a call to a non-static method from a static method, as follows:</span></span>

```csharp
// CS0120_2.cs
// CS0120 expected
using System;

public class MyClass
{
    public static void Main()  
    {  
        TestCall();   // CS0120
        // To call a non-static method from Main,
        // first create an instance of the class.
        // Use the following two lines instead:
        // var anInstanceofMyClass = new MyClass();
        // anInstanceofMyClass.TestCall();
   }

   public void TestCall()
   {
   }
}
```

 <span data-ttu-id="980c2-109">Benzer şekilde, statik bir yöntem, sınıfın bir örneğini aşağıdaki gibi açıkça vermediğiniz sürece örnek yöntemi ni çağıramaz:</span><span class="sxs-lookup"><span data-stu-id="980c2-109">Similarly, a static method cannot call an instance method unless you explicitly give it an instance of the class, as follows:</span></span>

```csharp
// CS0120_3.cs
using System;

public class MyClass
{
   public static void Main()
   {
      DoIt("Hello There");   // CS0120
   }
  
   private void DoIt(string sText)
   // You could also add the keyword static to the method definition.
   // private static void DoIt(string sText)
   {
      Console.WriteLine(sText);
   }
}
```
