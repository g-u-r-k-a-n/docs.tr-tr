# <a name="elementname-is-not-a-method-parameter"></a>' \<elementname> ' bir yöntem parametresi değil
Yordam çağrısı, yöntemi tarafından tanımlanmayan ek bir bağımsız değişken sağlar.  
  
 **Hata kimliği:** BC36568  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken listesinden gereksiz bağımsız değişkeni kaldırın.
