# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="61a9f-103">' ' ' ByRef ' parametresi \<parametername> lambda ifadesinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="61a9f-103">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>

<span data-ttu-id="61a9f-104">OR işlevi içinde belirtilen bir lambda ifadesi, `Sub` ya da işlevinin herhangi bir `ByRef` parametresini kullanamaz `Sub` .</span><span class="sxs-lookup"><span data-stu-id="61a9f-104">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="61a9f-105">Örneğin, `ByRef` parametre lambda ifadesinde kullanıldığı için aşağıdaki kod bu hataya neden olur `n` .</span><span class="sxs-lookup"><span data-stu-id="61a9f-105">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="61a9f-106">**Hata kimliği:** BC36639</span><span class="sxs-lookup"><span data-stu-id="61a9f-106">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="61a9f-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="61a9f-107">To correct this error</span></span>  
  
- <span data-ttu-id="61a9f-108">`ByRef`Parametreyi yerel bir değişkene atayın ve aşağıdaki kodda gösterildiği gibi lambda ifadesinde yerel değişkeni kullanın:</span><span class="sxs-lookup"><span data-stu-id="61a9f-108">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
