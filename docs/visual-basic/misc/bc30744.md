# <a name="referenced-array-element-has-a-value-of-nothing"></a>Başvurulan dizi öğesinde ' Nothing ' değeri bulunuyor
Başvurulan dizi öğesi ayarlanmamış veya daha önce olarak ayarlanmış `Nothing` .  
  
 **Hata kimliği:** BC30744  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizideki değerlerin doğru şekilde ayarlandığını doğrulayın.
