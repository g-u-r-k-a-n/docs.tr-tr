# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Bu sistem Japonca yerel ayar desteği içermiyor
`VbStrConv.Japanese` numaralandırma üyesini kullanmaya çalışıyorsunuz, ancak sisteminiz Japonca yerel ayarı desteklemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Japonca yerel ayarı desteklemek için makineyi yükseltin veya uygulamayı yerel ayarı destekleyen bir makinede çalıştırın.
