# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>Win32 bildirimi eklenirken hata oluştu: bildirim dosyası \<file> bulunamıyor
Derleyici seçeneği için belirtilen dosya `/win32manifest` bulunamıyor.  
  
 **Hata kimliği:** BC31192  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Derleyici seçeneği için geçerli bir dosya adı ve yol belirtin `/win32manifest` .
