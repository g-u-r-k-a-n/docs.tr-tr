# <a name="compiler-error-cs0200"></a>Derleyici Hatası CS0200
Özellik veya dizinleyici 'özellik' atanamıyor - sadece okunur

Bir [mülke](../programming-guide/classes-and-structs/using-properties.md)değer atama girişiminde bulunuldu, ancak özellik temalı bir erişime sahip değil veya atama oluşturucudışındaydı. Ayarlanmış bir erişimci ekleyerek hatayı giderin. Daha fazla bilgi için okuma [yazma özelliklerini nasıl beyan edin ve kullanacağınızı](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)öğrenin.

## <a name="examples"></a>Örnekler
Aşağıdaki örnek CS0200 üretir:

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

Aşağıdaki örnek [otomatik olarak uygulanan özellikleri](../programming-guide/classes-and-structs/auto-implemented-properties.md)kullanır , [nesne başlatılması](../programming-guide/classes-and-structs/object-and-collection-initializers.md), ve hala CS0200 oluşturur:

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

Yalnızca okunan bir özellik veya dizinleyici 'özellik'e atama, ayarlanmış bir erişimci ekleyerek veya nesnenin oluşturucusundaki özelliğe atayarak elde edilebilir.

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
