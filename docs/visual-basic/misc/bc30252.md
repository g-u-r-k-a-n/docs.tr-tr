# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>' End Interface ' öncesinde eşleşen bir ' Interface ' olmalıdır

`End Interface`Karşılık gelen bir ifade olmadan bir ifade oluşur `Interface` . `End Interface` önce karşılık gelen bir `Interface` deyimden önce gelmelidir.  
  
 **Hata kimliği:** BC30252  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Önceki bir deyimin yanlış yazılmış olup olmadığını kontrol edin `Interface` veya geçersiz.  
  
2. `Interface`Üyelerin doğru biçimlendirildiğinden emin olun.
