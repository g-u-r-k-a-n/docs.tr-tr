# <a name="compiler-error-cs0538"></a><span data-ttu-id="ce607-103">Derleyici hatası CS0538</span><span class="sxs-lookup"><span data-stu-id="ce607-103">Compiler Error CS0538</span></span>

<span data-ttu-id="ce607-104">Açık arabirim bildirimindeki ' name ' bir arabirim değil</span><span class="sxs-lookup"><span data-stu-id="ce607-104">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="ce607-105">Açıkça bir [arabirim](../language-reference/keywords/interface.md)bildirmek için bir girişimde bulunuldu, ancak arabirim belirtilmedi.</span><span class="sxs-lookup"><span data-stu-id="ce607-105">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="ce607-106">Aşağıdaki örnek CS0538 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ce607-106">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
