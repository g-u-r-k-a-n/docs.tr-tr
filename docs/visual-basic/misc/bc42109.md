# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>' \<variablename> ' Değişkenine bir değer atanmadan önce kullanılır (Visual Basic hatası)

Bir değişken, bir değer atanmadan önce kullanılıyor, sonuç olarak <xref:System.NullReferenceException> .  
  
 **Hata kimliği:** BC42109  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değişkene bir değer atandığından emin olun.
