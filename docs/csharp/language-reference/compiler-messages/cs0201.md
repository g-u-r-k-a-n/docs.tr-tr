## <a name="example"></a><span data-ttu-id="1deaf-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="1deaf-107">Example</span></span>

 <span data-ttu-id="1deaf-108">Aşağıdaki örnek CS0201 oluşturur, çünkü 2 \* 3 bir ifadedir, bir deyim değil.</span><span class="sxs-lookup"><span data-stu-id="1deaf-108">The following sample generates CS0201 because 2 \* 3 is an expression, not a statement.</span></span> <span data-ttu-id="1deaf-109">Kodu derlemek için ifadenin değerini bir değişkene atamayı deneyin.</span><span class="sxs-lookup"><span data-stu-id="1deaf-109">To make the code compile, try assigning the value of the expression to a variable.</span></span>  

```csharp
// CS0201.cs  
public class MainClass  
{  
   public static void Main()  
   {  
      2 * 3;   // CS0201  
      // Try the following line instead.  
      //   int i = 2 * 3;  
   }  
}  
```

## <a name="example"></a><span data-ttu-id="1deaf-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="1deaf-110">Example</span></span>

 <span data-ttu-id="1deaf-111">Bir artış işlemi yle parametrelendirilmiş olsa bile, tek başına denetlenen bir deyim olmadığından aşağıdaki örnek CS0201 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="1deaf-111">The following sample generates CS0201 because checked by itself is not a statement, even though it is parameterized by an increment operation.</span></span>  

```csharp
// CS0201_b.cs  
// compile with: /target:library  
public class MyList<T>
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
