# <a name="compiler-error-cs0113"></a><span data-ttu-id="0c95a-103">Derleyici hatası CS0113</span><span class="sxs-lookup"><span data-stu-id="0c95a-103">Compiler Error CS0113</span></span>
<span data-ttu-id="0c95a-104">Override olarak işaretlenen ' function ' üyesi New veya Virtual olarak işaretlenemez</span><span class="sxs-lookup"><span data-stu-id="0c95a-104">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="0c95a-105">Bir yöntemi [New](../language-reference/keywords/new-modifier.md) ve [override](../language-reference/keywords/override.md) anahtar sözcükleriyle işaretlemek birbirini dışlar.</span><span class="sxs-lookup"><span data-stu-id="0c95a-105">It is mutually exclusive to mark a method with the [new](../language-reference/keywords/new-modifier.md) and [override](../language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="0c95a-106">Aşağıdaki örnek CS0113 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0c95a-106">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
