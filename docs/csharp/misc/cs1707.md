# <a name="compiler-warning-level-1-cs1707"></a>Derleyici Uyarısı (düzey 1) CS1707

' DelegateName ' temsilcisi, yeni dil kuralları nedeniyle ' MethodName2 ' yerine ' MethodName1 ' ile bağlantılı  
  
 C# 2,0, bir temsilciyi bir yönteme bağlamak için yeni kurallar uygular. Geçmişte Not olmayan ek bilgiler göz önünde bulundurulmaz. Bu uyarı, temsilcinin daha önce bağlandığına göre metodun farklı bir aşırı yüklemesine bağlandığını gösterir. Temsilcinin gerçekten ' MethodName2 ' yerine ' MethodName1 ' öğesine bağlanması gerektiğini doğrulamak isteyebilirsiniz.  
  
 Derleyicinin hangi yönteme bağlanılacağını nasıl belirlediğini belirleyen bir açıklama için, bkz. [Temsilcilerde varyans kullanma](../programming-guide/concepts/covariance-contravariance/using-variance-in-delegates.md).
