# <a name="static-local-variable-variablename-is-already-declared"></a>' ' Statik yerel değişkeni \<variablename> zaten tanımlanmış
Aynı ada sahip başka bir statik yerel değişken zaten tanımlanmış.  
  
 **Hata kimliği:** BC31401  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Gereksiz statik yerel bildirimleri kaldırın.  
  
2. Her statik yerel değişkene benzersiz bir ad verin.
