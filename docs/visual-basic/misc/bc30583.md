# <a name="methodname-cannot-be-implemented-more-than-once"></a>' \<methodname> ' birden çok kez uygulanamaz

Zaten uygulanmış bir yöntemi uygulamaya çalıştınız.  
  
 **Hata kimliği:** BC30583  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Doğru yazdığınızdan emin olmak için yöntem adını denetleyin.  
  
2. Yöntem gereksiz ise, ek bildirimini kaldırın.
