# <a name="namespace-or-type-name-has-already-been-imported"></a>' ' Ad alanı veya türü \<name> zaten içeri aktarıldı
Bir ad alanını birden çok kez içeri aktarma girişiminde bulunuldu.  
  
 **Hata kimliği:** BC31051  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz `Imports` deyimleri kaldırın.
