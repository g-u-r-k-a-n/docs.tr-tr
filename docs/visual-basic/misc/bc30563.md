# <a name="name-is-ambiguous-in-the-application-objects-list"></a>' ' \<name> , ' ' uygulama nesnelerinde belirsiz \<list>

Belirsiz bir ad sağladınız ve bu nedenle başka bir adla çakışıyor. Visual Basic derleyicisinde herhangi bir çakışma çözümleme kuralı yok; adları kendiniz kesinleştirmeniz gerekir. Hata iletisinde belirtilen liste, adın belirsiz olduğu uygulama nesnelerinin ayrıntılarına bakın.  
  
 **Hata kimliği:** BC30563  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Adı tam olarak nitelendirin.
