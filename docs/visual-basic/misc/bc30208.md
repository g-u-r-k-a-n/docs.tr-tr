# <a name="compare-expected"></a>' Compare ' bekleniyor
Bir `Option` ifade `Text` or belirtir `Binary` , ancak değil `Compare` .  
  
 **Hata kimliği:** BC30208  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Compare`Deyimin anahtar sözcüğünü ekleyin `Option` ; Örneğin, `Option Compare Binary` .
