# <a name="bindingerror"></a>BindingError

Kimlik: 159  
  
 Önem derecesi: Hata  
  
 Kategori: SMSvcHost  
  
## <a name="description"></a>Açıklama  

 Bu olay, belirli bir sitenin belirli bir bağlamasını ayrıştırırken bir hata oluştuğunu gösterir, bu nedenle protokol site için geçici olarak devre dışı bırakılır. Olayda bağlama, kaynak, özel durum, işlem adı ve işlem KIMLIĞI listelenir.
