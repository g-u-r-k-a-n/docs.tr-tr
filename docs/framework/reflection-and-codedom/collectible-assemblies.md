<span data-ttu-id="4fe60-111">Bir tahsil derlemesinin ömrü, içerdiği türlere ve bu türlerden oluşturulan nesnelere yapılan başvuruların varlığıyla denetlenir.</span><span class="sxs-lookup"><span data-stu-id="4fe60-111">The lifetime of a collectible assembly is controlled by the existence of references to the types it contains and the objects that are created from those types.</span></span> <span data-ttu-id="4fe60-112">Ortak dil çalışma süresi, aşağıdakilerden biri veya birkaçı var olduğu`T` sürece (derlemede tanımlanan herhangi bir türdür) bir derlemeyi boşaltmaz:</span><span class="sxs-lookup"><span data-stu-id="4fe60-112">The common language runtime does not unload an assembly as long as one or more of the following exist (`T` is any type that is defined in the assembly):</span></span>

- <span data-ttu-id="4fe60-121">Kodu yürüterek `T` hala erişilebilen `T1` başka bir dinamik olarak tanımlanmış türtarafından statik bir başvuru.</span><span class="sxs-lookup"><span data-stu-id="4fe60-121">A static reference to `T` by another dynamically defined type `T1` that is still reachable by executing code.</span></span> <span data-ttu-id="4fe60-122">Örneğin, `T1` bir yöntemdeki `T`parametre türünden türetilebilir veya `T` parametre türü `T1`olabilir.</span><span class="sxs-lookup"><span data-stu-id="4fe60-122">For example, `T1` might derive from `T`, or `T` might be the type of a parameter in a method of `T1`.</span></span>

## <a name="restrictions-on-collectible-assemblies"></a><span data-ttu-id="4fe60-133">Tahsil edilebilir montajlarla ilgili kısıtlamalar</span><span class="sxs-lookup"><span data-stu-id="4fe60-133">Restrictions on collectible assemblies</span></span>

<span data-ttu-id="4fe60-134">Aşağıdaki kısıtlamalar tahsil montajları için geçerlidir:</span><span class="sxs-lookup"><span data-stu-id="4fe60-134">The following restrictions apply to collectible assemblies:</span></span>

- <span data-ttu-id="4fe60-135">**Statik referanslar** Olağan dinamik derlemedeki türlerin, tahsil edilebilir bir derlemede tanımlanan türlere statik başvuruları olamaz.</span><span class="sxs-lookup"><span data-stu-id="4fe60-135">**Static references** Types in an ordinary dynamic assembly cannot have static references to types that are defined in a collectible assembly.</span></span> <span data-ttu-id="4fe60-136">Örneğin, tahsil edilebilir bir derlemedeki bir türden devralınan sıradan <xref:System.NotSupportedException> bir tür tanımlarsanız, bir özel durum atılır.</span><span class="sxs-lookup"><span data-stu-id="4fe60-136">For example, if you define an ordinary type that inherits from a type in a collectible assembly, a <xref:System.NotSupportedException> exception is thrown.</span></span> <span data-ttu-id="4fe60-137">Tahsil edilebilir bir derlemedeki bir tür, başka bir tahsil edilebilir derlemedeki bir türe statik başvurulara sahip olabilir, ancak bu, başvurulan derlemenin kullanım ömrünü başvuru derlemesinin ömrüne kadar uzatır.</span><span class="sxs-lookup"><span data-stu-id="4fe60-137">A type in a collectible assembly can have static references to a type in another collectible assembly, but this extends the lifetime of the referenced assembly to the lifetime of the referencing assembly.</span></span>

- <span data-ttu-id="4fe60-138">**COM interop** Tahsil edilebilir bir derleme içinde com arabirimleri tanımlanamaz ve tahsil edilebilir bir derlemedeki türlerin hiçbir örneği COM nesnelerine dönüştürülemez.</span><span class="sxs-lookup"><span data-stu-id="4fe60-138">**COM interop** No COM interfaces can be defined within a collectible assembly, and no instances of types within a collectible assembly can be converted into COM objects.</span></span> <span data-ttu-id="4fe60-139">Tahsil edilebilir bir derlemedeki bir tür, COM çağrılabilir sarıcı (CCW) veya çalışma zamanı çağrılabilir sarıcı (RCW) olarak hizmet veremez.</span><span class="sxs-lookup"><span data-stu-id="4fe60-139">A type in a collectible assembly cannot serve as a COM callable wrapper (CCW) or runtime callable wrapper (RCW).</span></span> <span data-ttu-id="4fe60-140">Ancak, tahsil edilebilir derlemelerde türleri COM arabirimleri uygulayan nesneleri kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="4fe60-140">However, types in collectible assemblies can use objects that implement COM interfaces.</span></span>

- <span data-ttu-id="4fe60-141">**Platform çağırma** Öznitelik sahibi <xref:System.Runtime.InteropServices.DllImportAttribute> yöntemler, tahsil edilebilir bir derlemede beyan edildiğinde derlenecektir.</span><span class="sxs-lookup"><span data-stu-id="4fe60-141">**Platform invoke** Methods that have the <xref:System.Runtime.InteropServices.DllImportAttribute> attribute will not compile when they are declared in a collectible assembly.</span></span> <span data-ttu-id="4fe60-142">Yönerge, <xref:System.Reflection.Emit.OpCodes.Calli?displayProperty=nameWithType> tahsil edilebilir bir derlemede bir türün uygulanmasında kullanılamaz ve bu tür türler yönetilmeyen koda dönüştürülemez.</span><span class="sxs-lookup"><span data-stu-id="4fe60-142">The <xref:System.Reflection.Emit.OpCodes.Calli?displayProperty=nameWithType> instruction cannot be used in the implementation of a type in a collectible assembly, and such types cannot be marshaled to unmanaged code.</span></span> <span data-ttu-id="4fe60-143">Ancak, tahsil edilemeyen bir derlemede bildirilen bir giriş noktası kullanarak yerel koda çağrı yapabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="4fe60-143">However, you can call into native code by using an entry point that is declared in a non-collectible assembly.</span></span>

- <span data-ttu-id="4fe60-144">**Mareşallik** Tahsil meclislerinde tanımlanan nesneler (özellikle, temsilciler) marshaled olamaz.</span><span class="sxs-lookup"><span data-stu-id="4fe60-144">**Marshaling** Objects (in particular, delegates) that are defined in collectible assemblies cannot be marshaled.</span></span> <span data-ttu-id="4fe60-145">Bu, tüm geçici yayılan türleri için bir kısıtlamadır.</span><span class="sxs-lookup"><span data-stu-id="4fe60-145">This is a restriction on all transient emitted types.</span></span>

- <span data-ttu-id="4fe60-146">**Montaj yüklemesi** Yansıma yayı, tahsil edilebilir montajların yüklenmesi için desteklenen tek mekanizmadır.</span><span class="sxs-lookup"><span data-stu-id="4fe60-146">**Assembly loading** Reflection emit is the only mechanism that is supported for loading collectible assemblies.</span></span> <span data-ttu-id="4fe60-147">Başka bir montaj yükleme biçimi kullanılarak yüklenen derlemeler boşaltılamaz.</span><span class="sxs-lookup"><span data-stu-id="4fe60-147">Assemblies that are loaded by using any other form of assembly loading cannot be unloaded.</span></span>

- <span data-ttu-id="4fe60-148">**İçeribağlı nesneler** Bağlam statik değişkenler desteklenmez.</span><span class="sxs-lookup"><span data-stu-id="4fe60-148">**Context-bound objects** Context-static variables are not supported.</span></span> <span data-ttu-id="4fe60-149">Tahsil edilebilir bir derlemedeki <xref:System.ContextBoundObject>türler uzatılamaz.</span><span class="sxs-lookup"><span data-stu-id="4fe60-149">Types in a collectible assembly cannot extend <xref:System.ContextBoundObject>.</span></span> <span data-ttu-id="4fe60-150">Ancak, tahsil edilebilir derlemelerde kod başka bir yerde tanımlanan içeriğe bağlı nesneleri kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="4fe60-150">However, code in collectible assemblies can use context-bound objects that are defined elsewhere.</span></span>

- <span data-ttu-id="4fe60-151">**İş parçacığı statik verileri** İş parçacığı statik değişkenler desteklenmez.</span><span class="sxs-lookup"><span data-stu-id="4fe60-151">**Thread-static data** Thread-static variables are not supported.</span></span>
