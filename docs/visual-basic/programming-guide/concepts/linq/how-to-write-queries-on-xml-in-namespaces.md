Bu durumda, XML sabit değerlerinde bulunan öğeler varsayılan olarak ad alanında yer alabilir.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-105">You can define a global default namespace, in which case elements in the XML literals will be in the namespace by default.</span></span> <span data-ttu-id="a2e5b-106">Alternatif olarak, ön ek içeren bir genel ad alanı tanımlayabilir ve sonra öneki XML sabit değerlerinde ve XML özelliklerinde gereken şekilde kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-106">Alternatively, you can define a global namespace with a prefix, and then use the prefix as required in the XML literals, and in XML properties.</span></span> <span data-ttu-id="a2e5b-107">Diğer XML formlarında olduğu gibi, özniteliklerin her zaman varsayılan olarak ad alanı yoktur.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-107">As with other forms of XML, attributes are always in no namespace by default.</span></span>

## <a name="example"></a><span data-ttu-id="a2e5b-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="a2e5b-110">Example</span></span>  
 <span data-ttu-id="a2e5b-111">Aşağıdaki örnek, varsayılan bir ad alanında bulunan bir XML ağacı oluşturur.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-111">The following example creates an XML tree that is in a default namespace.</span></span> <span data-ttu-id="a2e5b-112">Daha sonra bir öğe koleksiyonu alır.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-112">It then retrieves a collection of elements.</span></span>  
  
```vb  
Imports <xmlns="http://www.adventure-works.com">  
  
Module Module1  
    Sub Main()  
        Dim root As XElement = _  
            <Root>  
                <Child>1</Child>  
                <Child>2</Child>  
                <Child>3</Child>  
                <AnotherChild>4</AnotherChild>  
                <AnotherChild>5</AnotherChild>  
                <AnotherChild>6</AnotherChild>  
            </Root>  
        Dim c1 As IEnumerable(Of XElement) = _  
            From el In root.<Child> _  
            Select el  
        For Each el As XElement In c1  
            Console.WriteLine(el.Value)  
        Next  
    End Sub  
End Module  
```  
  
 <span data-ttu-id="a2e5b-113">Bu örnek aşağıdaki çıktıyı üretir:</span><span class="sxs-lookup"><span data-stu-id="a2e5b-113">This example produces the following output:</span></span>  
  
```console

2  
3  
```  
  
## <a name="example"></a><span data-ttu-id="a2e5b-114">Örnek</span><span class="sxs-lookup"><span data-stu-id="a2e5b-114">Example</span></span>  
 <span data-ttu-id="a2e5b-115">Ancak Visual Basic ' de, ön ek içeren bir ad alanı kullanan bir XML ağacına sorgu yazmak, varsayılan bir ad alanında bir XML ağacını sorgulamadan oldukça farklıdır.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-115">In Visual Basic, however, writing queries on an XML tree that uses a namespace with a prefix is quite different from querying an XML tree in a default namespace.</span></span> <span data-ttu-id="a2e5b-116">Genellikle `Imports` ifadesini kullanarak ad alanını önekiyle içeri aktarabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-116">Typically you use the `Imports` statement to import the namespace with a prefix.</span></span> <span data-ttu-id="a2e5b-117">Ardından, XML ağacını oluştururken öğesi ve öznitelik adlarında öneki kullanın.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-117">You then use the prefix in the element and attribute names when you construct the XML tree.</span></span> <span data-ttu-id="a2e5b-118">Xml özelliklerini kullanarak bir XML ağacını sorgularken de ön eki kullanırsınız.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-118">You also use the prefix when querying an XML tree using XML properties.</span></span>  
  
 <span data-ttu-id="a2e5b-119">Aşağıdaki örnek, öneki olan bir ad alanında olan bir XML ağacı oluşturur.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-119">The following example creates an XML tree that is in a namespace with a prefix.</span></span> <span data-ttu-id="a2e5b-120">Daha sonra bir öğe koleksiyonu alır.</span><span class="sxs-lookup"><span data-stu-id="a2e5b-120">It then retrieves a collection of elements.</span></span>

```vb  
Module Module1  
    Sub Main()  
        Dim root As XElement = _  
            <aw:Root>  
                <aw:Child>1</aw:Child>  
                <aw:Child>2</aw:Child>  
                <aw:Child>3</aw:Child>  
                <aw:AnotherChild>4</aw:AnotherChild>  
                <aw:AnotherChild>5</aw:AnotherChild>  
                <aw:AnotherChild>6</aw:AnotherChild>  
            </aw:Root>  
        Dim c1 As IEnumerable(Of XElement) = _  
            From el In root.<aw:Child> _  
            Select el  
        For Each el As XElement In c1  
            Console.WriteLine(CInt(el))  
        Next  
    End Sub  
End Module  
```
