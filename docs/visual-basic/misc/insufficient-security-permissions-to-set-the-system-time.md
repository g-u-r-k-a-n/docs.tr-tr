# <a name="insufficient-security-permissions-to-set-the-system-time"></a>Sistem saatini ayarlamak için yetersiz güvenlik izinleri

Güvenlik izinleri, sistem saatini ayarlamak için yetersizdir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Güvenlik izinlerini değiştirin.
