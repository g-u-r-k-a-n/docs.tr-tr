# <a name="compiler-error-cs0031"></a><span data-ttu-id="41478-103">Derleyici hatası CS0031</span><span class="sxs-lookup"><span data-stu-id="41478-103">Compiler Error CS0031</span></span>

<span data-ttu-id="41478-104">' Value ' sabit değeri ' Type ' olarak dönüştürülemez.</span><span class="sxs-lookup"><span data-stu-id="41478-104">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="41478-105">Türü, değeri depolayabilen bir değişkene bir değer atamaya çalışıldı.</span><span class="sxs-lookup"><span data-stu-id="41478-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="41478-106">Daha fazla bilgi için bkz. [türler](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="41478-106">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="41478-107">Aşağıdaki örnek hem işaretli hem de işaretlenmemiş bağlamlarda CS0031 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="41478-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
