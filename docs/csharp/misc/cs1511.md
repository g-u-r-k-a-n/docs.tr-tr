# <a name="compiler-error-cs1511"></a><span data-ttu-id="7fb16-103">Derleyici hatası CS1511</span><span class="sxs-lookup"><span data-stu-id="7fb16-103">Compiler Error CS1511</span></span>
<span data-ttu-id="7fb16-104">' Base ' anahtar sözcüğü statik bir yöntemde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="7fb16-104">Keyword 'base' is not available in a static method</span></span>  
  
 <span data-ttu-id="7fb16-105">[Base](../language-reference/keywords/base.md) anahtar sözcüğü [statik](../language-reference/keywords/static.md) bir yöntemde kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="7fb16-105">The [base](../language-reference/keywords/base.md) keyword was used in a [static](../language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="7fb16-106">`base` yalnızca bir örnek oluşturucusunda, örnek yönteminde veya örnek erişimcisinde çağrılabilir.</span><span class="sxs-lookup"><span data-stu-id="7fb16-106">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7fb16-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="7fb16-107">Example</span></span>  
 <span data-ttu-id="7fb16-108">Aşağıdaki örnek CS1511 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7fb16-108">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
