# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>' ' Metodu ' \<methodname> ' arabiriminde zaten tanımlanmış \<interfacename>

İkinci kez bir arabirim yöntemi bildirdiniz, çoğaltma.  
  
 **Hata kimliği:** BC30377  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yinelenen bildirimi kaldırın.
