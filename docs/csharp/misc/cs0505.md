# <a name="compiler-error-cs0505"></a><span data-ttu-id="3fad1-103">Derleyici hatası CS0505</span><span class="sxs-lookup"><span data-stu-id="3fad1-103">Compiler Error CS0505</span></span>
<span data-ttu-id="3fad1-104">' member1 ': ' member2 ' bir işlev olmadığından geçersiz kılınamaz</span><span class="sxs-lookup"><span data-stu-id="3fad1-104">'member1': cannot override because 'member2' is not a function</span></span>  
  
 <span data-ttu-id="3fad1-105">Bir sınıf bildirimi, bir temel sınıfta bir yöntemi olmayan geçersiz kılmayı denedi.</span><span class="sxs-lookup"><span data-stu-id="3fad1-105">A class declaration attempted to override a non-method in a base class.</span></span> <span data-ttu-id="3fad1-106">Geçersiz kılmalar üye türüyle eşleşmelidir.</span><span class="sxs-lookup"><span data-stu-id="3fad1-106">Overrides must match the member type.</span></span> <span data-ttu-id="3fad1-107">Temel sınıftaki bir yöntem ile aynı ada sahip bir yöntem isteniyorsa, temel sınıftaki yöntem bildiriminde [Yeni](../language-reference/keywords/new-modifier.md) (ve [geçersiz kılma](../language-reference/keywords/override.md)) kullanın.</span><span class="sxs-lookup"><span data-stu-id="3fad1-107">If a method with the same name as a method in a base class is desired, use [new](../language-reference/keywords/new-modifier.md) (and not [override](../language-reference/keywords/override.md)) on the method declaration in the base class.</span></span>  
  
 <span data-ttu-id="3fad1-108">Aşağıdaki örnek CS0505 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3fad1-108">The following sample generates CS0505:</span></span>  
  
```csharp  
// CS0505.cs  
// compile with: /target:library  
public class clx  
{  
   public int i;  
}  
  
public class cly : clx  
{  
   public override int i() { return 0; }   // CS0505  
}  
```
