# <a name="name-is-already-declared-as-a-parameter-of-this-method"></a>' \<name> ' Bu yöntemin bir parametresi olarak zaten tanımlanmış
Belirtilen parametre adı bu yöntem için zaten tanımlanmış.  
  
 **Hata kimliği:** BC30734  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Parametreyi benzersiz olacak şekilde yeniden adlandırın.
