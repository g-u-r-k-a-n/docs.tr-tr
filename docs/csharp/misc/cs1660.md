# <a name="compiler-error-cs1660"></a><span data-ttu-id="6927a-103">Derleyici hatası CS1660</span><span class="sxs-lookup"><span data-stu-id="6927a-103">Compiler Error CS1660</span></span>

<span data-ttu-id="6927a-104">Anonim yöntem bloğu bir temsilci türü olmadığından ' Type ' türüne dönüştürülemiyor</span><span class="sxs-lookup"><span data-stu-id="6927a-104">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="6927a-105">Bu hata, bir anonim yöntem bloğunu atama türü olmayan bir türe atamayı veya başka bir şekilde dönüştürmeyi denerseniz oluşur.</span><span class="sxs-lookup"><span data-stu-id="6927a-105">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="6927a-106">Aşağıdaki örnek CS1660 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6927a-106">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
