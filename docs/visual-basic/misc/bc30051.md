# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>ParamArray parametresi tek boyutlu bir dizi olmalıdır

`ParamArray`Bağımsız değişken, tek boyutlu dizi türü olarak bildirilmelidir. Bildirim, bağımsız değişken için birden çok boyut belirtiyor `ParamArray` .  
  
 **Hata kimliği:** BC30051  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken adını izleyen parantez içindeki tüm virgülleri kaldırın `ParamArray` .
