# <a name="properties-cannot-be-declared-modifiername"></a>Özellikler ' ' olarak bildirilemez \<modifiername>
Bildirimde kullanılan sözdiziminde bir hata var `Property` . Bu hata genellikle, gibi belirtilen bir anahtar sözcüğünün `WithEvents` bildirimde kullanılamayacağını gösterir `Property` .  
  
 **Hata kimliği:** BC30639  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimin sözdizimini denetleyin `Property` .
