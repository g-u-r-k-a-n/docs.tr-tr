 <span data-ttu-id="ddab2-105">Bu hata, bir sınıfın yapıcısını doğrudan veya dolaylı olarak çağırdığınızda, ancak derleyici aynı sayıda parametreye sahip herhangi bir Oluşturucu bulamazsa oluşur.</span><span class="sxs-lookup"><span data-stu-id="ddab2-105">This error occurs when you either directly or indirectly invoke the constructor of a class but the compiler cannot find any constructors with the same number of parameters.</span></span> <span data-ttu-id="ddab2-106">Aşağıdaki örnekte, `test` sınıfının herhangi bir bağımsız değişken alan Oluşturucusu yoktur.</span><span class="sxs-lookup"><span data-stu-id="ddab2-106">In the following example, the `test` class has no constructors that take any arguments.</span></span> <span data-ttu-id="ddab2-107">Bu nedenle, sıfır bağımsız değişken alan parametresiz bir oluşturucuya sahiptir.</span><span class="sxs-lookup"><span data-stu-id="ddab2-107">It therefore has only a parameterless constructor that takes zero arguments.</span></span> <span data-ttu-id="ddab2-108">Hatanın oluşturulduğu ikinci satırda, türetilmiş sınıf kendisine ait hiçbir Oluşturucu bildirmediği için derleyicinin parametresiz bir Oluşturucu sağladığı anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="ddab2-108">Because in the second line in which the error is generated, the derived class declares no constructors of its own, the compiler provides a parameterless constructor.</span></span> <span data-ttu-id="ddab2-109">Bu Oluşturucu, temel sınıfta parametresiz bir oluşturucu çağırır.</span><span class="sxs-lookup"><span data-stu-id="ddab2-109">That constructor invokes a parameterless constructor in the base class.</span></span> <span data-ttu-id="ddab2-110">Temel sınıfta böyle bir Oluşturucu olmadığından, CS1729 oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="ddab2-110">Because the base class has no such constructor, CS1729 is generated.</span></span>

## <a name="example"></a><span data-ttu-id="ddab2-115">Örnek</span><span class="sxs-lookup"><span data-stu-id="ddab2-115">Example</span></span>  
 <span data-ttu-id="ddab2-116">Aşağıdaki örnek CS1729 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ddab2-116">The following example generates CS1729:</span></span>  
  
```csharp  
// cs1729.cs  
class Test  
{  
    static int Main()  
    {  
        // Class Test has only a parameterless constructor, which takes no arguments.  
        Test test1 = new Test(2); // CS1729  
        // The following line resolves the error.  
        Test test2 = new Test();  
  
        // Class Parent has only one constructor, which takes two int parameters.  
        Parent exampleParent1 = new Parent(10); // CS1729  
        // The following line resolves the error.  
        Parent exampleParent2 = new Parent(10, 1);  
  
        return 1;  
    }  
}  
  
public class Parent  
{  
    // The only constructor for this class has two parameters.  
    public Parent(int i, int j) { }  
}  
  
// The following declaration causes a compiler error because class Parent  
// does not have a constructor that takes no arguments. The declaration of  
// class Child2 shows how to resolve this error.  
public class Child : Parent { } // CS1729  
  
public class Child2 : Parent  
{  
    // The constructor for Child2 has only one parameter. To access the
    // constructor in Parent, and prevent this compiler error, you must provide
    // a value for the second parameter of Parent. The following example provides 0.  
    public Child2(int k)  
        : base(k, 0)  
    {  
        // Add the body of the constructor here.  
    }  
}  
```
