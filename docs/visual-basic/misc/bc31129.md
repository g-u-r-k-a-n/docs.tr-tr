# <a name="raiseevent-is-already-declared"></a>' RaiseEvent ' zaten tanımlanmış

`RaiseEvent`Özel bir olay bildiriminde birden fazla bildirim görüntülendi. `RaiseEvent`Bildirim, olayı yükseltmek için kullanılan bir yordamı bildirir.  
  
 **Hata kimliği:** BC31129  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ifadesini kaldırın `RaiseEvent` .
