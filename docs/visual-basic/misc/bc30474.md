# <a name="method1-cannot-override-method2-because-it-is-a-declare-statement"></a>' \<method1> \<method2> Declare ' bir ifade olduğundan ' ', ' ' öğesini geçersiz kılamaz
Bir deyimle belirtilen temel sınıf adındaki bir temsilciyi geçersiz kılmaya çalıştınız `Declare` .  
  
 **Hata kimliği:** BC30474  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Geçersiz kılınan üyeyi bir ifade olmaması için değiştirin `Declare` .  
  
2. Bu yöntemi geçersiz kılmayı denemeyin.
