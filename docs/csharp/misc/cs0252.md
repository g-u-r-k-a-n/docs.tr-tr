# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="28e85-103">Derleyici Uyarısı (düzey 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="28e85-103">Compiler Warning (level 2) CS0252</span></span>

<span data-ttu-id="28e85-104">Olası istenmeden yapılan başvuru karşılaştırması; değer karşılaştırması almak için sol tarafı ' Type ' türüne dönüştürün</span><span class="sxs-lookup"><span data-stu-id="28e85-104">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="28e85-105">Derleyici bir başvuru karşılaştırması yapıyor.</span><span class="sxs-lookup"><span data-stu-id="28e85-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="28e85-106">Dizelerin değerini karşılaştırmak istiyorsanız, ifadesinin sol tarafını olarak atayın `type` .</span><span class="sxs-lookup"><span data-stu-id="28e85-106">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="28e85-107">Aşağıdaki örnek CS0252 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="28e85-107">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
