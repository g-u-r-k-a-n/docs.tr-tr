# <a name="alias-namespacename-is-already-declared"></a>' ' Diğer adı \<namespacename> zaten tanımlanmış

`Imports`Zaten belirtilen bir ad alanını içeri aktarmak için kullanmayı denediniz.  
  
 **Hata kimliği:** BC30572  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İçeri aktarmaya çalıştığınız ad alanını denetleyin.
