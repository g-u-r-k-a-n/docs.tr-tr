<span data-ttu-id="63e0b-105"><xref:System.Diagnostics.ConditionalAttribute> , dışında bir dönüş türüne sahip bir yönteme uygulanamaz `void` .</span><span class="sxs-lookup"><span data-stu-id="63e0b-105"><xref:System.Diagnostics.ConditionalAttribute> cannot be applied to a method that has a return type other than `void`.</span></span> <span data-ttu-id="63e0b-106">Bunun nedeni, bir yöntem için başka bir dönüş türünün programınızın başka bir parçası için gerekli olabilir.</span><span class="sxs-lookup"><span data-stu-id="63e0b-106">The reason for this is that any other return type for a method may be needed by another part of your program.</span></span>

    [System.Diagnostics.ConditionalAttribute("a")]   // CS0578

    }
}
