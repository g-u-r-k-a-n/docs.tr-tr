# <a name="printpreviewcontrol-control-windows-forms"></a>PrintPreviewControl Denetimi (Windows Forms)
Windows Forms `PrintPreviewControl`, yazdırıldığında görüneceği şekilde bir belgeyi göstermek için kullanılır. Bu denetimde hiçbir düğme veya diğer kullanıcı arabirimi öğesi yoktur, bu nedenle genellikle `PrintPreviewControl` yalnızca kendi baskı önizleme Kullanıcı arabiriminizi yazmak istiyorsanız kullanırsınız. Standart Kullanıcı arabirimini istiyorsanız, bir <xref:System.Windows.Forms.PrintPreviewDialog> denetimi kullanın.
