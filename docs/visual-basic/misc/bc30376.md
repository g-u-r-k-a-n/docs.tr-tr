# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>\<classname>Geçersiz kılınamayan bir ' MustOverride ' üyesi içerdiğinden ' ' sınıfında ' New ' kullanılamaz
`New`Geçersiz kılınmamış bir üye içeren bir sınıfta kullanmayı denediniz `MustOverride` .  
  
 **Hata kimliği:** BC30376  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfadesini kaldırın `New` .
