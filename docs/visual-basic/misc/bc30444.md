# <a name="set-is-already-declared"></a>' Set ' zaten tanımlanmış
Bir `Set` ifade, kod bloğunda birden çok kez görünür. İfade, bir `Set` `Set` özelliğe değer atamak için kullanılan bir özellik yordamı bildirir.  
  
 **Hata kimliği:** BC30444  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ifadesini kaldırın `Set` .
