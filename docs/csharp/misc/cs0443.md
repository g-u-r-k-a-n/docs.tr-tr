# <a name="compiler-error-cs0443"></a>Derleyici Hatası CS0443
Sözdizimi hatası, beklenen değer  
  
 Bu hata, dizi dizi dizini için bir değer belirtmeden bir dizi başvuru oluşur.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki kod CS0443 oluşturur.  
  
```csharp  
// CS0443.cs
using System;
class MyClass
{  
    public static void Main()
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}
    }  
}  
```
