# <a name="breaking-changes-in-windows-forms-net-framework-to-net-core"></a>Windows Forms (.NET Framework .NET Core 'a) son değişiklikler

Son değişiklikler, tanıtıldıkları .NET Core sürümüne göre gruplandırılır.
