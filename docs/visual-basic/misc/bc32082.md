<span data-ttu-id="fb426-103">Genel tür, bir `MustInherit` sınıf ile tür bağımsız değişkeni olarak çağrılır, ancak karşılık gelen tür parametresi `New` kısıtlamayla birlikte bildirilmiştir.</span><span class="sxs-lookup"><span data-stu-id="fb426-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="fb426-104">`New`Kısıtlama, karşılık gelen tür bağımsız değişkeninde geçirilen türün nesne oluşturulmasını desteklemesini gerektirir.</span><span class="sxs-lookup"><span data-stu-id="fb426-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="fb426-105">Ancak, bir *soyut* sınıf olan, diğer bir deyişle, bir sınıf olarak belirtilen bir `MustInherit` nesne oluşturamadığı için herhangi bir Oluşturucu sunmaz.</span><span class="sxs-lookup"><span data-stu-id="fb426-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="fb426-106">**Hata kimliği:** BC32082</span><span class="sxs-lookup"><span data-stu-id="fb426-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fb426-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="fb426-107">To correct this error</span></span>

3. <span data-ttu-id="fb426-110">Karşılık gelen tür parametresinin kendisine geçirilen türden herhangi bir nesne oluşturması gerekmiyorsa, `New` kısıtlamayı onun bildiriminden kaldırın.</span><span class="sxs-lookup"><span data-stu-id="fb426-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>
