# <a name="unable-to-open-file-filename-error"></a>' ' Dosyası açılamıyor \<filename> : \<error>

Bu projede bir dosya açılırken hata oluştu.  
  
 **Hata kimliği:** BC31027  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosyanın var olduğunu ve bozulmuş ya da başka bir uygulama tarafından kilitlenip kilitlenmediğini doğrulayın.
