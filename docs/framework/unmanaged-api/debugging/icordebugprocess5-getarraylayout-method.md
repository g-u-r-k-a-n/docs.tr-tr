# <a name="icordebugprocess5getarraylayout-method"></a>ICorDebugProcess5::GetArrayLayout Yöntemi
Dizi türlerinin yerleşimi hakkında bilgi sağlar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT GetArrayLayout(    [in] COR_TYPEID id,     [out] COR_ARRAY_LAYOUT *pLayout);  
```  
  
## <a name="parameters"></a>Parametreler  
 `id`  
 'ndaki Düzeni istenen diziyi belirten bir [COR_TYPEID](../../../../docs/framework/unmanaged-api/debugging/cor-typeid-structure.md) belirteci.  
  
 `pLayout`  
 dışı [COR_ARRAY_LAYOUT](../../../../docs/framework/unmanaged-api/debugging/cor-array-layout-structure.md) yapısına yönelik bir işaretçi, bellekteki dizinin düzeni hakkında bilgi içerir.
