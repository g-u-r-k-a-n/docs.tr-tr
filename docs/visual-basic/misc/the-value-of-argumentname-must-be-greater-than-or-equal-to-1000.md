# <a name="the-value-of-argumentname-must-be-greater-than-or-equal-to-1000"></a>Değeri \<argumentname> 1000 değerinden büyük veya buna eşit olmalıdır

<xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A>Özelliğin değeri 1000 değerinden büyük veya buna eşit olmalıdır.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A>Özelliği, yeni bir günlük dosyası oluşturulmadan önce bir günlük dosyasında izin verilen en fazla bayt sayısını belirtir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A>Özelliği 1000 değerinden büyük veya buna eşit bir sayı olarak ayarlayın.
