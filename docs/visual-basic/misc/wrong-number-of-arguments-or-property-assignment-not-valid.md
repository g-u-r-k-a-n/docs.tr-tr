# <a name="wrong-number-of-arguments-or-property-assignment-not-valid"></a>Yanlış sayıda bağımsız değişken veya özellik ataması geçerli değil
Geçerli olmayan bir atama denendi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Sağladığınız bağımsız değişken sayısının hedefin gerektirdiği sayıyla eşleşip eşleşmediğini denetleyin.  
  
2. Özellik atamasını denetleyin.
