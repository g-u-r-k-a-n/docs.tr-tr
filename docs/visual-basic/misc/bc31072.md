# <a name="warning-treated-as-error-error"></a>Uyarı hata olarak değerlendirildi: \<error>

**Uyarıları hata olarak değerlendir** seçeneği etkinken derlenen bir derleme çalıştırılırken bir uyarıyla karşılaşıldı.  
  
 **Hata kimliği:** BC31072  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Uyarıyı düzeltin.  
  
2. **Uyarıları hata olarak değerlendir** seçeneği kapalıyken derlemeyi derleyin.
