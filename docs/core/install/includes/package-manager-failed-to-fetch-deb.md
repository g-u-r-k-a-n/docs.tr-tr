.NET Core paketini yüklerken `Failed to fetch ... File has unexpected size ... Mirror sync in progress?`benzer bir hata görebilirsiniz. Genellikle bu hata, .NET Core için paket akışı 'nın daha yeni paket sürümleriyle yükseltildiği ve daha sonra yeniden denemeniz gereken anlamına gelir. Yükseltme sırasında, paket akışı 30 dakikadan uzun bir süre için kullanılamayabilir. 30 dakikadan uzun bir süre içinde sürekli olarak bu hatayı alırsanız lütfen <https://github.com/dotnet/core/issues>bir sorun bildirin.
