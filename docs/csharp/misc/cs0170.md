 <span data-ttu-id="bbc99-105">Bir yapıdaki alan, ilk başlatılmadan kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="bbc99-105">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="bbc99-106">Bu sorunu çözmek için, önce hangi alanın başlatılmamış olduğunu belirleyip, daha sonra bu alana erişmeyi denemeden önce onu başlatın.</span><span class="sxs-lookup"><span data-stu-id="bbc99-106">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="bbc99-107">Yapıları başlatma hakkında daha fazla bilgi için bkz. [yapı türleri](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="bbc99-107">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="bbc99-108">Aşağıdaki örnek CS0170 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bbc99-108">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because
                                         //e.i was never assigned  
   }  
}  
```
