# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed

İletişim nesnesi durdurulamadı.  
  
## <a name="description"></a>Açıklama  

 Bir iletişim nesnesi, Abort yönteminin yürütülmesi sırasında bir özel durum oluşturulduğu için durdurulamadı. Başka bir yakın izleme özel durumla ilgili ayrıntıları içerebilir.
