# <a name="compiler-error-cs0743"></a>Derleyici hatası CS0743
' On ' bağlamsal anahtar sözcüğü bekleniyordu  
  
 `join` yan tümcesinin deseninin `join`...`in`...`on`...`equals`, bu örnekte gösterildiği gibi:  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `join` yan tümcesine `on` anahtar sözcüğünü ekleyin.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki kod CS0743 oluşturur:  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
