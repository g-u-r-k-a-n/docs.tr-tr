# <a name="compiler-error-cs1552"></a>Derleyici hatası CS1552
Dizi türü Belirleyicisi, [], parametre adından önce gelmelidir  
  
 Dizi türü belirticisinin konumu, dizi bildiriminde değişken adından sonra.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1552 oluşturur:  
  
```csharp  
// CS1552.cs  
public class C  
{  
    public static void Main(string args[])   // CS1552  
    // try the following line instead  
    // public static void Main(string [] args)  
    {  
    }  
}  
```
