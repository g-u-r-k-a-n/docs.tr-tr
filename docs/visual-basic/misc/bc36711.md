# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>' ' Parametresiyle eşleşen bağımsız değişken \<parametername> ' ' ile daralan \<typename>

Aşırı yüklenmiş bir yöntemde çağrı yapıldı, ancak derleyici, daraltma dönüştürmesi olmadan çağrılabilecek bir yöntem bulamıyor. *Daraltma dönüştürmesi* bir değeri, olası değerlerden bazılarını tutabilecek bir veri türüne dönüştürür.  
  
 **Hata kimliği:** BC36711
