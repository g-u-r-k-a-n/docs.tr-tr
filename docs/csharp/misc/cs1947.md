## <a name="example"></a><span data-ttu-id="43a11-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="43a11-109">Example</span></span>  
 <span data-ttu-id="43a11-110">Aşağıdaki kod CS1947 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="43a11-110">The following code generates CS1947:</span></span>

class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };
