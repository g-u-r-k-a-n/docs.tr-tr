# <a name="unable-to-open-module-file-filename-error"></a>Modül dosyası ' \<filename> ' açılamıyor: \<error>

Bu proje için modül içeren bir dosya açılamadı.  
  
 **Hata kimliği:** BC31007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosyanın belirtilen konumda olduğunu ve başka bir uygulama tarafından kilitlenmediğini doğrulayın.
