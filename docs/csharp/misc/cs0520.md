# <a name="compiler-error-cs0520"></a>Derleyici hatası CS0520
Önceden tanımlanmış ' name ' türü yanlış olarak bildiriliyor  
  
 Derleyici için gereken bir veya daha fazla dosya bulunamadı. .NET çalışma zamanını yeniden yükleyin.
