# <a name="expression-does-not-produce-a-value"></a>İfade bir değer üretmiyor

Değer üreten bir bağlamda bir değer üretmeyen bir ifade kullanmayı denediniz; Örneğin, bir bir bağlamda, bir değeri `Sub` `Function` olması beklendiğinde.  
  
 **Hata kimliği:** BC30491  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfadeyi bir değer üreten bir değerle değiştirin.
