# <a name="modifier-is-not-valid-on-a-declare"></a>' \<modifier> ', Declare üzerinde geçerli değil

Bir `Declare` ifade gibi geçersiz bir anahtar sözcük içeriyor `ReadOnly` .  
  
 **Hata kimliği:** BC30244  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Deyimden geçersiz anahtar sözcüğü Kaldır `Declare` .
