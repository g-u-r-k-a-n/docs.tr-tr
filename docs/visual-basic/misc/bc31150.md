# <a name="end-tag-tagname-expected"></a>Bitiş etiketi \</tagName> bekleniyor
XML öğesi değişmez değerindeki bitiş etiketi, karşılık gelen başlangıç etiketiyle eşleşmiyor.  
  
 **Hata kimliği:** BC31150  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML öğesi değişmez değeri için başlangıç XML etiketi ile eşleşecek şekilde son XML etiketini değiştirin.
