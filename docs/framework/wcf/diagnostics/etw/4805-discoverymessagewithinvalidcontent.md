|||  
|-|-|  
|ID|4805|  
|Anahtar sözcükler|Bulma|  
|Düzey|Uyarı|

## <a name="description"></a>Açıklama

## <a name="message"></a>İleti  

 MessageID = ' %2 ' olan bir %1 iletisi, geçersiz içeriğe sahip olduğundan bırakıldı.
