<span data-ttu-id="27b30-104">Lambda, tür bağımsız değişkeni ' Type ' bir temsilci türü olmayan bir ifade ağacına dönüştürülemez.</span><span class="sxs-lookup"><span data-stu-id="27b30-104">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="27b30-105">Lambda ifadesi bir ifade ağacına dönüştürülürse, ifade ağacının bağımsız değişkeni için bir temsilci türü olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="27b30-105">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="27b30-106">Ayrıca, lambda ifadesinin temsilci türüne dönüştürülebilir olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="27b30-106">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>

## <a name="example"></a><span data-ttu-id="27b30-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="27b30-109">Example</span></span>  
 <span data-ttu-id="27b30-110">Aşağıdaki örnek CS0835 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="27b30-110">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
