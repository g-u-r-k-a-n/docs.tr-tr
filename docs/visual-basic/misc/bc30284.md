# <a name="type1-typename-cannot-be-declared-overrides-because-it-does-not-override-a-type1-in-a-base-type2"></a>\<type1> ' \<typename> ', bir tabanında bir öğesini geçersiz kılmadığından ' override ' \<type1> olarak bildirilemez \<type2>

Bir `Sub` , `Function` veya `Property` deyimleri, bir `Overrides` temel sınıfta aynı ada sahip hiçbir tür olmadığında belirtir.  
  
 **Hata kimliği:** BC30284  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Tür adının doğru yazıldığından emin olun.  
  
2. Gereksiz `Overrides` anahtar sözcüğünü kaldırın.
