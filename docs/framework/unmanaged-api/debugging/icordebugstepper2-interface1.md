# <a name="icordebugstepper2-interface"></a>ICorDebugStepper2 Arabirimi
Yalnızca My Code (JMC) hata ayıklaması için destek sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[SetJMC Yöntemi](icordebugstepper2-setjmc-method.md)|Bu ICorDebugStepper adımlarının yalnızca bir uygulamanın geliştiricisi tarafından yazılan kodla ilgili olduğunu belirten bir değer ayarlar.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
> Bu arabirim, çapraz makine ya da çapraz işlem için uzaktan çağrılmakta değil.
