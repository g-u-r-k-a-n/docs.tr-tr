# <a name="compiler-error-cs0220"></a><span data-ttu-id="0e264-103">Derleyici hatası CS0220</span><span class="sxs-lookup"><span data-stu-id="0e264-103">Compiler Error CS0220</span></span>

<span data-ttu-id="0e264-104">İşlem, denetlenen modda derleme zamanında taşıyor</span><span class="sxs-lookup"><span data-stu-id="0e264-104">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="0e264-105">Bir işlem [denetlenen](../language-reference/keywords/checked.md)tarafından algılandı, bu varsayılan değer, bir veri kaybı ile sonuçlandı.</span><span class="sxs-lookup"><span data-stu-id="0e264-105">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="0e264-106">Atama girdilerini düzeltin ya da bu hatayı çözmek için [işaretlenmemiş](../language-reference/keywords/unchecked.md) kullanın.</span><span class="sxs-lookup"><span data-stu-id="0e264-106">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="0e264-107">Daha fazla bilgi için bkz. [Checked ve unchecked](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="0e264-107">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="0e264-108">Aşağıdaki örnek CS0220 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0e264-108">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
