# <a name="compiler-error-cs0415"></a><span data-ttu-id="f1964-103">Derleyici hatası CS0415</span><span class="sxs-lookup"><span data-stu-id="f1964-103">Compiler Error CS0415</span></span>

<span data-ttu-id="f1964-104">' IndexerName ' özniteliği yalnızca açık arabirim üyesi bildirimi olmayan bir dizin oluşturucuda geçerlidir</span><span class="sxs-lookup"><span data-stu-id="f1964-104">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="f1964-105">Bu hata, bir arabirim üzerinde açık bir uygulama olan bir dizin oluşturucuda IndexerName özniteliği kullanırsanız oluşur.</span><span class="sxs-lookup"><span data-stu-id="f1964-105">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="f1964-106">Bu hata, mümkünse, dizin oluşturucunun bildiriminden arabirim adı kaldırılarak kaçınılabilir.</span><span class="sxs-lookup"><span data-stu-id="f1964-106">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="f1964-107">Daha fazla bilgi için bkz. [IndexerNameAttribute nesnesinin oluşturucusu sınıfı](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="f1964-107">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="f1964-108">Aşağıdaki örnek CS0415 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f1964-108">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
