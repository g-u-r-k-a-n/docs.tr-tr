 <span data-ttu-id="4908e-106">Aşağıdaki örnek CS0642 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4908e-106">The following sample generates CS0642:</span></span>

```csharp  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?

         System.Console.WriteLine (i);  
      }  
   }  
}  
```
