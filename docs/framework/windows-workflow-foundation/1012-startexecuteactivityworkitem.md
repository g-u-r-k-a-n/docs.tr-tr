# <a name="1012---startexecuteactivityworkitem"></a>1012 - StartExecuteActivityWorkItem

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|1012|  
|Anahtar sözcükler|WFRuntime|  
|Level|Ayrıntılı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Description  

 Bir ExecuteActivityWorkItem 'ın yürütmeyi başlatdığını gösterir.  
  
## <a name="message"></a>İleti  

 DisplayName: ' %2 ', InstanceId: ' %3 ' olan ' %1 ' Activity 'si için bir ExecuteActivityWorkItem 'ın yürütülmesi başlatılıyor.
