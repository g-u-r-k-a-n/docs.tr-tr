# <a name="systemservicemodeltxsourcetxscoperequiredisattachedtransaction"></a>System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction

Belirtilen işlem, bir InstanceContext işleminden belirtilen işlem için alındı.  
  
## <a name="description"></a>Açıklama  

 Kullanıcı kodu için kullanılabilir hale getirilen işlem bu çağrıdan önce çağrılan bir AutoComplete = false yöntemi tarafından arkasında bırakılırsa izleniyor.
