# <a name="how-to-align-multiple-controls-on-windows-forms"></a>Nasıl yapılır: Windows Formlarında Birden Çok Denetimi Hizalama

Windows tabanlı uygulamanızın kullanıcı arabirimi (UI) yerleşimini standartlaştırmak için, denetim gruplarını tek bir komutla konumlandırabilirsiniz.

## <a name="to-align-multiple-controls-on-a-form"></a>Bir form üzerinde birden çok denetimi hizalamak için

1. Visual Studio 'da, **Windows Form Tasarımcısı**konumlandırmak istediğiniz denetimleri içeren formu açın.

2. Hizalamak istediğiniz denetimleri seçin. Seçtiğiniz ilk denetim, diğerlerinin hizalandığı birincil denetimdir.

3. **Biçim** menüsünde **Hizala**' yı seçin ve ardından kullanılabilen yedi seçenekten birini seçin.
