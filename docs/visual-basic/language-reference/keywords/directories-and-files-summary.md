 `My` özelliğinin kullanılması, bu üyelerin kullanılmasıyla dosya g/ç işlemlerinde daha fazla üretkenlik ve performans sağlar. Daha fazla bilgi için bkz. <xref:Microsoft.VisualBasic.FileIO.FileSystem>.
