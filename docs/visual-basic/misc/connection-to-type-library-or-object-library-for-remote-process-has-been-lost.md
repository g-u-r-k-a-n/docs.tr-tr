# <a name="connection-to-type-library-or-object-library-for-remote-process-has-been-lost"></a>Uzak işlem için tür kitaplığı veya nesne kitaplığı bağlantısı kesildi
Uzak işlemin nesne kitaplığıyla veya tür kitaplığıyla bağlantınızı kaybettiniz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Nesneyi yeniden başlatın `Application` .  
  
2. Bu yardım konusunu girdiğiniz hata iletişim kutusunda, **Başvurular** iletişim kutusunu göstermek için **Tamam** ' ı tıklatın. Kayıp başvuru, sol tarafında "eksık" kelimesiyle birlikte görüntülenir.  
  
3. Kayıp başvurusunu silin.  
  
4. **Başvurular** iletişim kutusunda nesnenin onay kutusunu seçin `Application` .
