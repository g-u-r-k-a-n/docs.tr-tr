# <a name="compiler-error-cs0522"></a><span data-ttu-id="cf08e-103">Derleyici hatası CS0522</span><span class="sxs-lookup"><span data-stu-id="cf08e-103">Compiler Error CS0522</span></span>

<span data-ttu-id="cf08e-104">' constructor ': yapılar temel sınıf oluşturucuları çağıramaz</span><span class="sxs-lookup"><span data-stu-id="cf08e-104">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="cf08e-105">[Struct](../language-reference/builtin-types/struct.md) bir temel sınıf oluşturucusu çağıramaz; Taban sınıf oluşturucusuna çağrıyı kaldırın.</span><span class="sxs-lookup"><span data-stu-id="cf08e-105">A [struct](../language-reference/builtin-types/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="cf08e-106">Aşağıdaki örnek CS0522 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="cf08e-106">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
