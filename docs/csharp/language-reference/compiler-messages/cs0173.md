# <a name="compiler-error-cs0173"></a>Derleyici Hatası CS0173

' Class1 ' ve ' Class2 ' arasında örtülü dönüşüm olmadığından koşullu ifadenin türü belirlenemiyor

Sınıflar arasındaki dönüştürmeler, farklı sınıfların nesnelerinin aynı kodla çalışmasını istediğinizde faydalıdır. Bununla birlikte, birlikte çalışan iki sınıf karşılıklı ve yedekli Dönüştürmelere veya örtük dönüştürmeler içeremez. `class1` ve `class2` türleri bağımsız olarak belirlenir ve koşullu ifadenin türü olarak daha genel tür seçilir. Türlerin nasıl belirlendiği hakkında daha fazla bilgi için bkz. [koşullu işleç örtük olarak atanamaz](https://stackoverflow.com/questions/2215745/conditional-operator-cannot-cast-implicitly/2215959#2215959).

CS0173 çözümlemek için, dönüştürmenin hangi yöne ait olduğuna ve dönüştürmenin hangi sınıftan olduğuna bakılmaksızın `class1` ve `class2`arasında tek bir örtük dönüştürme olduğunu doğrulayın. Daha fazla bilgi için bkz. [Kullanıcı tanımlı dönüştürme işleçleri](../operators/user-defined-conversion-operators.md) ve [yerleşik sayısal dönüşümler](../builtin-types/numeric-conversions.md).

## <a name="example"></a>Örnek

Aşağıdaki örnekler derleyici hatası oluşturur CS0173:

```csharp
public class C {}

public class A
{
    // The following code defines an implicit conversion operator from
    // type C to type A.
    //public static implicit operator A(C c)
    //{
    //    A a = new A();
    //    a = c;
    //    return a;
    //}
}

public class MyClass
{
    public static void F(bool b)
    {
        A a = new A();
        C c = new C();

        // The following line causes CS0173 because there is no implicit
        // conversion from a to c or from c to a.
        object o = b ? a : c;

        // To resolve the error, you can cast a and c.
        // object o = b ? (object)a : (object)c;

        // Alternatively, you can add a conversion operator from class C to
        // class A, or from class A to class C, but not both.
    }

   public static void Main()
   {
      F(true);
   }
}
```

```csharp
class M
{
    static int Main ()
    {
        int X = 1;
        // The following line causes CS0173.
        object o = (X == 0) ? null : null;
        return -1;
    }
}
```
