# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-narrowing-conversion"></a>\<methodname>Daraltma dönüştürmesi olmadan bu bağımsız değişkenlerle hiçbir erişilebilir aşırı yüklü ' ' çağrılamaz

Aşırı yüklenmiş bir yöntem çağrıldı, ancak bir daraltma dönüştürmesi olmadan belirtilen bağımsız değişkenlerin listesiyle hiçbir yöntem eşleşmedi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Option Strict Off` belirtin.  
  
2. Bağımsız değişkenleri, aşırı yüklenmiş metodun imzalarından biriyle eşleşecek şekilde değiştirin.
