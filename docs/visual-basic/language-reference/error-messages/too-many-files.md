# <a name="too-many-files"></a>Çok fazla sayıda dosya

Kök dizinde işletim sisteminin izin verdiğinden daha fazla dosya oluşturuldu veya CONFIG.SYS dosyanızda **Files =** Setting içinde belirtilen sayıdan daha fazla dosya açıldı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Programınız kök dizindeki dosyaları açıyor, kapatıyor veya kaydediyor ise, programınızı bir alt dizin kullanacak şekilde değiştirin.  
  
2. CONFIG.SYS dosyanızda **Files =** Setting dosyasında belirtilen dosya sayısını artırın ve bilgisayarınızı yeniden başlatın.
