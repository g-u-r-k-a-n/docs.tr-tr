Özniteliklerin ortak olması gerekir.  
  
 **Hata kimliği:** BC31516  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Public`Değiştiricisini Attribute sınıfına ekleyin.
