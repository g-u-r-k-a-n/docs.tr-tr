# <a name="messageloggingon"></a>MessageLoggingOn

Kimlik: 116  
  
 Önem derecesi: Hata  
  
 Kategori: ServiceModel  
  
## <a name="description"></a>Açıklama  

 İleti günlüğe kaydetme açıldı. Hassas bilgiler, hatta şifreli olsa bile günlüğe kaydedilebilir. Örneğin, ileti gövdeleri. Olay AppID, özel durum, işlem adı ve işlem KIMLIĞI ' ni listeler.
