# <a name="loop-control-variable-of-type-typename-does-not-implement-the-systemicomparable-interface"></a>' ' Türündeki döngü denetim değişkeni \<typename> ' System. IComparable ' arabirimini uygulamıyor

Değişken, arabirimi uygulamayan bir türdür `System.IComparable` . `System.IComparable` değer türü veya sınıf tarafından uygulanmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Değişkenin bir değer türü olduğundan emin olun.
