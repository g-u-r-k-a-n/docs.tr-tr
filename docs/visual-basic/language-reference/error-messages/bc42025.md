## <a name="example"></a><span data-ttu-id="3a66e-113">Örnek</span><span class="sxs-lookup"><span data-stu-id="3a66e-113">Example</span></span>

<span data-ttu-id="3a66e-114">Aşağıdaki örnek BC42025 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3a66e-114">The following example generates bc42025:</span></span>

```vb
Public Class TestClass
    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        Dim tc As New TestClass()
        tc.SayHello() ' BC42025.
    End Sub
End Module
```

## <a name="to-address-this-warning"></a><span data-ttu-id="3a66e-115">Bu uyarıyı çözmek için</span><span class="sxs-lookup"><span data-stu-id="3a66e-115">To address this warning</span></span>
