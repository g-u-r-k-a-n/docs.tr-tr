# <a name="compiler-error-cs1565"></a>Derleyici hatası CS1565
Çakışan seçenekler belirtildi: Win32 kaynak dosyası; Win32 simgesi  
  
 Aynı derlemede hem [-win32res](../language-reference/compiler-options/win32res-compiler-option.md) derleyici seçeneğini hem de [-win32ico](../language-reference/compiler-options/win32icon-compiler-option.md) derleyici seçeneğini belirtmek geçerli değildir.
