# <a name="compiler-error-cs1059"></a><span data-ttu-id="fe13e-103">Derleyici hatası CS1059</span><span class="sxs-lookup"><span data-stu-id="fe13e-103">Compiler Error CS1059</span></span>

## <a name="example"></a><span data-ttu-id="fe13e-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="fe13e-111">Example</span></span>  
 <span data-ttu-id="fe13e-112">Aşağıdaki örnek, bir `i` değişken değil bir sabit olduğu ve bir `E` tür olduğundan `Enum` , öğeleri de sabit değerler olan bir CS1059 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="fe13e-112">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
