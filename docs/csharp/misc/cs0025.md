# <a name="compiler-error-cs0025"></a>Derleyici hatası CS0025
' Dosya ' standart kitaplık dosyası bulunamadı  
  
 Derleyici için gerekli bir dosya bulunamadı. Yolun doğru olduğundan ve dosyanın var olduğundan emin olun.  
  
 Dosya bir Visual Studio sistem dosyası ise, Visual Studio yüklemenizi onarmanız veya tamamen yeniden yüklemeniz gerekebilir.
