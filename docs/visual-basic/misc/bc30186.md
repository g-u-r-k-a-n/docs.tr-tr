# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>' Using ' deyimleri Komut penceresinde geçerli değildir
`Using` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30186  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Using` ifadesini **komut** penceresinden kaldırın.
