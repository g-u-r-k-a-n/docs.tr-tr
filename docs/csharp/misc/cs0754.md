## <a name="example"></a><span data-ttu-id="2dc0d-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="2dc0d-108">Example</span></span>  
 <span data-ttu-id="2dc0d-109">Aşağıdaki kod CS0754 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2dc0d-109">The following code generates CS0754:</span></span>

        void Part();
