# <a name="synclock-operand-cannot-be-of-type-typename-because-typename-is-not-a-reference-type"></a>' \<typename> \<typename> ' Bir başvuru türü olmadığından, ' SyncLock ' işleneni ' ' türünde olamaz
`SyncLock`Deyimi, birden çok iş parçacığının aynı anda aynı deyimleri yürütmemesini sağlayarak deyimlerin tek bir ifadede eşitlenmesini sağlar. Bir deyimde ifade türü, `SyncLock` sınıf, modül, arabirim, dizi veya temsilci gibi bir başvuru türü olmalıdır.  
  
 **Hata kimliği:** BC30582  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Türü uygun bir başvuru türü olarak değiştirin.
