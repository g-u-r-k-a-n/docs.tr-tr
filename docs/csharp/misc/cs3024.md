# <a name="compiler-warning-cs3024"></a><span data-ttu-id="c4123-102">Derleyici Uyarısı CS3024</span><span class="sxs-lookup"><span data-stu-id="c4123-102">Compiler Warning CS3024</span></span>

<span data-ttu-id="c4123-103">' Type ' kısıtlama türü CLS uyumlu değil.</span><span class="sxs-lookup"><span data-stu-id="c4123-103">Constraint type 'type' is not CLS-compliant.</span></span>  
  
 <span data-ttu-id="c4123-104">Bu uyarı, CLS uyumlu olmayan bir türün genel bir tür kısıtlaması olarak kullanılması, bazı dillerde yazılan kodun genel sınıfınızı kullanmasını olanaksız hale getirecek olduğundan bu uyarıyı yayınlar.</span><span class="sxs-lookup"><span data-stu-id="c4123-104">The compiler issues this warning because the use of a non-CLS-compliant type as a generic type constraint could make it impossible for code written in some languages to consume your generic class.</span></span>  
  
### <a name="to-eliminate-this-warning"></a><span data-ttu-id="c4123-105">Bu uyarıyı kaldırmak için</span><span class="sxs-lookup"><span data-stu-id="c4123-105">To eliminate this warning</span></span>  
  
1. <span data-ttu-id="c4123-106">Tür kısıtlaması için CLS uyumlu bir tür kullanın.</span><span class="sxs-lookup"><span data-stu-id="c4123-106">Use a CLS-compliant type for the type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c4123-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="c4123-107">Example</span></span>  

 <span data-ttu-id="c4123-108">Aşağıdaki örnek çeşitli konumlarda CS3024 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c4123-108">The following example generates CS3024 in several locations:</span></span>  
  
```csharp  
// cs3024.cs  
// Compile with: /target:library  
 [assembly: System.CLSCompliant(true)]  
  
[type: System.CLSCompliant(false)]  
public class TestClass // CS3024  
{  
    public ushort us;  
}  
[type: System.CLSCompliant(false)]  
public interface ITest // CS3024  
{}  
public interface I<T> where T : TestClass  
{}  
public class TestClass_2<T> where T : ITest  
{}  
public class TestClass_3<T> : I<T> where T : TestClass  
{}  
public class TestClass_4<T> : TestClass_2<T> where T : ITest  
{}  
public class Test  
{  
    public static int Main()  
    {  
        return 0;  
    }  
}  
```
