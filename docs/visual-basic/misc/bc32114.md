# <a name="type-parameter-with-a-structure-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="6ca82-102">' Structure ' kısıtlaması olan tür parametresi kısıtlama olarak kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="6ca82-102">Type parameter with a 'Structure' constraint cannot be used as a constraint</span></span>
<span data-ttu-id="6ca82-103">Kısıtlama içeren bir tür parametresi `Structure` , başka bir tür parametresi için kısıtlama olarak kullanılır.</span><span class="sxs-lookup"><span data-stu-id="6ca82-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="6ca82-104">Kısıtlama, tür `Structure` bağımsız değişkeninin tür parametresine geçirilen değer türü olmasını gerektirir.</span><span class="sxs-lookup"><span data-stu-id="6ca82-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="6ca82-105">Ancak, bir değer türü uygulanamıyor veya devralınamaz, bu nedenle onu uygulamak veya ondan devralması için diğer tür parametresinin gerekli olduğu bir kısıtlama olarak kullanılması anlamlı değildir.</span><span class="sxs-lookup"><span data-stu-id="6ca82-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="6ca82-106">Bu durumun yalnızca anlamlı yorumu, her iki tür bağımsız değişkeni de tam olarak aynı türde olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="6ca82-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="6ca82-107">Bu durumda, genel türlerinizin yalnızca bir tür parametresi olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="6ca82-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="6ca82-108">Aşağıdaki ifade bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="6ca82-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="6ca82-109">Geçirilen tür `t2` `t1` `t1` bir değer türü olması gerektiğinden, geçirilen tür ' e uygulanamıyor veya devralma türünü alamaz.</span><span class="sxs-lookup"><span data-stu-id="6ca82-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="6ca82-110">**Hata kimliği:** BC32114</span><span class="sxs-lookup"><span data-stu-id="6ca82-110">**Error ID:** BC32114</span></span>
