# <a name="unable-to-get-type-information-for-typename"></a>' ' İçin tür bilgisi alınamıyor \<typename>

Ortak dil çalışma zamanı tarafından yüklenmeyen bir türe başvuruldu.  
  
 **Hata kimliği:** BC30751  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Türün geçerli kapsam içinde tanımlanması için kodunuzu yeniden yapılandır.
