# <a name="compiler-error-cs1562"></a>Derleyici hatası CS1562

Kaynağı olmayan çıkışlar için/out seçeneği belirtilmelidir  
  
 Derleme bir çıkış dosyası oluşturabilir, ancak çıkış dosyası adının çıkarsanbileceği girdi olarak kaynak kodu dosyası yoktu. Örneğin, bir meta veri veya yalnızca kaynak dosyası derlemeye çalışıyor olabilirsiniz.  
  
 Çıkış dosyasının adını belirtmek için [-Out](../language-reference/compiler-options/out-compiler-option.md) derleyici seçeneğini kullanın.
