# <a name="compiler-error-cs0540"></a><span data-ttu-id="3fff9-103">Derleyici hatası CS0540</span><span class="sxs-lookup"><span data-stu-id="3fff9-103">Compiler Error CS0540</span></span>

<span data-ttu-id="3fff9-104">' arabirim üyesi ': içeren tür ' Interface ' arabirimini uygulamıyor</span><span class="sxs-lookup"><span data-stu-id="3fff9-104">'interface member' : containing type does not implement interface 'interface'</span></span>  
  
 <span data-ttu-id="3fff9-105">Arabirimden türemeyen bir [sınıfta](../language-reference/keywords/class.md) bir arabirim üyesini uygulamaya [çalıştınız.](../language-reference/keywords/interface.md)</span><span class="sxs-lookup"><span data-stu-id="3fff9-105">You attempted to implement an interface member in a [class](../language-reference/keywords/class.md) that does not derive from the [interface](../language-reference/keywords/interface.md).</span></span> <span data-ttu-id="3fff9-106">Arabirim üyesinin uygulamasını silmeniz veya arabirimini sınıfının temel sınıf listesine eklemeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="3fff9-106">You should either delete the implementation of the interface member or add the interface to the base-class list of the class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3fff9-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="3fff9-107">Example</span></span>  

 <span data-ttu-id="3fff9-108">Aşağıdaki örnek CS0540 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="3fff9-108">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
