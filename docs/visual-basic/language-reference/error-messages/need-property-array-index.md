# <a name="need-property-array-index"></a>Need özelliği dizi dizini

Bu özellik değeri, tek bir değer yerine bir diziden oluşur. Erişmeye çalıştığınız özellik dizisinin dizinini belirtmediniz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Diziye uygun dizinlerin aralığını bulmak için bileşenin belgelerini denetleyin. Özellik erişim deyiminizde uygun bir dizin belirtin.
