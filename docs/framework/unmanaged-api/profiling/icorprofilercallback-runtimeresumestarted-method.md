# <a name="icorprofilercallbackruntimeresumestarted-method"></a>ICorProfilerCallback::RuntimeResumeStarted Yöntemi

## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT RuntimeResumeStarted();  
```
