# <a name="this-system-does-not-contain-support-for-the-simplifiedchinese-locale"></a>Bu sistem SimplifiedChinese yerel ayarı için destek içermiyor

`VbStrConv.SimplifiedChinese`Listeleme üyesini kullanmaya çalışıyorsunuz, ancak sisteminiz Basitleştirilmiş Çince yerel ayarı 'nı desteklemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Basitleştirilmiş Çince yerel ayarını desteklemek için makineyi yükseltin veya uygulamayı yerel ayarı destekleyen bir makinede çalıştırın.
