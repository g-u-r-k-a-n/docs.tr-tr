# <a name="using-metadata"></a>Meta Verileri Kullanma
Hizmet meta verileri, hizmetin makine tarafından okunabilen bir açıklamasını içerir. Hizmet meta verileri, hizmet uç noktalarının, bağlamaların, sözleşmelerin, işlemlerin ve mesajların açıklamalarını içerir. Hizmet meta verilerini hizmeti tüketmek için bir istemciyi otomatik olarak oluşturma, hizmet açıklamasını uygulama ve bir istemcinin bağlamasını dinamik olarak güncelleştirme dahil olmak üzere çeşitli amaçlar için kullanabilirsiniz.
