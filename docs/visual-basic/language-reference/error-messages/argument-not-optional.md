# <a name="argument-not-optional-visual-basic"></a>Bağımsız değişken isteğe bağlı değil (Visual Basic)

Bağımsız değişkenlerin sayısı ve türleri beklenenlerle aynı olmalıdır. Yanlış sayıda bağımsız değişken var veya atlanmış bir bağımsız değişken isteğe bağlı değil. Bağımsız değişken, yalnızca yordam tanımında bildirilirse Kullanıcı tanımlı yordama yapılan çağrıdan atlanabilir `Optional` .  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Tüm gerekli bağımsız değişkenleri sağlayın.  
  
2. Atlanan bağımsız değişkenlerin isteğe bağlı olduğundan emin olun. Aksi takdirde, çağrın bağımsız değişkenini sağlayın ya da parametreyi `Optional` tanımda bildirin.
