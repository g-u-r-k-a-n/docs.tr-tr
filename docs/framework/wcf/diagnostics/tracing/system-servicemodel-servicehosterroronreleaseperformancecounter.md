System.ServiceModel.ServiceHostErrorOnReleasePerformanceCounter  
  
## <a name="description"></a>Açıklama  
 ReleasePerformanceCounters çağrılırken ServiceHost 'da hatalar oluştu. Bu izleme, performans sayaçlarını serbest bırakırken hataları belirlemede kullanılabilir.
