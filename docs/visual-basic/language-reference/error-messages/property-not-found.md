# <a name="property-not-found"></a>Özellik bulunamadı
Bu nesne belirtilen özelliği desteklemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özelliğin adının yazımını denetleyin.  
  
2. Nesne gerçekten bir "Caption" veya benzer adlandırılmış özelliği desteklediğinde "metin" özelliği gibi bir şeye erişmeye çalışmadığınızdan emin olmak için nesnenin belgelerini denetleyin.
