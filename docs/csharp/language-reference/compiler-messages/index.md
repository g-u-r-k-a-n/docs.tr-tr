# <a name="c-compiler-errors"></a>C# Derleyici Hataları

Bazı C# derleyicisi hataları, hatanın neden oluşturulduğunu ve bazı durumlarda hatanın nasıl düzeltilebildiğini açıklayan ilgili konulara sahiptir. Belirli bir hata iletisi için yardım kullanılabilir olup olmadığını görmek için aşağıdaki adımlardan birini kullanın.  
  
- [Çıktı Penceresinde](/visualstudio/ide/reference/output-window)hata numarasını (örneğin CS0029) bulun ve ardından Microsoft Dokümanlar'da arayın.  
  
- [Çıkış Penceresindeki](/visualstudio/ide/reference/output-window)hata numarasını (örneğin CS0029) seçin ve ardından F1 tuşunu seçin.  
  
- Dizin'e, **Ara** kutusuna hata numarasını girin.  
  
 Bu adımlardan hiçbiri hatanızla ilgili bilgilere yol açmazsa, bu sayfanın sonuna gidin ve hatanın numarasını veya metnini içeren geri bildirim gönderin.  
  
 C#'da hata ve uyarı seçeneklerini yapılandırma hakkında bilgi için Bkz. [Yapı Sayfası, Proje Tasarımcısı (C#)](/visualstudio/ide/reference/build-page-project-designer-csharp).
