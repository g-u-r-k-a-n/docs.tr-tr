# <a name="compiler-error-cs1676"></a><span data-ttu-id="84798-102">Derleyici hatası CS1676</span><span class="sxs-lookup"><span data-stu-id="84798-102">Compiler Error CS1676</span></span>

<span data-ttu-id="84798-103">' Number ' parametresi ' anahtar sözcüğü ' anahtar sözcüğüyle bildirilmelidir</span><span class="sxs-lookup"><span data-stu-id="84798-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="84798-104">Bu hata, anonim bir yöntemde parametre türü değiştiricisi, yöntemini atamak istediğiniz temsilcinin bildiriminde kullanılmasından farklı olduğunda oluşur.</span><span class="sxs-lookup"><span data-stu-id="84798-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="84798-105">Aşağıdaki örnek CS1676 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="84798-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
