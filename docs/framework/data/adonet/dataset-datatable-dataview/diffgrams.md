## <a name="sample-diffgram"></a><span data-ttu-id="c9afe-177">Örnek DiffGram</span><span class="sxs-lookup"><span data-stu-id="c9afe-177">Sample DiffGram</span></span>  
 <span data-ttu-id="c9afe-178">DiffGram biçiminin bir örneği aşağıda gösterilmiştir.</span><span class="sxs-lookup"><span data-stu-id="c9afe-178">An example of the DiffGram format is shown below.</span></span> <span data-ttu-id="c9afe-179">Bu örnek, değişiklikler ayrılmadan önce bir tablodaki bir satıra güncelleştirmenin sonucunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="c9afe-179">This example shows the result of an update to a row in a table before the changes have been committed.</span></span> <span data-ttu-id="c9afe-180">"ALFKI" CustomerID ile satır değiştirildi, ancak güncelleştirilmedi.</span><span class="sxs-lookup"><span data-stu-id="c9afe-180">The row with a CustomerID of "ALFKI" has been modified, but not updated.</span></span> <span data-ttu-id="c9afe-181">Sonuç olarak, **\<** ***DataInstance*** **>** bloğunda "Müşteriler1" **diffgr:id** ve **diffgr:id** \*\* \<diffgr:id\*\* ile **özgün** bir satır diffgr:id ile **>** bloğu vardır.</span><span class="sxs-lookup"><span data-stu-id="c9afe-181">As a result, there is a **Current** row with a **diffgr:id** of "Customers1" in the **\<** ***DataInstance*** **>** block, and an **Original** row with a **diffgr:id** of "Customers1" in the **\<diffgr:before>** block.</span></span> <span data-ttu-id="c9afe-182">"ANATR" customerid ile satır **rowerror**içerir, bu yüzden açıklamalı `diffgr:hasErrors="true"` ve \*\* \<diffgr\*\* ilgili bir öğe var:hatalar>blok.</span><span class="sxs-lookup"><span data-stu-id="c9afe-182">The row with a CustomerID of "ANATR" includes a **RowError**, so it is annotated with `diffgr:hasErrors="true"` and there is a related element in the **\<diffgr:errors>** block.</span></span>

```xml  
    <Customers diffgr:id="Customers2" msdata:rowOrder="1" diffgram:hasErrors="true">

    </Customers>

    </Customers>  
  </CustomerDataSet>  
  <diffgr:before>  
    <Customers diffgr:id="Customers1" msdata:rowOrder="0">  
      <CustomerID>ALFKI</CustomerID>

    </Customers>  
  </diffgr:before>  
  <diffgr:errors>  
    <Customers diffgr:id="Customers2" diffgr:Error="An optimistic concurrency violation has occurred for this row."/>

</diffgr:diffgram>  
```
