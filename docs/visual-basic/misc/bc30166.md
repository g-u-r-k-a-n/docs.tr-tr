# <a name="cannot-create-an-instance-of-module-modulename"></a>' ' Modülünün örneği oluşturulamıyor \<modulename>

Modül yalnızca tek bir paylaşılan örnek olarak bulunur ve ek örnekler oluşturulamaz.  
  
 **Hata kimliği:** BC30166  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Modülü bir sınıf olarak değiştirin veya `New` yan tümcedeki bir sınıf adıyla değiştirin.
