# <a name="missing-required-white-space"></a>Gerekli boşluk eksik

Gerekli bir ayırıcı alanı olmadan bir XML sabit değeri belirtildi. Gerekli bir ayırıcı alanı örneği, bir XML öğesi değişmez değerinde sonraki öznitelikler arasındaki alandır.  
  
 **Hata kimliği:** BC31173  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gerekli alan ayırıcısını ekleyin.
