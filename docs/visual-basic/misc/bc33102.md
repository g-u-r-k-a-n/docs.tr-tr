<span data-ttu-id="edbd8-103">Null yapılabilir tür değiştiricisi (?) değişkeni, belirtilen değişken türünde dizi değiştiricilerin (parantezler) dahil olduğu değişken bildiriminde bulunur.</span><span class="sxs-lookup"><span data-stu-id="edbd8-103">The nullable type modifier (?) is included on the variable in a variable declaration in which the array modifiers (parentheses) are included on the specified variable type.</span></span> <span data-ttu-id="edbd8-104">Ya da null yapılabilir tür değiştiricisi, dizi değiştiricilerin değişkene dahil olduğu bir değişken bildiriminde belirtilen değişken türüne dahil edilir.</span><span class="sxs-lookup"><span data-stu-id="edbd8-104">Or, the nullable type modifier is included on the specified variable type in a variable declaration in which the array modifiers are included on the variable.</span></span>

1. <span data-ttu-id="edbd8-107">Aşağıdaki örnekte gösterildiği gibi, belirtilen değişkende veya belirtilen değişken türünde null yapılabilir tür değiştiricisi (?) ve dizi değiştiricilerini (parantezler) belirtin.</span><span class="sxs-lookup"><span data-stu-id="edbd8-107">Specify both the nullable type modifier (?) and the array modifiers (parentheses) on either the declared variable or the specified variable type, as shown in the following example.</span></span>
