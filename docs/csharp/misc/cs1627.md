# <a name="compiler-error-cs1627"></a><span data-ttu-id="506af-103">Derleyici hatası CS1627</span><span class="sxs-lookup"><span data-stu-id="506af-103">Compiler Error CS1627</span></span>
<span data-ttu-id="506af-104">Bırakma dönüşünden sonra ifade bekleniyor</span><span class="sxs-lookup"><span data-stu-id="506af-104">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="506af-105">Bu hata, `yield` bir ifade olmadan kullanılırsa oluşur.</span><span class="sxs-lookup"><span data-stu-id="506af-105">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="506af-106">Bu hatayı önlemek için, ifadeye uygun ifadeyi ekleyin.</span><span class="sxs-lookup"><span data-stu-id="506af-106">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="506af-107">Aşağıdaki örnek CS1627 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="506af-107">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
