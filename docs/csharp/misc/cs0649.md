# <a name="compiler-warning-level-4-cs0649"></a><span data-ttu-id="86b2c-103">Derleyici Uyarısı (düzey 4) CS0649</span><span class="sxs-lookup"><span data-stu-id="86b2c-103">Compiler Warning (level 4) CS0649</span></span>
<span data-ttu-id="86b2c-104">' Field ' alanı hiçbir zaman atanmaz ve her zaman varsayılan ' Value ' değerine sahip olur</span><span class="sxs-lookup"><span data-stu-id="86b2c-104">Field 'field' is never assigned to, and will always have its default value 'value'</span></span>  
  
 <span data-ttu-id="86b2c-105">Derleyici hiçbir değer atanmamış bir başlatılmamış özel veya iç alan bildirimi algıladı.</span><span class="sxs-lookup"><span data-stu-id="86b2c-105">The compiler detected an uninitialized private or internal field declaration that is never assigned a value.</span></span>  
  
 <span data-ttu-id="86b2c-106">Aşağıdaki örnek CS0649 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="86b2c-106">The following sample generates CS0649:</span></span>  
  
```csharp  
// CS0649.cs  
// compile with: /W:4  
using System.Collections;  
  
class MyClass  
{  
   Hashtable table;  // CS0649  
   // You may have intended to initialize the variable to null  
   // Hashtable table = null;  
  
   // Or you may have meant to create an object here  
   // Hashtable table = new Hashtable();  
  
   public void Func(object o, string p)  
   {  
      // Or here  
      // table = new Hashtable();  
      table[p] = o;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
