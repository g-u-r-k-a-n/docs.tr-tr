# <a name="expected--for-xml-end-tag"></a>For XML End etiketi için beklenen '/'

Bir XML sabit değerinde XML End etiketi gerekli '/' karakterini içermiyor.  
  
 **Hata kimliği:** BC31167  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML sabit değeri için XML bitiş etiketinin başlangıcına '/' karakterini ekleyin.
