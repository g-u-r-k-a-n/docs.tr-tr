# <a name="unable-to-open-key-file-filename-error"></a>' ' Anahtar dosyası açılamıyor \<filename> : \<error>

Bu proje için belirtilen anahtar dosyası açılamadı.  
  
 **Hata kimliği:** BC31025  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Anahtar dosyasının var olduğunu ve başka bir uygulama tarafından kilitlenmediğini doğrulayın.
