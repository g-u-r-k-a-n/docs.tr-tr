# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>İfade bir dizi veya yöntem değil ve bağımsız değişken listesine sahip olamaz
Dizi veya yöntem olmayan bir ifade için bir bağımsız değişken listesi sağladınız. Yalnızca diziler veya yöntemler bağımsız değişken listelerine sahip olabilir.  
  
 **Hata kimliği:** BC30471  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken listesini kaldırın.
