# <a name="systemservicemodeltxasyncabort"></a>System.ServiceModel.TxAsyncAbort

Belirtilen işlem zaman uyumsuz olarak iptal edildi.  
  
## <a name="description"></a>Açıklama  

 Geçerli işlem, başka bir katılımcının Iptali, zaman aşımları veya bir işlemin katılımcısı içinde başka bir hata nedeniyle sonlandırıldı.  
  
## <a name="troubleshooting"></a>Sorun giderme  

 Bu iptal işlemi için gerçek nedenin belirlenmesi için bu iptal beklenmiyorsa tüm sistem günlüklerine bakın.
