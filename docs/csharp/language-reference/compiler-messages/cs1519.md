# <a name="compiler-error-cs1519"></a><span data-ttu-id="82599-102">Derleyici Hatası CS1519</span><span class="sxs-lookup"><span data-stu-id="82599-102">Compiler Error CS1519</span></span>
<span data-ttu-id="82599-103">Sınıf, yapı veya arabirim üye bildiriminde geçersiz belirteç 'belirteç'</span><span class="sxs-lookup"><span data-stu-id="82599-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="82599-104">Bu hata, ait olmadığı bir konumda bir belirteç le karşılaşıldığında oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="82599-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="82599-105">*Belirteç* bir anahtar kelimedir; bir tanımlayıcı (bir sınıfın adı, yapı, yöntem vb.); 108, "Hello" veya "A" gibi bir dize, karakter veya sayısal gerçek değer; veya bir operatör veya noktalama `==` gibi `;`ya da .</span><span class="sxs-lookup"><span data-stu-id="82599-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="82599-106">Türden önce geçersiz değiştiriciler içeren herhangi bir [sınıf,](../keywords/class.md)yapı veya arabirim üye bildirimi bu hatayı oluşturur.</span><span class="sxs-lookup"><span data-stu-id="82599-106">Any [class](../keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="82599-107">Hatayı düzeltmek için geçersiz değiştiriciler kaldırın.</span><span class="sxs-lookup"><span data-stu-id="82599-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="82599-108">Aşağıdaki örnek, belirteçler geçerli olmadıkları yerlere yerleştirildiğinden, beş yerde CS1519 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="82599-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;
// Generates CS1519 because of "checked" on void method:  
    checked void f4();
  
// Generates CS1519 because of "num":  
    void f5(int a num){}
  
// Generates CS1519 because of namespace inside class:  
    namespace;
  
}  
```
