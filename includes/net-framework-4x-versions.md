> [!IMPORTANT]
> .NET Framework 4 ' teki tüm .NET Framework sürümleri yerinde güncelleştirmeler olduğundan, sistemde yalnızca tek bir 4. x sürümü bulunabilir. Ayrıca, .NET Framework belirli sürümleri, Windows işletim sisteminin bazı sürümlerinde önceden yüklenir. Bunun anlamı:
>
> - Zaten makinede daha sonraki 4. x sürümü yüklüyse, önceki 4. x sürümünü yükleyemezsiniz.
> - İşletim sistemi, belirli bir .NET Framework sürümüyle Önceden yüklenmiş olarak geliyorsa, önceki 4. x sürümünü aynı makineye yükleyemezsiniz.
> - Daha sonraki bir sürümünü yüklerseniz, önce önceki sürümü kaldırmanız gerekmez.
