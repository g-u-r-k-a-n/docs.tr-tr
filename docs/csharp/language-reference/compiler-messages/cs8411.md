# <a name="compiler-error-cs8411"></a><span data-ttu-id="b5ec2-103">Derleyici hatası CS8411</span><span class="sxs-lookup"><span data-stu-id="b5ec2-103">Compiler Error CS8411</span></span>

<span data-ttu-id="b5ec2-104">' Type ', ' GetAsyncEnumerator ' için uygun bir ortak örnek tanımı içermediğinden, zaman uyumsuz foreach ifadesi ' Type ' türündeki değişkenlerde çalışamaz</span><span class="sxs-lookup"><span data-stu-id="b5ec2-104">Asynchronous foreach statement cannot operate on variables of type 'type' because 'type' does not contain a suitable public instance definition for 'GetAsyncEnumerator'</span></span>

<span data-ttu-id="b5ec2-105">`await foreach` deyimlerde yalnızca tanımına sahip olan türler üzerinde çalışır `GetAsyncEnumerator` `IAsyncEnumerable<T>` .</span><span class="sxs-lookup"><span data-stu-id="b5ec2-105">`await foreach` statement operates only on types having a definition of `GetAsyncEnumerator`, such as `IAsyncEnumerable<T>`.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="b5ec2-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="b5ec2-106">To correct this error</span></span>

<span data-ttu-id="b5ec2-107">`await foreach` yerine `foreach` yazın.</span><span class="sxs-lookup"><span data-stu-id="b5ec2-107">Replace `await foreach` with `foreach`.</span></span>

## <a name="example"></a><span data-ttu-id="b5ec2-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="b5ec2-108">Example</span></span>

```csharp
using System.Collections.Generic;
using System.Threading.Tasks;

class Program
{
    async Task Example(IAsyncEnumerator<int> enumerator)
    {
        // error CS8411: Asynchronous foreach statement cannot operate on variables
        // of type 'IAsyncEnumerator<T>' because 'IAsyncEnumerator<T>' does not
        // contain a suitable public instance definition for 'GetAsyncEnumerator'
        await foreach (int i in enumerator)
        {
        }
    }
}
```
