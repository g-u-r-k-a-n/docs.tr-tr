# <a name="unable-to-get-serial-port-names-because-of-an-internal-system-error"></a>İç sistem hatası nedeniyle seri bağlantı noktası adları alınamıyor

Özellik çağrıldığında bir iç hata oluştu `My.Computer.Ports.SerialPortNames` .  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Daha fazla sorun giderme bilgisi için bkz. [hata ayıklayıcı temelleri](/visualstudio/debugger/debugger-basics) .  
  
2. Hatanın gerçekleştiği koşullara ve Microsoft Ürün Destek Hizmetleri 'ne çağrı yapın.
