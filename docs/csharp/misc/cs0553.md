# <a name="compiler-error-cs0553"></a><span data-ttu-id="cd8bd-103">Derleyici hatası CS0553</span><span class="sxs-lookup"><span data-stu-id="cd8bd-103">Compiler Error CS0553</span></span>
<span data-ttu-id="cd8bd-104">' dönüştürme yordamı ': taban sınıfına/öğesinden Kullanıcı tanımlı dönüştürme</span><span class="sxs-lookup"><span data-stu-id="cd8bd-104">'conversion routine' : user defined conversion to/from base class</span></span>  
  
 <span data-ttu-id="cd8bd-105">Bir taban sınıfın değerlerine Kullanıcı tanımlı Dönüştürmelere izin verilmiyor; Böyle bir işlece gerek yoktur.</span><span class="sxs-lookup"><span data-stu-id="cd8bd-105">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="cd8bd-106">Aşağıdaki örnek CS0553 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="cd8bd-106">The following sample generates CS0553:</span></span>  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
