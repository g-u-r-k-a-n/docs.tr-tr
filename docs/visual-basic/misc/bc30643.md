# <a name="property-propertyname-is-of-an-unsupported-type"></a>' \<propertyname> ' Özelliği desteklenmeyen bir türde
Başvurulan derlemedeki bir özellik temsil edilemeyecek bir tür içeriyor.  
  
 **Hata kimliği:** BC30643  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Uygulamanızla birlikte kullanılabilecek bir tür belirtin.
