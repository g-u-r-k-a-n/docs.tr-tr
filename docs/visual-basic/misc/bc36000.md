 <span data-ttu-id="00e4b-104">Amacı, kodunuzun kök düzeyi ad alanını `Global` engellediği bir ad alanı yapısının içinden kök düzeyinde bir ad alanına erişmesine imkan sağlamaktır.</span><span class="sxs-lookup"><span data-stu-id="00e4b-104">The purpose of `Global` is to allow your code to access a root-level namespace from inside a namespace structure that has blocked the root-level namespace.</span></span>
