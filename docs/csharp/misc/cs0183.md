# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="e9c24-103">Derleyici Uyarısı (düzey 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="e9c24-103">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="e9c24-104">Verilen ifade her zaman sağlanan (' tür ') türündedir</span><span class="sxs-lookup"><span data-stu-id="e9c24-104">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="e9c24-105">Bir koşullu ifade her zaman **true**olarak değerlendirilirse, koşullu bir deyime gerek kalmaz.</span><span class="sxs-lookup"><span data-stu-id="e9c24-105">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="e9c24-106">Bu uyarı, bir türü **,** bir tür işleci kullanarak değerlendirmeyi denediğinizde oluşur.</span><span class="sxs-lookup"><span data-stu-id="e9c24-106">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="e9c24-107">Değerlendirme bir değer türü ise, denetim gereksizdir.</span><span class="sxs-lookup"><span data-stu-id="e9c24-107">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="e9c24-108">Aşağıdaki örnek CS0183 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e9c24-108">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);
   }  
}  
```
