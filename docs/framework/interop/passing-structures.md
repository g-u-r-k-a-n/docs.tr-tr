## <a name="declaring-and-passing-structures"></a><span data-ttu-id="036bd-131">Yapıları bildirme ve geçirme</span><span class="sxs-lookup"><span data-stu-id="036bd-131">Declaring and Passing Structures</span></span>  

 <span data-ttu-id="036bd-132">Aşağıdaki örnek, `Point` yönetilen kodda ve yapıların nasıl tanımlanacağını gösterir `Rect` ve türleri parametre olarak User32.dll dosyasındaki **ptinrect** işlevine iletir.</span><span class="sxs-lookup"><span data-stu-id="036bd-132">The following example shows how to define the `Point` and `Rect` structures in managed code, and pass the types as parameter to the **PtInRect** function in the User32.dll file.</span></span> <span data-ttu-id="036bd-133">**Pınrect** aşağıdaki yönetilmeyen imzaya sahiptir:</span><span class="sxs-lookup"><span data-stu-id="036bd-133">**PtInRect** has the following unmanaged signature:</span></span>

 <span data-ttu-id="036bd-134">İşlev bir RECT türüne işaretçi beklediği için rect yapısını başvuruya göre geçirmeniz gerektiğini unutmayın.</span><span class="sxs-lookup"><span data-stu-id="036bd-134">Notice that you must pass the Rect structure by reference, since the function expects a pointer to a RECT type.</span></span>

Imports System.Runtime.InteropServices  
  
<StructLayout(LayoutKind.Sequential)> Public Structure Point  
    Public x As Integer

```csharp  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct Point {  
    public int x;  
    public int y;  
}
  
[StructLayout(LayoutKind.Explicit)]  
public struct Rect {  
    [FieldOffset(0)] public int left;  
    [FieldOffset(4)] public int top;  
    [FieldOffset(8)] public int right;  
    [FieldOffset(12)] public int bottom;  
}
  
internal static class NativeMethods
{  
    [DllImport("User32.dll")]  
    internal static extern bool PtInRect(ref Rect r, Point p);  
}  
```

[StructLayout(LayoutKind.Sequential)]  
public class MySystemTime {

{  
    public static void Main()
