# <a name="expected-closing--for-embedded-expression"></a>Katıştırılmış ifade için '% > ' kapanışı bekleniyor

Bir XML sabit değerinde eklenmiş bir ifadede gerekli kapanış `%>` karakterleri eksik.  
  
 **Hata kimliği:** BC31159  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `%>`Eklenmiş ifadenin sonuna gereken kapatma karakterlerini ekleyin.
