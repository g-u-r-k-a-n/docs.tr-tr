# <a name="class-classname-cannot-be-indexed-because-it-has-no-default-property"></a>' \<classname> ' Sınıfının varsayılan özelliği olmadığından dizini oluşturulamaz

Bir sınıf için varsayılan bir özellik sağlayatamamladınız.  
  
 **Hata kimliği:** BC30367  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimini anahtar sözcüğüyle başlatarak varsayılan özellik olarak bir özellik belirtin `Default` .
