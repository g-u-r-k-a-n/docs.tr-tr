# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>' ' Varsayılan bütünleştirilmiş kod başvurusu \<reference> zaten otomatik olarak eklenmiş; yoksayıldı

Varsayılan derleme başvurusunu eklemek için bir girişimde bulunuldu, ancak derleme zaten eklenmiş olduğundan, deneme yok sayılır.  
  
 **Hata kimliği:** BC2024  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz kodu kaldırın.
