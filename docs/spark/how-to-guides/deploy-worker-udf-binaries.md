<span data-ttu-id="abbf3-176">**Cevap:** YARN 'de Spark uygulamasını başlatmak için, ortam değişkenlerinin `spark.yarn.appMasterEnv.[EnvironmentVariableName]`olarak belirtilmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="abbf3-176">**Answer:** To launch the spark application on YARN, the environment variables should be specified as `spark.yarn.appMasterEnv.[EnvironmentVariableName]`.</span></span> <span data-ttu-id="abbf3-177">Lütfen `spark-submit`kullanarak aşağıda örnek olarak bakın:</span><span class="sxs-lookup"><span data-stu-id="abbf3-177">Please see below as an example using `spark-submit`:</span></span>

spark-submit \
--class org.apache.spark.deploy.dotnet.DotnetRunner \
--master yarn \
