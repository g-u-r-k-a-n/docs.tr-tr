# <a name="compiler-warning-level-2-cs0472"></a>Derleyici Uyarısı (düzey 2) CS0472

' Değer2 ' türündeki bir değer hiçbir zaman ' value3 ' türündeki ' null ' değerine eşit olmadığından, ifadenin sonucu her zaman ' değer1 ' olur  
  
 Sabit null değeri olan bir işleç kullanırsanız, derleyici uyar.  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS0472 oluşturur.  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;
        return counter;  
    }  
}  
```
