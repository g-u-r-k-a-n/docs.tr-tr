# <a name="compiler-error-cs0653"></a><span data-ttu-id="6dcf5-103">Derleyici hatası CS0653</span><span class="sxs-lookup"><span data-stu-id="6dcf5-103">Compiler Error CS0653</span></span>

<span data-ttu-id="6dcf5-104">Soyut olduğu için ' class ' öznitelik sınıfı uygulanamıyor</span><span class="sxs-lookup"><span data-stu-id="6dcf5-104">Cannot apply attribute class 'class' because it is abstract</span></span>  
  
 <span data-ttu-id="6dcf5-105">[Soyut](../language-reference/keywords/abstract.md) özel öznitelik sınıfı öznitelik olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="6dcf5-105">An [abstract](../language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>  
  
 <span data-ttu-id="6dcf5-106">Aşağıdaki örnek CS0653 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6dcf5-106">The following sample generates CS0653:</span></span>  
  
```csharp

{  
}  
  
public class My2Attribute : MyAttribute  
{  
}
```
