 <span data-ttu-id="c9dde-106">Parametre içermeyen bir genişletme yöntemi geçerli değil çünkü ilk parametre yöntemin hangi veri türünü genişlettiğini belirtiyor.</span><span class="sxs-lookup"><span data-stu-id="c9dde-106">An extension method without parameters is not valid because the first parameter specifies which data type the method extends.</span></span> <span data-ttu-id="c9dde-107">İlk parametre yöntemi çağıran veri türü örneğine bağlanır.</span><span class="sxs-lookup"><span data-stu-id="c9dde-107">The first parameter is bound to the instance of the data type that invokes the method.</span></span>

## <a name="example"></a><span data-ttu-id="c9dde-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="c9dde-111">Example</span></span>  

 <span data-ttu-id="c9dde-112">Aşağıdaki örnekteki ilk parametre, `Print` yöntemin veri türünü genişlettiğini gösterir `String` .</span><span class="sxs-lookup"><span data-stu-id="c9dde-112">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```vb

Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="c9dde-113">Uzantı yöntemi aşağıdaki gibi çağrıldığında, `str` yöntemindeki parametre öğesine bağlanır `greeting` , `String` Bu çağrının örneğidir `Print` .</span><span class="sxs-lookup"><span data-stu-id="c9dde-113">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="c9dde-114">Derleyici, `greeting` genişletme yöntemi için bağımsız değişken olarak kullanacaktır `Print` .</span><span class="sxs-lookup"><span data-stu-id="c9dde-114">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

```vb  
greeting.Print()  
```
