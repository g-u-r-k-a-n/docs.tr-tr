# <a name="interface-can-inherit-only-from-another-interface"></a>Arabirim yalnızca başka bir arabirimden devralınabilir

Arabirim olmayan bir şeyi devralma girişiminde bulunuldu.  
  
 **Hata kimliği:** BC30354  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Devralmayı denediğiniz temel arabirimin veya arabirimlerin gerçekte arabirimler olduğundan emin olun.
