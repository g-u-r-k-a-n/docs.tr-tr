## <a name="example"></a><span data-ttu-id="7c9fe-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="7c9fe-110">Example</span></span>  
 <span data-ttu-id="7c9fe-111">Aşağıdaki örnek CS0840 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7c9fe-111">The following example generates CS0840:</span></span>

{  
    public int myProp { get; } // CS0840
