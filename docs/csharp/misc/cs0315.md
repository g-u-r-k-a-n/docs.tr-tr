# <a name="compiler-error-cs0315"></a><span data-ttu-id="cbec1-102">Derleyici Hatası CS0315</span><span class="sxs-lookup"><span data-stu-id="cbec1-102">Compiler Error CS0315</span></span>

## <a name="example"></a><span data-ttu-id="cbec1-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="cbec1-108">Example</span></span>  
 <span data-ttu-id="cbec1-109">Aşağıdaki örnek CS0315 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="cbec1-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
