# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="45b9b-103">Derleyici Uyarısı (düzey 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="45b9b-103">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="45b9b-104">' interface ' arabirimi ' CoClassAttribute ' ile işaretlenmiş ' ComImportAttribute ' ile işaretlenmemiş</span><span class="sxs-lookup"><span data-stu-id="45b9b-104">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="45b9b-105">Bir arabirimde **CoClassAttribute** belirtirseniz, **ComImportAttribute**de belirtmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="45b9b-105">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="45b9b-106">Aşağıdaki örnek CS0684 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="45b9b-106">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
