# <a name="end-structure-must-be-preceded-by-a-matching-structure"></a>' End Structure ' öncesinde eşleşen bir ' Structure ' olmalıdır
`End Structure`İfade yalnızca bir ifadesiyle başlayan bir yapı bloğunun sonunda kullanılabilir `Structure` .  
  
 **Hata kimliği:** BC30621  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli yapı bloğunun bir ifadesiyle başladığı emin olun `Structure` .
