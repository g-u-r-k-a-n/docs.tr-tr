# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>Başvurulan '\<ObjectName > ' nesnesinin ' Nothing ' değeri vardır
Başvurulan nesne ayarlanmadı veya `Nothing`değere ayarlandı.  
  
 **Hata kimliği:** BC30705  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesnenin adının doğru yazıldığını ve nesnenin ayarlandığını doğrulayın.
