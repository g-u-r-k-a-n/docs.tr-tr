# <a name="compiler-error-cs1713"></a>Derleyici hatası CS1713

Typename1 türü için meta veri adı oluşturulurken beklenmeyen bir hata oluştu; onreason '  
  
 Bu hata genellikle iç derleyici hatasından kaynaklanır. Kodunuzda küçük değişiklikler yapma (örneğin, adın uzunluğunu kısaltmayı ve sonra yeniden derleme), bu hatayı çözebilir.
