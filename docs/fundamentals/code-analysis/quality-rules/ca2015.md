## <a name="how-to-fix-violations"></a><span data-ttu-id="070b8-117">İhlalleri çözme</span><span class="sxs-lookup"><span data-stu-id="070b8-117">How to fix violations</span></span>

```csharp

    public override bool Dispose(bool disposing)
    {
        if (disposing)
        {
            _handle.Dispose();
        }
    }

    ...

    // Violation occurs, remove the finalizer to fix the warning.
    ~DerivedClass() => Dispose(false);
}
```
