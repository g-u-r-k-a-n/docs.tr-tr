# <a name="compiler-warning-level-4-cs0402"></a>Derleyici Uyarısı (düzey 4) CS0402

' tanımlayıcı ': bir giriş noktası genel veya genel bir türde olamaz  
  
 Giriş noktası genel türde bulundu. Bu uyarıyı kaldırmak için, Main ' i genel olmayan bir sınıfta veya yapıda uygulayın.  
  
```csharp  
// CS0402.cs  
// compile with: /W:4  
class C<T>  
{  
   public static void Main()  // CS0402  
   {  
  
   }  
}  
  
class CMain  
{  
   public static void Main() {}  
}  
```
