## <a name="example"></a><span data-ttu-id="962dc-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="962dc-108">Example</span></span>  

 <span data-ttu-id="962dc-109">Aşağıdaki örnek CS0425 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="962dc-109">The following example generates CS0425:</span></span>

```csharp  
{  
}

{  
}

{  
    void F<ItemType>(ItemType item) where ItemType : C1;

    {  
    }

```
