# <a name="set-not-supported-at-run-time"></a>Çalışma zamanında desteklenmeyen ayarla

Yalnızca tasarım zamanında ayarlanmış olabilecek bir özelliği ayarlamaya veya değiştirmeye çalıştınız.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kodunuzda özelliğin başvurusunu kaldırın.  
  
2. Başvuruyu yalnızca çalışma zamanında özelliğin değerini döndürecek şekilde değiştirin.
