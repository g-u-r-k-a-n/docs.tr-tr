## <a name="example"></a><span data-ttu-id="3ddfe-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="3ddfe-108">Example</span></span>

 <span data-ttu-id="3ddfe-109">Aşağıdaki örnek CS0071 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="3ddfe-109">The following sample generates CS0071.</span></span>

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
