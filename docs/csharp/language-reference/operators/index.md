<span data-ttu-id="44856-173">İşleç ilişkilendirilebilirliği tarafından uygulanan değerlendirmenin sırasını değiştirmek için ayraçları kullanın:</span><span class="sxs-lookup"><span data-stu-id="44856-173">Use parentheses to change the order of evaluation imposed by operator associativity:</span></span>
