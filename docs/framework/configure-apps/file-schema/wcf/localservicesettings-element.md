 <span data-ttu-id="a06ad-159">Ayarlar, hizmetin güvenlik ilkesinin bir parçası olarak yayımlanmadığı ve istemcinin bağlamasını etkilemediği için yereldir.</span><span class="sxs-lookup"><span data-stu-id="a06ad-159">The settings are local because they are not published as part of the security policy of the service and do not affect the client's binding.</span></span>
