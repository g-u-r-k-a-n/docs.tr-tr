# <a name="reference-required-to-module-modulename-containing-the-implemented-interface-interfacename"></a>\<modulename>Uygulanan ' ' arabirimini içeren ' ' modülüne başvuru gerekiyor \<interfacename>

\<modulename>Uygulanan ' ' arabirimini içeren ' ' modülüne başvuru gerekiyor \<interfacename> . Projenize bir tane ekleyin.  
  
 Arabirim, projenizde doğrudan başvurulmayan bir modülde tanımlanmıştır. Visual Basic Derleyicisi, arabirimin birden fazla modülde tanımlanması durumunda karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata kimliği:** BC30010  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Başvurulmayan modülün adını Proje başvurularında ekleyin.
