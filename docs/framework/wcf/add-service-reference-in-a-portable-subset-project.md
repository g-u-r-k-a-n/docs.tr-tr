# <a name="add-service-reference-in-a-portable-subset-project"></a>Taşınabilir Alt Küme Projesine Hizmet Başvurusu Ekle
Taşınabilir alt küme projeleri, .NET derleme programcılarının tek bir kaynak ağacı ve derleme sistemi ile aynı zamanda birden çok .NET uygulaması (Masaüstü, Silverlight, Windows Phone ve XBOX) desteklemeye devam etmektedir. Taşınabilir alt küme projeleri yalnızca .NET uygulamalarında kullanılabilecek .NET Framework derlemesi olan .NET taşınabilir kitaplıklarına başvurur.  
  
## <a name="add-service-reference-details"></a>Hizmet Başvurusu Ekle ayrıntıları  
 Taşınabilir alt küme projesine hizmet başvurusu eklerken aşağıdaki kısıtlamalar uygulanır:  
  
1. <xref:System.Xml.Serialization.XmlSerializer>için yalnızca sabit kodlamalara izin verilir. SOAP kodlamaları içeri aktarma sırasında bir hata oluşturur.  
  
2. <xref:System.Runtime.Serialization.DataContractSerializer> senaryolar kullanan hizmetler için, yeniden kullanılan türlerin yalnızca taşınabilir alt kümeden gelmesini sağlamak üzere bir veri sözleşmesi yedeği sağlanır.  
  
3. Taşınabilir kitaplıklarda desteklenmeyen bağlamalara dayanan uç noktalar (<xref:System.ServiceModel.BasicHttpBinding>, işlem akışı olmadan <xref:System.ServiceModel.WSHttpBinding>, güvenilir oturumlar veya MTOM kodlaması ve eşdeğer özel bağlamalar) hariç tüm bağlamalar yok sayılır.  
  
4. İleti üstbilgileri, içeri aktarmadan önce tüm işlemlerde tüm ileti açıklamalarından silinir.  
  
5. Taşınabilir olmayan öznitelikler <xref:System.ComponentModel.DesignerCategoryAttribute>, <xref:System.SerializableAttribute>ve <xref:System.ServiceModel.TransactionFlowAttribute>, oluşturulan istemci proxy kodundan kaldırılır.  
  
6. Taşınabilir olmayan özellikler ProtectionLevel, SessionMode, IsInitiating, ısating <xref:System.ServiceModel.ServiceContractAttribute>, <xref:System.ServiceModel.OperationContractAttribute>ve <xref:System.ServiceModel.FaultContractAttribute>kaldırılır.  
  
7. Tüm hizmet işlemleri istemci proxy 'sinde zaman uyumsuz işlemler olarak oluşturulur.  
  
8. Taşınabilir olmayan türler kullanan oluşturulan istemci oluşturucuları kaldırılır.  
  
9. Oluşturulan istemcide <xref:System.Net.CookieContainer> bir örnek gösterilir.  
  
10. Kod oluşturucunun derlemesini ve sürümünü tanımlayan dosyanın üst kısmına bir açıklama eklenir:`// This code was auto-generated by Microsoft.VisualStudio.Portable.AddServiceReference, version 1.0.0.0`  
  
11. <xref:System.Runtime.Serialization.ISerializable> arabirimi desteklenmiyor.  
  
12. Net. TCP ve PollingDuplex bağlamaları desteklenmez  
  
13. <xref:System.Runtime.Serialization.DataContractSerializer> her zaman hatalar için kullanılacaktır.  
  
14. Taşınabilir alt küme projelerinde <xref:System.ServiceModel.MessageContractAttribute.IsWrapped%2A> desteklenmez.
