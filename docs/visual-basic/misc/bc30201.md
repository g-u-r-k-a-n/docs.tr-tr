# <a name="expression-expected"></a>İfade bekleniyor

Veya gibi bir operatör `+` `=` sağ işleneni için bir ifade gerektirir, ancak tanınabilir bir ifade görüntülenmez.  
  
 **Hata kimliği:** BC30201  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İşlecin sağ işleneni için geçerli bir ifade sağlayın.
