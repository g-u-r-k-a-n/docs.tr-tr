# <a name="datamemberlisteditor-class"></a>Datamemberlistedıtor sınıfı

Değer kaynağını seçmek için `DataSource` nesnesinin tüm özelliklerini listeleyerek, veri bağlantılı nesnelerin (null olmayan `DataSource` özelliğine sahip nesneler) düzenlenmesine yönelik açılan bir kullanıcı arabirimi sağlar.  
  
## <a name="syntax"></a>Sözdizimi
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberListEditor` sınıfı dahili ve doğrudan kodunuzda kullanılmamalıdır.
> 
> Microsoft, bu sınıfın herhangi bir koşulda bir üretim uygulamasında kullanımını desteklemez.
