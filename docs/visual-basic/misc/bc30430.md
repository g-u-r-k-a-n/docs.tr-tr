# <a name="end-function-must-be-preceded-by-a-matching-function"></a>' End Function ' öncesinde eşleşen bir ' function ' olmalıdır

`End Function`Kodunuzun önünde eşleşen yordam tanımı olmadan kodunuzda bir ifade görüntülenir `Function` .  
  
 **Hata kimliği:** BC30430  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Gereksiz ise, `End Function` ifadeyi kaldırın.  
  
2. Eksik `Function` yordamı bir varsa sağlayın.  
  
3. Öğesini `End Function` koddaki uygun yere taşıyın.
