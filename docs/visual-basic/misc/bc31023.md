# <a name="unable-to-parse-xml-error"></a>XML ayrıştırılamıyor: \<error>

Kullanılmakta olan XML dosyasında bir hata var.  
  
 **Hata kimliği:** BC31023  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML 'nin düzgün biçimlendirildiğini, geçerli olduğunu ve dosyanın bozuk olmadığını doğrulayın.
