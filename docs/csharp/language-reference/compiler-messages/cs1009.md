# <a name="compiler-error-cs1009"></a><span data-ttu-id="e8914-102">Derleyici Hatası CS1009</span><span class="sxs-lookup"><span data-stu-id="e8914-102">Compiler Error CS1009</span></span>
<span data-ttu-id="e8914-103">Tanınmayan kaçış sırası</span><span class="sxs-lookup"><span data-stu-id="e8914-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="e8914-104">Beklenmeyen bir karakter, bir [dizedeki](../builtin-types/reference-types.md#the-string-type)ters eğik çizgi (\\) izler.</span><span class="sxs-lookup"><span data-stu-id="e8914-104">An unexpected character follows a backslash (\\) in a [string](../builtin-types/reference-types.md#the-string-type).</span></span> <span data-ttu-id="e8914-105">Derleyici geçerli kaçış karakterlerinden birini bekliyor.</span><span class="sxs-lookup"><span data-stu-id="e8914-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="e8914-106">Daha fazla bilgi için bkz. [karakter kaçışları](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="e8914-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="e8914-107">Aşağıdaki örnek CS1009 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="e8914-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="e8914-108">Bu hatanın yaygın bir nedeni, aşağıdaki örnekte gösterildiği gibi bir dosya adında ters eğik çizgi karakterini kullanmaktır.</span><span class="sxs-lookup"><span data-stu-id="e8914-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="e8914-109">Bu hatayı çözmek için, aşağıdaki örnekte gösterildiği gibi "\\\\" veya @-quoted dize değişmez değerini kullanın.</span><span class="sxs-lookup"><span data-stu-id="e8914-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
