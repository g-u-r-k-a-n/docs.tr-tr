# <a name="compiler-error-cs1040"></a>Derleyici hatası CS1040

Önişlemci yönergeleri satırdaki ilk boşluk olmayan karakter olarak görünmelidir  
  
 Bir satırda ön [işlemci yönergesi](../language-reference/preprocessor-directives.md) bulundu ve satırdaki ilk belirteç değildi. Bir yönerge satırdaki ilk belirteç olmalıdır.  
  
 Aşağıdaki örnek CS1040 oluşturur:  
  
```csharp  
// CS1040.cs  
/* Define a symbol, X */ #define X   // CS1040  
  
// try the following two lines instead  
// /* Define a symbol, X */  
// #define X  
  
public class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
