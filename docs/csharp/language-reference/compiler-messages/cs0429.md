 <span data-ttu-id="9f57e-105">Bu hata, kodunuzdaki bir ifadenin bir parçası ulaşılamaz olduğunda oluşur.</span><span class="sxs-lookup"><span data-stu-id="9f57e-105">This error occurs whenever part of an expression in your code is unreachable.</span></span> <span data-ttu-id="9f57e-106">Aşağıdaki örnekte, `false && myTest()` `myTest()` işlemin sol tarafı `&&` her zaman false olduğu için yöntem hiçbir zaman değerlendirilmeyeceği için koşul bu ölçütleri karşılar.</span><span class="sxs-lookup"><span data-stu-id="9f57e-106">In the following example, the condition `false && myTest()` meets this criteria because the `myTest()` method will never get evaluated due to the fact that the left side of the `&&` operation is always false.</span></span> <span data-ttu-id="9f57e-107">`&&`İşleci, `false` ifadeyi yanlış olarak değerlendirir, değerlendirmeyi bırakır ve sağ tarafı hiçbir şekilde değerlendirmez.</span><span class="sxs-lookup"><span data-stu-id="9f57e-107">As soon as the `&&` operator evaluates the `false` statement as false, it stops the evaluation, and will never evaluate the right side.</span></span>
