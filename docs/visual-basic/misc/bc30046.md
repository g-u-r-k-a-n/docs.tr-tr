# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Metodun hem ParamArray hem de Isteğe bağlı parametreleri olamaz
`ParamArray`Bağımsız değişken otomatik olarak isteğe bağlıdır ve yordam bildiriminde tek bir isteğe bağlı bağımsız değişken olmalıdır. Önceki bağımsız değişkenlerin tümü gerekli olmalıdır.  
  
 **Hata kimliği:** BC30046  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimdeki isteğe bağlı bağımsız değişkenleri kaldırın.
