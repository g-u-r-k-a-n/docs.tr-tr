# <a name="compiler-error-cs1641"></a><span data-ttu-id="73b50-103">Derleyici hatası CS1641</span><span class="sxs-lookup"><span data-stu-id="73b50-103">Compiler Error CS1641</span></span>
<span data-ttu-id="73b50-104">Sabit boyutlu bir arabellek alanı, alan adından sonra dizi boyutu belirticisine sahip olmalıdır</span><span class="sxs-lookup"><span data-stu-id="73b50-104">A fixed size buffer field must have the array size specifier after the field name</span></span>  
  
 <span data-ttu-id="73b50-105">Normal dizilerin aksine, sabit boyutlu arabellekler, bildirim noktasında sabit bir boyut belirtilmesini gerektirir.</span><span class="sxs-lookup"><span data-stu-id="73b50-105">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="73b50-106">Bu hatayı çözmek için, pozitif bir tamsayı değişmez değeri veya sabit pozitif bir tamsayı ekleyin ve köşeli ayraçları tanımlayıcıdan sonra koyun.</span><span class="sxs-lookup"><span data-stu-id="73b50-106">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="73b50-107">Aşağıdaki örnek CS1641 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="73b50-107">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
