 <span data-ttu-id="681a2-117">Yöntem, <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> kaynak tablodaki bir satırda null ya da null değer türü ile karşılaştığında değeri ile değiştirir <xref:System.DBNull.Value> .</span><span class="sxs-lookup"><span data-stu-id="681a2-117">When the <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> method encounters either a null reference or nullable value type in a row in the source table, it replaces the value with <xref:System.DBNull.Value>.</span></span> <span data-ttu-id="681a2-118">Bu şekilde, null değerler döndürülen içinde doğru işlenir <xref:System.Data.DataTable> .</span><span class="sxs-lookup"><span data-stu-id="681a2-118">This way, null values are handled correctly in the returned <xref:System.Data.DataTable>.</span></span>
