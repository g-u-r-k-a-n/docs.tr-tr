# <a name="reference-required-to-module-modulename-containing-the-definition-for-event-eventname"></a>' ' \<modulename> Olayının tanımını içeren ' ' modülüne başvuru gerekiyor \<eventname>

' `modulename` <> ' olayının tanımını içeren ' <> ' modülüne başvuru gerekiyor `eventname` . Projenize bir tane ekleyin.  
  
 Olay, projenizde doğrudan başvurulmayan bir modülde tanımlanmıştır. Visual Basic derleyici, olayın birden fazla modülde tanımlanması durumunda belirsizlik olmaması için bir başvuru gerektirir.  
  
 **Hata kimliği:** BC30006  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Başvurulmayan modülün adını Proje başvurularında ekleyin.
