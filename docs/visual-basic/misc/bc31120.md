# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>' End RaiseEvent ' satırdaki ilk ifade olmalıdır
Bir `End RaiseEvent` ifade bir iki nokta üst üste (:) Ekstre ayırıcı. `End RaiseEvent`kaynak satırındaki tek bir ifade olmalıdır.  
  
 **Hata kimliği:** BC31120  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyimi farklı satırlara ayırın.
