# <a name="argument-cannot-be-nothing"></a>Bağımsız değişken Nothing olamaz

Bir değeri olması gereken bir bağımsız değişken için null değer sağlandı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nesne örneğini sağlamadan bir nesneyi kullanmayı denemiş olabilirsiniz. Böyle bir durumda `New` örneği oluşturmak için anahtar sözcüğünü kullanın.  
  
- Değerin doğru hesaplanıp hesaplanmadığını denetleyin.
