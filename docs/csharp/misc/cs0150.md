# <a name="compiler-error-cs0150"></a><span data-ttu-id="1a0cd-102">Derleyici Hatası CS0150</span><span class="sxs-lookup"><span data-stu-id="1a0cd-102">Compiler Error CS0150</span></span>
<span data-ttu-id="1a0cd-103">Sabit bir değer bekleniyor</span><span class="sxs-lookup"><span data-stu-id="1a0cd-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="1a0cd-104">Sabitin beklendiği yerde bir değişken bulundu.</span><span class="sxs-lookup"><span data-stu-id="1a0cd-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="1a0cd-105">Daha fazla bilgi için [bkz.](../language-reference/keywords/switch.md)</span><span class="sxs-lookup"><span data-stu-id="1a0cd-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="1a0cd-106">Aşağıdaki örnek CS0150 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1a0cd-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="1a0cd-107">Bu hata, bir dizi boyutu değişken değeri ile belirtilir ve bir dizi baş harfi ile başharfe de üretilir.</span><span class="sxs-lookup"><span data-stu-id="1a0cd-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="1a0cd-108">Hatayı kaldırmak için, diziyi ayrı bir deyim veya deyimde başlatma.</span><span class="sxs-lookup"><span data-stu-id="1a0cd-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
