## <a name="example"></a><span data-ttu-id="70933-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="70933-107">Example</span></span>  
 <span data-ttu-id="70933-108">Aşağıdaki örnek CS1642 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="70933-108">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
