# <a name="project-projectname-cannot-generate-a-reference-to-file-filename"></a>' \<projectname> ' Projesi ' ' dosyasına bir başvuru üretemiyor \<filename>

' <`projectname`> ' projesi ' <> ' dosyasına başvuru oluşturamıyor `filename` . Bu başvuruyu silip yeniden eklemeniz ve sonra çözümün tam yeniden oluşturulmasını yapmanız gerekebilir.  
  
 Bir dosyaya başvuru oluşturulurken hata oluştu.  
  
 **Hata kimliği:** BC31514  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. **Çözüm Gezgini** **Proje başvurularından** öğeye sağ tıklayıp **Kaldır**' a tıklayarak başvuruyu kaldırın.  
  
2. **Çözüm Gezgini** **Başvurular** ' a sağ tıklayıp **Başvuru Ekle**' ye tıklayarak başvuruyu geri yükleyin.
