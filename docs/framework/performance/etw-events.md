# <a name="etw-events-in-the-net-framework"></a>.NET Framework'te ETW Olayları
Windows için olay izleme (ETW), Windows işletim sistemleri tarafından sağlanan yüksek performanslı, düşük ek inişme, ölçeklenebilir bir izleme sistemidir. .NET Framework tarafından sağlanan profil oluşturma ve hata ayıklama desteğini tamamlar ve çeşitli senaryoları gidermek için kullanılabilir.  
  
 .NET Framework'de, ETW olay izleme ortak dil çalışma zamanı (CLR), [Görev Paralel Kitaplığı](../../standard/parallel-programming/task-parallel-library-tpl.md)ve [Paralel LINQ (PLINQ)](../../standard/parallel-programming/introduction-to-plinq.md)için kullanılabilir.

 Paralel uygulama kodunun nasıl profilinin nasıl açıklanır.  
  
 [Ortak Dil Çalışma Zamanında ETW Olayları](etw-events-in-the-common-language-runtime.md)  
 CLR ETW olaylarının ortak dil çalışma zamanı tarafından sağlanan profil oluşturma ve hata ayıklama desteğini nasıl tamamlandığını açıklar.
