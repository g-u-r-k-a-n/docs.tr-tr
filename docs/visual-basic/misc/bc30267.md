# <a name="declaration1-cannot-override-declaration2-because-it-is-declared-notoverridable"></a>' ' ' \<declaration1> \<declaration2> NotOverridable ' olarak bildirildiği için ' ', ' ' öğesini geçersiz kılamaz

Yordam veya özellik bildirimi aynı ada sahip devralınmış bir öğeyi geçersiz kılmaya çalışır, ancak devralınmış öğe olarak belirtilir `NotOverridable` .  
  
 **Hata kimliği:** BC30267  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `NotOverridable`Devralınan öğenin bildiriminden anahtar sözcüğünü kaldırın veya geçersiz kılma bildirimini kaldırın.
