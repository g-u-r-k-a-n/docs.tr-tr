# <a name="compiler-error-cs0517"></a>Derleyici hatası CS0517
' class ' temel sınıfa sahip değil ve temel Oluşturucu çağıramaz  
  
 CS0517 yalnızca .NET çalışma zamanı, temel sınıfı olmayan tek sınıf olan nesne sınıfının kaynak kodunu derlediğinde gerçekleşebilir.
