# <a name="compiler-error-cs1906"></a>Derleyici hatası CS1906
' Option ' seçeneği geçersiz; Kaynak görünürlüğü ' Public ' veya ' Private ' olmalıdır  
  
 Bu hata geçersiz bir [/Resource (kaynak dosyayı çıkışa ekle)](../language-reference/compiler-options/resource-compiler-option.md) veya [-linkresource (.NET Framework kaynağa bağlantı)](../language-reference/compiler-options/linkresource-compiler-option.md) komut satırı seçeneğini gösterir. **/Resource** veya **/linkresource** komut satırı seçeneğinin söz dizimini denetleyin ve kullanılan erişilebilirlik değiştiricisinin **ortak** veya `private` olduğundan emin olun.
