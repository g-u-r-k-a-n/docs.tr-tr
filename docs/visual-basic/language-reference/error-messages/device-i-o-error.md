# <a name="device-io-error"></a>Cihaz G/Ç hatası

Programınız yazıcı veya disk sürücüsü gibi bir cihaz kullanırken bir giriş veya çıkış hatası oluştu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Cihazın düzgün şekilde çalıştığından emin olun ve işlemi yeniden deneyin.
