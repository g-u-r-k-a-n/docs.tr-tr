# <a name="bc2012-cant-open-filename-for-writing"></a>BC2012: ' \<filename> ' yazma için açılamıyor

Belirtilen dosya, belki de zaten açık olduğundan yazmak için açılamıyor.

 **Hata kimliği:** BC2012

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Dosyayı kapatın ve yeniden açın.

2. Dosyanın izinlerini denetleyin.
