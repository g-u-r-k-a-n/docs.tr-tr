# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Bir Visual Basic çözümündeki tüm projeler aynı platformu hedeflemelidir, ancak eklemeye çalıştığınız proje, çözümünüz tarafından belirtilen bir platformu hedefliyor

Bir Visual Basic çözümündeki tüm projeler aynı platformu hedeflemelidir, ancak eklediğiniz proje, çözümünüz tarafından belirtilen bir platformu hedefler.  
  
 **Hata kimliği:** BC30765  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yalnızca çözümünüz tarafından belirtilen platformu hedefleyen projeleri ekleyin.
