# <a name="hosting-structures"></a>Barındırma Yapıları
Bu bölümde, barındırma API 'sinin kullandığı yönetilmeyen yapılar açıklanmaktadır.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [AssemblyBindInfo Yapısı](assemblybindinfo-structure.md)  
 Başvurulan derleme hakkında ayrıntılı bilgi sağlar.  
  
 [BucketParameters Yapısı](bucketparameters-structure.md)  
 Bir olayın tür adını ve olayla ilişkili geçerli özel durumun parametrelerini depolar.  
  
 [COR_GC_STATS Yapısı](cor-gc-stats-structure.md)  
 Ortak dil çalışma zamanının (CLR) çöp toplama mekanizması hakkında istatistikler sağlar.  
  
 [COR_GC_THREAD_STATS Yapısı](cor-gc-thread-stats-structure.md)  
 Çöp toplamadan ilgili iş parçacığı başına istatistikleri içerir.  
  
 [CustomDumpItem Yapısı](customdumpitem-structure.md)  
 Hata raporlamada özel bir döküme eklenecek bir öğe tanımlar.  
  
 [MDAInfo Yapısı](mdainfo-structure.md)  
 `Event_MDAFired`Yönetilen hata ayıklama Yardımcısı (MDA) oluşturulmasını tetikleyen olayla ilgili ayrıntıları sağlar.  
  
 [ModuleBindInfo Yapısı](modulebindinfo-structure.md)  
 Başvurulan modül ve onu içeren derleme hakkında ayrıntılı bilgi sağlar.  
  
 [StackOverflowInfo Yapısı](stackoverflowinfo-structure.md)  
 Oluşan taşma türünü ve taşma nedeniyle oluşan özel durum hakkında bilgi depolar.
