# <a name="array-initializer-has-too-few-dimensions"></a>Dizi başlatıcısı çok az boyuta sahip

Diziniz için başlatıcıda yeterli boyut sağlamadınız.  
  
 **Hata kimliği:** BC30565  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaç boyutun gerekli olduğunu öğrenmek için dizi başlatıcısını denetleyin.  
  
- Eksik bağımsız değişkeni veya bağımsız değişkenleri sağlayın.
