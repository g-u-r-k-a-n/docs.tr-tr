# <a name="modifier-is-not-valid-on-a-local-variable-declaration"></a>' \<modifier> ' yerel değişken bildiriminde geçerli değil

`Dim`Yordam içindeki bir ifade, gibi geçersiz bir anahtar sözcük içeriyor `Overloads` .  
  
 **Hata kimliği:** BC30247  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Deyimden geçersiz anahtar sözcüğü Kaldır `Dim` .
