# <a name="required-attribute-version-missing-from-xml-declaration"></a>Gerekli ' version ' özniteliği XML bildiriminde yok
`version`XML belgesi değişmez değeri için öznitelik eksik.  
  
 **Hata kimliği:** BC31153  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `version`Aşağıdaki örnekteki gibi, XML belgesi bildirimine özniteliği ekleyin:`<?xml version="1.0"?>`
