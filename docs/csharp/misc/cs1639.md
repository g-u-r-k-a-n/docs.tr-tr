# <a name="compiler-error-cs1639"></a>Derleyici hatası CS1639

' Interface ' arabiriminin ' Signature ' yönetilen coclass sarmalayıcı sınıfı imzası geçerli bir sınıf adı imzası değil  
  
 Arabirimin meta verileri geçersizse bu hata oluşur. Bu, arabirim C# veya desteklenen bir .NET dili kullanılarak oluşturulduysa asla gerçekleşmemelidir. Arabirimin satıcısına başvurun veya arabirim derlemesinin başarıyla oluşturulduğunu doğrulayın.
