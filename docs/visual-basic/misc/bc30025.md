# <a name="property-missing-end-property"></a>Özellikte ' End Property ' eksik

Bir `End` deyimin, sonlandırmakta olduğu blokla eşleşmesi gerekir. Bir `Property` tanım düzgün sonlandırılmadı.  
  
 **Hata kimliği:** BC30025  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `End Property`Özellik tanımının sonuna bir ifade ekleyin.
