# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>\<type1> \<type2> Dizi türlerinde farklı sayıda boyut olduğundan ' ' türünün değeri ' ' olarak dönüştürülemez
Farklı sayıda boyuta sahip bir diziyi diğerine dönüştürmeye çalıştınız.  
  
 **Hata kimliği:** BC30414  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Boyut sayısının eşleşmesi için bir veya her iki diziyi de değiştirin.
