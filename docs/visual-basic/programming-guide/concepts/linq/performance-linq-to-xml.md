# <a name="performance-linq-to-xml-visual-basic"></a>Performans (LINQ to XML) (Visual Basic)
Bu bölümde, özellikle işlevsel oluşturma ve sorguların performansı gibi [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)]performans hakkında bilgi sağlanır.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Zincirleme sorgularının performansı (LINQ to XML) (Visual Basic)](../../../../visual-basic/programming-guide/concepts/linq/performance-of-chained-queries-linq-to-xml.md)  
 Zincirleme [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] sorguları hakkında performans bilgileri sağlar.  
  
 [Atomlanmış XName ve XNamespace nesneleri (LINQ to XML) (Visual Basic)](../../../../visual-basic/programming-guide/concepts/linq/atomized-xname-and-xnamespace-objects-linq-to-xml.md)

 [XName nesnelerinin (LINQ to XML) ön atomda (Visual Basic)](../../../../visual-basic/programming-guide/concepts/linq/pre-atomization-of-xname-objects-linq-to-xml.md)
