# <a name="extension-method-methodname-defined-in-modulename-is-not-generic-or-has-no-free-type-parameters-and-so-cannot-have-type-arguments"></a><span data-ttu-id="41947-103">\<methodname>' ' İçinde tanımlanan ' ' genişletme metodu \<modulename> genel değil (veya hiç boş tür parametresi yok) ve bu nedenle tür bağımsız değişkenleri olamaz</span><span class="sxs-lookup"><span data-stu-id="41947-103">Extension method '\<methodname>' defined in '\<modulename>' is not generic (or has no free type parameters) and so cannot have type arguments</span></span>

<span data-ttu-id="41947-104">Bir tür bağımsız değişkeni, genel parametreleri olmayan ya da bir türün zaten belirtilmediği genel parametreleri olmayan bir genişletme yöntemine yapılan çağrıda belirtildi.</span><span class="sxs-lookup"><span data-stu-id="41947-104">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="41947-105">Örneğin, aşağıdaki kod bu hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="41947-105">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
