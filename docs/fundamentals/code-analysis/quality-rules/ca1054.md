## <a name="cause"></a><span data-ttu-id="9474c-111">Nedeni</span><span class="sxs-lookup"><span data-stu-id="9474c-111">Cause</span></span>

<span data-ttu-id="9474c-112">Bir tür, adı "Uri", "Uri", "urn", "urn", "URL" veya "URL" içeren bir dize parametresine sahip bir yöntem bildirir ve tür, bir parametreyi alan karşılık gelen bir aşırı yükleme bildirmiyor <xref:System.Uri?displayProperty=fullName> .</span><span class="sxs-lookup"><span data-stu-id="9474c-112">A type declares a method with a string parameter whose name contains "uri", "Uri", "urn", "Urn", "url", or "Url" and the type does not declare a corresponding overload that takes a <xref:System.Uri?displayProperty=fullName> parameter.</span></span>
