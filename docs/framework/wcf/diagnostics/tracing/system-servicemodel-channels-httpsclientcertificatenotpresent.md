# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent

İstemci sertifikası gereklidir. İstekte hiçbir sertifika bulunamadı.  
  
## <a name="description"></a>Description  

 Bu izleme, HTTPS dinleyicisinin bir istemci sertifikası ile ilişkilendirilmemiş bir HTTPS isteği aldığını gösterir. Dinleyici, tüm HTTPS isteklerinde istemci sertifikaları gerektirecek şekilde yapılandırıldığından, dinleyici istemcinin kimliğini doğrulayamadı.
