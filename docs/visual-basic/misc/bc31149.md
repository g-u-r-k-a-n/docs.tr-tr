# <a name="duplicate-xml-attribute-attributename"></a>Yinelenen XML özniteliği ' attributeName '

XML öğesi değişmez değeri için belirtilen öznitelik, daha önce XML öğesi değişmez değeri için tanımlanmış bir özniteliğin eşleşir. XML öğesi değişmez değerlerinde öznitelik adları benzersiz olmalıdır.  
  
 **Hata kimliği:** BC31149  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özniteliği, XML öğesi değişmez değeri için benzersiz olan bir adla yeniden adlandırın.
