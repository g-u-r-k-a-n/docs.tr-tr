# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="312c8-103">' ' \<derivedtypename> , ' ' \<type> türünün erişimini ' ' \<constructedbasetypename> olarak genişlettiğinden \<internaltypename> ' \<region> \<regionname> ' öğesinden ' ' öğesinden devralma yapılamıyor</span><span class="sxs-lookup"><span data-stu-id="312c8-103">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>

<span data-ttu-id="312c8-104">Türetilmiş bir sınıf veya arabirim, bir iç türün erişim düzeyini, bir temel sınıf veya arabirime tür bağımsız değişkeni olarak kullanarak genişletmeye çalışır.</span><span class="sxs-lookup"><span data-stu-id="312c8-104">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="312c8-105">Aşağıdaki kod bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="312c8-105">The following code can generate this error.</span></span>  
  
```vb  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="312c8-106">Dış kodun `containingClass` erişimine izin verilmiyor `internalStructure` .</span><span class="sxs-lookup"><span data-stu-id="312c8-106">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="312c8-107">Ancak, `derivedClass` aynı derlemedeki herhangi bir koddan erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="312c8-107">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="312c8-108">Bu nedenle, `derivedClass` `baseClass` `internalStructure` bir tür bağımsız değişkeni olarak kullanıyorsa, bu, `internalStructure` tanımlayıcı kod bölgesi genelinde ortaya çıkmasına neden olabileceğinden, devralma yapılamaz.</span><span class="sxs-lookup"><span data-stu-id="312c8-108">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="312c8-109">**Hata kimliği:** BC30921</span><span class="sxs-lookup"><span data-stu-id="312c8-109">**Error ID:** BC30921</span></span>

- <span data-ttu-id="312c8-113">Erişim düzeylerini ayarlayamadığından, temel sınıfı veya arabirimi oluştururken iç türü bir tür bağımsız değişkeni olarak kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="312c8-113">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
