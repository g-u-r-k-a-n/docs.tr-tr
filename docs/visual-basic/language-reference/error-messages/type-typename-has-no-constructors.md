# <a name="type-typename-has-no-constructors"></a>'\<typename>' türünün yapıcısı yok
Bir tür, çağrısını desteklemez `Sub New()` . Olası bir neden, bozuk bir derleyici veya ikili dosyadır.  
  
 **Hata kimliği:** BC30251  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Tür farklı bir projede veya başvurulan bir dosyada yer alıyorsa, projeyi veya dosyayı yeniden yükleyin.  
  
2. Tür aynı projem ise, türü içeren derlemeyi yeniden derleyin.  
  
3. Hata yinelenirse Visual Basic derleyicisini yeniden yükleyin.  
  
4. Hata devam ederse, koşullar hakkındaki bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
