# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>' ' Bağımsız değişkeni \<argumentname> sıfırdan büyük veya sıfıra eşit olmalı

Sıfırdan küçük olduğu için bağımsız değişken geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bağımsız değişkenlerin yazımını denetleyin. Yanlış yazılmış bir değişken adı, sıfır olarak başlatılan bir sayısal değişkeni örtülü olarak oluşturabilir.  
  
2. Deyimdeki değişkenlerde, özellikle de diğer yordamlardan bağımsız değişken olarak geçirilen işlemler için önceki işlemleri denetleyin.
