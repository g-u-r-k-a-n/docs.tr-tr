# <a name="cannot-find-net-framework-directory-error"></a>.NET Framework dizin bulunamıyor: \<error>

.NET Framework derlemelerinin depolandığı dizin bulunamadı.  
  
 **Hata kimliği:** BC31508  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Hatanın yinelenip yinelenmeyeceğini görmek için programı yeniden derleyin.  
  
2. Hata devam ederse, çalışmanızı kaydedin ve Visual Studio 'Yu yeniden başlatın.  
  
3. Hata yinelenirse Visual Basic yeniden yükleyin.  
  
4. Yeniden yükleme sonrasında hata devam ederse, Microsoft Ürün Destek Hizmetleri 'ne bildirin.
