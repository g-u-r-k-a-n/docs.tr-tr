<span data-ttu-id="bf2e3-104">Windows Communication Foundation (WCF), bir uygulamanın yazılımının diğer yazılımlarla nasıl iletişim kurduğuna nasıl yazıldığını ayırır.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-104">Windows Communication Foundation (WCF) separates how the software for an application is written from how it communicates with other software.</span></span> <span data-ttu-id="bf2e3-105">Bağlamalar, istemcilerin ve hizmetlerin birbirleriyle iletişim kurması için gereken aktarım, kodlama ve protokol ayrıntılarını belirtmek için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-105">Bindings are used to specify the transport, encoding, and protocol details required for clients and services to communicate with each other.</span></span> <span data-ttu-id="bf2e3-106">WCF, uç noktanın temel alınan tel temsilini oluşturmak için bağlamaları kullanır, bu nedenle bağlama ayrıntılarının çoğu, iletişim kuran taraflar tarafından üzerinde anlaşılmalıdır.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-106">WCF uses bindings to generate the underlying wire representation of the endpoint, so most of the binding details must be agreed upon by the parties that are communicating.</span></span> <span data-ttu-id="bf2e3-107">Bunu başarmanın en kolay yolu, bir hizmetin istemcilerinin hizmet için uç noktanın kullandığı bağlamayı kullanması içindir.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-107">The easiest way to achieve this is for clients of a service to use the same binding that the endpoint for the service uses.</span></span> <span data-ttu-id="bf2e3-108">Bunun nasıl yapılacağı hakkında daha fazla bilgi için bkz.  <span data-ttu-id="bf2e3-109">Bağlama, bağlama öğeleri koleksiyonundan oluşur.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-109">A binding is made up of a collection of binding elements.</span></span> <span data-ttu-id="bf2e3-110">Her öğe, uç noktanın istemcilerle nasıl iletişim kuracağını açıklar.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-110">Each element describes some aspect of how the endpoint communicates with clients.</span></span> <span data-ttu-id="bf2e3-111">Bağlama en az bir tane aktarım bağlama öğesi, en az bir ileti kodlama bağlama öğesi (taşıma bağlama öğesinin varsayılan olarak sağlayabileceği) ve diğer birçok protokol bağlama öğesini içermelidir.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-111">A binding must include at least one transport binding element, at least one message-encoding binding element (which the transport binding element can provide by default), and any number of other protocol binding elements.</span></span> <span data-ttu-id="bf2e3-112">Bu açıklamanın dışında bir çalışma zamanı oluşturan işlem, her bağlama öğesinin bu çalışma zamanına kod katkıda bulunmasını sağlar.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-112">The process that builds a runtime out of this description allows each binding element to contribute code to that runtime.</span></span>

[System-Provided bağlamalarını yapılandırma](configuring-system-provided-bindings.md).</span><span class="sxs-lookup"><span data-stu-id="bf2e3-117">For details, see [Configuring System-Provided Bindings](configuring-system-provided-bindings.md).</span></span>  
  
 <span data-ttu-id="bf2e3-118">Bu sistem tarafından belirtilen bağlamalardan biri tarafından sağlanmayan bağlama öğeleri koleksiyonuna ihtiyacınız varsa, gereken bağlama öğeleri koleksiyonundan oluşan özel bir bağlama oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-118">If you need a collection of binding elements not provided by one of these system-provided bindings, you can create a custom binding that consists of the collection of binding elements required.</span></span> <span data-ttu-id="bf2e3-119">Bu özel bağlamaların oluşturulması ve yeni bir sınıf gerektirilmesi kolaydır, ancak bağlama öğelerini veya ayarlarını denetlemek için özellikler sağlamaz.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-119">These custom bindings are easy to create and do not require a new class, but they do not provide properties for controlling the binding elements or their settings.</span></span> <span data-ttu-id="bf2e3-120">Bağlama öğelerine erişebilir ve bunları içeren koleksiyon aracılığıyla ayarlarını değiştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="bf2e3-120">You can access the binding elements and modify their settings through the collection that contains them.</span></span> <span data-ttu-id="bf2e3-121">Ayrıntılar için bkz.
