# <a name="too-many-dll-application-clients"></a>Çok fazla sayıda DLL uygulama istemcisi
Visual Basic için dinamik bağlantı kitaplığı (DLL) yalnızca sınırlı sayıda ana bilgisayar uygulaması tarafından erişim sağlayabilir. Uygulamanız ve Visual Basic ana bilgisayar olan diğer uygulamalar (bazıları uygulamanız tarafından erişilebilir), Visual Basic DLL 'sine aynı anda erişmeyi deniyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Visual Basic erişen açık uygulama sayısını azaltın.
