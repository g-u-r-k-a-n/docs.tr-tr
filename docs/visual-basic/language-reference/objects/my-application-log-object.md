## <a name="remarks"></a>Açıklamalar  
 Nesnesinin yöntemleri ve özellikleri hakkında daha fazla bilgi için `My.Application.Log` bkz <xref:Microsoft.VisualBasic.Logging.Log> ..  
  
 Daha fazla bilgi için bkz. [uygulamadan bilgileri günlüğe kaydetme](../../developing-apps/programming/log-info/index.md).
