# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="9c8ec-102">Derleyici Uyarısı (düzey 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="9c8ec-102">Compiler Warning (level 2) CS0469</span></span>
<span data-ttu-id="9c8ec-103">'Goto case' değeri örtülü olarak 'tür' türüne dönüştürülemez</span><span class="sxs-lookup"><span data-stu-id="9c8ec-103">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="9c8ec-104">`goto case`Kullandığınızda, goto servis talebinin değerinden anahtarın türüne kadar örtülü bir dönüştürme olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="9c8ec-104">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9c8ec-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="9c8ec-105">Example</span></span>  
 <span data-ttu-id="9c8ec-106">Aşağıdaki örnek CS0469 üretir.</span><span class="sxs-lookup"><span data-stu-id="9c8ec-106">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
