<span data-ttu-id="45c6e-104">Geçersiz kılma ve açık arabirim uygulama yöntemlerinin kısıtlamaları temel yöntemden devralınır ve bu nedenle doğrudan belirtilemez</span><span class="sxs-lookup"><span data-stu-id="45c6e-104">Constraints for override and explicit interface implementation methods are inherited from the base method, so they cannot be specified directly</span></span>  
  
 <span data-ttu-id="45c6e-105">Türetilmiş bir sınıfın parçası olan genel bir yöntem, temel sınıftaki bir yöntemi geçersiz kıldığında, geçersiz kılınan yöntemde kısıtlama belirtemezsiniz.</span><span class="sxs-lookup"><span data-stu-id="45c6e-105">When a generic method that is part of a derived class overrides a method in the base class, you may not specify constraints on the overridden method.</span></span> <span data-ttu-id="45c6e-106">Türetilmiş sınıftaki geçersiz kılma yöntemi, temel sınıftaki yönteminden kısıtlamalarını devralır.</span><span class="sxs-lookup"><span data-stu-id="45c6e-106">The override method in the derived class inherits its constraints from the method in the base class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="45c6e-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="45c6e-107">Example</span></span>  

 <span data-ttu-id="45c6e-108">Aşağıdaki örnek CS0460 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="45c6e-108">The following sample generates CS0460.</span></span>  
  
```csharp  
// CS0460.cs  
// compile with: /target:library  
class BaseClass
{  
   BaseClass() { }  
}  
  
interface I  
{  
   void F1<T>() where T : BaseClass;  
   void F2<T>() where T : struct;  
   void F3<T>() where T : BaseClass;  
}  
  
class ExpImpl : I  
{  
   void I.F1<T>() where T : BaseClass {}   // CS0460  
   void I.F2<T>() where T : class {}  // CS0460  
}  
```
