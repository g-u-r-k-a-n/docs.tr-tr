# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>Tanımlayıcılar yalnızca bir bildirimin başlangıcında geçerlidir
Erişilebilirlik ve diğer göstergeleri belirten anahtar sözcükler, veya gibi bildirim anahtar sözcüğünden önce gelmelidir `Dim` `Function` .  
  
 **Hata kimliği:** BC30181  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öznitelik listesi (varsa) ve bildirim anahtar sözcüğü arasındaki tüm belirticileri taşıyın.
