# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-list"></a>Bu bağımsız değişkenlerle hiçbir erişilebilir aşırı yüklü ' \<methodname> ' çağrılamaz:\<list>
Aşırı yüklenmiş bir yöntem çağrılır, ancak belirtilen bağımsız değişkenlerin listesiyle hiçbir yöntem eşleştirilenebilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken listesini, çağırmayı amaçladığınız aşırı yüklemeye göre eşleştirin.  
  
- Bağımsız değişken listesini, imza aşırı yüklemeden eşleşecek şekilde ayarlayın.
