verimli şekilde kullanılması için eşzamanlılık derecesini dinamik olarak ölçeklendirir.</span><span class="sxs-lookup"><span data-stu-id="63afa-105">The TPL scales the degree of concurrency çıkarabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="63afa-107">By using TPL, you can maximize the performance of your code while focusing on the work that your program is designed to
