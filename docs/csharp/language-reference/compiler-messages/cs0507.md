# <a name="compiler-error-cs0507"></a><span data-ttu-id="b440b-103">Derleyici Hatası CS0507</span><span class="sxs-lookup"><span data-stu-id="b440b-103">Compiler Error CS0507</span></span>

<span data-ttu-id="b440b-104">' işlev1 ': ' Access ' devralınan üyesi ' function2 ' geçersiz kılınırken erişim değiştiricileri değiştirilemez</span><span class="sxs-lookup"><span data-stu-id="b440b-104">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="b440b-105">Yöntem geçersiz kılma içindeki erişim belirtimini değiştirme girişiminde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="b440b-105">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b440b-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="b440b-106">Example</span></span>  

 <span data-ttu-id="b440b-107">Aşağıdaki örnek CS0507 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="b440b-107">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="b440b-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="b440b-108">Example</span></span>  

 <span data-ttu-id="b440b-109">Ayrıca, bir sınıf `protected internal` başvurulan meta verilerde tanımlanmış olarak işaretlenen bir yöntemi geçersiz kılmayı denerse CS0507 de oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="b440b-109">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="b440b-110">Bu durumda, geçersiz kılma yöntemi olarak işaretlenmelidir `protected` .</span><span class="sxs-lookup"><span data-stu-id="b440b-110">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="b440b-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="b440b-111">Example</span></span>  

 <span data-ttu-id="b440b-112">Aşağıdaki örnek CS0507 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="b440b-112">The following sample generates CS0507.</span></span>
