# <a name="vbstrconvwide-and-vbstrconvnarrow-are-not-applicable-to-the-locale-specified"></a>VbStrConv. Wide ve VbStrConv. dar belirtilen yerel ayara uygulanamaz

Uygulama, `VbStrConv` sabit listesi üyelerini kullanmaya çalışıyor `Wide` veya `Narrow` belirtilen yerel ayar için geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Ya da ' i kaldırın `VbStrConv.Wide` `VbStrConv.Narrow` .
