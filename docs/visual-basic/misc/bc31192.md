# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>Win32 bildirimi eklenirken hata oluştu: bildirim dosyası \<> dosyası bulunamıyor
`/win32manifest` derleyici seçeneği için belirtilen dosya bulunamıyor.  
  
 **Hata kimliği:** BC31192  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `/win32manifest` derleyici seçeneği için geçerli bir dosya adı ve yol girin.
