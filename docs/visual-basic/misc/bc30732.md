# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Web metotları hata ayıklama pencerelerinde değerlendirilemiyor

XML Web hizmeti metotları hata ayıklama penceresi içinde test edilemez.  
  
 **Hata kimliği:** BC30732  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kod düzenleyicisinden XML Web hizmeti yöntemlerini test edin.
