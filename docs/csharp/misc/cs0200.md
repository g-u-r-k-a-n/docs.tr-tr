<span data-ttu-id="bd0e5-104">Bir [mülke](../programming-guide/classes-and-structs/using-properties.md)değer atama girişiminde bulunuldu, ancak özellik temalı bir erişime sahip değil veya atama oluşturucudışındaydı.</span><span class="sxs-lookup"><span data-stu-id="bd0e5-104">An attempt was made to assign a value to a [property](../programming-guide/classes-and-structs/using-properties.md), but the property does not have a set accessor or the assignment was outside of the constructor.</span></span> <span data-ttu-id="bd0e5-105">Ayarlanmış bir erişimci ekleyerek hatayı giderin.</span><span class="sxs-lookup"><span data-stu-id="bd0e5-105">Resolve the error by adding a set accessor.</span></span> <span data-ttu-id="bd0e5-106">Daha fazla bilgi için okuma [yazma özelliklerini nasıl beyan edin ve kullanacağınızı](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)öğrenin.</span><span class="sxs-lookup"><span data-stu-id="bd0e5-106">For more information, see [How to declare and use read write properties](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span></span>

## <a name="examples"></a><span data-ttu-id="bd0e5-107">Örnekler</span><span class="sxs-lookup"><span data-stu-id="bd0e5-107">Examples</span></span>
<span data-ttu-id="bd0e5-108">Aşağıdaki örnek CS0200 üretir:</span><span class="sxs-lookup"><span data-stu-id="bd0e5-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="bd0e5-109">Aşağıdaki örnek [otomatik olarak uygulanan özellikleri](../programming-guide/classes-and-structs/auto-implemented-properties.md)kullanır , [nesne başlatılması](../programming-guide/classes-and-structs/object-and-collection-initializers.md), ve hala CS0200 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bd0e5-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="bd0e5-110">Yalnızca okunan bir özellik veya dizinleyici 'özellik'e atama, ayarlanmış bir erişimci ekleyerek veya nesnenin oluşturucusundaki özelliğe atayarak elde edilebilir.</span><span class="sxs-lookup"><span data-stu-id="bd0e5-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
