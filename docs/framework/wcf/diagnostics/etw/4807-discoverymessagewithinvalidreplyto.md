# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 - DiscoveryMessageWithInvalidReplyTo

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|4807|  
|Anahtar sözcükler|Bulma|  
|Level|Uyarı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Description  

 Bu olay, bulma iletisi geçersiz bir ReplyTo adresine sahip olduğu için bırakıldığında yayınlanır.  
  
## <a name="message"></a>İleti  

 MessageID = ' %1 ' olan bir bulma isteği iletisi, geçersiz bir ReplyTo adresine sahip olduğundan bırakıldı.
