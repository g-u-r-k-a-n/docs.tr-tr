# <a name="resource-name-resourcename-cannot-be-used-more-than-once"></a>' @No__t_0resourcename > ' kaynak adı birden çok kez kullanılamaz
Dinamik bağlantılı bir kitaplıkta veya yürütülebilir dosyada belirtilen bir kaynağın adı birden çok kez kullanıldı.  
  
 **Hata kimliği:** BC31502  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaynakları benzersiz şekilde adlandırın.
