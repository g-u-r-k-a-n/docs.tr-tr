# <a name="overload-resolution-failed-because-no-accessible-method-accepts-this-number-of-arguments"></a>Erişilebilir ' \<method> ' Bu sayıda bağımsız değişken kabul etmediğinden aşırı yükleme çözümlemesi başarısız oldu

Aşırı yüklenmiş bir yönteme çağrı yaptınız, ancak derleyici eşleşen bir bağımsız değişken listesi olan bir yöntem bulamıyor.  
  
 **Hata kimliği:** BC30516  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sağladığınız bağımsız değişken listesini denetleyin.
