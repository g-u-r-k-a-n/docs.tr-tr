# <a name="end-set-must-be-the-first-statement-on-a-line"></a>' End Set ' satırdaki ilk ifade olmalıdır
Bir `End Set` ifade bir iki nokta üst üste (:) Ekstre ayırıcı. `End Set`kaynak satırındaki tek bir ifade olmalıdır.  
  
 **Hata kimliği:** BC32034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyimi farklı satırlara ayırın.
