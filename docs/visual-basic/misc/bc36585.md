# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>' \<elementname> ', ' \<methodname> ' içinde tanımlanan ' ' Genişletme yönteminin parametresi değil \<typename>
Yordam çağrısı, genişletme yöntemi tarafından tanımlanmayan ek bir bağımsız değişken sağlar.  
  
 **Hata kimliği:** BC36585  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken listesinden gereksiz bağımsız değişkeni kaldırın.
