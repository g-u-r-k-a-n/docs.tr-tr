# <a name="interface-interfacename-can-be-implemented-only-once-by-this-type"></a>' \<interfacename> ' Arabirimi bu tür tarafından yalnızca bir kez uygulanabilir

Belirtilen arabirim bu tür tarafından zaten uygulanmış ve yeniden uygulanamaz.  
  
 **Hata kimliği:** BC31033  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu arabirimi bu türü yalnızca bir kez kullanarak uygulayın.
