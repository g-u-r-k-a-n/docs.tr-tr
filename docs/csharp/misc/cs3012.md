# <a name="compiler-warning-level-1-cs3012"></a>Derleyici Uyarısı (düzey 1) CS3012

Derlemedeki CLSCompliant özniteliğinden farklı olan bir modülde CLSCompliant özniteliğini belirtemezsiniz

Bir modülün ile ortak dil belirtimi (CLS) ile uyumlu olabilmesi için `[module:System.CLSCompliant(true)]` , [-target: Module](../language-reference/compiler-options/target-module-compiler-option.md) derleyici seçeneğiyle oluşturulmalıdır. CLS hakkında daha fazla bilgi için bkz. [Dil bağımsızlığı ve Language-Independent bileşenleri](../../standard/language-independence-and-language-independent-components.md).

## <a name="example"></a>Örnek

Aşağıdaki örnek, olmadan inşa edildiğinde `/target:module` , CS3012 oluşturur:

```csharp
// CS3012.cs
// compile with: /W:1

[module:System.CLSCompliant(true)]   // CS3012
public class C
{
    public static void Main()
    {
    }
}
```
