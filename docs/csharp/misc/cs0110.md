# <a name="compiler-error-cs0110"></a><span data-ttu-id="e9e96-103">Derleyici hatası CS0110</span><span class="sxs-lookup"><span data-stu-id="e9e96-103">Compiler Error CS0110</span></span>
<span data-ttu-id="e9e96-104">' Const declaration ' için sabit değerin değerlendirilmesi döngüsel bir tanım içeriyor</span><span class="sxs-lookup"><span data-stu-id="e9e96-104">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="e9e96-105">Bir [const](../language-reference/keywords/const.md) değişkeninin bildirimi (), `a` `b` aynı zamanda () başvurduğu başka bir const değişkenine () başvuramaz `a` .</span><span class="sxs-lookup"><span data-stu-id="e9e96-105">The declaration of a [const](../language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="e9e96-106">Aşağıdaki örnek CS0110 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e9e96-106">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
