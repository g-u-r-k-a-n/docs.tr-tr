# <a name="compiler-warning-level-1-cs0626"></a><span data-ttu-id="7dd31-103">Derleyici Uyarısı (düzey 1) CS0626</span><span class="sxs-lookup"><span data-stu-id="7dd31-103">Compiler Warning (level 1) CS0626</span></span>

<span data-ttu-id="7dd31-104">Yöntem, işleç veya erişimci ' Method ' dış olarak işaretlenmiş ve üzerinde hiç özniteliği yok.</span><span class="sxs-lookup"><span data-stu-id="7dd31-104">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="7dd31-105">Dış uygulamayı belirtmek için DllImport özniteliği eklemeyi düşünün.</span><span class="sxs-lookup"><span data-stu-id="7dd31-105">Consider adding a DllImport attribute to specify the external implementation.</span></span>
  
 <span data-ttu-id="7dd31-106">İşaretlenmiş bir yöntem `extern` aynı zamanda bir özniteliğiyle işaretlenmelidir, örneğin [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) özniteliği.</span><span class="sxs-lookup"><span data-stu-id="7dd31-106">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>

 <span data-ttu-id="7dd31-109">Aşağıdaki örnek CS0626 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7dd31-109">The following sample generates CS0626:</span></span>  
  
```csharp

public class MyClass  
{  
   static extern public void M(); // CS0626  
   // try the following line  
   // [DllImport("mydll.dll")] static extern public void M();

   {  
   }  
}  
```
