# <a name="compiler-error-cs1033"></a>Derleyici hatası CS1033

Kaynak dosya PDB 'de 16.707.565 satır gösterilebilir tablo sınırını aştı; hata ayıklama bilgileri hatalı olacak

Kaynak kodu dosyası, derleyicinin işleyey, izin verilen en fazla satır sayısını aştı. Bu hatayı çözmek için özgün dosyadan iki veya daha fazla kaynak kodu dosyası oluşturun. Maksimum satır sayısı 268.435.454 satırlık bir sayıdır. **/Debug**kullanıyorsanız, 16.707.566 ' den fazla kullanım, hata ayıklama bilgilerinin bozulmasına neden olur.

> [!NOTE]
> Bu derleyici hatası artık Roslyn içinde kullanılmıyor.
