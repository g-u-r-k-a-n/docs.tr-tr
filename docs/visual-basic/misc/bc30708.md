# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Loop deyimleri Komut penceresi içinde geçerli değildir

`Loop` deyimler **komut** penceresinde kullanılamaz.  
  
 **Hata kimliği:** BC30708  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Loop`Hata ayıklama kodundan deyimleri kaldır.
