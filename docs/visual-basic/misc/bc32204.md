# <a name="event-name-length-cannot-exceed-1011-characters"></a>Olay adı uzunluğu 1011 karakteri aşamaz
Bir ifadesiyle belirtilen her olayın adı veya tanımlayıcısı `Event` 1011 karakterle sınırlıdır.  
  
 **Hata kimliği:** BC32204  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tanımlayıcının uzunluğunu azaltın.
