# <a name="optional-parameters-cannot-be-declared-as-the-type-type"></a>İsteğe bağlı parametreler ' ' türü olarak bildirilemez \<type>

İsteğe bağlı parametreler, bir yapının veri türünde olamaz.  
  
 **Hata kimliği:** BC30423  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bir yapıyı isteğe bağlı bir parametreye geçirmek için parametresini olarak bildirin `Object` .  
  
2. `CType`Parametresini yordam içindeki bu yapı türüne dönüştürmek için kullanın.
