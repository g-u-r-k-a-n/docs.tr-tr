# <a name="withevents-variables-must-have-an-as-clause"></a>' WithEvents ' değişkenlerinde ' As ' yan tümcesi olmalıdır
`As`Anahtar sözcüğü ile bir yan tümce sağlamadınız `WithEvents` . Değişkeni, olayları yükseltebilen belirli bir sınıf olarak bildirin.  
  
 **Hata kimliği:** BC30412  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `As`Olayları yükseltecan sınıfı belirten yan tümceyi ekleyin.
