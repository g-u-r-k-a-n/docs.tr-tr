# <a name="end-get-must-be-the-first-statement-on-a-line"></a>' End Get ' satırdaki ilk ifade olmalıdır
Bir `End Get` ifade bir iki nokta üst üste (:) Ekstre ayırıcı. `End Get`kaynak satırındaki tek bir ifade olmalıdır.  
  
 **Hata kimliği:** BC32033  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyimi farklı satırlara ayırın.
