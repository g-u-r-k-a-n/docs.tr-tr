# <a name="c-compiler-options"></a>C# Derleyici Seçenekleri

Derleyici çalıştırılabilir (.exe) dosyaları, dinamik bağlantı kitaplıkları (.dll) veya kod modülleri (.netmodule) üretir.

Her derleyici seçeneği iki şekilde kullanılabilir: **-option** ve **/option**. Belgeler yalnızca **seçenek** formunu gösterir.

Visual Studio'da, *web.config* dosyasında derleyici seçeneklerini ayarlarsınız. Daha fazla bilgi için [ \<derleyici> Öğesi'ne](../../../framework/configure-apps/file-schema/compiler/compiler-element.md)bakın.
