# <a name="native-wpf-browser-hosting-support-apis"></a>Destek API'leri Barındıran Yerel WPF Tarayıcısı
WPF uygulamalarının Web tarayıcılarında barındırılan etkin belge sunucusu (DocObject olarak da bilinir) WPF Ana Bilgisayar'dan kaydedilmiştir. Internet Explorer doğrudan etkinleştirebilir ve etkin bir belgeyle tümleştirebilirsiniz. Mozilla tarayıcılarında XBAP'ların ve gevşek XAML belgelerinin barındırıldığı için WPF, Internet Explorer'ın yaptığı gibi WPF Active Document sunucusuna benzer bir barındırma ortamı sağlayan bir NPAPI eklentisi sağlar. Ancak, diğer tarayıcılarda ve bağımsız uygulamalarda XBAPs ve XAML belgeleri barındırmak için en kolay pratik yolu Internet Explorer Web Tarayıcısı kontrolü üzerinden. Web Tarayıcısı denetimi karmaşık Active Document sunucu barındırma ortamısağlar, ancak kendi ana bilgisayar ını bu ortamı özelleştirmesini ve genişletmesini ve geçerli Etkin Belge nesnesiyle doğrudan iletişim kurmasını sağlar.  
  
 WPF Active Document [sunucusu, IOleObject](/windows/win32/api/oleidl/nn-oleidl-ioleobject), [IOleDocument](/windows/win32/api/docobj/nn-docobj-ioledocument), [IOleInPlaceActiveObject](/windows/win32/api/oleidl/nn-oleidl-ioleinplaceactiveobject), [IPersistMoniker](https://docs.microsoft.com/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms775042(v=vs.85)), [IOleCommandTarget](/windows/win32/api/docobj/nn-docobj-iolecommandtarget)dahil olmak üzere birçok ortak barındırma arabirimleri uygular. Web Tarayıcısı denetiminde barındırıldığında, bu arabirimler [IWebBrowser2::Document](https://docs.microsoft.com/previous-versions/aa752116(v=vs.85)) özelliği tarafından döndürülen nesneden sorgulanabilir.  
  
## <a name="iolecommandtarget"></a>ıolecommandtarget  
 WPF Active Document sunucusunun [IOleCommandTarget](/windows/win32/api/docobj/nn-docobj-iolecommandtarget) uygulaması, standart OLE komut grubunun (null komut grubu GUID ile) çok sayıda navigasyonla ilgili ve tarayıcıya özgü komutlarını destekler. Buna ek olarak, CGID_PresentationHost adlı özel bir komut grubu tanır. Şu anda, bu grup içinde tanımlanan tek bir komut vardır.  
  
```cpp  
DEFINE_GUID(CGID_PresentationHost, 0xd0288c55, 0xd6, 0x4f5e, 0xa8, 0x51, 0x79, 0xde, 0xc5, 0x1b, 0x10, 0xec);  
enum PresentationHostCommands {
   PHCMDID_TABINTO = 1
};  
```  
  
 PHCMDID_TABINTO, Shift tuşu'nun durumuna bağlı olarak, PresentationHost'a odağı içeriğindeki ilk veya son odaklanabilir öğeye çevirmesini bildirir.
