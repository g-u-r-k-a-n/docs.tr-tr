# <a name="argument-per-is-not-valid"></a>' Per ' bağımsız değişkeni geçerli değil

İçin sağlanan bir bağımsız değişken `Per` , `Double` 1 ile arasında bir ödeme dönemi belirtmeye `NPer` , geçerli değildir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişkenin geçerli Aralık içinde olduğundan emin olun.
