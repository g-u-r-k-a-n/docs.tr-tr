# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose

System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Description  

 Bir ServiceHost kapatma işlemi zaman aşımına uğradı. Bu, bazı oturumlardaki kanalların istemci tarafından düzgün şekilde kapatmadığını ve durdurulmuş olduğunu gösterir. Varsayılan saati, kapatma zaman aşımını ayarlamak için değiştirin.
