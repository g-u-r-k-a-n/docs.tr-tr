.NET yüklemenizi yönetmek için bir paket yöneticisi kullanırken, daha önce bir önizleme sürümü yüklediyseniz bir çakışma ile karşılaşabilirsiniz. Paket Yöneticisi, önizleme dışı sürümü önceki bir .NET sürümü olarak yorumlayabilir. Önizleme dışı sürümü yüklemek için önce önizleme sürümlerini kaldırın. .NET kaldırma hakkında daha fazla bilgi için bkz. [.NET çalışma zamanını ve SDK 'yı kaldırma](../remove-runtime-sdk-versions.md?pivots=os-linux#uninstall-net).
