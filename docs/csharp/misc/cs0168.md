# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="4c7d6-103">Derleyici Uyarısı (düzey 3) CS0168</span><span class="sxs-lookup"><span data-stu-id="4c7d6-103">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="4c7d6-104">' Var ' değişkeni tanımlanmış ancak hiç kullanılmadı</span><span class="sxs-lookup"><span data-stu-id="4c7d6-104">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="4c7d6-105">Derleyici bir değişken bildirdiğinizde, ancak kullanmıyorsanız, bir düzey üç uyarı verir.</span><span class="sxs-lookup"><span data-stu-id="4c7d6-105">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="4c7d6-106">Aşağıdaki örnek bir CS0168 uyarısı üretir:</span><span class="sxs-lookup"><span data-stu-id="4c7d6-106">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
