# <a name="compiler-error-cs1061"></a><span data-ttu-id="9ce7b-103">Derleyici Hatası CS1061</span><span class="sxs-lookup"><span data-stu-id="9ce7b-103">Compiler Error CS1061</span></span>

<span data-ttu-id="9ce7b-104">' Type ', ' Member ' için bir tanım içermiyor ve ' Type ' türünde bir ilk bağımsız değişken kabul eden ' name ' genişletme metodu bulunamadı (bir using yönergeniz veya derleme başvurunuz eksik olabilir mi?).</span><span class="sxs-lookup"><span data-stu-id="9ce7b-104">'type' does not contain a definition for 'member' and no extension method 'name' accepting a first argument of type 'type' could be found (are you missing a using directive or an assembly reference?).</span></span>  
  
 <span data-ttu-id="9ce7b-105">Bu hata, bir yöntemi çağırmayı veya var olmayan bir sınıf üyesine erişmeyi denediğinizde oluşur.</span><span class="sxs-lookup"><span data-stu-id="9ce7b-105">This error occurs when you try to call a method or access a class member that does not exist.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9ce7b-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="9ce7b-106">Example</span></span>  

 <span data-ttu-id="9ce7b-107">Aşağıdaki örnek, `TestClass1` bir yöntemine sahip olmadığı IÇIN CS1061 oluşturur `DisplaySomething` .</span><span class="sxs-lookup"><span data-stu-id="9ce7b-107">The following example generates CS1061 because `TestClass1` does not have a `DisplaySomething` method.</span></span> <span data-ttu-id="9ce7b-108">Çağrılan bir yöntemi vardır `WriteSomething` .</span><span class="sxs-lookup"><span data-stu-id="9ce7b-108">It does have a method that is called `WriteSomething`.</span></span> <span data-ttu-id="9ce7b-109">Bu kaynak kodun yazarı ne olabilir?</span><span class="sxs-lookup"><span data-stu-id="9ce7b-109">Perhaps that is what the author of this source code meant to write.</span></span>  
  
```csharp  
// cs1061.cs  
public class TestClass1  
{  
    // TestClass1 has one method, called WriteSomething.  
    public void WriteSomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestClass2  
{  
    // TestClass2 has one method, called DisplaySomething.  
    public void DisplaySomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestTheClasses  
{  
    public static void Main()  
    {  
        TestClass1 tc1 = new TestClass1();  
        TestClass2 tc2 = new TestClass2();  
        // The following call fails because TestClass1 does not have
        // a method called DisplaySomething.  
        tc1.DisplaySomething("Hello");      // CS1061  
  
        // To correct the error, change the method call to either
        // tc1.WriteSomething or tc2.DisplaySomething.  
        tc1.WriteSomething("Hello from TestClass1");  
        tc2.DisplaySomething("Hello from TestClass2");  
    }  
}  
```
