# <a name="argument-cannot-be-less-than-zero"></a>Bağımsız değişken sıfırdan küçük olamaz

Sıfırdan küçük bir değer sağlandı, ancak bağımsız değişken sıfırdan küçük olamaz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sorunun doğru olduğundan emin olmak için söz konusu değerin nasıl hesaplanacağını denetleyin.
