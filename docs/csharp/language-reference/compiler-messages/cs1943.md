# <a name="compiler-error-cs1943"></a><span data-ttu-id="429b3-103">Derleyici Hatası CS1943</span><span class="sxs-lookup"><span data-stu-id="429b3-103">Compiler Error CS1943</span></span>

<span data-ttu-id="429b3-104">' Type ' kaynak türündeki bir sorgu ifadesindeki sonraki from yan tümcesinde ' Type ' türünde bir ifadeye izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="429b3-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="429b3-105">' Method ' çağrısında tür çıkarımı başarısız oldu.</span><span class="sxs-lookup"><span data-stu-id="429b3-105">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="429b3-106">Tüm Aralık değişkenleri sorgulanabilir türleri temsil etmelidir.</span><span class="sxs-lookup"><span data-stu-id="429b3-106">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="429b3-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="429b3-107">To correct this error</span></span>  
  
1. <span data-ttu-id="429b3-108">Türün `IEnumerable` , `IEnumerable<T>` ya da türetilmiş bir arabirim ya da kendisi için tanımlanmış bir sorgu düzenine sahip başka herhangi bir türü uygulayan bir sorgulanabilir tür olduğundan emin olun.</span><span class="sxs-lookup"><span data-stu-id="429b3-108">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2. <span data-ttu-id="429b3-109">Tür genel olmayan bir değer ise `IEnumerable` , Aralık değişkeninde açık bir tür belirtin.</span><span class="sxs-lookup"><span data-stu-id="429b3-109">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="429b3-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="429b3-110">Example</span></span>  

 <span data-ttu-id="429b3-111">Aşağıdaki kod CS1943 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="429b3-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
