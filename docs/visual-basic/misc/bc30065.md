# <a name="exit-sub-is-not-valid-in-a-function-or-property"></a>' Exit Sub ', Function veya Property içinde geçerli değil

`Exit Sub``Function`yordamda veya `Property` yordamda görüntülenir. Bir `Exit` deyimin gerçekleştiği blokla eşleşmesi gerekir.  
  
 **Hata kimliği:** BC30065  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öğesini `Exit Sub` `Exit Function` veya `Exit Property` ifadesiyle uygun şekilde değiştirin.
