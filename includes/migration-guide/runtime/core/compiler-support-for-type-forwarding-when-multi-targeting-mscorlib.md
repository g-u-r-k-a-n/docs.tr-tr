Yeni bir CodeDOM özelliği, bir derleyicinin mscorlib.dll .NET Framework 4,5 sürümü yerine hedeflenen mscorlib.dll göre derlenmesini sağlar.

| Name    | Değer       |
|:--------|:------------|
| Kapsam   |Edge|
|Sürüm|4,5|
|Tür|Çalışma Zamanı|
