# <a name="specifier1-and-specifier2-cannot-be-combined"></a>' \<specifier1> ' ve ' \<specifier2> ' birleştirilemez

 **Hata kimliği:** BC31408  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir veya başka tanımlayıcı kaldırın.
