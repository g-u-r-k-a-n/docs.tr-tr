<span data-ttu-id="4bc7a-104">Arabirimde bildirilen bir [olayı](../keywords/event.md) açıkça uygularken, genellikle derleyici `add` `remove` tarafından sağlanan olay ve olay erişimcilerini el ile sağlamanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="4bc7a-104">When explicitly implementing an [event](../keywords/event.md) that was declared in an interface, you must manually provide the `add` and `remove` event accessors that are typically provided by the compiler.</span></span> <span data-ttu-id="4bc7a-105">Erişimci kodu, arabirim olayını sınıfınızdaki başka bir olaya (bu konuda daha sonra gösterilir) veya kendi temsilci türüne bağlayabilir.</span><span class="sxs-lookup"><span data-stu-id="4bc7a-105">The accessor code can connect the interface event to another event in your class (shown later in this topic) or to its own delegate type.</span></span> <span data-ttu-id="4bc7a-106">Daha fazla bilgi için [arabirim olaylarının nasıl uygulanacağını](../../programming-guide/events/how-to-implement-interface-events.md)görün.</span><span class="sxs-lookup"><span data-stu-id="4bc7a-106">For more information, see [How to implement interface events](../../programming-guide/events/how-to-implement-interface-events.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="4bc7a-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="4bc7a-107">Example</span></span>

 <span data-ttu-id="4bc7a-108">Aşağıdaki örnek CS0071 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="4bc7a-108">The following sample generates CS0071.</span></span>

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

interface ITest
{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
