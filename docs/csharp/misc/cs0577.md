<span data-ttu-id="eae9b-104">' Function ' bir Oluşturucu, yıkıcı, işleç veya açık arabirim uygulama olduğundan koşullu öznitelik geçerli değil</span><span class="sxs-lookup"><span data-stu-id="eae9b-104">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>

```csharp  
// CS0577.cs

{  
   void m();  
}

   void I.m() {}  
}  
```
