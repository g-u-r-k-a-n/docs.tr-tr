# <a name="compiler-error-cs1520"></a><span data-ttu-id="f5abc-102">Derleyici hatası CS1520</span><span class="sxs-lookup"><span data-stu-id="f5abc-102">Compiler Error CS1520</span></span>
<span data-ttu-id="f5abc-103">Metot bir dönüş türüne sahip olmalıdır</span><span class="sxs-lookup"><span data-stu-id="f5abc-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="f5abc-104">Bir sınıf, yapı veya arabirimde tanımlanmış bir yöntem açık bir dönüş türüne sahip olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="f5abc-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="f5abc-105">Aşağıdaki örnekte, kare yönteminin dönüş değeri [String](../language-reference/builtin-types/reference-types.md)' dir:</span><span class="sxs-lookup"><span data-stu-id="f5abc-105">In the following example, the Square method has a return value of [string](../language-reference/builtin-types/reference-types.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="f5abc-106">Aşağıdaki örnek CS1520 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f5abc-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="f5abc-107">Alternatif olarak, aşağıdaki örnekte olduğu gibi, bir oluşturucunun adının durumu sınıf veya yapı bildiriminden farklıysa, bu hata ile karşılaşabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f5abc-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="f5abc-108">Ad, sınıf adıyla tam olarak aynı olmadığından, derleyici onu bir Oluşturucu değil normal bir yöntem olarak yorumlar ve şu hatayı üretir:</span><span class="sxs-lookup"><span data-stu-id="f5abc-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
