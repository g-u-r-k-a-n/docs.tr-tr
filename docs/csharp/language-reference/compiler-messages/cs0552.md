# <a name="compiler-error-cs0552"></a><span data-ttu-id="a6052-102">Derleyici Hatası CS0552</span><span class="sxs-lookup"><span data-stu-id="a6052-102">Compiler Error CS0552</span></span>
<span data-ttu-id="a6052-103">'dönüşüm yordamı' : kullanıcı tanımlı arayüze/arabirimden dönüştürme</span><span class="sxs-lookup"><span data-stu-id="a6052-103">'conversion routine' : user defined conversion to/from interface</span></span>  
  
 <span data-ttu-id="a6052-104">Bir arabirime veya arabirimden kullanıcı tanımlı bir dönüşüm oluşturamazsınız.</span><span class="sxs-lookup"><span data-stu-id="a6052-104">You cannot create a user-defined conversion to or from an interface.</span></span> <span data-ttu-id="a6052-105">Dönüştürme yordamına ihtiyacınız varsa, arabirimi bir sınıf haline getirerek bu hatayı giderin veya arabirimden bir sınıf türetin.</span><span class="sxs-lookup"><span data-stu-id="a6052-105">If you need the conversion routine, resolve this error by making the interface a class or derive a class from the interface.</span></span>  
  
 <span data-ttu-id="a6052-106">Aşağıdaki örnek CS0552 üretir:</span><span class="sxs-lookup"><span data-stu-id="a6052-106">The following sample generates CS0552:</span></span>  
  
```csharp  
// CS0552.cs  
public interface ii  
{  
}  
  
public class a  
{  
   // delete the routine to resolve CS0552  
   public static implicit operator ii(a aa) // CS0552  
   {  
      return new ii();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
