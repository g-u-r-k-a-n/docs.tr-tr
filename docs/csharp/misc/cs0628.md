# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="22de5-103">Derleyici Uyarısı (düzey 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="22de5-103">Compiler Warning (level 4) CS0628</span></span>

<span data-ttu-id="22de5-104">' üye ': korumalı sınıfta yeni korunan üye tanımlandı</span><span class="sxs-lookup"><span data-stu-id="22de5-104">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="22de5-105">[Kapalı](../language-reference/keywords/sealed.md) bir sınıf, sınıftan devralması ve üyeyi kullanabilmesi için [korumalı](../language-reference/keywords/protected.md) bir üye sunmaz `sealed` `protected` .</span><span class="sxs-lookup"><span data-stu-id="22de5-105">A [sealed](../language-reference/keywords/sealed.md) class cannot introduce a [protected](../language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="22de5-106">Aşağıdaki örnek CS0628 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="22de5-106">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
