# <a name="ehostapplicationpolicy-enumeration"></a>EHostApplicationPolicy Numaralandırması
[IHostAssemblyManager](../../../../docs/framework/unmanaged-api/hosting/ihostassemblymanager-interface.md) arabirimi nesnesinin nasıl değiştirileceğini gösterir. Bu numaralandırma kullanım dışı bırakıldı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
typedef enum _HostApplicationPolicy {  
    HOST_APPLICATION_BINDING_POLICY = 1  
} EHostApplicationPolicy;  
```
