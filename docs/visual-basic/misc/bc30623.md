# <a name="end-namespace-must-be-preceded-by-a-matching-namespace"></a>' End Namespace ' öncesinde eşleşen bir ' namespace ' olmalıdır
Ad alanı blokları bir `Namespace` ifadesiyle başlar ve bir yapı ile `End Namespace` biter.  
  
 **Hata kimliği:** BC30623  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ad alanı bloğunun bir ifadesiyle başladığından emin olun `Namespace` .
