# <a name="compiler-error-cs1563"></a>Derleyici hatası CS1563

' Çıktı dosyası ' çıkışının hiç kaynak dosyası yok  
  
 [**OutputAssembly**](../language-reference/compiler-options/output.md#outputassembly) derleyici seçeneği belirtildi, ancak hiçbir kaynak kodu dosyası izmedi. **OutputAssembly** kullanmamalısınız ya da **OutputAssembly**' den sonra kaynak kodu dosyalarını belirtmeniz gerekir.
