# <a name="systemservicemodeltxcompletionstatuscompletedforerror"></a>System.ServiceModel.TxCompletionStatusCompletedForError
Belirtilen işlem için belirtilen işlem, işlenmemiş bir yürütme özel durumu nedeniyle tamamlandı.  
  
## <a name="description"></a>Açıklama  
 Geçerli işlemi tamamlamaya çalışırken bir hata oluştuğunda izlenebilir. Bu, çağrıyı yapana bir yanıt veya hata gönderilmeden önce oluşur.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Özel durum iletisi ve herhangi bir işlem yapılabilir öğe için izlenen iletiyi inceleyin.
