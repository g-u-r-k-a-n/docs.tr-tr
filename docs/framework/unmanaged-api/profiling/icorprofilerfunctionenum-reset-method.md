# <a name="icorprofilerfunctionenumreset-method"></a>ICorProfilerFunctionEnum::Reset Yöntemi
Moves the enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT Reset();  
```
