# <a name="printer-error"></a>Yazıcı hatası

Yazıcıda bir hata oluştu, ancak dosyayı gönderen bilgisayara başka bilgi döndürülmedi.  
  
### <a name="to-correct-the-error"></a>Hatayı düzeltmek için  
  
- Yazıcıyı fiziksel olarak inceleyin. Bilgisayar ve yazıcı arasındaki tüm bağlantıların dolu olduğundan emin olun. Çoğu yazıcı, "çevrimdışı" veya "kağıt dışı" gibi hata bilgileri için bir ekran sağlar.
