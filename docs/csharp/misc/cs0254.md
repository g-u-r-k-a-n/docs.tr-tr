# <a name="compiler-error-cs0254"></a><span data-ttu-id="29979-103">Derleyici hatası CS0254</span><span class="sxs-lookup"><span data-stu-id="29979-103">Compiler Error CS0254</span></span>

<span data-ttu-id="29979-104">Sabit deyim atamasının sağ tarafı bir atama ifadesi olamaz</span><span class="sxs-lookup"><span data-stu-id="29979-104">The right hand side of a fixed statement assignment may not be a cast expression</span></span>  
  
 <span data-ttu-id="29979-105">[Sabit](../language-reference/keywords/fixed-statement.md) ifadenin sağ tarafı bir tür dönüştürme kullanamaz.</span><span class="sxs-lookup"><span data-stu-id="29979-105">The right side of a [fixed](../language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="29979-106">Daha fazla bilgi için bkz. [güvenli olmayan kod ve işaretçiler](../language-reference/unsafe-code.md).</span><span class="sxs-lookup"><span data-stu-id="29979-106">For more information, see [Unsafe Code and Pointers](../language-reference/unsafe-code.md).</span></span>  
  
 <span data-ttu-id="29979-107">Aşağıdaki örnek CS0254 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="29979-107">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
