<span data-ttu-id="066da-103">Kodunuz, `For...Next` aşağıdaki koşullar doğru olduğundan, derleyicinin döngü denetim değişkeni için bir veri türünü çıkaramayacağı bir döngü içerir:</span><span class="sxs-lookup"><span data-stu-id="066da-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>

 <span data-ttu-id="066da-108">Örneğin, aşağıdaki döngü türü `Integer` bir döngü bağlı ve türü `UInteger`bir döngü bağlı:</span><span class="sxs-lookup"><span data-stu-id="066da-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>

' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="066da-109">Arasında `Integer` olası bir dönüşüm `UInteger`ve , ve `UInteger` `Integer`arasında olası bir dönüşüm vardır ve , arasında olası bir dönüşüm, ancak her ikisi de dönüşümleri daraltıyor, bu yüzden ne en iyi seçimdir.</span><span class="sxs-lookup"><span data-stu-id="066da-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="066da-110">Sonraki örnekte, üçüncü bir `Double` tür değişkeni eklenir.</span><span class="sxs-lookup"><span data-stu-id="066da-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="066da-111">Her `Integer` `UInteger` ikisi de ve en `Double`iyi seçim e `Double` dönüştürme yapar standart genişletme dönüşümleri var.</span><span class="sxs-lookup"><span data-stu-id="066da-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="066da-112">Döngü denetimi değişkenine `i` veri türüatatını `Double`atayan yazı çıkarımı.</span><span class="sxs-lookup"><span data-stu-id="066da-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>

For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```
