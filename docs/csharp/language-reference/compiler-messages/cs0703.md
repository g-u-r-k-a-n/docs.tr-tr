# <a name="compiler-error-cs0703"></a><span data-ttu-id="ea65d-103">Derleyici Hatası CS0703</span><span class="sxs-lookup"><span data-stu-id="ea65d-103">Compiler Error CS0703</span></span>

<span data-ttu-id="ea65d-104">Tutarsız erişilebilirlik: ' Identifier ' kısıtlama türü, ' Identifier ' değerinden daha az erişilebilir</span><span class="sxs-lookup"><span data-stu-id="ea65d-104">Inconsistent accessibility: constraint type 'identifier' is less accessible than 'identifier'</span></span>  
  
 <span data-ttu-id="ea65d-105">Kısıtlama genel parametreyi genel sınıfın kendisinden daha az erişilebilir olacak şekilde zorlamayabilir.</span><span class="sxs-lookup"><span data-stu-id="ea65d-105">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="ea65d-106">Aşağıdaki örnekte, C genel sınıfı \<T> genel olarak bildirildiği sürece kısıtlama, bir iç arabirim uygulamak için zorla vermeye çalışır.</span><span class="sxs-lookup"><span data-stu-id="ea65d-106">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="ea65d-107">Buna izin verilse de, yalnızca iç erişimi olan istemciler sınıf için parametre oluşturabiliyor, bu nedenle, sınıf yalnızca iç erişimi olan istemciler tarafından kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="ea65d-107">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="ea65d-108">Bu hatadan kurtulmak için, genel sınıfın erişim düzeyinin, sınırlar içinde görünen sınıflardan veya arabirimlerden daha az kısıtlayıcı olmadığından emin olun.</span><span class="sxs-lookup"><span data-stu-id="ea65d-108">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>  
  
 <span data-ttu-id="ea65d-109">Aşağıdaki örnek CS0703 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ea65d-109">The following sample generates CS0703:</span></span>  
  
```csharp  
// CS0703.cs  
internal interface I {}  
public class C<T> where T : I  // CS0703 – I is internal; C<T> is public  
{  
}  
```
