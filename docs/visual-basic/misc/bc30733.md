# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>' \<membername> ', ' ' tarafından zaten bildirildi \<procedurename> , bu, bu için üretildi \<containername>

Ortak dil çalışma zamanı tarafından belirtilen üyeyle aynı ada sahip bir üye bildirildi.

**Hata kimliği:** BC30733

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Çakışmayı kaldırmak için üyenin adını değiştirin.
