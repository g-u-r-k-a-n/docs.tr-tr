# <a name="nullable-parameters-must-specify-a-type"></a>Null yapılabilir parametrelerin bir tür belirtmesi gerekir
Parametre null yapılabilir tür değiştiricisini (?) içerir, ancak bir tür belirtmez.  
  
 **Hata kimliği:** BC36632  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Null yapılabilir değer türü parametresini bildirirken bir tür belirtin.
