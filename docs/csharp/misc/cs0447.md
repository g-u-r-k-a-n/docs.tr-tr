# <a name="compiler-error-cs0447"></a><span data-ttu-id="a972a-102">Derleyici hatası CS0447</span><span class="sxs-lookup"><span data-stu-id="a972a-102">Compiler Error CS0447</span></span>

<span data-ttu-id="a972a-103">Öznitelikler tür bağımsız değişkenlerinde kullanılamaz, yalnızca tür parametrelerinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="a972a-103">Attributes cannot be used on type arguments, only on type parameters</span></span>

<span data-ttu-id="a972a-104">Bu hata, bir çağırma ifadesinde oluşan bir tür bağımsız değişkenine bir öznitelik uyguladığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="a972a-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="a972a-105">Aşağıdaki gibi bir sınıf veya yöntem bildirimi deyimindeki bir tür parametresine bir öznitelik uygulamak kabul edilebilir:</span><span class="sxs-lookup"><span data-stu-id="a972a-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```csharp
class C<[some attribute] T> {…}
```

<span data-ttu-id="a972a-106">Aşağıdaki kod satırı bu hatayı oluşturacaktır.</span><span class="sxs-lookup"><span data-stu-id="a972a-106">The following line of code will generate this error.</span></span> <span data-ttu-id="a972a-107">Önceki kod satırında tanımlanan `C` sınıfının, `MyStaticMethod` adlı statik bir yöntemi olduğu varsayılır.</span><span class="sxs-lookup"><span data-stu-id="a972a-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```csharp
C<[some attribute] T>.MyStaticMethod();
```

## <a name="example"></a><span data-ttu-id="a972a-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="a972a-108">Example</span></span>

<span data-ttu-id="a972a-109">Aşağıdaki kod hata CS0447 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a972a-109">The following code generates error CS0447:</span></span>

```csharp
// CS0447.cs
using System;

namespace Test41
{
    public interface I<A>
    {
        void Meth<B>();
    }
    public class B : I<int>
    {
        void I<[Test] int>.Meth<X>() { }  // CS0447
    }
}
```
