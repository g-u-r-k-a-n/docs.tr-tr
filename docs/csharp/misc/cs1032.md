# <a name="compiler-error-cs1032"></a><span data-ttu-id="a4c25-103">Derleyici hatası CS1032</span><span class="sxs-lookup"><span data-stu-id="a4c25-103">Compiler Error CS1032</span></span>

<span data-ttu-id="a4c25-104">Dosyadaki ilk belirteçten sonra önişlemci sembolleri tanımlanamaz/tanımlanamıyor</span><span class="sxs-lookup"><span data-stu-id="a4c25-104">Cannot define/undefine preprocessor symbols after first token in file</span></span>  
  
 <span data-ttu-id="a4c25-105">`#define`Ve `#undef` [Önişlemci yönergeleri](../language-reference/preprocessor-directives/index.md) , bir programın başlangıcında, ad alanı bildiriminde kullanılanlar gibi diğer anahtar sözcüklerden önce kullanılmalıdır.</span><span class="sxs-lookup"><span data-stu-id="a4c25-105">The `#define` and `#undef` [preprocessor directives](../language-reference/preprocessor-directives/index.md) must be used at the beginning of a program, before any other keywords, such as those used in the namespace declaration.</span></span>  
  
 <span data-ttu-id="a4c25-106">Aşağıdaki örnek CS1032 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a4c25-106">The following sample generates CS1032:</span></span>  
  
```csharp  
// CS1032.cs  
namespace x  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace  
      public static void Main()  
      {  
      }  
   }  
}  
```
