|||  
|-|-|  
|ID|5403|  
|Anahtar sözcükler|Güvenlik|  
|Level|Ayrıntılı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Description  

 Bu olay, SecurityToken doğrulaması başarılı olduğunda yayınlanır.

 SecurityToken (' %1 ' türü ve ' %2 ' kimliği) doğrulaması başarılı oldu.
