 <span data-ttu-id="23dc9-105">Statik `this` Oluşturucu, sınıfın herhangi bir örneği oluşturulmadan önce otomatik olarak çağrıldığından, statik oluşturucuda çağırmaya izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="23dc9-105">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="23dc9-106">Ayrıca, statik oluşturucular devralınmaz ve doğrudan çağrılamaz.</span><span class="sxs-lookup"><span data-stu-id="23dc9-106">Also, static constructors are not inherited, and cannot be called directly.</span></span>

```csharp  
    }

```
