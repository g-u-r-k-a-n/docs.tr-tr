# <a name="expected-one-of-dim-const-public-private-protected-friend-shadows-readonly-or-shared"></a>' Dim ', ' const ', ' Public ', ' Private ', ' protected ', ' Friend ', ' Shadows ', ' ReadOnly ' veya ' Shared ' dışında bir tane bekleniyor

Bildirim ifadesinde bir bildirim anahtar sözcüğü eksik. Olası nedenlerden biri, öznitelik bildiriminin bir yöntemi çağırmasına neden olur.  
  
 **Hata kimliği:** BC30195  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bir yöntemin öznitelik bildiriminde bildirilip bildirilmemiş olup olmadığını denetleyin.  
  
2. Bildirimini uygun bildirim anahtar sözcüğüyle başlatın.
