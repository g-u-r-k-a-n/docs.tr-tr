# <a name="compiler-error-cs1725"></a>Derleyici Hatası CS1725
Arkadaş derleme başvurusu 'başvuru' geçersizdir. InternalsVisibleTo bildirimleri bir sürüm, kültür, ortak anahtar belirteci veya işlemci mimarisi belirtilemez.  
  
 Arkadaş derleme başvurusuna sürüm kültürü ekleyemezsiniz. Kısmi sınıflar arkadaş derlemeleri tarafından görülebilmeli.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1725 oluşturur.  
  
```csharp  
// CS1725.cs  
// compile with: /target:library  
using System.Runtime.CompilerServices;  
[assembly:InternalsVisibleTo("partial01,version=1.1.0.0")]   // CS1725  
// try the following line instead  
// [assembly:InternalsVisibleTo("partial01")]  
  
partial class TestClass
{  
   public static string strBar = "my string";  
}  
```
