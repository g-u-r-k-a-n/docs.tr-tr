# <a name="local-variable-variablename-is-already-declared-in-the-current-block"></a>' ' Yerel değişkeni \<variablename> geçerli blokta zaten tanımlanmış

Aynı ada sahip iki değişken aynı blokta (bir yordam veya gibi bir denetim bloğu olabilir `If...Then` ) belirtilir.  
  
 **Hata kimliği:** BC30288  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimlerden birini farklı bir bloğa taşıyın veya değişkenlerden birinin adını değiştirin.
