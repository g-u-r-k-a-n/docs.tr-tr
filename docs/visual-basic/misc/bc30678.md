# <a name="end-statement-not-valid"></a>' End ' deyimleri geçerli değil
`End`Anahtar sözcüğü yanlış kullanılıyor veya geçerli bağlamda uygun değil.  
  
 **Hata kimliği:** BC30678  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `End`Bir yordamın kapsam kaybetmesine neden olmak için kendisini kullanın.  
  
2. `End`Yapı veya özellik gibi bir kod bloğunu sonlandırmak için diğer anahtar sözcüklerle kullanın.
