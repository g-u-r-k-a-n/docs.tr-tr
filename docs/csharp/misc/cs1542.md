# <a name="compiler-error-cs1542"></a>Derleyici hatası CS1542

' dll ' zaten bir derleme olduğundan bu derlemeye eklenemez; Bunun yerine '/R ' seçeneğini kullanın  
  
 [**AddModules**](../language-reference/compiler-options/inputs.md#addmodules) derleyici seçeneğiyle başvurulan dosya, [ **TargetType** derleyici seçeneğinin](../language-reference/compiler-options/output.md#targettype) **module** öğesiyle oluşturulmamış; Bu derlemedeki dosyaya başvurmak için [**başvuruları**](../language-reference/compiler-options/inputs.md#references) kullanın.
