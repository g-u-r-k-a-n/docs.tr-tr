# <a name="resume-or-goto-expected"></a>' Özgeçmişi ' veya ' GoTo ' bekleniyor
Bir `On Error` ifade, bir `Resume` yan tümce veya `GoTo` yan tümce içermez. Tek bir yan tümce veya diğerini kullanmalıdır.  
  
 **Hata kimliği:** BC32019  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Resume`Deyime bir yan tümce ya da `GoTo` yan tümce ekleyin `On Error` .
