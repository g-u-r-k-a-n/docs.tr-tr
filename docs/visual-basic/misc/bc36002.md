# <a name="global-not-allowed-in-handles-local-name-expected"></a>Tanıtıcılarda ' Global ' öğesine izin verilmiyor; Yerel ad bekleniyor

`Handles`Yan tümce yerel bir olaya başvurmalıdır. `Global`Anahtar sözcüğü, genel programlama öğelerine erişim sağlar.  
  
 **Hata kimliği:** BC36002  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Handles`Yan tümcesini, genel örnek yerine bir olayın yerel örneğine başvuracak şekilde değiştirin.
