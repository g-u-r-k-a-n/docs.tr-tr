# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="1c1a9-103">Derleyici Uyarısı (düzey 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="1c1a9-103">Compiler Warning (level 2) CS0618</span></span>

<span data-ttu-id="1c1a9-104">' Member ' artık kullanılmıyor: ' text '</span><span class="sxs-lookup"><span data-stu-id="1c1a9-104">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="1c1a9-105">Sınıf üyesi, `Obsolete` sınıf üyesine başvurulduğunda bir uyarı verilecek şekilde, özniteliğiyle işaretlenmiş.</span><span class="sxs-lookup"><span data-stu-id="1c1a9-105">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="1c1a9-106">Daha fazla bilgi için bkz. [ortak öznitelikler](../attributes/global.md).</span><span class="sxs-lookup"><span data-stu-id="1c1a9-106">For more information, see [Common Attributes](../attributes/global.md).</span></span>  
  
 <span data-ttu-id="1c1a9-107">Aşağıdaki örnek CS0618 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1c1a9-107">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
