# <a name="compiler-error-cs0545"></a>Derleyici Hatası CS0545
'fonksiyon' : 'özellik' geçersiz kılınabilir erişime sahip olmadığı için geçersiz kılamaz  
  
 Taban sınıfın geçersiz kılacak böyle bir tanımı olmadığında, özellik erişimcisi için geçersiz kılma yı tanımlamak için bir deneme yapıldı. Bu hatayı şu şekilde çözebilirsiniz:  
  
- Taban `set` sınıfa bir erişimci ekleme.  
  
- `set` Erişime gireni türetilmiş sınıftan kaldırma.  
  
- Türemiş bir sınıftaki bir özelliğe [yeni](../keywords/new-modifier.md) anahtar sözcük ekleyerek taban sınıf özelliğini gizleme.  
  
- Taban sınıf [özelliğini sanal](../keywords/virtual.md)hale getirme.  
  
 Daha fazla bilgi için [Bkz. Özellikleri Kullanma.](../../programming-guide/classes-and-structs/using-properties.md)  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0545 üretir.  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
