# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="a2f01-103">Derleyici Uyarısı (düzey 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="a2f01-103">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="a2f01-104">' class ' Object. Equals (Object o) öğesini geçersiz kılar, ancak Object. GetHashCode () öğesini geçersiz kılmaz</span><span class="sxs-lookup"><span data-stu-id="a2f01-104">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="a2f01-105">Derleyici, yöntemi geçersiz kılmayı algıladı, <xref:System.Object.Equals%2A?displayProperty=nameWithType> ancak metodun geçersiz kılınması <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> .</span><span class="sxs-lookup"><span data-stu-id="a2f01-105">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="a2f01-106">Geçersiz kılma, <xref:System.Object.Equals%2A> geçersiz kılmak istediğiniz anlamına gelir <xref:System.Object.GetHashCode%2A> .</span><span class="sxs-lookup"><span data-stu-id="a2f01-106">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="a2f01-107">Aşağıdaki kod CS0659 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a2f01-107">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
