# <a name="join-expected"></a><span data-ttu-id="01104-102">' JOIN ' bekleniyor</span><span class="sxs-lookup"><span data-stu-id="01104-102">'Join' expected</span></span>

<span data-ttu-id="01104-103">Bir `Group Join` yan tümce, `Join` anahtar sözcük olmadan belirtildi.</span><span class="sxs-lookup"><span data-stu-id="01104-103">A `Group Join` clause has been specified without a `Join` keyword.</span></span>  
  
 <span data-ttu-id="01104-104">**Hata kimliği:** BC36631</span><span class="sxs-lookup"><span data-stu-id="01104-104">**Error ID:** BC36631</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="01104-105">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="01104-105">To correct this error</span></span>  
  
1. <span data-ttu-id="01104-106">`Join` `Group Join` Aşağıdaki örnekte gösterildiği gibi yan tümcesine anahtar sözcüğü ekleyin:</span><span class="sxs-lookup"><span data-stu-id="01104-106">Add the `Join` keyword to the `Group Join` clause, as shown in the following example:</span></span>  
  
```vb  
Dim query = From var1 In collection1 _  
            Join var2 In collection2 _  
            Group Join var3 In collection3 _  
            On var2.ID Equals var3 Into Matches = Group _  
            On var1 Equals var2.ID _  
            Select JoinID = var1, var2.Name, Matches  
```
