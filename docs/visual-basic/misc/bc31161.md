# <a name="expected-closing----for-xml-comment"></a>XML açıklaması için '--> ' kapanışı bekleniyor

Bir XML açıklama sabit değerinde gerekli '--> ' karakter eksik.  
  
 **Hata kimliği:** BC31161  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gerekli kapatma '--> ' karakterlerini XML açıklama değişmez değerinin sonuna ekleyin.
