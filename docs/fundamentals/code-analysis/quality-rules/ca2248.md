## <a name="how-to-fix-violations"></a><span data-ttu-id="bac97-116">İhlalleri çözme</span><span class="sxs-lookup"><span data-stu-id="bac97-116">How to fix violations</span></span>

<span data-ttu-id="bac97-117">İhlalleri onarmak için hem bağımsız değişkende hem de çağıranda aynı numaralandırma türünü kullanın:</span><span class="sxs-lookup"><span data-stu-id="bac97-117">To fix violations, use the same enum type on both the argument and the caller:</span></span>

```csharp
public class C
{
    [Flags]
    public enum MyEnum { A, B, }

    [Flags]
    public enum OtherEnum { A, }

    public void Method(MyEnum m)
    {
        m.HasFlag(OtherEnum.A); // Enum types are different, this call will cause an `ArgumentException` to be thrown at runtime

        m.HasFlag(MyEnum.A); // Valid call
    }
}
```
