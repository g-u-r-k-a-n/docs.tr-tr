<span data-ttu-id="34f4d-104">' Global:: ' bir diğer ad değil, her zaman genel ad alanına başvurduğundan, ' Global ' adlı bir diğer ad tanımlanması önerilmez</span><span class="sxs-lookup"><span data-stu-id="34f4d-104">Defining an alias named 'global' is ill-advised since 'global::' always references the global namespace and not an alias</span></span>

## <a name="example"></a><span data-ttu-id="34f4d-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="34f4d-106">Example</span></span>  
 <span data-ttu-id="34f4d-107">Aşağıdaki örnek CS0440 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="34f4d-107">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
