## <a name="example"></a><span data-ttu-id="936f7-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="936f7-109">Example</span></span>

 <span data-ttu-id="936f7-110">Aşağıdaki örnek CS0122 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="936f7-110">The following sample generates CS0122:</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
