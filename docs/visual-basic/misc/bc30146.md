# <a name="unable-to-sign-assembly-error-message"></a>Bütünleştirilmiş kod imzalanamıyor: \<error message>

Visual Basic derleyici, bildirime sahip bir derleme oluşturmak için derleme bağlayıcı (Al.exe, Alink olarak da bilinir) çağırır. Bağlayıcı, oluşturulduktan sonra derlemeyi imzalayan bir hata bildirdi. Bunun nedeni geçersiz bir anahtar olabilir.  
  
 **Hata kimliği:** BC30146  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Alıntı yapılan hata iletisini inceleyin ve daha fazla açıklama ve öneri için  [Al.exe (derleme bağlayıcı)](../../framework/tools/al-exe-assembly-linker.md) konusuna başvurun.  
  
2. Hata devam ederse, koşullar hakkındaki bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
