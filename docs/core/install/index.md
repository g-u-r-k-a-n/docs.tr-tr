# <a name="download-and-install-net-core"></a>.NET Core'u indirip yükleyin

Bu makalede, .NET Core'un nasıl indirilip yüklenirken yüklenmesi hakkında bilgi verilmektedir. Uygulamaları çalıştırmak için kullanılan çalışma **zamanı** ve uygulamaları oluşturmak için kullanılan **SDK** olmak üzere .NET Core'un iki bölümü vardır. SDK çalışma süresini içerir.

- Uygulamaları çalıştırmak için .NET Core'a ihtiyaç duyan bir kullanıcıysanız, [bkz.](runtime.md)

- Uygulama oluşturmak için .NET Core'a ihtiyaç duyan bir geliştiriciyseniz, [.NET Core SDK'yı yükleyin.](sdk.md)

> [!NOTE]
> .NET Core SDK çalışma süresini içerir.

## <a name="dependencies"></a>Bağımlılıklar

.NET Core Windows, Linux ve macOS'ta desteklenir. Gereksinimlerin tam listesi için [desteklenen işletim sistemlerine](dependencies.md)bakın.
