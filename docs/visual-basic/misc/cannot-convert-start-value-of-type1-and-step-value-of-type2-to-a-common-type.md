# <a name="cannot-convert-start-value-of-type-type1-and-step-value-of-type-type2-to-a-common-type"></a>' ' Türündeki başlangıç değeri \<type1> ve ' ' türündeki adım değeri \<type2> ortak bir türe dönüştürülemiyor

Belirtilen `For...Next` Denetim değerlerinde farklı türler var ve bu nedenle ortak bir türe dönüştürülemediğinden geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Değerlerin bir veya daha fazla türünü, tüm değerlerin ortak bir türe dönüştürülebilmesi için değiştirin.
