# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-optional"></a>' Optional ' olarak bildirilmeyen parametreler için varsayılan değerler verilemez

Yordam çağrıldığında gerekli bağımsız değişkenlerin sağlanması gerekir ve bildirimde varsayılan değerler almaz.  
  
 **Hata kimliği:** BC32024  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişkenini bildirin `Optional` veya varsayılan değeri kaldırın.
