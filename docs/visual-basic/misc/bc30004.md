# <a name="character-constant-must-contain-exactly-one-character"></a>Karakter sabiti tam olarak bir karakter içermelidir
Bir karakter sabiti, birden fazla karakter içerdiğinden veya null bir karakter (tüm bitler sıfır) olduğundan geçerli değil.  
  
 **Hata kimliği:** BC30004  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Karakter sabitini tek bir karakter olarak değiştirin veya null karakteri geçerli bir karakter olarak değiştirin.
