# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-filesystem-apis"></a>Dize uzunluğu, ' FileSystem ' API 'Leri için maksimum 32767 karakter uzunluğunu aşıyor
Bir dizenin uzunluğu 32767 karakterlik en fazla uzunluğu aşıyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dizeyi kısaltın.
