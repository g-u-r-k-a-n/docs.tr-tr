# <a name="end-sub-expected"></a>' End Sub ' bekleniyor

Bir `End` deyimin, sonlandırmakta olduğu blokla eşleşmesi gerekir. `Sub`Yordam tanımı düzgün sonlandırılmadı.  
  
 **Hata kimliği:** BC30026  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `End Sub`Yordamın sonuna bir ifade ekleyin.
