# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>' Error ' deyimleri Komut penceresinde geçerli değildir
`Error` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30724  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Error` deyimlerini hata ayıklama kodundan kaldırın.
