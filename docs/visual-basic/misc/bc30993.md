<span data-ttu-id="85d3d-104">Bir nesne başlatıcısı listesinde başlatılan Üyeler alan veya özellik olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="85d3d-104">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="85d3d-105">Ayrıca, Başlatıcı listesindeki Özellikler parametrelere sahip olamaz.</span><span class="sxs-lookup"><span data-stu-id="85d3d-105">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="85d3d-106">Bu hataya neden olan özellik aşırı yüklenmiş ve sürümlerinin her biri bağımsız değişken gerektiriyor.</span><span class="sxs-lookup"><span data-stu-id="85d3d-106">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="85d3d-107">Bu nedenle, özellik bir nesne başlatıcısı listesinde başlatılamaz.</span><span class="sxs-lookup"><span data-stu-id="85d3d-107">Therefore, the property cannot be initialized in an object initializer list.</span></span>

## <a name="example"></a><span data-ttu-id="85d3d-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="85d3d-111">Example</span></span>  

 <span data-ttu-id="85d3d-112">Aşağıdaki sınıf üç özellik tanımı içerir: için bir diğeri `TotalItems` ve `Item` olan.</span><span class="sxs-lookup"><span data-stu-id="85d3d-112">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```vb  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="85d3d-113">`TotalItems`Özelliği, aşağıdaki bildirimde gösterildiği gibi bir bağımsız değişken gerektirmez ve bir nesne başlatma listesinde başlatılabilir.</span><span class="sxs-lookup"><span data-stu-id="85d3d-113">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="85d3d-114">`Item`Özelliği aşırı yüklendi ve her aşırı yükleme bir bağımsız değişken gerektiriyor.</span><span class="sxs-lookup"><span data-stu-id="85d3d-114">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="85d3d-115">Bu nedenle, `Item` bir nesne başlatıcısı listesinde görünemez.</span><span class="sxs-lookup"><span data-stu-id="85d3d-115">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```vb  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="85d3d-116">Bu hatayı önlemek için, `Item` özelliği nesne Başlatıcısı dışında başlatın.</span><span class="sxs-lookup"><span data-stu-id="85d3d-116">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
