# <a name="compiler-error-cs1007"></a><span data-ttu-id="5933d-103">Derleyici hatası CS1007</span><span class="sxs-lookup"><span data-stu-id="5933d-103">Compiler Error CS1007</span></span>

 <span data-ttu-id="5933d-105">Bir [özellik](../programming-guide/classes-and-structs/using-properties.md)bildirdiğinizde erişimci yöntemlerini de bildirmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="5933d-105">When you declare a [property](../programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="5933d-106">Ancak, bir özelliğin birden fazla `get` erişimci yöntemi veya birden fazla `set` erişimci yöntemi olamaz.</span><span class="sxs-lookup"><span data-stu-id="5933d-106">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5933d-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="5933d-107">Example</span></span>  

 <span data-ttu-id="5933d-108">Aşağıdaki örnek CS1007 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5933d-108">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs

    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
