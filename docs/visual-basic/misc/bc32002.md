# <a name="keyword-is-valid-only-within-a-class"></a>' \<keyword> ' yalnızca bir sınıf içinde geçerlidir

Veya gibi sınıflarla ilişkili bir anahtar sözcük, `Me` `MyClass` sınıf tanımının dışında kullanılır.  
  
 **Hata kimliği:** BC32002  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Anahtar sözcüğü kullanan kod sınıf örnekleri içeriyorsa, sınıf uygulamasına taşıyın.  
  
- Anahtar sözcüğü kullanan kod sınıflar için geçerli değilse, geçersiz anahtar sözcüğü kaldırın.
