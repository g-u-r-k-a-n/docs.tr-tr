2. <span data-ttu-id="5ac34-115">Analitik izlemeyi etkinleştirmediyseniz, **uygulamalar ve hizmetler günlükleri**, **Microsoft**, **Windows**, **uygulama sunucusu-uygulamalar**' ı genişletin.</span><span class="sxs-lookup"><span data-stu-id="5ac34-115">If you haven’t enabled analytic tracing, expand **Applications and Services Logs**, **Microsoft**, **Windows**, **Application Server-Applications**.</span></span> <span data-ttu-id="5ac34-116">**Görünüm**, **analitik ve hata ayıklama günlüklerini göster**' i seçin.</span><span class="sxs-lookup"><span data-stu-id="5ac34-116">Select **View**, **Show Analytic and Debug Logs**.</span></span> <span data-ttu-id="5ac34-117">**Analitik** öğesine sağ tıklayın ve **günlüğü etkinleştir**' i seçin.</span><span class="sxs-lookup"><span data-stu-id="5ac34-117">Right-click **Analytic** and select **Enable Log**.</span></span> <span data-ttu-id="5ac34-118">İzlemelerin görüntülenebilmesi için Olay Görüntüleyicisi açık bırakın.</span><span class="sxs-lookup"><span data-stu-id="5ac34-118">Leave Event Viewer open so that traces can be viewed.</span></span>

4. <span data-ttu-id="5ac34-122">**Hizmet** projesine sağ tıklayın ve **Ekle**, **Yeni öğe**' yi seçin.</span><span class="sxs-lookup"><span data-stu-id="5ac34-122">Right-click the **Service** project and select **Add**, **New Item**.</span></span> <span data-ttu-id="5ac34-123">**Uygulama yapılandırma dosyası** ' nı seçin ve **Tamam**' ı tıklatın.</span><span class="sxs-lookup"><span data-stu-id="5ac34-123">Select **Application Configuration File** and click **OK**.</span></span>

    ```xml
    </diagnostics>
    ```

9. <span data-ttu-id="5ac34-129">İstemci projesindeki program.cs dosyasındaki Main yönteminde, Izleme GUID 'sini olay günlüğüne yayılacağı şekilde ayarlayın.</span><span class="sxs-lookup"><span data-stu-id="5ac34-129">In the Main method in the program.cs file in the client project, set the Trace GUID to be propagated in the event log.</span></span>

    ```csharp
    Trace.CorrelationManager.ActivityId = guid;
    ```

10. <span data-ttu-id="5ac34-130">**Analitik** günlüğü yenileyin ve inceleyin.</span><span class="sxs-lookup"><span data-stu-id="5ac34-130">Refresh and examine the **Analytic**  log.</span></span>  <span data-ttu-id="5ac34-131">Olay KIMLIĞI 220 olan bir olay arayın.</span><span class="sxs-lookup"><span data-stu-id="5ac34-131">Look for an event with Event ID 220.</span></span>  <span data-ttu-id="5ac34-132">Olayı seçin ve Önizleme bölmesindeki **Ayrıntılar** sekmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="5ac34-132">Select the event, and click the **Details** tab in the preview pane.</span></span> <span data-ttu-id="5ac34-133">Bu olay çağıran etkinliğin bağıntı KIMLIĞINI içerir.</span><span class="sxs-lookup"><span data-stu-id="5ac34-133">This event will contain the correlation ID for the calling activity.</span></span>

    > [!NOTE]

    ```xml
        <System>

            ...
            <Correlation ActivityID="{A066CCF1-8AB3-459B-B62F-F79F957A5036}" RelatedActivityID="{85FC0930-9C49-42DA-804B-A7368104BD1B}" />

       </System>
    ```

    > [!NOTE]
    > <span data-ttu-id="5ac34-140">Aktarım olayı, etkin ActivityId 'nin değişikliğini ActivityId olarak belirtilen GUID 'den relatedActivityId olarak belirtilen GUID 'ye kaydeder.</span><span class="sxs-lookup"><span data-stu-id="5ac34-140">The transfer event records the change of the active ActivityID from the GUID specified as the ActivityID to the GUID specified as RelatedActivityID.</span></span> <span data-ttu-id="5ac34-141">Aktarım olayı oluşturulduktan sonra, tüm olaylar ActivityId olarak yeni GUID 'YI içerir.</span><span class="sxs-lookup"><span data-stu-id="5ac34-141">After the transfer event is emitted, all events will contain the new GUID as the ActivityID.</span></span>
