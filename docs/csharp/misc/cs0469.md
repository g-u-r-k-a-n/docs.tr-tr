# <a name="compiler-warning-level-2-cs0469"></a>Derleyici Uyarısı (düzey 2) CS0469

' Goto case ' değeri ' Type ' türüne örtük olarak dönüştürülebilir değil  
  
 Kullandığınızda `goto case` , goto case değerinden anahtar türüne örtük bir dönüştürme olmalıdır.  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS0469 oluşturur.  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
