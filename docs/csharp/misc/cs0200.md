## <a name="examples"></a><span data-ttu-id="43efb-108">Örnekler</span><span class="sxs-lookup"><span data-stu-id="43efb-108">Examples</span></span>

<span data-ttu-id="43efb-109">Aşağıdaki örnek CS0200 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="43efb-109">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="43efb-110">Aşağıdaki örnek [Otomatik uygulanan özellikleri](../programming-guide/classes-and-structs/auto-implemented-properties.md) ve [nesne başlatıcıları](../programming-guide/classes-and-structs/object-and-collection-initializers.md) KULLANıR ve yine de CS0200 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="43efb-110">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md) and [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md) and still generates CS0200:</span></span>

```csharp
    int I
    {
        get;

    }

    {

        };

```

<span data-ttu-id="43efb-111">Salt okunurdur ' Property ' özelliğine veya dizin oluşturucusuna atamak için bir küme erişimcisi ekleyin veya nesne oluşturucusunda değeri atayın.</span><span class="sxs-lookup"><span data-stu-id="43efb-111">To assign to a property or indexer 'property' that's read-only, add a set accessor or assign the value in the object's constructor.</span></span>
