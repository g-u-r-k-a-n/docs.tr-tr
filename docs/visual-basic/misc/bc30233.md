# <a name="specifier-is-not-valid-on-a-constant-declaration"></a>' \<specifier> ' Sabit bildiriminde geçerli değil

Bir `Const` ifade geçersiz bir anahtar sözcük içeriyor. Bir `Const` bildirimde yalnızca,,, `Friend` `Private` `Protected` `Public` veya `Shadows` anahtar sözcükler bulunabilir.  
  
 **Hata kimliği:** BC30233  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Anahtar sözcüğünün doğru yazıldığından emin olun.  
  
2. Deyimden geçersiz anahtar sözcüğü Kaldır `Const` .
