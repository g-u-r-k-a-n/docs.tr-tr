# <a name="compiler-error-cs0103"></a><span data-ttu-id="c01b5-103">Derleyici Hatası CS0103</span><span class="sxs-lookup"><span data-stu-id="c01b5-103">Compiler Error CS0103</span></span>

<span data-ttu-id="c01b5-104">' Tanımlayıcı ' adı geçerli bağlamda yok</span><span class="sxs-lookup"><span data-stu-id="c01b5-104">The name 'identifier' does not exist in the current context</span></span>

 <span data-ttu-id="c01b5-105">Sınıf, [ad alanı](../keywords/namespace.md)veya kapsamda bulunmayan bir adı kullanma girişiminde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="c01b5-105">An attempt was made to use a name that does not exist in the class, [namespace](../keywords/namespace.md), or scope.</span></span> <span data-ttu-id="c01b5-106">Kullanmaya çalıştığınız adın kullanılabilir olduğundan emin olmak için adın yazımını denetleyin ve kullandığınız yönergeleri ve derleme başvurularını denetleyin.</span><span class="sxs-lookup"><span data-stu-id="c01b5-106">Check the spelling of the name and check your using directives and assembly references to make sure that the name that you are trying to use is available.</span></span>

 <span data-ttu-id="c01b5-107">Bu hata, bir döngü veya bir ya da blok içinde bir değişken bildirirseniz `try` `if` ve ardından bunu kapsayan bir kod bloğundan veya ayrı bir kod bloğundan erişmeyi denerseniz, aşağıdaki örnekte gösterildiği gibi sıklıkla meydana gelir:</span><span class="sxs-lookup"><span data-stu-id="c01b5-107">This error frequently occurs if you declare a variable in a loop or a `try` or `if` block and then attempt to access it from an enclosing code block or a separate code block, as shown in the following example:</span></span>

```csharp
using System;

class MyClass1
{
    public static void Main()
    {
        try
        {
            // The following declaration is only available inside the try block.
            var conn = new MyClass1();
        }
        catch (Exception e)
        {  
            // The following expression causes error CS0103, because variable
            // conn only exists in the try block.
            if (conn != null)
                Console.WriteLine("{0}", e);
        }
    }
}
```

 <span data-ttu-id="c01b5-108">Aşağıdaki örnek, hatayı çözer:</span><span class="sxs-lookup"><span data-stu-id="c01b5-108">The following example resolves the error:</span></span>

```csharp
using System;

class MyClass2
{
    public static void Main()
    {
        // To resolve the error in the example, the first step is to
        // move the declaration of conn out of the try block. The following
        // declaration is available throughout the Main method.
        MyClass2 conn = null;
        try
        {
            // Inside the try block, use the conn variable that you declared
            // previously.
            conn = new MyClass2();
        }
        catch (Exception e)
        {
            // The following expression no longer causes an error, because
            // the declaration of conn is in scope.
            if (conn != null)
                Console.WriteLine("{0}", e);
        }
    }
}
```
