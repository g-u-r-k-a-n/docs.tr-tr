# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="c4886-103">Derleyici Uyarısı (düzey 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="c4886-103">Compiler warning (level 1) CS0612</span></span>

<span data-ttu-id="c4886-104">' üye ' artık kullanılmıyor</span><span class="sxs-lookup"><span data-stu-id="c4886-104">'member' is obsolete</span></span>  
  
 <span data-ttu-id="c4886-105">Sınıf Tasarımcısı, [eski özniteliğiyle](../language-reference/attributes/general.md#obsolete-attribute)bir üye işaretledi.</span><span class="sxs-lookup"><span data-stu-id="c4886-105">The class designer marked a member with the [Obsolete attribute](../language-reference/attributes/general.md#obsolete-attribute).</span></span> <span data-ttu-id="c4886-106">Bu, üyenin sınıfın gelecekteki bir sürümünde desteklenmeyebilir anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="c4886-106">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="c4886-107">Aşağıdaki örnek, eski bir üyenin nasıl CS0612 üretmediğini göstermektedir:</span><span class="sxs-lookup"><span data-stu-id="c4886-107">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
