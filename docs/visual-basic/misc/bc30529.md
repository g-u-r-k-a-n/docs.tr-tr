# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Varsa, tüm parametrelerin açıkça yazılması gerekir

Bir işlevde tür bildirimlerinin farklı türlerini karıştırın.  
  
 **Hata kimliği:** BC30529  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm parametrelerin açıkça yazıldığından emin olun.
