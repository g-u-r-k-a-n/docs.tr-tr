## <a name="example"></a><span data-ttu-id="a1866-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="a1866-110">Example</span></span>  
 <span data-ttu-id="a1866-111">Aşağıdaki örnekteki ilk parametre, `Print` yöntemin veri türünü genişlettiğini gösterir `String` .</span><span class="sxs-lookup"><span data-stu-id="a1866-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```vb  
<Extension()> _  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="a1866-112">Uzantı yöntemi aşağıdaki gibi çağrıldığında, `str` yöntemindeki parametre öğesine bağlanır `greeting` , `String` Bu çağrının örneğidir `Print` .</span><span class="sxs-lookup"><span data-stu-id="a1866-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="a1866-113">Derleyici, `greeting` genişletme yöntemi için bağımsız değişken olarak kullanacaktır `Print` .</span><span class="sxs-lookup"><span data-stu-id="a1866-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

```vb  
greeting.Print()  
```
