# <a name="label-labelname-is-already-defined-in-the-current-methodmultiline-lambda"></a>' \<labelname> ' Etiketi, geçerli metotta/çok satırlı lambda içinde zaten tanımlanmış

Etiket adı yordam içinde yinelenir. Bir yordamın içindeki tüm adlar benzersiz olmalıdır.  
  
 **Hata kimliği:** BC30094  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Yinelenen etiketi yeniden adlandırın.
