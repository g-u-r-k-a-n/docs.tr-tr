# <a name="file-not-found-visual-basic-run-time-error"></a>Dosya bulunamadı (Visual Basic Çalışma Süresi Hatası)
Dosya belirtilen yerde bulunamadı. Hatanın olası nedenleri şunlardır:  
  
- Bir ifade, varolmayan bir dosyaya başvurur.  
  
- Dinamik bağlantı kitaplığındaki (DLL) bir yordamı çağırma girişiminde bulunuldu, ancak `Lib` deyimin yan tümcesinde belirtilen kitaplık `Declare` bulunamıyor.  
  
- Bir projeyi açmaya veya var olmayan bir metin dosyasını yüklemeye çalıştınız.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosya adı ve yol belirtiminin yazımını denetleyin.
