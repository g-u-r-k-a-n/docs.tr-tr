# <a name="the-moduleassemblyname-option-may-only-be-specified-when-building-a-target-of-type-module"></a>/Moduleassemblyname seçeneği yalnızca ' Module ' türünde bir hedef oluşturulurken belirtilebilir
@No__t_1 seçeneği `module` dışında bir değere ayarlandığında Visual Basic Derleyicisi `/moduleassemblyname` derleyici seçeneğini geçirdi.  
  
 **Hata kimliği:** BC2030  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. @No__t_0 derleyici seçeneğini kaldırın veya `/target` seçeneğini `module` olarak ayarlayın.
