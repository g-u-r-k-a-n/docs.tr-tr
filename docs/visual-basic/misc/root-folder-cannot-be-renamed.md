# <a name="root-folder-cannot-be-renamed"></a>Kök klasör yeniden adlandırılamıyor
Kök dizinin yeniden adlandırılması için bir girişimde bulunuldu. Bu, yanlış ayrıştırılmış dosya yolunun bir sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Doğru olduğundan emin olmak için dosya yolunu denetleyin.
