# <a name="xml-dtds-are-not-supported"></a>XML DTD 'Leri desteklenmez

XML sabit değeri olarak bir belge türü tanımı (DTD) belirtildi.  
  
 **Hata kimliği:** BC31175  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- DTD 'yi kaldırın. XML belgesi şemasını belirtmek için XML şema tanımı (XSD) kullanın.
