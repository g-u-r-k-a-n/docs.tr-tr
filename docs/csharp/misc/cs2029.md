# <a name="compiler-warning-level-1-cs2029"></a>Derleyici Uyarısı (düzey 1) CS2029
'/Define ' için geçersiz değer; ' Identifier ' geçerli bir tanımlayıcı değil  
  
 [-Define](../language-reference/compiler-options/define-compiler-option.md) seçeneğinde kullanılan değer bazı geçersiz karakterler içeriyorsa bu uyarı oluşur.  
  
 Bu uyarı [-nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) seçeneği tarafından bastırılamaz.
