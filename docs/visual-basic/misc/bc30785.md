# <a name="parameter-specifier-is-duplicated"></a>Parametre belirleyicisi yineleniyor

Parametre belirleyicisi parametreye birden çok kez uygulanır.  
  
 **Hata kimliği:** BC30785  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yinelenen belirticileri kaldırın.
