# <a name="typename-membername-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>\<typename>\<membername>Bu ada sahip başka bir üye ' Shadows ' olarak bildirildiği için ' ' ' Shadows ' olarak bildirilmelidir

Bu özellik veya yordam, değiştiriciyle işaretlenen başka bir özellik veya yordamla aynı ada sahip `Shadows` .  
  
 **Hata kimliği:** BC30695  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştiricisini kullanın `Shadows` .
