# <a name="baudrate-must-be-greater-than-0"></a>Baud hızı 0 ' dan büyük olmalıdır

`BaudRate`Yöntem için belirtilen bağımsız değişken `My.Computer.Ports.OpenSerialPort` sıfırdan büyük olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `BaudRate`Bağımsız değişkenin değerini pozitif bir sayı olarak değiştirin.
