# <a name="compiler-error-cs0214"></a><span data-ttu-id="da541-103">Derleyici hatası CS0214</span><span class="sxs-lookup"><span data-stu-id="da541-103">Compiler Error CS0214</span></span>

<span data-ttu-id="da541-104">İşaretçiler ve sabit boyutlu arabellekler yalnızca güvenli olmayan bir bağlamda kullanılabilir</span><span class="sxs-lookup"><span data-stu-id="da541-104">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="da541-105">İşaretçiler yalnızca [unsafe](../language-reference/keywords/unsafe.md) anahtar sözcüğüyle birlikte kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="da541-105">Pointers can only be used with the [unsafe](../language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="da541-106">Daha fazla bilgi için bkz. [güvenli olmayan kod ve işaretçiler](../language-reference/unsafe-code.md).</span><span class="sxs-lookup"><span data-stu-id="da541-106">For more information, see [Unsafe Code and Pointers](../language-reference/unsafe-code.md).</span></span>  
  
 <span data-ttu-id="da541-107">Aşağıdaki örnek CS0214 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="da541-107">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
