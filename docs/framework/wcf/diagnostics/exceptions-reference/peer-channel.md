# <a name="peer-channel"></a>Eş Kanal

Bu konu, Windows Communication Foundation (WCF) eş kanalı tarafından oluşturulan tüm özel durumları listeler.  
  
## <a name="exception-list"></a>Özel durum listesi  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|InsufficientCredentials|Belirtilen kimlik bilgileri, istenen işlemi yürütmek için yeterli değil. Lütfen belirtilen işlem için geçerli bir değer belirtin|  
|Invalidresolvermode|Belirtilen PeerResolverMode değeri geçersiz. PeerResolveMode. Auto, Default veya PNRP seçeneklerinden birini belirtin.|  
|Peernodedurdurulan|PeerNode sonlandırıldığı için açılamıyor.|  
|Peernullregistrationınfo|Kayıt bilgileri null olamaz. Lütfen Register işleminin geçerli bir RegistrationInfo nesnesi ile çağrıldığından emin olun.|  
|Peernullresolveınfo|Bilgi çözümleme olamaz `null` . Lütfen Resolve işleminin geçerli bir ResolveInfo nesnesiyle çağrıldığından emin olun.|
