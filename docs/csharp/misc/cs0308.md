# <a name="compiler-error-cs0308"></a><span data-ttu-id="be357-103">Derleyici hatası CS0308</span><span class="sxs-lookup"><span data-stu-id="be357-103">Compiler Error CS0308</span></span>

<span data-ttu-id="be357-104">Genel olmayan tür-veya-yöntemi ' tanımlayıcı ' tür bağımsız değişkenleriyle kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="be357-104">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="be357-105">Yöntem veya tür genel değil, ancak tür bağımsız değişkenleriyle kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="be357-105">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="be357-106">Bu hatayı önlemek için, açılı ayraçları ve tür bağımsız değişkenlerini kaldırın ya da yöntemi veya türü genel bir yöntem veya tür olarak yeniden bildirin.</span><span class="sxs-lookup"><span data-stu-id="be357-106">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="be357-107">Aşağıdaki örnek CS0308 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="be357-107">The following example generates CS0308:</span></span>  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="be357-108">Aşağıdaki örnek ayrıca CS0308 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="be357-108">The following example also generates CS0308.</span></span> <span data-ttu-id="be357-109">Hatayı gidermek için, "System. Collections. Generic" yönergesini kullanın.</span><span class="sxs-lookup"><span data-stu-id="be357-109">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
