# <a name="compiler-error-cs0031"></a><span data-ttu-id="45927-102">Derleyici hatası CS0031</span><span class="sxs-lookup"><span data-stu-id="45927-102">Compiler Error CS0031</span></span>

<span data-ttu-id="45927-103">' Value ' sabit değeri ' Type ' olarak dönüştürülemez.</span><span class="sxs-lookup"><span data-stu-id="45927-103">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="45927-104">Türü, değeri depolayabilen bir değişkene bir değer atamaya çalışıldı.</span><span class="sxs-lookup"><span data-stu-id="45927-104">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="45927-105">Daha fazla bilgi için bkz. [türler](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="45927-105">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="45927-106">Aşağıdaki örnek hem işaretli hem de işaretlenmemiş bağlamlarda CS0031 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="45927-106">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
