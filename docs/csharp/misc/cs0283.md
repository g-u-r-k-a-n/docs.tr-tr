# <a name="compiler-error-cs0283"></a>Derleyici hatası CS0283
' Type ' türü const olarak bildirilemez  
  
[Sabit](../language-reference/keywords/const.md) bildiriminde belirtilen tür `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, bir [sabit listesi](../language-reference/builtin-types/enum.md) türü veya `null`değeri atanan bir başvuru türü olmalıdır. Her sabit ifade, hedef türünde bir değer veya hedef türüne örtük olarak dönüştürülebilir bir tür değeri vermelidir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0283 oluşturur.  
  
```csharp  
// CS0283.cs  
struct MyTest  
{  
}  
class MyClass   
{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.  
    const MyTest test = new MyTest();   // CS0283  
  
    public static int Main() {  
        return 1;  
    }  
}  
```
