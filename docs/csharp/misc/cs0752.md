# <a name="compiler-error-cs0752"></a><span data-ttu-id="bc0af-103">Derleyici hatası CS0752</span><span class="sxs-lookup"><span data-stu-id="bc0af-103">Compiler Error CS0752</span></span>
<span data-ttu-id="bc0af-104">Kısmi bir yöntemin out parametreleri olamaz</span><span class="sxs-lookup"><span data-stu-id="bc0af-104">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="bc0af-105">Kısmi bir yöntemin [Out](../language-reference/keywords/out-parameter-modifier.md) parametresi olamaz.</span><span class="sxs-lookup"><span data-stu-id="bc0af-105">A partial method cannot have an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="bc0af-106">Kısmi yöntem derleyici tarafından kaldırılacağından Out parametrelerine izin verilmez, aksi takdirde Out parametresinin atandığı garanti yoktur.</span><span class="sxs-lookup"><span data-stu-id="bc0af-106">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bc0af-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="bc0af-107">To correct this error</span></span>
