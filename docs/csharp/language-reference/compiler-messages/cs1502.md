# <a name="compiler-error-cs1502"></a><span data-ttu-id="74726-102">Derleyici Hatası CS1502</span><span class="sxs-lookup"><span data-stu-id="74726-102">Compiler Error CS1502</span></span>
<span data-ttu-id="74726-103">'Ad' için en iyi aşırı yüklenen yöntem eşleşmesi bazı geçersiz bağımsız değişkenleri vardır</span><span class="sxs-lookup"><span data-stu-id="74726-103">The best overloaded method match for 'name' has some invalid arguments</span></span>  
  
 <span data-ttu-id="74726-104">Yönteme geçirilen bağımsız değişken türleri bu yöntemin parametre türleri yle eşleşmediğinde bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="74726-104">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="74726-105">Çağrılan yöntem aşırı yüklenmişse, aşırı yüklenen sürümlerin hiçbirinde bağımsız değişken türleri ile eşleşen bir imza yoktur.</span><span class="sxs-lookup"><span data-stu-id="74726-105">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="74726-106">Bu sorunu gidermek için aşağıdakilerden birini yapın:</span><span class="sxs-lookup"><span data-stu-id="74726-106">To resolve this problem, do one of the following:</span></span>  
  
- <span data-ttu-id="74726-107">Geçirilen bağımsız değişken türlerini iki kez denetleyin.</span><span class="sxs-lookup"><span data-stu-id="74726-107">Double-check the types of the arguments being passed.</span></span> <span data-ttu-id="74726-108">Çağrılan yöntemin bağımsız değişkenlerini eşleştirdiklerinden emin olun.</span><span class="sxs-lookup"><span data-stu-id="74726-108">Make sure that they match the arguments of the method being called.</span></span>  
  
- <span data-ttu-id="74726-109">Uygunsa, sınıfı kullanarak eşleşmemiş <xref:System.Convert> parametreleri dönüştürün.</span><span class="sxs-lookup"><span data-stu-id="74726-109">If appropriate, convert any mismatched parameters using the <xref:System.Convert> class.</span></span>  
  
- <span data-ttu-id="74726-110">Uygunsa, yöntemin beklediği türle eşleşecek şekilde eşleşmemiş parametreleri atın.</span><span class="sxs-lookup"><span data-stu-id="74726-110">If appropriate, cast any mismatched parameters to match the type that the method is expecting.</span></span>  
  
- <span data-ttu-id="74726-111">Uygunsa, gönderilen parametre türleri yle eşleşecek şekilde yöntemin başka bir aşırı yüklenmiş sürümünü tanımlayın.</span><span class="sxs-lookup"><span data-stu-id="74726-111">If appropriate, define another overloaded version of the method to match the parameter types that are being sent.</span></span>

   public class a  
   {  
      public a(char i)

      // public a(int i)

         a aa = new a(2222);   // CS1502  
      }  
   }
