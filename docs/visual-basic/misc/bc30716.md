# <a name="finally-statements-are-not-valid-in-the-immediate-window"></a>' Finally ' deyimleri Komut penceresinde geçerli değildir

`Finally` deyimler yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30716  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Finally`Hata ayıklama kodundaki deyimleri kaldır.
