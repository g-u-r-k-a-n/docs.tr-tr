# <a name="compiler-error-cs0752"></a><span data-ttu-id="5de1d-103">Derleyici hatası CS0752</span><span class="sxs-lookup"><span data-stu-id="5de1d-103">Compiler Error CS0752</span></span>

<span data-ttu-id="5de1d-104">Kısmi bir yöntemin out parametreleri olamaz</span><span class="sxs-lookup"><span data-stu-id="5de1d-104">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="5de1d-105">Kısmi bir yöntemin [Out](../language-reference/keywords/out-parameter-modifier.md) parametresi olamaz.</span><span class="sxs-lookup"><span data-stu-id="5de1d-105">A partial method cannot have an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="5de1d-106">Kısmi yöntem derleyici tarafından kaldırılacağından Out parametrelerine izin verilmez, aksi takdirde Out parametresinin atandığı garanti yoktur.</span><span class="sxs-lookup"><span data-stu-id="5de1d-106">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5de1d-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="5de1d-107">To correct this error</span></span>  
  
1. <span data-ttu-id="5de1d-108">Parametresinden Out değiştiricisini kaldırın ve bunun yerine yöntemin dönüş değerini kullanın veya yöntem bildiriminden kısmi değiştiricisini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="5de1d-108">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5de1d-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="5de1d-109">Example</span></span>  

 <span data-ttu-id="5de1d-110">Aşağıdaki kod CS0752 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5de1d-110">The following code generates CS0752:</span></span>

```csharp  
// cs0752.cs

    partial void Part(out int num); // CS0752  
    // try the following line instead

        return 1;  
    }  
}  
```
