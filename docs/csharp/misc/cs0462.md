# <a name="compiler-error-cs0462"></a><span data-ttu-id="bbb39-103">Derleyici hatası CS0462</span><span class="sxs-lookup"><span data-stu-id="bbb39-103">Compiler Error CS0462</span></span>

<span data-ttu-id="bbb39-104">' Member1 ' ve ' member2 ' devralınan üyeleri ' Type ' türünde aynı imzaya sahip ve bu nedenle geçersiz kılınamazlar</span><span class="sxs-lookup"><span data-stu-id="bbb39-104">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="bbb39-105">Bu hata, genel türler kullanıma sunulmasıyla birlikte oluşur.</span><span class="sxs-lookup"><span data-stu-id="bbb39-105">This error arises with the introduction of generics.</span></span> <span data-ttu-id="bbb39-106">Normalde, aynı imzaya sahip bir sınıftaki yöntemin iki sürümüne sahip olamaz.</span><span class="sxs-lookup"><span data-stu-id="bbb39-106">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="bbb39-107">Ancak, genel türler ile, belirli bir türle örneği oluşturulmuş ise başka bir yöntemi yinegerekebilecek bir genel yöntem belirtebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="bbb39-107">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bbb39-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="bbb39-108">Example</span></span>  

 <span data-ttu-id="bbb39-109">`C<int>`Örneği oluşturulduğunda, yöntemin iki sürümü `F` aynı imzayla oluşturulur, bu nedenle sınıftaki geçersiz kılma, `D` geçersiz kılmanın uygulanacağı bir karar alamaz.</span><span class="sxs-lookup"><span data-stu-id="bbb39-109">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="bbb39-110">Aşağıdaki örnek CS0462 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="bbb39-110">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
