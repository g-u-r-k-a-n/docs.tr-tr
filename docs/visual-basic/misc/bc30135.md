# <a name="synclock-statements-are-not-valid-in-the-immediate-window"></a>' SyncLock ' deyimleri Komut penceresinde geçerli değildir

`SyncLock`İfade, iş parçacıklarını eşitler ve bir hata ayıklama bağlamında buna izin verilmez.  
  
 **Hata kimliği:** BC30135  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `SyncLock` **Komut** penceresinde bir ifade vermeyin.
