# <a name="creating-event-handlers-in-windows-forms"></a>Windows Forms'ta Olay İşleyicileri Oluşturma

Olay işleyicisi, bir olay gerçekleştiğinde, kullanıcının bir düğmeye tıkladığı veya ileti sırasının bir ileti aldığı durumlarda olduğu gibi, hangi eylemlerin gerçekleştirileceğini belirleyen kodunuzda bir yordamdır. Bir olay ortaya çıktığında olayı alan olay işleyicisi veya işleyiciler yürütülür. Olaylar birden çok işleyicilere atanabilir ve belirli olayları işleyen yöntemler dinamik olarak değiştirilebilir. Olay işleyicileri oluşturmak için Visual Studio 'daki Windows Form Tasarımcısı de kullanabilirsiniz.
