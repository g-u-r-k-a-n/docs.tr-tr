## <a name="example"></a><span data-ttu-id="acc33-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="acc33-109">Example</span></span>  

 <span data-ttu-id="acc33-110">Aşağıdaki örnek CS0403 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="acc33-110">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
