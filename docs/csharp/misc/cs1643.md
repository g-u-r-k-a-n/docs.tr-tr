 <span data-ttu-id="5246c-105">Bu hata, bir temsilci gövdesinin Return ifadesine sahip olmaması veya derleyicinin doğrulayamadığını belirten bir return bildirimine sahip olması durumunda meydana gelir.</span><span class="sxs-lookup"><span data-stu-id="5246c-105">This error occurs if a delegate body does not have a return statement, or has a return statement that the compiler is unable to verify will be reached.</span></span> <span data-ttu-id="5246c-106">Aşağıdaki örnekte, derleyici, anonim yöntem bloğunun her zaman bir değer döndürdüğünü doğrulamak için dallanma koşulunun sonucunu tahmin etmeye çalışmaz.</span><span class="sxs-lookup"><span data-stu-id="5246c-106">In the example below, the compiler does not attempt to predict the result of the branching condition in order to verify that the anonymous method block always returns a value.</span></span>
