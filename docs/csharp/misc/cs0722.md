# <a name="compiler-error-cs0722"></a><span data-ttu-id="691cb-103">Derleyici hatası CS0722</span><span class="sxs-lookup"><span data-stu-id="691cb-103">Compiler Error CS0722</span></span>
<span data-ttu-id="691cb-104">' Type ': statik türler dönüş türleri olarak kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="691cb-104">'type': static types cannot be used as return types</span></span>  
  
 <span data-ttu-id="691cb-105">Statik türlerin örnekleri oluşturulemediğinden, dönüş türü olarak statik bir tür anlamlı değildir.</span><span class="sxs-lookup"><span data-stu-id="691cb-105">A static type as a return type is not meaningful since instances of static types cannot be created.</span></span>  
  
 <span data-ttu-id="691cb-106">Aşağıdaki örnek CS0722 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="691cb-106">The following sample generates CS0722:</span></span>  
  
```csharp  
// CS0722.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public SC F()  // CS0722  
   {  
      return null;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
