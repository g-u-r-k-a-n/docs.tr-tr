# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>'\<TypeName > ' türü tanımlı değil veya kendisini içeren modül hata ayıklama oturumunda yüklü değil
Kullanılabilir olmayan veya tanımlı olmayan bir türe başvuruldu.  
  
 **Hata kimliği:** BC30743  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Türün adının doğru yazıldığından emin olun.  
  
2. Türü geçerli sınıf veya modül içinde tanımlayın.
