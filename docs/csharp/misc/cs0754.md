# <a name="compiler-error-cs0754"></a><span data-ttu-id="3d050-103">Derleyici hatası CS0754</span><span class="sxs-lookup"><span data-stu-id="3d050-103">Compiler Error CS0754</span></span>

 <span data-ttu-id="3d050-105">Kısmi bir yöntem, bir arabirimde tanımlanmış bir metodun açık bir uygulama olarak bildirilemez.</span><span class="sxs-lookup"><span data-stu-id="3d050-105">A partial method cannot be declared as an explicit implementation of a method defined in an interface.</span></span>

## <a name="example"></a><span data-ttu-id="3d050-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="3d050-108">Example</span></span>  

 <span data-ttu-id="3d050-109">Aşağıdaki kod CS0754 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3d050-109">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
