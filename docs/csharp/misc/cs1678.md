# <a name="compiler-error-cs1678"></a><span data-ttu-id="2663d-102">Derleyici hatası CS1678</span><span class="sxs-lookup"><span data-stu-id="2663d-102">Compiler Error CS1678</span></span>

<span data-ttu-id="2663d-103">' Number ' parametresi ' type1 ' türü olarak bildiriliyor, ancak ' type2 ' olmalıdır</span><span class="sxs-lookup"><span data-stu-id="2663d-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="2663d-104">Bu hata, anonim bir yöntemde parametre türü, yöntemi atamak istediğiniz temsilcinin bildiriminden farklı olduğunda oluşur.</span><span class="sxs-lookup"><span data-stu-id="2663d-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="2663d-105">Aşağıdaki örnek CS1678 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2663d-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors
{  
   static void Main()
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
