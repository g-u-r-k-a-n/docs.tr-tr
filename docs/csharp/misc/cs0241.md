## <a name="example"></a><span data-ttu-id="974fa-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="974fa-108">Example</span></span>  
 <span data-ttu-id="974fa-109">Aşağıdaki örnek CS0241 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="974fa-109">The following sample generates CS0241.</span></span> <span data-ttu-id="974fa-110">Buna ek olarak, örnek, varsayılan bağımsız değişkenlerle bir yöntemi aşırı yükleme ile nasıl benzeten olduğunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="974fa-110">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {
      B x = new B();  
      x.Test();  
   }  
}  
```
