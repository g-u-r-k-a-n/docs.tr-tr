# <a name="lambda-expression-cannot-be-converted-to-typename-because-typename-is-not-a-delegate-type"></a><span data-ttu-id="a853b-102">\<typename>' \<typename> ' Bir temsilci türü olmadığından lambda ifadesi ' ' olarak dönüştürülemez</span><span class="sxs-lookup"><span data-stu-id="a853b-102">Lambda expression cannot be converted to '\<typename>' because '\<typename>' is not a delegate type</span></span>
<span data-ttu-id="a853b-103">Lambda ifadeleri, temsilcilerin geçerli olduğu yerde kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="a853b-103">Lambda expressions can be used where delegates are valid.</span></span> <span data-ttu-id="a853b-104">Bunlar, uyumlu temsilci türlerine dönüştürülebilirler, ancak başka hiçbir tür için kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="a853b-104">They can be converted to compatible delegate types, but not to any other type.</span></span> <span data-ttu-id="a853b-105">Örneğin, bir temsilci türü tanımlayabilir ve buna bir lambda ifadesi atayabilir ya da bir parametreye bağımsız değişken olarak bir lambda ifadesi gönderebilirsiniz <xref:System.Func%601> .</span><span class="sxs-lookup"><span data-stu-id="a853b-105">For example, you can define a delegate type and assign a lambda expression to it, or send a lambda expression as the argument to a <xref:System.Func%601> parameter.</span></span> <span data-ttu-id="a853b-106">Bu örnekler aşağıdaki kodda gösterilmiştir.</span><span class="sxs-lookup"><span data-stu-id="a853b-106">These examples are shown in the following code.</span></span>  
  
```vb  
Module Module1  
  
    Delegate Function FunDel(ByVal m As Integer) As Boolean  
  
    Sub Main()  
  
        ' Assign a lambda expression to a function delegate.  
        Dim negative As FunDel = Function(n As Integer) n < 0  
        Console.WriteLine(negative(-3))  
  
        ' Send a lambda as the argument to a delegate parameter.  
        Dim numbers() As Integer = {3, 4, 2, 8, 1, 0, 9, 13, 42}  
        Dim evens = numbers.Where(Function(n) n Mod 2 = 0)  
  
        For Each even In evens  
            Console.WriteLine(even)  
        Next  
  
    End Sub  
  
End Module  
```
