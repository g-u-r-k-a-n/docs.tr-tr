# <a name="compiler-error-cs0268"></a>Derleyici hatası CS0268

İçeri aktarılan ' Type ' türü geçersiz. Döngüsel temel sınıf bağımlılığı içerir.  
  
 Başka bir dilden içeri aktarılan bir türün döngüsel temel sınıf bağımlılığı varsa bu hata oluşur. Böyle bir tür bir C# programında kullanılamaz. Bu hatayı çözmek için, başvurulan derlemelerin veya modüllerdeki diğer dillerden içeri aktarılan türleri kontrol edin.
