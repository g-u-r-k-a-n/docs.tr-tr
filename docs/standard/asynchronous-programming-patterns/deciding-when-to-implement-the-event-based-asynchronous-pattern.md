  - <span data-ttu-id="7f03e-127">Ortaya çıkan nesne modeli <xref:System.IAsyncResult> karmaşıklığı uygulamaları ayırma avantajından daha ağır basıyorsa, olay tabanlı deseni ve deseni aynı sınıfta ortaya çıkar.</span><span class="sxs-lookup"><span data-stu-id="7f03e-127">Expose the event-based pattern and the <xref:System.IAsyncResult> pattern on the same class if the resulting object model complexity outweighs the benefit of separating the implementations.</span></span> <span data-ttu-id="7f03e-128">Olay tabanlı deseni açığa çıkarmaktan kaçınmaktansa, her iki deseni tek bir sınıfta ortaya çıkarmak daha iyidir.</span><span class="sxs-lookup"><span data-stu-id="7f03e-128">It is better to expose both patterns on a single class than to avoid exposing the event-based pattern.</span></span>
