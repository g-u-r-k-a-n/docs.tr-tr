# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>' Public ', ' Private ', ' protected ', ' Friend ' veya ' Protected Friend ' için yalnızca bir tane belirtilebilir

Çakışan erişilebilirlik öğeleri bir öğe bildiriminde oluşur. Yalnızca bir erişilebilirlik belirtilebilir.  
  
 **Hata kimliği:** BC30176  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir erişilebilirlik seçin ve diğer anahtar sözcükleri bildirimden kaldırın.
