# <a name="expected-closing--for-xml-cdata-section"></a>For XML CDATA bölümü için kapatma ']] > ' bekleniyor

Bir XML CDATA sabit değerinde gerekli kapanış '] ' > ' karakteri eksik.  
  
 **Hata kimliği:** BC31162  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gerekli kapanış '] ' > ' karakterini XML CDATA sabit değerinin sonuna ekleyin.
