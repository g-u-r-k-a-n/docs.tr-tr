# <a name="compiler-error-cs0243"></a><span data-ttu-id="f550a-102">Derleyici hatası CS0243</span><span class="sxs-lookup"><span data-stu-id="f550a-102">Compiler error CS0243</span></span>

<span data-ttu-id="f550a-103">Koşullu öznitelik bir geçersiz kılma yöntemi olduğundan 'yöntem' geçerli değildir</span><span class="sxs-lookup"><span data-stu-id="f550a-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="f550a-104">Geçersiz <xref:System.Diagnostics.ConditionalAttribute> [kılma](../language-reference/keywords/override.md) anahtar sözcüğüyle işaretlenmiş bir yöntemde öznitelik izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="f550a-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="f550a-105">Daha fazla bilgi için [bkz.](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)</span><span class="sxs-lookup"><span data-stu-id="f550a-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="f550a-106">Derleyici yöntemleri geçersiz kılmak için asla bağlamaz.</span><span class="sxs-lookup"><span data-stu-id="f550a-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="f550a-107">Yalnızca temel yönteme bağlanır ve ortak dil çalışma zamanı uygun şekilde geçersiz kılmayı çağırır.</span><span class="sxs-lookup"><span data-stu-id="f550a-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="f550a-108">Aşağıdaki kod CS0243 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f550a-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
