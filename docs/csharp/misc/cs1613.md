# <a name="compiler-error-cs1613"></a><span data-ttu-id="e6f16-103">Derleyici hatası CS1613</span><span class="sxs-lookup"><span data-stu-id="e6f16-103">Compiler Error CS1613</span></span>

<span data-ttu-id="e6f16-104">' Interface ' arabirimi için ' class ' yönetilen coclass sarmalayıcı sınıfı bulunamıyor (bir derleme başvurunuz mu eksik?)</span><span class="sxs-lookup"><span data-stu-id="e6f16-104">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="e6f16-105">Bir arabirimden COM nesnesi örneğini oluşturma girişiminde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="e6f16-105">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="e6f16-106">Arabirim **ComImport** ve `CoClass` özniteliklerine sahiptir, ancak derleyici öznitelik için verilen türü bulamıyor `CoClass` .</span><span class="sxs-lookup"><span data-stu-id="e6f16-106">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="e6f16-107">Bu hatayı çözmek için aşağıdakilerden birini deneyebilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="e6f16-107">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="e6f16-108">Coclass 'a sahip olan derlemeye bir başvuru ekleyin (çoğu zaman arabirimin ve coclass 'ın aynı derlemede olması gerekir).</span><span class="sxs-lookup"><span data-stu-id="e6f16-108">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="e6f16-109">Bilgi için bkz. [**Başvurular**](../language-reference/compiler-options/inputs.md#references) veya [başvuru ekleme iletişim kutusu](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) .</span><span class="sxs-lookup"><span data-stu-id="e6f16-109">See [**References**](../language-reference/compiler-options/inputs.md#references) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="e6f16-110">`CoClass`Arabirimdeki özniteliği onarın.</span><span class="sxs-lookup"><span data-stu-id="e6f16-110">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="e6f16-111">Aşağıdaki örnek, **CoClassAttribute**'in doğru kullanımını gösterir:</span><span class="sxs-lookup"><span data-stu-id="e6f16-111">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
