# <a name="compiler-error-cs1504"></a>Derleyici hatası CS1504
' Dosya ' kaynak dosyası açılamadı (' Reason ')  
  
 Kaynak dosya, derleyici tarafından açılamıyor veya okunamadı. Dosya başka bir uygulama tarafından kilitlenmiş olabilir ya da başka bir işletim sistemi sorunu olabilir. İleti, derleyicinin neden dosyayı açamadığına veya okuyamamasının nedenini içerir.
