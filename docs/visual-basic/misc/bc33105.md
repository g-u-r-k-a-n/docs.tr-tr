# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="c9a57-102">' If ' işlenenleri adlandırılmış bağımsız değişkenler olamaz</span><span class="sxs-lookup"><span data-stu-id="c9a57-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="c9a57-103">İşlecin işlenenlerinde adlandırılmış bağımsız değişkenlerin kullanılması `If` geçerli değil.</span><span class="sxs-lookup"><span data-stu-id="c9a57-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="c9a57-104">Aşağıdaki örnek bu hataya neden olur:</span><span class="sxs-lookup"><span data-stu-id="c9a57-104">The following example causes this error:</span></span>  
  
```vb  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="c9a57-105">Bu, `IIf` aşağıdaki kodda gösterildiği gibi, adlandırılmış bağımsız değişkenlere izin veren işlevinden farklıdır:</span><span class="sxs-lookup"><span data-stu-id="c9a57-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```

