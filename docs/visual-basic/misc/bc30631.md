# <a name="get-statement-must-end-with-a-matching-end-get"></a>' Get ' ifadesinin eşleşen bir ' End Get ' ile bitmesi gerekir

`Get` Özellik yordamlarının bir ifadesiyle bitmesi gerekir `End Get` .  
  
 **Hata kimliği:** BC30631  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Get`Özellik yordamının bir yapı ile bitdiğinizden emin olun `End Get` .
