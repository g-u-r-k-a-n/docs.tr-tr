# <a name="file-format-not-valid"></a>Dosya biçimi geçerli değil

Bir bileşeni bir bileşene yüklemeye çalıştınız, ancak dosyadaki verilerin biçimi bileşeniyle uyumlu değildi veya bileşen verilerini bir dosyaya kaydetmeye çalıştınız, ancak verilerin biçimi dosyanın biçimiyle uyumlu değil olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Disk dosyası verilerinin doğru biçimini öğrenmek için bileşene yönelik belgelere başvurun.  
  
2. Bir biçimden diğerine dönüştürme desteği verip sunmadığını öğrenmek için bileşene yönelik belgelere başvurun.
