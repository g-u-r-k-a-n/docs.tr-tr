# <a name="compiler-error-cs0076"></a>Derleyici hatası CS0076
' Value__ ' Numaralandırıcı adı ayrılmış ve kullanılamaz  
  
 [Numaralandırmadaki](../language-reference/builtin-types/enum.md) bir öğe **value__** adlı bir tanımlayıcıya sahip olamaz.
