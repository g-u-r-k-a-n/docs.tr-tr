# <a name="if-block-must-end-with-a-matching-end-if"></a>' #If ' bloğunun eşleşen bir ' #End If ' ile bitmesi gerekir
`#If`, koşullu bir derleme yönergedir. Bir `#If` blok, bir yönerge tarafından sonlandırılmaz `#End If` .  
  
 **Hata kimliği:** BC30012  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `#End If`Koşullu derleme bloğunun sonuna bir yönerge ekleyin.
