# <a name="3501---inferredcontractdescription"></a>3501 - InferredContractDescription

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|3501|  
|Anahtar sözcükler|WFServices|  
|Düzey|Bilgi|  
|Kanal|Microsoft-Windows-uygulama sunucusu-uygulamalar/analitik|  
  
## <a name="description"></a>Açıklama  

 WorkflowService 'dan bir ContractDescription çıkardığını gösterir.  
  
## <a name="message"></a>İleti  

 Name = ' %1 ' ve Namespace = ' %2 ' olan ContractDescription WorkflowService 'dan çıkarsandı.
