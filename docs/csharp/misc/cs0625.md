# <a name="compiler-error-cs0625"></a><span data-ttu-id="b30d5-102">Derleyici Hatası CS0625</span><span class="sxs-lookup"><span data-stu-id="b30d5-102">Compiler Error CS0625</span></span>
<span data-ttu-id="b30d5-103">'alan': StructLayout(LayoutKind.Explicit) ile işaretlenmiş örnek alan türlerinin FieldOffset özniteliği olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="b30d5-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="b30d5-104">Bir yapı açık bir **StructLayout** özniteliği ile işaretlendiğinde, yapıdaki tüm alanların [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) özniteliği olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="b30d5-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="b30d5-105">Daha fazla bilgi için Bkz. [StructLayoutAttribute Sınıf.](xref:System.Runtime.InteropServices.StructLayoutAttribute)</span><span class="sxs-lookup"><span data-stu-id="b30d5-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span>

<span data-ttu-id="b30d5-106">Aşağıdaki örnek CS0625 üretir:</span><span class="sxs-lookup"><span data-stu-id="b30d5-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
