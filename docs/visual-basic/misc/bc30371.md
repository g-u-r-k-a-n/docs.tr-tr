# <a name="module-modulename-cannot-be-used-as-a-type"></a>' \<modulename> ' Modülü tür olarak kullanılamaz
Bir modülü tür olarak kullanmayı denediniz. Modül, tür olarak değil kullanılamaz `Get` .  
  
 **Hata kimliği:** BC30371  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Modülünü geçerli bir tür olarak kullanılabilecek bir şekilde değiştirin.
