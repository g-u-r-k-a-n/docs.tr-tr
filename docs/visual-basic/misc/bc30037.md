# <a name="character-is-not-valid"></a>Karakter geçerli değil

Geçerli bağlam bu karaktere izin vermiyor. Örneğin, bir tilde ( `~` ) veya dikey çubuk ( `|` ), belirtilen bir öğe adının parçası olamaz.  
  
 **Hata kimliği:** BC30037  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bir ek karakterin yanlışlıkla programlama öğesine eklenmiş olup olmadığını kontrol edin.  
  
2. Programlama öğesinin yalnızca bu tür öğe için geçerli olan karakterler kullanılarak yazıldığını veya oluşturulduğunu denetleyin.
