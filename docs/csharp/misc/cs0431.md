# <a name="compiler-error-cs0431"></a><span data-ttu-id="117f0-103">Derleyici hatası CS0431</span><span class="sxs-lookup"><span data-stu-id="117f0-103">Compiler Error CS0431</span></span>

<span data-ttu-id="117f0-104">Takma ad bir türe başvurduğundan ' identifier ' diğer adı ':: ' ile kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="117f0-104">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="117f0-105">Bunun yerine '. ' kullanın.</span><span class="sxs-lookup"><span data-stu-id="117f0-105">Use '.' instead.</span></span>  
  
 <span data-ttu-id="117f0-106">Bir türe başvuran diğer adla "::" kullandınız.</span><span class="sxs-lookup"><span data-stu-id="117f0-106">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="117f0-107">Bu hatayı çözmek için "." işlecini kullanın.</span><span class="sxs-lookup"><span data-stu-id="117f0-107">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="117f0-108">Aşağıdaki örnek CS0431 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="117f0-108">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
