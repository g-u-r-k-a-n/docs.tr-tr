# <a name="partial-trust"></a>Kısmi Güven

.NET Framework 3,5 ' den başlayarak, kısmen güvenilen çağıranlar, <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>ve <xref:System.ServiceModel.Web>uygulanan Genel türlere ve yöntemlere erişebilir. Bu bölümde, kısmen güvenilen bir uygulama içinde Windows Communication Foundation (WCF) kullanmak için desteklenen senaryolar ve azaltılmış kod erişim güvenliği (CAS) ile çalışan uygulamalar için kullanılabilen WCF işlevselliğinin sınırlı alt kümesi açıklanmaktadır izinleri.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Desteklenen Dağıtım Senaryoları](../../../../docs/framework/wcf/feature-details/supported-deployment-scenarios.md)  
 WCF çalıştırmaya yönelik ana kısmi güven senaryolarını açıklar.  
  
 [Kısmi Güven Özelliği Uyumluluğu](../../../../docs/framework/wcf/feature-details/partial-trust-feature-compatibility.md)  
 Kısmi güvenle kullanılamayan WCF özelliklerini açıklar.  
  
 [Kısmi Güven En İyi Uygulamaları](../../../../docs/framework/wcf/feature-details/partial-trust-best-practices.md)  
 Kısmen güvenilen uygulamalarda WCF kullanmak için en iyi uygulamaları içerir.
