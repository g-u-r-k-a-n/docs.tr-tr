# <a name="compiler-error-cs0445"></a>Derleyici Hatası CS0445
Kutulama yıkıntı dönüşüm sonucunu değiştiremez  
  
 Unboxing dönüşüm sonucu geçici bir değişkendir. Derleyici, geçici değişken kaybolduğunda herhangi bir değişiklik ortadan kalkar, bu tür değişkenleri değiştirmenizi engeller. Bunu düzeltmek için, ara ifadeyi depolamak için yeni bir değer türü değişkeni bildirin ve bu değişkene kutulama dönüştürmenin sonucunu atayın.  
  
 Aşağıdaki kod CS0455 oluşturur.  
  
```csharp  
// CS0445.CS  
class UnboxingTest  
{  
    public static void Main()  
    {  
        Point p;  
        p.x = 1;  
        p.y = 2;  
        object obj = p;  
        // The following line generates CS0445, because the result  
        // of unboxing obj is a temporary variable.  
        ((Point)obj).x = 2;  
  
        // The following lines resolve the error.  
  
        // Store the result of the unboxing conversion in p2.  
        Point p2;
        p2 = (Point)obj;  
        // Then you can modify the unboxed value.  
        p2.x = 2;  
    }  
}  
  
struct Point  
{  
    public int x, y;  
}  
```
