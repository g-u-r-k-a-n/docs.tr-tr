# <a name="compiler-error-cs1631"></a>Derleyici hatası CS1631
Catch yan tümcesinin gövdesinde değer değerlendirilemiyor  
  
 Catch yan tümcesinin gövdesinden yield ifadesine izin verilmiyor. Bu hatayı önlemek için, yield ifadesini catch yan tümcesinin gövdesi dışına taşıyın.  
  
 Aşağıdaki örnek CS1631 oluşturur:  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
