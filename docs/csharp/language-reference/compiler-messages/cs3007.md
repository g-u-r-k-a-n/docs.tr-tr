# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="ab271-102">Derleyici Uyarısı (düzey 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="ab271-102">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="ab271-103">Yalnızca adlandırılmamış dizi türleri farklı olan 'method' tekrar yüklenen yöntemi CLS uyumlu değil</span><span class="sxs-lookup"><span data-stu-id="ab271-103">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="ab271-104">Basit bir dizi kullanan aşırı yüklü bir yönteminiz varsa ve yöntem imzaları arasındaki tek fark dizideki öğe türüyse bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="ab271-104">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="ab271-105">Bu hatayı önlemek için, pürüzlü bir dizi yerine dikdörtgen bir dizi kullanmayı düşünün; işlev çağrısını ayrıştırmak için ek bir parametre kullanın; aşırı yüklenilen yöntemlerden birini veya daha fazlasını yeniden adlandırmak; veya CLS Uyumluluğu gerekli değilse, <xref:System.CLSCompliantAttribute> özniteliği kaldırın.</span><span class="sxs-lookup"><span data-stu-id="ab271-105">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="ab271-106">CLS Uyumluluğu hakkında daha fazla bilgi için Dil [Bağımsızlığı ve Dilden Bağımsız Bileşenler'e](../../../standard/language-independence-and-language-independent-components.md)bakın.</span><span class="sxs-lookup"><span data-stu-id="ab271-106">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](../../../standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="ab271-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="ab271-107">Example</span></span>  
 <span data-ttu-id="ab271-108">Aşağıdaki örnek CS3007 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ab271-108">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
