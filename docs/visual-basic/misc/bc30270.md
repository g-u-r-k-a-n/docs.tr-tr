# <a name="specifier-is-not-valid-on-an-interface-method-declaration"></a>' \<specifier> ', arabirim yöntemi bildiriminde geçerli değil

`Function` `Sub` Arabirim içindeki or deyimleri, gibi geçersiz bir anahtar sözcük içeriyor `Implements` . Arabirim yalnızca üyeleri tanımlayabilir, uygulamaz.  
  
 **Hata kimliği:** BC30270  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bildirim ifadesinden geçersiz anahtar sözcüğü kaldır.  
  
2. Arabirim üyelerinin uygulamasını arabirimini uygulayan bir sınıfa taşıyın.
