# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>' Let ' ve ' Set ' atama deyimleri artık desteklenmiyor
`Let`ve `Set` atamalarında kullanılamaz.  
  
 **Hata kimliği:** BC30807  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Let` `Set` Atama deyimlerini atlayın; örneğin:  
  
     `X = 2`
