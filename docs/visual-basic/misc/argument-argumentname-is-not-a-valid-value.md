# <a name="argument-argumentname-is-not-a-valid-value"></a>' ' Bağımsız değişkeni \<argumentname> geçerli bir değer değil

Bağımsız değişkenin değeri geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bağımsız değişkenin değerini denetleyin.  
  
2. Bağımsız değişken bir hesaplama tarafından sağlandıysa, hesaplama işlemlerinin değerlerini denetleyin.
