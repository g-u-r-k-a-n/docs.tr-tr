<span data-ttu-id="f24fb-103">' ParamArray ' bir genişletme yönteminin ilk parametresine uygulanamaz.</span><span class="sxs-lookup"><span data-stu-id="f24fb-103">'ParamArray' cannot be applied to the first parameter of an extension method.</span></span> <span data-ttu-id="f24fb-104">İlk parametre hangi türün genişletileceğini belirtir.</span><span class="sxs-lookup"><span data-stu-id="f24fb-104">The first parameter specifies which type to extend.</span></span>

<span data-ttu-id="f24fb-105">Bir genişletme yönteminin ilk parametresi, yöntemin genişlettiği veri türünü belirtir.</span><span class="sxs-lookup"><span data-stu-id="f24fb-105">The first parameter of an extension method specifies the data type that the method extends.</span></span> <span data-ttu-id="f24fb-106">Bu nedenle, ilk parametre gereklidir ve isteğe bağlı olamaz.</span><span class="sxs-lookup"><span data-stu-id="f24fb-106">Therefore, the first parameter is required and cannot be optional.</span></span> <span data-ttu-id="f24fb-107">Bir parametre dizisi otomatik olarak isteğe bağlı olduğundan, bir genişletme yönteminin ilk bağımsız değişkeni olarak geçerli değildir.</span><span class="sxs-lookup"><span data-stu-id="f24fb-107">Because a parameter array is automatically optional, it is not valid as the first argument of an extension method.</span></span>

> [!NOTE]
> <span data-ttu-id="f24fb-108">Yöntemi yürütüldüğünde, yöntemi çağıran genişletilmiş veri türünün örneği metodun ilk parametresi için bağımsız değişken olur.</span><span class="sxs-lookup"><span data-stu-id="f24fb-108">When the method is executed, the instance of the extended data type that invokes the method becomes the argument for the first parameter of the method.</span></span> <span data-ttu-id="f24fb-109">Örneğin, içindeki örneği, `greeting` `greeting.Print()` ilk parametre olan bağımsız değişkendir, örneğin `str` genişletme yöntemi `Public Sub Print (ByVal str As String)` .</span><span class="sxs-lookup"><span data-stu-id="f24fb-109">For example, the instance `greeting` in `greeting.Print()` is the argument for the first parameter, `str`, in extension method `Public Sub Print (ByVal str As String)`.</span></span>

<span data-ttu-id="f24fb-110">**Hata kimliği:** BC36554</span><span class="sxs-lookup"><span data-stu-id="f24fb-110">**Error ID:** BC36554</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="f24fb-111">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="f24fb-111">To correct this error</span></span>

- <span data-ttu-id="f24fb-112">Parametre dizisi genişletmek istediğiniz veri türünü belirtmezse, bu türü belirten yeni bir ilk parametre ekleyin.</span><span class="sxs-lookup"><span data-stu-id="f24fb-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>

  ```vb
  <Extension()>
  Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)
  ' Concatenate the strings in addOns to str.
  End Sub
  ```

- <span data-ttu-id="f24fb-113">Parametre dizisi genişletmek istediğiniz veri türünü belirtse, bir parametre dizisi yerine bir bağımsız değişken gerektiren normal bir dizi olarak değiştirmeyi düşünün.</span><span class="sxs-lookup"><span data-stu-id="f24fb-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="f24fb-114">Normal diziler genişletilebilir.</span><span class="sxs-lookup"><span data-stu-id="f24fb-114">Regular arrays can be extended.</span></span>

  ```vb
  <Extension()>
  Public Function Sum(ByVal ints() As Integer) As Integer
      Dim total As Integer = 0
      For Each i As Integer In ints
          total = total + i
      Next i
      Return total
  End Function
  ```
