## <a name="nullable-annotation-context"></a><span data-ttu-id="5ddfa-181">Null yapılabilir ek açıklama bağlamı</span><span class="sxs-lookup"><span data-stu-id="5ddfa-181">Nullable annotation context</span></span>

<span data-ttu-id="5ddfa-182">Derleyici, devre dışı bırakılmış bir null yapılabilir ek açıklama bağlamında aşağıdaki kuralları kullanır:</span><span class="sxs-lookup"><span data-stu-id="5ddfa-182">The compiler uses the following rules in a disabled nullable annotation context:</span></span>

<span data-ttu-id="5ddfa-188">Derleyici, etkinleştirilmiş bir null yapılabilir ek açıklama bağlamında aşağıdaki kuralları kullanır:</span><span class="sxs-lookup"><span data-stu-id="5ddfa-188">The compiler uses the following rules in an enabled nullable annotation context:</span></span>

- <span data-ttu-id="5ddfa-189">Başvuru türündeki herhangi bir değişken **null atanamaz bir başvurudur**.</span><span class="sxs-lookup"><span data-stu-id="5ddfa-189">Any variable of a reference type is a **non-nullable reference**.</span></span>

<span data-ttu-id="5ddfa-195">Etkin bir null yapılabilir ek açıklama bağlamında, bir başvuru türüne eklenen `?` karakteri **null olabilen bir başvuru türü**bildirir.</span><span class="sxs-lookup"><span data-stu-id="5ddfa-195">In an enabled nullable annotation context, the `?` character appended to a reference type declares a **nullable reference type**.</span></span> <span data-ttu-id="5ddfa-196">İfadenin null olmadığını bildirmek için `!` **null-forverme işleci** bir ifadeye eklenebilir.</span><span class="sxs-lookup"><span data-stu-id="5ddfa-196">The **null-forgiving operator** `!` may be appended to an expression to declare that the expression isn't null.</span></span>

1. <span data-ttu-id="5ddfa-204">Değişken, null olmayan bir değere kesin olarak atandı.</span><span class="sxs-lookup"><span data-stu-id="5ddfa-204">The variable has been definitely assigned to a non-null value.</span></span>
1. <span data-ttu-id="5ddfa-205">Değişken veya ifade, kendisine başvurulmadan önce null değere karşı denetlendi.</span><span class="sxs-lookup"><span data-stu-id="5ddfa-205">The variable or expression has been checked against null before de-referencing it.</span></span>
