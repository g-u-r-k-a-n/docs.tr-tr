# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>' ' XML açıklama parametresi \<parameter> , karşılık gelen deyimdeki bir parametreyle eşleşmiyor \<keyword>

XML açıklama parametresi \<parameter> karşılık gelen deyimdeki bir parametreyle eşleşmiyor \<keyword> . XML açıklaması yoksayılacak.  
  
 Bir yöntemde hiçbir parametre değil, bazıları için bir XML açıklama etiketi belirtildi.  
  
 **Hata kimliği:** BC42307  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yöntemdeki tüm parametrelerin XML açıklama etiketini girin.
