# <a name="unable-to-create-temp-file-in-path-filename-error-message"></a>' ' Yolunda geçici dosya oluşturulamıyor \<filename> : \<error message>

Visual Basic derleyici, bildirime sahip bir derleme oluşturmak için derleme bağlayıcı (Al.exe, Alink olarak da bilinir) çağırır. Bağlayıcı bir dosya oluştururken veya bir bellek içi kaynağı yazarken bir hata bildirdi. Bu bir yapılandırma sorunu olabilir.  
  
 **Hata kimliği:** BC30138  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Alıntı yapılan hata iletisini inceleyin ve daha fazla açıklama ve öneri için  [Al.exe (derleme bağlayıcı)](../../framework/tools/al-exe-assembly-linker.md) konusuna başvurun.  
  
2. Hata devam ederse, koşullar hakkındaki bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
