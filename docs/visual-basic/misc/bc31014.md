# <a name="unable-to-find-entry-point-name-in-dll-filename-error"></a>' ' DLL dosyasında ' ' giriş noktası bulunamadı \<name> \<filename> : \<error>

Belirtilen DLL 'de yordam adı bulunamadı.  
  
 **Hata kimliği:** BC31014  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Yordam adının doğru yazıldığını ve büyük/küçük harf işlevinin DLL 'deki işlevle eşleştiğini doğrulayın.
