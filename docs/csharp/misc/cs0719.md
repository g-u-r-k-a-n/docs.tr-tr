# <a name="compiler-error-cs0719"></a><span data-ttu-id="1038a-103">Derleyici hatası CS0719</span><span class="sxs-lookup"><span data-stu-id="1038a-103">Compiler Error CS0719</span></span>

<span data-ttu-id="1038a-104">' Type ': dizi öğeleri statik türde olamaz</span><span class="sxs-lookup"><span data-stu-id="1038a-104">'type': array elements cannot be of static type</span></span>  
  
 <span data-ttu-id="1038a-105">Dizi öğeleri örnek olduğundan ve statik türlerin örnekleri oluşturmak mümkün olmadığından, statik türdeki bir dizi mantıklı değildir.</span><span class="sxs-lookup"><span data-stu-id="1038a-105">An array of static type does not make sense since array elements are instances, but it is not possible to create instances of static types.</span></span>  
  
 <span data-ttu-id="1038a-106">Aşağıdaki örnek CS0719 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1038a-106">The following sample generates CS0719:</span></span>  
  
```csharp  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
