# <a name="expected-closing--for-xml-tag"></a>For XML etiketi için kapatma ' > ' bekleniyor
Bir XML sabit değerinin XML etiketinde gereken ' > ' karakteri eksik.  
  
 **Hata kimliği:** BC31166  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- ' > ' karakterini XML sabit değerinin XML etiketinin sonuna ekleyin.
