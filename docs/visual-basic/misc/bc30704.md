# <a name="run-time-exception-thrown"></a>Çalışma zamanı özel durumu oluşturuldu

Hata ayıklama penceresindeki ifadelerden biri bir çalışma zamanı hatası verdi.  
  
 **Hata kimliği:** BC30704  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kodu denetleyin.
