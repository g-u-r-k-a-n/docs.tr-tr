# <a name="compiler-warning-level-3-cs0675"></a>Derleyici Uyarısı (düzey 3) CS0675
Bitwise veya operatör bir işaret genişletilmiş operand kullanılan; önce daha küçük bir imzasız türe döküm yapmayı düşünün  
  
 Derleyici örtülü olarak genişledi ve bir değişkeni işaretledi ve sonra elde edilen değeri bitwise VEYA işleminde kullandı. Bu beklenmeyen davranışlara neden olabilir.  
  
 Aşağıdaki örnek CS0675 oluşturur:  
  
```csharp  
// CS0675.cs  
// compile with: /W:3  
using System;  
  
public class sign  
{  
   public static void Main()  
   {  
      int hi = 1;  
      int lo = 1;  
      long value = (((long)hi) << 32) | lo;              // CS0675  
      // try the following line instead  
      // long value = (((long)hi) << 32) | ((uint)lo);   // correct  
   }  
}  
```
