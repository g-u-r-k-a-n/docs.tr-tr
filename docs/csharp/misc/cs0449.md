# <a name="compiler-error-cs0449"></a><span data-ttu-id="87f34-102">Derleyici Hatası CS0449</span><span class="sxs-lookup"><span data-stu-id="87f34-102">Compiler Error CS0449</span></span>
<span data-ttu-id="87f34-103">'Sınıf' veya 'struct' kısıtlaması diğer kısıtlamalardan önce gelmelidir</span><span class="sxs-lookup"><span data-stu-id="87f34-103">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="87f34-104">Genel bir tür veya yöntemin tür parametresi üzerindeki kısıtlamalar `class` belirli `struct` bir sırada oluşmalıdır: ya da ilk olarak, varsa, daha sonra herhangi bir arabirim kısıtlamaları ve son olarak herhangi bir oluşturucu kısıtlamaları olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="87f34-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="87f34-105">Bu hata, önce `class` `struct` görünmeyen kısıtlama veya kısıtlamadan kaynaklanır.</span><span class="sxs-lookup"><span data-stu-id="87f34-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="87f34-106">Bu hatayı gidermek için kısıtlama yan tümcelerini yeniden sırala.</span><span class="sxs-lookup"><span data-stu-id="87f34-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="87f34-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="87f34-107">Example</span></span>  
 <span data-ttu-id="87f34-108">Aşağıdaki örnek CS0449 üretir.</span><span class="sxs-lookup"><span data-stu-id="87f34-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
