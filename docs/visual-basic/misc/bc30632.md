# <a name="end-set-must-be-preceded-by-a-matching-set"></a>' End Set ' öncesinde eşleşen bir ' Set ' olmalıdır

`End Set` özellik yordamlarını sonlandırmak için kullanılır `Set` . `End Set`Bir özellik yordamının dışında bir yapı ile karşılaşıldı `Set` .  
  
 **Hata kimliği:** BC30632  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Set`Özellik yordamının bir `Property` anahtar sözcükten sonra ve yapılandırmadan önce bildirildiği emin olun `End Property` .  
  
2. `Set`Özellik yordamının `Set` anahtar sözcüğüyle başladığından ve bir yapı ile bitdiğinizden emin olun `End Set` .
