- <span data-ttu-id="f12e0-116">Tüm parametreler ve özellik değerleri mutlak ve yerel bağımsızdır.</span><span class="sxs-lookup"><span data-stu-id="f12e0-116">All parameters and property values are absolute and independent of locale.</span></span>
