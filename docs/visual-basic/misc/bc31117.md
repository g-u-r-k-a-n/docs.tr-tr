# <a name="raiseevent-declaration-must-end-with-a-matching-end-raiseevent"></a>' RaiseEvent ' bildirimi eşleşen bir ' End RaiseEvent ' ile bitmelidir

Bir `RaiseEvent` bildirimin ifadesiyle bitmesi gerekir `End RaiseEvent` .  
  
 **Hata kimliği:** BC31117  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `RaiseEvent`Bildirimin bir ifadesiyle bitmediğini denetleyin `End RaiseEvent` .
