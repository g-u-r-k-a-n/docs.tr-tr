# <a name="evaluation-of-expression-or-statement-stopped"></a>İfade veya deyimin değerlendirmesi durduruldu

**Değerlendirmeyi durdur** menü öğesi seçildi ve değerlendirmenin durdurulmasına neden oldu.  
  
 **Hata kimliği:** BC30721  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yürütmenin **değerlendirmeyi durdur** seçimini yapmadan devam etmesine izin verin.
