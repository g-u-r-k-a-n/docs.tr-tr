# <a name="systemservicemodelchannelspeernodeauthenticationtimeout"></a>System.ServiceModel.Channels.PeerNodeAuthenticationTimeout

## <a name="description"></a>Açıklama  
 Bu izleme, güvenli bir komşu bağlantı kurmaya çalışırken oluşur.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Tüm komşuların uygun güvenlik kimlik bilgilerine sahip olduğundan ve 1 dakikadan kısa bir sürede tüm iletilere yanıt verdiğini doğrulayın.
