# <a name="you-must-specify-a-file-name"></a>Bir dosya adı belirtmeniz gerekir

Dosya adı sağlamaya gerek duyan bir dosyayı taşıma veya kopyalama gibi bir işlem yapılmaya çalıştınız.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gerekli dosya adını sağlayın.  
  
- Yolun bir dizin ayırıcı karakteriyle () bitmediğinden emin olun \\ .
