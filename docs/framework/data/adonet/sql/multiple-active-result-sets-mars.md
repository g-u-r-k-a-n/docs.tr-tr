# <a name="multiple-active-result-sets-mars"></a>Birden Çok Etkin Sonuç Kümesi (MARS)

Birden çok etkin sonuç kümesi (MARS), tek bir bağlantıda birden çok toplu işi yürütmeye olanak tanıyan bir özelliktir. Önceki sürümlerde, tek bir bağlantıda tek seferde yalnızca bir toplu iş yürütülebilir. Birden çok toplu işi MARS ile yürütmek, işlemlerin eşzamanlı olarak yürütülmesini göstermez.
