<span data-ttu-id="5fddd-106">Tür çıkarımı, bir genel yordama yapılan çağrıda tür veya parametre türlerini veya parametrelerini belirleyebilmek için kullanılmaya çalışıldı.</span><span class="sxs-lookup"><span data-stu-id="5fddd-106">An attempt has been made to use type inference to determine the type or types of the type parameter or parameters in a call to a generic procedure.</span></span> <span data-ttu-id="5fddd-107">Derleyici bir veya daha fazla tür parametresi için birden fazla olası veri türü bulur ve bu hatayı raporlar.</span><span class="sxs-lookup"><span data-stu-id="5fddd-107">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>
