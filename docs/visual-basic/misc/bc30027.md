# <a name="end-function-expected"></a>' End Function ' bekleniyor

Bir `End` deyimin, sonlandırmakta olduğu blokla eşleşmesi gerekir. `Function`Yordam tanımı düzgün sonlandırılmadı.  
  
 **Hata kimliği:** BC30027  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `End Function`Yordamın sonuna bir ifade ekleyin.
