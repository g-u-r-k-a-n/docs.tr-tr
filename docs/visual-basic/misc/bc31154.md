# <a name="xml-declaration-does-not-allow-attribute-attributename"></a>XML bildirimi ' attributeName ' özniteliğine izin vermiyor

XML belgesi değişmez değeri için geçersiz bir öznitelik belirtildi. XML belgesi değişmez değeri için geçerli öznitelik değerleri, XML 1,0 belirtimine dayanır.  
  
 **Hata kimliği:** BC31154  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML belgesi değişmez değerinden geçersiz özniteliği kaldırın.
