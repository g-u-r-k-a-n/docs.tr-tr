<span data-ttu-id="6ef3e-103">Bir lambda ifadesinde `As` , ile birlikte bir yan tümce kullanmadan bir parametre bildirdiniz `Option Strict` .</span><span class="sxs-lookup"><span data-stu-id="6ef3e-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>

```vb  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="6ef3e-104">Öğesinin türü çıkarsanamıyor, önceki bildirim geçerlidir `n` .</span><span class="sxs-lookup"><span data-stu-id="6ef3e-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="6ef3e-105">Örneğin, önceki lambda ifadesini bir işlev temsilcisine atarken `Del` :</span><span class="sxs-lookup"><span data-stu-id="6ef3e-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>

 <span data-ttu-id="6ef3e-106">Artık türü `n` parametresinden çıkarsanamıyor `p` :</span><span class="sxs-lookup"><span data-stu-id="6ef3e-106">Now the type of `n` can be inferred from parameter `p`:</span></span>
