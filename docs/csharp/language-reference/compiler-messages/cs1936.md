# <a name="compiler-error-cs1936"></a><span data-ttu-id="fad75-103">Derleyici Hatası CS1936</span><span class="sxs-lookup"><span data-stu-id="fad75-103">Compiler Error CS1936</span></span>

<span data-ttu-id="fad75-104">' Type ' kaynak türü için sorgu deseninin bir uygulama bulunamadı.</span><span class="sxs-lookup"><span data-stu-id="fad75-104">Could not find an implementation of the query pattern for source type 'type'.</span></span>  <span data-ttu-id="fad75-105">' Method ' bulunamadı.</span><span class="sxs-lookup"><span data-stu-id="fad75-105">'method' not found.</span></span>  
  
<span data-ttu-id="fad75-106">Bir kaynak türünü sorgulamak için, bu türün sorguda çağırdığınız standart sorgu işleci yöntemlerini uygulaması gerekir.</span><span class="sxs-lookup"><span data-stu-id="fad75-106">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="fad75-107">Uygulama, uygun yönergeyle kapsama getirilen sınıf üyeleri veya uzantı yöntemleri biçiminde olabilir `using` .</span><span class="sxs-lookup"><span data-stu-id="fad75-107">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>

## <a name="example"></a><span data-ttu-id="fad75-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="fad75-111">Example</span></span>

<span data-ttu-id="fad75-112">Aşağıdaki örnek CS1936 üretir:</span><span class="sxs-lookup"><span data-stu-id="fad75-112">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="fad75-113">Bu hata genellikle yanlışlıkla, bu nesnelerin bir koleksiyonu yerine, bazı türden bir nesneyi sorgulamaya çalıştığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="fad75-113">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
