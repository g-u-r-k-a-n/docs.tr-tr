# <a name="compiler-error-cs0542"></a><span data-ttu-id="23087-103">Derleyici hatası CS0542</span><span class="sxs-lookup"><span data-stu-id="23087-103">Compiler Error CS0542</span></span>

<span data-ttu-id="23087-104">' Kullanıcı tanımlı tür ': üye adları kendilerini kapsayan türle aynı olamaz</span><span class="sxs-lookup"><span data-stu-id="23087-104">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="23087-105">Bir sınıf veya yapının üyeleri, üye bir Oluşturucu olmadığı sürece sınıf veya struct ile aynı ada sahip olamaz.</span><span class="sxs-lookup"><span data-stu-id="23087-105">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="23087-106">Aşağıdaki örnek CS0542 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="23087-106">The following sample generates CS0542:</span></span>  
  
```csharp

{  
    public int C;  
}  
```  
  
 <span data-ttu-id="23087-107">Bu hata, bir oluşturucuya yanlışlıkla bir dönüş türü yerleştirirseniz, aslında onu sıradan bir yönteme getiren bu hataya neden olmuş olabilir.</span><span class="sxs-lookup"><span data-stu-id="23087-107">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="23087-108">Aşağıdaki örnek, bir `F` Oluşturucu değil, bir dönüş türü olduğundan bir yöntem olduğundan, CS0542 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="23087-108">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp

{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="23087-109">Sınıfınız ' Item ' olarak adlandırılmışsa ve bir dizin oluşturucu olarak bildirilirse `this` , bu hatayı alabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="23087-109">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="23087-110">Varsayılan bir dizin oluşturucuya, yayımlanmış kodda ' Item ' adı verilir ve çakışma oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="23087-110">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>
