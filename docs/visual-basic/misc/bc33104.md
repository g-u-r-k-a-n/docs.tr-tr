# <a name="if-operator-requires-either-two-or-three-operands"></a>' If ' işleci iki ya da üç işlenen gerektirir

`If`İşleç yalnızca iki veya üç bağımsız değişkeni kabul eder.  
  
 **Hata kimliği:** BC33104  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `If`İşleç sayısının iki ya da üç bağımsız değişkenle azaltılabilir veya uzatılamayacağını görmek için ifadeyi inceleyin.  
  
- `If...Then...Else`Bağımsız değişkenleri düzenlemek için bir ifade veya iç içe geçmiş bir ifade gibi başka bir koşullu oluşturma kullanmayı düşünün `If...Then...Else` .
