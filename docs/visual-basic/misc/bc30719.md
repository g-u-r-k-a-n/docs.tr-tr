# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>' Case ' deyimleri Komut penceresinde geçerli değildir
`Case` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30719  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Case` deyimlerini **hemen** pencereden kaldır.
