# <a name="reference-required-to-assembly-assemblyname-containing-the-implemented-interface-interfacename"></a>\<assemblyname>Uygulanan ' ' arabirimini içeren ' ' derlemesine başvuru gerekiyor \<interfacename>

\<assemblyname>Uygulanan ' ' arabirimini içeren ' ' derlemesine başvuru gerekiyor \<interfacename> . Projenize bir tane ekleyin.  
  
 Arabirim, projenizde doğrudan başvurulmayan bir dinamik bağlantı kitaplığı (DLL) veya bütünleştirilmiş kodda tanımlanır. Visual Basic Derleyicisi, arabirimin birden fazla DLL veya derlemede tanımlanması durumunda karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata kimliği:** BC30009  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Başvurulmayan DLL veya derlemenin adını Proje başvurularında ekleyin.
