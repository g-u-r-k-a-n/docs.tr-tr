# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError

TCP bağlanma işlemi başarısız oldu.  
  
## <a name="description"></a>Description  

 Bu uyarı düzeyi izleme, bir TCP uç noktasına bağlanma başarısızlığının olduğunu gösterir. Bu durum, uzak uç noktanın belirli bir IP adresi ve bağlantı noktasında yanıt vermemesi durumunda meydana gelebilir. Sonraki geçerli IP adreslerine (örneğin, IPv4 veya IPv6 adresleri ya da belirli bir ana bilgisayar adını temsil eden diğer IP adresleri) bağlanmayı denediğinde bu izleme yoksayılabilir. Bu izleme içindeki özel durum, hata hakkındaki ek bilgileri açığa çıkaryükleyebilir.
