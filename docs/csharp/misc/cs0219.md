# <a name="compiler-warning-level-3-cs0219"></a><span data-ttu-id="fdd8f-102">Derleyici Uyarısı (seviye 3) CS0219</span><span class="sxs-lookup"><span data-stu-id="fdd8f-102">Compiler Warning (level 3) CS0219</span></span>
<span data-ttu-id="fdd8f-103">Değişken 'değişken' atanır, ancak değeri asla kullanılmaz</span><span class="sxs-lookup"><span data-stu-id="fdd8f-103">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="fdd8f-104">Derleyici, bir değişkeni beyan edip atadığınızda, düzey-üç uyarısı yayınlar, ancak bunu kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="fdd8f-104">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="fdd8f-105">Derleyici bu uyarıyı yalnızca değişken değeri derleme zamanı sabiti olduğunda oluşturur.</span><span class="sxs-lookup"><span data-stu-id="fdd8f-105">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="fdd8f-106">Sabit olmayan bir ifade yi veya yöntem sonucunu yerel bir değişkene atamak, hata ayıklayıcıdaki bu ifadeleri gözlemlememizi kolaylaştırır.</span><span class="sxs-lookup"><span data-stu-id="fdd8f-106">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="fdd8f-107">Ayrıca, bu değişkene ulaşılabildiği sırada çöp toplamayı engelleyerek sonuca ulaşılabilir hale getirir.</span><span class="sxs-lookup"><span data-stu-id="fdd8f-107">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="fdd8f-108">Aşağıdaki örnek, uyarının ne zaman ve ne zaman oluşturulmadığını gösterir:</span><span class="sxs-lookup"><span data-stu-id="fdd8f-108">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
