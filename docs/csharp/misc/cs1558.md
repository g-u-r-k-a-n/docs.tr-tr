# <a name="compiler-error-cs1558"></a><span data-ttu-id="c219c-102">Derleyici Hatası CS1558</span><span class="sxs-lookup"><span data-stu-id="c219c-102">Compiler Error CS1558</span></span>
<span data-ttu-id="c219c-103">'sınıf' uygun bir statik Ana metodu yok</span><span class="sxs-lookup"><span data-stu-id="c219c-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="c219c-104">[-ana](../language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği, **Ana** yöntemin bakılabilmek için bir sınıf belirtir.</span><span class="sxs-lookup"><span data-stu-id="c219c-104">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="c219c-105">Ancak, [Ana](../programming-guide/main-and-command-args/index.md) yöntem doğru tanımlanmadı.</span><span class="sxs-lookup"><span data-stu-id="c219c-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="c219c-106">Aşağıdaki örnek, geçersiz iade türü nedeniyle CS1558 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c219c-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
