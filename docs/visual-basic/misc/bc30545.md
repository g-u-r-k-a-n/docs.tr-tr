# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Özellik erişimi özelliğe atanmalıdır veya değerini kullanmalıdır
Bir özelliğe atamaya veya değerini kullanmaya gerek kalmadan erişmeye çalıştınız.
  
 **Hata kimliği:** BC30545  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özelliğe bir değer atayın.  
  
     \-veya  
  
- Özelliğin değerini kullanın.
