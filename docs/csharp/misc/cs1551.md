# <a name="compiler-error-cs1551"></a>Derleyici hatası CS1551

Dizin oluşturucular en az bir parametreye sahip olmalıdır  
  
 Bağımsız değişken alan bir [Dizin Oluşturucu](../programming-guide/indexers/index.md) bildirildi.  
  
 Aşağıdaki örnek CS1551 oluşturur:  
  
```csharp  
// CS1551.cs  
public class MyClass  
{  
   int intI;  
  
   int this[]   // CS1551  
   // try the following line instead  
   // int this[int i]  
   {  
      get  
      {  
         return intI;  
      }  
      set  
      {  
         intI = value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
