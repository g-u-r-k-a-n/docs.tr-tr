# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>' Namespace ' ifadesinin eşleşen bir ' End Namespace ' ile bitmesi gerekir
`Namespace`blokların bir ifadesiyle bitmesi gerekir `End Namespace` .  
  
 **Hata kimliği:** BC30626  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `End Namespace`Bloğun sonuna bir ifade ekleyin `Namespace` .
