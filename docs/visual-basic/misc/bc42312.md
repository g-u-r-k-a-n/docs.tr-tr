# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>XML açıklama bloğu XML belge açıklamalarının uygulamasını destekleyen herhangi bir dil öğesiyle ilişkilendirilemez
XML açıklama bloğu hiçbir geçerli dil öğesiyle ilişkilendirilemez. XML açıklaması yoksayılacak. Bu hata, açıklama bloğunda bir XML etiketinin eşleşen Language öğesi yoksa da oluşur.  
  
 **Hata kimliği:** BC42312  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yabancı XML etiketini bulup kaldırın veya açıkladığı dil öğesini sağlayın.
