# <a name="create-an-azure-account"></a>Azure hesabı oluşturma

Azure 'u kullanmak için bir Azure hesabınızın olması gerekir.  Azure hesabınız, Azure [portalı](https://portal.azure.com) veya [Cloud Shell](https://shell.azure.com)gibi Azure hizmetlerinde oturum açmak için kullandığınız kimlik bilgileridir.

## <a name="option-1-use-monthly-azure-credits-for-visual-studio-subscribers"></a>Seçenek 1: Visual Studio aboneleri için aylık Azure kredilerini kullanma

Visual Studio aboneliğiniz varsa, aboneliğiniz Azure kullanmaya yönelik krediler içerir.  [Visual Studio aboneleri Için aylık Azure kredileri](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/) sayfasını ziyaret ederek kredilerinizi etkinleştirin.

## <a name="option-2-sign-up-for-a-free-azure-account"></a>2. seçenek: ücretsiz bir Azure hesabı için kaydolun

Ücretsiz olarak bir [Azure hesabı](https://azure.microsoft.com/free/dotnet/) oluşturabilir ve 12 aylık popüler hizmetleri $200 ücretsiz olarak alabilir ve Azure 'ı 30 gün boyunca keşfetmeye başlayabilirsiniz

## <a name="option-3-use-a-corporate-account"></a>3. seçenek: kurumsal hesap kullanma

İş sırasında Azure kullanıyorsanız, Azure kimlik bilgilerinizi almak için şirketinizin bulut yöneticisiyle konuşun ve ardından [Azure Portal](https://portal.azure.com)ile hesabınızda oturum açın.

## <a name="next-steps"></a>Sonraki adımlar

Azure hesabı oluşturduktan sonra, Azure 'a kolay erişim sağlamak için Web tarayıcınızda [Azure portalına](https://portal.azure.com) yer işaretinin bulunduğundan emin olun.

Daha sonra, kullandığınız IDE 'ye bağlı olarak Visual Studio 'Yu veya Visual Studio Code Azure geliştirme için yapılandırmak isteyeceksiniz.
