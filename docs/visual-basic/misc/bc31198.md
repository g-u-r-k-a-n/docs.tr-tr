# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="65887-102">Parantez içine alınmamışsa XML sabit değeri burada bulunamaz</span><span class="sxs-lookup"><span data-stu-id="65887-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="65887-103">Bir XML sabit değeri bildirimi, Visual Basic derleyicisine belirsiz bir konumdaki ifadede kullanılır.</span><span class="sxs-lookup"><span data-stu-id="65887-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="65887-104">Diğer bir deyişle Visual Basic derleyici, daha küçüktür karakteri (<) bir karşılaştırıcı işleci veya bir XML sabit değerinin başlangıcı olarak düşünülmediğini belirleyemez.</span><span class="sxs-lookup"><span data-stu-id="65887-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="65887-105">Aşağıdaki kod bir örnek sağlar:</span><span class="sxs-lookup"><span data-stu-id="65887-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="65887-106">**Hata kimliği:** BC31198</span><span class="sxs-lookup"><span data-stu-id="65887-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="65887-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="65887-107">To correct this error</span></span>  
  
- <span data-ttu-id="65887-108">Aşağıdaki örnekte gösterildiği gibi, XML değişmez bildirimini parantez içine alın:</span><span class="sxs-lookup"><span data-stu-id="65887-108">Enclose the XML literal declaration in parentheses, as shown in the following example:</span></span>

                  Select element  
```  
  
- <span data-ttu-id="65887-109">Aşağıdaki örnekte gösterildiği gibi, kodunuzu mevcut bir XML sabit değerine başvuracak şekilde değiştirin:</span><span class="sxs-lookup"><span data-stu-id="65887-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>

Dim queryResult = From element In elements _  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
