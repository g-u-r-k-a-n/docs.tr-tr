# <a name="unable-to-read-fixed-width-fields-because-fieldwidths-is-nothing-or-empty"></a>Fieldwidth Nothing veya boş olduğundan sabit genişlik alanları okunamıyor

`TextFieldParser` `FieldWidths` Özelliği, veya boş olarak ayarlandığından, sabit genişlikte alanları okuyamıyor `Nothing` .  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `FieldWidths`Geçerli bir değere ayarlayın.
