# <a name="module-statement-must-end-with-a-matching-end-module"></a>' Module ' ifadesinin eşleşen bir ' End Module ' ile bitmesi gerekir
Modül blokları bir `End Module` ifadesiyle bitmelidir.  
  
 **Hata kimliği:** BC30625  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `End Module`Bloğun sonuna bir ifade ekleyin `Module` .
