# <a name="statement-cannot-appear-within-a-method-body"></a>İfade bir metot gövdesinde görünemez

Yordam, başka bir yordam bildirimi gibi geçersiz bir ifade içeriyor. Yordam bildirimleri iç içe geçirilemez.  
  
 **Hata kimliği:** BC30289  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yordam bildirimini kaldırın veya kapsayan yordamın dışına taşıyın.
