# <a name="parameter-already-declared-with-name-argumentname"></a>Parametre ' ' adıyla zaten tanımlanmış \<argumentname>
Yordam bildirimi aynı ada sahip iki bağımsız değişkeni tanımlar. Yordam bildirimindeki tüm bağımsız değişken adları benzersiz olmalıdır.  
  
 **Hata kimliği:** BC30237  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişkenlerden birinin adını değiştirin.
