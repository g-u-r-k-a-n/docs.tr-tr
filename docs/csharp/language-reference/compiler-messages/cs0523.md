 <span data-ttu-id="6441c-109">Aşağıdaki örnek CS0523 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6441c-109">The following sample generates CS0523:</span></span>

```csharp  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
