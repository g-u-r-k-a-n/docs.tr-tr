# <a name="project-projectname-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-classname"></a>' ' Projesi, ' ' içeren ' ' \<projectname> derlemesine dolaylı bir başvuru yapar \<assemblyname> \<classname>

' ' Projesi, ' ' \<projectname> içeren ' ' derlemesine dolaylı bir başvuru yapar \<assemblyname> \<classname> . Projenize ' ' öğesine bir başvuru ekleyin \<assemblyname> .  
  
 Bu proje, Proje başvurularında bulunmayan bir derlemeye bağlı Üyeler içeriyor.  
  
 **Hata kimliği:** BC31515  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- **Çözüm Gezgini** başvuruya sağ tıklayıp **Başvuru Ekle** **' ye** tıklayarak derlemeye bir başvuru ekleyin.
