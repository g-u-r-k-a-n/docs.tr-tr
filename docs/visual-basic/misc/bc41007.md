# <a name="attribute-conditional-is-only-valid-on-sub-declarations"></a>' Conditional ' özniteliği yalnızca ' Sub ' bildirimlerinde geçerlidir
<xref:System.Diagnostics.ConditionalAttribute>Özniteliği bir yordam dışında bir yöntem bildirimine uygulandı `Sub` .  
  
 **Hata kimliği:** BC41007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. <xref:System.Diagnostics.ConditionalAttribute>Geçersiz bildirimden kaldırın.
