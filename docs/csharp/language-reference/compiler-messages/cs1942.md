# <a name="compiler-error-cs1942"></a>Derleyici Hatası CS1942

' Clause ' yan tümcesindeki ifadenin türü yanlış. ' Method ' çağrısında tür çıkarımı başarısız oldu.  
  
Bu hata genellikle Aralık değişkenine yanlış bir açık tür verildiğinde oluşturulur.  

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Aralık değişkeni açıkça yazılmışsa, türün, yineleme yaptığı koleksiyondaki öğelerin türü ile aynı ya da örtülü olarak dönüştürülebilir olduğundan emin olun. Aralık değişkenine önünde `var` anahtar sözcüğü varsa `var`kaldırın.  
  
## <a name="example"></a>Örnek

Aşağıdaki kod CS1942 oluşturur:  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

CS1942, CS1949 ile ilgilidir çünkü `var` bir tür olmadığından, bir Aralık değişkeniyle birlikte `var` kullanımı temeldeki `Cast<T>` işleminin başarısız olmasına neden olur.
