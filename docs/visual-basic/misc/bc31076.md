# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>' \<modulename> ' bir modüldür ve derleme olarak başvurulamaz

Bir modül derleme olarak kullanılmaya çalışıldı.  
  
 **Hata kimliği:** BC31076  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir derlemeye başvuruyu değiştirme.
