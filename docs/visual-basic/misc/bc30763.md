# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>.NET Compact Framework hedeflenen sürümü ANSI, Auto veya UNICODE değiştiricisini kullanmayı desteklemiyor

Üzerinde çalıştığınız .NET Compact Framework sürümü `Ansi` ,, `Auto` ve `Unicode` değiştiricilerini desteklemez.  
  
 **Hata kimliği:** BC30763  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştiricisini kaldırın.
