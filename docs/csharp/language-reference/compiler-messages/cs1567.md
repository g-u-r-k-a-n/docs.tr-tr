# <a name="compiler-error-cs1567"></a>Derleyici Hatası CS1567
Hata üreten Win32 kaynak: 'dosya'  
  
 Derlemeniz [ya -win32icon](../compiler-options/win32icon-compiler-option.md) derleyici seçeneğini kullandı ya da [-win32res](../compiler-options/win32res-compiler-option.md)kullanmadı, bu da derleyicinin kaynak bilgileri içeren bir dosya oluşturmasına neden oldu, ancak derleyici yetersiz disk alanı veya başka bir hata nedeniyle dosyayı oluşturamadı.  
  
 Dosya oluşturma sorununu çözemiyorsanız, kaynak bilgileri içeren bir dosya oluşturmayan [-win32res'i](../compiler-options/win32res-compiler-option.md)kullanabilirsiniz.
