# <a name="compiler-error-cs0629"></a><span data-ttu-id="0069d-102">Derleyici hatası CS0629</span><span class="sxs-lookup"><span data-stu-id="0069d-102">Compiler error CS0629</span></span>

<span data-ttu-id="0069d-103">Koşullu üye 'üye' 'Tür Adı' türünde arayüz üyesi 'taban sınıf üyesi' uygulayamaz</span><span class="sxs-lookup"><span data-stu-id="0069d-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
<span data-ttu-id="0069d-104">[Koşullu öznitelik](../language-reference/attributes/general.md#conditional-attribute) bir arabirim uygulamasında kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="0069d-104">The [Conditional attribute](../language-reference/attributes/general.md#conditional-attribute) cannot be used on the implementation of an interface.</span></span>  
  
<span data-ttu-id="0069d-105">Aşağıdaki örnek CS0629 üretir:</span><span class="sxs-lookup"><span data-stu-id="0069d-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
