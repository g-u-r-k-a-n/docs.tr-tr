# <a name="para-c-programming-guide"></a>\<para > (C# Programlama Kılavuzu)
## <a name="syntax"></a>Sözdizimi  
  
```xml  
<para>content</para>  
```  
  
## <a name="parameters"></a>Parametreler  
 `content`  
 Paragrafın metni.  
  
## <a name="remarks"></a>Açıklamalar  
 @No__t_0para > etiketi, [\<summary >](./summary.md), [\<remarks >](./remarks.md)veya [\<returns](./returns.md)> gibi bir etiket içinde kullanım içindir ve metne yapı eklemenizi sağlar.  
  
 Belge açıklamalarını bir dosyaya işlemek için [-doc](../../language-reference/compiler-options/doc-compiler-option.md) ile derleyin.  
  
## <a name="example"></a>Örnek  
 @No__t_2para > kullanma örneği için bkz. [\<summary >](./summary.md) .
