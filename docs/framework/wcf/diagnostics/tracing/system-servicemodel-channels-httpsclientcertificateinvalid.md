# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid

İstemci sertifikası geçersiz.  
  
## <a name="description"></a>Açıklama  

 Bu izleme, istemci tarafından sunulan sertifikanın HTTPS dinleyicisi tarafından geçersiz olduğunu gösteriyor. HTTPS dinleyicisi, bu sertifikayı kullanarak istemcinin gerçekliğini doğrulamaya çalışıyor. Sertifika yetkilisi, hizmeti barındıran sunucu tarafından tanınmıyorsa bir sertifika geçersiz olabilir.
