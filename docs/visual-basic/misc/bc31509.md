# <a name="unable-to-open-resource-file-filename-error"></a>' ' Kaynak dosyası açılamadı \<filename> :\<error>
Belirtilen kaynak dosyası açılırken bir hata oluştu.  
  
 **Hata kimliği:** BC31509  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Belirtilen ad ve konumun doğru olduğundan emin olun.  
  
2. Dosyanın hasarlı veya bozuk olmadığını doğrulayın.
