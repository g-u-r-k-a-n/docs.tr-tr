# <a name="set-statement-must-end-with-a-matching-end-set"></a>' Set ' ifadesinin eşleşen bir ' End Set ' ile bitmesi gerekir
`Set`Özellik yordamlarının bir ifadesiyle bitmesi gerekir `End Set` .  
  
 **Hata kimliği:** BC30633  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Set`Özellik yordamının bir yapı ile bitdiğinizden emin olun `End Set` .
