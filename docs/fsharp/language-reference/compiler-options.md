# <a name="compiler-options"></a>Derleyici seçenekleri

Bu konu, F # derleyicisi için derleyici komut satırı seçeneklerini açıklar, fsc.exe.

Derleme ortamı, proje özellikleri ayarlanarak de denetlenebilir. .NET Core 'u hedefleyen projeler için, içindeki "diğer bayraklar" özelliği `<OtherFlags>...</OtherFlags>` , `.fsproj` ek komut satırı seçeneklerini belirtmek için kullanılır.
