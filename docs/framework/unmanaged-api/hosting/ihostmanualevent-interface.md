# <a name="ihostmanualevent-interface"></a>IHostManualEvent Arabirimi
Konağın el ile sıfırlama olayının temsili için bir uygulama sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Reset Yöntemi](ihostmanualevent-reset-method.md)|Geçerli örneği, `IHostManualEvent` sinyal olmayan bir duruma sıfırlar.|  
|[Set Yöntemi](ihostmanualevent-set-method.md)|Geçerli `IHostManualEvent` örneği bir sinyal durumuna ayarlar.|  
|[Wait Yöntemi](ihostmanualevent-wait-method.md)|Geçerli örneğin, `IHostManualEvent` sahip olana kadar beklemesini veya belirli bir süre geçtikten sonra gelmesini sağlar.|
