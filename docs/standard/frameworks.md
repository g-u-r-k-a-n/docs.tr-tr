## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="490d8-198">Hedef çerçeveler nasıl belirtilir?</span><span class="sxs-lookup"><span data-stu-id="490d8-198">How to specify target frameworks</span></span>

<span data-ttu-id="490d8-199">Hedef çerçeveler proje dosyanızda belirtilir.</span><span class="sxs-lookup"><span data-stu-id="490d8-199">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="490d8-200">Tek bir hedef çerçevesi belirtildiğinde, **TargetFramework** öğesini kullanın.</span><span class="sxs-lookup"><span data-stu-id="490d8-200">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="490d8-201">Aşağıdaki konsol uygulaması proje dosyası ,NET Core 3.0'ı nasıl hedeflenebildiğini gösterir:</span><span class="sxs-lookup"><span data-stu-id="490d8-201">The following console app project file demonstrates how to target .NET Core 3.0:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp3.0</TargetFramework>
  </PropertyGroup>

</Project>
```

<span data-ttu-id="490d8-202">Birden çok hedef çerçevesi belirttiğiniz zaman, her hedef çerçeve için koşullu başvuru derlemeleri olabilir.</span><span class="sxs-lookup"><span data-stu-id="490d8-202">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="490d8-203">Kodunuzda, *if-then-else* mantığına sahip önişlemci sembolleri kullanarak bu derlemelere karşı koşullu olarak derleyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="490d8-203">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="490d8-204">Aşağıdaki kitaplık proje dosyası ,NET Standard`netstandard1.4`( ) api'lerini`net40` ve `net45`.NET Framework (ve) API'lerini hedefler.</span><span class="sxs-lookup"><span data-stu-id="490d8-204">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="490d8-205">Birden çok hedef çerçevesi olan çoğul **TargetFrameworks** öğesini kullanın.</span><span class="sxs-lookup"><span data-stu-id="490d8-205">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="490d8-206">Kitaplık `Condition` iki .NET Framework TFMs için derlendiğinde özniteliklerin uygulamaya özgü paketleri nasıl içerdiğine dikkat edin:</span><span class="sxs-lookup"><span data-stu-id="490d8-206">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml

  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->

  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net45' ">
    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```

<span data-ttu-id="490d8-207">Kitaplığınız veya uygulamanızda, her hedef çerçeve için derlemek üzere koşullu kod yazarsınız:</span><span class="sxs-lookup"><span data-stu-id="490d8-207">Within your library or app, you write conditional code to compile for each target framework:</span></span>

```csharp

    {

        Console.WriteLine("Target framework: .NET Framework 4.0");

        Console.WriteLine("Target framework: .NET Framework 4.5");
```
