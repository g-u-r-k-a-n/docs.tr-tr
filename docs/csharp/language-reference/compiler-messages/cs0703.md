# <a name="compiler-error-cs0703"></a><span data-ttu-id="42022-102">Derleyici Hatası CS0703</span><span class="sxs-lookup"><span data-stu-id="42022-102">Compiler Error CS0703</span></span>
<span data-ttu-id="42022-103">Tutarsız erişilebilirlik: kısıtlama türü 'tanımlayıcı' 'tanımlayıcı' daha az erişilebilir</span><span class="sxs-lookup"><span data-stu-id="42022-103">Inconsistent accessibility: constraint type 'identifier' is less accessible than 'identifier'</span></span>  
  
 <span data-ttu-id="42022-104">Kısıtlama, genel parametreyi genel sınıfın kendisinden daha az erişilebilir olmaya zorlayamaz.</span><span class="sxs-lookup"><span data-stu-id="42022-104">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="42022-105">Aşağıdaki örnekte, genel sınıf\<C T> genel olarak bildirilirken, kısıtlama T'yi bir iç arabirim uygulamaya zorlamaya çalışır.</span><span class="sxs-lookup"><span data-stu-id="42022-105">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="42022-106">Buna izin verilse bile, yalnızca dahili erişimi olan istemciler sınıf için parametre oluşturabilecektir, bu nedenle sınıf yalnızca dahili erişimi olan istemciler tarafından kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="42022-106">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="42022-107">Bu hatadan kurtulmak için, genel sınıfın erişim düzeyinin sınırlarda görünen tüm sınıflardan veya arabirimlerden daha az kısıtlayıcı olmadığından emin olun.</span><span class="sxs-lookup"><span data-stu-id="42022-107">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>  
  
 <span data-ttu-id="42022-108">Aşağıdaki örnek CS0703 üretir:</span><span class="sxs-lookup"><span data-stu-id="42022-108">The following sample generates CS0703:</span></span>  
  
```csharp  
// CS0703.cs  
internal interface I {}  
public class C<T> where T : I  // CS0703 – I is internal; C<T> is public  
{  
}  
```
