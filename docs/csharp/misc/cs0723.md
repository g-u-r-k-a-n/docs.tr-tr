# <a name="compiler-error-cs0723"></a>Derleyici hatası CS0723
' Type ' statik türünün değişkeni bildirilemez  
  
 Statik türlerin örnekleri oluşturulamıyor.  
  
 Aşağıdaki örnek CS0723 oluşturur:  
  
```csharp  
// CS0723.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC sc = null;  // CS0723  
   }  
}  
```
