# <a name="compiler-warning-level-4-cs1610"></a>Derleyici Uyarısı (düzey 4) CS1610
Varsayılan Win32 kaynağı için kullanılan geçici dosya 'dosya' sileme - kaynak  
  
 [-win32res](../compiler-options/win32res-compiler-option.md) derleyicisi seçeneğini kullanırken ve **%TEMP%** dizininizin DELETE izni yoksa, bu uyarı derleyicinin oluşturduğu geçici bir dosyayı silemeyeceğini gösterir.  
  
 **%TEMP%** dizini için izinleri okuduğunuz/yazdığınızdan/sildiğinizden emin olun.  
  
 Gerekirse, bu dosyaları el ile silebilirsiniz ve C# veya herhangi bir program için hiçbir zarar olmayacaktır.
