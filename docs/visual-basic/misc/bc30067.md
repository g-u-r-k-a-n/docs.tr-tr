# <a name="exit-function-is-not-valid-in-a-sub-or-property"></a>' Exit Function ', Sub veya Property içinde geçerli değildir

`Exit Function``Sub`yordamda veya `Property` yordamda görüntülenir. Bir `Exit` deyimin gerçekleştiği blokla eşleşmesi gerekir.  
  
 **Hata kimliği:** BC30067  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öğesini `Exit Function` `Exit Sub` veya `Exit Property` ifadesiyle uygun şekilde değiştirin.
