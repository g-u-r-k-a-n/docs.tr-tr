# <a name="compiler-error-cs0304"></a><span data-ttu-id="29166-103">Derleyici Hatası CS0304</span><span class="sxs-lookup"><span data-stu-id="29166-103">Compiler Error CS0304</span></span>

<span data-ttu-id="29166-104">New () kısıtlamasına sahip olmadığından ' Type ' değişken türünün bir örneği oluşturulamıyor</span><span class="sxs-lookup"><span data-stu-id="29166-104">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="29166-105">Bir genel sınıf uyguladığınızda ve bir `new` tür parametresi için sağlanan herhangi bir türün yeni bir örneğini oluşturmak için anahtar sözcüğünü kullanmak istiyorsanız, `T` Aşağıdaki örnekte gösterildiği gibi, [New () kısıtlamasını](../keywords/new-constraint.md) `T` sınıf bildiriminde uygulamanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="29166-105">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../keywords/new-constraint.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="29166-106">`new()`Kısıtlama, için sağlanan herhangi bir somut türün `T` Varsayılan, parametresiz bir oluşturucuya sahip olduğunu garanti ederek tür güvenliğini zorlar.</span><span class="sxs-lookup"><span data-stu-id="29166-106">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="29166-107">CS0304, `new` kısıtlama belirtmeyen bir tür parametresi örneği oluşturmak için sınıfının gövdesinde işleci kullanmayı denerseniz oluşur `T` `T` `new()` .</span><span class="sxs-lookup"><span data-stu-id="29166-107">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="29166-108">İstemci tarafında, kod parametresiz oluşturucusu olmayan bir türle genel sınıf örneğini oluşturmaya çalışırsa, bu kod [derleyici hatası CS0310](./cs0310.md)oluşturacaktır.</span><span class="sxs-lookup"><span data-stu-id="29166-108">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](./cs0310.md).</span></span>  
  
 <span data-ttu-id="29166-109">Aşağıdaki örnek CS0304 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="29166-109">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="29166-110">`new`Ayrıca, işlecine sınıf yöntemlerinde de izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="29166-110">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="29166-111">Hatayı önlemek için, `new()` Aşağıdaki örnekte gösterildiği gibi, kısıtlamasını kullanarak sınıfı bildirin.</span><span class="sxs-lookup"><span data-stu-id="29166-111">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
