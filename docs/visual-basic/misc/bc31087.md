# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>Dizi değiştiricileri hem değişken hem de değişkenin türü üzerinde belirtilemez
Dizi değiştiricileri hem değişken hem de türü üzerinde bulunur ve dizi dizisi belirtir.  
  
 **Hata kimliği:** BC31087  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizi değiştiricisini tür belirticisinden kaldırın.
