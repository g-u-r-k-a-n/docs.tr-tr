## <a name="methods"></a>Yöntemler  
  
|Yöntem|Description|  
|------------|-----------------|  
|[GetFileMapping Yöntemi](imetadatainfo-getfilemapping-method.md)|Eşlenen dosyanın bellek bölgesini ve eşleme türünü alır.|
