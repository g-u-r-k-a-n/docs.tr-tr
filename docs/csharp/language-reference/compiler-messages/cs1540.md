# <a name="compiler-error-cs1540"></a><span data-ttu-id="c7c4d-103">Derleyici Hatası CS1540</span><span class="sxs-lookup"><span data-stu-id="c7c4d-103">Compiler Error CS1540</span></span>
<span data-ttu-id="c7c4d-104">' Type1 ' türünde bir niteleyici aracılığıyla korunan ' Member ' üyesine erişilemez; Niteleyici ' type2 ' türünde (veya bundan türetilmiş) olmalıdır</span><span class="sxs-lookup"><span data-stu-id="c7c4d-104">Cannot access protected member 'member' via a qualifier of type 'type1'; the qualifier must be of type 'type2' (or derived from it)</span></span>  
  
 <span data-ttu-id="c7c4d-105">Türetilmiş bir [sınıf](../keywords/class.md) temel sınıfın bir örneği aracılığıyla temel sınıfının korunan üyelerine erişemez.</span><span class="sxs-lookup"><span data-stu-id="c7c4d-105">A derived [class](../keywords/class.md) cannot access protected members of its base class through an instance of the base class.</span></span> <span data-ttu-id="c7c4d-106">Türetilmiş sınıfta belirtilen temel sınıfın bir örneği, çalışma zamanında, aynı temelden türetilmiş ancak başka türlü türetilmiş sınıfla ilişkili olmayan bir türün örneği olabilir.</span><span class="sxs-lookup"><span data-stu-id="c7c4d-106">An instance of the base class declared in the derived class might, at run time, be an instance of another type that is derived from the same base but is not otherwise related to the derived class.</span></span> <span data-ttu-id="c7c4d-107">Korunan üyelere yalnızca türetilmiş türler tarafından erişilebildiğinden, çalışma zamanında geçerli olmayan korumalı üyelere erişim girişimleri derleyici tarafından geçerli değil olarak işaretlenir.</span><span class="sxs-lookup"><span data-stu-id="c7c4d-107">Because protected members can be accessed only by derived types, any attempts to access protected members that might not be valid at run time are marked by the compiler as not valid.</span></span>  
  
 <span data-ttu-id="c7c4d-108">`Employee`Aşağıdaki örnekteki sınıfında `emp2` ve `emp3` her ikisinde de `Person` derleme zamanında türü vardır, ancak `emp2` `Manager` çalışma zamanında türü vardır.</span><span class="sxs-lookup"><span data-stu-id="c7c4d-108">In the `Employee` class in the following example, `emp2` and `emp3` both have type `Person` at compile time, but `emp2` has type `Manager` at run time.</span></span> <span data-ttu-id="c7c4d-109">, `Employee` Öğesinden türetilmediği `Manager` için, temel sınıfın korunan üyelerine `Person` sınıfının bir örneği aracılığıyla erişemez `Manager` .</span><span class="sxs-lookup"><span data-stu-id="c7c4d-109">Because `Employee` is not derived from `Manager`, it cannot access the protected members of the base class, `Person`, through an instance of the `Manager` class.</span></span> <span data-ttu-id="c7c4d-110">Derleyici, iki nesnenin çalışma zamanı türünün ne olacağını belirleyemez `Person` .</span><span class="sxs-lookup"><span data-stu-id="c7c4d-110">The compiler cannot determine what the run-time type of the two `Person` objects will be.</span></span> <span data-ttu-id="c7c4d-111">Bu nedenle, hem çağrısı hem de `emp2` çağrı `emp3` nedeni DERLEYICI hatası CS1540.</span><span class="sxs-lookup"><span data-stu-id="c7c4d-111">Therefore, both the call from `emp2` and the call from `emp3` cause compiler error CS1540.</span></span>  
  
```csharp  
using System;  
using System.Text;  
  
namespace CS1540  
{  
    class Program1  
    {  
        static void Main()  
        {  
            Employee.PreparePayroll();  
        }  
    }  
  
    class Person  
    {  
        protected virtual void CalculatePay()
        {  
            Console.WriteLine("CalculatePay in Person class.");  
        }  
    }  
  
    class Manager : Person  
    {  
        protected override void CalculatePay()
        {  
            Console.WriteLine("CalculatePay in Manager class.");
  
        }  
    }  
  
    class Employee : Person  
    {  
        public static void PreparePayroll()  
        {  
            Employee emp1 = new Employee();  
            Person emp2 = new Manager();  
            Person emp3 = new Employee();  
            // The following line calls the method in the Employee base class,  
            // Person.  
            emp1.CalculatePay();
  
            // The following lines cause compiler error CS1540. The compiler
            // cannot determine at compile time what the run-time types of
            // emp2 and emp3 will be.  
            //emp2.CalculatePay();
            //emp3.CalculatePay();  
  
        }  
    }  
}  
```
