# <a name="syntax-error-visual-basic-error"></a>Sözdizimi hatası (Visual Basic hatası)

Kaynak kodu satırı, özel olmayan bir sözdizimi hatası oluşturur.  
  
 **Hata kimliği:** BC30035  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kaynak kodu satırında kullanılan anahtar sözcüklerin her biri için belgeleri inceleyin. Belgelerdeki örnek kodu, bu hatayı oluşturan kaynak satıra göre karşılaştırın.  
  
2. Hatanın nedenini belirleyemezseniz, bu koşullara ilişkin bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
