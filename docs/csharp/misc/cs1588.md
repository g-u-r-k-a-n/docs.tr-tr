# <a name="compiler-error-cs1588"></a>Derleyici hatası CS1588
Ortak dil çalışma zamanı dizini belirlenemiyor--' Reason '  
  
 Derleyici .NET çalışma zamanının yüklendiği dizini alamıyor. Bu hata, ortak dil çalışma zamanının yanlış yüklü olduğunu gösterir.
