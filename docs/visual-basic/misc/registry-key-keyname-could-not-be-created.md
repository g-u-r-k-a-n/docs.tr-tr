# <a name="registry-key-keyname-could-not-be-created"></a>' ' Kayıt defteri anahtarı \<keyname> oluşturulamadı
Belirtilen kayıt defteri anahtarı yetersiz güvenlik izinleri nedeniyle oluşturulamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Anahtara erişmek için yeterli güvenlik izninizin olduğundan emin olun.
