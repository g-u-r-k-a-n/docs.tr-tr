# <a name="reference-required-to-assembly-assemblyname-containing-the-type-membername"></a>' ' \<assemblyname> Türünü içeren ' ' derlemesine başvuru gerekiyor \<membername>

' ' Türünü içeren ' ' derlemesine başvuru gerekiyor \<assemblyname> \<membername> . Projenize bir tane ekleyin.  
  
 Proje, proje başvurularına dahil olmayan bir derlemeden başka bir türe bağımlı olan bir tür içerir.  
  
 **Hata kimliği:** BC30652  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Projedeki bağımlı türler için bir proje başvurusu ekleyin.
