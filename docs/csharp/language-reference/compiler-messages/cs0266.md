# <a name="compiler-error-cs0266"></a><span data-ttu-id="9bb8c-102">Derleyici Hatası CS0266</span><span class="sxs-lookup"><span data-stu-id="9bb8c-102">Compiler Error CS0266</span></span>

<span data-ttu-id="9bb8c-103">'type1' türünü örtülü olarak 'type2'ye dönüştüremez.</span><span class="sxs-lookup"><span data-stu-id="9bb8c-103">Cannot implicitly convert type 'type1' to 'type2'.</span></span> <span data-ttu-id="9bb8c-104">Açık bir dönüştürme var (bir dökümü eksik mi?)</span><span class="sxs-lookup"><span data-stu-id="9bb8c-104">An explicit conversion exists (are you missing a cast?)</span></span>

 <span data-ttu-id="9bb8c-105">Bu hata, kodunuz örtülü olarak dönüştürülemeyen, ancak açık bir dönüştürmenin kullanılabildiği iki tür arasında dönüştürmeye çalıştığında oluşur.</span><span class="sxs-lookup"><span data-stu-id="9bb8c-105">This error occurs when your code tries to convert between two types that cannot be implicitly converted, but where an explicit conversion is available.</span></span> <span data-ttu-id="9bb8c-106">Daha fazla bilgi için [bkz.](../../programming-guide/types/casting-and-type-conversions.md)</span><span class="sxs-lookup"><span data-stu-id="9bb8c-106">For more information, see [Casting and Type Conversions](../../programming-guide/types/casting-and-type-conversions.md).</span></span>

 <span data-ttu-id="9bb8c-107">Aşağıdaki kod, CS0266 üreten örnekleri gösterir:</span><span class="sxs-lookup"><span data-stu-id="9bb8c-107">The following code shows examples that generate CS0266:</span></span>

```csharp
// CS0266.cs
class MyClass
{
    public static void Main()
    {
        // You cannot implicitly convert a double to an integer.
        double d = 3.2;

        // The following line causes compiler error CS0266.
        int i1 = d;

        // However, you can resolve the error by using an explicit conversion.
        int i2 = (int)d;  

        // You cannot implicitly convert an object to a class type.
        object obj = new MyClass();

        // The following assignment statement causes error CS0266.
        MyClass myClass = obj;

        // You can resolve the error by using an explicit conversion.
        MyClass c = (MyClass)obj;

        // You cannot implicitly convert a base class object to a derived class type.
        MyClass mc = new MyClass();
        DerivedClass dc = new DerivedClass();

        // The following line causes compiler error CS0266.
        dc = mc;

        // You can resolve the error by using an explicit conversion.
        dc = (DerivedClass)mc;
    }  
}  
  
class DerivedClass : MyClass  
{  
}  
```
