# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="204db-103">Derleyici Uyarısı (düzey 1) CS1635</span><span class="sxs-lookup"><span data-stu-id="204db-103">Compiler Warning (level 1) CS1635</span></span>

<span data-ttu-id="204db-104">Genel olarak devre dışı bırakıldığı için ' uyarı kodu ' uyarısı geri yüklenemiyor</span><span class="sxs-lookup"><span data-stu-id="204db-104">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="204db-105">Bu uyarı, tüm derleme birimi için bir uyarı devre dışı bırakmak üzere **/nowarn** komut satırı seçeneğini veya proje ayarını kullanırsanız, ancak bu `#pragma warning restore` uyarıyı geri yüklemeyi denemek için kullandığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="204db-105">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="204db-106">Bu hatayı çözmek için,/nowarn komut satırı seçeneğini veya proje ayarını kaldırın veya `#pragma warning restore` komut satırı veya proje ayarları aracılığıyla devre dışı bırakamakta olduğunuz tüm uyarılar için öğesini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="204db-106">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="204db-107">Daha fazla bilgi için [#pragma uyarı](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) konusuna bakın.</span><span class="sxs-lookup"><span data-stu-id="204db-107">For more information, see the [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="204db-108">Aşağıdaki örnek CS1635 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="204db-108">The following sample generates CS1635:</span></span>  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
