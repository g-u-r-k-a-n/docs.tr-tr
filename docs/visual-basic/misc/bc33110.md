# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="6f998-103">İkili ' If ' işlecinin birinci ve ikinci işlenenleri için ortak bir tür çıkarsanamıyor</span><span class="sxs-lookup"><span data-stu-id="6f998-103">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="6f998-104">İkili ' If ' işlecinin birinci ve ikinci işlenenleri için ortak bir tür çıkarsanamıyor.</span><span class="sxs-lookup"><span data-stu-id="6f998-104">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="6f998-105">Birinin birbirlerine bir genişletme dönüştürmesi olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="6f998-105">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="6f998-106">İkili `If` işleç, bağımsız değişkenlerden biri ile diğer bağımsız değişkenden bir genişletme dönüştürmesi gerektirir.</span><span class="sxs-lookup"><span data-stu-id="6f998-106">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="6f998-107">Örneğin, ve arasında her iki yönde de genişleyen bir dönüştürme olmadığından `Integer` `String` , aşağıdaki kod bu hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="6f998-107">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim first? As Integer
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="6f998-108">**Hata kimliği:** BC33110</span><span class="sxs-lookup"><span data-stu-id="6f998-108">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="6f998-109">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="6f998-109">To correct this error</span></span>

- <span data-ttu-id="6f998-110">Kodunuzda Mümkünse, işlenenden biri için açık bir dönüştürme sağlayın:</span><span class="sxs-lookup"><span data-stu-id="6f998-110">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>

  ```vb
  Console.WriteLine(If(first, CInt(second)))
  ```

- <span data-ttu-id="6f998-111">Farklı bir koşullu oluşturma kullanarak kodu yeniden yazın.</span><span class="sxs-lookup"><span data-stu-id="6f998-111">Rewrite the code by using a different conditional construction.</span></span>

  ```vb
  If first IsNot Nothing Then
      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```
