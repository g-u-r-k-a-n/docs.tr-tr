# <a name="compiler-error-cs1019"></a><span data-ttu-id="b03bb-102">Derleyici Hatası CS1019</span><span class="sxs-lookup"><span data-stu-id="b03bb-102">Compiler Error CS1019</span></span>
<span data-ttu-id="b03bb-103">Aşırı yüklenebilir unary operatörü bekleniyor</span><span class="sxs-lookup"><span data-stu-id="b03bb-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="b03bb-104">Aşırı yüklenmiş boşaltılmış işleç gibi görünen bir şey beyan edildi, ancak operatör eksik veya imzada yanlış konumda.</span><span class="sxs-lookup"><span data-stu-id="b03bb-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="b03bb-105">Unary *işleci,* tek bir operand üzerinde çalışan bir işleçtir.</span><span class="sxs-lookup"><span data-stu-id="b03bb-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="b03bb-106">Örneğin, `++` bir unary işlecidir.</span><span class="sxs-lookup"><span data-stu-id="b03bb-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="b03bb-107">Anahtar kelimeyi `operator` kullanarak ve işlecinin üzerinde çalıştığı türe ait tek bir parametre belirterek bazı unary işleçleri aşırı yükleyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="b03bb-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="b03bb-108">Örneğin, kullanıcı tanımlı bir sınıf `++` `Temperature` için işleci aşırı yüklemek istiyorsanız, böylece yazabilirsiniz `Temperature++`, bu şekilde tanımlayabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="b03bb-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="b03bb-109">Bu hatayı aldığınızda, operatörün kendisinin eksik olması veya imzada yanlış konumda olması dışında, aşırı yüklenmiş boşaltıcı işleci gibi görünen bir şey beyan etmiş olursunuz.</span><span class="sxs-lookup"><span data-stu-id="b03bb-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="b03bb-110">Önceki örnekte `++` imzayı kaldırırsanız, CS1019 oluşturursunuz.</span><span class="sxs-lookup"><span data-stu-id="b03bb-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="b03bb-111">Aşağıdaki kod CS1019 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b03bb-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;
   }  
  
   public static void Main()  
   {  
   }  
}  
```
