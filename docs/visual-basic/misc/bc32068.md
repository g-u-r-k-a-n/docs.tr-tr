# <a name="local-variables-within-generic-methods-cannot-be-declared-static"></a>Genel metotlar içindeki yerel değişkenler ' static ' olarak bildirilemez

Genel yordam içindeki yerel bir değişkenin bildirimi, belirtir `Static` .  
  
 Visual Basic ve .NET Framework şu anda herhangi bir statik değişken ve genel yordam birleşimini desteklemez.  
  
 **Hata kimliği:** BC32068  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Static`Anahtar sözcüğünü değişken bildiriminden kaldırın.
