# <a name="type1-typename-and-type2-typename-conflict-in-namespace-namespacename"></a>\<type1>' ' \<typename> içinde ' ' ve \<type2> ' \<typename> ' çakışması \<namespace> \<namespacename>

Bir sınıf ve modül gibi tanımlanmış iki öğe aynı ad alanında aynı ada sahiptir.  
  
 **Hata kimliği:** BC30179  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öğelerinden birinin bildirimini farklı bir ad alanına taşıyın.  
  
- Öğelerden birinin adını değiştirin.
