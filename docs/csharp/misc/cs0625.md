# <a name="compiler-error-cs0625"></a><span data-ttu-id="933b8-103">Derleyici hatası CS0625</span><span class="sxs-lookup"><span data-stu-id="933b8-103">Compiler Error CS0625</span></span>

<span data-ttu-id="933b8-104">' alan ': StructLayout (LayoutKind. Explicit) ile işaretlenen örnek alan türleri bir FieldOffset özniteliğine sahip olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="933b8-104">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="933b8-105">Bir struct bir açık **StructLayout** özniteliğiyle işaretlendiğinde, yapı içindeki tüm alanlar [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) özniteliğine sahip olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="933b8-105">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="933b8-106">Daha fazla bilgi için bkz. [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="933b8-106">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span>

<span data-ttu-id="933b8-107">Aşağıdaki örnek CS0625 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="933b8-107">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
