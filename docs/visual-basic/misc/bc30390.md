# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>' ' olduğundan \<qualifiedmethod> Bu bağlamda ' ' erişilebilir değil \<modifier>

 **Hata kimliği:** BC30390  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değiştiricisini kaldırın.
