# <a name="division-by-zero-visual-basic-run-time-error"></a>Sıfıra bölme (Visual Basic çalışma zamanı hatası)
Bölen olarak kullanılan bir ifadenin değeri sıfır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İfadedeki değişkenlerin yazımını denetleyin. Yanlış yazılmış bir değişken adı, sıfır olarak başlatılan bir sayısal değişkeni örtülü olarak oluşturabilir.  
  
2. Deyimdeki değişkenlerde, özellikle de diğer yordamlardan bağımsız değişken olarak geçirilen işlemler için önceki işlemleri denetleyin.
