# <a name="events-in-interfaces-cannot-be-declared-implements"></a>Arabirimlerdeki olaylar ' ' olarak bildirilemez \<implements>

Arabirimlerde belirtilen olaylar, diğer arabirimlerin olaylarını uygulayamaz.  
  
 **Hata kimliği:** BC30688  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İfadesini kaldırın `Implements` .  
  
2. Olayı bir sınıf veya yapı içinde uygulayın.
