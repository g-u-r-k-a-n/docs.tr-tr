# <a name="initialization-error"></a>Başlatma hatası
Program başlatılırken bir hata oluştu. Bu, devam eden bir yükleme iptal etme sonucu olabilir.  
  
 **Hata kimliği:** BC2000  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yüklemeyi yeniden deneyin.
