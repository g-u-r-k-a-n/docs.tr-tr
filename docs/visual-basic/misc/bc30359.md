# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>' Default ', bir üzerinde yalnızca bir özellik adına uygulanabilir \<type>

`Default`Bir sınıf, arabirim veya yapı içinde birden fazla özellik adına uygulanmaya çalıştınız.  
  
 **Hata kimliği:** BC30359  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimlerden biri hariç tümünü kaldırın `Default` .
