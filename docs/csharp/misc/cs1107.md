# <a name="compiler-error-cs1107"></a><span data-ttu-id="63118-102">Derleyici Hatası CS1107</span><span class="sxs-lookup"><span data-stu-id="63118-102">Compiler Error CS1107</span></span>
<span data-ttu-id="63118-103">Bir parametrenin yalnızca bir 'değiştirici adı' değiştiriciolabilir.</span><span class="sxs-lookup"><span data-stu-id="63118-103">A parameter can only have one 'modifier name' modifier.</span></span>  
  
 <span data-ttu-id="63118-104">Parametre değiştiriciler için bir hatadır, `this`, `out` `ref` `in`, , ve bir parametre tanımında birden fazla kez görünmesi.</span><span class="sxs-lookup"><span data-stu-id="63118-104">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="63118-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="63118-105">Example</span></span>  
 <span data-ttu-id="63118-106">Aşağıdaki örnek, CS1107 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="63118-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
