# <a name="4810---discoverymessagewithnullmessagesequence"></a>4810 - DiscoveryMessageWithNullMessageSequence

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|4810|  
|Anahtar sözcükler|Bulma|  
|Level|Uyarı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Description  

 Bu olay, DiscoveryMessageSequence özelliğine sahip olmadığı için bulma iletisi DiscoveryClient tarafından bırakıldığında yayınlanır.  
  
## <a name="message"></a>İleti  

 MessageID = ' %2 ' olan bir %1 iletisi, DiscoveryMessageSequence özelliğine sahip olmadığı için DiscoveryClient tarafından bırakıldı.
