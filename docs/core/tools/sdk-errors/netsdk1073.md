# <a name="netsdk1073-the-frameworkreference-was-not-recognized"></a>NETSDK1073: FrameworkReference tanınmadı

**Bu makale şu şekilde geçerlidir:** ✔️ .NET 2.1.100 SDK ve sonraki sürümleri

Bu hata genellikle, SDK 'nın bulamadığı belirli bir FrameworkReference sürümü olduğu anlamına gelir. *Obj* ve *bin* klasörlerinizi silmeyi ve `dotnet restore` en son hedefleme paketlerini yeniden indirmek için çalıştırmayı deneyin.

Alternatif olarak, yüklemenize bir sorun olabilir, bu nedenle .NET ve Visual Studio 'nun en son sürümlerinde olduğunuzdan emin olun
