# <a name="compiler-error-cs0426"></a><span data-ttu-id="1b9c0-102">Derleyici Hatası CS0426</span><span class="sxs-lookup"><span data-stu-id="1b9c0-102">Compiler Error CS0426</span></span>
<span data-ttu-id="1b9c0-103">'Tanımlayıcı' türünde 'tanımlayıcı' türünde yok</span><span class="sxs-lookup"><span data-stu-id="1b9c0-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="1b9c0-104">Bu hata, başka bir tür içinde iç içe bir tür başvuru, ancak böyle bir iç içe tür yok oluşur.</span><span class="sxs-lookup"><span data-stu-id="1b9c0-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="1b9c0-105">İç içe geçen türün adını yanlış yazarsanız bu oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="1b9c0-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="1b9c0-106">Kullanılan adların yazımını denetleyin ve çevreleyen türün beklenen üyeye sahip olduğunu doğrulayın.</span><span class="sxs-lookup"><span data-stu-id="1b9c0-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="1b9c0-107">Aşağıdaki örnek CS0426 oluşturur, çünkü C sınıfı iç içe a türü yoktur:</span><span class="sxs-lookup"><span data-stu-id="1b9c0-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
