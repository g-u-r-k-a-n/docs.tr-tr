# <a name="compiler-error-cs1613"></a><span data-ttu-id="3d57c-102">Derleyici hatası CS1613</span><span class="sxs-lookup"><span data-stu-id="3d57c-102">Compiler Error CS1613</span></span>
<span data-ttu-id="3d57c-103">' Interface ' arabirimi için ' class ' yönetilen coclass sarmalayıcı sınıfı bulunamıyor (bir derleme başvurunuz mu eksik?)</span><span class="sxs-lookup"><span data-stu-id="3d57c-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="3d57c-104">Bir arabirimden COM nesnesi örneğini oluşturma girişiminde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="3d57c-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="3d57c-105">Arabirimde **ComImport** ve `CoClass` öznitelikleri vardır, ancak derleyici `CoClass` özniteliği için verilen türü bulamıyor.</span><span class="sxs-lookup"><span data-stu-id="3d57c-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="3d57c-106">Bu hatayı çözmek için aşağıdakilerden birini deneyebilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="3d57c-106">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="3d57c-107">Coclass 'a sahip olan derlemeye bir başvuru ekleyin (çoğu zaman arabirimin ve coclass 'ın aynı derlemede olması gerekir).</span><span class="sxs-lookup"><span data-stu-id="3d57c-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="3d57c-108">Bilgi için bkz. [başvuru](../language-reference/compiler-options/reference-compiler-option.md) veya [başvuru ekleme iletişim kutusu](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) .</span><span class="sxs-lookup"><span data-stu-id="3d57c-108">See [-reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="3d57c-109">Arabirimindeki `CoClass` özniteliğini onarın.</span><span class="sxs-lookup"><span data-stu-id="3d57c-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="3d57c-110">Aşağıdaki örnek, **CoClassAttribute**'in doğru kullanımını gösterir:</span><span class="sxs-lookup"><span data-stu-id="3d57c-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
