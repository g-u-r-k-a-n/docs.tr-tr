# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>' ' \<propertyname> \<membername> , ' \<type> \<typename> ' içinde aynı ada sahip bir üyeyle çakışan ' ' öğesini örtük olarak tanımlıyor

Bir tür üyesinin adı, bir özellik için örtük olarak oluşturulan üyenin adıyla çakışıyor. Özellikler, tanımlı üyelerle çakışabilecek birkaç örtük değişken örtülü olarak oluşturulur.  
  
 **Hata kimliği:** BC31060  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Adlandırma çakışmasını kaldırmak için açıkça belirtilen üyeyi yeniden adlandırın.
