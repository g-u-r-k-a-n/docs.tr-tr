# <a name="specifier-is-not-valid-on-an-interface-property-declaration"></a>' \<specifier> ', arabirim özelliği bildiriminde geçerli değil

`Property`Arabirim içindeki bir ifade, gibi geçersiz bir anahtar sözcük içeriyor `Implements` . Arabirim yalnızca üyeleri tanımlayabilir, uygulamaz.  
  
 **Hata kimliği:** BC30273  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirim ifadesinden geçersiz anahtar sözcüğü kaldır.  
  
- Arabirim üyelerinin uygulamasını arabirimini uygulayan bir sınıfa taşıyın.
