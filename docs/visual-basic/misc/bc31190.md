# <a name="xml-literals-and-xml-axis-properties-are-not-available"></a>XML sabit değerleri ve XML eksen özellikleri kullanılamıyor
XML sabit değerleri ve XML eksen özellikleri kullanılamaz. System. xml, System. xml. LINQ ve System. Core 'a başvurular ekleyin.  
  
 Derlenen kod, XML değişmez değerlerini veya xml eksen özelliklerini içerir, ancak XML değişmez değerlerini veya XML eksen özelliklerini derlemek için gereken derlemelere başvurmaz.  
  
 **Hata kimliği:** BC31190  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- System. xml. dll, System. xml. Linq. dll ve System. Core. dll derlemelerine başvurular ekleyin.
