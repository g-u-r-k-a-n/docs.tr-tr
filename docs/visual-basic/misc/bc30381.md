# <a name="finally-can-only-appear-once-in-a-try-statement"></a>' TRY ' ifadesinde yalnızca bir kez ' finally ' bulunabilir

`Finally` bir bloğu tamamlaması için kullanılır `Try...Catch...Finally` ; Bu nedenle, bloğun sonunda yalnızca bir kez yer alabilir.  
  
 **Hata kimliği:** BC30381  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Gereksiz öğesini bulun ve kaldırın `Finally` .
