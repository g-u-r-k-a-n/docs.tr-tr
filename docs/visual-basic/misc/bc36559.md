<span data-ttu-id="20e4e-103">Anonim tür özelliğinin türü oluşturuluncaya kadar, başka bir özelliğin türünü oluşturmak için kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="20e4e-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="20e4e-104">Örneğin, şu bildirimde `.IDName = .LastName` henüz başlatılmamış olduğundan geçerli değil `.LastName` .</span><span class="sxs-lookup"><span data-stu-id="20e4e-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

