# <a name="error-setting-assembly-manifest-option-error-message"></a>Bütünleştirilmiş kod bildirim seçeneği ayarlanırken hata oluştu: \<error message>

Visual Basic derleyici, bildirime sahip bir derleme oluşturmak için derleme bağlayıcı (Al.exe, Alink olarak da bilinir) çağırır. Bağlayıcı, bir anahtar dosyası veya derlemenin imzalanacağı anahtar adını bulmakta bir hata bildirdi.  
  
 **Hata kimliği:** BC30139  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Alıntı yapılan hata iletisini inceleyin ve daha fazla açıklama ve öneri için [Al.exe (derleme bağlayıcı)](../../framework/tools/al-exe-assembly-linker.md) sayfasında listelenen hatalara başvurun.  
  
2. Hata devam ederse, koşullar hakkındaki bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
