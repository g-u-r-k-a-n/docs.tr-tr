|||  
|-|-|  
|ID|3815|  
|Anahtar sözcükler|Yönlendirme hizmetleri|  
|Level|Bilgi|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Description  

 Bu olay, yönlendirme hizmeti bir iletiyi işlerken yayınlanır.

 Yönlendirme hizmeti KIMLIK: ' %1 ', eylem: ' %2 ', gelen URL: ' %3 ', Işlem: %4 ile alınmış bir iletiyi işliyor.
