# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="69025-103">Derleyici Uyarısı (düzey 3) CS0419</span><span class="sxs-lookup"><span data-stu-id="69025-103">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="69025-104">Cref özniteliğinde belirsiz başvuru: ' Method name1 '.</span><span class="sxs-lookup"><span data-stu-id="69025-104">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="69025-105">' Method AD2 ' varsayılıyor, ancak ' Method name3 ' dahil diğer aşırı yüklemelerin de eşleşebilirdi.</span><span class="sxs-lookup"><span data-stu-id="69025-105">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="69025-106">Koddaki XML belge açıklamasında bir başvuru çözümlenemedi.</span><span class="sxs-lookup"><span data-stu-id="69025-106">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="69025-107">Bu, yöntem aşırı yüklenmişse veya aynı ada sahip iki farklı tanımlayıcı bulunursa meydana gelebilir.</span><span class="sxs-lookup"><span data-stu-id="69025-107">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="69025-108">Uyarıyı çözmek için, başvuruyu ortadan kaldırmak için nitelenmiş bir ad kullanın veya belirli bir aşırı yüklemeyi parantez içine ekleyin.</span><span class="sxs-lookup"><span data-stu-id="69025-108">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="69025-109">Aşağıdaki örnek CS0419 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="69025-109">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
