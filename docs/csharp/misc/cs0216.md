# <a name="compiler-error-cs0216"></a><span data-ttu-id="dfad0-103">Derleyici hatası CS0216</span><span class="sxs-lookup"><span data-stu-id="dfad0-103">Compiler Error CS0216</span></span>
<span data-ttu-id="dfad0-104">' Operator ' işleci, eşleşen ' missing_operator ' işlecinin de tanımlanmasını gerektirir</span><span class="sxs-lookup"><span data-stu-id="dfad0-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="dfad0-105">Kullanıcı tanımlı bir [true](../language-reference/operators/true-false-operators.md) işleci, Kullanıcı tanımlı bir [false](../language-reference/operators/true-false-operators.md) işleci gerektirir, tersi de geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="dfad0-105">A user-defined [true](../language-reference/operators/true-false-operators.md) operator requires a user-defined [false](../language-reference/operators/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="dfad0-106">Aşağıdaki örnek CS0216 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="dfad0-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
