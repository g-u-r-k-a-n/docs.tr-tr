# <a name="specified-registry-key-is-not-valid-because-it-contains-two-or-more-consecutive-backslashes"></a>Belirtilen kayıt defteri anahtarı iki veya daha fazla ardışık ters eğik çizgi içerdiğinden geçerli değil

Bir yol ile belirtilen bir kayıt defteri anahtarı iki veya daha fazla ardışık ters eğik çizgi içerir. Bu, yolu oluşturmak için birkaç dizeyi birleştirmenin ve yanlışlıkla çok fazla ters eğik çizgi dahil olmak üzere bir sonuç olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirtilen kayıt defteri anahtarını inceleyerek, ek ters eğik çizgileri nerede ve neden eklendiğini belirlersiniz.
