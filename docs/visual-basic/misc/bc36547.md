# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="053f1-103">Anonim tür üyesi veya ' \<propertyname> ' özelliği zaten tanımlanmış</span><span class="sxs-lookup"><span data-stu-id="053f1-103">Anonymous type member or property '\<propertyname>' is already declared</span></span>

<span data-ttu-id="053f1-104">Bir özellik adı, anonim bir türün bildiriminde yalnızca bir kez oluşturulabilir.</span><span class="sxs-lookup"><span data-stu-id="053f1-104">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="053f1-105">Örneğin, aşağıdaki bildirimler geçerli değildir:</span><span class="sxs-lookup"><span data-stu-id="053f1-105">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="053f1-106">**Hata kimliği:** BC36547</span><span class="sxs-lookup"><span data-stu-id="053f1-106">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="053f1-107">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="053f1-107">To correct this error</span></span>  
  
- <span data-ttu-id="053f1-108">Özelliklerden biri için farklı bir ad seçin.</span><span class="sxs-lookup"><span data-stu-id="053f1-108">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
- <span data-ttu-id="053f1-109">Ad ve değerleri verdiğiniz değişkenler veya özellik adları için yeni adlar sağlayın.</span><span class="sxs-lookup"><span data-stu-id="053f1-109">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
