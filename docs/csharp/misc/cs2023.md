# <a name="compiler-warning-level-1-cs2023"></a>Derleyici Uyarısı (düzey 1) CS2023

Bir yanıt dosyası içinde belirtildiğinden/noconfig seçeneği yoksayılıyor  
  
 [-Noconfig](../language-reference/compiler-options/miscellaneous.md#noconfig) derleyici seçeneği bir yanıt dosyasında belirtildi; buna izin verilmez.  
  
 Bu uyarı, [**Disableduyarılar**](../language-reference/compiler-options/errors-warnings.md#disabledwarnings) seçeneği tarafından bastırılamaz.
