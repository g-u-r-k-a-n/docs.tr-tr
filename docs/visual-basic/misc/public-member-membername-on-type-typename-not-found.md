# <a name="public-member-membername-on-type-typename-not-found"></a>\<membername>' ' Türündeki ' ' ortak üyesi \<typename> bulunamadı

Erişmeye çalıştığınız üye bulunamıyor. Bu hata, geç bağlantılı nesnelerin genel olmayan üyelerine erişilirken ortaya çıkabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Üye varsa, olarak bildirildiği için olduğundan emin olun `Public` .
