# <a name="properties-declared-readonly-cannot-have-a-set"></a>' ReadOnly ' olarak tanımlanan özelliklerde ' Set ' bulunamaz
`Set`Yordam bir özelliğin değerini yazar. `ReadOnly`Özellikler yazılamıyor.  
  
 **Hata kimliği:** BC30022  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `ReadOnly`Deyimden anahtar sözcüğü kaldırın `Property` veya `Set` yordamı Özellik gövdesinden kaldırın.
