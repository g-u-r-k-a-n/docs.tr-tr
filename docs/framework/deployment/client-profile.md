# <a name="net-framework-client-profile"></a>.NET Framework İstemci Profili
.NET İstemci Profili,.NET Framework 4 ve önceki sürümlerle sağlanan ve istemci uygulamaları için optimize edilmiş .NET Framework'ün bir alt kümesidir. .NET Framework, Windows, Windows Phone ve Microsoft Azure için bir geliştirme platformudur ve yönetilen bir uygulama yürütme ortamı ve .NET Framework sınıf kitaplığı sağlar. .NET Framework 4 ve önceki sürümler iki dağıtım seçeneği sunar: tam .NET Framework ve İstemci Profili. İstemci Profili daha hızlı dağıtım ve daha küçük uygulama yükleme paketlerini tam .NET Framework'ten etkinleştirdi.  
  
 .NET Framework 4.5'ten başlayarak İstemci Profili durduruldu ve yalnızca yeniden dağıtılabilir paketin tamamı kullanılabilir. .NET Framework 4.5 tarafından sağlanan ve daha küçük indirme boyutu ve daha hızlı dağıtım gibi optimizasyonlar, ayrı bir dağıtım paketi gereksinimini ortadan kaldırmış olur. Tek yeniden dağıtılabilir yükleme işlemini kolaylaştırır ve uygulamaya dağıtım seçeneklerinizi daha da basitleştirir.  
  
 Ancak, .NET Framework 4 veya 3.5'i hedefliyorsanız ve İstemci Profili hakkında daha fazla bilgi edinmek ve ne zaman kullanacağınız hakkında daha fazla bilgi edinmek istiyorsanız, .NET Framework 4 belgelerindeki [.NET Framework İstemci Profili'ne](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/cc656912%28v=vs.100%29) bakın.  
  
 .NET Framework 4.5'i yüklediğinizde,.NET Framework 4 İstemci Profili .NET Framework'ün tam sürümüne güncelleştirilir. .NET Framework 4.5'i yükleme hakkında daha fazla bilgi [için geliştiriciler için .NET Framework'e bakın.](../install/guide-for-developers.md)
