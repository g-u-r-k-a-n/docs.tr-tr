<span data-ttu-id="99885-104">Tutarsız erişilebilirlik: ' Interface 1 ' temel arabirimi, ' interface 2 ' arabiriminden daha az erişilebilir</span><span class="sxs-lookup"><span data-stu-id="99885-104">Inconsistent accessibility: base interface 'interface 1' is less accessible than interface 'interface 2'</span></span>  
  
 <span data-ttu-id="99885-105">[Ortak](../language-reference/keywords/public.md) bir yapı herkese açık bir şekilde erişilebilir nesne döndürmelidir.</span><span class="sxs-lookup"><span data-stu-id="99885-105">A [public](../language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>  
  
 <span data-ttu-id="99885-106">Arabirim erişilebilirliği türetilmiş bir arabirimde daha dar olamaz.</span><span class="sxs-lookup"><span data-stu-id="99885-106">Interface accessibility cannot be narrowed in a derived interface.</span></span> <span data-ttu-id="99885-107">Daha fazla bilgi için bkz. [arabirimler](../programming-guide/interfaces/index.md) ve [erişim değiştiricileri](../programming-guide/classes-and-structs/access-modifiers.md).</span><span class="sxs-lookup"><span data-stu-id="99885-107">For more information, see [Interfaces](../programming-guide/interfaces/index.md) and [Access Modifiers](../programming-guide/classes-and-structs/access-modifiers.md).</span></span>  
  
 <span data-ttu-id="99885-108">Aşağıdaki örnek CS0061 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="99885-108">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
