# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Nullable değiştiricisi hem değişken hem de türü üzerinde belirtilemez

Hem tanımlanmış bir değişken hem de bildirildiği tür için Nullable tür değiştiricisi (?) eklenmiştir.  
  
 **Hata kimliği:** BC33100  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Null yapılabilir tür değiştiricisini (?) değişkenden ya da onun bildirildiği türden kaldırın.
