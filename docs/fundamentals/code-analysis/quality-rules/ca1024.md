<span data-ttu-id="571f9-149">Hata ayıklayıcının, özellikleri uygulayarak, özellikleri genişletmeyi önleyebilirsiniz <xref:System.Diagnostics.DebuggerBrowsableAttribute?displayProperty=fullName> .</span><span class="sxs-lookup"><span data-stu-id="571f9-149">You can prevent the debugger from autoexpanding properties by applying <xref:System.Diagnostics.DebuggerBrowsableAttribute?displayProperty=fullName>.</span></span> <span data-ttu-id="571f9-150">Aşağıdaki örnek, bir örnek özelliğine uygulanan bu özniteliği gösterir.</span><span class="sxs-lookup"><span data-stu-id="571f9-150">The following example shows this attribute being applied to an instance property.</span></span>
