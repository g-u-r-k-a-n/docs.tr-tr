# <a name="notoverridable-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>' NotOverridable ', başka bir yöntemi geçersiz kılmayan metotlarda belirtilemez
Özellikler ve yöntemler `NotOverridable` Varsayılan olarak ' dir. `NotOverridable`Değiştirici yalnızca, başka bir özelliği veya yöntemi geçersiz kılan yöntemlerde kullanılabilir.  
  
 **Hata kimliği:** BC31088  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `NotOverridable`Farklı bir üyeyi geçersiz kılmayan özelliklerden ve metotlardan değiştiricisini kaldırın.
