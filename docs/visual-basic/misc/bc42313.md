# <a name="xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property"></a>' WriteOnly ' özelliğinde ' döndürüyor ' XML açıklama etiketine izin verilmez

' WriteOnly ' özelliğinde ' döndürüyor ' XML açıklama etiketine izin verilmez. XML açıklaması yoksayılacak.  
  
 Salt yazılır özellik bildirimi için bir XML yorumu \<returns> etiketi içeremez.  
  
 **Hata kimliği:** BC42313  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Desteklenmeyen etiketi kaldırın \<returns> .
