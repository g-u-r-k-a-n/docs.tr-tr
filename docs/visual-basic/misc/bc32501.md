# <a name="microsoftvisualbasiccomclassattribute-and-attribute-cannot-both-be-applied-to-the-same-class"></a>' Microsoft. VisualBasic. ComClassAttribute ' ve ' \<attribute> ' aynı sınıfa birlikte uygulanamaz

`COMClassAttribute`Öznitelik bloğu, com nesneleri için uygulanan bir öznitelikle birlikte kullanılır. Olası bir neden .NET Framework ve COM özniteliklerini karıştırdır.  
  
 **Hata kimliği:** BC32501  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `COMClassAttribute`Öznitelik bloğunu veya com için uygulanan özniteliği kaldırın.
