# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
{0}ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute değeri true olarak ayarlandığından, hizmet örneği ' ' işleminin tamamlamada serbest bırakıldı.  
  
## <a name="description"></a>Açıklama  
 Geçerli etkin işlemin tamamlanması ve ReleaseServiceInstanceOnTransactionComplete olarak ayarlanması nedeniyle geçerli hizmet örneği serbest bırakıldığında veya atıldığı zaman izleniyor `true` .
