# <a name="custom-modifier-can-only-be-used-immediately-before-an-event-declaration"></a>' Custom ' değiştiricisi yalnızca ' Event ' bildiriminden hemen önce kullanılabilir
`Custom`Anahtar sözcüğü yalnızca `Event` özel bir olay bildirimindeki anahtar kelimeden önce gelmelidir.  
  
 **Hata kimliği:** BC31140  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Anahtar sözcüğü `Custom` `Event` anahtar kelimeden önce taşıyın.  
  
     —veya—  
  
- `Custom`Anahtar sözcüğünü kaldırın.
