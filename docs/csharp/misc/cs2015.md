# <a name="compiler-error-cs2015"></a>Derleyici hatası CS2015

' File ' bir metin dosyası yerine bir ikili dosyadır  
  
 Bir ikili dosya olan derleyiciye bir dosya geçirildi. Derleyici bir kaynak kodu dosyası bekliyor.
