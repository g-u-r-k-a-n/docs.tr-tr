# <a name="compiler-error-cs1567"></a>Derleyici Hatası CS1567

Win32 kaynağı oluşturulurken hata oluştu: ' dosya '  
  
 Derlemeniz [**Win32Icon**](../compiler-options/resources.md#win32icon) derleyici seçeneğini kullandı veya bir derleyicinin kaynak bilgilerini içeren bir dosya oluşturmasına neden olan [**Win32Resource**](../compiler-options/resources.md#win32resource)kullanmadı, ancak derleyici yetersiz disk alanı veya başka bir hata nedeniyle dosyayı oluşturamadı.  
  
 Dosya oluşturma sorununu çözemezse, kaynak bilgilerini içeren bir dosya oluşturabilen [**Win32Resource**](../compiler-options/resources.md#win32resource)kullanabilirsiniz.
