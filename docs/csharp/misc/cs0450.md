# <a name="compiler-error-cs0450"></a>Derleyici hatası CS0450
' Tür parametre adı ': hem bir kısıtlama sınıfı hem de ' class ' veya ' struct ' kısıtlaması belirtilemez  
  
 Bir tür parametresi yapı türü kısıtlaması tarafından kısıtlandığından, yapı ve sınıf birbirini dışlayan Kategoriler olduğundan, bu, belirli bir sınıf türü tarafından sınırlandırılır ve mantıksal olarak çelişkili. Bir tür parametresi belirli bir sınıf türü kısıtlaması tarafından kısıtlandığından, tanım, sınıf türü kısıtlaması tarafından kısıtlanıyor ve bu nedenle sınıf türü kısıtlamasını belirtmek gereksizdir.  
  
## <a name="example"></a>Örnek  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
