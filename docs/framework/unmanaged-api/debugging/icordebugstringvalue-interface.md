# <a name="icordebugstringvalue-interface"></a>ICorDebugStringValue Arabirimi
Dize değerlerine uygulanan ICorDebugHeapValue öğesinin alt sınıfı.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetLength Yöntemi](icordebugstringvalue-getlength-method.md)|Bu `ICorDebugStringValue`başvurduğu dizedeki karakter sayısını alır.|  
|[GetString Yöntemi](icordebugstringvalue-getstring-method.md)|Bu `ICorDebugStringValue`başvurulan dizeyi alır.|
