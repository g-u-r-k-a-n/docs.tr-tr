# <a name="compiler-error-cs0027"></a><span data-ttu-id="b3ef1-103">Derleyici hatası CS0027</span><span class="sxs-lookup"><span data-stu-id="b3ef1-103">Compiler Error CS0027</span></span>
<span data-ttu-id="b3ef1-104">' This ' anahtar sözcüğü geçerli bağlamda kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="b3ef1-104">Keyword 'this' is not available in the current context</span></span>  
  
 <span data-ttu-id="b3ef1-105">[Bu](../language-reference/keywords/this.md) anahtar sözcük, bir özelliğin, yöntemin veya oluşturucunun dışında bulundu.</span><span class="sxs-lookup"><span data-stu-id="b3ef1-105">The [this](../language-reference/keywords/this.md) keyword was found outside of a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="b3ef1-106">Bu hatayı gidermek için, anahtar sözcüğünün kullanımını ortadan kaldırmak `this` ve/veya bir özelliğin, yöntemin veya oluşturucunun içindeki deyimin kısmını veya tamamını taşımak için ifadeyi değiştirin.</span><span class="sxs-lookup"><span data-stu-id="b3ef1-106">To fix this error, either modify the statement to eliminate use of the `this` keyword, and/or move part or all of the statement inside a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="b3ef1-107">Aşağıdaki örnek CS0027 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b3ef1-107">The following example generates CS0027:</span></span>  
  
```csharp  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```
