<span data-ttu-id="a0508-104">Bir dönüştürme işleci, parametre türünden türetilen bir dönüş türüyle birlikte bildirilmiştir.</span><span class="sxs-lookup"><span data-stu-id="a0508-104">A conversion operator is declared with a return type that derives from the parameter type.</span></span>  
  
 <span data-ttu-id="a0508-105">Derleme zamanında Visual Basic, önceden tanımlanmış bir dönüştürmeyi herhangi bir başvuru türünden devralma hiyerarşisinde herhangi bir türe, yani türettiği veya ondan türetilen herhangi bir türe kabul eder.</span><span class="sxs-lookup"><span data-stu-id="a0508-105">At compile time, Visual Basic considers a predefined conversion to exist from any reference type to any type in its inheritance hierarchy, that is, any type from which it derives or which derives from it.</span></span> <span data-ttu-id="a0508-106">Bu tür bir dönüştürme çalışma zamanında başarısız olabilir, ancak derleyici çalışma zamanı sonuçlarını tahmin edemez, bu nedenle bu tür dönüştürmenin derlenmesi için izin verir.</span><span class="sxs-lookup"><span data-stu-id="a0508-106">Such a conversion might fail at run time, but the compiler cannot predict run-time results, so it allows any such conversion to compile.</span></span>
