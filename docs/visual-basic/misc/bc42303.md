# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>XML açıklaması bir metot veya özellik içinde bulunamaz

Bir yöntem veya özellik içinde bir XML açıklaması bulunamaz. XML açıklaması yoksayılacak. XML açıklama blokları metotlar veya Özellikler içinde bulunamaz.  
  
 **Hata kimliği:** BC42303  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Açıklama bloğunu kaldırın veya daha uygun bir konuma taşıyın.
