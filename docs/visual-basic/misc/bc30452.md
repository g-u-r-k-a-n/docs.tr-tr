# <a name="operator-operatorname-is-not-defined-for-types-type1-and-type2"></a>' ' İşleci ' ' \<operatorname> \<type1> ve ' ' türleri için tanımlı değil \<type2>

Dahil edilen iki değer döndüren kod öğesi için geçerli olmayan bir işlem gerçekleştirmek için bir ikili işleç kullanmaya çalıştınız.  
  
 **Hata kimliği:** BC30452  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İki öğeyi kontrol edin ve uyumlu olduklarından emin olun.
