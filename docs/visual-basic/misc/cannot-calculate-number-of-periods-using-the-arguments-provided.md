# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>Belirtilen bağımsız değişkenler kullanılarak dönemlerin sayısı hesaplanamıyor

İşleve yapılan bir çağrı, `NPer` gerekli tüm bağımsız değişkenleri içermez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Rate`, `Prnt` Ve `PV` değerlerinin işlev çağrısına eklendiğinden emin olun.
