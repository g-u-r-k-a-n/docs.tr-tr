# <a name="compiler-error-cs0170"></a><span data-ttu-id="c25f6-102">Derleyici Hatası CS0170</span><span class="sxs-lookup"><span data-stu-id="c25f6-102">Compiler Error CS0170</span></span>
<span data-ttu-id="c25f6-103">Muhtemelen atanmamış alan 'alan' kullanımı</span><span class="sxs-lookup"><span data-stu-id="c25f6-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="c25f6-104">Bir yapıdaki alan ilk olarak başharflere alınmadan kullanılmıştır.</span><span class="sxs-lookup"><span data-stu-id="c25f6-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="c25f6-105">Bu sorunu çözmek için, önce hangi alanın baş harfe getirilmemiş olduğunu belirleyin ve sonra erişmeye çalışmadan önce sorunu baş harfe getirin.</span><span class="sxs-lookup"><span data-stu-id="c25f6-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="c25f6-106">Structs'u başlatma hakkında daha fazla bilgi için [Yapı türlerine](../language-reference/builtin-types/struct.md)bakın.</span><span class="sxs-lookup"><span data-stu-id="c25f6-106">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="c25f6-107">Aşağıdaki örnek CS0170 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c25f6-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because
                                         //e.i was never assigned  
   }  
}  
```
