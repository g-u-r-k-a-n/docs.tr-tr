# <a name="end-operator-expected"></a>' End operator ' bekleniyor
Bir `End` deyimin, sonlandırmakta olduğu blokla eşleşmesi gerekir. `Operator`Yordam tanımı düzgün sonlandırılmadı.  
  
 **Hata kimliği:** BC33005  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `End Operator`Yordamın sonuna bir ifade ekleyin.
