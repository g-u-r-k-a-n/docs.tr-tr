# <a name="compiler-error-cs0081"></a><span data-ttu-id="e36d6-103">Derleyici hatası CS0081</span><span class="sxs-lookup"><span data-stu-id="e36d6-103">Compiler Error CS0081</span></span>

<span data-ttu-id="e36d6-104">Tür parametresi bildirimi bir tür değil bir tanımlayıcı olmalıdır</span><span class="sxs-lookup"><span data-stu-id="e36d6-104">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="e36d6-105">Genel bir yöntem veya tür bildirdiğinizde, tür parametresini tanımlayıcı olarak belirtin, örneğin "T" veya "InputType".</span><span class="sxs-lookup"><span data-stu-id="e36d6-105">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="e36d6-106">İstemci kodu yöntemini çağırdığında, yöntem veya sınıf gövdesinde tanımlayıcı oluşumlarını değiştiren türü sağlar.</span><span class="sxs-lookup"><span data-stu-id="e36d6-106">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="e36d6-107">Daha fazla bilgi için bkz. [genel tür parametreleri](../programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="e36d6-107">For more information, see [Generic Type Parameters](../programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
