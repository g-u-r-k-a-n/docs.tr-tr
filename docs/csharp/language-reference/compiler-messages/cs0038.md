# <a name="compiler-error-cs0038"></a><span data-ttu-id="19455-102">Derleyici Hatası CS0038</span><span class="sxs-lookup"><span data-stu-id="19455-102">Compiler Error CS0038</span></span>

<span data-ttu-id="19455-103">Dış tip 'type1' için nonstatik bir üyeye iç içe geçmiş tip '2' üzerinden erişemez</span><span class="sxs-lookup"><span data-stu-id="19455-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>

 <span data-ttu-id="19455-104">Sınıftaki bir alan iç içe olan bir sınıf için otomatik olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="19455-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="19455-105">İç içe geçmiş bir sınıfın kullanılabilmesi için alanın [statik](../keywords/static.md)olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="19455-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="19455-106">Aksi takdirde, dış sınıfın bir örneğini oluşturmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="19455-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="19455-107">Daha fazla bilgi için İç [Içe Türler'e](../../programming-guide/classes-and-structs/nested-types.md)bakın.</span><span class="sxs-lookup"><span data-stu-id="19455-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>

 <span data-ttu-id="19455-108">Aşağıdaki örnek CS0038 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="19455-108">The following sample generates CS0038:</span></span>

```csharp
// CS0038.cs
class OuterClass
{
    public int count;
    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();
            // int count2 = class_inst.count;
            int count2 = count;   // CS0038
        }
    }

    public static void Main()
    {
    }
}
```
