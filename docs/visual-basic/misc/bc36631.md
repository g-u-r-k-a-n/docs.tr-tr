 <span data-ttu-id="b802c-104">**Hata kimliği:** BC36631</span><span class="sxs-lookup"><span data-stu-id="b802c-104">**Error ID:** BC36631</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b802c-105">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="b802c-105">To correct this error</span></span>  
  
1. <span data-ttu-id="b802c-106">`Join` `Group Join` Aşağıdaki örnekte gösterildiği gibi yan tümcesine anahtar sözcüğü ekleyin:</span><span class="sxs-lookup"><span data-stu-id="b802c-106">Add the `Join` keyword to the `Group Join` clause, as shown in the following example:</span></span>

```vb  
            Join var2 In collection2 _  
            Group Join var3 In collection3 _  
            On var2.ID Equals var3 Into Matches = Group _  
            On var1 Equals var2.ID _  
            Select JoinID = var1, var2.Name, Matches  
```
