# <a name="property-propertyname-cannot-be-declared-default-because-it-overrides-a-property-on-the-base-type-typename-that-is-not-default"></a>\<propertyname> \<type> \<typename> Varsayılan olmayan ' ' tabanında bir özelliği geçersiz kıldığından ' ' özelliği ' default ' olarak bildirilemez
`Default`Temel sınıfta bildirilmemiş bir özelliği geçersiz kıldığından geçerli bir bildirim olmayan bir özellik bildirmeye çalıştınız `Default` .  
  
 **Hata kimliği:** BC30504  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimi kaldırın `Default` .
