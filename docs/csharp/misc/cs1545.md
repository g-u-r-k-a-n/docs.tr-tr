# <a name="compiler-error-cs1545"></a><span data-ttu-id="c3528-103">Derleyici hatası CS1545</span><span class="sxs-lookup"><span data-stu-id="c3528-103">Compiler Error CS1545</span></span>
<span data-ttu-id="c3528-104">' Property ' özelliği, Dizin Oluşturucusu veya olayı dil tarafından desteklenmiyor; ' set accessor ' veya ' Get accessor ' erişimci yöntemlerini doğrudan çağırmayı deneyin</span><span class="sxs-lookup"><span data-stu-id="c3528-104">Property, indexer, or event 'property' is not supported by the language; try directly calling accessor methods 'set accessor' or 'get accessor'</span></span>  
  
 <span data-ttu-id="c3528-105">Kod, varsayılan olmayan bir [dizin oluşturucuya](../programming-guide/indexers/index.md) sahip olan ve dizinli sözdizimini kullanmaya çalışmış bir nesne kullanıyor.</span><span class="sxs-lookup"><span data-stu-id="c3528-105">The code is consuming an object that has a non-default [indexer](../programming-guide/indexers/index.md) and tried to use the indexed syntax.</span></span> <span data-ttu-id="c3528-106">Bu hatayı çözmek için, özelliğin `get` veya `set` erişimci yöntemini çağırın.</span><span class="sxs-lookup"><span data-stu-id="c3528-106">To resolve this error, call the property's `get` or `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c3528-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="c3528-107">Example</span></span>  
  
```cpp  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="c3528-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="c3528-108">Example</span></span>  
 <span data-ttu-id="c3528-109">Aşağıdaki örnek CS1545 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c3528-109">The following sample generates CS1545.</span></span>

```csharp  
   public static void Main() {  
      Manager Ed = new Manager();

   }  
}  
```
