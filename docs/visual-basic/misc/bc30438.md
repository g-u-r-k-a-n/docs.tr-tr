# <a name="constants-must-have-a-value"></a>Sabitler bir değere sahip olmalıdır

Bir değeri atamadan bir sabit olarak bildirdiniz.  
  
 **Hata kimliği:** BC30438  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Sabitine bir değer atayın.
