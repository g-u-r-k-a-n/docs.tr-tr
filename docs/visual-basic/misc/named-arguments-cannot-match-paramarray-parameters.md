# <a name="named-arguments-cannot-match-paramarray-parameters"></a>Adlandırılmış bağımsız değişkenler ParamArray parametreleriyle eşleşemez

Parametre dizilerinin değere göre geçirilmesi gerekir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bağımsız değişkenleri `ByVal` , onları adlandırarak değil, geçirin.
