# <a name="isymunmanagedscope2-interface"></a>ISymUnmanagedScope2 Arabirimi
Bir yöntem içindeki bir sözcük temelli kapsamı temsil eder. Bu arabirim, [ıstreamunmanagedscope](isymunmanagedscope-interface.md) arabirimini kapsam içinde tanımlanan sabitler hakkında bilgi alan yöntemlerle genişletir.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetConstantCount Yöntemi](isymunmanagedscope2-getconstantcount-method.md)|Bu kapsam içinde tanımlanan sabitlerin sayısını alır.|  
|[GetConstants Yöntemi](isymunmanagedscope2-getconstants-method.md)|Bu kapsam içinde tanımlanan yerel sabitleri alır.|  
  
## <a name="requirements"></a>Gereksinimler  
 **Üst bilgi:** CorSym. IDL, CorSym. h
