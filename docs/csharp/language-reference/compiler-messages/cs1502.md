# <a name="compiler-error-cs1502"></a><span data-ttu-id="87f43-103">Derleyici Hatası CS1502</span><span class="sxs-lookup"><span data-stu-id="87f43-103">Compiler Error CS1502</span></span>

<span data-ttu-id="87f43-104">' Name ' için en iyi aşırı yüklenmiş yöntem eşleşmesi bazı geçersiz bağımsız değişkenlere sahip</span><span class="sxs-lookup"><span data-stu-id="87f43-104">The best overloaded method match for 'name' has some invalid arguments</span></span>  
  
 <span data-ttu-id="87f43-105">Bu hata, yönteme geçilen bağımsız değişken türleri bu yöntemin parametre türleriyle eşleşmediği zaman oluşur.</span><span class="sxs-lookup"><span data-stu-id="87f43-105">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="87f43-106">Çağrılan yöntem aşırı yüklenmişse, aşırı yüklenmiş sürümlerden hiçbirinin geçilen bağımsız değişken türleriyle eşleşen bir imzası yoktur.</span><span class="sxs-lookup"><span data-stu-id="87f43-106">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="87f43-107">Bu sorunu çözmek için aşağıdakilerden birini yapın:</span><span class="sxs-lookup"><span data-stu-id="87f43-107">To resolve this problem, do one of the following:</span></span>  
  
- <span data-ttu-id="87f43-108">Geçilen bağımsız değişkenlerin türlerini çift kontrol edin.</span><span class="sxs-lookup"><span data-stu-id="87f43-108">Double-check the types of the arguments being passed.</span></span> <span data-ttu-id="87f43-109">Çağrılmakta olan yöntemin bağımsız değişkenleriyle eşleştiğinden emin olun.</span><span class="sxs-lookup"><span data-stu-id="87f43-109">Make sure that they match the arguments of the method being called.</span></span>

- <span data-ttu-id="87f43-112">Uygunsa, gönderilmekte olan parametre türleriyle eşleşecek şekilde metodun başka bir aşırı yüklenmiş sürümünü tanımlayın.</span><span class="sxs-lookup"><span data-stu-id="87f43-112">If appropriate, define another overloaded version of the method to match the parameter types that are being sent.</span></span>
