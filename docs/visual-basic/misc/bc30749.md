# <a name="unable-to-access-member"></a>Üyeye erişilemiyor

Belirtilen üyeye erişilemedi.  
  
 **Hata kimliği:** BC30749  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Üyenin tanımlandığını ve üye adını doğru yazdığınızdan emin olun.  
  
2. Modülde belirtilen üyelerden birine erişmeyi deneyin. Bazı durumlarda, bildirildiği modüller henüz yüklenmediği için hata ayıklama ortamı üyeleri bulamaz.
