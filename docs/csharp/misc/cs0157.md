 <span data-ttu-id="3a8df-105">[Finally](../language-reference/keywords/try-catch-finally.md) yan tümcesindeki tüm deyimlerin yürütülmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="3a8df-105">All of the statements in a [finally](../language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="3a8df-106">Daha fazla bilgi için bkz. [özel durumlar ve özel durum işleme](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="3a8df-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="3a8df-107">Aşağıdaki örnek CS0157 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3a8df-107">The following sample generates CS0157:</span></span>  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
