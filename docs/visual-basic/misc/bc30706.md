# <a name="expression-or-statement-is-not-valid-in-debug-windows"></a>İfade veya deyim hata ayıklama penceresinde geçerli değil
`Try...Catch...Finally`deyimler hata ayıklama bağlamında kullanılamaz.  
  
 **Hata kimliği:** BC30706  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Try...Catch...Finally`Hata ayıklama kodundaki deyimleri kaldır.
