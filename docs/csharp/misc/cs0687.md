## <a name="example"></a><span data-ttu-id="aac88-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="aac88-108">Example</span></span>  
 <span data-ttu-id="aac88-109">Aşağıdaki örnek CS0687 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="aac88-109">The following sample generates CS0687:</span></span>  
  
```csharp  
// CS0687.cs

using System;  
  
public class Test
{  
    public static int x = 77;  
  
    public static void Main()
    {  
        Console.WriteLine(M::x); // CS0687  
        // To resolve use the following line instead:  
        // Console.WriteLine(M.x);  
    }

```
