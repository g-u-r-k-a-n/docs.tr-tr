# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>' Operator ' işleci ' type1 ' ve ' type2 ' türleri için tanımlı değil

' Operator ' işleci ' type1 ' ve ' type2 ' türleri için tanımlı değil. ' ' Öğesinin ilk öğesinin dize değerini almak için ' Value ' özelliğini kullanabilirsiniz \<parentElement> .  
  
 Bir ikili veya karşılaştırmada bir XML sabit değeri kullanıldı `Boolean` .  
  
 **Hata kimliği:** BC31195  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Value`İkili veya karşılaştırma IÇIN XML sabit değerinin özelliğini kullanın `Boolean` .
