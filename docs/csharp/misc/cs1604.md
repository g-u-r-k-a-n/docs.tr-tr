# <a name="compiler-error-cs1604"></a>Derleyici hatası CS1604
' Variable ' öğesine atanamıyor çünkü salt okunurdur  
  
 Salt okunurdur bir değişkene bir atama yapıldı. Bu hatayı önlemek için, bu bağlamda bu değişkene atama veya değiştirme girişiminde bulunuldu.
