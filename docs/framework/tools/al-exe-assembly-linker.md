# <a name="alexe-assembly-linker"></a>Al.exe (Derleme Bağlayıcı)

Assembly Linker, modül veya kaynak dosyalar olan bir veya daha fazla dosyadan, derleme bildirimi içeren bir dosya oluşturur. Modül, derleme bildirimi içermeyen bir ara dil (IL) dosyasıdır.

> [!NOTE]
> Visual Studio 2008'den başlayarak, hem C# hem de Visual Basic derleyicileri otomatik olarak bir Win32 manifestosunu derlemeye katıştırır. Daha fazla bilgi için [-win32manifest (C# Derleyici Seçenekleri)](../../csharp/language-reference/compiler-options/win32manifest-compiler-option.md)bakın.

Bu araç, Visual Studio ile birlikte otomatik olarak yüklenir. To run the tool, use the Developer Command Prompt for Visual Studio (or the Visual Studio Command Prompt in Windows 7). Daha fazla bilgi için [Komut İstemleri'ne](developer-command-prompt-for-vs.md)bakın.

Komut satırına şunu yazın:

## <a name="syntax"></a>Sözdizimi

```console
al sources options
```

## <a name="remarks"></a>Açıklamalar

Tüm Visual Studio derleyicileri derlemeler üretir. Ancak, bir veya daha fazla modül (bir bildirim olmadan meta veri) varsa, ayrı bir dosyada bildirim içeren bir derleme oluşturmak için *Al.exe'yi* kullanabilirsiniz.

Önbelleğe derlemeler yüklemek için, önbellekten derlemeleri kaldırmak veya önbelleğin içeriğini listelemek için [Genel Derleme Önbellek Aracı'nı *(Gacutil.exe)*](gacutil-exe-gac-tool.md)kullanın.

## <a name="example"></a>Örnek

Aşağıdaki `t2.netmodule` komut, modülden bir derleme ile çalıştırılabilir bir dosya *t2a.exe* oluşturur. Giriş `MyClass`noktası' `Main` ndaki yöntemdir.

```console
al t2.netmodule /target:exe /out:t2a.exe /main:MyClass.Main
```
