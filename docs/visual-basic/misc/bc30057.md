# <a name="too-many-arguments-to-procedurename"></a>' ' İçin çok fazla bağımsız değişken var \<procedurename>
Bir yordam çağrısında sağlanan bağımsız değişkenlerin sayısı, yordam için tanımlanan bağımsız değişken sayısını aşıyor.  
  
 **Hata kimliği:** BC30057  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yordamın tanımına başvurun ve çağrıdan ek bağımsız değişkenleri kaldırın.
