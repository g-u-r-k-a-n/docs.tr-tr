# <a name="compiler-error-cs0669"></a>Derleyici hatası CS0669

ComImport özniteliğine sahip bir sınıfın Kullanıcı tanımlı Oluşturucusu olamaz.

Ortak dil çalışma zamanındaki COM birlikte çalışma katmanı, [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) sınıfları için Oluşturucu sağlar. Sonuç olarak, bir COM nesnesi çalışma zamanında yönetilen bir nesne olarak kullanılabilir.

Aşağıdaki örnek CS0669 oluşturur:

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
