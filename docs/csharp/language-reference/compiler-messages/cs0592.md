# <a name="compiler-error-cs0592"></a><span data-ttu-id="f3df3-102">Derleyici Hatası CS0592</span><span class="sxs-lookup"><span data-stu-id="f3df3-102">Compiler Error CS0592</span></span>

<span data-ttu-id="f3df3-103">Öznitelik 'öznitelik' bu bildirim türünde geçerli değildir.</span><span class="sxs-lookup"><span data-stu-id="f3df3-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="f3df3-104">Yalnızca 'tür' bildirimlerinde geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="f3df3-104">It is valid on 'type' declarations only.</span></span>  
  
 <span data-ttu-id="f3df3-105">Bir öznitelik tanımladığınızda, bir `AttributeTargets` değer belirterek uygulanabilecek yapıları tanımlarsınız.</span><span class="sxs-lookup"><span data-stu-id="f3df3-105">When you define an attribute, you define what constructs it can be applied to by specifying an `AttributeTargets` value.</span></span> <span data-ttu-id="f3df3-106">Aşağıdaki örnekte, `MyAttribute` öznitelik belirtir, çünkü `AttributeUsage` öznitelik, yalnızca arabirimlere `AttributeTargets.Interface`uygulanabilir.</span><span class="sxs-lookup"><span data-stu-id="f3df3-106">In the following example, the `MyAttribute` attribute can be applied to interfaces only, because the `AttributeUsage` attribute specifies `AttributeTargets.Interface`.</span></span> <span data-ttu-id="f3df3-107">Öznitelik bir sınıfa (sınıf) `A`uygulandığından hata oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="f3df3-107">The error is generated because the attribute is applied to a class (class `A`).</span></span>  
  
## <a name="example"></a><span data-ttu-id="f3df3-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="f3df3-108">Example</span></span>

 <span data-ttu-id="f3df3-109">Aşağıdaki örnek CS0592 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f3df3-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
