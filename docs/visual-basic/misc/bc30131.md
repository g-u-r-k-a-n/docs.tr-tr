# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>' ' Güvenlik özniteliği \<attributename> bir modüle uygulanamaz

Ortak dil çalışma zamanı, modüldeki güvenlik özniteliğini kabul etmez.

**Hata kimliği:** BC30131

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- Özniteliği bildirimden kaldırın.
