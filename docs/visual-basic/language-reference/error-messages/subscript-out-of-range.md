# <a name="subscript-out-of-range-visual-basic"></a>Alt simge aralık dışı (Visual Basic)

Bir dizi alt simge, izin verilen aralığın dışında olduğundan geçerli değil. Bir boyut için en düşük alt simge değeri her zaman 0 ' dır ve en yüksek alt indis değeri `GetUpperBound` Bu boyut için yöntem tarafından döndürülür.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Alt indisi geçerli aralık dahilinde olacak şekilde değiştirin.
