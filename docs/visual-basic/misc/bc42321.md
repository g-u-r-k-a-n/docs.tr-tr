# <a name="badly-formed-xml-file-commentfile-cannot-be-included"></a>Hatalı oluşturulmuş XML. ' \<commentfile> ' Dosyası dahil edilemez

`<include>`XML açıklamalarındaki bir etiket, varolmayan bir dosyayı veya kötü BIÇIMLI XML içeren bir dosyayı ifade eder.  
  
 **Hata kimliği:** BC42321  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosyanın var olduğunu doğrulayın.  
  
2. Dosyadaki XML 'nin geçerli olduğunu doğrulayın.
