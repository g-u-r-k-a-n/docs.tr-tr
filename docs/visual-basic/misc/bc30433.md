# <a name="methods-in-a-module-cannot-be-declared-specifier"></a>Modül içindeki metotlar ' ' olarak bildirilemez \<specifier>
Bir deyimin içindeki bir yöntemde geçerli olmayan bir tanımlayıcı kullandınız `Module` . Modüller hiçbir şekilde örneklenemez, devralmayı desteklemez ve arabirimler uygulayamaz.  
  
 **Hata kimliği:** BC30433  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirteci kaldırın.
