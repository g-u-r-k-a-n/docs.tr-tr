 <span data-ttu-id="17c60-108">Aşağıdaki örnek CS0438 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="17c60-108">The following example generates CS0438:</span></span>

```csharp  
   public class A { }

```  
  
 <span data-ttu-id="17c60-110">Sonra bu dosyayı derleyin:</span><span class="sxs-lookup"><span data-stu-id="17c60-110">Then compile this file:</span></span>

```csharp  
   public class A { }  
}  
```  
  
 <span data-ttu-id="17c60-111">Sonra bu dosyayı derleyin:</span><span class="sxs-lookup"><span data-stu-id="17c60-111">And then compile this file:</span></span>

```csharp  
{  
   public static void Main() {  
      Console.WriteLine(typeof(Util.A));   // CS0438  
   }  
}  
```
