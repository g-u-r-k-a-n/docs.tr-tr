# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>' RemoveHandler ' bildirimi eşleşen bir ' End RemoveHandler ' ile bitmelidir

Bir `RemoveHandler` bildirimin ifadesiyle bitmesi gerekir `End RemoveHandler` .  
  
 **Hata kimliği:** BC31116  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `RemoveHandler`Bildirimin bir ifadesiyle bitmediğini denetleyin `End RemoveHandler` .
