# <a name="option-optionname-requires-argument"></a>Seçenek \<optionname> gerekir \<argument>

Gerekli bir bağımsız değişken atlandı.  
  
 **Hata kimliği:** BC2006  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Eksik bağımsız değişkeni sağlayın.
