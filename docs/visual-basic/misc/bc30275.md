# <a name="specifier-is-not-valid-on-an-interface-event-declaration"></a>' \<specifier> ', arabirim olay bildiriminde geçerli değil

`Event`Arabirim içindeki bir ifade, izin verilmeyen bir anahtar sözcük içerir, örneğin `Implements` . Arabirim yalnızca üyeleri tanımlayabilir, uygulamaz.  
  
 **Hata kimliği:** BC30275  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Anahtar sözcüğünü bildirim ifadesinden kaldırın.  
  
2. Arabirim üyelerinin uygulamasını arabirimini uygulayan bir sınıfa taşıyın.
