# <a name="method-cannot-contain-both-a-try-statement-and-an-on-error-or-resume-statement"></a>Yöntem hem bir ' TRY ' ifadesiyle hem de ' On Error ' veya ' özgeçmişi ' ifadesine sahip olamaz

`Try`Veya ile bir ifadesini birleştirdik `On Error` `Resume` .  
  
 **Hata kimliği:** BC30544  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Veya ' i kaldırın `On Error` `Resume` .
