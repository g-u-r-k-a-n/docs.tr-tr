 <span data-ttu-id="e09b9-108">Aşağıdaki örnek CS0155 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e09b9-108">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs

{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }

    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
