# <a name="file-filename-cannot-be-deleted-because-it-is-open"></a>' \<filename> ' Dosyası açık olduğundan silinemiyor

Açık bir dosya üzerinde silme yapılmaya çalışıldı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Silmeden önce dosyayı kapatın.
