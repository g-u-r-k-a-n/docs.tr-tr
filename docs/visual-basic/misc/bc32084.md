<span data-ttu-id="b9ff0-105">`New`Kısıtlama, bu tür parametresine sağlanan tür bağımsız değişkeninin, içinden nesneler oluşturan koda erişilebilen parametresiz bir oluşturucuyu kullanıma sunması gerektiği anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="b9ff0-105">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="b9ff0-106">Tüm değer türlerinde parametresiz oluşturucular vardır, ancak tüm başvuru türleri uygulanmaz.</span><span class="sxs-lookup"><span data-stu-id="b9ff0-106">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="b9ff0-107">Bu nedenle `Structure` kısıtlama kısıtlamayı karşılar `New` , ancak `Class` kısıtlama veya bir sınıf ya da arabirim adı değildir.</span><span class="sxs-lookup"><span data-stu-id="b9ff0-107">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>

<span data-ttu-id="b9ff0-109">Sınıf `c2` öğesinden bir nesne oluşturduğunda `c1` , tür parametresi `u` `New` tür parametresindeki kısıtlamayı karşılamıyor `t` .</span><span class="sxs-lookup"><span data-stu-id="b9ff0-109">When class `c2` creates an object from `c1`, type parameter `u` does not satisfy the `New` constraint on type parameter `t`.</span></span>
