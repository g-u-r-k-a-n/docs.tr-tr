# <a name="compiler-error-cs0466"></a><span data-ttu-id="9f795-103">Derleyici hatası CS0466</span><span class="sxs-lookup"><span data-stu-id="9f795-103">Compiler Error CS0466</span></span>
<span data-ttu-id="9f795-104">' Method1 ', ' Method2 ' olmadığı için bir params parametresine sahip olmamalıdır</span><span class="sxs-lookup"><span data-stu-id="9f795-104">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="9f795-105">`params`Uygulanan arabirim bunu kullanmıyorsa, sınıf üyesi üzerinde parametresini kullanamazsınız.</span><span class="sxs-lookup"><span data-stu-id="9f795-105">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9f795-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="9f795-106">Example</span></span>  
 <span data-ttu-id="9f795-107">Aşağıdaki örnek CS0466 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="9f795-107">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
