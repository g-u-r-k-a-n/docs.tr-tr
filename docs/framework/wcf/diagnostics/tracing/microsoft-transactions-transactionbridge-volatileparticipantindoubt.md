# <a name="microsofttransactionstransactionbridgevolatileparticipantindoubt"></a>Microsoft.Transactions.TransactionBridge.VolatileParticipantInDoubt

WS-AT protokol hizmeti, tanınmayan bir geçici katılımcıdan hazırlanmış veya yeniden yürütme iletisi aldı. Katılımcıya bir hata döndürüldü, işlemin sonucunun şüpheli olduğunu bildirir.  
  
## <a name="description"></a>Description  

 Yerel Işlem Yöneticisi zaten unutmakta olan geçici bir listeden hazırlanmış veya yeniden yürütme iletisi aldığında izlenebilir.  
  
## <a name="troubleshooting"></a>Sorun giderme  

 Geçici katılımcıdan gelen geç mesajların olası nedenlerini araştırın.
