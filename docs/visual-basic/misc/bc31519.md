# <a name="filename-cannot-be-referenced-because-it-is-not-an-assembly"></a>\<filename>Derleme olmadığından ' ' öğesine başvurulamaz

.NET derlemesi olmayan bir dosyaya başvuru yapıldı.  
  
 **Hata kimliği:** BC31519  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Derlemenin adını ve konumunu doğrulayın.  
  
2. Derlemenin hasarlı veya bozuk olmadığını doğrulayın.
