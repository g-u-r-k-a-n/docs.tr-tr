# <a name="isymunmanagedsymbolsearchinfogethresult-method"></a>ISymUnmanagedSymbolSearchInfo::GetHRESULT Yöntemi
HRESULT 'yi alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT GetHRESULT(  
    [out] HRESULT *phr);  
```  
  
## <a name="parameters"></a>Parametreler  
 `phr`
