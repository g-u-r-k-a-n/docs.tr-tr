# <a name="xml-documentation-parse-error-start-tag-tag-doesnt-have-a-matching-end-tag"></a>XML belgesi ayrıştırma hatası: ' ' başlangıç etiketinin \<tag> eşleşen bitiş etiketi yok

XML belgesi ayrıştırma hatası: başlangıç etiketinin \<tag> eşleşen bitiş etiketi yok. XML açıklaması yoksayılacak.  
  
 XML yorumu başlangıç etiketi içeriyor ancak eşleşen bir bitiş etiketi içermiyor.  
  
 **Hata kimliği:** BC42316  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Başlangıç etiketiyle eşleşen bir bitiş etiketi ekleyin.  
  
     veya  
  
- Etiket, gibi bir iç metin içermiyorsa, [\<seealso>](../language-reference/xmldoc/seealso.md) kapanış açılı ayracından önce eğik çizgi belirtin.
