# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed

Bir komşu bağlantı açma girişimi başarısız oldu.  
  
## <a name="description"></a>Açıklama  

 Bu izleme, bir komşu bağlantı kurma denemesinden sonra oluşur. Olası nedenler, yanıt vermeyen bir çözümleyici, yanlış bağlama ayarı veya geçici ağ arızalarına sahip olabilir.  
  
## <a name="troubleshooting"></a>Sorun giderme  

 Her iki düğümün de geçerli adreslere sahip olduğundan ve uygun şekilde yapılandırıldığından emin olun.
