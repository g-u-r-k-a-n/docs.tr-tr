# <a name="property1-and-property2-cannot-overload-each-other-because-only-one-is-declared-default"></a>\<property1> \<property2> yalnızca biri ' default ' olarak bildirildiği için ' ' ve ' ' birbirini aşırı yükleyemez

Bir özellik belirtiyorsa `Default` , bu adda aşırı yüklenmiş tüm özellikler de belirtilmelidir `Default` .  
  
 **Hata kimliği:** BC30361  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tüm aşırı yüklenmiş özelliklerin bildirildiği emin olun `Default` .
