# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="50ba5-102">Derleyici Uyarısı (seviye 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="50ba5-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="50ba5-103">Verilen ifade her zaman sağlanan ('type') türündedir</span><span class="sxs-lookup"><span data-stu-id="50ba5-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="50ba5-104">Koşullu bir deyim her zaman **doğru**değerlendirirse, o zaman koşullu bir ifade gerekmez.</span><span class="sxs-lookup"><span data-stu-id="50ba5-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="50ba5-105">Bu **uyarı, is** işleci kullanarak bir türü değerlendirmeye çalıştığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="50ba5-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="50ba5-106">Değerlendirme bir değer türüyse, denetim gereksizdir.</span><span class="sxs-lookup"><span data-stu-id="50ba5-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="50ba5-107">Aşağıdaki örnek CS0183 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="50ba5-107">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);
   }  
}  
```
