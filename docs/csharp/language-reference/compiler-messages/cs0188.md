## <a name="example"></a><span data-ttu-id="d77d1-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="d77d1-107">Example</span></span>

 <span data-ttu-id="d77d1-108">Aşağıdaki örnek CS0188 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d77d1-108">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
