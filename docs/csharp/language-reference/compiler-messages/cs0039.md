# <a name="compiler-error-cs0039"></a><span data-ttu-id="b9d8c-102">Derleyici Hatası CS0039</span><span class="sxs-lookup"><span data-stu-id="b9d8c-102">Compiler Error CS0039</span></span>

<span data-ttu-id="b9d8c-103">'Type1' türünü referans dönüştürme, kutulama dönüştürme, kutulama dönüştürmeyi açma, sarma dönüştürme veya null tür dönüştürme yoluyla 'type2'ye dönüştüremez</span><span class="sxs-lookup"><span data-stu-id="b9d8c-103">Cannot convert type 'type1' to 'type2' via a reference conversion, boxing conversion, unboxing conversion, wrapping conversion, or null type conversion</span></span>

## <a name="example"></a><span data-ttu-id="b9d8c-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="b9d8c-105">Example</span></span>

<span data-ttu-id="b9d8c-106">Aşağıdaki örnek CS0039 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b9d8c-106">The following example generates CS0039:</span></span>

```csharp
using System;

class A { }
class B : A { }
class C : A { }

class Example
{
    static void Main()
    {
        C c;

        // This compiles, because
        // there is an explicit reference conversion from type A to type C.
        A a = new C();
        c = a as C;

        // This generates CS0039, because be there is no
        // there is no implicit or explicit reference conversion between B and C types.
        B b = new B();
        c = b as C;  // CS0039
    }
}
```
