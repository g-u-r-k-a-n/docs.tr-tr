## <a name="example"></a><span data-ttu-id="7b565-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="7b565-108">Example</span></span>  
 <span data-ttu-id="7b565-109">Aşağıdaki örnek CS3007 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7b565-109">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
