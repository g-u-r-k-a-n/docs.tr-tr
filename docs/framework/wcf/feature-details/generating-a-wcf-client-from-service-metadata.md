<span data-ttu-id="863ae-104">Bu konuda, meta veri belgelerinden istemciler oluşturmak için Svcutil.exe çeşitli anahtarların nasıl kullanılacağı açıklanmaktadır.</span><span class="sxs-lookup"><span data-stu-id="863ae-104">This topic describes how to use the various switches in Svcutil.exe to generate clients from metadata documents.</span></span>

- <span data-ttu-id="863ae-108">Sağlanan adrese WS-MetadataExchange (MEX) isteği.</span><span class="sxs-lookup"><span data-stu-id="863ae-108">WS-MetadataExchange (MEX) request to the supplied address.</span></span>

 <span data-ttu-id="863ae-111">Svcutil.exe, hizmetten alınan Web Hizmetleri Açıklama Dili (WSDL) veya ilke dosyasını temel alarak istemciyi oluşturur.</span><span class="sxs-lookup"><span data-stu-id="863ae-111">Svcutil.exe generates the client based on the Web Services Description Language (WSDL) or policy file received from the service.</span></span> <span data-ttu-id="863ae-112">Kullanıcı asıl adı (UPN), Kullanıcı adının "" ile bitiştirerek \@ ve tam nitelikli etki alanı adı (FQDN) eklenerek oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="863ae-112">The user principal name (UPN) is generated by concatenating the user name with "\@" and then adding a fully-qualified domain name (FQDN).</span></span> <span data-ttu-id="863ae-113">Ancak, Active Directory kaydolan kullanıcılar için, bu biçim geçerli değil ve aracın oluşturduğu UPN Kerberos kimlik doğrulamasında şu hata iletisiyle bir hataya neden oluyor: **oturum açma girişimi başarısız oldu.**</span><span class="sxs-lookup"><span data-stu-id="863ae-113">However, for users who registered on Active Directory, this format is not valid and the UPN that the tool generates causes a failure in the Kerberos authentication with the following error message: **The logon attempt failed.**</span></span> <span data-ttu-id="863ae-114">Bu sorunu çözmek için, aracın ürettiği istemci dosyasını el ile onarın.</span><span class="sxs-lookup"><span data-stu-id="863ae-114">To resolve this problem, manually fix the client file that the tool generated.</span></span>
