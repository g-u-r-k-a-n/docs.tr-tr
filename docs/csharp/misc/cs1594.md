# <a name="compiler-error-cs1594"></a>Derleyici hatası CS1594

' Delegate ' temsilcisinin bazı geçersiz bağımsız değişkenleri var  
  
 Bir [temsilci](../language-reference/builtin-types/reference-types.md) çağrısına geçirilen bir bağımsız değişkenin türü, temsilci bildirimindeki parametrenin türünü kabul etmez.  
  
 Aşağıdaki örnek CS1594 oluşturur:  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
