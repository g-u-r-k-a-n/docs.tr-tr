# <a name="install-the-net-framework-on-windows-10-and-windows-server-2016-and-later"></a>.NET Framework'u Windows 10 ve Windows Server 2016 ve sonraki

.NET Framework Windows'da birçok uygulama çalıştırmak için gereklidir. Bu makaledeki yönergeler, ihtiyacınız olan .NET Framework sürümlerini yüklemenize yardımcı olmalıdır. [.NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) en son kullanılabilir sürümüdür.

Bir uygulamayı çalıştırmaya çalıştıktan ve makinenizde aşağıdakine benzer bir iletişim gördükten sonra bu sayfaya gelmiş olabilirsiniz:

![Bu uygulama başlatılamadı](./media/this-application-could-not-be-started.png)

## <a name="net-framework-48"></a> .NET Framework 4.8

.NET Framework 4.8 aşağıdakilerle birlikte

- [Windows 10 Mayıs 2019 Güncelleştirmesi](https://support.microsoft.com/help/4028685/windows-10-get-the-update)

> [!div class="button"]
> [.NET Framework 4.8'i İndirin](https://dotnet.microsoft.com/download/dotnet-framework/net48)

[.NET Framework 4.8,.NET](https://dotnet.microsoft.com/download/dotnet-framework/net48) Framework 4.0 ile 4.7.2 için oluşturulmuş uygulamaları çalıştırmak için kullanılabilir.

[.NET Framework 4.8'i](https://dotnet.microsoft.com/download/dotnet-framework/net48) şu üzerine yükleyebilirsiniz:

- Windows 10 Ekim 2018 Güncelleştirmesi (sürüm 1809)
- Windows 10 Nisan 2018 Güncelleştirmesi (sürüm 1803)
- Windows 10 Güz Creators Güncelleme (sürüm 1709)
- Windows 10 Creators Update (sürüm 1703)
- Windows 10 Yıldönümü Güncelleştirmesi (sürüm 1607)
- Windows Server 2019
- Windows Server, sürüm 1809
- Windows Server, sürüm 1803
- Windows Server 2016

.NET Framework 4.8 aşağıdaki konularda desteklenmez:

- Windows 10 1507
- Windows 10 1511

Windows 10 1507 veya 1511 kullanıyorsanız ve .NET Framework 4.8'i yüklemek istiyorsanız, öncelikle daha sonraki bir Windows 10 sürümüne yükseltmeniz gerekir.

## <a name="net-framework-462"></a>.NET Framework 4.6.2

[.NET Framework 4.6.2,](https://dotnet.microsoft.com/download/dotnet-framework/net462) Windows 10 1507 ve 1511'de en son desteklenen .NET Framework sürümüdür.

.NET Framework 4.6.2 ,.NET Framework 4.0 ile 4.6.2 için oluşturulmuş uygulamaları destekler.

## <a name="net-framework-35"></a>.NET Framework 3.5

[Windows 10'a .NET Framework 3.5'i](dotnet-35-windows-10.md)yüklemek için yönergeleri izleyin.

.NET Framework 3.5, .NET Framework 1.0 ile 3.5 arasında oluşturulmuş uygulamaları destekler.

## <a name="additional-information"></a>Ek bilgiler

.NET Framework 4.x sürümleri, önceki sürümlere yer güncelleştirmeleridir. Bu aşağıdaki anlamına gelir:

- Makinenizde .NET Framework 4.x'in yalnızca bir sürümü yüklenebilir.

- Daha sonraki bir sürüm zaten yüklenmişse, .NET Framework'ün önceki bir sürümünü makinenize yükleyemezsiniz.

- .NET Framework'ün 4.x sürümleri, .NET Framework 4.0 için oluşturulmuş uygulamaları bu sürüm üzerinden çalıştırmak için kullanılabilir. Örneğin, .NET Framework 4.7 ,.NET Framework 4.0 ile 4.7 için oluşturulmuş uygulamaları çalıştırmak için kullanılabilir. En son sürüm (.NET Framework 4.8), 4.0 ile başlayan .NET Framework'ün tüm sürümleriyle oluşturulmuş uygulamaları çalıştırmak için kullanılabilir.

İndirilebilecek .NET Framework'ün tüm sürümlerinin listesi için [.NET İndirmeler](https://dotnet.microsoft.com/download) sayfasına bakın.
