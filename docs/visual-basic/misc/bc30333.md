# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-a-reference-type"></a>' ' \<type1> \<type2> \<type3> Bir başvuru türü olmadığından, ' ' türünün değeri ' ' olarak dönüştürülemez
Bir ifade bir dizi türünü, hedef dizinin öğelerinin veri türünün bir başvuru türü olmadığı başka bir dizi türüne dönüştürmeye çalışır.  
  
 **Hata kimliği:** BC30333  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Hatanın kaynağını öğrenmek için her iki dizinin öğelerinin veri türlerini kontrol edin.
