# <a name="locale-id-name-is-not-supported-on-this-system"></a>' ' Yerel ayar KIMLIĞI \<name> Bu sistemde desteklenmiyor

Kullanmaya çalışıyorsunuz `VbStrConv` , ancak sisteminiz belirtilen yerel ayarı desteklemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Yerel ayarı desteklemek için makineyi yükseltin veya uygulamayı belirtilen yerel ayarı destekleyen bir makinede çalıştırın.
