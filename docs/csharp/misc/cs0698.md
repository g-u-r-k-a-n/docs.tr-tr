# <a name="compiler-error-cs0698"></a>Derleyici hatası CS0698

Bir genel tür, öznitelik sınıfı olduğu için ' class ' öğesinden türetilemez  
  
 Öznitelik sınıfından türetilen tüm sınıflar bir özniteliktir. Özniteliklerin genel tür olmasına izin verilmez.  
  
 Aşağıdaki örnek CS0698 oluşturur:  
  
```csharp  
// CS0698.cs  
class C<T> : System.Attribute  // CS0698  
{  
}  
```
