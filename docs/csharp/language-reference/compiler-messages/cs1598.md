# <a name="compiler-warning-level-1-cs1598"></a>Derleyici Uyarısı (düzey 1) CS1598
XML parser aşağıdaki nedenle yüklenmiş olamazdı: 'neden'. XML dokümantasyon dosyası 'dosya' oluşturulmayacak.  
  
 [-doc](../compiler-options/doc-compiler-option.md) seçeneği belirtildi, ancak derleyici msxml3.dll'yi bulup yükleyemedi. Msxml3.dll dosyasının yüklü ve kayıtlı olduğundan emin olun.
