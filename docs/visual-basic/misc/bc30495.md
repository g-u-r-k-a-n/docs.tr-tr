# <a name="exponent-is-not-valid"></a>Üs geçerli değil

Kayan nokta değişmez değeri ile geçerli olmayan bir üs kullanmaya çalıştınız. Bu hata genellikle kayan nokta değişmez değerinin, kayan nokta değişmez değerini tutmak için gereken boyutta Odd 'de olduğunda oluşur.  
  
 **Hata kimliği:** BC30495  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kayan nokta değişmez değerinin türünü denetleyin ve geçerli bir türle değiştirin.
