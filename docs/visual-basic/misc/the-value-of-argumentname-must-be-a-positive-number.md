# <a name="the-value-of-argumentname-must-be-a-positive-number"></a>Değeri \<argumentname> pozitif bir sayı olmalıdır

<xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A>Özelliğin değeri sıfırdan büyük olmalıdır.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A>Özelliği, iletilerin günlük dosyasına yazılması için gereken boş disk alanı miktarını bayt cinsinden belirtir  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A>Özelliği pozitif bir sayı olarak ayarlayın.
