# <a name="compiler-warning-level-3-cs0067"></a>Derleyici Uyarısı (düzey 3) CS0067

' Event ' olayı hiç kullanılmadı  
  
 Bir [olay](../language-reference/keywords/event.md) bildirildi ancak bildirildiği sınıfta hiç kullanılmadı.  
  
 Aşağıdaki örnek CS0067 oluşturur:  
  
```csharp  
// CS0067.cs  
// compile with: /W:3  
using System;  
delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067  
   // uncomment TestMethod to resolve this CS0067  
/*  
   private void TestMethod()  
   {  
      if (evt != null)  
         evt();  
   }  
*/  
   public static void Main()  
   {  
   }  
}  
```
