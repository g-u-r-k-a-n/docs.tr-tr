# <a name="compiler-error-cs1022"></a>Derleyici hatası CS1022
Tür veya ad alanı tanımı ya da dosya sonu bekleniyor  
  
 Kaynak kodu dosyası eşleşen bir ayraç kümesine sahip değil.  
  
 Aşağıdaki örnek CS1022 oluşturur:  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
