# <a name="xml-comment-block-must-immediately-precede-the-language-element-to-which-it-applies"></a>XML açıklama bloğu, uygulandığı dil öğesinden hemen önce gelmelidir
XML açıklama bloğu, uygulandığı dil öğesinden hemen önce gelmelidir. XML açıklaması yoksayılacak.  
  
 Bu hata, bir etiketin yanlış yerleştirilmiş olması veya yanlış biçimlendirilmiş olması durumunda da oluşur.  
  
 **Hata kimliği:** BC42300  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Açıklama bloğunu, uygulandığı dil öğesinden önce taşıyın. Gereksiz karakterlerin ilk etiketinden önce yanlışlıkla eklenmediğinden emin olun.  
  
2. Etiketin geçerli olduğundan emin olun.
