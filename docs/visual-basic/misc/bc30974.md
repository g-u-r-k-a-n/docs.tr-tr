# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Comment deyimleri değerlendirilemiyor (TOC 'de değil)
Hata ayıklama oturumu sırasında anında pencereye bir açıklama açıklaması girildi. Komut penceresindeki Açıklama deyimleri değerlendirilmez.  
  
 **Hata kimliği:** BC30974
