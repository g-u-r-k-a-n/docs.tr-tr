# <a name="compiler-error-cs1728"></a><span data-ttu-id="19eed-102">Derleyici hatası CS1728</span><span class="sxs-lookup"><span data-stu-id="19eed-102">Compiler Error CS1728</span></span>

<span data-ttu-id="19eed-103">' Type ' üyesi olduğundan temsilciyi ' Member ' öğesine bağlayamıyor</span><span class="sxs-lookup"><span data-stu-id="19eed-103">Cannot bind delegate to 'member' because it is a member of 'type'</span></span>  
  
 <span data-ttu-id="19eed-104">Temsilcileri `Nullable` değer türlerinin üyelerine bağlayamazsınız.</span><span class="sxs-lookup"><span data-stu-id="19eed-104">You cannot bind delegates to members of `Nullable` value types.</span></span>  
  
## <a name="example"></a><span data-ttu-id="19eed-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="19eed-105">Example</span></span>  

 <span data-ttu-id="19eed-106">Aşağıdaki örnek CS1728 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="19eed-106">The following sample generates CS1728:</span></span>  
  
```csharp  
// CS1728.cs  
class Test  
{  
   delegate T GetT<T>();  
   delegate T GetT1<T>(T t);  
  
   delegate bool E(object o);  
   delegate int I();  
   delegate string S();  
  
   static void Main()  
   {  
      int? x = null;  
      int? y = 5;  
  
      GetT<int> d1 = x.GetValueOrDefault;   // CS1728  
      GetT<int> d2 = y.GetValueOrDefault;   // CS1728  
      GetT1<int> d3 = x.GetValueOrDefault;   // CS1728  
      GetT1<int> d4 = y.GetValueOrDefault;   // CS1728  
   }  
}  
```
