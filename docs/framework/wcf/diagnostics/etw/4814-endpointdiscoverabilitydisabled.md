|||  
|-|-|  
|ID|4814|  
|Anahtar sözcükler|Bulma|  
|Düzey|Bilgi|

## <a name="description"></a>Açıklama  

 Uç noktanın bulunabilirliği devre dışı bırakıldığında bu olay yayınlanır.

 EndpointAddress = ' %1 ' ve ListenUri = ' %2 ' olan bitiş noktasının bulunabilirliği devre dışı bırakıldı.
