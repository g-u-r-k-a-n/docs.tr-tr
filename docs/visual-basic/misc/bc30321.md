# <a name="case-cannot-follow-a-case-else-in-the-same-select-statement"></a>Aynı ' Select ' deyimindeki ' Case Else ' bir ' Case ' gelemez
Bir `Case Else` deyim, ilk için eşleşme bulunmazsa yürütülecek deyimleri tanıtır `Case` . `Case` `Case Else` Aynı bloktan sonra bir ifade bulundu `Select` .  
  
 **Hata kimliği:** BC30321  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- ' İ `Case Else` deyimden sonra uygun konuma taşıyın `Case` .
