# <a name="codepage-name-is-invalid-or-not-installed"></a>'\<name > ' kod sayfası geçersiz veya yüklü değil
Belirtilen kod sayfası geçerli değil veya bulunamıyor.  
  
 **Hata kimliği:** BC2016  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kodunuzu inceleyin ve belirli bir duruma uygun çözümü saptayın
