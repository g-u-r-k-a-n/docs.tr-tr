# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>' WithEvents ' değişkeni, ' ' tarafından erişilebilen herhangi bir örnek olayı oluşturmaz \<containername>

Başvurduğunuz nesne, erişilebilir olaylar oluşturmaz.  
  
 **Hata kimliği:** BC30600  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Olay için erişim değiştiricilerini değiştirin veya geçerli bağlamda erişilebilen bir olay seçin.
