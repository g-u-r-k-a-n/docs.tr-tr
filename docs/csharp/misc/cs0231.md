# <a name="compiler-error-cs0231"></a><span data-ttu-id="5de3a-103">Derleyici hatası CS0231</span><span class="sxs-lookup"><span data-stu-id="5de3a-103">Compiler Error CS0231</span></span>

<span data-ttu-id="5de3a-104">Bir params parametresi, biçimsel parametre listesindeki son parametre olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="5de3a-104">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="5de3a-105">[Params](../language-reference/keywords/params.md) parametresi, değişken sayıda bağımsız değişkeni destekler ve diğer tüm parametrelerden sonra olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="5de3a-105">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="5de3a-106">Daha fazla bilgi için bkz. [Yöntemler](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="5de3a-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="5de3a-107">Aşağıdaki örnek CS0231 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5de3a-107">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```
