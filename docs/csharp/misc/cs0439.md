# <a name="compiler-error-cs0439"></a>Derleyici hatası CS0439

Extern diğer ad bildirimi, ad alanında tanımlanan diğer tüm öğelerden önce gelmelidir

Bu hata, bir `extern` bildirim gibi bir bildirimin bundan sonra `using` aynı ad alanında geldiği durumlarda oluşur. `extern`Bildirimlerin diğer tüm ad alanı öğelerinden önce gelmesi gerekir.

## <a name="example"></a>Örnek

Aşağıdaki örnek CS0439 oluşturur:

```csharp
// CS0439.cs
using System;

extern alias MyType;   // CS0439
// To resolve the error, make the extern alias the first line in the file.

public class Test
{
    public static void Main()
    {
    }
}
```
