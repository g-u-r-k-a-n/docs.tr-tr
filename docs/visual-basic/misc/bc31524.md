# <a name="systemruntimeinteropservicesdllimportattribute-cannot-be-applied-to-a-get-or-set"></a>' System.Runtime.InteropServices.DllImportAttribute ' bir ' Get ' veya ' Set ' öğesine uygulanamaz

`DllImportAttribute`Özniteliği bir `Get` veya `Set` özellik yordamına uygulandı.  
  
 **Hata kimliği:** BC31524  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `DllImportAttribute` `Get` Ve `Set` özellik yordamlarını kaldırın.
