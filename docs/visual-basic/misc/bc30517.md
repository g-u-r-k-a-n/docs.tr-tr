# <a name="overload-resolution-failed-because-no-method-is-accessible"></a>' ' Erişilebilir olmadığından aşırı yükleme çözümlemesi başarısız oldu \<method>

Aşırı yüklenmiş bir yönteme çağrı yaptınız, ancak derleyici parametre sayısı bakımından eşleşen bir parametre listesi ile erişilebilir bir yöntem bulamıyor.  
  
 **Hata kimliği:** BC30517  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Sağladığınız bağımsız değişken listesini denetleyin.  
  
2. Erişilebilir olup olmadığını öğrenmek için çağırmayı denediğiniz yöntemi denetleyin.
