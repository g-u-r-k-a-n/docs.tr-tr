# <a name="compiler-error-cs0523"></a><span data-ttu-id="42311-102">Derleyici Hatası CS0523</span><span class="sxs-lookup"><span data-stu-id="42311-102">Compiler Error CS0523</span></span>
<span data-ttu-id="42311-103">'struct1' türüyapı üyesi 'struct2 alan' yapı düzeninde bir döngüneden olur</span><span class="sxs-lookup"><span data-stu-id="42311-103">Struct member 'struct2 field' of type 'struct1' causes a cycle in the struct layout</span></span>  
  
 <span data-ttu-id="42311-104">İki yapının tanımları özyinelemeli başvurular içerir.</span><span class="sxs-lookup"><span data-stu-id="42311-104">The definitions of two structs include recursive references.</span></span> <span data-ttu-id="42311-105">Yapı [tanımlarını,](../builtin-types/struct.md) her birinin diğerinde tanımlaşmayacak şekilde değiştirin.</span><span class="sxs-lookup"><span data-stu-id="42311-105">Change the [struct](../builtin-types/struct.md) definitions such that each does not define itself on the other.</span></span> <span data-ttu-id="42311-106">Bu sınırlama yalnızca yapı türleri için geçerlidir, çünkü yapı türleri değer dir.</span><span class="sxs-lookup"><span data-stu-id="42311-106">This limitation applies only to structs, since structs are value types.</span></span> <span data-ttu-id="42311-107">Özyinelemeli başvurular kullanıyorsanız, türlerinizi sınıf olarak bildirin.</span><span class="sxs-lookup"><span data-stu-id="42311-107">If using recursive references, declare your types as classes.</span></span>  
  
 <span data-ttu-id="42311-108">Aşağıdaki örnek CS0523 üretir:</span><span class="sxs-lookup"><span data-stu-id="42311-108">The following sample generates CS0523:</span></span>  
  
```csharp

{  
   public RecursiveLayoutStruct2 field;

{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
