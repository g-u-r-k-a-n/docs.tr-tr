<span data-ttu-id="5fb85-104">Bir dönüştürme işleci, dönüş türünden türetilen bir parametre türüyle bildirilmiştir.</span><span class="sxs-lookup"><span data-stu-id="5fb85-104">A conversion operator is declared with a parameter type that derives from the return type.</span></span>  
  
 <span data-ttu-id="5fb85-105">Derleme zamanında Visual Basic, önceden tanımlanmış bir dönüştürmeyi herhangi bir başvuru türünden devralma hiyerarşisinde herhangi bir türe, yani türettiği veya ondan türetilen herhangi bir türe kabul eder.</span><span class="sxs-lookup"><span data-stu-id="5fb85-105">At compile time, Visual Basic considers a predefined conversion to exist from any reference type to any type in its inheritance hierarchy, that is, any type from which it derives or which derives from it.</span></span> <span data-ttu-id="5fb85-106">Bu tür bir dönüştürme çalışma zamanında başarısız olabilir, ancak derleyici çalışma zamanı sonuçlarını tahmin edemez, bu nedenle bu tür dönüştürmenin derlenmesi için izin verir.</span><span class="sxs-lookup"><span data-stu-id="5fb85-106">Such a conversion might fail at run time, but the compiler cannot predict run-time results, so it allows any such conversion to compile.</span></span>
