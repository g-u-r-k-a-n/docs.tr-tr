# <a name="bc31168-xml-axis-properties-do-not-support-late-binding"></a>BC31168: XML eksen özellikleri geç bağlamayı desteklemiyor

Türsüz bir nesne için bir XML ekseni özelliğine başvuruldu.

 **Hata kimliği:** BC31168

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

- XML eksen özelliğine başvurmadan önce nesnenin kesin türü belirtilmiş bir nesne olduğundan emin olun <xref:System.Xml.Linq.XElement> .
