# <a name="name-cannot-refer-to-itself-through-its-default-instance-use-me-instead"></a>' \<name> ' kendi varsayılan örneği aracılığıyla kendi kendine başvuramaz; bunun yerine ' Me ' kullanın

Bir form içinden, varsayılan örnek olarak bu forma başvurabileceğiniz bir girişimde bulunuldu. Bu, formun kendisini yinelemeli olarak çağırmasını sağlar.  
  
 Çoğu durumda, `Me` varsayılan örneği kullanmak yerine, formun geçerli örneğine başvuru yaparken ' i kullanmanız gerekir.  
  
 **Hata kimliği:** BC31139  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Me`Nesnesine başvurmak için kullanın.
