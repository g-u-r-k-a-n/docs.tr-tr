# <a name="structures-cannot-have-inherits-statements"></a>Yapılar ' Inherits ' deyimlerine sahip olamaz

Sınıfların aksine, yapılar devralmayı desteklemez.  
  
 **Hata kimliği:** BC30628  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Inherits`Deyimin yapısını kaldırın.  
  
- Yerine sınıfları kullanmak için uygulamanızı yeniden tasarlama.
