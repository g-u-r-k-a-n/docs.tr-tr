<span data-ttu-id="15b8e-103">Bu konu, bir XML belgesi yapısından bir XML şeması tanım dili (XSD) şeması çıkarsmak için <xref:System.Xml.Schema.XmlSchemaInference> sınıfının nasıl kullanılacağını açıklar.</span><span class="sxs-lookup"><span data-stu-id="15b8e-103">This topic describes how to use the <xref:System.Xml.Schema.XmlSchemaInference> class to infer an XML Schema definition language (XSD) schema from the structure of an XML document.</span></span>  
  
## <a name="the-schema-inference-process"></a><span data-ttu-id="15b8e-104">Şema çıkarımı Işlemi</span><span class="sxs-lookup"><span data-stu-id="15b8e-104">The Schema Inference Process</span></span>  
 <span data-ttu-id="15b8e-105"><xref:System.Xml.Schema?displayProperty=nameWithType> ad alanının <xref:System.Xml.Schema.XmlSchemaInference> sınıfı, bir XML belgesi yapısından bir veya daha fazla XML şeması tanım dili (XSD) şeması oluşturmak için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="15b8e-105">The <xref:System.Xml.Schema.XmlSchemaInference> class of the <xref:System.Xml.Schema?displayProperty=nameWithType> namespace is used to generate one or more XML Schema definition language (XSD) schemas from the structure of an XML document.</span></span> <span data-ttu-id="15b8e-106">Oluşturulan şemalar özgün XML belgesini doğrulamak için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="15b8e-106">The generated schemas may be used to validate the original XML document.</span></span>  
  
 <span data-ttu-id="15b8e-107">Bir XML belgesi <xref:System.Xml.Schema.XmlSchemaInference> sınıfı tarafından işlendiği için <xref:System.Xml.Schema.XmlSchemaInference> sınıfı, XML belgesindeki öğeleri ve öznitelikleri tanımlayan şema bileşenleriyle ilgili varsayımlar yapar.</span><span class="sxs-lookup"><span data-stu-id="15b8e-107">As an XML document is processed by the <xref:System.Xml.Schema.XmlSchemaInference> class, the <xref:System.Xml.Schema.XmlSchemaInference> class makes assumptions about the schema components that describe the elements and attributes in the XML document.</span></span> <span data-ttu-id="15b8e-108"><xref:System.Xml.Schema.XmlSchemaInference> sınıfı, belirli bir öğe veya öznitelik için en kısıtlayıcı türü azaltarak, şema bileşenlerini kısıtlanmış bir şekilde de algılar.</span><span class="sxs-lookup"><span data-stu-id="15b8e-108">The <xref:System.Xml.Schema.XmlSchemaInference> class also infers schema components in a constrained way by inferring the most restrictive type for a particular element or attribute.</span></span> <span data-ttu-id="15b8e-109">XML belgesi hakkında daha fazla bilgi toplandığından, bu kısıtlamalar daha az kısıtlayıcı olan türleri azaltarak gevşur.</span><span class="sxs-lookup"><span data-stu-id="15b8e-109">As more information about the XML document is gathered, these constraints are loosened by inferring less restrictive types.</span></span> <span data-ttu-id="15b8e-110">Çıkarsanan en az kısıtlayıcı tür `xs:string`.</span><span class="sxs-lookup"><span data-stu-id="15b8e-110">The least restrictive type that can be inferred is `xs:string`.</span></span>  
  
 <span data-ttu-id="15b8e-111">Örneğin, bir XML belgesinin aşağıdaki parçasını alın.</span><span class="sxs-lookup"><span data-stu-id="15b8e-111">Take, for example, the following piece of an XML document.</span></span>  
  
```xml  
<parent attribute1="6">  
    <child>One</child>  
    <child>Two</child>  
</parent>  
<parent attribute1="A">  
```  
  
 <span data-ttu-id="15b8e-112">Yukarıdaki örnekte, `attribute1` özniteliğine <xref:System.Xml.Schema.XmlSchemaInference> işlem tarafından `6` değeri ile karşılaşıldığında, `xs:unsignedByte`türünde olduğu varsayılır.</span><span class="sxs-lookup"><span data-stu-id="15b8e-112">In the example above, when the `attribute1` attribute is encountered with a value of `6` by the <xref:System.Xml.Schema.XmlSchemaInference> process, it is assumed to be of type `xs:unsignedByte`.</span></span> <span data-ttu-id="15b8e-113">İkinci `parent` öğesi <xref:System.Xml.Schema.XmlSchemaInference> işlem tarafından karşılaşıldığında, `attribute1` özniteliğinin değeri artık `A`olduğundan, kısıtlama, türü `xs:string` olarak değiştirilerek gevşendi.</span><span class="sxs-lookup"><span data-stu-id="15b8e-113">When the second `parent` element is encountered by the <xref:System.Xml.Schema.XmlSchemaInference> process, the constraint is loosened by modifying the type to `xs:string` because the value of the `attribute1` attribute is now `A`.</span></span> <span data-ttu-id="15b8e-114">Benzer şekilde, şemada gösterilen tüm `child` öğeleri için `minOccurs` özniteliği, ikinci üst öğenin alt öğeleri olmadığından `minOccurs="0"` için gevşulmuş.</span><span class="sxs-lookup"><span data-stu-id="15b8e-114">Similarly, the `minOccurs` attribute for all the `child` elements inferred in the schema are loosened to `minOccurs="0"` because the second parent element has no child elements.</span></span>

 <span data-ttu-id="15b8e-117">İlk <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> yöntemi, bir XML belgesini temel alan bir şema oluşturmak için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="15b8e-117">The first <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> method is used to create a schema based on an XML document.</span></span> <span data-ttu-id="15b8e-118">İkinci <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> yöntemi, birden çok XML belgesini açıklayan bir şemayı çıkarmakta kullanılır.</span><span class="sxs-lookup"><span data-stu-id="15b8e-118">The second <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> method is used to infer a schema that describes multiple XML documents.</span></span> <span data-ttu-id="15b8e-119">Örneğin, tüm XML belgesi kümesini açıklayan bir şema oluşturmak için birden çok XML belgesini tek seferde <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> yöntemine ekleyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="15b8e-119">For example, you can feed multiple XML documents to the <xref:System.Xml.Schema.XmlSchemaInference.InferSchema%2A?displayProperty=nameWithType> method one at a time to produce a schema that describes the entire set of XML documents.</span></span>
