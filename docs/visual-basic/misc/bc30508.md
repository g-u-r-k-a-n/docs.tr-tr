# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>' \<name> ' \<type1> , ' \<specifier> \<type2> \<typename> ' içinde türü gösteremez \<name>

Ortak sınıf dışında özel bir tür kullanıma sunmaya çalıştınız `MyClass` .  
  
 **Hata kimliği:** BC30508  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Türü bildirin `Public` .
