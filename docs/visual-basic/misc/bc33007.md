# <a name="end-operator-must-be-preceded-by-a-matching-operator"></a>' End operator ' öncesinde eşleşen bir ' operator ' olmalıdır

`End Operator`Kendisinden önce eşleşen bildirim olmadan bir ifade görüntülenir `Operator` .  
  
 **Hata kimliği:** BC33007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ise, `End Operator` ifadeyi kaldırın.  
  
- Eksik `Operator` yordamı bir varsa sağlayın.  
  
- `End Operator`İfadeyi koddaki uygun yere taşıyın.
