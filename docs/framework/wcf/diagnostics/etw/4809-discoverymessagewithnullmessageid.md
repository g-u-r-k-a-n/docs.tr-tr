# <a name="4809---discoverymessagewithnullmessageid"></a>4809 - DiscoveryMessageWithNullMessageId

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|4809|  
|Anahtar sözcükler|Bulma|  
|Level|Uyarı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Description  

 Bu olay, ileti üst bilgisi gerekli MessageID özelliğini içermediği için bulma iletisi bırakıldığında yayınlanır.  
  
## <a name="message"></a>İleti  

 İleti üst bilgisi gerekli MessageID özelliğini içermediği için bir %1 iletisi bırakıldı.
