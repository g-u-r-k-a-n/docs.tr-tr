# <a name="para-c-programming-guide"></a>\<paragraf > (C# Programlama Kılavuzu)
## <a name="syntax"></a>Sözdizimi  
  
```xml  
<para>content</para>  
```  
  
## <a name="parameters"></a>Parametreler  
 `content`  
 Paragrafın metni.  
  
## <a name="remarks"></a>Açıklamalar  
 \<paragraf > etiketi, [\<özet >](./summary.md), [\<açıklamalar >](./remarks.md)veya [\<> döndürür](./returns.md)ve metne yapı eklemenizi sağlar.  
  
 Belge açıklamalarını bir dosyaya işlemek için [-doc](../../language-reference/compiler-options/doc-compiler-option.md) ile derleyin.
