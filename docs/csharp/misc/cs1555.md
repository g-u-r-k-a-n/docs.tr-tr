# <a name="compiler-error-cs1555"></a>Derleyici hatası CS1555

 [-Main](../language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği için bir sınıf belirtildi, ancak kaynak kodunda sınıf adı bulunamadı.
