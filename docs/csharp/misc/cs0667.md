# <a name="compiler-error-cs0667"></a>Derleyici hatası CS0667
' Geçersiz özellik ' özelliği kullanım dışıdır. Lütfen bunun yerine ' geçerli özellik ' kullanın '.  
  
 Kullanmaya çalıştığınız özellik artık kullanım dışıdır. Kodunuzu geçerli özelliği kullanacak şekilde güncelleştirin.
