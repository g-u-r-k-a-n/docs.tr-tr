# <a name="type-character-character-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>' ' Tür karakteri \<character> Açık türe sahip bir bildirimde kullanılamaz
`$`Açıkça bir tür belirten bir tür karakteri kullandınız.  
  
 **Hata kimliği:** BC30302  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Tür karakterini kaldırın veya `As <Type>` yan tümceyi bildirimden kaldırın.
