# <a name="variablename-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-catch-variable"></a>' \<variablename> ' bir yerel değişken veya parametre olmadığından, ' catch ' değişkeni olarak kullanılamaz
`Try...Catch...Finally`Deyimlerdeki değişkenler yerel olarak bildirilmelidir veya geçerli yordamın parametreleri olmalıdır.  
  
 **Hata kimliği:** BC31082  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Deyimler için yerel değişkenler veya parametreler bildirin `Try...Catch...Finally` .
