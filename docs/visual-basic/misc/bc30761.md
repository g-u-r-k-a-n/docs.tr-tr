# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>' ' Projesi ' ' \<projectname> projesine başvuramaz \<projectname> çünkü ' ' \<projectname> doğrudan veya dolaylı olarak ' ' öğesine başvuruyor \<projectname>

Bir proje, ikinci bir projeye yönelik bir başvuru içerir. Sırasıyla ikinci proje, başvurduğu projeye bir başvuru içerir.  
  
 **Hata kimliği:** BC30761  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Başvurulardan birini kaldırın.
