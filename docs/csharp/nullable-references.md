## <a name="nullable-annotation-context"></a><span data-ttu-id="b419d-182">Null yapılabilir ek açıklama bağlamı</span><span class="sxs-lookup"><span data-stu-id="b419d-182">Nullable annotation context</span></span>

<span data-ttu-id="b419d-183">Derleyici, devre dışı bırakılmış bir null yapılabilir ek açıklama bağlamında aşağıdaki kuralları kullanır:</span><span class="sxs-lookup"><span data-stu-id="b419d-183">The compiler uses the following rules in a disabled nullable annotation context:</span></span>

<span data-ttu-id="b419d-189">Derleyici, etkinleştirilmiş bir null yapılabilir ek açıklama bağlamında aşağıdaki kuralları kullanır:</span><span class="sxs-lookup"><span data-stu-id="b419d-189">The compiler uses the following rules in an enabled nullable annotation context:</span></span>

- <span data-ttu-id="b419d-190">Başvuru türündeki herhangi bir değişken **null atanamaz bir başvurudur**.</span><span class="sxs-lookup"><span data-stu-id="b419d-190">Any variable of a reference type is a **non-nullable reference**.</span></span>
- <span data-ttu-id="b419d-191">Null olamayan herhangi bir başvuruya, güvenli bir şekilde başvurulmalıdır.</span><span class="sxs-lookup"><span data-stu-id="b419d-191">Any non-nullable reference may be dereferenced safely.</span></span>

<span data-ttu-id="b419d-196">Etkin bir null yapılabilir ek açıklama bağlamında, bir başvuru türüne eklenen `?` karakteri **null yapılabilir bir başvuru türü**bildirir.</span><span class="sxs-lookup"><span data-stu-id="b419d-196">In an enabled nullable annotation context, the `?` character appended to a reference type declares a **nullable reference type**.</span></span> <span data-ttu-id="b419d-197">İfadenin null olmadığını bildirmek için **null-forverme işleci** `!` bir ifadeye eklenebilir.</span><span class="sxs-lookup"><span data-stu-id="b419d-197">The **null-forgiving operator** `!` may be appended to an expression to declare that the expression isn't null.</span></span>

1. <span data-ttu-id="b419d-205">Değişken, null olmayan bir değere kesin olarak atandı.</span><span class="sxs-lookup"><span data-stu-id="b419d-205">The variable has been definitely assigned to a non-null value.</span></span>
