## <a name="syntax"></a>Sözdizimi
  
```csharp  
private ArrayList m_ConnectionList
```

> [!WARNING]
> Alan `ConnectionGroup.m_ConnectionList` özeldir ve doğrudan kodunuzda kullanılmamalıdır.
