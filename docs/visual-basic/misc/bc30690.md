# <a name="structure-structurename-cannot-be-indexed-because-it-has-no-default-property"></a>' \<structurename> ' Yapısının varsayılan özelliği olmadığından dizini oluşturulamaz

Varsayılan özelliği olmayan bir yapıyla varsayılan özellik sözdizimini kullanma girişiminde bulunuldu.  
  
 **Hata kimliği:** BC30690  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yapının özelliklerine erişmek için standart özellik söz dizimini kullanın.
