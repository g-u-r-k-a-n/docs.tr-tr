# <a name="f-compiler-messages"></a>F#Derleyici iletileri

Bu bölümde F# derleyicinin belirli yapılar için yayacaktır. Varsayılan hata kümeleri şu şekilde değiştirilebilir:

- `-warnaserror+` derleyici seçeneğini kullanarak belirli uyarıları hatalar gibi davranma

- `-nowarn` derleyici seçeneğini kullanarak belirli uyarıları yok sayma

Bu bölümde belirli bir uyarı veya hata henüz kaydedilmediyse, bu sayfanın sonuna gidin ve hatanın numarasını veya metnini içeren geri bildirim gönderin.
