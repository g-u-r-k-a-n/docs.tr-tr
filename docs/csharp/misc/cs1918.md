# <a name="compiler-error-cs1918"></a><span data-ttu-id="17ba0-103">Derleyici hatası CS1918</span><span class="sxs-lookup"><span data-stu-id="17ba0-103">Compiler Error CS1918</span></span>

<span data-ttu-id="17ba0-104">Değer türünde olduğundan, ' Type ' türünün ' name ' özelliğinin üyeleri bir nesne başlatıcısı ile atanamaz.</span><span class="sxs-lookup"><span data-stu-id="17ba0-104">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="17ba0-105">Bu hata, başlatılmakta olan sınıfın bir özelliği olan bir yapı türünün özelliklerini başlatmak için bir nesne Başlatıcısı kullanmayı denediğinizde oluşur.</span><span class="sxs-lookup"><span data-stu-id="17ba0-105">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="17ba0-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="17ba0-106">To correct this error</span></span>  
  
1. <span data-ttu-id="17ba0-107">Nesne başlatıcısında özelliğin alanlarını tam olarak başlatmak istiyorsanız, yapısını bir sınıf türü olarak değiştirin.</span><span class="sxs-lookup"><span data-stu-id="17ba0-107">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="17ba0-108">Aksi takdirde, nesne Başlatıcısı 'nı kullanarak nesneyi oluşturduktan sonra yapı üyelerini ayrı bir yöntem çağrısında başlatın.</span><span class="sxs-lookup"><span data-stu-id="17ba0-108">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="17ba0-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="17ba0-109">Example</span></span>  

 <span data-ttu-id="17ba0-110">Aşağıdaki örnek CS1918 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="17ba0-110">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
