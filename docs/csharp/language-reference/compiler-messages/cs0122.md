## <a name="example"></a><span data-ttu-id="76f16-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="76f16-109">Example</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
