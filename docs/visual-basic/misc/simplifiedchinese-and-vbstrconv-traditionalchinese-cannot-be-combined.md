# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>SimplifiedChinese ve VbStrConv. TraditionalChinese birleştirilemez

Uygulamanız, `VbStrConv` sabit listesi üyelerini birleştirmeye çalışıyor `SimplifiedChinese` ve `TraditionalChinese` birbirini dışlıyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ya da ' i kaldırın `VbStrConv.SimplifiedChinese` `VbStrConv.TraditionalChinese` .
