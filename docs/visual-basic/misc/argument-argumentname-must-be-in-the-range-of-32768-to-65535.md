# <a name="argument-argumentname-must-be-in-the-range-of--32768-to-65535"></a>' ' Bağımsız değişkeni \<argumentname> -32768 ile 65535 arasında olmalıdır

Bağımsız değişkenin değeri, izin verilen-32.768 aralığının 65.535 ' nin dışında.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bağımsız değişkenin değerinin geçerli Aralık içinde olduğundan emin olun.  
  
2. Bağımsız değişken bir hesaplama tarafından sağlandıysa, hesaplama işlemlerinin değerlerini kontrol edin.
