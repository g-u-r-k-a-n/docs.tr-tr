<span data-ttu-id="1bfe9-104">Bu örnek, bir Windows Communication Foundation (WCF) istemci uygulamasının mevcut bir ASMX Web hizmetiyle nasıl tümleştirileceğini gösterir.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-104">This sample demonstrates how to integrate a Windows Communication Foundation (WCF) client application with an existing ASMX Web service.</span></span>

 <span data-ttu-id="1bfe9-106">Bu örnek, Internet Information Services (IIS) tarafından barındırılan bir istemci konsol programından (. exe) ve hizmet kitaplığından (. dll) oluşur.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-106">This sample consists of a client console program (.exe) and a service library (.dll) hosted by Internet Information Services (IIS).</span></span> <span data-ttu-id="1bfe9-107">Hizmet, istek-yanıt iletişim modelini tanımlayan bir sözleşmeyi uygulayan bir ASMX Web hizmetidir.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-107">The service is an ASMX Web Service that implements a contract that defines a request-reply communication pattern.</span></span> <span data-ttu-id="1bfe9-108">Hizmet matematik işlemlerini (, `Add` ,, `Subtract` `Multiply` ve `Divide` ) kullanıma sunar.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-108">The service exposes math operations (`Add`, `Subtract`, `Multiply`, and `Divide`).</span></span> <span data-ttu-id="1bfe9-109">İstemci, zaman uyumlu istekleri bir matematik işlemine yapar ve hizmet sonuçla yanıt verir.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-109">The client makes synchronous requests to a math operation and the service replies with the result.</span></span> <span data-ttu-id="1bfe9-110">İstemci etkinliği konsol penceresinde görünür.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-110">Client activity is visible in the console window.</span></span>  
  
 <span data-ttu-id="1bfe9-111">Aşağıdaki örnek kodda gösterilen ASMX Web hizmeti uygulama, uygun sonucu hesaplar ve döndürür.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-111">The ASMX Web service implementation shown in the following sample code calculates and returns the appropriate result.</span></span>  
  
```csharp  
[WebService(Namespace="http://Microsoft.ServiceModel.Samples")]  
public class CalculatorService : System.Web.Services.WebService  
    {  
        [WebMethod]  
        public double Add(double n1, double n2)  
        {  
            return n1 + n2;  
        }  
        [WebMethod]  
        public double Subtract(double n1, double n2)  
        {  
            return n1 - n2;  
        }  
        [WebMethod]  
        public double Multiply(double n1, double n2)  
        {  
            return n1 * n2;  
        }  
        [WebMethod]  
        public double Divide(double n1, double n2)  
        {  
            return n1 / n2;  
        }  
    }  
```  
  
 <span data-ttu-id="1bfe9-112">Yapılandırıldığı gibi, hizmete `http://localhost/servicemodelsamples/service.asmx` aynı makinede bulunan bir istemci tarafından erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-112">As configured, the service can be accessed at `http://localhost/servicemodelsamples/service.asmx` by a client on the same machine.</span></span> <span data-ttu-id="1bfe9-113">Uzak makinelerdeki istemciler hizmete erişmek için, localhost yerine tam etki alanı adı belirtilmelidir.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-113">For clients on remote machines to access the service, a qualified domain name must be specified instead of localhost.</span></span>  
  
 <span data-ttu-id="1bfe9-114">İletişim, [ServiceModel meta veri yardımcı programı Aracı (Svcutil.exe)](../servicemodel-metadata-utility-tool-svcutil-exe.md)tarafından oluşturulan bir istemci aracılığıyla yapılır.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-114">Communication is done through a client generated by the [ServiceModel Metadata Utility Tool (Svcutil.exe)](../servicemodel-metadata-utility-tool-svcutil-exe.md).</span></span> <span data-ttu-id="1bfe9-115">İstemci generatedClient.cs dosyasında bulunur.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-115">The client is contained in the file generatedClient.cs.</span></span> <span data-ttu-id="1bfe9-116">Güncelleştirilmiş meta verileri almak için kullanıldığından, ASMX hizmeti proxy kodunu oluşturmak için kullanılabilir olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-116">The ASMX service must be available to generate the proxy code, because it is used to retrieve the updated metadata.</span></span> <span data-ttu-id="1bfe9-117">Yazılan proxy 'yi oluşturmak için istemci dizinindeki bir komut isteminden aşağıdaki komutu çalıştırın.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-117">Run the following command from a command prompt in the client directory to generate the typed proxy.</span></span>  
  
```console  
svcutil.exe /n:http://Microsoft.ServiceModel.Samples,Microsoft.ServiceModel.Samples http://localhost/servicemodelsamples/service.svc?wsdl /out:generatedClient.cs  
```  
  
 <span data-ttu-id="1bfe9-118">Oluşturulan istemciyi kullanarak, uygun adresi ve bağlamayı yapılandırarak bir hizmet uç noktasına erişebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-118">By using the generated client, you can access a service endpoint by configuring the appropriate address and binding.</span></span> <span data-ttu-id="1bfe9-119">Hizmet gibi istemci, iletişim kuracak uç noktayı belirtmek için bir yapılandırma dosyası (App.config) kullanır.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-119">Like the service, the client uses a configuration file (App.config) to specify the endpoint to communicate with.</span></span> <span data-ttu-id="1bfe9-120">İstemci uç noktası yapılandırması, aşağıdaki örnek yapılandırmada gösterildiği gibi hizmet uç noktası, bağlama ve sözleşme için mutlak bir adresten oluşur.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-120">The client endpoint configuration consists of an absolute address for the service endpoint, the binding, and the contract, as shown in the following sample configuration.</span></span>  
  
```xml  
<client>  
   <endpoint
      address="http://localhost/ServiceModelSamples/service.asmx"
      binding="basicHttpBinding"
      contract="Microsoft.ServiceModel.Samples.CalculatorServiceSoap" />  
</client>  
```  
  
 <span data-ttu-id="1bfe9-121">İstemci uygulama, oluşturulan istemcinin bir örneğini oluşturur.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-121">The client implementation constructs an instance of the generated client.</span></span> <span data-ttu-id="1bfe9-122">Oluşturulan istemci, hizmet ile iletişim kurmak için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-122">The generated client can then be used to communicate with the service.</span></span>  
  
```csharp  
// Create a client.  
CalculatorServiceSoapClient client = new CalculatorServiceSoapClient();  
  
// Call the Add service operation.  
double value1 = 100.00D;  
double value2 = 15.99D;  
double result = client.Add(value1, value2);  
Console.WriteLine("Add({0},{1}) = {2}", value1, value2, result);  
  
// Call the Subtract service operation.  
value1 = 145.00D;  
value2 = 76.54D;  
result = client.Subtract(value1, value2);  
Console.WriteLine("Subtract({0},{1}) = {2}", value1, value2, result);  
  
// Call the Multiply service operation.  
value1 = 9.00D;  
value2 = 81.25D;  
result = client.Multiply(value1, value2);  
Console.WriteLine("Multiply({0},{1}) = {2}", value1, value2, result);  
  
// Call the Divide service operation.  
value1 = 22.00D;  
value2 = 7.00D;  
result = client.Divide(value1, value2);  
Console.WriteLine("Divide({0},{1}) = {2}", value1, value2, result);  
  
//Closing the client gracefully closes the connection and cleans up resources.  
client.Close();  
  
Console.WriteLine();  
Console.WriteLine("Press <ENTER> to terminate client.");  
Console.ReadLine();  
```  
  
 <span data-ttu-id="1bfe9-123">Örneği çalıştırdığınızda, işlem istekleri ve yanıtları istemci konsol penceresinde görüntülenir.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-123">When you run the sample, the operation requests and responses are displayed in the client console window.</span></span> <span data-ttu-id="1bfe9-124">İstemcisini kapatmak için istemci penceresinde ENTER tuşuna basın.</span><span class="sxs-lookup"><span data-stu-id="1bfe9-124">Press ENTER in the client window to shut down the client.</span></span>
