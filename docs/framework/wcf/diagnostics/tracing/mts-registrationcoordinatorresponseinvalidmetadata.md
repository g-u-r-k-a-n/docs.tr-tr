# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorresponseinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorResponseInvalidMetadata

WS-Atomic Transaction Protocol hizmeti, geçersiz veya uyumsuz meta verilerle bir uç nokta başvurusu içeren düzenleyicinden bir RegisterResponse iletisi aldı.  
  
## <a name="description"></a>Açıklama  

 Yerel Işlem yöneticisi üst Işlem yöneticisi ile kaydolmayı denediğinde izlenen ve üst öğesi RegisterResponse iletisi içinde geçersiz bir adres döndürüyor.
