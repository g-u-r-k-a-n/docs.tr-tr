# <a name="end-synclock-must-be-preceded-by-a-matching-synclock"></a>' End SyncLock ' öncesinde eşleşen bir ' SyncLock ' olmalıdır

`SyncLock` bloklar `SyncLock` anahtar sözcüğüyle başlar ve yapı ile biter `End SyncLock` .  
  
 **Hata kimliği:** BC30674  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `SyncLock`Bloğun bir ifadesiyle başladığından emin olun `SyncLock` .
