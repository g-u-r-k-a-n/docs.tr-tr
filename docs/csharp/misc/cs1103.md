# <a name="compiler-error-cs1103"></a>Derleyici hatası CS1103

Genişletme metodunun ilk parametresi ' Type ' türünde olamaz.  
  
 Bir genişletme yönteminin ilk parametresi bir işaretçi türü olamaz.  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS1103 oluşturur:  
  
```csharp  
// cs1103.cs  
public static class Extensions  
{  
    public unsafe static char* Test(this char* charP) { return charP; } // CS1103  
}
```
