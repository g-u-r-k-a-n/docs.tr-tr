# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="3a725-103">Derleyici Uyarısı (düzey 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="3a725-103">Compiler Warning (level 1) CS0420</span></span>

<span data-ttu-id="3a725-104">' tanımlayıcı ': geçici alana yapılan başvuru geçici olarak değerlendirilmeyecektir</span><span class="sxs-lookup"><span data-stu-id="3a725-104">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="3a725-105">Geçici bir alan, `ref` işlevin kapsamı içinde geçici olarak değerlendirilmeyeceği için normalde bir veya **Out** parametresi kullanılarak geçirilmemelidir.</span><span class="sxs-lookup"><span data-stu-id="3a725-105">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="3a725-106">Bunun için, birbirine kenetlenmiş bir API çağrılırken olduğu gibi özel durumlar vardır.</span><span class="sxs-lookup"><span data-stu-id="3a725-106">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="3a725-107">Herhangi bir uyarıda olduğu gibi, kasıtlı olarak geçici bir alanı bir başvuru parametresi olarak kullandığınız nadir durumlarda bu uyarıyı devre dışı bırakmak için [#pragma uyarısını](../preprocessor-directives.md#pragma-warning) kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="3a725-107">As with any warning, you may use the [#pragma warning](../preprocessor-directives.md#pragma-warning) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="3a725-108">Aşağıdaki örnek CS0420 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3a725-108">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```
