# <a name="compiler-warning-level-3-cs1717"></a><span data-ttu-id="6b965-103">Derleyici Uyarısı (düzey 3) CS1717</span><span class="sxs-lookup"><span data-stu-id="6b965-103">Compiler Warning (level 3) CS1717</span></span>

<span data-ttu-id="6b965-104">Atama aynı değişkene yapıldı; başka bir şey atamanız mı istediniz?</span><span class="sxs-lookup"><span data-stu-id="6b965-104">Assignment made to same variable; did you mean to assign something else?</span></span>  
  
 <span data-ttu-id="6b965-105">Bu uyarı, kendi öğesine bir değişken atadığınızda oluşur `a = a` .</span><span class="sxs-lookup"><span data-stu-id="6b965-105">This warning occurs when you assign a variable to itself, such as `a = a`.</span></span>  
  
 <span data-ttu-id="6b965-106">Yaygın olarak karşılaşılan birkaç hata, bu uyarıyı oluşturabilir:</span><span class="sxs-lookup"><span data-stu-id="6b965-106">Several common mistakes can generate this warning:</span></span>  
  
- <span data-ttu-id="6b965-107">Gibi `a = a` bir **IF** ifadesinin koşulu olarak yazma `if (a = a)` .</span><span class="sxs-lookup"><span data-stu-id="6b965-107">Writing `a = a` as the condition of an **if** statement, such as `if (a = a)`.</span></span> <span data-ttu-id="6b965-108">Büyük olasılıkla `if (a == a)` her zaman doğru olan olduğunu varsayalım, bu nedenle bu öz daha fazla yazabilirsiniz `if (true)` .</span><span class="sxs-lookup"><span data-stu-id="6b965-108">You probably meant to say `if (a == a)`, which is always true, so you could write this more concisely as `if (true)`.</span></span>  
  
- <span data-ttu-id="6b965-109">Hatalı yazma.</span><span class="sxs-lookup"><span data-stu-id="6b965-109">Mistyping.</span></span> <span data-ttu-id="6b965-110">Büyük olasılıkla söylemeniz gerekir `a = b` .</span><span class="sxs-lookup"><span data-stu-id="6b965-110">You probably meant to say `a = b`.</span></span>  
  
- <span data-ttu-id="6b965-111">**Bu** anahtar sözcüğünün kullanıldığı, parametresinin alanla aynı ada sahip olduğu bir oluşturucuda: büyük olasılıkla söylemeniz gerekir `this.a = a` .</span><span class="sxs-lookup"><span data-stu-id="6b965-111">In a constructor where the parameter has the same name as the field, not using the **this** keyword: you probably meant to say `this.a = a`.</span></span>
