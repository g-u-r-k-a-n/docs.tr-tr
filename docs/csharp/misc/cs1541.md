# <a name="compiler-error-cs1541"></a>Derleyici hatası CS1541
Geçersiz başvuru seçeneği: ' symbol ' — dizinlere başvuramaz  
  
 Derleyici belirli bir dosya yerine bir dizin belirtme girişimi algıladı. Örneğin, [-Reference](../language-reference/compiler-options/reference-compiler-option.md) derleyici seçeneğini kullandığınızda bir dosya belirtmeniz gerekir; bir dizin belirtmek mümkün değildir.  
  
 Örneğin, derleyiciye geçiş `/reference:c:\` CS1541 oluşturur.
