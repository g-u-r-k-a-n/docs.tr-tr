# <a name="targetfilepath-specifies-an-existing-folder"></a>TargetFilePath var olan bir klasörü belirtir

Parametresi `TargetFilePath` var olan bir dizini belirtir, ancak yeni dizinin adını belirtmelidir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `TargetFilePath`Var olan dizin yerine yeni bir yeni belirtin.
