# <a name="helpprovider-component-windows-forms"></a>HelpProvider Bileşeni (Windows Forms)
Windows Forms `HelpProvider` bileşeni, bir HTML Yardım 1. x yardım dosyasını (HTML yardım Atölyiyle oluşturulan bir. chm dosyası ya da bir. htm dosyası) Windows tabanlı uygulamanızla ilişkilendirmek için kullanılır.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [HelpProvider Bileşenine Genel Bakış](helpprovider-component-overview-windows-forms.md)  
 , Bir HTML Yardım dosyasını Windows tabanlı bir uygulamayla ilişkilendirmenizi sağlayan `HelpProvider` bileşeninin genel kavramlarını tanıtır.
