## <a name="example"></a><span data-ttu-id="7b0c6-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="7b0c6-107">Example</span></span>  

 <span data-ttu-id="7b0c6-108">Aşağıdaki örnek CS1510 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7b0c6-108">The following sample generates CS1510:</span></span>  
  
```csharp  
// CS1510.cs  
public class C  
{  
   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
