# <a name="compiler-error-cs0311"></a><span data-ttu-id="b3924-102">Derleyici Hatası CS0311</span><span class="sxs-lookup"><span data-stu-id="b3924-102">Compiler Error CS0311</span></span>

<span data-ttu-id="b3924-103">'Type1' türü, genel türde 'T' türü veya 'ad\<>' yönteminde 'T' türü olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="b3924-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="b3924-104">'Type1'den 'type2'ye örtülü başvuru dönüştürme yoktur.</span><span class="sxs-lookup"><span data-stu-id="b3924-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="b3924-105">Genel bir tür parametresine bir kısıtlama uygulandığında, somut bağımsız değişkenden kısıtlama türüne örtülü bir kimlik veya başvuru dönüştürme sayılmalı.</span><span class="sxs-lookup"><span data-stu-id="b3924-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b3924-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="b3924-106">To correct this error</span></span>  
  
1. <span data-ttu-id="b3924-107">Sınıfı oluşturmak için kullandığınız bağımsız değişkeni değiştirin.</span><span class="sxs-lookup"><span data-stu-id="b3924-107">Change the argument you are using to create the class.</span></span>  
  
2. <span data-ttu-id="b3924-108">Sınıfın sahibiyseniz, kısıtlamayı kaldırabilir veya örtülü bir başvuru veya kimlik dönüştürmesini etkinleştirmek için bir şeyler yapabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="b3924-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="b3924-109">Örneğin, ikinci tür ilk devralabilir yapabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="b3924-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b3924-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="b3924-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="b3924-111">Bu hata, değer türünden bir bağımsız değişken kullanmaya çalışırken oluşursa, örneğin `short` `int`, genel bir tür parametresini karşılamadığını, örtük bir sayısal dönüştürmenin, genel bir tür parametresini karşılamadığını unutmayın.</span><span class="sxs-lookup"><span data-stu-id="b3924-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
