# <a name="specified-event-log-does-not-exist-on-this-machine"></a>Belirtilen olay günlüğü bu makinede yok

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Doğru dosya adını ve yolunu tedarik edin.
