# <a name="as-expected"></a>' As ' bekleniyor

Bildirimi yapılan bir `As` veya daha fazla öğe için bir yan tümce sağlamayan bir bildirim oluşur.  
  
 **Hata kimliği:** BC30197  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `As`Yan tümceyi gerekli olarak ifade edin veya öğesini açın `Option Strict Off` .
