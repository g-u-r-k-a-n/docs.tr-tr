# <a name="class-classname-does-not-implement-the-systemcollectionsicollection-interface"></a>' \<classname> ' Sınıfı System. Collections. ICollection arabirimini uygulamıyor

Sınıfı, `System.Collections.ICollection` tüm koleksiyonlar için boyut, Numaralandırıcılar ve eşitleme yöntemlerini tanımlayan arabirimi uygulamaz. Visual Studio 'Yu yeniden yüklemeniz gerekebilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Visual Studio'yu yeniden yükleyin.
