# <a name="catch-statements-are-not-valid-in-the-immediate-window"></a>' Catch ' deyimleri Komut penceresinde geçerli değildir
`Catch` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30715  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Catch` deyimlerini hata ayıklama kodundan kaldırın.
