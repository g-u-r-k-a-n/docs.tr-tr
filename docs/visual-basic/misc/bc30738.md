# <a name="sub-main-is-declared-more-than-once-in-namespace-message"></a>' Sub Main ', ' ' içinde birden çok kez bildiriliyor \<namespace> : \<message>

`Sub Main` bir ad alanı içinde yalnızca bir kez belirtilebilir.  
  
 **Hata kimliği:** BC30738  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Projenizde yalnızca tek bir yordamın bulunduğundan emin olun `Sub Main` .
