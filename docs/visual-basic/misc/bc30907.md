<span data-ttu-id="3da91-103">Devralma yapısı, iç içe sınıflar, diğer bir deyişle birbirinden devralan iki sınıf arasında döngüsel bağımlılığa neden olur.</span><span class="sxs-lookup"><span data-stu-id="3da91-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>  
  
 <span data-ttu-id="3da91-104">Aşağıdaki kod bu hata iletisini oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="3da91-104">The following code can generate this error message.</span></span>  
  
```vb  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 <span data-ttu-id="3da91-105">Önceki kodda, sınıf `c1` sınıfından devralır `c4` , ancak içinden devralınan, içinden devralan iç `c4` içe `c3` `c2` `c1` .</span><span class="sxs-lookup"><span data-stu-id="3da91-105">In the preceding code, class `c1` inherits from class `c4`, but `c4` is nested inside `c3`, which inherits from `c2`, nested inside `c1`.</span></span>
