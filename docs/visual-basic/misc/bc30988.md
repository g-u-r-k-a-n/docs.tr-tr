# <a name="type-or-with-expected"></a><span data-ttu-id="0a7c9-102">Tür veya ' with ' bekleniyor</span><span class="sxs-lookup"><span data-stu-id="0a7c9-102">Type or 'With' expected</span></span>
<span data-ttu-id="0a7c9-103">Bir sınıfın örneğini bildirdiğinizde, `New` anahtar kelimesinin ardından bir tür adı veya tarafından gelmelidir `With` .</span><span class="sxs-lookup"><span data-stu-id="0a7c9-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="0a7c9-104">Örneğin, aşağıdaki deyimler `client` sınıfının bir örneği olarak bildirir `Customer` .</span><span class="sxs-lookup"><span data-stu-id="0a7c9-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="0a7c9-105">Tür adı `Customer` aşağıdaki gibidir `New` .</span><span class="sxs-lookup"><span data-stu-id="0a7c9-105">The type name `Customer` follows `New`.</span></span>  
  
```vb  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```

