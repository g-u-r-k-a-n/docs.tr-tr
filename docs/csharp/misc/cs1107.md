## <a name="example"></a><span data-ttu-id="54ae5-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="54ae5-106">Example</span></span>  

 <span data-ttu-id="54ae5-107">Aşağıdaki örnek CS1107 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="54ae5-107">The following example generates CS1107:</span></span>  
  
```csharp

{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
