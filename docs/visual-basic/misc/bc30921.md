# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="014d8-102">' ' \<derivedtypename> , ' ' \<type> türünün erişimini ' ' \<constructedbasetypename> olarak genişlettiğinden \<internaltypename> ' \<region> \<regionname> ' öğesinden ' ' öğesinden devralma yapılamıyor</span><span class="sxs-lookup"><span data-stu-id="014d8-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>

<span data-ttu-id="014d8-103">Türetilmiş bir sınıf veya arabirim, bir iç türün erişim düzeyini, bir temel sınıf veya arabirime tür bağımsız değişkeni olarak kullanarak genişletmeye çalışır.</span><span class="sxs-lookup"><span data-stu-id="014d8-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="014d8-104">Aşağıdaki kod bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="014d8-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="014d8-105">Dış kodun `containingClass` erişimine izin verilmiyor `internalStructure` .</span><span class="sxs-lookup"><span data-stu-id="014d8-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="014d8-106">Ancak, `derivedClass` aynı derlemedeki herhangi bir koddan erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="014d8-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="014d8-107">Bu nedenle, `derivedClass` `baseClass` `internalStructure` bir tür bağımsız değişkeni olarak kullanıyorsa, bu, `internalStructure` tanımlayıcı kod bölgesi genelinde ortaya çıkmasına neden olabileceğinden, devralma yapılamaz.</span><span class="sxs-lookup"><span data-stu-id="014d8-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="014d8-108">**Hata kimliği:** BC30921</span><span class="sxs-lookup"><span data-stu-id="014d8-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="014d8-109">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="014d8-109">To correct this error</span></span>  
  
- <span data-ttu-id="014d8-110">Türetilmiş türün iç türün erişim düzeyini genişletmemesi için sınıfların veya arabirimlerin erişim düzeylerini ayarlayın.</span><span class="sxs-lookup"><span data-stu-id="014d8-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="014d8-111">-veya-</span><span class="sxs-lookup"><span data-stu-id="014d8-111">-or-</span></span>  
  
- <span data-ttu-id="014d8-112">Erişim düzeylerini ayarlayamadığından, temel sınıfı veya arabirimi oluştururken iç türü bir tür bağımsız değişkeni olarak kullanmayın.</span><span class="sxs-lookup"><span data-stu-id="014d8-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
