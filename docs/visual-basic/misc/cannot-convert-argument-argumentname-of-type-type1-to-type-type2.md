# <a name="cannot-convert-argument-argumentname-of-type-type1-to-type-type2"></a>' ' Türündeki ' ' bağımsız değişkeni \<argumentname> \<type1> ' \<type2> ' türüne dönüştürülemiyor
Bir dönüştürme işlemi, bir bağımsız değişkeni uyumsuz bir türe dönüştürmeye çalışır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bağımsız değişkeni dönüştürmeye çalıştığınız türün geçerli olduğundan emin olun.
