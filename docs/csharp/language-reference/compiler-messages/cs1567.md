# <a name="compiler-error-cs1567"></a>Derleyici Hatası CS1567

Win32 kaynağı oluşturulurken hata oluştu: ' dosya '  
  
 Derlemeniz [-win32icon](../compiler-options/win32icon-compiler-option.md) derleyici seçeneğini kullandı veya, derleyicinin kaynak bilgilerini içeren bir dosya oluşturmasına neden olan [-win32res](../compiler-options/win32res-compiler-option.md)öğesini kullanmıştı, ancak derleyici yetersiz disk alanı veya başka bir hata nedeniyle dosyayı oluşturamadı.  
  
 Dosya oluşturma sorununu çözemezse, kaynak bilgilerini içeren bir dosya oluşturabilen [-win32res](../compiler-options/win32res-compiler-option.md)kullanabilirsiniz.
