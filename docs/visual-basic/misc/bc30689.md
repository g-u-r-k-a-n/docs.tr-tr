# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>İfade bir metot gövdesi/çok satırlı lambda dışında görünemez
Çalıştırılabilir deyimler yordamlar içinde kullanılmalıdır.  
  
 **Hata kimliği:** BC30689  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfadeyi bir yordama veya alt yordama taşıyın.
