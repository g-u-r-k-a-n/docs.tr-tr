# <a name="linklabel-control-windows-forms"></a>LinkLabel Denetimi (Windows Forms)
Windows Forms `LinkLabel` denetimi, Windows Forms uygulamalara Web stili bağlantılar eklemenize olanak sağlar. <xref:System.Windows.Forms.Label> denetimini kullanabileceğiniz her şey için `LinkLabel` denetimini kullanabilirsiniz; Ayrıca, metnin bir bölümünü bir nesne veya Web sayfasına bağlantı olarak da ayarlayabilirsiniz.  
  
## <a name="in-this-section"></a>Bu Bölümde

 Bu denetimin ne olduğunu ve temel özelliklerini ve özelliklerini açıklar.  
  
 [Nasıl yapılır: Windows Forms LinkLabel Denetiminin Görünüşünü Değiştirme](how-to-change-the-appearance-of-the-windows-forms-linklabel-control.md)  
 Bağlantı etiketlerindeki Web stili bağlantıların rengini ve altını belirtmenin yollarını listeler.  
  
 [Nasıl yapılır: Windows Forms LinkLabel Denetimi ile Bir Nesneye veya Web Sayfasına Bağlama](link-to-an-object-or-web-page-with-wf-linklabel-control.md)  
 Bir bağlantı tıklandığında bir form veya Web sayfası açmak için nasıl yapılır yönergeleri sağlar.  
  
 [Nasıl yapılır: Bir Windows Forms LinkLabel Denetiminden Web Sayfası Görüntüleme (Visual Basic)](display-a-web-page-from-a-wf-linklabel-control-visual-basic.md)  
 Kullanıcı Windows Forms `LinkLabel` denetimine tıkladığında bir Web sayfasının varsayılan tarayıcıda nasıl görüntüleneceğini gösterir.
