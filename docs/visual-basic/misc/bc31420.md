# <a name="member-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>' \<member> ', tüm sabit listelerinde örtük olarak belirtilen bu ada sahip ayrılmış üyeyle çakışıyor

Bir tür üyesinin adı, tüm numaralandırmalarda örtük olarak bildirildiği bir üyenin adı ile çakışıyor.  
  
 Visual Basic derleyici, bildirdiğiniz belirli programlama öğelerine karşılık gelen örtük Üyeler oluşturur. Numaralandırmalar üyeyi örtük olarak bildirir `value__member` .  
  
 **Hata kimliği:** BC31420  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Üyenin adını değiştirin.
