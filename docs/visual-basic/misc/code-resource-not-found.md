# <a name="code-resource-not-found"></a>Kod kaynağı bulunamadı

Kod kaynağındaki bir yordama bir çağrı yapıldı, ancak kod kaynağı bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kaynağın kullanılabilir ve doğru şekilde başvurulduğundan emin olun.
