<span data-ttu-id="4c841-103">Genel bir arabirim iki veya daha fazla genel arabirimden kalıtımla alır ve tür bağımsız değişkenlerin belirli değerleri için devrasallarından ikisi çakışabilir.</span><span class="sxs-lookup"><span data-stu-id="4c841-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="4c841-104">Aşağıdaki deyimler bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="4c841-104">The following statements can generate this error.</span></span>  
  
```vb  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="4c841-105">`derivedInterface`Oluşturulmuş veya uygulanmış ise, her ikisine de aynı tür sağlamak `t1` için `t2` , aynı tür bağımsız değişkenleriyle iki sürümünü devralması gerekir `interfaceX` .</span><span class="sxs-lookup"><span data-stu-id="4c841-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="4c841-106">Bunun yapılması, hangi sürümün erişecağına bir belirsizlik üretir.</span><span class="sxs-lookup"><span data-stu-id="4c841-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="4c841-107">**Hata kimliği:** BC32121</span><span class="sxs-lookup"><span data-stu-id="4c841-107">**Error ID:** BC32121</span></span>
