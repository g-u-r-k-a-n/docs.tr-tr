# <a name="compiler-error-cs0077"></a><span data-ttu-id="8d49a-103">Derleyici hatası CS0077</span><span class="sxs-lookup"><span data-stu-id="8d49a-103">Compiler Error CS0077</span></span>
<span data-ttu-id="8d49a-104">As işleci bir başvuru türüyle veya null olabilen bir türle birlikte kullanılmalıdır (' int ', null olamayan bir değer türüdür).</span><span class="sxs-lookup"><span data-stu-id="8d49a-104">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="8d49a-105">[As](../language-reference/operators/type-testing-and-cast.md#as-operator) işleci bir [değer türü](../language-reference/builtin-types/value-types.md)geçirdi.</span><span class="sxs-lookup"><span data-stu-id="8d49a-105">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/builtin-types/value-types.md).</span></span> <span data-ttu-id="8d49a-106">`as` [Null](../language-reference/keywords/null.md)döndürebileceğinden, yalnızca bir [başvuru türü](../language-reference/keywords/reference-types.md) veya [null olabilen bir değer türü](../language-reference/builtin-types/nullable-value-types.md)geçirilebilir.</span><span class="sxs-lookup"><span data-stu-id="8d49a-106">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed a [reference type](../language-reference/keywords/reference-types.md) or a [nullable value type](../language-reference/builtin-types/nullable-value-types.md).</span></span>
  
 <span data-ttu-id="8d49a-107">Aşağıdaki örnek CS0077 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8d49a-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
