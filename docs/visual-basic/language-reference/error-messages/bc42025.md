## <a name="example"></a><span data-ttu-id="aadf8-114">Örnek</span><span class="sxs-lookup"><span data-stu-id="aadf8-114">Example</span></span>

<span data-ttu-id="aadf8-115">Aşağıdaki örnek BC42025 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="aadf8-115">The following example generates bc42025:</span></span>

```vb

    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        Dim tc As New TestClass()
        tc.SayHello() ' BC42025.
    End Sub
End Module
```

## <a name="to-address-this-warning"></a><span data-ttu-id="aadf8-116">Bu uyarıyı çözmek için</span><span class="sxs-lookup"><span data-stu-id="aadf8-116">To address this warning</span></span>
