# <a name="icordebugprocess7-interface"></a>ICorDebugProcess7 Arabirimi

[.NET Framework 4.5.2 ve sonraki sürümlerde desteklenir]  
  
 Hata ayıklayıcıyı, hedef işlemdeki bellek içi meta veri güncelleştirmelerini işleyecek şekilde yapılandıran bir yöntem sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[SetWriteableMetadataUpdateMode Yöntemi](icordebugprocess7-setwriteablemetadataupdatemode-method.md)|Hata ayıklayıcının, hedef işlem içindeki meta veriler için bellek içi güncelleştirmeleri nasıl işleyeceğini belirleyen bir değer ayarlar.|
