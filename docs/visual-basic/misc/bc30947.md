# <a name="suspend-request-is-pending"></a>Askıya alma isteği bekliyor
Visual Studio hata ayıklayıcısında bir ifade bir yordam çağrısını belirtir, ancak iş parçacığını askıya alma isteği vardır.  
  
 Hata ayıklayıcı etkin olmayan bir iş parçacığında bir yordamı çağırmayı denemez.  
  
 **Hata kimliği:** BC30947  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Mümkünse, iş parçacığını askıya almak için isteğin kaynağını saptayın, bu sayede yeniden oluşmasını engelleyebilirsiniz.  
  
- Sonlandırın ve hata ayıklamayı yeniden başlatın.
