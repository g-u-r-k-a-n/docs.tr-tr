# <a name="compiler-error-cs0413"></a><span data-ttu-id="b73bc-102">Derleyici Hatası CS0413</span><span class="sxs-lookup"><span data-stu-id="b73bc-102">Compiler Error CS0413</span></span>
<span data-ttu-id="b73bc-103">Sınıf türü kısıtlaması veya 'sınıf' kısıtlaması olmadığı için 'as' işleci ile 'tür parametresi' türü parametresi kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="b73bc-103">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
<span data-ttu-id="b73bc-104">Genel bir tür [işleyici olarak](../operators/type-testing-and-cast.md#as-operator) kullanır, ancak bu genel tür bir sınıf türü kısıtlaması yoksa bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="b73bc-104">This error occurs if a generic type uses the [as](../operators/type-testing-and-cast.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="b73bc-105">Operatöryalnızca `as` başvuru ve nullable değer türleri ile izin verilir, bu nedenle tür parametresi bir değer türü olmadığını garanti etmek için kısıtlanmış olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="b73bc-105">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="b73bc-106">Bu hatayı önlemek için bir sınıf türü kısıtlaması veya başvuru türü kısıtlaması kullanın.</span><span class="sxs-lookup"><span data-stu-id="b73bc-106">To avoid this error, use a class type constraint or a reference type constraint.</span></span>
  
<span data-ttu-id="b73bc-107">Bunun nedeni, `as` işleç `null`bir değer türü için olası bir değer değildir ve sınıf türü kısıtlaması veya başvuru türü kısıtlaması olmadığı sürece tür parametresi bir değer türü olarak kabul edilmelidir döndürebilir olmasıdır.</span><span class="sxs-lookup"><span data-stu-id="b73bc-107">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>
  
## <a name="example"></a><span data-ttu-id="b73bc-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="b73bc-108">Example</span></span>

<span data-ttu-id="b73bc-109">Aşağıdaki örnek CS0413 üretir.</span><span class="sxs-lookup"><span data-stu-id="b73bc-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
