# <a name="compiler-error-cs0508"></a><span data-ttu-id="3092d-103">Derleyici Hatası CS0508</span><span class="sxs-lookup"><span data-stu-id="3092d-103">Compiler Error CS0508</span></span>
<span data-ttu-id="3092d-104">' Type 1 ': geçersiz kılınan ' üye adı ' üyesini eşleştirmek için dönüş türünün ' Type 2 ' olması gerekir</span><span class="sxs-lookup"><span data-stu-id="3092d-104">'Type 1': return type must be 'Type 2' to match overridden member 'Member Name'</span></span>  
  
 <span data-ttu-id="3092d-105">Yöntem geçersiz kılmada dönüş türünü değiştirme girişiminde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="3092d-105">An attempt was made to change the return type in a method override.</span></span> <span data-ttu-id="3092d-106">Bu hatayı çözmek için her iki yöntemin de aynı dönüş türünü bildirdiğinizden emin olun.</span><span class="sxs-lookup"><span data-stu-id="3092d-106">To resolve this error, make sure both methods declare the same return type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3092d-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="3092d-107">Example</span></span>  
 <span data-ttu-id="3092d-108">Aşağıdaki örnek CS0508 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="3092d-108">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
