# <a name="argument-path-is-nothing-or-empty"></a>' Path ' bağımsız değişkeni Nothing veya boş

Bir yol adı, veya boş olarak başlatıldığı için geçerli değil `Nothing` . Bir yol, belirli bir veri türüne ve geçerli bir ada sahip olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli bir yol adı ekleyin.
