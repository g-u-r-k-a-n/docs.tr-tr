## <a name="example"></a><span data-ttu-id="4c868-112">Örnek</span><span class="sxs-lookup"><span data-stu-id="4c868-112">Example</span></span>

<span data-ttu-id="4c868-113">Aşağıdaki örnek, `AttributeUsage` yeni sürümünün hangi program öğelerine uygulanacağını belirtmek için özniteliğini kullanır `ExtensionAttribute` .</span><span class="sxs-lookup"><span data-stu-id="4c868-113">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="4c868-114">Örnek, numaralandırmanın üç üyesini belirtir `AttributeTargets` : `Assembly` , `Class` ve `Method` .</span><span class="sxs-lookup"><span data-stu-id="4c868-114">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="4c868-115">Bu öğelerden herhangi birinin atlanması bu hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="4c868-115">The omission of any one of these elements will cause this error.</span></span>

```vb

    <AttributeUsage(AttributeTargets.Assembly Or _
        AttributeTargets.Class Or AttributeTargets.Method)>
    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```

<span data-ttu-id="4c868-116">Alternatif olarak, `ExtensionAttribute` öğesinin üyesini kullanarak tüm program öğelerine uygulanmasına izin verebilirsiniz `All` `AttributeTargets` .</span><span class="sxs-lookup"><span data-stu-id="4c868-116">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>
