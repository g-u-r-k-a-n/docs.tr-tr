# <a name="end-statement-cannot-be-used-in-class-library-projects"></a>' End ' deyimleri sınıf kitaplığı projelerinde kullanılamaz

DLL oluşturmak için kullanılan sınıf kitaplığı projeleri, `End` bir yordamdaki kodun yürütülmesini durdurmak için anahtar sözcüğünün kullanılmasına izin vermez.  
  
 **Hata kimliği:** BC30615  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `While` `For` Program yürütme akışını denetlemek için ve gibi denetim yapılarını kullanın.
