# <a name="compiler-error-cs1578"></a><span data-ttu-id="eb930-103">Derleyici hatası CS1578</span><span class="sxs-lookup"><span data-stu-id="eb930-103">Compiler Error CS1578</span></span>

<span data-ttu-id="eb930-104">Dosya adı, tek satırlık açıklama veya satır sonu bekleniyor</span><span class="sxs-lookup"><span data-stu-id="eb930-104">Filename, single-line comment or end-of-line expected</span></span>  
  
 <span data-ttu-id="eb930-105">[#Line](../language-reference/preprocessor-directives/preprocessor-line.md) yönergesinden sonra, yalnızca bir dosya adı (çift tırnak işareti içinde) veya tek satırlık açıklamaya izin verilir.</span><span class="sxs-lookup"><span data-stu-id="eb930-105">After a [#line](../language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="eb930-106">Aşağıdaki örnek CS1578 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="eb930-106">The following sample generates CS1578:</span></span>  
  
```csharp  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
