# <a name="compiler-error-cs0431"></a><span data-ttu-id="94ecb-102">Derleyici Hatası CS0431</span><span class="sxs-lookup"><span data-stu-id="94ecb-102">Compiler Error CS0431</span></span>
<span data-ttu-id="94ecb-103">Takma ad bir türe atıfta bulunuldığından, 'tanımlayıcı' takma adını '::' ile kullanamazsınız.</span><span class="sxs-lookup"><span data-stu-id="94ecb-103">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="94ecb-104">Bunun yerine '.' kullanın.</span><span class="sxs-lookup"><span data-stu-id="94ecb-104">Use '.' instead.</span></span>  
  
 <span data-ttu-id="94ecb-105">Bir türe başvuruyapan bir takma adla "::" sözcü.</span><span class="sxs-lookup"><span data-stu-id="94ecb-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="94ecb-106">Bu hatayı gidermek için "." işlecini kullanın.</span><span class="sxs-lookup"><span data-stu-id="94ecb-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="94ecb-107">Aşağıdaki örnek CS0431 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="94ecb-107">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
