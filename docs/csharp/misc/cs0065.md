# <a name="compiler-error-cs0065"></a><span data-ttu-id="cbf93-103">Derleyici hatası CS0065</span><span class="sxs-lookup"><span data-stu-id="cbf93-103">Compiler Error CS0065</span></span>

<span data-ttu-id="cbf93-104">' Event ': olay özelliğinin hem ekleme hem de kaldırma erişimcileri olmalıdır</span><span class="sxs-lookup"><span data-stu-id="cbf93-104">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="cbf93-105">Bir alan olmayan bir olay, her iki erişim yöntemine de sahip olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="cbf93-105">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="cbf93-106">Aşağıdaki örnek CS0065 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="cbf93-106">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
