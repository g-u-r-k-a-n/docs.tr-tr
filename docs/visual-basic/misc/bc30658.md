# <a name="property-propertyname-with-no-parameters-cannot-be-found"></a>\<propertyname>Parametresi olmayan ' ' özelliği bulunamıyor

Kullanılan özniteliğin bir veya daha fazla parametre gerektiren bir özelliği vardır.  
  
 **Hata kimliği:** BC30658  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özellik için parametreleri belirtin.
