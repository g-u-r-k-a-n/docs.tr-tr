# <a name="file-io-with-type-typename-is-not-valid"></a>' ' Türündeki dosya g/ç \<typename> geçerli değil

Bir dosyaya desteklenmeyen bir tür yazmaya çalışıyorsunuz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Desteklenmeyen türün nerede oluştuğunu öğrenmek için yapıyı denetleyin ve sonra geçerli bir türle değiştirin.
