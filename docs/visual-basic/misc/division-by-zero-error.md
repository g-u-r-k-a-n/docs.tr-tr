# <a name="division-by-zero-visual-basic-error"></a>Sıfıra bölme (Visual Basic hatası)

Bölen olarak kullanılan bir ifadenin değeri sıfırdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İfadedeki değişkenlerin yazımını denetleyin. Yanlış yazılmış bir değişken, sıfır olarak başlatılan bir sayısal değişkeni örtülü olarak oluşturabilir.  
  
2. İfadede, özellikle diğer yordamlardan bağımsız değişken olarak geçirilen bir değişkende önceki işlemleri denetleyin.
