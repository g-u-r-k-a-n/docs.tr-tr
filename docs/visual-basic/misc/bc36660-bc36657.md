<span data-ttu-id="0475e-106">Tür çıkarımı, genel yordama yönelik bir çağrı değerlendirilirken tür parametresinin veya parametrelerinin veri türünü veya türlerini belirlemekte kullanılmak üzere bir girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="0475e-106">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="0475e-107">Derleyici, tüm bağımsız değişkenlerin kısıtlamalarını karşılayan bir veri türü bulamadı.</span><span class="sxs-lookup"><span data-stu-id="0475e-107">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="0475e-108">Bu nedenle, bu hatayı bildirdi.</span><span class="sxs-lookup"><span data-stu-id="0475e-108">Therefore, it reported this error.</span></span>

> <span data-ttu-id="0475e-109">Bağımsız değişkenlerin belirtilmesi bir seçenek değil (örneğin, sorgu ifadelerinde sorgu işleçleri için), ikinci tümce olmadan hata iletisi görüntülenir.</span><span class="sxs-lookup"><span data-stu-id="0475e-109">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="0475e-112">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="0475e-112">To correct this error</span></span>
