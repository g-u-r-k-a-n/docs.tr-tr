## <a name="exception-list"></a>Özel durum listesi  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|MessageMustHaveViaOrToSetForSendingOnServerSideCompositeDuplexChannels|Sunucu bileşik çift yönlü kanallarda bir ileti göndermek için iletinin ' Via ' özelliğine ya da ' to ' üst bilgisi kümesine sahip olması gerekir.|
