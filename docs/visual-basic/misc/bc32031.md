# <a name="end-sub-must-be-the-first-statement-on-a-line"></a>' End Sub ' satırdaki ilk ifade olmalıdır
Bir `End Sub` ifade bir iki nokta üst üste (:) Ekstre ayırıcı. `End Sub`kaynak satırındaki tek bir ifade olmalıdır.  
  
 **Hata kimliği:** BC32031  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyimi farklı satırlara ayırın.
