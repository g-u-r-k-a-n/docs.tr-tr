# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>Otomasyon işlemi sırasında dosya adı veya sınıf adı bulunamadı (Visual Basic)

İşleve yapılan çağrıda dosya adı veya sınıf için belirtilen ad `GetObject` bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Adları denetleyip yeniden deneyin. Parametresi için kullanılan adın `class` sistemle birlikte kaydedilmiş olarak eşleştiğinden emin olun.
