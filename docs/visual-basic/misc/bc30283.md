# <a name="sub-new-cannot-be-declared-overrides"></a>' Sub New ' ' geçersiz kılmalar ' olarak bildirilemez

Oluşturucu devralınmış bir oluşturucuyu geçersiz kıldığını belirtir. Oluşturucular geçersiz kılınamaz.  
  
 **Hata kimliği:** BC30283  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Overrides`Anahtar sözcüğünü `Sub` bildirimden kaldırın.
