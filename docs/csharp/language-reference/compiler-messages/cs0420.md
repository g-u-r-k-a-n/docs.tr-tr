# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="eb5ab-102">Derleyici Uyarısı (düzey 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="eb5ab-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="eb5ab-103">'tanımlayıcı': uçucu bir alana yapılan bir başvuru geçici olarak ele alınmayacaktır</span><span class="sxs-lookup"><span data-stu-id="eb5ab-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="eb5ab-104">İşlev kapsamında geçici olarak kabul `ref` edilmeyeceğiniz için, geçici bir alan normalde bir veya **dış** parametre kullanılarak geçirilmemelidir.</span><span class="sxs-lookup"><span data-stu-id="eb5ab-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="eb5ab-105">Bunun, birbirine bağlı bir API'yi çağırma gibi istisnalar vardır.</span><span class="sxs-lookup"><span data-stu-id="eb5ab-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="eb5ab-106">Herhangi bir uyarıda olduğu gibi, başvuru parametresi olarak geçici bir alan kasıtlı olarak kullandığınız nadir durumlarda bu uyarıyı devre dışı bırakma#pragma [uyarısını](../preprocessor-directives/preprocessor-pragma-warning.md) kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="eb5ab-106">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="eb5ab-107">Aşağıdaki örnek CS0420 üretir:</span><span class="sxs-lookup"><span data-stu-id="eb5ab-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```
