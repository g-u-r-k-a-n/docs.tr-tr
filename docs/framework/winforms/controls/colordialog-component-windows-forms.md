# <a name="colordialog-component-windows-forms"></a>ColorDialog Bileşeni (Windows Forms)
Windows Forms <xref:System.Windows.Forms.ColorDialog> bileşeni, kullanıcının bir paletten renk seçmesine ve bu palete özel renkler eklemesine olanak tanıyan önceden yapılandırılmış bir iletişim kutusudur. Renkleri seçmek için diğer Windows tabanlı uygulamalarda gördüğünüz iletişim kutusudur. Kendi iletişim kutunuzu yapılandırmak yerine basit bir çözüm olarak Windows tabanlı uygulamanızda kullanın.
