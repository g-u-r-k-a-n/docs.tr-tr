# <a name="piiloggingnotallowed"></a>PiiLoggingNotAllowed

Kimlik: 108  
  
 Önem derecesi: Hata  
  
 Kategori: Izleme  
  
## <a name="description"></a>Description  

 Bu olay, hiç bilinen PII günlüğe kaydedilmeyeceğini gösterir. Bilinen PII günlüğüne izin verilmiyor. Bilinen PII kaydına izin vermek için, "enableLoggingKnownPII" değerini Machine.config olarak ayarlayın `true` . Olayda işlem adı ve işlem KIMLIĞI listelenir.
