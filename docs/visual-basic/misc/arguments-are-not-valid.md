# <a name="arguments-are-not-valid"></a>Bağımsız değişkenler geçerli değil
Geçerli olmayan bağımsız değişkenler bir işleve veya alt yordama geçirilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Neden geçersiz olduklarını tespit etmek için bağımsız değişkenleri denetleyin.  
  
2. Bağımsız değişkenler hesaplamalar tarafından sağlandıysa, hesaplamalar işleminin değerlerini denetleyin.
