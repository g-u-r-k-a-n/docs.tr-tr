# <a name="addressof-expressions-are-not-valid-in-debug-windows"></a>' AddressOf ' ifadeleri hata ayıklama pencereleri içinde geçerli değil
`AddressOf` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30731  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `AddressOf` deyimlerini hata ayıklama kodundan kaldırın.
