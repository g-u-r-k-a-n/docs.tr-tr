# <a name="compiler-error-cs1644"></a>Derleyici Hatası CS1644
' Feature ' özelliği Standartlaştırılmış ISO C# dili belirtiminin bir parçası değil ve diğer derleyiciler tarafından kabul edilemez  
  
 Bu hata, [-langversion](../compiler-options/langversion-compiler-option.md) seçeneğini ISO-1 belirttiyseniz ve derlediğiniz kod ISO 1,0 standardının parçası olmayan özellikler kullanıyorsa oluşur. Bu hatayı çözmek için, genel türler veya anonim yöntemler gibi C# 2,0 derleyici ÖZELLIKLERININ hiçbirini ISO-1 uyumluluk seçeneğiyle kullanmayın.  
  
 Aşağıdaki örnek CS1644 oluşturur:  
  
```csharp  
// CS1644.cs  
// compile with: /langversion:ISO-1 /target:library  
class C<T> {}   // CS1644  
```
