# <a name="get-not-supported-write-only-property"></a>Get desteklenmiyor (salt yazılır özellik)

Bir özellik salt yazılır olduğundan okunamaz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özelliği denetleyin ve hangi koşullara ayarlanbileceğinize göre belirleyin.  
  
2. Özelliği başvurusunu kaldırın.
