# <a name="compiler-error-cs2016"></a>Derleyici hatası CS2016

' CodePage ' kod sayfası geçersiz veya yüklü değil  
  
 [**CodePage**](../language-reference/compiler-options/advanced.md#codepage) derleyici seçeneği geçersiz bir değer geçirdi.  
  
 Aşağıdaki örnek CS2016 oluşturur:  
  
```csharp  
// CS2016.cs  
// compile with: /codepage:x  
// CS2016 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
