# <a name="compiler-error-cs1509"></a>Derleyici hatası CS1509

Başvurulan ' File ' dosyası bir derleme değil; Bunun yerine **AddModules** seçeneğini kullanın  
  
 [**TargetType**](../language-reference/compiler-options/output.md#targettype) (bir derleme bildirimine sahip olmayan) **Modül** öğesini kullanan derlemede üretilen bir çıkış dosyası (çıkış dosyası 1), [**başvuru**](../language-reference/compiler-options/inputs.md#references)olarak belirtilmiştir. Bu nedenle, geçerli program için derlemeye bir derleme eklemek yerine, çıkış dosyası 1 ' deki meta veri bilgileri geçerli program için derlemeye eklenecektir.
