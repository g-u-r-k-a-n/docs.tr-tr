# <a name="combobox-control-windows-forms"></a>ComboBox Denetimi (Windows Forms)
Windows Forms `ComboBox` denetimi, verileri açılan bir açılan kutuda göstermek için kullanılır. Varsayılan olarak, `ComboBox` denetimi iki bölümden oluşur: üstteki bölüm, kullanıcının bir liste öğesi yazarak bir metin kutusudur. İkinci bölüm, kullanıcının seçim bileceği öğelerin bir listesini görüntüleyen bir liste kutusudur.
