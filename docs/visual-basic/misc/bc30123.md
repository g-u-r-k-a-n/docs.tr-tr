# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>' End ' deyimleri Komut penceresinde geçerli değildir

`Stop`Ve `End` deyimleri yürütmeyi askıya alır ve bir hata ayıklama bağlamında buna izin verilmez.  
  
 **Hata kimliği:** BC30123  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir `End` veya `Stop` ifadesini **komut** penceresinde vermeyin.
