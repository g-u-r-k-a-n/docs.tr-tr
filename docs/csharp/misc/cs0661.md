# <a name="compiler-warning-level-3-cs0661"></a><span data-ttu-id="bed8e-103">Derleyici Uyarısı (düzey 3) CS0661</span><span class="sxs-lookup"><span data-stu-id="bed8e-103">Compiler Warning (level 3) CS0661</span></span>
<span data-ttu-id="bed8e-104">' class ' işleç = = veya operator! = tanımlar, ancak Object. GetHashCode () öğesini geçersiz kılmaz</span><span class="sxs-lookup"><span data-stu-id="bed8e-104">'class' defines operator == or operator != but does not override Object.GetHashCode()</span></span>  
  
 <span data-ttu-id="bed8e-105">Derleyici Kullanıcı tanımlı eşitlik veya eşitsizlik işlecini algıladı, ancak **GetHashCode** işlevi için geçersiz kılma yok.</span><span class="sxs-lookup"><span data-stu-id="bed8e-105">The compiler detected the user-defined equality or inequality operator, but no override for the **GetHashCode** function.</span></span> <span data-ttu-id="bed8e-106">Kullanıcı tanımlı eşitlik veya eşitsizlik operatörü, **GetHashCode** işlevini de geçersiz kılmak istediğinizi belirtir.</span><span class="sxs-lookup"><span data-stu-id="bed8e-106">A user-defined equality or inequality operator implies that you also want to override the **GetHashCode** function.</span></span>  
  
 <span data-ttu-id="bed8e-107">Aşağıdaki örnek CS0661 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bed8e-107">The following sample generates CS0661:</span></span>  
  
```csharp  
// CS0661.cs  
// compile with: /W:3  
class Test   // CS0661  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   public static bool operator != (object o, Test t)  
   {  
      return true;  
   }  
  
   public override bool Equals(object o)  
   {  
      return true;  
   }  
  
   // uncomment the GetHashCode function to resolve  
   // public override int GetHashCode()  
   // {  
   //    return 0;  
   // }  
  
   public static void Main()  
   {  
   }  
}  
```
