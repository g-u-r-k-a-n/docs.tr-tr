# <a name="declare-statements-in-a-module-cannot-be-declared-specifier"></a>Modül içindeki Declare deyimleri ' ' olarak bildirilemez \<specifier>

`Declare`Bildirimin içinde geçerli olmayan bir tanımlayıcı vardır `Module` . Modüller hiçbir şekilde örneklenemez, devralmayı desteklemez ve arabirimler uygulayamaz.  
  
 **Hata kimliği:** BC30786  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirteci kaldırın.
