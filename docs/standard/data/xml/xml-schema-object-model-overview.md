# <a name="xml-schema-object-model-overview"></a><span data-ttu-id="e41a1-102">XML Şema Nesne Modeline (SOM) Genel Bakış</span><span class="sxs-lookup"><span data-stu-id="e41a1-102">XML Schema Object Model Overview</span></span>
<span data-ttu-id="e41a1-103">Microsoft .NET çerçevesindeki şema nesne modeli (SOM), şemaları programlı bir şekilde oluşturmanıza, düzenlemenize ve doğrulamanıza olanak sağlayan zengin bir API 'dir.</span><span class="sxs-lookup"><span data-stu-id="e41a1-103">The Schema Object Model (SOM) in the Microsoft .NET Framework is a rich API that allows you to create, edit, and validate schemas programmatically.</span></span> <span data-ttu-id="e41a1-104">SOM, XML şeması belgelerinde Belge Nesne Modeli (DOM) XML belgelerinde çalışma biçimine benzer şekilde çalışır.</span><span class="sxs-lookup"><span data-stu-id="e41a1-104">The SOM operates on XML schema documents similarly to the way the Document Object Model (DOM) operates on XML documents.</span></span> <span data-ttu-id="e41a1-105">XML şeması belgeleri, SOM 'a bir kez yüklendikten sonra şemaya uygun diğer XML belgelerinin yapısı ve geçerliliği hakkında anlam veren geçerli XML dosyalarıdır.</span><span class="sxs-lookup"><span data-stu-id="e41a1-105">XML schema documents are valid XML files that, once loaded into the SOM, convey meaning about the structure and validity of other XML documents which conform to the schema.</span></span>  
  
 <span data-ttu-id="e41a1-106">Şema, belirli bir şema için XML belgelerinin yapısını veya modelini belirterek XML belgelerinin bir sınıfını tanımlayan bir XML belgesidir.</span><span class="sxs-lookup"><span data-stu-id="e41a1-106">A schema is an XML document that defines a class of XML documents by specifying the structure or model of XML documents for a particular schema.</span></span> <span data-ttu-id="e41a1-107">Bir şema, XML belgelerinin içeriğiyle ilgili kısıtlamaları tanımlar ve uyumlu XML belgelerinin, şema için geçerli olan şemayla birlikte kabul edilebilmesi için izlenmesi gereken sözlük (kurallar veya dilbilgisi) açıklanmaktadır.</span><span class="sxs-lookup"><span data-stu-id="e41a1-107">A schema identifies the constraints on the content of the XML documents, and describes the vocabulary (rules or grammar) that compliant XML documents must follow in order to be considered schema-valid with that particular schema.</span></span> <span data-ttu-id="e41a1-108">Bir XML belgesinin doğrulanması, belgenin şema tarafından belirtilen dilbilgisine uyduğundan emin olmanızı sağlayan işlemdir.</span><span class="sxs-lookup"><span data-stu-id="e41a1-108">Validation of an XML document is the process that ensures that the document conforms to the grammar specified by the schema.</span></span>  
  
 <span data-ttu-id="e41a1-109">Aşağıda, .NET Framework 'teki SOM API 'sinin şemaları oluşturmanıza, düzenlemenize ve doğrulamanıza olanak tanıyan yollar verilmiştir.</span><span class="sxs-lookup"><span data-stu-id="e41a1-109">The following are ways the SOM API in the .NET Framework enables you to create, edit, and validate schemas.</span></span>  
  
- <span data-ttu-id="e41a1-110">Dosyalardan ve dosyalarından geçerli şemaları yükleyin ve kaydedin.</span><span class="sxs-lookup"><span data-stu-id="e41a1-110">Load and save valid schemas to and from files.</span></span>  
  
- <span data-ttu-id="e41a1-111">Kesin olarak belirlenmiş sınıfları kullanarak bellek içi şemalar oluşturun.</span><span class="sxs-lookup"><span data-stu-id="e41a1-111">Create in-memory schemas using strongly typed classes.</span></span>  
  
- <span data-ttu-id="e41a1-112">Şemaları önbelleğe alma <xref:System.Xml.Schema.XmlSchemaSet> , derleme ve alma amacıyla sınıfla etkileşime geçin.</span><span class="sxs-lookup"><span data-stu-id="e41a1-112">Interact with the <xref:System.Xml.Schema.XmlSchemaSet> class to cache, compile, and retrieve schemas.</span></span>  
  
- <span data-ttu-id="e41a1-113">XML örnek belgelerinin <xref:System.Xml.XmlReader.Create%2A> şemalara karşı doğrulanması <xref:System.Xml.XmlReader> için sınıfının yöntemiyle etkileşime geçin.</span><span class="sxs-lookup"><span data-stu-id="e41a1-113">Interact with the <xref:System.Xml.XmlReader.Create%2A> method of the <xref:System.Xml.XmlReader> class to validate XML instance documents against schemas.</span></span>  
  
- <span data-ttu-id="e41a1-114">Şemaları oluşturmaya ve korumaya yönelik düzenleyiciler oluşturun.</span><span class="sxs-lookup"><span data-stu-id="e41a1-114">Build editors for creating and maintaining schemas.</span></span>  
  
- <span data-ttu-id="e41a1-115">Karmaşık ve XML örnek belgelerinin doğrulamasında kullanılmak üzere kaydedilebilecek bir şemayı dinamik olarak düzenleyin.</span><span class="sxs-lookup"><span data-stu-id="e41a1-115">Dynamically edit a schema that can be complied and saved for use in the validation of XML instance documents.</span></span>  
  
## <a name="the-schema-object-model"></a><span data-ttu-id="e41a1-116">Şema nesne modeli</span><span class="sxs-lookup"><span data-stu-id="e41a1-116">The Schema Object Model</span></span>  
 <span data-ttu-id="e41a1-117">SOM, bir XML şemasındaki öğelere karşılık gelen <xref:System.Xml.Schema?displayProperty=nameWithType> ad alanındaki kapsamlı bir sınıf kümesinden oluşur.</span><span class="sxs-lookup"><span data-stu-id="e41a1-117">The SOM consists of an extensive set of classes in the <xref:System.Xml.Schema?displayProperty=nameWithType> namespace corresponding to the elements in an XML schema.</span></span> <span data-ttu-id="e41a1-118">Örneğin, `<xsd:schema>...</xsd:schema>` öğesi <xref:System.Xml.Schema.XmlSchema?displayProperty=nameWithType> sınıfıyla eşlenir ve bir `<xsd:schema/>` öğe içinde yer alan tüm bilgiler <xref:System.Xml.Schema.XmlSchema> sınıfı kullanılarak temsil edilebilir.</span><span class="sxs-lookup"><span data-stu-id="e41a1-118">For example, the `<xsd:schema>...</xsd:schema>` element maps to the <xref:System.Xml.Schema.XmlSchema?displayProperty=nameWithType> class, and all the information that can be contained within an `<xsd:schema/>` element can be represented using the <xref:System.Xml.Schema.XmlSchema> class.</span></span> <span data-ttu-id="e41a1-119">Benzer şekilde, `<xsd:element>...</xsd:element>` ve `<xsd:attribute>...</xsd:attribute>` öğeleri sırasıyla <xref:System.Xml.Schema.XmlSchemaElement?displayProperty=nameWithType> ve <xref:System.Xml.Schema.XmlSchemaAttribute?displayProperty=nameWithType> sınıflarıyla eşlenir.</span><span class="sxs-lookup"><span data-stu-id="e41a1-119">Similarly, the `<xsd:element>...</xsd:element>` and `<xsd:attribute>...</xsd:attribute>` elements map to the <xref:System.Xml.Schema.XmlSchemaElement?displayProperty=nameWithType> and <xref:System.Xml.Schema.XmlSchemaAttribute?displayProperty=nameWithType> classes respectively.</span></span> <span data-ttu-id="e41a1-120">Bu eşleme, izleyen diyagramda gösterilen <xref:System.Xml.Schema> ad alanında bir XML şeması nesne modeli oluşturan bir XML şemasının tüm öğeleri için devam eder.</span><span class="sxs-lookup"><span data-stu-id="e41a1-120">This mapping continues for all the elements of an XML schema creating an XML schema object model in the <xref:System.Xml.Schema> namespace illustrated in the diagram that follows.</span></span>
