# <a name="goto-labelname-is-not-valid-because-labelname-is-inside-a-try-catch-or-finally-statement-that-does-not-contain-this-statement"></a>' \<labelname> \<labelname> ', Bu ifadeyi içermeyen bir ' TRY ', ' catch ' veya ' finally ' deyimindeki içinde olduğundan ' goto ' geçerli değil

Bir bloğa dallandırılamıyor `Try...Catch...Finally` .  
  
 **Hata kimliği:** BC30754  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Etiketi bloğundan önce yeniden yapılandırmak için kodunuzu yeniden yapılandır `Try...Catch...Finally` .
