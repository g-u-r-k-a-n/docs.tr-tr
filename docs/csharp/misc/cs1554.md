# <a name="compiler-error-cs1554"></a><span data-ttu-id="9a09d-103">Derleyici hatası CS1554</span><span class="sxs-lookup"><span data-stu-id="9a09d-103">Compiler Error CS1554</span></span>

<span data-ttu-id="9a09d-104">Bildirim geçerli değil; ' \<type> operator op (... ' kullanın yerine</span><span class="sxs-lookup"><span data-stu-id="9a09d-104">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="9a09d-105">[Aşırı yüklenmiş bir işlecin](../language-reference/operators/operator-overloading.md) dönüş türü, anahtar sözcüğünden önce gelmelidir `operator` .</span><span class="sxs-lookup"><span data-stu-id="9a09d-105">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>
  
<span data-ttu-id="9a09d-106">Aşağıdaki örnek CS1554 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9a09d-106">The following sample generates CS1554:</span></span>  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
