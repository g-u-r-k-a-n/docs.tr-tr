 <span data-ttu-id="d3144-167">Düzen çok basit yöneliktir.</span><span class="sxs-lookup"><span data-stu-id="d3144-167">The pattern is very simple.</span></span> <span data-ttu-id="d3144-168">Yapmanız gereken tek şey <xref:System.Runtime.Serialization.ISerializable> arabirimini uygular ve nesne seri durumdan çıkarıldığınızda kullanılan özel bir oluşturucu sağlar.</span><span class="sxs-lookup"><span data-stu-id="d3144-168">All you need to do is implement the <xref:System.Runtime.Serialization.ISerializable> interface and provide a special constructor that is used when the object is deserialized.</span></span>  
  
 <span data-ttu-id="d3144-169">**✓ DO** korumalı serileştirme Oluşturucusu oluşturarak yazılan ve tam olarak burada aşağıdaki örnekte gösterildiği gibi adlı iki parametre sağlayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="d3144-169">**✓ DO** make the serialization constructor protected and provide two parameters typed and named exactly as shown in the sample here.</span></span>

```csharp
{  
    protected Person(SerializationInfo info, StreamingContext context)
    {  
        // ...  
    }

```
  
 <span data-ttu-id="d3144-170">**✓ DO** uygulamak <xref:System.Runtime.Serialization.ISerializable> üyeleri açıkça.</span><span class="sxs-lookup"><span data-stu-id="d3144-170">**✓ DO** implement the <xref:System.Runtime.Serialization.ISerializable> members explicitly.</span></span>  
  
 <span data-ttu-id="d3144-171">**✓ DO** bir bağlantı isteği uygulamak <xref:System.Runtime.Serialization.ISerializable.GetObjectData%2A?displayProperty=nameWithType> uygulaması.</span><span class="sxs-lookup"><span data-stu-id="d3144-171">**✓ DO** apply a link demand to <xref:System.Runtime.Serialization.ISerializable.GetObjectData%2A?displayProperty=nameWithType> implementation.</span></span> <span data-ttu-id="d3144-172">Bu, yalnızca tam olarak güvenilen çekirdek ve çalışma zamanı serileştirici üyeye erişimine sahip olmasını sağlar.</span><span class="sxs-lookup"><span data-stu-id="d3144-172">This ensures that only fully trusted core and the Runtime Serializer have access to the member.</span></span>
