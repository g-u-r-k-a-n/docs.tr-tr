# <a name="compiler-error-cs1715"></a><span data-ttu-id="12e06-102">Derleyici hatası CS1715</span><span class="sxs-lookup"><span data-stu-id="12e06-102">Compiler Error CS1715</span></span>

<span data-ttu-id="12e06-103">' Type1 ': geçersiz kılınan ' MemberName ' üyesini eşleştirmek için tür ' type2 ' olmalıdır</span><span class="sxs-lookup"><span data-stu-id="12e06-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="12e06-104">Bu hata, [Derleyici Hatası CS0508](./cs0508.md)ile aynıdır, ancak CS0508 artık yalnızca dönüş türlerine sahip yöntemler için geçerli olsa da, CS1715, yalnızca ' Return Types ' yerine yalnızca ' Types ' olan özellikler ve Dizin oluşturucular için geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="12e06-104">This error is the same as [Compiler Error CS0508](./cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="12e06-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="12e06-105">Example</span></span>  

 <span data-ttu-id="12e06-106">Aşağıdaki kod CS1715 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="12e06-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
