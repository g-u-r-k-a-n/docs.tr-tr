# <a name="hosting-wpf-applications"></a>WPF Uygulamaları Barındırma
WPF XAML tarayıcı uygulamaları (XBAP 'ler), bir Web sunucusuna dağıtılabilecek ve bir tarayıcıda başlatılan zengin istemci uygulamalardır. WPF konağı (PresentationHost. exe), XBAP ve XAML dosyaları için kabuk ve MIME işleyicisi olarak kaydedilir. Bu nedenle, Internet Explorer bir XBAP başlatıldığında WPF konağını başlatmayı bilir. Firefox kullanıcıları, Firefox 'u aynı zamanda barındırmanıza olanak tanıyan Firefox eklentilerini yükleyebilir. Bir XBAP, WPF tarafından sunulan yerel tarayıcı barındırma API 'Leri kullanılarak diğer tarayıcılarda veya tek başına uygulamalarda barındırılabilir.
