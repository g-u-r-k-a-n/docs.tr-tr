# <a name="labels-are-not-valid-in-the-immediate-window"></a>Etiketler komut penceresinde geçerli değildir

Çizgi etiketleri dallanmayı etkinleştirir ve hata ayıklama bağlamında buna izin verilmez.  
  
 **Hata kimliği:** BC30134  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- **Komut** penceresine bir satır etiketi girmeyin.
