# <a name="return-statement-in-a-function-get-or-operator-must-return-a-value"></a>Bir Işlevde, Get veya Işlecinde ' Return ' deyimleri bir değer döndürmelidir
`Return`çağırma yordamına bir değer döndürmek için deyimlerinin kullanılması gerekir. `Return`Deyimleri kendi program akışını denetlemek için kullanamazsınız.  
  
 **Hata kimliği:** BC30654  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. İşlevin veya yordamın döndüre, bir değer belirtin.  
  
2. `End`Programın geçerli yordamdan çıkmasına neden olacak şekilde ifadesini kullanın.
