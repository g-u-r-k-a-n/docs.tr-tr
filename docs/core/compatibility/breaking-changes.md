## <a name="github-issues-and-announcements"></a>GitHub sorunları ve duyurular

.NET Core'da tanıtılan son dakika değişikliklerini ayrıntılı olarak açıklayan tek tek sorunları aşağıdaki GitHub depolarında da görüntüleyebilirsiniz:

- .NET Core için [dotnet/docs](https://github.com/dotnet/docs/issues?q=is%3Aissue+label%3Abreaking-change) deposu.
- ASP.NET Core için [aspnet/Duyuru](https://github.com/aspnet/Announcements/issues?q=is%3Aissue+is%3Aopen+label%3A%22Breaking+change%22+label%3A3.0.0) deposu.
- Varlık Framework Core [için, dotnet/efcore](https://github.com/dotnet/efcore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change) deposu.
