# <a name="compiler-error-cs0653"></a><span data-ttu-id="ea34b-103">Derleyici hatası CS0653</span><span class="sxs-lookup"><span data-stu-id="ea34b-103">Compiler Error CS0653</span></span>
<span data-ttu-id="ea34b-104">Soyut olduğu için ' class ' öznitelik sınıfı uygulanamıyor</span><span class="sxs-lookup"><span data-stu-id="ea34b-104">Cannot apply attribute class 'class' because it is abstract</span></span>  
  
 <span data-ttu-id="ea34b-105">[Soyut](../language-reference/keywords/abstract.md) özel öznitelik sınıfı öznitelik olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="ea34b-105">An [abstract](../language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>  
  
 <span data-ttu-id="ea34b-106">Aşağıdaki örnek CS0653 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ea34b-106">The following sample generates CS0653:</span></span>  
  
```csharp  
// CS0653.cs

public abstract class MyAttribute : Attribute  
{  
}  
  
public class My2Attribute : MyAttribute  
{  
}  
  
[My]   // CS0653  
// try the following line instead  
// [My2]

{  
   public static void Main()  
   {  
   }  
}  
```
