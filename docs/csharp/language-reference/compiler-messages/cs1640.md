# <a name="compiler-error-cs1640"></a><span data-ttu-id="74f27-103">Derleyici Hatası CS1640</span><span class="sxs-lookup"><span data-stu-id="74f27-103">Compiler Error CS1640</span></span>
<span data-ttu-id="74f27-104">foreach ifadesi, ' Interface ' öğesinin birden çok örneğini uyguladığından ' Type ' türündeki değişkenlerde çalışamaz, belirli bir arabirim örneklemesine atamayı deneyin</span><span class="sxs-lookup"><span data-stu-id="74f27-104">foreach statement cannot operate on variables of type 'type' because it implements multiple instantiations of 'interface', try casting to a specific interface instantiation</span></span>  
  
 <span data-ttu-id="74f27-105">Tür, IEnumerator öğesinin iki veya daha fazla değerinden kalıtımla alır, bu, kullanılabilecek \<T> türde benzersiz bir sabit listesi olmadığı anlamına gelir `foreach` .</span><span class="sxs-lookup"><span data-stu-id="74f27-105">The type inherits from two or more instances of IEnumerator\<T>, which means there is not a unique enumeration of the type that `foreach` could use.</span></span> <span data-ttu-id="74f27-106">IEnumerator türünü belirtin \<T> veya başka bir döngü yapısı kullanın.</span><span class="sxs-lookup"><span data-stu-id="74f27-106">Specify the type of IEnumerator\<T> or use another looping construct.</span></span>  
  
## <a name="example"></a><span data-ttu-id="74f27-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="74f27-107">Example</span></span>  
 <span data-ttu-id="74f27-108">Aşağıdaki örnek CS1640 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="74f27-108">The following sample generates CS1640:</span></span>  
  
```csharp  
// CS1640.cs  
  
using System;  
using System.Collections;  
using System.Collections.Generic;  
  
public class C : IEnumerable, IEnumerable<int>, IEnumerable<string>  
{  
    IEnumerator<int> IEnumerable<int>.GetEnumerator()  
    {  
        yield break;  
    }  
  
    IEnumerator<string> IEnumerable<string>.GetEnumerator()  
    {  
        yield break;  
    }  
  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        return (IEnumerator)((IEnumerable<string>)this).GetEnumerator();  
    }  
}  
  
public class Test  
{  
    public static int Main()  
    {  
        foreach (int i in new C()){}    // CS1640  
  
        // Try specifying the type of IEnumerable<T>  
        // foreach (int i in (IEnumerable<int>)new C()){}  
        return 1;  
    }  
}  
```
