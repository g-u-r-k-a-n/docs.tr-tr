# <a name="compiler-error-cs1681"></a>Derleyici hatası CS1681

Genel extern diğer adını yeniden tanımlayamazsınız  
  
 Genel diğer ad, diğer ad dışı başvuruların dahil olduğu şekilde zaten tanımlanmış ve bu nedenle yeniden tanımlanamaz.  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS1681 oluşturur.  
  
```csharp  
// CS1681.cs  
// compile with: /reference:global=System.dll  
// CS1681 expected  
  
// try this instead: /reference:System.dll  
class A  
{  
   static void Main() {}  
}  
```
