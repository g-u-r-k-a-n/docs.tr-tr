# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>' \<assemblyname> ' bir derlemedir ve modül olarak başvurulamaz

Bir derlemeyi modül olarak kullanma girişiminde bulunuldu.  
  
 **Hata kimliği:** BC31077  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir modülün başvurusunu değiştirin.
