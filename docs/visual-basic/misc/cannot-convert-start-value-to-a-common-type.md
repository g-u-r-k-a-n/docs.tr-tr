# <a name="cannot-convert-start-value-of-type-type1-limit-value-of-type2-and-step-value-of-type3-to-a-common-type"></a>' ' Türündeki başlangıç değeri, ' ' \<type1> değerini \<type2> ve ' ' adım değerini \<type3> ortak bir tür olarak dönüştüremiyor

Belirtilen `For...Next` Denetim değerlerinde farklı türler var ve bu nedenle ortak bir türe dönüştürülemediğinden geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bir veya daha fazla değerin türünü, tümünün ortak bir türe dönüştürülebilmesi için değiştirin.
