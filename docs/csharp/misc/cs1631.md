# <a name="compiler-error-cs1631"></a><span data-ttu-id="97940-102">Derleyici Hatası CS1631</span><span class="sxs-lookup"><span data-stu-id="97940-102">Compiler Error CS1631</span></span>
<span data-ttu-id="97940-103">Yakalama yan tümcesinin gövdesinde bir değer verilemez</span><span class="sxs-lookup"><span data-stu-id="97940-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="97940-104">Verim deyimi bir yakalama maddesi gövdesi içinden izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="97940-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="97940-105">Bu hatayı önlemek için, verim deyimini yakalama yan tümcesinin gövdesinin dışına taşıyın.</span><span class="sxs-lookup"><span data-stu-id="97940-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="97940-106">Aşağıdaki örnek CS1631 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="97940-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
