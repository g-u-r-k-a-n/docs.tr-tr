# <a name="procedurename-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>' \<procedurename> ' desteklenmeyen bir dönüş türü veya desteklenmeyen parametre türleri içeriyor
Yordam, parametrelerle veya Visual Studio tarafından gösterilemeyen bir dönüş türüyle bildirilmiştir.  
  
 **Hata kimliği:** BC30657  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dönüş türlerini veya parametreleri desteklenen bir tür olarak tanımlayın.  
  
- Bu yordam tarafından kullanılan verileri açıklayan yeni bir tür geliştirin.
