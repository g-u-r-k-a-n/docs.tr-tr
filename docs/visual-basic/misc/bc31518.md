# <a name="assembly-assemblyname-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>\<assemblyname>Yolu 259 karakterden uzun olduğundan ' ' derlemesi oluşturulamıyor

Bu derlemenin yolu 259 karakterlik en fazla uzunluğu aşıyor.  
  
 **Hata kimliği:** BC31518  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Daha kısa adlar kullanarak veya bir düzdür dosya hiyerarşisi seçerek daha kısa bir yol oluşturun.
