<span data-ttu-id="2bdda-104">' özellik geçersiz kılma ': ' Özellik dışı ' bir özellik olmadığından geçersiz kılınamaz</span><span class="sxs-lookup"><span data-stu-id="2bdda-104">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="2bdda-105">Özellik olmayan bir veri türünü [özellik](../programming-guide/classes-and-structs/properties.md)olarak geçersiz kılma girişiminde bulunuldu; buna izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="2bdda-105">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="2bdda-106">Aşağıdaki örnek CS0544 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2bdda-106">The following sample generates CS0544:</span></span>  
  
```csharp

// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
