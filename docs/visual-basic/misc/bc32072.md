# <a name="cannot-implement-interface-interfacename1-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-interfacename2-for-some-type-arguments"></a><span data-ttu-id="62fbd-103">\<interfacename1>Uygulaması \<interfacename2> bazı tür bağımsız değişkenleri için uygulanan başka bir ' ' arabiriminin uygulamasıyla çakışabileceğinden ' ' arabirimi uygulanamıyor</span><span class="sxs-lookup"><span data-stu-id="62fbd-103">Cannot implement interface '\<interfacename1>' because its implementation could conflict with the implementation of another implemented interface '\<interfacename2>' for some type arguments</span></span>

<span data-ttu-id="62fbd-104">Bir sınıf bildirimi `Implements` iki veya daha fazla arabirimi belirten bir ifade içerir, ancak en az bir arabirimler geneldir ve uygulamalardan ikisi tür bağımsız değişkenlerinin belirli değerleri için çakışabilir.</span><span class="sxs-lookup"><span data-stu-id="62fbd-104">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>

<span data-ttu-id="62fbd-105">Aşağıdaki deyimler bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="62fbd-105">The following statements can generate this error.</span></span>

```vb
Public Interface iFace1
    Sub testSub(ByVal arg As String)
End Interface
Public Interface iFace2(Of t)
    Sub testSub(ByVal arg As t)
End Interface
Public Class testClass
    Implements iFace1, iFace2(Of String)
End Class
```

<span data-ttu-id="62fbd-106">Kullanılarak oluşturulduğu için `iFace2` `String` , `testClass` aynı imzalara sahip iki sürümünün uygulanması gerekir `testSub` .</span><span class="sxs-lookup"><span data-stu-id="62fbd-106">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="62fbd-107">Bunun yapılması, hangi sürümün erişecağına bir belirsizlik üretir.</span><span class="sxs-lookup"><span data-stu-id="62fbd-107">Doing so would produce an ambiguity about which version to access.</span></span>

<span data-ttu-id="62fbd-108">**Hata kimliği:** BC32072</span><span class="sxs-lookup"><span data-stu-id="62fbd-108">**Error ID:** BC32072</span></span>
