## <a name="example"></a><span data-ttu-id="5a3ca-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="5a3ca-108">Example</span></span>

 <span data-ttu-id="5a3ca-109">Aşağıdaki örnek CS0122 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5a3ca-109">The following sample generates CS0122:</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
