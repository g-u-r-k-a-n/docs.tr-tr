## <a name="example"></a><span data-ttu-id="c23ea-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="c23ea-106">Example</span></span>  

 <span data-ttu-id="c23ea-107">Aşağıdaki örnek CS1686 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c23ea-107">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs

class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
