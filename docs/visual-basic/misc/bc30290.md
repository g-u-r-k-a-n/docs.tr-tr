# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>Yerel değişken, kendisini içeren işlevle aynı ada sahip olamaz

Bir `Dim` ifade, kendisini içeren yordamla aynı ada sahip bir değişken belirtir `Function` .  
  
 **Hata kimliği:** BC30290  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değişken bildirimini kaldırın veya değişkenin adını değiştirin.
