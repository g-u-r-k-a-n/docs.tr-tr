 <span data-ttu-id="17708-104">Bu hata, bir atlama deyimi `goto`**(break**, **continue,** vb.) denetimi anonim bir yöntem bloğundan taşımaya çalışırsa oluşur.</span><span class="sxs-lookup"><span data-stu-id="17708-104">This error occurs if a jump statement (**break**, `goto`, **continue**, etc.) attempts to move control out of an anonymous method block.</span></span> <span data-ttu-id="17708-105">Anonim bir yöntem bloğu bir işlev gövdesidir ve yalnızca bir iade deyimi yle veya bloğun sonuna ulaşarak çıkarılabilir.</span><span class="sxs-lookup"><span data-stu-id="17708-105">An anonymous method block is a function body and can only be exited by a return statement or by reaching the end of the block.</span></span>

      }  
   }
