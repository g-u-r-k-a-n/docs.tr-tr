# <a name="compiler-error-cs1021"></a><span data-ttu-id="b9f72-102">Derleyici Hatası CS1021</span><span class="sxs-lookup"><span data-stu-id="b9f72-102">Compiler Error CS1021</span></span>

<span data-ttu-id="b9f72-103">İntegral sabiti çok büyük</span><span class="sxs-lookup"><span data-stu-id="b9f72-103">Integral constant is too large</span></span>  
  
<span data-ttu-id="b9f72-104">Tamsayı ile temsil edilen bir değer, <xref:System.UInt64.MaxValue?displayProperty=nameWithType>'den büyüktür.</span><span class="sxs-lookup"><span data-stu-id="b9f72-104">A value represented by an integer literal is greater than <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span></span>  
  
<span data-ttu-id="b9f72-105">Aşağıdaki örnek CS1021 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b9f72-105">The following sample generates CS1021:</span></span>  

```csharp
// CS1021.cs  
class Program
{
    static void Main(string[] args)
    {
        int a = 18_446_744_073_709_552_000;
    }
}  
```

<span data-ttu-id="b9f72-106">Aşağıdaki kod da CS1021 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b9f72-106">The following code also generates CS1021:</span></span>

```csharp

class Program
{
    static void Main(string[] args)
    {
        var a = new BigInteger(18_446_744_073_709_552_000);
    }
}
```
