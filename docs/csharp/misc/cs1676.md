# <a name="compiler-error-cs1676"></a><span data-ttu-id="5cd17-102">Derleyici Hatası CS1676</span><span class="sxs-lookup"><span data-stu-id="5cd17-102">Compiler Error CS1676</span></span>
<span data-ttu-id="5cd17-103">Parametre 'sayı' 'anahtar kelime' anahtar kelimesi ile bildirilmelidir</span><span class="sxs-lookup"><span data-stu-id="5cd17-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="5cd17-104">Bu hata, anonim bir yöntemdeki parametre türü değiştirici, yöntemi döküm yaptığınız temsilcinin bildiriminde kullanılandan farklı olduğunda oluşur.</span><span class="sxs-lookup"><span data-stu-id="5cd17-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="5cd17-105">Aşağıdaki örnek CS1676 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5cd17-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
