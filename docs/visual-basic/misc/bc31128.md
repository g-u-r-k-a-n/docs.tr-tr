# <a name="removehandler-is-already-declared"></a>' RemoveHandler ' zaten tanımlanmış

`RemoveHandler`Özel bir olay bildiriminde birden fazla bildirim görüntülendi. Bir `RemoveHandler` bildirim, bir olay işleyicisini kaldırmak için kullanılan yordamı bildirir.  
  
 **Hata kimliği:** BC31128  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ifadesini kaldırın `RemoveHandler` .
