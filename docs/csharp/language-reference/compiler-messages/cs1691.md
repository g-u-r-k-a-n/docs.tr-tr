# <a name="compiler-warning-level-1-cs1691"></a>Derleyici Uyarısı (düzey 1) CS1691
' number ' geçerli bir uyarı numarası değil  
  
 [#Pragma uyarı](../preprocessor-directives/preprocessor-pragma-warning.md) Önişlemci yönergesine geçirilen bir sayı geçerli bir uyarı numarası değil. Sayının bir hatayı veya başka bir karakter dizisini temsil ettiğini doğrulayın.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1691 oluşturur.  
  
```csharp  
// CS1691.cs  
public class C  
{  
    int i = 1;  
    public static void Main()  
    {  
        C myC = new C();  
#pragma warning disable 151  // CS1691  
// Try the following line instead:  
// #pragma warning disable 1645
        myC.i++;  
#pragma warning restore 151  // CS1691  
// Try the following line instead:  
// #pragma warning restore 1645
    }  
}  
```
