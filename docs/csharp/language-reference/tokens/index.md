# <a name="c-special-characters"></a>C# Özel Karakterleri

Özel karakterler, program öğesini (gerçek bir dize, tanımlayıcı veya öznitelik adı) değiştirerek önceden tanımlanmış, bağlamsal karakterlerdir. C# aşağıdaki özel karakterleri destekler:

- [@](./verbatim.md), harfi harfini tanımlayıcı karakter.

- [$](./interpolated.md), enterpolasyonlu dize karakteri.
