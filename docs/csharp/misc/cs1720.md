# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="f1f4a-102">Derleyici Uyarısı (düzey 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="f1f4a-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="f1f4a-103">'Genel tür' varsayılan değeri null olduğundan ifade her zaman bir System.NullReferenceException neden olur</span><span class="sxs-lookup"><span data-stu-id="f1f4a-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="f1f4a-104">Başvuru türü (örneğin, bir sınıf) olan genel bir tür değişkeninin varsayılanını içeren bir ifade yazarsanız, bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="f1f4a-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="f1f4a-105">Aşağıdaki ifadeyi göz önünde bulundurun:</span><span class="sxs-lookup"><span data-stu-id="f1f4a-105">Consider the following expression:</span></span>

```csharp  
// CS1720.cs  
using System;  
public class Tester
{  
    public static void GenericClass<T>(T t1) where T : class
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
