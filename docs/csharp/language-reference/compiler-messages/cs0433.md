## <a name="example"></a><span data-ttu-id="97cc7-114">Örnek</span><span class="sxs-lookup"><span data-stu-id="97cc7-114">Example</span></span>  
 <span data-ttu-id="97cc7-115">Aşağıdaki örnek, CS0433 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="97cc7-115">The following example generates CS0433.</span></span>

```csharp  
public class Test
{  
   public static void Main()

      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433

```  
  
## <a name="example"></a><span data-ttu-id="97cc7-116">Örnek</span><span class="sxs-lookup"><span data-stu-id="97cc7-116">Example</span></span>
