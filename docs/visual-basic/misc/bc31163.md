# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>XML öznitelik değeri için eşleşen kapatma tek tırnak işareti bekleniyordu
Tek tırnak işaretleriyle ayrılmış bir öznitelik değerinde kapanış tırnak işareti karakteri eksik.  
  
 **Hata kimliği:** BC31163  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öznitelik değerini sınırlandırmak için, kapanış tek tırnak işareti karakterini (') ekleyin.
