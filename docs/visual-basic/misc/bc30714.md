# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>' Özgeçmişi ' deyimleri Komut penceresinde geçerli değildir

Özgeçmişi deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30714  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Resume`Hata ayıklama kodundan deyimleri kaldır.
