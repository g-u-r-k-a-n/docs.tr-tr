# <a name="field-fieldname-is-of-an-unsupported-type"></a>' \<fieldname> ' Alanı desteklenmeyen bir türde
Bir alan, Visual Studio tarafından gösterilemeyen bir tür olarak bildirilmiştir.  
  
 **Hata kimliği:** BC30656  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Alanı desteklenen bir tür olarak tanımlayın.  
  
2. Alandaki verileri açıklayan yeni bir tür geliştirin.
