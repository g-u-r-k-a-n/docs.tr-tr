# <a name="icordebugthreadgetprocess-method"></a>ICorDebugThread::GetProcess Yöntemi

Bu ICorDebugThread 'in bir parçasını oluşturan işlem için bir arabirim işaretçisi alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
HRESULT GetProcess (  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
## <a name="parameters"></a>Parametreler  

 `ppProcess`  
 dışı İşlemi temsil eden ICorDebugProcess arabirimi nesnesinin adresine yönelik bir işaretçi.
