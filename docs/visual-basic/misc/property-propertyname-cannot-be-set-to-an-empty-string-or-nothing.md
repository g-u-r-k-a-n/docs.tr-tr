# <a name="property-propertyname-cannot-be-set-to-an-empty-string-or-nothing"></a>Özellik \<propertyname> boş bir dizeye veya Nothing olarak ayarlanamaz

Özelliği, boş bir dize ("") veya ya da dışında bir değer gerektirir `Nothing` . Bu, değerin yanlış hesaplanmasının bir sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli bir değer sağlayın.
