# <a name="functions-entity-sql"></a>İşlevler (Entity SQL)

Entity SQL, Kullanıcı tanımlı işlevleri, kurallı işlevleri ve sağlayıcıya özgü işlevleri destekler. Kullanıcı tanımlı işlevler, kavramsal modelde veya sorgudaki satır içi olarak belirtilir. Daha fazla bilgi için bkz. [Kullanıcı tanımlı işlevler](user-defined-functions-entity-sql.md).  
  
 Kurallı işlevler Entity Framework önceden tanımlanmıştır ve veri sağlayıcıları tarafından desteklenmelidir. Bir Kullanıcı bir sağlayıcı tarafından desteklenmeyen bir işlevi çağırırsa Entity SQL komutları başarısız olur. Bu nedenle, kurallı işlevler genellikle sağlayıcıya özgü bir ad alanında bulunan mağazaya özgü işlevlerde önerilir. Daha fazla bilgi için bkz. [kurallı işlevler](canonical-functions.md).  
  
 Microsoft SQL Istemci tarafından yönetilen sağlayıcı, sağlayıcıya özgü işlevler kümesi sağlar. Daha fazla bilgi için bkz. [Entity Framework işlevleri Için SqlClient](../sqlclient-for-ef-functions.md).
