# <a name="compiler-error-cs0828"></a><span data-ttu-id="69531-103">Derleyici hatası CS0828</span><span class="sxs-lookup"><span data-stu-id="69531-103">Compiler Error CS0828</span></span>

 <span data-ttu-id="69531-105">Anonim bir tür, null bir değer veya güvenli olmayan bir tür ya da bir yöntem grubu ya da anonim işlev ile başlatılamaz.</span><span class="sxs-lookup"><span data-stu-id="69531-105">An anonymous type cannot be initialized with a null value or an unsafe type, or a method group or anonymous function.</span></span>

1. <span data-ttu-id="69531-107">Atamanın sol tarafına bir tür bildirimi ekleyin ya da sağ taraftaki ifadeyi, kabul edilebilir bir türe sahip olacak şekilde değiştirin.</span><span class="sxs-lookup"><span data-stu-id="69531-107">Either add a type declaration to the left side of the assignment, or change the expression on the right side so that it has an acceptable type.</span></span>
