# <a name="imetadataerror-interface"></a>IMetaDataError Arabirimi

> [!NOTE]
> `IMetaDataError`Arabirimin istemci tarafından uygulanması gerekir.  
  
## <a name="methods"></a>Yöntemler

|[OnError Yöntemi](imetadataerror-onerror-method.md)|Meta veri birleştirme sırasında oluşan hataların bildirimini sağlar.|
