 <span data-ttu-id="5a60a-104">[Void](../language-reference/builtin-types/void.md) dönüş türüne sahip bir yöntem değer döndüremez.</span><span class="sxs-lookup"><span data-stu-id="5a60a-104">A method with a [void](../language-reference/builtin-types/void.md) return type cannot return a value.</span></span> <span data-ttu-id="5a60a-105">Daha fazla bilgi için bkz. [Yöntemler](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="5a60a-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="5a60a-106">Aşağıdaki örnek CS0127 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5a60a-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
