# <a name="other-control-structures-visual-basic"></a>Diğer Denetim Yapıları (Visual Basic)
Visual Basic, bir kaynağı atmayı veya bir nesne başvurusunu tekrarlamanız için gereken süreyi azaltmanızı sağlayan denetim yapıları sağlar.  
  
## <a name="usingend-using-construction"></a>Kullanılıyor... Oluşturma kullanarak bitir  
 `Using...End Using` oluşturma, SQL bağlantısı gibi bir kaynağı kullandığınız bir bildirim bloğu oluşturur. İsteğe bağlı olarak `Using` ifadesiyle kaynağı elde edebilirsiniz. `Using` bloğundan çıktığınızda, diğer kodun kullanabilmesi için kaynağın otomatik olarak ortadan kaldırmasına Visual Basic. Kaynak yerel ve atılabilir olmalıdır. Daha fazla bilgi için bkz. [using deyimleri](../../../../visual-basic/language-reference/statements/using-statement.md).  
  
## <a name="withend-with-construction"></a>İle... Yapım Ile bitir  
 `With...End With` oluşturma bir kez nesne başvurusu belirtmenizi ve sonra üyelerine erişen bir dizi deyim çalıştırmanızı sağlar. Bu, kodunuzun basitleşmesini ve performansını iyileştirebildiğinden, Visual Basic kendisine erişen her bir deyimin başvurusunu yeniden oluşturmak zorunda değildir. Daha fazla bilgi için bkz [.... WITH Ifadesiyle biter](../../../../visual-basic/language-reference/statements/with-end-with-statement.md).
