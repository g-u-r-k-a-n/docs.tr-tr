# <a name="cannot-save-file-to-temp"></a>Dosya TEMP 'e kaydedilemiyor

Bir bileşen TEMP adlı bir dizin bulamıyor veya TEMP dizinini içeren sürücü veya bölümün bilgileri kaydetmek için yeterli alanı yok.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. "TEMP" adlı bir dizin oluşturun ve TEMP ortam değişkenini yoluna eşit olarak ayarlayın.  
  
2. Gereksiz dosyaları silerek sürücüde yer açın veya başka bir bölümde GEÇICI bir dizin oluşturun ve TEMP ortam değişkenini yoluna eşit olarak ayarlayın.
