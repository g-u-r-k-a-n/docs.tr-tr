# <a name="systemservicemodelchannelshttpchannelresponseaborted"></a>System.ServiceModel.Channels.HttpChannelResponseAborted

## <a name="description"></a>Açıklama  
 Bu uyarı düzeyi izleme bir HTTP kanal yanıtının durdurulduğunu belirtir. Bu izlemede ek bilgiler, nedeni veya Abort yöntemine geçirilen iletiyi açığa çıkarır.
