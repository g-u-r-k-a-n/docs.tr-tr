<span data-ttu-id="f670f-103">Genel tür, bir tür parametresinde çakışan kısıtlamalarla bildirilmiştir.</span><span class="sxs-lookup"><span data-stu-id="f670f-103">A generic type is declared with conflicting constraints on a type parameter.</span></span>  
  
 <span data-ttu-id="f670f-104">Aşağıdaki ifade bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="f670f-104">The following statement can generate this error.</span></span>  
  
 `Public Class testClass(Of t As {Structure, Class })`  
  
 <span data-ttu-id="f670f-105">Kısıtlama, `Structure` `Class` `t` `Structure` karşılık gelen tür bağımsız değişkeninin bir değer türü olmasını gerektirdiğinden ve `Class` bunun bir başvuru türü olması gerektiğinden tür parametresi için bir çakışmaya neden olur.</span><span class="sxs-lookup"><span data-stu-id="f670f-105">The constraints `Structure` and `Class` cause a conflict for type parameter `t`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
