## <a name="declaring-a-message"></a><span data-ttu-id="0eeb4-106">İleti bildirme</span><span class="sxs-lookup"><span data-stu-id="0eeb4-106">Declaring a message</span></span>

<span data-ttu-id="0eeb4-107">WCF 'de, hisse senedi Pazar ticareti uygulaması için bir `Stock` sınıfı aşağıdaki örnekte olduğu gibi tanımlanabilir:</span><span class="sxs-lookup"><span data-stu-id="0eeb4-107">In WCF, a `Stock` class for a stock market trading application might be defined like the following example:</span></span>

```csharp
    [DataContract]
    public class Stock
    {
        [DataMember]
        public int Id { get; set;}
        [DataMember]
        public string Symbol { get; set;}
        [DataMember]
        public string DisplayName { get; set;}
        [DataMember]
        public int MarketId { get; set; }
    }
}
```

<span data-ttu-id="0eeb4-108">Protoarabelleğe denk sınıfı uygulamak için, `.proto` dosyasında bildirilmelidir.</span><span class="sxs-lookup"><span data-stu-id="0eeb4-108">To implement the equivalent class in Protobuf, it must be declared in the `.proto` file.</span></span> <span data-ttu-id="0eeb4-109">`protoc` derleyici daha sonra yapı sürecinin bir parçası olarak .NET sınıfını oluşturur.</span><span class="sxs-lookup"><span data-stu-id="0eeb4-109">The `protoc` compiler will then generate the .NET class as part of the build process.</span></span>

```protobuf

    int32 id = 1;
    string symbol = 2;
    string display_name = 3;
    int32 market_id = 4;

}  
```

<span data-ttu-id="0eeb4-110">İlk satır, kullanılmakta olan sözdizimi sürümünü bildirir.</span><span class="sxs-lookup"><span data-stu-id="0eeb4-110">The first line declares the syntax version being used.</span></span> <span data-ttu-id="0eeb4-111">Dilin sürüm 3 2016 ' de yayımlanmıştır ve gRPC Hizmetleri için önerilen sürümdür.</span><span class="sxs-lookup"><span data-stu-id="0eeb4-111">Version 3 of the language was released in 2016 and is the recommended version for gRPC services.</span></span>

<span data-ttu-id="0eeb4-112">`option csharp_namespace` satırı, oluşturulan C# türler için kullanılacak ad alanını belirtir.</span><span class="sxs-lookup"><span data-stu-id="0eeb4-112">The `option csharp_namespace` line specifies the namespace to be used for the generated C# types.</span></span> <span data-ttu-id="0eeb4-113">`.proto` dosyası diğer diller için derlendiğinde Bu seçenek yok sayılır.</span><span class="sxs-lookup"><span data-stu-id="0eeb4-113">This option will be ignored when the `.proto` file is compiled for other languages.</span></span> <span data-ttu-id="0eeb4-114">Prototip dosyalarının birkaç dilde dile özgü seçenekleri içermesi yaygındır.</span><span class="sxs-lookup"><span data-stu-id="0eeb4-114">It is common for Protobuf files to contain language-specific options for several languages.</span></span>

<span data-ttu-id="0eeb4-115">`Stock` ileti tanımı, her biri türü, adı ve alan numarası olan dört alanı belirtir.</span><span class="sxs-lookup"><span data-stu-id="0eeb4-115">The `Stock` message definition specifies four fields, each with a type, a name, and a field number.</span></span>
