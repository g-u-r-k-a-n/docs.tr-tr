## <a name="example"></a><span data-ttu-id="42d4e-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="42d4e-107">Example</span></span>  

 <span data-ttu-id="42d4e-108">Aşağıdaki örnek CS0406 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="42d4e-108">The following sample generates CS0406.</span></span>
