# <a name="compiler-error-cs0014"></a>Derleyici hatası CS0014
Gerekli ' dosya ' dosyası bulunamadı  
  
 Derleyici için bir dosya gerekiyor ancak sistemde yok. Yolun doğru olduğundan emin olun. Dosya bir Visual Studio sistem dosyası ise, yüklemenizi onarmanız veya Visual Studio 'Yu tamamen kaldırmanız ve yeniden yüklemeniz gerekebilir.
