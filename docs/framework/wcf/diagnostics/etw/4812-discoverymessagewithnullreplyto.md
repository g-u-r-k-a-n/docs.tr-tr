# <a name="4812---discoverymessagewithnullreplyto"></a>4812 - DiscoveryMessageWithNullReplyTo

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|4812|  
|Anahtar sözcükler|Bulma|  
|Düzey|Uyarı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Açıklama  

 Bu olay, bulma iletisi bir ReplyTo Address içermediğinden DiscoveryClient tarafından bırakıldığında yayınlanır.  
  
## <a name="message"></a>İleti  

 MessageID = ' %1 ' olan bir bulma isteği iletisi, ReplyTo adresi olmadığından bırakıldı.
