# <a name="cast-from-type-typename1-to-type-typename2-is-not-valid"></a>' \<typename1> ' Türünden ' ' türüne dönüştürme \<typename2> geçerli değil
Bir değişkeni bir türden diğerine dönüştürme girişimi geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dönüştürmede yer alan türleri denetleyin ve anlamlı bir türe dönüştürmeye çalışırken emin olun.
