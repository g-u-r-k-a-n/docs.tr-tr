# <a name="new-is-not-valid-in-this-context"></a>' New ' Bu bağlamda geçerli değil

`New`Anahtar sözcüğü bir tür adı gerekli olduğunda gerçekleşir. Bağlam, ile birleştirilemeyen bir değer türü için çağırır `New` .  
  
 **Hata kimliği:** BC30200  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `New`Anahtar sözcüğünü kaldırın.
