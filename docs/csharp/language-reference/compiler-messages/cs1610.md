# <a name="compiler-warning-level-4-cs1610"></a>Derleyici Uyarısı (düzey 4) CS1610
Varsayılan Win32 kaynağı için kullanılan ' File ' geçici dosyası silinemiyor--kaynak  
  
 [-Win32res](../compiler-options/win32res-compiler-option.md) derleyici seçeneği kullanılırken ve **% Temp%** dizininiz silme iznine sahip olmadığında, bu uyarı derleyicinin oluşturduğu geçici bir dosyayı silemediğini gösterir.  
  
 **% Temp%** dizini için okuma/yazma/silme izinlerine sahip olduğunuzdan emin olun.  
  
 Gerekirse, bu dosyaları el ile silebilirsiniz ve programlarınızdan C# veya herhangi birine zarar vermez.
