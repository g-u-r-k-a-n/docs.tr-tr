<span data-ttu-id="7b839-105">Bu hata, bir çağırma ifadesinde oluşan bir tür bağımsız değişkenine bir öznitelik uyguladığınızda oluşur.</span><span class="sxs-lookup"><span data-stu-id="7b839-105">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="7b839-106">Aşağıdaki gibi bir sınıf veya yöntem bildirimi deyimindeki bir tür parametresine bir öznitelik uygulamak kabul edilebilir:</span><span class="sxs-lookup"><span data-stu-id="7b839-106">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

        void Meth<B>();
