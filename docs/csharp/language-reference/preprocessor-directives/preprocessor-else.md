# <a name="else-c-reference"></a>#else (C# Başvurusu)
`#else`, yukarıdaki [#if](./preprocessor-if.md) veya (isteğe bağlı) [#elif](./preprocessor-elif.md) yönergelerinden hiçbiri `true`değerlendiriyorsa derleyici, `#else` ve sonraki `#endif`arasındaki tüm kodları değerlendirmek için bileşik bir koşullu yönerge oluşturmanıza olanak sağlar.  
  
## <a name="remarks"></a>Açıklamalar  
 `#else`sonrasında [#endif](./preprocessor-endif.md) sonraki Önişlemci yönergesi olmalıdır. `#else`nasıl kullanılacağına ilişkin bir örnek için bkz. [#if](./preprocessor-if.md) .
