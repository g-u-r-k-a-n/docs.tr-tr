# <a name="exit-statements-are-not-valid-in-the-immediate-window"></a>' Exit ' deyimleri Komut penceresinde geçerli değildir
`Exit` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30713  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Hata ayıklama kodundan `Exit` deyimlerini kaldırın.
