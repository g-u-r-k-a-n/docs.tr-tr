# <a name="compiler-warning-level-1-cs2029"></a>Derleyici Uyarısı (düzey 1) CS2029

'/Define ' için geçersiz değer; ' Identifier ' geçerli bir tanımlayıcı değil  
  
 [**Definesabitleri**](../language-reference/compiler-options/language.md#defineconstants) seçeneğinde kullanılan değer bazı geçersiz karakterler içeriyorsa bu uyarı oluşur.  
  
 Bu uyarı, [**Disableduyarılar**](../language-reference/compiler-options/errors-warnings.md#disabledwarnings) seçeneği tarafından bastırılamaz.
