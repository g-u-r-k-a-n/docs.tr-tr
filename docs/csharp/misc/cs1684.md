 <span data-ttu-id="427a5-104">Bu hata, bir ad alanının içindeki bir başvurunun ikinci bir ad alanı içinde var olduğunu belirten bir başvuruya neden olabilir, ancak tür yok.</span><span class="sxs-lookup"><span data-stu-id="427a5-104">This error can be caused by a reference inside one namespace referring to a type that it says exists inside a second namespace, but the type does not exist.</span></span> <span data-ttu-id="427a5-105">Örneğin mydll.dll `A` , türün yourdll.dll içinde var olduğunu, ancak yourdll.dll içinde böyle bir tür bulunmadığını söyler.</span><span class="sxs-lookup"><span data-stu-id="427a5-105">For example, mydll.dll says that type `A` exists inside yourdll.dll, but no such type exists inside yourdll.dll.</span></span> <span data-ttu-id="427a5-106">Bu hatanın olası nedenlerinden biri, kullanmakta olduğunuz yourdll.dll sürümünün çok eski ve `A` henüz tanımlanmamasına neden olur.</span><span class="sxs-lookup"><span data-stu-id="427a5-106">One possible cause of this error is that the version of yourdll.dll you are using is too old and `A` has not yet been defined.</span></span>  
  
 <span data-ttu-id="427a5-107">Aşağıdaki örnek CS1684 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="427a5-107">The following sample generates CS1684.</span></span>  
  
## <a name="example"></a><span data-ttu-id="427a5-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="427a5-108">Example</span></span>  
  
```csharp

// compile with: /target:library /keyfile:CS1684.key  
public class A {  
   public void Test() {}  
}  
  
public class C2 {}  
```  
  
## <a name="example"></a><span data-ttu-id="427a5-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="427a5-109">Example</span></span>  
  
```csharp

   public static A GetA() { return new A(); }  
   public static C2 GetC() { return new C2(); }  
}  
```
