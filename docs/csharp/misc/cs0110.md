# <a name="compiler-error-cs0110"></a>Derleyici hatası CS0110
' Const declaration ' için sabit değerin değerlendirilmesi döngüsel bir tanım içeriyor  
  
 Bir [const](../language-reference/keywords/const.md) değişkeninin bildirimi (), `a` `b` aynı zamanda () başvurduğu başka bir const değişkenine () başvuramaz `a` .  
  
 Aşağıdaki örnek CS0110 oluşturur:  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
