# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>' End AddHandler ' satırdaki ilk ifade olmalıdır
Bir `End AddHandler` ifade bir iki nokta üst üste (:) Ekstre ayırıcı. `End AddHandler`kaynak satırındaki tek bir ifade olmalıdır.  
  
 **Hata kimliği:** BC31118  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Birden çok deyimi farklı satırlara ayırın.
