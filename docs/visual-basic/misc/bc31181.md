# <a name="attribute-value-is-not-valid-expecting-value"></a>Öznitelik değeri geçerli değil; ' Value ' bekleniyor

XML belgesi sabit değerinde bir öznitelik için geçersiz bir değer belirtildi. XML belgesi değişmez değeri için geçerli öznitelik değerleri XML 1,0 belirtiminde belirtilmiştir.  
  
 **Hata kimliği:** BC31181  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML 1,0 belirtiminde belirtilen şekilde öznitelik değerini geçerli bir değere güncelleştirin.
