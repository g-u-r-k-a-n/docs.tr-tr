# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>\<methodname>Yalnızca bir ' ' metodu ' \<methodname> partial ' olarak işaretleneceği için ' ' metodu ' partial ' olarak bildirilemez

Aynı ada ve imzaya sahip iki kısmi yöntem bildirimi olamaz.  
  
 **Hata kimliği:** BC31433  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Kısmi Yöntem bildirimlerinden birinin adını veya imzasını değiştirin.
