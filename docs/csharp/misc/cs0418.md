# <a name="compiler-error-cs0418"></a>Derleyici hatası CS0418

' sınıf adı ': soyut bir sınıf korumalı veya statik olamaz  
  
 Soyut bir sınıf, öğesinden türetilmediği takdirde nesneleri oluşturmak için kullanılamaz, bu nedenle korumalı hale gelmesini mantıklı hale getirir. Soyut bir sınıf herhangi bir şekilde anlamlı olamaz; soyut sınıflar, soyut sınıfı temel olarak kullanacak bir nesne hiyerarşisini destekleyecek şekilde tasarlanmıştır.  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS0418 oluşturur:  
  
```csharp  
// CS0418.cs  
public abstract sealed class C  // CS0418  
{  
}  
  
sealed static class S  // CS0418  
{  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
    }  
}  
```
