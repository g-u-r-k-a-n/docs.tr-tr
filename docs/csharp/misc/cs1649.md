# <a name="compiler-error-cs1649"></a><span data-ttu-id="04140-102">Derleyici Hatası CS1649</span><span class="sxs-lookup"><span data-stu-id="04140-102">Compiler Error CS1649</span></span>
<span data-ttu-id="04140-103">Sadece okuma alanı 'tanımlayıcı' üyeleri ref veya dışarı geçirilemez (bir yapıcı hariç)</span><span class="sxs-lookup"><span data-stu-id="04140-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="04140-104">Bir değişkeni bir `readonly` alanın üyesi olan bir işleve `ref` veya `out` bağımsız değişkene geçirirseniz bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="04140-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="04140-105">`ref` İşlev `out` tarafından değiştirilebileceğinden ve parametrelere izin verilmedi.</span><span class="sxs-lookup"><span data-stu-id="04140-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="04140-106">Bu hatayı gidermek için, alandaki anahtar kelimeyi `readonly` kaldırın veya `readonly` alan üyelerini işleve geçirmeyin.</span><span class="sxs-lookup"><span data-stu-id="04140-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="04140-107">Örneğin, aşağıdaki örnekte gösterildiği gibi değiştirilebilir ve geçici bir `ref` bağımsız değişken olarak geçirilebilir geçici bir değişken oluşturmayı deneyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="04140-107">For example, you might try creating a temporary variable which can be modified and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="04140-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="04140-108">Example</span></span>  
 <span data-ttu-id="04140-109">Aşağıdaki örnek CS1649 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="04140-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
