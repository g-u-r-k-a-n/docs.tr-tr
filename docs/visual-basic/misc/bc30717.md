# <a name="try-statements-are-not-valid-in-the-immediate-window"></a>' TRY ' deyimleri Komut penceresinde geçerli değildir

`Try` deyimler yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata kimliği:** BC30717  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Try`Hata ayıklama kodundaki deyimleri kaldır.
