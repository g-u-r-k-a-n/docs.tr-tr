# <a name="array-exceeds-the-limit-of-32-dimensions"></a>Dizi 32 boyut sınırını aşıyor
Diziler 32 taneden fazla boyuta sahip olamaz.  
  
 **Hata kimliği:** BC30052  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizi bildirimindeki boyut sayısını azaltın.
