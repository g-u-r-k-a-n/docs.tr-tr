## <a name="example"></a><span data-ttu-id="34de2-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="34de2-105">Example</span></span>  
 <span data-ttu-id="34de2-106">Aşağıdaki örnek, CS1018 oluşturur ve hatayı çözmek için çeşitli yollar önerir:</span><span class="sxs-lookup"><span data-stu-id="34de2-106">The following example generates CS1018, and suggests several ways to resolve the error:</span></span>  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's parameterless constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
