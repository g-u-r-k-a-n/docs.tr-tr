# <a name="constant-expression-not-representable-in-type-typename"></a>Sabit ifade '\<typename>' türünde gösterilemez

Genellikle aralığı taşdığı için hedef türüne sığmayan bir sabiti değerlendirmeye çalışıyorsunuz.  
  
 **Hata kimliği:** BC30439  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Hedef türünü, sabiti işleyebilen bir olacak şekilde değiştirin.
