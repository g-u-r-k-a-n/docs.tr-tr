# <a name="4811---discoverymessagewithnullrelatesto"></a>4811 - DiscoveryMessageWithNullRelatesTo

## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|ID|4811|  
|Anahtar sözcükler|Bulma|  
|Düzey|Uyarı|  
|Kanal|Microsoft-Windows-Application Server-uygulamalar/hata ayıkla|  
  
## <a name="description"></a>Açıklama  

 Bu olay, ileti üst bilgisi gerekli RelatesTo özelliğini içermediğinden, bulma iletisi DiscoveryClient tarafından bırakıldığında yayınlanır.  
  
## <a name="message"></a>İleti  

 MessageID = ' %2 ' olan bir %1 iletisi, ileti üst bilgisi gerekli RelatesTo özelliğini içermediğinden DiscoveryClient tarafından bırakıldı.
