# <a name="codepage-name-is-invalid-or-not-installed"></a>' ' Kod sayfası \<name> geçersiz veya yüklü değil

 **Hata kimliği:** BC2016  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kodunuzu inceleyin ve belirli bir duruma uygun çözümü saptayın
