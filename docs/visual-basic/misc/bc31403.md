# <a name="imports-alias-name-conflicts-with-name2-declared-in-the-root-namespace"></a>' ' Imports diğer adı \<name> \<name2> kök ad alanında belirtilen ' ' ile çakışıyor

Deyimde kullanılan diğer ad, `Imports` kök ad alanında belirtilen başka bir adla çakışıyor.  
  
 **Hata kimliği:** BC31403  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Deyimin kullandığı diğer ad için benzersiz bir ad seçin `Imports` .
