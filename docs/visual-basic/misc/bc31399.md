<span data-ttu-id="c90b5-103">Bir sınıf, bir temel sınıf belirtmek için [Inherits ifadesini](../language-reference/statements/inherits-statement.md) kullanır, ancak hedeflenen temel sınıftaki herhangi bir oluşturucuya erişemez.</span><span class="sxs-lookup"><span data-stu-id="c90b5-103">A class uses an [Inherits Statement](../language-reference/statements/inherits-statement.md) to specify a base class, but it cannot access any constructor on the intended base class.</span></span>  
  
 <span data-ttu-id="c90b5-104">Bu durum, hedeflenen temel sınıfın hiçbir Oluşturucusu yoksa veya başka bir sınıftan erişimi önleyen erişim düzeylerine sahip oluşturucuları varsa meydana gelir.</span><span class="sxs-lookup"><span data-stu-id="c90b5-104">This can happen if the intended base class has no constructors or if it has constructors with access levels that prevent access from another class.</span></span>  
  
 <span data-ttu-id="c90b5-105">Bir sınıfı aktardığınızda, oluşturucunuzu [MyBase](../programming-guide/program-structure/me-my-mybase-and-myclass.md#mybase)kullanarak temel sınıf oluşturucusunu çağırmalıdır.</span><span class="sxs-lookup"><span data-stu-id="c90b5-105">When you inherit a class, your constructor should call the base class constructor using [MyBase](../programming-guide/program-structure/me-my-mybase-and-myclass.md#mybase).</span></span> <span data-ttu-id="c90b5-106">Bu çağrıyı yapmazsanız ya da açık bir Oluşturucu yazarsanız, Visual Basic çağıran bir örtük Oluşturucu oluşturur `MyBase.New()` .</span><span class="sxs-lookup"><span data-stu-id="c90b5-106">If you do not make this call, or if you do not even write an explicit constructor, Visual Basic generates an implicit constructor that calls `MyBase.New()`.</span></span>  
  
 <span data-ttu-id="c90b5-107">**Hata kimliği:** BC31399</span><span class="sxs-lookup"><span data-stu-id="c90b5-107">**Error ID:** BC31399</span></span>

1. <span data-ttu-id="c90b5-109">Hedeflenen temel sınıf üzerinde kaynak denetiminiz varsa, başka bir sınıfın erişebilmesi için oluşturucularından en az birinin erişim düzeyini değiştirin.</span><span class="sxs-lookup"><span data-stu-id="c90b5-109">If you have source control over the intended base class, then change the access level of at least one of its constructors so that another class can access them.</span></span>
