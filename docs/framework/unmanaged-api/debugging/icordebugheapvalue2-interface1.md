# <a name="icordebugheapvalue2-interface"></a>ICorDebugHeapValue2 Arabirimi

Ortak dil çalışma zamanı (CLR) tanıtıcıları için destek sağlayan ICorDebugHeapValue uzantısı.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[CreateHandle Yöntemi](icordebugheapvalue2-createhandle-method.md)|Bu nesne için belirtilen türde bir tanıtıcı oluşturur `ICorDebugHeapValue2` .|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
> Bu arabirim, çapraz makine ya da çapraz işlem için uzaktan çağrılmakta değil.
