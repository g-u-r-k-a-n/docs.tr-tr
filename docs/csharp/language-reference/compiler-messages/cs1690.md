# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="6fb2b-103">Derleyici Uyarısı (düzey 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="6fb2b-103">Compiler Warning (level 1) CS1690</span></span>

<span data-ttu-id="6fb2b-104">Başvuruya göre sıralama sınıfının bir alanı olduğundan, ' Member ' üzerindeki bir üyeye erişmek çalışma zamanı özel durumuna neden olabilir</span><span class="sxs-lookup"><span data-stu-id="6fb2b-104">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="6fb2b-105">Bu uyarı, öğesinden türetilen bir sınıfın üyesi üzerinde bir yöntem, özellik veya Dizin Oluşturucu çağırmaya çalıştığınızda oluşur <xref:System.MarshalByRefObject> ve üye bir değer türüdür.</span><span class="sxs-lookup"><span data-stu-id="6fb2b-105">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="6fb2b-106">Öğesinden devraldığı nesneler `MarshalByRefObject` , genellikle bir uygulama etki alanı genelinde başvuruya göre sıralanmalıdır.</span><span class="sxs-lookup"><span data-stu-id="6fb2b-106">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="6fb2b-107">Herhangi bir kod, bir uygulama etki alanında böyle bir nesnenin değer türü üyesine doğrudan erişmeyi denerse, bir çalışma zamanı <xref:System.InvalidOperationException> meydana gelir.</span><span class="sxs-lookup"><span data-stu-id="6fb2b-107">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="6fb2b-108">Uyarıyı çözmek için önce üyeyi yerel bir değişkene kopyalayın ve bu değişkende yöntemi çağırın.</span><span class="sxs-lookup"><span data-stu-id="6fb2b-108">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="6fb2b-109">Aşağıdaki örnek CS1690 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6fb2b-109">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {
        AppDomain domain = AppDomain.CreateDomain("MyDomain");
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;
    }
}
```
