 <span data-ttu-id="d6196-112">Language-Integrated Query (LINQ) çerçevesi iki JOIN işleci sağlar <xref:System.Linq.Enumerable.Join%2A> ve <xref:System.Linq.Enumerable.GroupJoin%2A> .</span><span class="sxs-lookup"><span data-stu-id="d6196-112">The Language-Integrated Query (LINQ) framework provides two join operators, <xref:System.Linq.Enumerable.Join%2A> and <xref:System.Linq.Enumerable.GroupJoin%2A>.</span></span> <span data-ttu-id="d6196-113">Bu işleçler *eş birleştirme* gerçekleştirir: diğer bir deyişle, yalnızca anahtarları eşitse iki veri kaynağıyla eşleşen birleşimler.</span><span class="sxs-lookup"><span data-stu-id="d6196-113">These operators perform *equi-joins*: that is, joins that match two data sources only when their keys are equal.</span></span> <span data-ttu-id="d6196-114">(Bunun aksine, Transact-SQL, işleç gibi dışında bir JOIN işleçlerini destekler `equals` `less than` .)</span><span class="sxs-lookup"><span data-stu-id="d6196-114">(By contrast, Transact-SQL supports join operators other than `equals`, such as the `less than` operator.)</span></span>
