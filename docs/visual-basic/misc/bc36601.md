# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Tür karakterleri Aralık değişkeni bildirimlerinde kullanılamaz

Bir LINQ sorgusunda Aralık değişkeni bildiriminin adının sonuna bir tür karakteri eklenmiştir.  
  
 **Hata kimliği:** BC36601  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Aralık değişkeni bildiriminden tür karakterini kaldırın.
