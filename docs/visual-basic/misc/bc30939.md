# <a name="addressof-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>' \<typename> ' Türü ' \<typename> MustInherit ' olarak bildirildiği ve oluşturulamadığından, ' AddressOf ' ifadesi ' ' olarak dönüştürülemez

Deyim, bir `AddressOf` ifadeyi yalnızca temel sınıf olabilecek bir türe dönüştürmeye çalışır ve bir örnek oluşturmak için kullanılamaz.  
  
 `AddressOf`İşleci, belirli bir yordama başvuran bir yordam temsilci örneği oluşturur.  
  
 **Hata kimliği:** BC30939  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `AddressOf`İfadeyi belirli bir temsilci türüne atayın.
