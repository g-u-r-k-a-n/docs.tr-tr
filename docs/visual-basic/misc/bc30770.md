# <a name="event-eventname-event-specified-by-the-defaultevent-attribute-is-not-a-publicly-accessible-event-for-this-class"></a>\<eventname>' DefaultEvent ' özniteliği tarafından belirtilen ' ' olayı, bu sınıf için genel olarak erişilebilir bir olay değil

<xref:System.ComponentModel.DefaultEventAttribute>Öznitelik, özniteliğin uygulandığı sınıfta herkese açık olarak erişilebilen olay adını belirtmelidir.  
  
 **Hata kimliği:** BC30770  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Sınıfın herkese açık olarak erişilebilen bir olay tanımladığından emin olun.  
  
2. Sınıftaki olayın adının öznitelik tarafından belirtilen ad ile eşleştiğinden emin olun <xref:System.ComponentModel.DefaultEventAttribute> .
