# <a name="tlbexp-helper-functions-unmanaged-api-reference"></a>Tlbexp Yardımcı İşlevleri (Yönetilmeyen API Başvurusu)
[Tip Kitaplığı İhracatçısı aracı](../../tools/tlbexp-exe-type-library-exporter.md) (Tlbexp.exe) TlbRef.dll adlı dinamik bir bağlantı kitaplığı yükler. Bu DLL, iki yardımcı işlev ve ihracatçı aracının derleme-tip kitaplık dönüştürme işlemi sırasında kullandığı bir arabirim içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [GetTypeLibInfo İşlevi](gettypelibinfo-function.md)  
 Tür kitaplığı için yerelleştirme ve işletim sistemi bilgileri sağlar.  
  
 [LoadTypeLibWithResolver İşlevi](loadtypelibwithresolver-function.md)  
 Başvurulan tür kitaplıklarını çözmek için [ITypeLibResolver arabiriminin](itypelibresolver-interface.md) bir uygulamasını kullanarak bir tür kitaplığı yükler.  
  
 [ITypeLibResolver Arabirimi](itypelibresolver-interface.md)  
 Bir tür kitaplığın tam nitelikli yolunu döndüren [ResolveTypeLib yöntemini](resolvetypelib-method.md)sağlar.
