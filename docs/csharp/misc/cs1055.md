# <a name="compiler-error-cs1055"></a><span data-ttu-id="c8d20-103">Derleyici hatası CS1055</span><span class="sxs-lookup"><span data-stu-id="c8d20-103">Compiler Error CS1055</span></span>
<span data-ttu-id="c8d20-104">Ekleme veya kaldırma erişimcisi bekleniyor</span><span class="sxs-lookup"><span data-stu-id="c8d20-104">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="c8d20-105">[Olaylarınız](../language-reference/keywords/event.md) bir alan olarak bildirilirse, hem **ekleme** hem de **kaldırma** erişimci işlevlerini tanımlamalıdır.</span><span class="sxs-lookup"><span data-stu-id="c8d20-105">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="c8d20-106">Aşağıdaki örnek CS1055 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c8d20-106">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
