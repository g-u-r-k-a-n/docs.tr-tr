# <a name="dynamicresource-markup-extension"></a><span data-ttu-id="e7a7d-102">DynamicResource Biçimlendirme Uzantısı</span><span class="sxs-lookup"><span data-stu-id="e7a7d-102">DynamicResource Markup Extension</span></span>
<span data-ttu-id="e7a7d-103">Bu değeri tanımlı bir kaynağa başvuru olarak erteleyerek herhangi [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] bir özellik özniteliği için bir değer sağlar.</span><span class="sxs-lookup"><span data-stu-id="e7a7d-103">Provides a value for any [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] property attribute by deferring that value to be a reference to a defined resource.</span></span> <span data-ttu-id="e7a7d-104">Bu kaynağın arama davranışı, çalışma zamanı aramaya benzer.</span><span class="sxs-lookup"><span data-stu-id="e7a7d-104">Lookup behavior for that resource is analogous to run-time lookup.</span></span>

```xml

```

```xml  
  </object.property>

```
