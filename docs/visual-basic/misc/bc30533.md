# <a name="conversion-from-double-to-date-requires-calling-the-datefromoadate"></a>' Double ' türünden ' Date ' türüne dönüştürme için ' Date. FromOADate ' çağrısı gerekir

Bir değeri bir değere dönüştürmeye çalıştınız `Date` `Double` , bu, yöntemi kullanılmadan gerçekleştirilemez <xref:System.DateTime.FromOADate%2A?displayProperty=nameWithType> .  
  
 **Hata kimliği:** BC30533  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- <xref:System.DateTime.FromOADate%2A>Değeri dönüştürmek için yöntemini kullanın.
