# <a name="debugging-unmanaged-api-reference"></a>Hata Ayıklama (Yönetilmeyen API Başvurusu)
Hata ayıklama API 'SI, bir hata ayıklayıcının ortak dil çalışma zamanı (CLR) ortamında çalışan kodun hatalarını ayıklamasına olanak sağlar. Ayıklanacak kod, CLR tarafından desteklenen herhangi bir kod türü olabilir.
