# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>\<membername1> \<membername2> isteğe bağlı parametrelerinin türleri farklı olduğundan ' ', ' ' öğesini geçersiz kılamaz

Aşırı yüklenmiş Üyeler bir veya daha fazla gerekli parametre için farklı veri türlerine sahip olmalıdır.  
  
 **Hata kimliği:** BC30697  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Farklı bir veri türü olarak en az bir gerekli parametre tanımlayın.
