# <a name="name-is-ambiguous-between-declarations-in-modules-modulename1-and-modulename2"></a>' ' \<name> , ' \<modulename1> ' ve ' ' modüllerindeki bildirimler arasında \<modulename2> belirsiz

Belirsiz bir ad sağladınız ve bu nedenle başka bir adla çakışıyor. Visual Basic derleyicisinde herhangi bir çakışma çözümleme kuralı yok; adları kendiniz kesinleştirmeniz gerekir.  
  
 **Hata kimliği:** BC30562  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Adı tam olarak nitelendirin.
