# <a name="warning-number-number-for-the-option-optionname-is-either-not-configurable-or-not-valid"></a>\<number>' ' Seçeneği için ' ' uyarı numarası \<optionname> yapılandırılamaz ya da geçerli değil

Uyarı KIMLIĞI geçerli değil. Kimlikler 0 ' dan büyük tamsayı değerleri olmalıdır.  
  
 **Hata kimliği:** BC2026  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli olduğundan emin olmak için sağlanan uyarı KIMLIĞINI denetleyin.
