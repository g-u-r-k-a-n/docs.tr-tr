# <a name="methods-declared-overrides-cannot-be-declared-overridable-because-they-are-implicitly-overridable"></a>Örtülü olarak geçersiz kılınabilir olduklarından ' geçersiz kılmalar ' olarak belirtilen metotlar ' geçersiz kılınabilir ' olarak bildirilemez

Çoğu yöntemden farklı olarak, `Overrides` değiştiriciyle işaretlenen Yöntemler varsayılan olarak geçersiz kılınabilir.  
  
 **Hata kimliği:** BC30730  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Overridable`Değiştiriciyle işaretlenen yöntemlerin değiştiricisini atlayın `Overrides` .
