## <a name="example"></a><span data-ttu-id="9f435-129">Örnek</span><span class="sxs-lookup"><span data-stu-id="9f435-129">Example</span></span>  

 <span data-ttu-id="9f435-130">Aşağıdaki kod örneği <xref:System.Threading.Thread.Suspend%2A> , etkinleştirmesine neden olan yöntemine yapılan çağrıyı gösterir `dangerousThreadingAPI` .</span><span class="sxs-lookup"><span data-stu-id="9f435-130">The following code example demonstrates a call to the <xref:System.Threading.Thread.Suspend%2A> method that causes the activation of the `dangerousThreadingAPI`.</span></span>
