# <a name="compiler-error-cs1625"></a><span data-ttu-id="25170-103">Derleyici hatası CS1625</span><span class="sxs-lookup"><span data-stu-id="25170-103">Compiler Error CS1625</span></span>
<span data-ttu-id="25170-104">Finally yan tümcesinin gövdesinde ödeme yapılamaz</span><span class="sxs-lookup"><span data-stu-id="25170-104">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="25170-105">Finally yan tümcesinin gövdesinde bir yield ifadesine izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="25170-105">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="25170-106">Bu hatayı önlemek için, yield ifadesini finally yan tümcesinin dışına taşıyın.</span><span class="sxs-lookup"><span data-stu-id="25170-106">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="25170-107">Aşağıdaki örnek CS1625 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="25170-107">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
