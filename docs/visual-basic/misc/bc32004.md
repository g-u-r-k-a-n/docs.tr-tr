# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>' ', ' ' \<name> \<projectname1> projesi tarafından başvurulmayan ' ' projesinde bildiriliyor \<projectname2>
Bir programlama öğesi, geçerli projenin bir başvurusu olmayan bir projede tanımlanır.  
  
 **Hata kimliği:** BC32004  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Geçerli projede, istenen öğeyi içeren projeye bir başvuru ekleyin.
