# <a name="information-for-the-type-of-typename-has-not-been-loaded-into-the-runtime"></a>' ' Türü için bilgi \<typename> çalışma zamanına yüklenmedi

Çalışma zamanı tarafından yüklenmeyen bir türe başvuruldu.  
  
 **Hata kimliği:** BC30750  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Türün geçerli kapsam içinde tanımlanması için kodunuzu yeniden yapılandır.  
  
2. Üyenin tanımlandığını ve üye adını doğru yazdığınızdan emin olun.  
  
3. Modülde belirtilen üyelerden birine erişmeyi deneyin. Bazı durumlarda, bildirildiği modüller yüklü olmadığı için hata ayıklama ortamı üyeleri bulamaz.
