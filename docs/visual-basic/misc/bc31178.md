# <a name="expected-closing--for-xml-entity"></a>XML varlığı için '; ' kapanışı bekleniyor

Bir XML varlık başvurusu (örneğin, `>` ), bir kapanış noktalı virgül (;) eksik.  
  
 **Hata kimliği:** BC31178  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML varlık başvurusunun sonuna noktalı virgül ekleyin.
