4. <span data-ttu-id="f4912-109">Declare the necessary constants in your code.</span><span class="sxs-lookup"><span data-stu-id="f4912-109">Declare the necessary constants in your code.</span></span> <span data-ttu-id="f4912-110">Many Windows constants are 32-bit unsigned values, and you should declare these `As UInteger`.</span><span class="sxs-lookup"><span data-stu-id="f4912-110">Many Windows constants are 32-bit unsigned values, and you should declare these `As UInteger`.</span></span>

5. <span data-ttu-id="f4912-111">Call the function in the normal way.</span><span class="sxs-lookup"><span data-stu-id="f4912-111">Call the function in the normal way.</span></span> <span data-ttu-id="f4912-112">The following example calls the Windows function `MessageBox`, which takes an unsigned integer argument.</span><span class="sxs-lookup"><span data-stu-id="f4912-112">The following example calls the Windows function `MessageBox`, which takes an unsigned integer argument.</span></span>

    ```vb
    Public Class windowsMessage
        Private Declare Auto Function mb Lib "user32.dll" Alias "MessageBox" (
            ByVal hWnd As Integer,
            ByVal lpText As String,
            ByVal lpCaption As String,
            ByVal uType As UInteger) As Integer
        Private Const MB_OK As UInteger = 0
        Private Const MB_ICONEXCLAMATION As UInteger = &H30
        Private Const IDOK As UInteger = 1
        Private Const IDCLOSE As UInteger = 8
        Private Const c As UInteger = MB_OK Or MB_ICONEXCLAMATION
        Public Function messageThroughWindows() As String
            Dim r As Integer = mb(0, "Click OK if you see this!",
                "Windows API call", c)
            Dim s As String = "Windows API MessageBox returned " &
                 CStr(r)& vbCrLf & "(IDOK = " & CStr(IDOK) &
                 ", IDCLOSE = " & CStr(IDCLOSE) & ")"
            Return s
        End Function
    End Class
    ```

     <span data-ttu-id="f4912-113">You can test the function `messageThroughWindows` with the following code.</span><span class="sxs-lookup"><span data-stu-id="f4912-113">You can test the function `messageThroughWindows` with the following code.</span></span>

    ```vb

        w.messageThroughWindows()
    End Sub
    ```
