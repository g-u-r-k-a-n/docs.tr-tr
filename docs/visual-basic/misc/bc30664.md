# <a name="conversions-from-typename1-to-typename2-must-be-explicit"></a>' \<typename1> ' İle ' ' arasındaki dönüşümler \<typename2> açık olmalıdır
Bu dönüştürme örtük olarak gerçekleştirilemez.  
  
 **Hata kimliği:** BC30664  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Ya da gibi `CType` belirli dönüştürme işlevlerinden birini kullanın `CInt` `CDec` .
