# <a name="exit-must-be-followed-by-sub-function-property-do-for-while-select-or-try"></a>' Exit ' arkasından ' Sub ', ' function ', ' Property ', ' do ', ' for ', ' while ', ' Select ' veya ' TRY ' gelmelidir

Bir `Exit` ifade geçersiz bir anahtar sözcük içeriyor. `Exit` denetimin bloğundan sonraki deyime aktarılacağı blok belirtilmelidir; Örneğin, `End While` .  
  
 **Hata kimliği:** BC30240  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Aşağıdaki geçerli bir anahtar sözcük ekleyin `Exit` veya ifadesini kaldırın `Exit` .
