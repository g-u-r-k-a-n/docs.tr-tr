<span data-ttu-id="74d0a-103">'tanımlayıcı': değişken türünde bir örnek oluştururken bağımsız değişkenler sağlayamazsınız</span><span class="sxs-lookup"><span data-stu-id="74d0a-103">'identifier': cannot provide arguments when creating an instance of a variable type</span></span>  
  
 <span data-ttu-id="74d0a-104">Bu hata, bir tür `new` parametresi üzerinde işleci bir çağrı bağımsız değişkenleri varsa oluşur.</span><span class="sxs-lookup"><span data-stu-id="74d0a-104">This error occurs if a call to the `new` operator on a type parameter has arguments.</span></span> <span data-ttu-id="74d0a-105">Bilinmeyen bir parametre türünde `new` işleci kullanılarak çağrılabilen tek oluşturucu, bağımsız değişkeni olmayan bir oluşturucudur.</span><span class="sxs-lookup"><span data-stu-id="74d0a-105">The only constructor that can be called by using the `new` operator on an unknown parameter type is a constructor that has no arguments.</span></span> <span data-ttu-id="74d0a-106">Başka bir oluşturucuçağırmanız gerekiyorsa, sınıf türü kısıtlaması veya arabirim kısıtlaması kullanmayı düşünün.</span><span class="sxs-lookup"><span data-stu-id="74d0a-106">If you need to call another constructor, consider using a class type constraint or interface constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="74d0a-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="74d0a-107">Example</span></span>  

 <span data-ttu-id="74d0a-108">Aşağıdaki örnek CS0417 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="74d0a-108">The following example generates CS0417:</span></span>  

```csharp
// CS0417

{  
    // The following line causes CS0417.  
    T instance1 = new T(1);
  
    // The following line doesn't cause the error.  
    T instance2 = new T();  
}  
```
