# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="9d123-102">Derleyici Uyarısı (düzey 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="9d123-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="9d123-103">'member1' devralınan üye 'member2'yi gizler.</span><span class="sxs-lookup"><span data-stu-id="9d123-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="9d123-104">Gizlemek amaçlanıyorsa yeni anahtar sözcüğü kullanın.</span><span class="sxs-lookup"><span data-stu-id="9d123-104">Use the new keyword if hiding was intended.</span></span>

 <span data-ttu-id="9d123-105">Bir değişken, taban sınıftaki bir değişkenle aynı ada sahip olarak bildirildi.</span><span class="sxs-lookup"><span data-stu-id="9d123-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="9d123-106">Ancak, [yeni](../keywords/new-modifier.md) anahtar kelime kullanılmadı.</span><span class="sxs-lookup"><span data-stu-id="9d123-106">However, the [new](../keywords/new-modifier.md) keyword was not used.</span></span> <span data-ttu-id="9d123-107">Bu **uyarı, yeni**kullanmanız gerektiğini bildirir; değişken, beyannamede **yeni** kullanılmış gibi bildirilir.</span><span class="sxs-lookup"><span data-stu-id="9d123-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>

 <span data-ttu-id="9d123-108">Aşağıdaki örnek CS0108 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9d123-108">The following sample generates CS0108:</span></span>

```csharp
// CS0108.cs
// compile with: /W:2
using System;

namespace x
{
    public class clx
    {
        public int i = 1;
    }

    public class cly : clx
    {
        public static int i = 2;   // CS0108, use the new keyword
        // Use the following line instead:
        // public static new int i = 2;

      public static void Main()
      {
         Console.WriteLine(i);
      }
   }
}
```
