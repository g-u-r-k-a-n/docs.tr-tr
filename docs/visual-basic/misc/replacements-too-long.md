# <a name="replacements-too-long"></a>Değişiklikler çok uzun

İzin verilen uzunluğu aşan bir değiştirme belirttiniz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Uzunluk kısıtlamasının bileşen belgelerini inceleyin.
