# <a name="core-communications-tcp-transport-channels"></a>Temel İletişimler: TCP Taşıma Kanalları

Bu konu, TCP taşıma kanalları tarafından oluşturulan tüm özel durumları listeler.  
  
## <a name="exception-list"></a>Özel durum listesi  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|SocketCloseReadTimeout|Belirtilen yuvanın uzak uç noktası belirtilen ayrılan zaman aşımı süresi içinde bir kapatma isteğine yanıt vermedi. Alma işleminden EOF sinyali (null) alındıktan sonra uzak uç nokta kapatma çağrısı yapılamaz. Bu işlem için ayrılan süre daha uzun bir zaman aşımı değerinin bir bölümü olabilir.|
