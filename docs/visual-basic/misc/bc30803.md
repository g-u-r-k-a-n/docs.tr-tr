# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>' Currency ' artık desteklenen bir tür değil; Bunun yerine ' Decimal ' türünü kullanın
`Currency`Veri türü `Decimal` veri türüyle değiştirilmiştir.  
  
 **Hata kimliği:** BC30803  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Decimal`Yerine veri türünü kullanın `Currency` .
