## <a name="cause"></a><span data-ttu-id="0cd35-111">Nedeni</span><span class="sxs-lookup"><span data-stu-id="0cd35-111">Cause</span></span>

<span data-ttu-id="0cd35-112">Bir sembol ve üyesi aynı deyime atandı.</span><span class="sxs-lookup"><span data-stu-id="0cd35-112">A symbol and its member were assigned in the same statement.</span></span> <span data-ttu-id="0cd35-113">Örnek:</span><span class="sxs-lookup"><span data-stu-id="0cd35-113">For example:</span></span>

```csharp
// 'a' and 'a.Field' are assigned in the same statement

```

## <a name="rule-description"></a><span data-ttu-id="0cd35-114">Kural açıklaması</span><span class="sxs-lookup"><span data-stu-id="0cd35-114">Rule description</span></span>

## <a name="how-to-fix-violations"></a><span data-ttu-id="0cd35-118">İhlalleri çözme</span><span class="sxs-lookup"><span data-stu-id="0cd35-118">How to fix violations</span></span>

<span data-ttu-id="0cd35-119">İhlalleri onarmak için, çoklu atama deyimini iki veya daha fazla basit atama bildirimine ayırın.</span><span class="sxs-lookup"><span data-stu-id="0cd35-119">To fix violations, split the multi-assign statement into two or more simple assignment statements.</span></span> <span data-ttu-id="0cd35-120">Örneğin, aşağıdaki kod parçacığı, kural ihlalini ve Kullanıcı amacına bağlı olarak bunu gidermenin birkaç yolunu gösterir:</span><span class="sxs-lookup"><span data-stu-id="0cd35-120">For example, the following code snippet shows a violation of the rule and a couple of ways to fix it based on the user intent:</span></span>

```csharp
public class C
{
    public C Field;
}

public class Test
{
    public void M(C a, C b)
    {
        // Let us assume 'a' points to 'Instance1' and 'b' points to 'Instance2' at the start of the method.
        // It is not clear if the user intent in the below statement is to assign to 'Instance1.Field' or 'Instance2.Field'.
        // CA2246: Symbol 'a' and its member 'Field' are both assigned in the same statement. You are at risk of assigning the member of an unintended object.
        a.Field = a = b;
    }
}
```

```csharp
public class C
{
    public C Field;
}

public class Test
{
    public void M(C a, C b)
    {
        // Let us assume 'a' points to 'Instance1' and 'b' points to 'Instance2' at the start of the method.
        // 'Instance1.Field' is intended to be assigned.
        var instance1 = a;
        a = b;
        instance1.Field = a;
    }
}
```

```csharp
public class C
{
    public C Field;
}

public class Test
{
    public void M(C a, C b)
    {
        // Let us assume 'a' points to 'Instance1' and 'b' points to 'Instance2' at the start of the method.
        // 'Instance2.Field' is intended to be assigned.
        a = b;
        b.Field = a; // or 'a.Field = a;'
    }
}
```
