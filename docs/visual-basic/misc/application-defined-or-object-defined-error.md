# <a name="application-defined-or-object-defined-error"></a>Uygulama tanımlı veya nesne tanımlı hata

Uygulama tanımlı veya Kullanıcı tanımlı bir hata oluştu. Hatayı denetleyin.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Nerede tanımlandığını ve nasıl ele alınacağını görmek için hataya bakın.
