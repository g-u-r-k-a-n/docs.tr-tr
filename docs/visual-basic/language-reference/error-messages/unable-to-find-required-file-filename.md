# <a name="unable-to-find-required-file-filename"></a>Gerekli '\<filename > ' dosyası bulunamıyor
Visual Studio'nun gerek duyduğu bir dosya eksik veya zarar görmüş.  
  
 **Hata kimliği:** BC30655  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Visual Studio'yu yeniden yükleyin.
