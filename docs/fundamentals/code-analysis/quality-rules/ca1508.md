## <a name="rule-description"></a><span data-ttu-id="1d416-114">Kural açıklaması</span><span class="sxs-lookup"><span data-stu-id="1d416-114">Rule description</span></span>

<span data-ttu-id="1d416-115">Yöntemler, IF deyimleri, ikili ifadeler ( `==` , `!=` , `<` , `>` ), null denetimleri vb. gibi koşullu koda sahip olabilir. Örneğin, aşağıdaki kodu göz önünde bulundurun:</span><span class="sxs-lookup"><span data-stu-id="1d416-115">Methods can have conditional code, such as if statements, binary expressions (`==`, `!=`, `<`, `>`), null checks, etc. For example, consider the following code:</span></span>

```csharp
public void M(int i, int j)
{
    if (i != 0)
    {
        return;
    }

    if (j != 0)
    {
        return;
    }

    // Below condition will always evaluate to 'false' as 'i' and 'j' are both '0' here.
    if (i != j)
    {
        // Code in this 'if' branch is dead code.
        // It can either be removed or refactored.
        ...
    }
}
```
