    - <span data-ttu-id="ae310-116">[Güçlü Ad aracını (Sn.exe)](../../framework/tools/sn-exe-strong-name-tool.md) kullanarak bir şifreleme anahtar çifti oluşturma ve bu anahtar çiftini bir komut satırı derleyicisi veya [Derleme Bağlayıcısı (Al.exe)](../../framework/tools/al-exe-assembly-linker.md)kullanarak derlemeye atama.</span><span class="sxs-lookup"><span data-stu-id="ae310-116">Creating a cryptographic key pair using the [Strong Name tool (Sn.exe)](../../framework/tools/sn-exe-strong-name-tool.md) and assigning that key pair to the assembly using either a command-line compiler or the [Assembly Linker (Al.exe)](../../framework/tools/al-exe-assembly-linker.md).</span></span> <span data-ttu-id="ae310-117">Windows SDK hem Sn.exe hem de Al.exe sağlar.</span><span class="sxs-lookup"><span data-stu-id="ae310-117">The Windows SDK provides both Sn.exe and Al.exe.</span></span>
