# <a name="internal-error-visual-basic"></a>İç hata (Visual Basic)

Visual Basic bir iç hatalı çalışma oluştu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bu hatanın deyimin veya metodun oluşturulmadığından emin olun `Error` `Raise` . Olmadığı durumlar dışında, iletinin göründüğü koşulları bildirmek için Microsoft Ürün Destek Hizmetleri 'ne başvurun.
