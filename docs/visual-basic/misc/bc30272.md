# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>' \<argumentname> ' bir ' ' parametresi değil \<methoddeclaration>
Yordam çağrısı, yordam tarafından tanımlanmayan ek bir bağımsız değişken sağlar.  
  
 **Hata kimliği:** BC30272  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken listesinden gereksiz bağımsız değişkeni kaldırın.
