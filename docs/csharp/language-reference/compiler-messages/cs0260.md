# <a name="compiler-error-cs0260"></a><span data-ttu-id="4ed65-103">Derleyici Hatası CS0260</span><span class="sxs-lookup"><span data-stu-id="4ed65-103">Compiler Error CS0260</span></span>

<span data-ttu-id="4ed65-104">' Type ' türündeki bildirimde kısmi değiştirici eksik; Bu türün başka bir kısmi bildirimi var</span><span class="sxs-lookup"><span data-stu-id="4ed65-104">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="4ed65-105">Bu hata, aynı ada sahip birden çok sınıf bildirdiğin olduğunu gösterir.</span><span class="sxs-lookup"><span data-stu-id="4ed65-105">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="4ed65-106">Ayrıca, en az bir tane, bildirimlerin hepsi `partial` değiştirici içermez.</span><span class="sxs-lookup"><span data-stu-id="4ed65-106">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="4ed65-107">Birkaç bölümde bir sınıf tanımlamak istiyorsanız, her parçayı anahtar sözcüğünü kullanarak bildirmeniz gerekir `partial` .</span><span class="sxs-lookup"><span data-stu-id="4ed65-107">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="4ed65-108">Bu hata ayrıca bir sınıf bildirdiğinizde ve yanlışlıkla aynı ad alanında başka bir yerde tanımlanmış kısmi bir sınıf ile aynı adı verirseniz oluşur.</span><span class="sxs-lookup"><span data-stu-id="4ed65-108">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="4ed65-109">Aşağıdaki örnek CS0260 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4ed65-109">The following sample generates CS0260:</span></span>  

```csharp
// CS0260.cs  
// You must mark both parts of the definition of class C
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C
{  
}  
  
partial class C  
{  
}  
```
