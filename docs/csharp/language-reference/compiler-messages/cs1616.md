# <a name="compiler-warning-level-1-cs1616"></a>Derleyici Uyarısı (düzey 1) CS1616
' Option ' seçeneği bir kaynak dosyasında veya eklenen modülde verilen ' Attribute ' özniteliğini geçersiz kılar  
  
 Bu uyarı, derleme öznitelikleri <xref:System.Reflection.AssemblyKeyFileAttribute> veya <xref:System.Reflection.AssemblyKeyNameAttribute> kaynak üzerinde bulunursa, [-keyfile](../compiler-options/keyfile-compiler-option.md) veya [-keycontainer](../compiler-options/keycontainer-compiler-option.md) komut satırı seçeneği veya anahtar dosya adı ya da proje özelliklerinde belirtilen anahtar kapsayıcısı ile çakışıyorsa oluşur.  
  
 Aşağıdaki örnekte adlı bir anahtar dosyanız olduğunu varsayalım `cs1616.snk` . Komut satırıyla bu dosyayı oluşturun:  
  
```console  
sn -k CS1616.snk  
```  
  
 Aşağıdaki örnek CS1616 oluşturur:  
  
```csharp  
// CS1616.cs  
// compile with: /keyfile:cs1616.snk  
using System.Reflection;  
  
// To fix the error, remove the next line  
[assembly: AssemblyKeyFile("cs1616b.snk")]  // CS1616  
  
class C  
{  
  public static void Main()  
  {  
  }  
}  
```
