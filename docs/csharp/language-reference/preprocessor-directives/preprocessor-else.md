# <a name="else-c-reference"></a>#else (C# Başvurusu)
`#else`, önceki [#if](./preprocessor-if.md) veya (isteğe bağlı) [#elif](./preprocessor-elif.md) yönergelerinden hiçbiri olarak değerlendiriyorsa `true` derleyici, ve arasındaki tüm kodları değerlendirmek için bileşik bir koşullu yönerge oluşturmanızı sağlar `#else` `#endif`  
  
## <a name="remarks"></a>Açıklamalar  
 [#endif](./preprocessor-endif.md) sonraki ön işlemci yönergesi olmalıdır `#else` . Öğesinin nasıl kullanılacağına ilişkin bir örnek için bkz. [#if](./preprocessor-if.md) `#else` .
