# <a name="namespace-statements-can-occur-only-at-file-or-namespace-level"></a>' Namespace ' deyimleri yalnızca dosya veya ad alanı düzeyinde bulunabilir

`Namespace` deyimler, `Option` `Imports` kaynak dosyanızdaki diğer tüm bildirimlerden önce deyimler, deyimler ve genel özniteliklerden sonra gelmelidir.  
  
 **Hata kimliği:** BC30618  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Namespace`İfadeyi ad alanı bildiriminin veya kaynak dosyanın en üstüne taşıyın.
