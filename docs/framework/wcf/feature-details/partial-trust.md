# <a name="partial-trust"></a>Kısmi Güven

.NET Framework 3,5 ' den başlayarak, kısmen güvenilen çağıranlar, ve ' de uygulanan Genel türlere ve yöntemlere erişebilir <xref:System.ServiceModel> <xref:System.Runtime.Serialization> <xref:System.ServiceModel.Web> . Bu bölümde, kısmen güvenilen bir uygulama içinde Windows Communication Foundation (WCF) kullanmak için desteklenen senaryolar ve azaltılmış kod erişim güvenliği (CAS) izinleriyle çalışan uygulamalar için kullanılabilen WCF işlevselliğinin sınırlı alt kümesi açıklanmaktadır.  
  
## <a name="in-this-section"></a>Bu Bölümde  

 [Desteklenen Dağıtım Senaryoları](supported-deployment-scenarios.md)  
 WCF çalıştırmaya yönelik ana kısmi güven senaryolarını açıklar.  
  
 [Kısmi Güven Özelliği Uyumluluğu](partial-trust-feature-compatibility.md)  
 Kısmi güvenle kullanılamayan WCF özelliklerini açıklar.  
  
 [Kısmi Güven En İyi Uygulamaları](partial-trust-best-practices.md)  
 Kısmen güvenilen uygulamalarda WCF kullanmak için en iyi uygulamaları içerir.
