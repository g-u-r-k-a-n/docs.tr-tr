# <a name="expected-closing--for-xml-processor-instruction"></a>XML işlemci yönergesi için '? > ' kapanışı bekleniyor

XML işleme yönergesi sabit değerinde gereken kapatma `?>` karakterleri eksik.  
  
 **Hata kimliği:** BC31160  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `?>`XML işleme yönergesi sabit değerinin sonuna gereken kapatma karakterlerini ekleyin.
