# <a name="unable-to-create-a-net-runtime-interface-error"></a>.NET çalışma zamanı arabirimi oluşturulamıyor: \<error>

Arabirim oluşturulurken bir iç derleyici hatası oluştu veya bir arabirimde Yöntem çağrısı başarısız oldu.  
  
 **Hata kimliği:** BC31024  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Çalışmanızı kaydedin ve Visual Studio 'Yu yeniden başlatın.  
  
2. Hata yinelenirse, Visual Studio 'Yu yeniden yükleyin.  
  
3. Yeniden yükleme sonrasında hata devam ederse, Microsoft Ürün Destek Hizmetleri 'ne bildirin.
