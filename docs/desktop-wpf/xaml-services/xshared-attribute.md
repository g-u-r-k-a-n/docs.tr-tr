# <a name="xshared-attribute"></a><span data-ttu-id="c3baf-102">x:Shared Özniteliği</span><span class="sxs-lookup"><span data-stu-id="c3baf-102">x:Shared Attribute</span></span>

<span data-ttu-id="c3baf-103">`false`Ayarlandığında, wpf kaynak alma davranışını değiştirir, böylece atfedilen kaynak istekleri tüm istekler için aynı örneği paylaşmak yerine her istek için yeni bir örnek oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c3baf-103">When set to `false`, modifies WPF resource-retrieval behavior so that requests for the attributed resource create a new instance for each request instead of sharing the same instance for all requests.</span></span>

```xaml
  <object x:Shared="false".../>
</ResourceDictionary>
```
