 <span data-ttu-id="8ee84-105">Bu hata, değişkenin salt okunur olduğu bir bağlamda [ref](../language-reference/keywords/ref.md) veya [Out](../language-reference/keywords/out-parameter-modifier.md) bağımsız değişkeni olarak geçirildiğinde oluşur.</span><span class="sxs-lookup"><span data-stu-id="8ee84-105">This error occurs when a variable is passed as a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="8ee84-106">ReadOnly bağlamları, değişkenleri ve değişkenleri [kullanarak](../language-reference/keywords/using-statement.md) [foreach](../language-reference/keywords/foreach-in.md) yineleme değişkenlerini içerir `fixed` .</span><span class="sxs-lookup"><span data-stu-id="8ee84-106">Readonly contexts include [foreach](../language-reference/keywords/foreach-in.md) iteration variables, [using](../language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="8ee84-107">Bu hatayı çözmek için `foreach` `using` `fixed` `ref` `out` `using` blok, `foreach` deyimler ve deyimlerde bir veya parametresi `fixed` olarak veya değişkenini alan işlevleri çağırmayın.</span><span class="sxs-lookup"><span data-stu-id="8ee84-107">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8ee84-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="8ee84-108">Example</span></span>  
 <span data-ttu-id="8ee84-109">Aşağıdaki örnek CS1657 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8ee84-109">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
