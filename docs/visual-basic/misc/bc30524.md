# <a name="property-propertyname-is-writeonly"></a>' \<propertyname> ' Özelliği ' WriteOnly '

Belirtilen bir özellikten değer almaya çalıştınız `WriteOnly` .  
  
 **Hata kimliği:** BC30524  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `WriteOnly`Özellik bildiriminden belirteci kaldırın.
