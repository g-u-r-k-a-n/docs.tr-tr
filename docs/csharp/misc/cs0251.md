# <a name="compiler-warning-level-2-cs0251"></a>Derleyici Uyarısı (düzey 2) CS0251

Bir diziyi negatif bir dizinle dizine ekleme (dizi dizinleri her zaman sıfırdan başlar)  
  
 Bir dizide dizin eklemek için negatif bir sayı kullanmayın.  
  
 Aşağıdaki örnek CS0251 oluşturur:  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
