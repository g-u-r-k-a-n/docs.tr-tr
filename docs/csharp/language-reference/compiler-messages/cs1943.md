<span data-ttu-id="b6c94-103">Kaynak türü 'türü' olan bir sorgu ifadesindeki tümceden sonraki bir sözcükte 'tür' ifadesine izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="b6c94-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="b6c94-104">'Yöntem' çağrısında tür çıkarım başarısız oldu.</span><span class="sxs-lookup"><span data-stu-id="b6c94-104">Type inference failed in the call to 'method'.</span></span>

## <a name="example"></a><span data-ttu-id="b6c94-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="b6c94-109">Example</span></span>  
 <span data-ttu-id="b6c94-110">Aşağıdaki kod CS1943 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b6c94-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
