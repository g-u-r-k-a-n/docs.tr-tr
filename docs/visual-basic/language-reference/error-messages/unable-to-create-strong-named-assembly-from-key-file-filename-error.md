# <a name="bc31026-unable-to-create-strong-named-assembly-from-key-file-filename-error"></a>BC31026: ' ' anahtar dosyasından tanımlayıcı adlı derleme oluşturulamıyor \<filename> : \<error>

Belirtilen anahtar dosyasından tanımlayıcı adlı bir bütünleştirilmiş kod oluşturulamadı.

 **Hata kimliği:** BC31026

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Doğru anahtar dosyasının belirtildiğinden ve başka bir uygulama tarafından kilitlenmediğinden emin olun.
