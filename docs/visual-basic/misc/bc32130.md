# <a name="nullable-types-are-not-supported-because-the-current-version-of-mscorlibdll-does-not-define-the-type-nullableof-t"></a>Geçerli ' mscorlib. dll ' sürümü ' Nullable (Of T) ' türünü tanımlamıyor, null yapılabilir türler desteklenmiyor
Kullanmakta olduğunuz Framework sürümü null yapılabilir değer türlerini desteklemiyor.  
  
 **Hata kimliği:** BC32130
