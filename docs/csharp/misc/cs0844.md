## <a name="example"></a><span data-ttu-id="19a48-112">Örnek</span><span class="sxs-lookup"><span data-stu-id="19a48-112">Example</span></span>  
 <span data-ttu-id="19a48-113">Aşağıdaki kod CS0844 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="19a48-113">The following code generates CS0844:</span></span>

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="19a48-114">`this.num`Sınıf alanına başvurmak için kullanarak hatayı düzeltin</span><span class="sxs-lookup"><span data-stu-id="19a48-114">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
