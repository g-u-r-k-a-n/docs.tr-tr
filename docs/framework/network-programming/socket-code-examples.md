# <a name="socket-code-examples"></a>Yuva Kod Örnekleri
Aşağıdaki kod örnekleri, <xref:System.Net.Sockets.Socket> uzak ağ hizmetlerine bağlanmak için istemci olarak sınıfın nasıl kullanılacağını ve uzak istemcilerden gelen bağlantıları dinlemek için sunucu olarak nasıl kullanılacağını gösterir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Zaman Uyumlu İstemci Yuvası Örneği](synchronous-client-socket-example.md)  
 Sunucuya bağlanan ve <xref:System.Net.Sockets.Socket> sunucudan döndürülen verileri görüntüleyen eşzamanlı istemcinin nasıl uygulanacağını gösterir.  
  
 [Zaman Uyumlu Sunucu Yuvası Örneği](synchronous-server-socket-example.md)  
 İstemciden gelen bağlantıları <xref:System.Net.Sockets.Socket> kabul eden ve istemciden alınan verileri yankılayan eşzamanlı bir sunucunun nasıl uygulanacağını gösterir.  
  
 [Zaman Uyumsuz İstemci Yuvası Örneği](asynchronous-client-socket-example.md)  
 Sunucuya bağlanan ve sunucudan döndürülen verileri görüntüleyen eşzamanlı bir <xref:System.Net.Sockets.Socket> istemcinin nasıl uygulanacağını gösterir.  
  
 [Zaman Uyumsuz Sunucu Yuvası Örneği](asynchronous-server-socket-example.md)  
 İstemciden gelen bağlantıları <xref:System.Net.Sockets.Socket> kabul eden ve istemciden alınan verileri yankılayan eşzamanlı bir sunucunun nasıl uygulanacağını gösterir.
