# <a name="shared-cannot-be-combined-with-specifier-on-a-property-declaration"></a>Özellik bildiriminde ' Shared ' ' ' ile birleştirilemez \<specifier>

Değiştirici, bir `Shared` özellik bildiriminde bu tür bir kombinasyonda geçerli olmayan bir belirticiyle birleştirmeye çalıştınız.  
  
 **Hata kimliği:** BC30502  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirteci kaldırın.
