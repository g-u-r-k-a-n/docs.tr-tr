# <a name="compiler-error-cs0174"></a>Derleyici hatası CS0174
' Base ' başvurusu için temel sınıf gerekir  
  
 Bu hata yalnızca .NET Framework ortak dil çalışma zamanı, `System.Object` sınıfının temel sınıfı olmayan tek sınıf olan kaynak kodunu derlediğinde oluşur.
