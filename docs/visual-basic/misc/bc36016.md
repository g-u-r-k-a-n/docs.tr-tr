# <a name="implicit-variable-variablename-is-invalid-because-of-message"></a>'\<VariableName > ' örtük değişkeni, '\<Message > ' nedeniyle geçersizdir
Hata ayıklama sırasında geçersiz bir örtük değişken oluşturuldu.  
  
 **Hata kimliği:** BC36016  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Neden ve olası çözüm için alıntı yapılan iletiyi inceleyin.  
  
2. Hata devam ederse, koşullar hakkındaki bilgileri toplayın ve Microsoft Ürün Destek Hizmetleri 'ne bildirin.
