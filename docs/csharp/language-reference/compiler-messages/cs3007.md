<span data-ttu-id="d62ed-104">Yalnızca adlandırılmamış dizi türleri farklı olan 'method' tekrar yüklenen yöntemi CLS uyumlu değil</span><span class="sxs-lookup"><span data-stu-id="d62ed-104">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="d62ed-105">Basit bir dizi kullanan aşırı yüklü bir yönteminiz varsa ve yöntem imzaları arasındaki tek fark dizideki öğe türüyse bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="d62ed-105">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="d62ed-106">Bu hatayı önlemek için, pürüzlü bir dizi yerine dikdörtgen bir dizi kullanmayı düşünün; işlev çağrısını belirsizliğini ortadan kaldırmak için ek bir parametre kullanın; aşırı yüklenmiş yöntemlerin birini veya birkaçını yeniden adlandırın; ya da CLS uyumluluğu gerekmiyorsa, <xref:System.CLSCompliantAttribute> özniteliğini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="d62ed-106">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="d62ed-107">CLS uyumluluğu hakkında daha fazla bilgi için bkz. ## <a name="example"></a><span data-ttu-id="d62ed-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="d62ed-108">Example</span></span>  

 <span data-ttu-id="d62ed-109">Aşağıdaki örnek CS3007 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d62ed-109">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
