<span data-ttu-id="66a83-104">' Method ' için en iyi aşırı yüklenmiş yöntem eşleşmesi, başlatıcı öğesi için yanlış imzaya sahip.</span><span class="sxs-lookup"><span data-stu-id="66a83-104">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="66a83-105">Başlatılabilir ekleme, erişilebilir bir örnek yöntemi olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="66a83-105">The initializable Add must be an accessible instance method.</span></span>
