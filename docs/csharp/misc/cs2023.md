# <a name="compiler-warning-level-1-cs2023"></a>Derleyici Uyarısı (düzey 1) CS2023

Bir yanıt dosyası içinde belirtildiğinden/noconfig seçeneği yoksayılıyor  
  
 [-Noconfig](../language-reference/compiler-options/noconfig-compiler-option.md) derleyici seçeneği bir yanıt dosyasında belirtildi; buna izin verilmez.  
  
 Bu uyarı [-nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) seçeneği tarafından bastırılamaz.
