# <a name="error-structurename1-contains-structurename2"></a>\<error>: ' \<structurename1> ', ' \<structurename2> ' içerir

Bir yapı kendisini kendi üyelerinden biri olarak bir tasarladığı zaman dairesel bir yapı tanımı oluşur.  
  
 **Hata kimliği:** BC30293  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yapı üyesinin adını değiştirin.
