# <a name="compiler-error-cs0568"></a><span data-ttu-id="84d71-102">Derleyici hatası CS0568</span><span class="sxs-lookup"><span data-stu-id="84d71-102">Compiler Error CS0568</span></span>
<span data-ttu-id="84d71-103">Yapılar, açık parametresiz oluşturucular içeremez</span><span class="sxs-lookup"><span data-stu-id="84d71-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="84d71-104">Her [yapının](../language-reference/builtin-types/struct.md) zaten nesneyi sıfıra Başlatan parametresiz bir Oluşturucusu vardır.</span><span class="sxs-lookup"><span data-stu-id="84d71-104">Each [struct](../language-reference/builtin-types/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="84d71-105">Bu nedenle, bir yapı için oluşturabileceğiniz oluşturucuların bir veya daha fazla parametre olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="84d71-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="84d71-106">Aşağıdaki örnek CS0568 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="84d71-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
