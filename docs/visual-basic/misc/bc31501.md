# <a name="readonly-attribute-property-propertyfield-cannot-be-the-target-of-an-assignment"></a>' ' ' ReadOnly ' öznitelik özelliği \<propertyfield> bir atamanın hedefi olamaz

Bir öznitelik içindeki bir özelliğe değer atama girişiminde bulunuldu `ReadOnly` .  
  
 **Hata kimliği:** BC31501  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özellik atama ifadesini kaldırın.  
  
2. Geliştirmiş olduğunuz özellikleri kullanıyorsanız, `ReadOnly` veya `Shared` değiştiricilerini öznitelik özelliğinden kaldırın.
