# <a name="systemservicemodelmanualflowthrottlelimitreached"></a>System.ServiceModel.ManualFlowThrottleLimitReached

System.ServiceModel.ManualFlowThrottleLimitReached  
  
## <a name="description"></a>Açıklama  

 Sistem ManualFlowControlLimit kısıtlaması için belirlenen sınıra ulaştı. Kısıtlama değeri, varsa ServiceHost veya InstanceContext üzerindeki ManualFlowControlLimit özelliği değiştirilerek değiştirilebilir.  
  
 Bu izleme, el ile akış denetim sınırı başlangıçta 0 ' a azaltılacağı zaman yayılır. 0 ' a yapılan sonraki değişiklikler izlenmez. Örnek bağlamındaki akış denetim sınırı her bağlam için bir kez izleniyor.
