# <a name="compiler-error-cs1593"></a><span data-ttu-id="7c05e-103">Derleyici hatası CS1593</span><span class="sxs-lookup"><span data-stu-id="7c05e-103">Compiler Error CS1593</span></span>

<span data-ttu-id="7c05e-104">' Del ' temsilcisi ' number ' bağımsız değişkeni almaz</span><span class="sxs-lookup"><span data-stu-id="7c05e-104">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="7c05e-105">Bir [temsilci](../language-reference/builtin-types/reference-types.md) çağrısına geçirilen bağımsız değişkenlerin sayısı, temsilci bildirimindeki parametre sayısıyla kabul etmez.</span><span class="sxs-lookup"><span data-stu-id="7c05e-105">The number of arguments passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="7c05e-106">Aşağıdaki örnek CS1593 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7c05e-106">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
