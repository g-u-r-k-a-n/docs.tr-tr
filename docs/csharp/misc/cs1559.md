# <a name="compiler-error-cs1559"></a>Derleyici hatası CS1559

İçeri aktarıldığından Main yöntemi için ' Object ' kullanılamaz  
  
 [**StartupObject**](../language-reference/compiler-options/advanced.md#mainentrypoint-or-startupobject) derleyici seçeneği için geçersiz bir sınıf belirtildi; sınıf, [Main](../programming-guide/main-and-command-args/index.md) yöntemi için bir konum olarak kullanılamaz.
