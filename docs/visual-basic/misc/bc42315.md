# <a name="xml-comment-tag-returns-is-not-permitted-on-a-declare-sub-language-element"></a>' Declare Sub ' dil öğesinde ' döndürüyor ' XML açıklama etiketine izin verilmez

' Declare Sub ' dil öğesinde ' döndürüyor ' XML açıklama etiketine izin verilmez. XML açıklaması yoksayılacak.  
  
 Bir bildirimin XML yorumu `Declare Sub` , bir `<` dönüş etiketi içeremez `>` .  
  
 **Hata kimliği:** BC42315  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Desteklenmeyen `<` dönüşler `>` etiketini kaldırın.
