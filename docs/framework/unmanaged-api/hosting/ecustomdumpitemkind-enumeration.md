# <a name="ecustomdumpitemkind-enumeration"></a>ECustomDumpItemKind Numaralandırması
[CustomDumpItem](../../../../docs/framework/unmanaged-api/hosting/customdumpitem-structure.md) yapısının gelecekteki uzantısı için ayrılmıştır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
typedef enum {  
    DUMP_ITEM_None = 0  
} ECustomDumpItemKind;  
```  
  
## <a name="members"></a>Üyeler  
  
|Üye|Açıklama|

|`DUMP_ITEM_None`|Daha sonraki kullanımlar için ayrılmıştır.|
