|||  
|-|-|  
|ID|5403|  
|Anahtar sözcükler|Güvenlik|  
|Düzey|Ayrıntılı|

 SecurityToken (' %1 ' türü ve ' %2 ' kimliği) doğrulaması başarılı oldu.
