# <a name="compiler-error-cs0186"></a><span data-ttu-id="06859-103">Derleyici hatası CS0186</span><span class="sxs-lookup"><span data-stu-id="06859-103">Compiler Error CS0186</span></span>

<span data-ttu-id="06859-104">Bu bağlamda null kullanımı geçerli değil</span><span class="sxs-lookup"><span data-stu-id="06859-104">Use of null is not valid in this context</span></span>  
  
 <span data-ttu-id="06859-105">Aşağıdaki örnek CS0186 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="06859-105">The following sample generates CS0186:</span></span>  
  
```csharp  
// CS0186.cs  
using System;  
using System.Collections;  
  
class MyClass
{  
   static void Main()
   {  
      // Each of the following lines generates CS0186:  
      foreach (int i in null) {}   // CS0186  
      foreach (int i in (IEnumerable) null) { };   // CS0186  
   }  
}  
```
