# <a name="bc36548-cannot-convert-anonymous-type-to-an-expression-tree-because-a-property-of-the-type-is-used-to-initialize-another-property"></a><span data-ttu-id="2c854-103">BC36548: başka bir özelliği başlatmak için türün bir özelliği kullanıldığından, anonim tür bir ifade ağacına dönüştürülemez</span><span class="sxs-lookup"><span data-stu-id="2c854-103">BC36548: Cannot convert anonymous type to an expression tree because a property of the type is used to initialize another property</span></span>

<span data-ttu-id="2c854-104">Anonim türün başka bir özelliğini başlatmak için anonim türdeki bir özellik kullanıldığında, derleyici anonim bir türün bir ifade ağacına dönüştürülmesini kabul etmez.</span><span class="sxs-lookup"><span data-stu-id="2c854-104">The compiler does not accept conversion of an anonymous type to an expression tree when one property of the anonymous type is used to initialize another property of the anonymous type.</span></span> <span data-ttu-id="2c854-105">Örneğin, aşağıdaki kodda, `Prop1` başlatma listesinde bildirilmiştir ve için başlangıç değeri olarak kullanılır `Prop2` .</span><span class="sxs-lookup"><span data-stu-id="2c854-105">For example, in the following code, `Prop1` is declared in the initialization list and then used as the initial value for `Prop2`.</span></span>

<span data-ttu-id="2c854-106">**Hata kimliği:** BC36548</span><span class="sxs-lookup"><span data-stu-id="2c854-106">**Error ID:** BC36548</span></span>

## <a name="example"></a><span data-ttu-id="2c854-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="2c854-107">Example</span></span>

```vb
Module M2

    Sub ExpressionExample(Of T)(x As Expressions.Expression(Of Func(Of T)))
    End Sub

    Sub Main()
        ' The following line causes the error.
        ExpressionExample(Function() New With {.Prop1 = 2, .Prop2 = .Prop1})
    End Sub
End Module
```

## <a name="to-correct-this-error"></a><span data-ttu-id="2c854-108">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="2c854-108">To correct this error</span></span>

<span data-ttu-id="2c854-109">İçin başlangıç değerini `Prop1` yerel bir değişkene atayın.</span><span class="sxs-lookup"><span data-stu-id="2c854-109">Assign the initial value for `Prop1` to a local variable.</span></span> <span data-ttu-id="2c854-110">`Prop1` `Prop2` Aşağıdaki kodda gösterildiği gibi, bu değişkeni hem hem de öğesine atayın:</span><span class="sxs-lookup"><span data-stu-id="2c854-110">Assign that variable to both `Prop1` and `Prop2`, as shown in the following code:</span></span>

```vb
Sub Main()
    Dim temp = 2
    ExpressionExample(Function() New With {.Prop1 = temp, .Prop2 = temp})
End Sub
```
