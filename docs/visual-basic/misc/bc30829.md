# <a name="get-statements-are-no-longer-supported"></a>' Get ' deyimleri artık desteklenmiyor
`Get`deyimler artık desteklenmiyor. Dosya g/ç işlevselliği `Microsoft.VisualBasic` ad alanında kullanılabilir.  
  
 `Get`dosya işlemleri için desteklenmez ve yalnızca özellik yordamı sözdiziminde kullanılabilir.  
  
 **Hata kimliği:** BC30829  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `System.IO`, `FileSystemObject` Ve Visual Basic çalışma zamanı işlevlerinin üyelerini kullanarak dosya işlemleri gerçekleştirin.
