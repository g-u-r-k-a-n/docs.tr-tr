 <span data-ttu-id="9dfb1-104">Bu hata, ikinci bir ad alanı içinde var olduğunu söylediği bir türe atıfta bulunan bir ad alanı içindeki bir başvurudan kaynaklanabilir, ancak tür yok.</span><span class="sxs-lookup"><span data-stu-id="9dfb1-104">This error can be caused by a reference inside one namespace referring to a type that it says exists inside a second namespace, but the type does not exist.</span></span> <span data-ttu-id="9dfb1-105">Örneğin, mydll.dll bu `A` tür yourdll.dll içinde var diyor, ama böyle bir tür yourdll.dll içinde var.</span><span class="sxs-lookup"><span data-stu-id="9dfb1-105">For example, mydll.dll says that type `A` exists inside yourdll.dll, but no such type exists inside yourdll.dll.</span></span> <span data-ttu-id="9dfb1-106">Bu hatanın olası bir nedeni, kullandığınız yourdll.dll sürümünün `A` çok eski olması ve henüz tanımlanmamış olmasıdır.</span><span class="sxs-lookup"><span data-stu-id="9dfb1-106">One possible cause of this error is that the version of yourdll.dll you are using is too old and `A` has not yet been defined.</span></span>  
  
 <span data-ttu-id="9dfb1-107">Aşağıdaki örnek CS1684 üretir.</span><span class="sxs-lookup"><span data-stu-id="9dfb1-107">The following sample generates CS1684.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9dfb1-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="9dfb1-108">Example</span></span>  
  
```csharp

// compile with: /target:library /keyfile:CS1684.key  
public class A {  
   public void Test() {}  
}  
  
public class C2 {}  
```  
  
## <a name="example"></a><span data-ttu-id="9dfb1-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="9dfb1-109">Example</span></span>  
  
```csharp

   public static A GetA() { return new A(); }

}  
```
