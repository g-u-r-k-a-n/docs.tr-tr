# <a name="datamemberlisteditor-class"></a>Datamemberlistedıtor sınıfı

`DataSource`, `DataSource` Öğesinden değer kaynağını seçmek üzere nesnenin tüm özelliklerini listeleyerek, veri bağlantılı nesnelerin (null olmayan bir özelliğine sahip nesneler) düzenlenmesine yönelik açılan bir kullanıcı arabirimi sağlar.  
  
## <a name="syntax"></a>Syntax
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberListEditor`Sınıfı dahili ve doğrudan kodunuzda kullanılması amaçlıyordu.
>
> Microsoft, bu sınıfın herhangi bir koşulda bir üretim uygulamasında kullanımını desteklemez.
