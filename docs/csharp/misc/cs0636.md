# <a name="compiler-error-cs0636"></a><span data-ttu-id="330c1-102">Derleyici Hatası CS0636</span><span class="sxs-lookup"><span data-stu-id="330c1-102">Compiler Error CS0636</span></span>
<span data-ttu-id="330c1-103">FieldOffset özniteliği yalnızca StructLayout(LayoutKind.Explicit) ile işaretlenmiş türlerin üyelerine yerleştirilebilir</span><span class="sxs-lookup"><span data-stu-id="330c1-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="330c1-104">**Alan Ofset** özniteliği ile işaretlenmiş herhangi bir üye içeriyorsa, yapı bildiriminde **StructLayout(LayoutKind.Explicit)** özniteliğini kullanmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="330c1-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="330c1-105">Daha fazla bilgi için [FieldOffset'e](xref:System.Runtime.InteropServices.FieldOffsetAttribute)bakın.</span><span class="sxs-lookup"><span data-stu-id="330c1-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="330c1-106">Aşağıdaki örnek CS0636 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="330c1-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
