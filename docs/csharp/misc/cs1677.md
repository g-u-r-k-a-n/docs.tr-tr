 <span data-ttu-id="1bdb9-104">Bu hata, anonim bir yöntemde parametre türü değiştiricisi, yöntemi atamak istediğiniz temsilcinin bildiriminde kullanılan ile eşleşmezse oluşur.</span><span class="sxs-lookup"><span data-stu-id="1bdb9-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1bdb9-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="1bdb9-105">Example</span></span>  

 <span data-ttu-id="1bdb9-106">Aşağıdaki örnek CS1677 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1bdb9-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
