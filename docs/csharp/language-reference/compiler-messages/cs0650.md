## <a name="example"></a><span data-ttu-id="13d83-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="13d83-109">Example</span></span>

 <span data-ttu-id="13d83-110">Aşağıdaki örnek kod CS0650 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="13d83-110">The following example code generates CS0650.</span></span>  
  
```csharp  
// CS0650.cs  
public class MyClass  
{  
   public static void Main()  
   {  
// Generates CS0650. Incorrect array declaration syntax:  
      int myarray[2];
  
      // Correct declaration.  
      int[] myarray2;  
  
      // Declaration and initialization in one statement  
      int[] myArray3= new int[2] {1,2}  
  
      // Access an array element.  
      myarray3[0] = 0;  
    }  
}  
```  
  
 <span data-ttu-id="13d83-111">Aşağıdaki örnek, `fixed` sabit boyutlu bir arabellek oluştururken anahtar sözcüğünün nasıl kullanılacağını gösterir:</span><span class="sxs-lookup"><span data-stu-id="13d83-111">The following example shows how to use the `fixed` keyword when you create a fixed size buffer:</span></span>  
  
```csharp  
// This code must appear in an unsafe block.
public struct MyArray
{  
    public fixed char pathName[128];  
}  
```
