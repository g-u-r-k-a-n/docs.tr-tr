# <a name="keyword-is-not-valid-within-a-module"></a>' \<keyword> ' bir modül içinde geçerli değil

Veya gibi sınıf örnekleriyle ilgili bir anahtar sözcük `Me` `MyBase` bir modül içinde kullanılır. Modüller devralma veya örneklere sahip değildir.  
  
 **Hata kimliği:** BC32001  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Anahtar sözcüğü kullanan kod sınıf örnekleri içeriyorsa, sınıf uygulamasına taşıyın.  
  
- Anahtar sözcüğünü kullanan kod modül için geçerliyse, geçersiz anahtar sözcüğünü kaldırın.
