   }

## <a name="example"></a><span data-ttu-id="be662-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="be662-109">Example</span></span>  
 <span data-ttu-id="be662-110">Aşağıdaki örnek CS0559 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="be662-110">The following sample generates CS0559.</span></span>  
  
```csharp  
// CS0559_b.cs  
// compile with: /target:library  
public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559

}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```
