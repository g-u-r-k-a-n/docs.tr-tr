# <a name="argument-access-is-not-valid-valid-values-for-append-mode-are-openaccesswrite-and-openaccessdefault"></a>' Access ' bağımsız değişkeni geçerli değil. Append modu için geçerli değerler şunlardır ' OpenAccess. Write ' ve ' OpenAccess. Default '

Ekleme modunda bir erişim değeri geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Değeri ya da olarak değiştirin `OpenAccess.Write` `OpenAccess.Default` .
