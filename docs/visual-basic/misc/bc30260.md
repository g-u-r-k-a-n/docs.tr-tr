# <a name="name-is-already-declared-as-declaration-in-this-declarationspace"></a>' \<name> ' Bu öğesinde zaten ' ' olarak bildiriliyor \<declaration>\<declarationspace>

İki programlama öğesi aynı bildirim alanında (bir sınıf, modül, arabirim veya yapı) aynı adla bildirilmiştir.  
  
 **Hata kimliği:** BC30260  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bildirimlerden birini farklı bir bildirim alanına taşıyın veya belirtilen öğelerden birinin adını değiştirin.
