# <a name="compiler-error-cs0601"></a><span data-ttu-id="f0b86-103">Derleyici hatası CS0601</span><span class="sxs-lookup"><span data-stu-id="f0b86-103">Compiler Error CS0601</span></span>

<span data-ttu-id="f0b86-104">DllImport özniteliği ' static ' ve ' extern ' olarak işaretlenmiş bir yöntemde belirtilmelidir</span><span class="sxs-lookup"><span data-stu-id="f0b86-104">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="f0b86-105">`DllImport`Öznitelik, doğru erişim anahtar sözcüklere sahip olmayan bir yöntemde kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="f0b86-105">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="f0b86-106">Aşağıdaki örnek CS0601 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f0b86-106">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
