# <a name="specifier-is-duplicated"></a>Belirtici yineleniyor
Bir anahtar sözcük aynı bildirimde birden çok kez görünür.  
  
 **Hata kimliği:** BC30178  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Anahtar sözcüğünün yinelenen görünümlerini bildirimden kaldırın.
