# <a name="compiler-error-cs0535"></a><span data-ttu-id="34733-103">Derleyici hatası CS0535</span><span class="sxs-lookup"><span data-stu-id="34733-103">Compiler Error CS0535</span></span>

 <span data-ttu-id="34733-105">Bir [arabirimden](../language-reference/keywords/interface.md)türetilmiş bir [sınıf](../language-reference/keywords/class.md) , ancak sınıf bir veya daha fazla arabirimin üyesini gerçekleştirmedi.</span><span class="sxs-lookup"><span data-stu-id="34733-105">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="34733-106">Bir sınıf, türetilen ya da başka bir şekilde bildirildiği tüm arabirimlerin üyelerini uygulamalıdır `abstract` .</span><span class="sxs-lookup"><span data-stu-id="34733-106">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="34733-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="34733-107">Example</span></span>  

 <span data-ttu-id="34733-108">Aşağıdaki örnek CS0535 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="34733-108">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
