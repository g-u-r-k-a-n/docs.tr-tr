# <a name="dialog-boxes-in-windows-forms"></a>Windows Forms'ta İletişim Kutuları
İletişim kutuları, kullanıcıyla etkileşime geçmek ve bilgi almak için kullanılır. Basit koşullarda, bir iletişim kutusu <xref:System.Windows.Forms.FormBorderStyle> numaralandırma özelliği `FixedDialog`olarak ayarlanmış bir formdur. Visual Studio 'daki Windows Form Tasarımcısı kullanarak kendi özel iletişim kutularınızı oluşturabilirsiniz. İletişim kutularını özel gereksinimlerinize göre özelleştirmek için `Label`, `Textbox`ve `Button` gibi denetimler ekleyin. .NET Framework Ayrıca, kendi uygulamalarınız için uyarlayabileceğiniz **dosya açma** ve ileti kutuları gibi önceden tanımlanmış iletişim kutularını da içerir. Daha fazla bilgi için bkz. [Iletişim kutusu denetimleri ve bileşenleri](./controls/dialog-box-controls-and-components-windows-forms.md).
