# <a name="xml-attribute-attribute1-must-appear-before-xml-attribute-attribute2"></a>XML özniteliği ' Attribute1 ', ' attribute2 ' XML özniteliğinden önce gelmelidir

XML belgesi değişmez değerindeki öznitelikler geçersiz bir sırada belirtildi. XML belgesi değişmez değeri için geçerli öznitelik sırası, XML 1,0 belirtimini temel alır. Örneğin, `encoding` BIR XML belgesi değişmez değerinin özniteliği hemen arkasından gelmelidir `version` .  
  
 **Hata kimliği:** BC31157  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- XML belgesi değişmez değeri için öznitelik sırasını, XML 1,0 belirtiminde belirtilen yönergeleri kullanarak güncelleştirin.
