- <span data-ttu-id="f3bd9-115">[Azure SQL Veritabanı](https://azure.microsoft.com/services/sql-database/)gibi uzak ilişkisel veritabanları, Azure [Cosmos DB](https://docs.microsoft.com/azure/cosmos-db/introduction)gibi NoSQL veritabanları veya [Redis](https://redis.io/)gibi önbellek hizmetleri.</span><span class="sxs-lookup"><span data-stu-id="f3bd9-115">Remote relational databases like [Azure SQL Database](https://azure.microsoft.com/services/sql-database/), NoSQL databases like [Azure Cosmos DB](https://docs.microsoft.com/azure/cosmos-db/introduction), or cache services like [Redis](https://redis.io/).</span></span>

<span data-ttu-id="f3bd9-116">Docker konteynerinden:</span><span class="sxs-lookup"><span data-stu-id="f3bd9-116">From the Docker container:</span></span>
