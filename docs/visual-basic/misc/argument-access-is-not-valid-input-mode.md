# <a name="argument-access-is-not-valid-valid-values-for-input-mode-are-openaccessread-and-openaccessdefault"></a>' Access ' bağımsız değişkeni geçerli değil. Giriş modu için geçerli değerler şunlardır ' OpenAccess. Read ' ve ' OpenAccess. Default '

Modda bir erişim değeri geçerli değil `Input` .  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değeri ya da olarak değiştirin `OpenAccess.Read` `OpenAccess.Default` .
