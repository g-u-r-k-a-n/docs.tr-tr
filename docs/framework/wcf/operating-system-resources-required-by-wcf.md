# <a name="operating-system-resources-required-by-wcf"></a>WCF Tarafından Gerektirilen İşletim Sistemi Kaynakları
Windows Communication Foundation (WCF), işletim sistemi tarafından çalışması için sunulan çeşitli kaynaklara bağlıdır. Aşağıdaki tabloda bu kaynaklar listelenmektedir.  
  
|Kaynak|Açıklama|  
|--------------|-----------------|  
|Microsoft Dağıtılmış İşlem Düzenleyicisi (MSDTC)|OleTx işlemlerini desteklemek için gereklidir.|  
|Internet Information Services (IIS)|Uygulamanızı barındırmak için IIS kullanmak istiyorsanız gereklidir.|  
|Windows Işlem etkinleştirme hizmeti (WAS)|Uygulamanızı barındırmak için kullanmak istiyorsanız gereklidir.|
