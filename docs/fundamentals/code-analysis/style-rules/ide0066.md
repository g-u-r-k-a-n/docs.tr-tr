### <a name="example"></a><span data-ttu-id="6727e-129">Örnek</span><span class="sxs-lookup"><span data-stu-id="6727e-129">Example</span></span>

```csharp
// csharp_style_prefer_switch_expression = true
return x switch
{
    1 => 1 * 1,
    2 => 2 * 2,
    _ => 0,
};

// csharp_style_prefer_switch_expression = false
switch (x)
{
    case 1:
        return 1 * 1;
    case 2:
        return 2 * 2;
    default:
        return 0;
}
```
