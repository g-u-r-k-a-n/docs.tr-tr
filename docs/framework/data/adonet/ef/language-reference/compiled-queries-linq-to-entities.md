<span data-ttu-id="4eb4d-103">Varlık Çerçevesi'nde yapısal olarak benzer sorguları birçok kez yürüten bir uygulamanız olduğunda, sorguyu bir kez derleyerek ve farklı parametrelerle birkaç kez çalıştırarak performansı sık sık artırabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-103">When you have an application that executes structurally similar queries many times in the Entity Framework, you can frequently increase performance by compiling the query one time and executing it several times with different parameters.</span></span> <span data-ttu-id="4eb4d-104">Örneğin, bir uygulamanın belirli bir şehirdeki tüm müşterileri alması gerekebilir; şehir bir formda kullanıcı tarafından çalışma zamanında belirtilir.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-104">For example, an application might have to retrieve all the customers in a particular city; the city is specified at runtime by the user in a form.</span></span> <span data-ttu-id="4eb4d-105">LINQ to Varlıklar, bu amaçla derlenmiş sorguları kullanmayı destekler.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-105">LINQ to Entities supports using compiled queries for this purpose.</span></span>  
  
 <span data-ttu-id="4eb4d-106">.NET Framework 4.5 ile başlayarak LINQ sorguları otomatik olarak önbelleğe alınır.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-106">Starting with the .NET Framework 4.5, LINQ queries are cached automatically.</span></span> <span data-ttu-id="4eb4d-107">Ancak, daha sonraki yürütmelerde bu maliyeti azaltmak için derlenmiş LINQ sorgularını kullanmaya devam edebilirsiniz ve derlenen sorgular otomatik olarak önbelleğe alınmış LINQ sorgularından daha verimli olabilir.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-107">However, you can still use compiled LINQ queries to reduce this cost in later executions and compiled queries can be more efficient than LINQ queries that are automatically cached.</span></span> <span data-ttu-id="4eb4d-108">`Enumerable.Contains` İşleçteni bellek içi koleksiyonlara uygulayan VARLıKLARa LINQ sorgularının otomatik olarak önbelleğe alınmadığını unutmayın.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-108">Note that LINQ to Entities queries that apply the `Enumerable.Contains` operator to in-memory collections are not automatically cached.</span></span> <span data-ttu-id="4eb4d-109">Ayrıca derlenmiş LINQ sorgularında bellek içi koleksiyonları parametrelendirmeye izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-109">Also parameterizing in-memory collections in compiled LINQ queries is not allowed.</span></span>

class="sxs-lookup"><span data-stu-id="4eb4d-114">The query compiles once during only the first execution.</span></span> <span data-ttu-id="4eb4d-115">Derleme sırasında sorgu için ayarlanan birleştirme  <span data-ttu-id="4eb4d-118"><xref:System.Data.Objects.CompiledQuery>'ın `Compile` derletir dediği LINQ- Varlıklar sorgu ifadesi, `Func` <xref:System.Func%605>genel temsilcilerden biri tarafından temsil edilir.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-118">The LINQ to Entities query expression that the <xref:System.Data.Objects.CompiledQuery>'s `Compile` method compiles is represented by one of the generic `Func` delegates, such as <xref:System.Func%605>.</span></span> <span data-ttu-id="4eb4d-119">En fazla, sorgu ifadesi bir `ObjectContext` parametre, bir dönüş parametresi ve 16 sorgu parametrelerini kapsülleyebilir.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-119">At most, the query expression can encapsulate an `ObjectContext` parameter, a return parameter, and 16 query parameters.</span></span> <span data-ttu-id="4eb4d-120">16'dan fazla sorgu parametresi gerekiyorsa, özellikleri sorgu parametrelerini temsil eden bir yapı oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-120">If more than 16 query parameters are required, you can create a structure whose properties represent query parameters.</span></span> <span data-ttu-id="4eb4d-121">Daha sonra özellikleri ayarladıktan sonra sorgu ifadesinde yapı daki özellikleri kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="4eb4d-121">You can then use the properties on the structure in the query expression after you set the properties.</span></span>
