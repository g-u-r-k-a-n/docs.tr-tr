 <span data-ttu-id="bc532-105">Özellik olmayan bir veri türünü [özellik](../programming-guide/classes-and-structs/properties.md)olarak geçersiz kılma girişiminde bulunuldu; buna izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="bc532-105">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

   public int i;  
}
