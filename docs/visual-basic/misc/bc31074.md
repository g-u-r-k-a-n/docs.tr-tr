# <a name="delegate-constructor-for-type-typename-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Temsilcilerin uygulanması için gerekli olan ' ' türü için temsilci Oluşturucu \<typename> bulunamıyor

Türü içeren derleme hasarlı veya geçersiz olabilir.  
  
 **Hata kimliği:** BC31074  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Bu türü içeren derlemenin geçerli olduğunu doğrulayın.  
  
2. Hatanın yinelenip yinelenmeyeceğini görmek için programı yeniden derleyin.  
  
3. Hata devam ederse, çalışmanızı kaydedin ve Visual Studio 'Yu yeniden başlatın.
