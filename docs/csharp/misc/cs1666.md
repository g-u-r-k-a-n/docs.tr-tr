# <a name="compiler-error-cs1666"></a><span data-ttu-id="ee252-102">Derleyici hatası CS1666</span><span class="sxs-lookup"><span data-stu-id="ee252-102">Compiler Error CS1666</span></span>

<span data-ttu-id="ee252-103">Sabitlenmemiş ifadelerde bulunan sabit boyutlu arabellekleri kullanamazsınız.</span><span class="sxs-lookup"><span data-stu-id="ee252-103">You cannot use fixed size buffers contained in unfixed expressions.</span></span> <span data-ttu-id="ee252-104">Fixed ifadesini kullanmayı deneyin.</span><span class="sxs-lookup"><span data-stu-id="ee252-104">Try using the fixed statement.</span></span>  
  
 <span data-ttu-id="ee252-105">Sabit boyutlu arabelleği sabit bir sınıf içeren bir ifadede kullanırsanız bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="ee252-105">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="ee252-106">Çalışma zamanı, sabit boyutlu arabellek kullanılırken hatalara yol açabilecek şekilde, bellek erişimini iyileştirmek için sabit bir sınıfın etrafında taşınmasını ücretsizdir.</span><span class="sxs-lookup"><span data-stu-id="ee252-106">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="ee252-107">Bu hatayı önlemek için, `fixed` deyimindeki anahtar sözcüğünü kullanın.</span><span class="sxs-lookup"><span data-stu-id="ee252-107">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ee252-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="ee252-108">Example</span></span>  

 <span data-ttu-id="ee252-109">Aşağıdaki örnek CS1666 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="ee252-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);
   }
}  
```
