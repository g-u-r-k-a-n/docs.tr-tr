# <a name="custom-event-must-end-with-a-matching-end-event"></a>' Özel olay ', eşleşen bir ' End Event ' ile bitmelidir

Bir `Custom Event` bildirimin ifadesiyle bitmesi gerekir `End Event` .  
  
 **Hata kimliği:** BC31114  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Custom Event`Bildirimin bir ifadesiyle bitmediğini denetleyin `End Event` .
