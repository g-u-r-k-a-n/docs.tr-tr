# <a name="wcf-directive-syntax"></a>WCF Yönerge Söz Dizimi
Derleyicileri yönlendirmek için. svc dosyalarında Windows Communication Foundation (WCF) yönergeleri tarafından kullanılan ayarları belirtir. Her yönerge, bu yönergeye özgü bir veya daha fazla öznitelik (değerleriyle eşleştirilmiş) içerebilir. WCF yalnızca [ \@ ServiceHost](servicehost.md) yönergesine sahiptir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [@ServiceHost](servicehost.md)  
 . Svc derleyicisi tarafından kullanılan sayfaya özgü öznitelikleri tanımlar. Yalnızca. svc dosyalarına dahil edilebilir.
