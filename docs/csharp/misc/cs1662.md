# <a name="compiler-error-cs1662"></a><span data-ttu-id="a74a5-103">Derleyici hatası CS1662</span><span class="sxs-lookup"><span data-stu-id="a74a5-103">Compiler Error CS1662</span></span>

<span data-ttu-id="a74a5-104">Bloktaki dönüş türlerinden bazıları örtük olarak temsilci dönüş türüne dönüştürülemediğinden, anonim yöntem bloğu ' temsilci türü ' temsilci türüne dönüştürülemiyor</span><span class="sxs-lookup"><span data-stu-id="a74a5-104">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="a74a5-105">Anonim yöntem bloğunun Return ifadesinde, temsilcinin dönüş türüne örtük olarak dönüştürülebilir bir tür varsa bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="a74a5-105">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="a74a5-106">Aşağıdaki örnek CS1662 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a74a5-106">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```
