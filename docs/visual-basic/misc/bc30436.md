# <a name="members-in-a-module-cannot-be-declared-specifier"></a>Modül içindeki Üyeler ' ' olarak bildirilemez \<specifier>

Bir deyimin içindeki bir üyede geçerli olmayan bir tanımlayıcı kullandınız `Module` . Modüller hiçbir şekilde örneklenemez, devralmayı desteklemez ve arabirimler uygulayamaz.  
  
 **Hata kimliği:** BC30436  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirteci kaldırın.
