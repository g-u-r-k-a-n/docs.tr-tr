# <a name="install-the-azure-cli"></a>Azure CLI'yi yükleme

Azure portalına ek olarak, Azure Azure kaynaklarını oluşturmak ve yönetmek için Azure [CLI](/cli/azure/) 'yi bir komut satırı aracı olarak da sunmaktadır. Azure CLı, verimlilik, yinelenebilirlik ve yinelenen görevleri betikleştirime avantajları sunar.  

Uygulamada çoğu geliştirici hem Azure portalını hem de Azure CLı 'yi kullanır. Azure Portal 'ın yeni hizmetleri araştırırken ve Azure hesabınızdaki tüm kaynaklara genel bir bakış edinmesinde yararlı olduğu durumlarda, çoğu geliştirici Azure CLı 'yi daha hızlı ve daha verimli olacak şekilde bulur.  Azure CLı, genellikle Azure portalında birden çok adımı alan tek bir komutta gerçekleştirilebilir.  Ayrıca, Azure CLı komutları bir dosyaya kaydedilemediğinden, geliştiriciler yinelenen görevlerin her seferinde aynı şekilde çalışmasını güvence altına alabilir.

Azure CLı, Windows, macOS ve Linux 'ta kullanılabilir.

### <a name="azure-cloud-shell"></a>Azure Cloud Shell

Azure CLı 'yi de Azure Cloud Shell de kullanabilirsiniz [https://shell.azure.com](https://shell.azure.com) .  Azure Cloud Shell, Azure kaynaklarını yönetmeye yönelik tam işlevli, tarayıcı tabanlı bir kabuktur.  Azure Cloud Shell, bir komut satırı ortamına ihtiyacınız olduğunda, ancak Azure CLı 'yi yükleyemedik bir cihazda çalışırken yararlıdır.
