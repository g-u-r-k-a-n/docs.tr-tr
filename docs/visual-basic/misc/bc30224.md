# <a name="is-expected"></a>' SIS ' bekleniyor
Bir `TypeOf` işleç bir yan tümce olmadan meydana gelir `Is` .  
  
 **Hata kimliği:** BC30224  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Is`İşlecini izleyen bir yan tümce ekleyin `TypeOf` ; Örneğin, `TypeOf MyControl Is ComboBox` .
