# <a name="unable-to-load-dll-filename-error"></a>' ' DLL dosyası yüklenemiyor \<filename> : \<error>

Bu proje tarafından başvurulan bir DLL yüklenemedi.  
  
 **Hata kimliği:** BC31013  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Dosyanın belirtilen konumda olduğunu ve başka bir uygulama tarafından kilitlenmediğini doğrulayın.
