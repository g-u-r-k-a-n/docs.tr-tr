## <a name="findresponse"></a><span data-ttu-id="35795-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="35795-140">FindResponse</span></span>

<span data-ttu-id="35795-141"><xref:System.ServiceModel.Discovery.FindResponse><xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A>, ağda eşleşen hizmetler tarafından gönderilen yanıtları içeren bir koleksiyon özelliğine sahiptir.</span><span class="sxs-lookup"><span data-stu-id="35795-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="35795-142">Hiçbir hizmet yanıtlanmadan, koleksiyon boştur.</span><span class="sxs-lookup"><span data-stu-id="35795-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="35795-143">Bir veya daha fazla hizmet yanıtladıysanız, her yanıt bir nesne içinde depolanır <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> ve bu hizmetle ilgili adres, sözleşme ve bazı ek bilgiler bulunur.</span><span class="sxs-lookup"><span data-stu-id="35795-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>

<span data-ttu-id="35795-144">Aşağıdaki örnek, kodda bulma işleminin nasıl gerçekleştirileceğini gösterir.</span><span class="sxs-lookup"><span data-stu-id="35795-144">The following example shows how to perform a find operation in code.</span></span>

```csharp
// Create DiscoveryClient
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());

// Create FindCriteria
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));
findCriteria.Duration = TimeSpan.FromSeconds(10);

// Find ICalculatorService endpoints
FindResponse findResponse = discoveryClient.Find(findCriteria);

Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)
```
