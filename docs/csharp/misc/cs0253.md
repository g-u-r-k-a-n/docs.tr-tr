# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="85b0b-103">Derleyici Uyarısı (düzey 2) CS0253</span><span class="sxs-lookup"><span data-stu-id="85b0b-103">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="85b0b-104">Olası istenmeden yapılan başvuru karşılaştırması; değer karşılaştırması almak için sağ tarafı ' Type ' türüne dönüştürün</span><span class="sxs-lookup"><span data-stu-id="85b0b-104">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="85b0b-105">Derleyici bir başvuru karşılaştırması yapıyor.</span><span class="sxs-lookup"><span data-stu-id="85b0b-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="85b0b-106">Dizelerin değerini karşılaştırmak istiyorsanız, ifadenin sağ tarafını olarak atayın `type` .</span><span class="sxs-lookup"><span data-stu-id="85b0b-106">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="85b0b-107">Aşağıdaki örnek CS0253 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="85b0b-107">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
