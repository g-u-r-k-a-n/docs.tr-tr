# <a name="compiler-error-cs0730"></a>Derleyici hatası CS0730

' Type ' öğesinin iç içe yerleştirilmiş bir türü olduğundan ' Type ' türü iletilemiyor  
  
 Bu hata, iç içe yerleştirilmiş bir sınıfı iletmeyi denediğinizde oluşturulur.  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS0730 oluşturur. İki kaynak dosyadan oluşur. İlk olarak, kitaplık dosyasını derleyin `CS0730a.cs` ve `CS0730.cs` kitaplık dosyasına başvuran dosyayı derleyin.  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
