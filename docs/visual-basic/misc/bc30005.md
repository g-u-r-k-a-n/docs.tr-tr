# <a name="reference-required-to-assembly-assemblyname-containing-the-definition-for-event-eventname"></a>' ' \<assemblyname> Olayının tanımını içeren ' ' derlemesine başvuru gerekiyor \<eventname>

' `assemblyname` <> ' olayının tanımını içeren ' <> ' derlemesine başvuru gerekiyor `eventname` . Projenize bir başvuru ekleyin.  
  
 Bu olay, projenizde doğrudan başvurulmayan bir dinamik bağlantı kitaplığı (DLL) veya bütünleştirilmiş kodda tanımlanır. Visual Basic derleyici, olayın birden fazla DLL veya derlemede tanımlanması durumunda karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata kimliği:** BC30005  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Başvurulmayan DLL veya derlemenin adını Proje başvurularında ekleyin.
