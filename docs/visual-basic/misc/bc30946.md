# <a name="stop-request-is-pending"></a>Durdurma isteği bekliyor

Visual Studio hata ayıklayıcısında bir ifade bir yordam çağrısını belirtir, ancak iş parçacığını durdurmak için bir istek vardır.  
  
 Hata ayıklayıcı etkin olmayan bir iş parçacığında bir yordamı çağırmayı denemez.  
  
 **Hata kimliği:** BC30946  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Mümkünse, yeniden oluşmasını engellemek için iş parçacığını durdurma isteğinin kaynağını saptayın.  
  
- Sonlandırın ve hata ayıklamayı yeniden başlatın.
