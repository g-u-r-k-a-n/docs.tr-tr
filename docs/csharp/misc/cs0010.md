# <a name="compiler-error-cs0010"></a>Derleyici hatası CS0010

Beklenmeyen önemli hata--' Error '.  
  
 Derlemeyi durdurmak için tamamen beklenmeyen bir durum oluştuğunda bu hata oluşturulur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Başka bir projeyi yeniden derleyin. Aynı hatayı alırsanız, Visual Studio 'Yu yeniden yüklemeyi deneyin. Bu hatayı alırsanız, Microsoft 'a bir hata raporu gönderin.
