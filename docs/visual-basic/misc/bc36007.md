# <a name="end-using-must-be-preceded-by-a-matching-using"></a>' End using ' öncesinde eşleşen bir ' Using ' olmalıdır

`End Using`Kendisinden önce eşleşen bildirim olmadan bir ifade görüntülenir `Using` .  
  
 **Hata kimliği:** BC36007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Gereksiz ise, `End Using` ifadeyi kaldırın.  
  
- Eksik bir [using ifadesini](../language-reference/statements/using-statement.md) , varsa belirtin.  
  
- `End Using`İfadeyi koddaki uygun yere taşıyın.
