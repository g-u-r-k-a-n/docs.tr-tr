# <a name="compiler-error-cs0557"></a><span data-ttu-id="b6b3e-103">Derleyici hatası CS0557</span><span class="sxs-lookup"><span data-stu-id="b6b3e-103">Compiler Error CS0557</span></span>
<span data-ttu-id="b6b3e-104">' Class ' türünde yinelenen kullanıcı tanımlı dönüştürme</span><span class="sxs-lookup"><span data-stu-id="b6b3e-104">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="b6b3e-105">Bir sınıfta yinelenen dönüştürme yordamlarına izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="b6b3e-105">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="b6b3e-106">Aşağıdaki örnek CS0557 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b6b3e-106">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
