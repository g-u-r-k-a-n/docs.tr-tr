# <a name="compiler-error-cs0228"></a>Derleyici hatası CS0228
' Type ' bir ' Member ' tanımı içermiyor veya erişilebilir değil  
  
 System. **Hashtable**, **System. String**veya **System. Array**sistem sürümünü değiştirmiş olabilirsiniz ve değiştirme içermez `member` .  
  
 Aksi takdirde, Visual Studio 'Yu onarmanız veya yeniden yüklemeniz gerekebilir.
