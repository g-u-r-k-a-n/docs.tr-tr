<span data-ttu-id="d63e4-105"><xref:System.Diagnostics.ConditionalAttribute> [Override](../language-reference/keywords/override.md) anahtar sözcüğüyle işaretlenmiş bir yöntemde özniteliğe izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="d63e4-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="d63e4-106">Daha fazla bilgi için bkz. [geçersiz kılmayı ne zaman kullanacağınızı bilme ve yeni anahtar sözcükler](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="d63e4-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="d63e4-107">Derleyici hiçbir şekilde geçersiz kılma yöntemlerine bağlanmaz.</span><span class="sxs-lookup"><span data-stu-id="d63e4-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="d63e4-108">Yalnızca temel yönteme bağlanır ve ortak dil çalışma zamanı, uygun şekilde geçersiz kılma işlemini çağırır.</span><span class="sxs-lookup"><span data-stu-id="d63e4-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="d63e4-109">Aşağıdaki kod CS0243 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d63e4-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
