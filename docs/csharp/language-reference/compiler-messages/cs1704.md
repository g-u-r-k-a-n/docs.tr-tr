# <a name="compiler-error-cs1704"></a>Derleyici Hatası CS1704

Aynı ' derleme adı ' basit adına sahip bir derleme zaten içeri aktarılmış. Başvurulardan birini kaldırmayı deneyin veya yan yana etkinleştirmek için bunları imzalayın.  
  
 Bu hata, söz konusu derlemeler tanımlayıcı adlara sahip olmadığından ve derleyicinin meta verilerde aralarında ayrım yaptığından, bu iki başvuruyu aynı derleme kimliğine sahip olduğunu gösterir. Bu nedenle, çalışma zamanı sürüm ve kültür derleme adı özelliklerini yoksayar. Kullanıcı, gereksiz başvuruyu kaldırmalı, başvurulardan birini yeniden adlandırmalı ya da bunlar için tanımlayıcı bir ad sağlamalıdır.  
  
## <a name="example"></a>Örnek  

 Bu örnek, bir derleme oluşturur ve kök dizine kaydeder.  
  
```csharp  
// CS1704_a.cs  
// compile with: /target:library /out:c:\\cs1704.dll  
public class A {}  
```  
  
## <a name="example"></a>Örnek  

 Bu örnek, önceki örnekle aynı ada sahip bir derleme oluşturur, ancak bunu farklı bir konuma kaydeder.  
  
```csharp  
// CS1704_b.cs  
// compile with: /target:library /out:cs1704.dll  
public class A {}  
```  
  
## <a name="example"></a>Örnek  

 Bu örnek her iki derlemeye de başvurulmasına çalışır. Aşağıdaki örnek CS1704 oluşturur.  
  
```csharp  
// CS1704_c.cs  
// compile with: /target:library /r:A2=cs1704.dll /r:A1=c:\\cs1704.dll  
// CS1704 expected  
extern alias A1;  
extern alias A2;  
```
