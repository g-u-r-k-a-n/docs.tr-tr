# <a name="property-parameters-cannot-be-declared-byref"></a>\<Property> Parametreler ' ByRef ' olarak bildirilemez

Özellikler için parametreler değiştirici kullanılarak bildirilmelidir `ByVal` .  
  
 **Hata kimliği:** BC30651  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Özellik parametrelerini olarak tanımlayın `ByVal` .
