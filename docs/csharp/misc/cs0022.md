# <a name="compiler-error-cs0022"></a>Derleyici hatası CS0022
[] İçinde yanlış sayıda dizin var; ' number ' bekleniyor  
  
 Dizi erişim işlemi, köşeli ayraç içinde yanlış sayıda boyut belirtti. Daha fazla bilgi için bkz. [diziler](../programming-guide/arrays/index.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0022 oluşturur:  
  
```csharp  
// CS0022.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        int[] a = new int[10];  
        a[0] = 0;     // single-dimension array  
        a[0,1] = 9;   // CS0022, the array does not have two dimensions  
    }  
}  
```
