# <a name="end-module-must-be-preceded-by-a-matching-module"></a>' End Module ' öncesinde eşleşen bir ' Module ' olmalıdır

Modül blokları bir ifadesiyle başlar `Module` ve bir yapı ile biter `End Module` .  
  
 **Hata kimliği:** BC30622  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Modül bloğunun bir ifadesiyle başladığından emin olun `Module` .
