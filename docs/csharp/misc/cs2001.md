# <a name="compiler-error-cs2001"></a>Derleyici hatası CS2001

' Dosya ' kaynak dosyası bulunamadı  
  
 Derleyiciye kaynak dosya adı geçirildi, ancak bulunamadı. Dosya adının ve dosyanın konumunun yazımını denetleyin.
