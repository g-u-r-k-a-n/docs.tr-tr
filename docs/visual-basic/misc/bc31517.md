# <a name="type-typename-cannot-be-used-as-an-attribute-because-its-container-containername-is-not-declared-public"></a>' ' \<typename> Kapsayıcısı ' \<containername> Public ' olarak bildirilmemiş olduğundan ' ' türü öznitelik olarak kullanılamaz

Bu özniteliğin tanımlandığı sınıf veya modül değiştirici kullanılarak bildirilmemiş `Public` . Bir erişim değiştiricisi belirtmeyen sınıflar ve modüller varsayılan olarak olarak bildirilmiştir `Friend` .  
  
 **Hata kimliği:** BC31517  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `Public`Değiştiricisini bu özniteliğin tanımlandığı sınıfa veya modüle ekleyin.
