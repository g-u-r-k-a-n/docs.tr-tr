## <a name="remarks"></a><span data-ttu-id="564b2-178">Açıklamalar</span><span class="sxs-lookup"><span data-stu-id="564b2-178">Remarks</span></span>  
 <span data-ttu-id="564b2-179">`<Assembly>` öğesi bir derlemedeki tüm türler için çalışma zamanı ilkesini tanımlar.</span><span class="sxs-lookup"><span data-stu-id="564b2-179">The `<Assembly>` element defines runtime policy for all the types in an assembly.</span></span> <span data-ttu-id="564b2-180">Bu, bir kitaplığı belirten, ancak çalışma zamanı yansıtma ilkesini tanımlamak için alt öğelerine bağımlı [\<kitaplığı >](library-element-net-native.md) öğesinden farklıdır.</span><span class="sxs-lookup"><span data-stu-id="564b2-180">It differs from the [\<Library>](library-element-net-native.md) element, which specifies a library but depends on its child elements to define runtime reflection policy.</span></span> <span data-ttu-id="564b2-181">`<Assembly>` öğesi bir alt öğe tarafından geçersiz kılınmadıkça derlemedeki tüm türlere uygulanır.</span><span class="sxs-lookup"><span data-stu-id="564b2-181">The `<Assembly>` element applies to all the types in an assembly unless they are overridden by a child element.</span></span>  
  
 <span data-ttu-id="564b2-182">Aşağıdaki örnek, `Name` özniteliği bir "\* Application\*" değeri atayarak uygulama paketinizdeki derlemelerdeki tüm türlere çalışma zamanı ilkesini nasıl uygulayacağınızı gösterir.</span><span class="sxs-lookup"><span data-stu-id="564b2-182">The following example shows how you can apply runtime policy to all the types in assemblies within your app package by assigning the `Name` attribute a value of "\*Application\*".</span></span> <span data-ttu-id="564b2-183">`<Assembly>` öğesi [\<uygulama >](application-element-net-native.md) öğesinin bir alt öğesi olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="564b2-183">The `<Assembly>` element must be a child of the [\<Application>](application-element-net-native.md) element.</span></span>

     <Assembly Name="*Application*" Dynamic="Required All" />   
  </Application>
