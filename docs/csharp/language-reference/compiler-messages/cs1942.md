## <a name="example"></a><span data-ttu-id="f0807-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="f0807-110">Example</span></span>

<span data-ttu-id="f0807-111">Aşağıdaki kod CS1942 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f0807-111">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="f0807-112">CS1942, CS1949 ile ilgilidir çünkü `var` bir Aralık değişkeni ile kullanılması temeldeki `Cast<T>` işlemin başarısız olmasına neden olur çünkü `var` bir tür değildir.</span><span class="sxs-lookup"><span data-stu-id="f0807-112">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
