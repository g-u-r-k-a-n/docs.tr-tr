# <a name="compiler-error-cs1023"></a>Derleyici hatası CS1023
Gömülü ifade bir bildirim veya etiketli bir ifade olamaz  
  
 Bir **if** deyimini izleyen deyimler gibi gömülü bir deyim, hiçbir bildirim veya etiketli deyim içerebilir.  
  
 Aşağıdaki örnek, iki kez CS1023 üretir:  
  
```csharp  
// CS1023.cs  
public class a  
{  
   public static void Main()  
   {  
      if (1)  
         int i;      // CS1023, declaration is not valid here  
  
      if (1)  
         xx : i++;   // CS1023, labeled statement is not valid here  
   }  
}  
```
