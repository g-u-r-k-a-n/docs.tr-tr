# <a name="withevents-variables-can-only-be-typed-as-classes-interfaces-or-type-parameters-with-class-constraints"></a>' WithEvents ' değişkenleri yalnızca sınıf kısıtlamalarına sahip sınıflar, arabirimler veya tür parametreleri olarak yazılabilir

`WithEvents`' İn geçerli bir değiştirici kullanımı olmayan, ile birlikte bir yapı olarak yazılmış bir değişken bildirdiniz `WithEvents` .  
  
 **Hata kimliği:** BC30413  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. `AddHandler`Bir yapıda tanımlanan olayları işlemek için kullanın.
