# <a name="repair-the-net-framework"></a>.NET Framework’ü Onarma

Bazı durumlarda,.NET Framework yüklemeniz zarar görebilir ve onarım gerektirebilir. Uygulamanız başlatmayı denedikten hemen sonra çöküyorsa veya daha yeni .NET Framework sürümleri yükleyemiyorsanız bu durum söz konusu olabilir.

[.NET Framework Repair Tool](https://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe)'u kullanarak .NET Framework yüklemenizi onarabilirsiniz.

Uygulamanız .NET Framework'u onardıktan sonra hala başlatılmıyorsa, uygulamanın bir sorunu olabilir. Bu durumda, uygulama yayımcısına başvurmalısınız.
