# <a name="name-cannot-be-named-as-a-parameter-in-an-attribute-specifier-because-it-is-not-a-field-or-property"></a>' \<name> ' bir alan veya özellik olmadığından, öznitelik tanımlayıcıda parametre olarak adlandırılamaz

Öznitelik bloğu özniteliğin değişken olmayan üyesi için bir değer ayarlar. Yalnızca alanlar veya özellikler gibi değişken üyelere değer atayabilirsiniz.  
  
 **Hata kimliği:** BC32010  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Özniteliğin ve üye adlarının doğru yazıldığından emin olun.  
  
2. Üye değişken olmayan bir değişkense öznitelik bloğundan bağımsız değişkeni kaldırın.
